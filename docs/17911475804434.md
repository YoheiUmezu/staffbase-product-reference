# ユーザープロファイルからMicrosoft Teamsを使用する

**URL**: https://support.staffbase.com/hc/ja/articles/17911475804434
**取得日**: 2026-05-02 18:49:15 JST

## 内容

アクティベーション
デスクトップまたはモバイルアプリのStaffbaseプラットフォームから、Microsoft Teamsでシームレスにコラボレーションできます。同僚のユーザープロファイルにあるMicrosoft Teamsのドロップダウンを使用すると、簡単にチャット、音声通話、またはビデオ通話を開始できます。
前提条件
再設計されたユーザープロファイルページ（ベータ、ヘッダー画像で認識可能）がアクティブになっていること。
Staffbaseプラットフォームから Microsoft 365にサインインしていること。
連絡を取りたい同僚が、Staffbaseプラットフォーム上で有効なMicrosoftメールアドレスを公開メールアドレスとして持っていること。
Teamsでのコラボレーション があなたのプラットフォームでアクティブになっていること。
Microsoft Teamsのドロップダウンは他のユーザーのプロファイルにのみ表示されます。自分のユーザープロファイルには表示されません。
ユーザープロファイルからMicrosoft Teamsでユーザーとチャットする
アプリまたはイントラネットで、コラボレーションしたいユーザーのユーザープロファイルに移動します。
ユーザーの名前の下にある Microsoft Teams ドロップダウンをクリックします。
もし Microsoft Teams ドロップダウンが表示されない場合、以下の理由が考えられます：
連絡を取るユーザーのMicrosoftアカウントが、Staffbaseプラットフォームの公開メールアドレスフィールドに追加されていない。
どちらかが有効なMicrosoft Teamsライセンスを持っていない。
チャットをクリックします。
Microsoft Teamsのチャットウィンドウが開きます。
これで同僚にメッセージを入力できます。
ユーザープロファイルからMicrosoft Teamsでユーザーに電話する
アプリまたはイントラネットで、コラボレーションしたいユーザーのユーザープロファイルに移動します。
ユーザーの名前の下にある Microsoft Teams ドロップダウンをクリックします。
もし Microsoft Teams ドロップダウンが表示されない場合、以下の理由が考えられます：
連絡を取るユーザーのMicrosoftアカウントが、Staffbaseプラットフォームの公開メールアドレスフィールドに追加されていない。
どちらかが有効なMicrosoft Teamsライセンスを持っていない。
電話をクリックします。
Microsoft Teamsのウィンドウが確認ダイアログと共に開きます。
通話を開始をクリックします。
これでMicrosoft Teamsで同僚と話すことができます。
ユーザープロファイルからMicrosoft Teamsでユーザーにビデオ通話する
アプリまたはイントラネットで、コラボレーションしたいユーザーのユーザープロファイルに移動します。
ユーザーの名前の下にある Microsoft Teams ドロップダウンをクリックします。
もし Microsoft Teams ドロップダウンが表示されない場合、以下の理由が考えられます：
連絡を取るユーザーのMicrosoftアカウントが、Staffbaseプラットフォームの公開メールアドレスフィールドに追加されていない。
どちらかが有効なMicrosoft Teamsライセンスを持っていない。
ビデオ通話をクリックします。
Microsoft Teamsのウィンドウが確認ダイアログと共に開きます。
通話を開始をクリックします。
これでMicrosoft Teamsで同僚と見ながら話すことができ、関連するすべての機能を使用できます。
