# CSVインポートのレビュー

**URL**: https://support.staffbase.com/hc/ja/articles/24593148323090
**取得日**: 2026-05-02 18:53:25 JST

## 内容

ベータ
アクティベーション
管理者
実際のCSVインポートの前に、インポートの詳細をレビューおよび編集できます。
プレビューの表示
プレビュー準備完了のステータスのファイルのみプレビューできます。
スタジオで、ユーザー > ユーザーインポートと同期に移動します。
インポートしたいファイルに対応する三点リーダーのアクションメニューから、プレビューを表示をクリックします。
プレビューを表示オプションは、プレビュー準備完了のステータスのファイルにのみ表示されます。
新しいインポートページが開き、インポートに伴う変更の概要が表示されます。詳細を確認し、すべてが正しいかどうかを確認できます。
次のステップ
プレビューを確認した後、次のいずれかを行うことができます：
設定を編集
マッピングを編集
ファイルをインポート
設定の編集
スタジオで、ユーザー > ユーザーインポートと同期に移動します。
インポートしたいファイルに対応する三点リーダーのアクションメニューから、設定を編集をクリックします。
設定を編集オプションは、プレビュー準備完了のステータスのファイルにのみ表示されます。
好みに応じて設定を編集します。
次へをクリックします。
マッピングページが開きます。
オプションで、好みに応じてマッピングを編集します。
設定を編集しました。これで、変更を確認するためのプレビューを作成するか、ファイルをインポートすることができます。
マッピングの編集
スタジオで、ユーザー > ユーザーインポートと同期に移動します。
インポートしたいファイルに対応する三点リーダーのアクションメニューから、マッピングを編集をクリックします。
マッピングを編集オプションは、プレビュー準備完了のステータスのファイルにのみ表示されます。
好みに応じてマッピングを編集します。
マッピングを編集しました。これで、変更を確認するためのプレビューを作成するか、ファイルをインポートすることができます。
