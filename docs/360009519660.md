# オープングループの作成

**URL**: https://support.staffbase.com/hc/ja/articles/360009519660
**取得日**: 2026-05-02 18:50:22 JST

## 内容

管理者
オープングループは、興味のあるグループをサポートしたり、場所に基づく情報を提供したり、会社全体の透明性を促進したりする素晴らしい機会です。ユーザーはオープングループの選択ツールを通じて、グループコンテンツに自由に登録および登録解除できます。これにより、ユーザーはアプリをパーソナライズでき、ユーザー管理の手間を省くことができます。
スタジオで、ユーザー設定>ユーザーグループに移動します。
グループを作成をクリックします。
ユーザーグループを作成するためのダイアログが開きます。
グループの名前を入力します。
ドロップダウンメニューからタイプオープングループを選択します。
ドロップダウンメニューから希望の可視性を選択します。
グループの可視性を制限すると、定義された可視性内のユーザーのみが登録できるようになります。これは、多くのオープングループがある場合に便利です。たとえば、関連する場所によってオープングループの可視性を制限すると、その場所にいる人だけがグループを表示できるようになります。
作成をクリックします。
オープングループが作成されました。これで、グループ設定を編集し、メンバーを手動でグループに追加できます。
オープングループは事前に入力することはできません。ユーザーを手動で追加でき、ユーザー自身がオープングループの選択ツールを通じて登録を選択できます。
