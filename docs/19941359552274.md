# Microsoft StoreからStaffbase Connectアプリを追加する

**URL**: https://support.staffbase.com/hc/ja/articles/19941359552274
**取得日**: 2026-05-02 18:49:06 JST

## 内容

アクティベーション
ボタンをクリックするだけで、Microsoft Teams内から直接Staffbase従業員アプリにアクセスできます。常に最新の情報を把握し、組織内で何が起こっているかを把握できます。次の方法でMicrosoft Teamsに従業員アプリを追加できます：
カスタムアプリとして追加する
Microsoft Storeから追加する
Staffbaseは、組織がMicrosoft Teams内のStaffbase従業員アプリに新しいカスタムウィジェットを許可リストに追加する必要がある場合、カスタムアプリの使用を推奨します。
前提条件
Microsoft StoreからMicrosoft Teamsにアプリを追加する権限があります。
Microsoft TeamsでのStaffbase Connectアプリのリクエスト
Microsoft Teamsの左側のナビゲーションペインで、アプリをクリックします。
Staffbase Connectアプリを検索します。
Staffbase Connectアプリのリンクをクリックします。
アプリの説明ページが開きます。
承認をリクエストをクリックします。
Staffbase ConnectアプリをMicrosoft Teamsに追加するリクエストをしました。組織のMicrosoft管理者にリクエストが通知されます。彼らはリクエストを承認し、権限を付与し、ポリシーを設定します。これが完了すると、アプリが左側のナビゲーションペインに表示されます。これには数時間かかる場合があります。その後、従業員アプリにログインできます。
