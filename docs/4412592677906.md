# 従業員メールの設定変更

**URL**: https://support.staffbase.com/hc/ja/articles/4412592677906
**取得日**: 2026-05-02 18:52:00 JST

## 内容

製品名に関する注意: 2022年以降、BananatagはStaffbaseの従業員メール製品です。
正確で役立つメトリクスを収集するために、タイムゾーンやアドイン/拡張機能の設定を調整してください。
設定によっては、同じメニューを使用して追加の送信エイリアスアドレスを管理できる場合もあります。
メトリクスのためのタイムゾーン選択
従業員メールを、あなたのローカルタイムまたは受信者のいるローカルタイムでメトリクスを表示するように設定します:
従業員メールウェブアプリで、イニシャルを含むアバターをクリックします。
設定 > アカウントに移動します。
ドロップダウンからタイムゾーンを選択します。
保存して更新をクリックします。
アドインと拡張機能の設定
Outlook Classic
OutlookリボンでStaffbaseで作成をクリックします。
保存したEmail Designerのドラフトのリストが表示されます。
ギアホイールをクリックして設定メニューを開きます。
オプションで、以下の設定を調整します:
トラッキングを有効にする
Outlookで作成するすべてのメールに対してメールを追跡設定を自動的に有効にします。
リンクを追跡を有効にする
メール内の各リンクへのクリック数を測定します。
リストBCCを有効にする
送信された各メールの統計ダッシュボードにBCCされたアドレスを自動的に追加します。
このリストには最大30のメールアドレスしか含めることができません。
受信者を自動カウント
Staffbaseが送信された各メールの受信者数をカウントできるようにします。
受信者のカウントは、リストの設定によって送信の遅延を引き起こす可能性があります。
受信者が1人だけの場合は、Staffbaseサポートまたはアカウントマネージャーに連絡してください。
受信者数は、メールメトリクス内のメールサマリーレポートで手動で入力することもできます。
xをクリックしてメニューを閉じます。
設定は自動的に保存されます。
Outlook 365
新しいドラフトメールの従業員メールサイドバーで、ギアホイールをクリックして設定メニューを開きます。
オプションで、以下の設定を調整します:
一般 > リンクを追跡
メール内の各リンクへのクリック数を測定します。
エイリアス > エイリアスを追加
このオプションがインストールされたアドインのバージョンに表示されている場合は、追加の送信エイリアスアドレスを管理するために使用します。
最新のOutlook 365アドインのバージョンを有効にした場合、エイリアスの選択と管理のためのツールが変更されました。
何が異なるかを学び、どのように試すことができるかを確認してください。
xをクリックしてメニューを閉じます。
設定は自動的に保存されます。
Gmail
Gmailで設定をクリックします。
サイドバーにメニューオプションのセットが開きます。
オプションで一般をクリックし、以下の設定を調整します:
トラッキングチェックボックス
Gmailで作成するメールのデフォルトのトラッキング状態を選択します。
リンクを追跡
メール内の各リンクへのクリック数を測定します。
ドラフトサイズ警告を表示
Gmailは102 KBの厳格な最大メールメッセージサイズを強制します。
警告を有効にしてメッセージがクリップされるのを防ぎます。
連絡先へのアクセスを許可
設定とキャッシュ
デフォルトにリセットをクリックして、拡張機能のオプションをリセットし、エイリアス設定をクリアします。
Gmailに複数の追加送信エイリアスアドレスが設定されている場合は、設定 > エイリアスマネジメントに移動します。
xをクリックしてメニューを閉じます。
設定は自動的に保存されます。
