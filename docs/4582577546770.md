# 従業員のメールユーザーの編集

**URL**: https://support.staffbase.com/hc/ja/articles/4582577546770
**取得日**: 2026-05-02 18:53:53 JST

## 内容

マネージャー
あなたが利用できる他の 従業員メール ユーザーの設定を編集し（アカウントの種類に応じて）、オプションでアクセスを削除できます。
注意: スタンドアロンのメールの場合、従業員メール ユーザーを無効化することができます。
無効化されたユーザーのリソース（配信リスト、メールデザイナのドラフト、またはテンプレート）は、自動的にそれらを無効化したアカウントに移転します。
統合された アプリ/イントラネット のユーザー管理
Staffbase アプリ/イントラネット が 従業員メール と統合されている場合、両方の アプリ/イントラネット と メール にアクセスできるユーザーを削除する必要がある場合は、あなたの アプリ/イントラネット から削除してください。詳細については、Staffbaseサポートまたはカスタマーサクセスマネージャーにお問い合わせください。
統合された アプリ/イントラネット と メール のユーザーを管理する際、スタジオのメインユーザー設定メニューとこのメニューに埋め込まれたチームページは異なる機能を果たします。
次の スタジオ を使用して:
ユーザーを追加
ユーザーを削除
ユーザー情報を更新
次のチームページを使用して:
有効な場合、従業員メール におけるユーザーの機能アクセス 設定を管理し、メールの送信やスタイリング、レポート生成のオプションを設定します。
メールのコラボレーションのためのグループメンバーシップを編集します。
他のユーザーとしてサインイン（例えば、共有メールアドレス用に作成された共有ユーザーアカウント）
ユーザーの編集
チームページで、編集したいユーザーの三点リーダーをクリックします。
ユーザーを編集をクリックして、設定を変更します。
アカウントの種類に応じてアクセス可能なユーザーの詳細を編集します。
保存して更新をクリックします。
機能アクセス設定
有効な場合、これらの設定により、個々のユーザーの権限をカスタマイズし、アカウントの種類に対するデフォルト設定を超えたより具体的なアクセスや制限を追加できます。
詳細については、Staffbaseサポートまたはカスタマーサクセスマネージャーにお問い合わせください。

メールドラフトとテンプレート
デフォルトでは、すべての 従業員メール アカウントタイプは、メールデザイナーでドラフトを作成および編集する能力を持っています。
特定の状況では、ユーザーがメールコンテンツを追加または変更する能力を制限することが有益な場合があります。
ユーザーのアクセスを制限するには、作成/編集とラベル付けされたボックスのチェックを外します。
レポート
ユーザーがレポートを作成および/またはエクスポートする能力を有効または制限するには、レポート生成とラベル付けされたボックスのチェックを入れるか外します。
メールデザイナー
組織内のユーザーが、メールデザイナーのドラフトやテンプレートでデフォルトのブランドに沿った色とフォントのみを使用できるように制限できます。
この制限を解除するには、スタイリングアクセスとラベル付けされたボックスにチェックを入れます。
連絡先と送信
連絡先と送信の権限がない場合、ユーザーは既に所有している配信リストや共有されているリストの表示名のみを表示または編集できます。
ユーザーにこの制限されたレベルのアクセスを与えるには、連絡先と送信へのアクセスとラベル付けされたボックスのチェックを外します。
ユーザーが所有している配信リストや共有されているリストを表示または編集できるようにしつつ、組織のすべての連絡先にアクセスできないように安全に防ぐことができます。
ユーザーにこの制限されたレベルのアクセスを与えるには、すべての連絡先を表示とラベル付けされたボックスのチェックを外します。
ユーザーがEmployee Emailに連絡先をインポートする方法を選択します。
ユーザーがCSVファイルを介して連絡先リストをアップロードできるようにするには、Upload CSVとラベル付けされたボックスにチェックを入れるか、チェックを外してください。
ユーザーが統合の作成と編集を行うことを許可しながら、フィールドマッピングと除外の編集を制限することができます。
これらの権限の両方が、ユーザーがEmployee EmailをMicrosoft Entra ID、Workday、またはStaffbaseApp/Intranetに接続するために必要です。
詳細については、Staffbaseサポートまたはカスタマーサクセスマネージャーにお問い合わせください。
ユーザーがドラフトメールを送信またはスケジュールする能力を有効にするか制限するには、Send/Schedule emailsとラベル付けされたボックスにチェックを入れるか、チェックを外してください。
