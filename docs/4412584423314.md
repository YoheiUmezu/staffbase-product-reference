# ウェブアプリでのバウンスしたメール連絡先の修正

**URL**: https://support.staffbase.com/hc/ja/articles/4412584423314
**取得日**: 2026-05-02 18:57:10 JST

## 内容

バウンスは、さまざまな理由で、Employee Emailが特定のメールアドレスにメールメッセージを配信できない場合に記録されます。
ハードバウンスは、受信者のアドレスが無効であるために、メールメッセージが送信者に返送されるときに発生します。受信者のメールサーバーがハードバウンスと見なす条件は、設定によって異なります。
ソフトバウンスは、受信ボックスが満杯である、接続が確立できない、またはその他の一時的な技術的問題など、明らかに一時的な条件のためにサーバーがメールを拒否する場合に発生します。
ソフトバウンスはバウンスとしてマークされません。代わりに、Employee Emailは、メッセージが配信されるまで、または受信者のメールサーバーがハードバウンスを返すまで、メッセージの配信を試み続けます。
Employee Emailウェブアプリの機能の一部として、自分自身の配信リストを作成および管理することができ、メール統計でバウンスを返す受信者のメールアドレスに遭遇することがあります。
ウェブアプリの送信が有効になっている場合、アプリは特定のメールに対するすべてのハードバウンスした連絡先のレポートを提供し、これらの問題のある連絡先をダウンロードする方法と修正手順を提供します。
バウンスした連絡先リストに含まれる連絡先は、ディレクトリからフィルタリングされるため、送信しても安全とマークされない限り、今後の通信には含まれません。
バウンスしたレコードを送信しても安全とマークすることの意味は？
バウンスしたレコードを送信しても安全とマークすることは、この受信者のメールアドレスが正しいことを示し、無効なメールアドレス以外の理由でバウンスが発生したことを示します。
受信者のメールアドレスが正しく入力されていることを確認でき、メールがバウンスした理由が一時的および/または断続的な技術的問題である可能性が高い場合は、これを行うことを検討してください。
