# 公開エリアでのコンテンツの表示

**URL**: https://support.staffbase.com/hc/ja/articles/360001297552
**取得日**: 2026-05-02 18:50:35 JST

## 内容

公開エリアで利用可能なコンテンツは、Staffbaseプラットフォームにサインインしている従業員以外のすべてのオーディエンスがアクセスできます。
コンテンツを公開する
管理エディター
デフォルトでは、新しいコンテンツはサインインしているユーザーにのみ表示されます。サインインしていないユーザーに表示するために、公開エリアに配置するコンテンツを選択できます。
スタジオで、コンテンツに移動します。
公開したいページ、ニュースチャンネル、またはプラグインに移動します。
設定を開くためにギアアイコンをクリックします。
設定ページで、トグル公開エリアでの表示を調整してオプションを有効にします。
保存をクリックします。
公開エリアでコンテンツが表示されるようになりました。
公開エリアのプレビューを表示する
管理者
Staffbaseプラットフォームの設定で、すべての公開コンテンツのプレビューを見ることができます。
スタジオで、設定 > 公開エリアに移動します。
公開エリアページが開きます。
