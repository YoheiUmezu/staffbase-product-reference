# 概要ページからキャンペーンを分析する

**URL**: https://support.staffbase.com/hc/ja/articles/26111417873298
**取得日**: 2026-05-02 18:55:07 JST

## 内容

キャンペーンマネージャー
キャンペーンの概要ページにあるアナリティクスサイドパネルは、ページを離れることなく主要なキャンペーンの指標を素早く簡単に確認できる方法を提供します。キャンペーンに割り当てられたニュース投稿記事やメールの重要なアナリティクスが表示されます。さらに、より詳細なインサイトが必要な場合は、サイドパネルから直接キャンペーン詳細や、組織で利用可能な場合はSmart Insightsダッシュボードにアクセスできます。
アナリティクスサイドパネルは、少なくとも1つのニュース投稿記事またはメールが割り当てられているキャンペーンでのみ利用できます。
サイドパネルの指標
ご利用の製品パッケージや組織で有効化されている追加機能によって、アナリティクスサイドパネルに表示される指標が異なります。

アナリティクスサイドパネルへアクセスする
Studioでキャンペーン > 指標を確認したいキャンペーンに移動します。
インサイトアイコンをクリックします。.
アナリティクスサイドパネルが開きます。
これでサイドパネルでキャンペーンの指標にアクセスできました。さらに詳細なインサイトが必要な場合は、キャンペーン詳細を開くまたはアナリティクスを開くをクリックしてください（組織で利用可能な場合）。キャンペーン詳細ページまたはSmart Insightsダッシュボードが開きます。
