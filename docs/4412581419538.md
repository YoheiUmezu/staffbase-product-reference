# 従業員メールAPIの課題解決

**URL**: https://support.staffbase.com/hc/ja/articles/4412581419538
**取得日**: 2026-05-02 19:01:56 JST

## 内容

製品名に関する注意: 2022年以降、BananatagはStaffbaseの従業員メール製品です。
一部の組織では、ユーザーのアクセスを制限し、APIアクセスには管理者の承認を必要とする場合があります。
このオプションはREST APIとして表示されるべきですが、これを行う手順は管理者インターフェースによって異なる場合がありますので、これらの変更を行っても課題が解決しない場合は、Staffbaseサポートにお問い合わせください。
