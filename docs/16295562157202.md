# SMSメッセージの作成と送信

**URL**: https://support.staffbase.com/hc/ja/articles/16295562157202
**取得日**: 2026-05-02 18:56:51 JST

## 内容

ベータ
アクティベーション
管理者
イントラネットおよび従業員アプリでSMSを使用することで、デスクレスおよびフロントラインの作業者と直接的に関わるコミュニケーションが実現します。StudioへのシームレスなTwilio統合により、プラットフォームを離れることなくSMSメッセージを簡単に送信でき、効率的で便利なコミュニケーションが保証されます。個々のユーザーやユーザーグループ全体にSMSを送信できます。
SMSサービスを使用する際には、追加のサードパーティコストが発生することに注意してください。これらの費用は当社の管理外であり、関連する料金を理解し管理する責任はお客様にあります。Staffbaseはサードパーティの料金に対して責任を負わず、潜在的な追加コストを評価する際には注意を払うことをお勧めします。
前提条件
TwilioでSMSサービスが完全に設定されていること。
StudioでSMSコミュニケーションが構成されていること。
受信者がSMSを受け取ることに同意していること。
受信者が構成されたプロファイルフィールドに有効な電話番号を持っていること（E.164形式）。
SMSの文字数制限、エンコーディング、およびメッセージの長さの制限を理解していること。詳細はTwilioのドキュメントで確認してください。
手順
StudioでSMS > SMSを作成に移動します。
SMSを作成をクリックします。
SMSを作成ページが開きます。
メッセージの下に、送信するSMSメッセージを記入します。
注意: メッセージに完全なURLを含めることができますが、これによりメッセージ全体の長さが延びる可能性があることを忘れないでください。
表示されている送信者の電話番号を再確認します。
受信者の下で、ユーザーのドロップダウンから個々の受信者を選択します。
注意: 複数のユーザーを選択する前に、限られたユーザー（できれば自分自身）でテストしてください。メッセージが送信されたことを確認するために、自分自身を受信者として追加します。
オプションで、グループのドロップダウンから受信者として全ユーザーグループを選択します。
送信をクリックします。
確認メッセージが表示されます。
今すぐ送信をクリックします。
成功した送信の通知が表示されます。
メッセージは選択した受信者に送信されます。
メッセージページは送信後すぐにクリアされます。メッセージの履歴とログは、Twilioコンソールのモニター > ログ > メッセージングで確認できます。
