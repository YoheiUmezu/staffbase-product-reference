# タスクハブウィジェットのoverview

**URL**: https://support.staffbase.com/hc/ja/articles/31917314395538
**取得日**: 2026-05-02 18:45:37 JST

## 内容

アクティベーション
タスクウィジェットは、従業員が参加しているすべてのタスクプロジェクトへの中央の入り口を提供します。特に複数のチーム、プロジェクト、拠点で働くユーザーが、メニューを移動せずにプロジェクト間を素早く切り替えてアクセスできるようにします。
ユーザーは自分が閲覧権限を持つプロジェクトのみを表示します
各プロジェクトは選択可能なエントリーポイントとして表示されます
プロジェクトを選択すると、そのタスクビューが開きます
ユーザーは簡単にハブに戻り、他のプロジェクトに切り替えることができます
タスクハブウィジェットのメリット
タスクプロジェクトへの集中かつ効率的なアクセス
ウィジェットは、割り当てられたすべてのタスクプロジェクトへの単一のアクセス拠点を提供し、ユーザーがプロジェクト間を素早く切り替えて、タスクをより効率的に完了できるようにします。
パーソナライズされたプロジェクトの表示
プロジェクトはユーザーのアクセス権に基づいて自動的にフィルタリングされ、従業員は自分に関連するプロジェクトのみを表示します。
業務担当者の効率向上
タスクハブは、ユーザーがタスクや業務の状況を頻繁に切り替える場合でも、迅速な意思決定をサポートします。
タスクハブウィジェットのユーザー向け表示
タスクハブは、Staffbaseの任意のページに追加することができるウィジェットです。ログインしている各ユーザーに、割り当てられているプロジェクトの概要を表示します。プロジェクトを選択すると、直接タスク内で開き、ユーザーはすぐにタスク作業を開始できます。
