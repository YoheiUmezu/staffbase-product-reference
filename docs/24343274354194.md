# Staffbase Emailの追加ブランドカラーの追加

**URL**: https://support.staffbase.com/hc/ja/articles/24343274354194
**取得日**: 2026-05-02 18:56:40 JST

## 内容

管理者
Staffbase Emailのために追加のブランドカラーを追加できます。これらのカラーは、カラー設定が利用可能な場所で追加された順に表示されます。
スタジオで、設定 > メールに移動します。
カラーをクリックします。
カラーページが開きます。
カラーを追加をクリックします。
カラー名と値の新しいフィールドが表示されます。
カラー名の下に、カラーの名前を入力します。
対応する値フィールドに、カラーの16進コードを入力します。
保存をクリックします。
Staffbase Emailのために追加のデフォルトブランドカラーを正常に追加しました。最初の2つのデフォルトカラーとは異なり、追加のデフォルトカラーを削除したり、任意の数の追加カラーを追加したりできます。
