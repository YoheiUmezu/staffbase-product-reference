# クイズカレンダーの設定の構成

**URL**: https://support.staffbase.com/hc/ja/articles/4411480464402
**取得日**: 2026-05-02 18:52:02 JST

## 内容

クイズカレンダープラグインを使用すると、各クイズを組織の特定の好みに合わせて構成できます。特定のクイズの設定を変更して、クイズの外観、結果が表示されるタイミング、持続時間を調整できます。
前提条件
クイズカレンダープラグインをインストールしています。
クイズカレンダーのインスタンスを少なくとも1つ設定しています。
カスタム背景画像の追加
スタジオで、コンテンツ > クイズカレンダーに移動します。
カスタマイズしたいクイズカレンダーを開き、設定をクリックします。
オプションカスタム画像を選択します。
画像をアップロードをクリックします。
コンピュータからアップロードしたい画像を選択します。
画像はクイズカレンダーの中央に表示されます。同じ画像がモバイルとデスクトップの画面で使用されるため、最も重要な視覚要素が画像の中心に配置されていることを確認してください。
保存をクリックします。
クイズカレンダーにカスタム背景画像を追加しました。これで、ユーザーに対して画像がどのように表示されるかをプレビューできます。
正解の表示を遅らせる
正解は、ユーザーが質問に回答した直後に表示されるのがデフォルトです。この機能を無効にして、競争を健全に保ち、回答が早期に共有されないようにすることができます。回答は翌日に公開されます。
スタジオで、コンテンツ > クイズカレンダーに移動します。
カスタマイズしたいクイズカレンダーを開き、設定をクリックします。
オプション回答後に正解をすぐに表示のチェックを外します。
保存をクリックします。
クイズカレンダーで正解がすぐに表示されないようにしました。ユーザーは翌日になって初めて正解を見ることができます。
ランキングタブの有効化
クイズカレンダーのランキングタブを使用すると、従業員は特定のクイズに対するポイント数と、他の同僚と比較して自分の位置を確認できます。デフォルトでは、ランキングタブは表示されません。ランキングタブをオンにして、クイズカレンダーにさらなる興奮を加えましょう。
オプション回答後に正解をすぐに表示のチェックを外した場合、ランキングタブは翌日に更新されます。
スタジオで、コンテンツ > クイズカレンダーに移動します。
カスタマイズしたいクイズカレンダーを開き、設定をクリックします。。
チェックボックスランキングを表示を選択します。
オプションで、チェックボックスランキングでユーザー名を非表示を選択します。これにより、従業員は自分の名前と同僚に対する自分の順位を確認できますが、個々の名前は表示されません。
クリックして保存します。
ユーザーがランキングタブを見る方法
期間の変更
クイズ設定内でカレンダーの期間を延長または短縮し、すべての既存コンテンツを保持します。これは、開始日または終了日を誤って入力した場合や、タイムラインを変更したい場合に便利です。
公開済みのクイズの期間を変更した後に残るもの：
ユーザーが提出した回答
ユーザーが収集したすべてのポイント
ランキングリスト
公開されたクイズの日付を変更して、将来のクイズのコピーやテンプレートを作成することは推奨されません。
期間を短縮する
クイズの最小期間は2日です。クイズの期間を短縮すると、削除された日数に対して以前に入力されたコンテンツは利用できなくなります。
スタジオで、コンテンツ > クイズカレンダーに移動します。
カスタマイズしたいクイズカレンダーを開き、設定をクリックします。。
開始日および/または終了日を短縮された日付範囲に変更します。
期間を短縮するための確認ダイアログが開きます。
クリックして短縮します。
クリックして  保存。
クイズの期間を短縮しました。削除した日数に以前入力したコンテンツはもはや利用できません。
クイズの期間を延長する
クイズの最大期間は40日です。クイズの期間を延長した後は、新たに追加された日数のコンテンツを作成する必要があります。
スタジオで、 コンテンツ > クイズカレンダーに移動します。
カスタマイズしたいクイズカレンダーを開き、設定をクリックします。.
開始日および/または終了日を短縮された日付範囲に変更します。
期間を延長するための確認ダイアログが開きます。
延長をクリックします。
保存をクリックします。
クイズの期間を延長しました。新たに追加された日数にコンテンツを追加できます。
