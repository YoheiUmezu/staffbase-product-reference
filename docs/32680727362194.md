# RTMP接続を使用したライブ配信

**URL**: https://support.staffbase.com/hc/ja/articles/32680727362194
**取得日**: 2026-05-02 18:50:47 JST

## 内容

モデレーター
Real-Time Messaging Protocol（RTMP）接続を使用してイベントをライブ配信できます。RTMPを使うと、OBSなどの配信ツールからStaffbaseへ動画信号を送信できます。イベントを作成した後、Studioのイベント概要からRTMP接続情報をコピーできます。これらの情報には、サーバーURLとストリームキーが含まれており、配信ツールに入力して動画信号を送信する必要があります。
配信ツールでストリームを開始しても、ライブ配信は自動的に開始されません。Broadcast Studioでもストリームを開始する必要があります。それまでは、受信した信号は代替入力として表示されます。
RTMP情報の入力場所が分からない場合は、ご利用の配信ツールのドキュメントを参照してください。
前提条件：
Studioでイベントがすでに作成されていること。
イベントのモデレーターであること。RTMP接続情報はモデレーターのみが確認できます。
RTMPに対応した外部配信ツールを使用していること。
Studioでコンテンツ > イベントセンター > スケジュール > コピーしたいイベントのRTMP接続に移動します。
三点リーダーをクリックしてアクションメニューを開きます。
表示と編集をクリックします。
概要タブのイベントページが開きます。
RTMP接続の下で、サーバーURLのコピーアイコンをクリックします。
サーバーURLのリンクがクリップボードにコピーされます。
ストリームキーのコピーアイコンをクリックします。
ストリームキーのリンクがクリップボードにコピーされます。
コピーしたリンクは配信ツールで使用できます。
