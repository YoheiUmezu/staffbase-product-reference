# クイズカレンダーの概要とよくある質問

**URL**: https://support.staffbase.com/hc/ja/articles/26714917474578
**取得日**: 2026-05-02 18:41:36 JST

## 内容

クイズカレンダーは、組織内のユーザーエンゲージメントを高めるために設計された、動的で魅力的なプラグインです。テーマ別のカレンダー形式で毎日クイズを提供します。これにより、従業員は毎日新しいコンテンツを楽しみに戻ってきて、重要な会社のイベントやマイルストーン、祝日までの楽しい期間を過ごすことができます。
カスタムクイズの質問、画像、またはテキストをカレンダーの各日に追加することができます。これにより、ユーザーは毎日戻ってくる理由ができ、プラットフォームを楽しく利用できます。カレンダーを公開したいタイミングに応じて、クイズカレンダーには2つのオプションがあります。
クリスマスカレンダーは、あらかじめ設定された日付範囲があります。12月1日に開始し、12月24日に終了します。クリスマスシーズンのエンゲージメントに最適で、日付が固定されているため手動でのスケジューリングは不要です。
カスタムカレンダーは、任意のタイミングで開始できるカスタマイズ可能な期間を持っています。このクイズタイプは2日から40日までの任意の期間で実施でき、さまざまな取り組みに適しています。
クイズカレンダーは非常にカスタマイズ可能です。カレンダーの各日にクイズの質問、画像、またはテキストを追加でき、組織のテーマやコミュニケーション目標に合わせてコンテンツを調整できます。さらに、プラグインには各クイズの表示方法、結果の表示タイミング、クイズの公開期間を設定するための設定があります。これらのオプションにより、利用目的に合わせてクイズ体験を調整できます。
FAQ
ランキングリストはどのように並べ替えられますか？
ユーザーは正解ごとに1ポイントを獲得します。これらのポイントは、すべてのデイリークイズと質問で累積されます。また、各質問の回答にかかった時間も記録されます。獲得したポイントと回答にかかった時間の両方に基づいてランキングが生成されます。有効にすると、上位20位までのランキングを表示できます。
ランキングはリアルタイムで更新されますか？
はい。ユーザー向けには、「回答後すぐに正解を表示」設定が有効になっている場合、ランキングはリアルタイムで更新されます。それ以外の場合は、ランキングの更新は翌日に反映されます。編集者向けには、CSVファイルとしてエクスポートすると、ランキングは常に即時に更新されます。
クイズを表示する日を選択できますか？
いいえ、クイズは連続した日でのみ設定できます。複数日間のクイズで個別の日をスキップすることはできません。
クイズはいつ開始されますか？
クイズは、各クイズで設定された日付とタイムゾーンに基づき、深夜0時に開始されます。クイズカレンダーはユーザーのローカルタイムゾーンには自動で調整されません。
