# ニュースの概要

**URL**: https://support.staffbase.com/hc/ja/articles/29227790087954
**取得日**: 2026-05-02 18:41:17 JST

## 内容

ニュースは、スタッフベースアプリおよびイントラネットで、視聴者ごとに動的なコンテンツを公開することができます。コンテンツを効率的に整理し、従業員が簡単に理解できる形式で配信します。ニュースは、最近の更新や人事の改善など、短期から中期の寿命を持つコンテンツに最適です。投稿は読者の注目を集めるように設計されています。ニュースは3層構造で整理されており、これによりコンテンツをタイムリーかつターゲットに合わせて配信できます。ニュースページ、チャンネル、投稿の関係について詳しくはこちらをご覧ください。
ニュースページ
チャンネルとその投稿を表示するハブです。最新の投稿が上部に表示され、古い投稿は下に移動し、最終的には表示されなくなります。
ニュースチャンネル
ニュースチャンネルは、ニュースページ内のテーマ別スペースです。チャンネルごとに視聴者ごとの表示および編集権限を設定できます。
記事：ストーリーやレポートなどの詳細なコミュニケーション。見出し、テキスト、画像、動画、カスタマイズ可能なレイアウトなどの機能があります。
更新：お知らせやマイクロブログなど、短いテキストと画像や動画を組み合わせた、迅速でタイムリーな投稿。
メディア：画像や動画と短い説明文を組み合わせたビジュアルコンテンツ。
投稿を作成する
投稿は、チャンネル内で実際に作成されるコンテンツです。各投稿のコンテンツ形式は、チャンネルの種類によって異なります。
投稿は柔軟かつカスタマイズ可能で、編集者はコンテンツに対して創造的な自由を持つことができます。読者はコメントやリアクションを行い、さらにエンゲージメントを高めることができます。これらはチャンネルの設定や個別投稿の設定で定義できます。これにより、プラットフォーム内で包括的かつサポートし合う環境が生まれます。
追加のニュース機能
ニュースコンテンツの作成を簡単にし、より魅力的にコンテンツを表示するための追加機能を利用できます。例：
コンパニオン編集者
コンテンツレビュー
@メンション
ウィジェット
モバイルポッドキャストプレーヤー
メトリクス
ニュースのメトリクスを追跡し、コンテンツがどのように受け取られているかを把握できます：
投稿の統計情報：読者が投稿にどのようにアクセスしたか、コンテンツの全体的なリーチなどの情報。
アナリティクスダッシュボード：ニュースタブでは、エンゲージメントメトリクスのビジュアル表示など、より詳細なインサイトを提供します。
コメント： コメントは、従業員と作成したコンテンツをつなげ、トピックに対する直接的なフィードバックを提供するインタラクションの一種です。
