# PnP モダン検索ボックス Web パーツの構成

**URL**: https://support.staffbase.com/hc/ja/articles/11192156596370
**取得日**: 2026-05-02 18:49:27 JST

## 内容

アクティベーション
ページに専用の検索ボックスを追加するために、PnP モダン検索ボックス Web パーツをオプションで構成できます。たとえば、ランディングページに検索ボックスを追加する場合です。PnP モダン検索ボックス Web パーツを構成するには、PnP 検索結果 Web パーツと一緒に構成する必要があります。したがって、この構成には、PnP モダン検索ボックスと PnP 検索結果の両方の Web パーツが必要です。
PnP モダン検索ボックス Web パーツの追加
Staffbase News と一緒に PnP 検索結果 Web パーツを構成したページに移動します。
ページの右上にある 編集 をクリックします。
ページの上にマウスをホバーします。
新しい Web パーツを追加するための + アイコンが表示されます。
+ アイコンをクリックします。
利用可能な Web パーツのダイアログが開きます。
PnP 検索ボックスを検索して選択します。
追加した Web パーツに移動します。
鉛筆アイコンをクリックして Web パーツを編集します。
PnP - 検索ボックスのダイアログが開きます。
クリア時にクエリをリセットのトグルを オン にします。
保存 をクリックしてページを保存します。
ページを保存し、PnP モダン検索ボックス Web パーツを構成しました。次に、PnP モダン検索ボックスを PnP 検索結果 Web パーツと構成する必要があります。
PnP 検索結果 Web パーツとの PnP モダン検索ボックス Web パーツの構成
PnP モダン検索ボックスを接続のソースとして PnP 検索結果 Web パーツを構成する必要があります。
PnP 検索ボックス Web パーツを追加したページに移動します。
PnP 検索結果 Web パーツの Web パーツを編集 のための鉛筆アイコンをクリックします。
PnP - 検索結果のダイアログが開きます。
PnP - 検索結果 ダイアログのページ 3 に移動します。
利用可能な接続の下で、入力クエリテキストを使用のトグルを オン にします。
動的値のオプションを選択します。
接続先のドロップダウンメニューから PnP - 検索ボックス を選択します。
PnP - 検索ボックスのプロパティのドロップダウンメニューから 検索クエリ を選択します。
保存 をクリックしてページを保存します。
ページを保存し、Staffbase ニュースのための検索ボックスを構成しました。
ユーザーにとっての検索の見え方
ユーザーが検索ボックスにキーワードを入力すると、検索結果はキーワードに基づいて表示されます。
