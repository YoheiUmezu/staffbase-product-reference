# Formsプラグインを使用した自動化ワークフローの設定

**URL**: https://support.staffbase.com/hc/ja/articles/360017377559
**取得日**: 2026-05-02 18:52:11 JST

## 内容

アクティベーション
システム全体のエディター
IT環境において、フォームを自動化ワークフローに統合できます。ユーザーがStaffbaseプラットフォームでフォームを送信すると、例えば、HRシステム内の従業員の住所データの自動更新や、外部の保守会社への故障修理の自動通知をトリガーすることができます。
前提条件
IT部門から自動化フレームワークで作成されたワークフローのトリガーURLが提供されています（例：Power AutomateのHTTP Post URL）。
外部ワークフローをトリガーするフォームを設定しています。
スタジオで、フォームに移動します。
ワークフローを統合したいフォームをクリックします。
フォームページが開きます。
ワークフロータブのトリガーURLフィールドに、IT部門から受け取ったURLを入力します。
保存をクリックします。
フォームを使用して自動化ワークフローを設定しました。
Staffbaseは、より大きなグループに展開する前に、小さなユーザーグループでワークフローをテストすることを推奨します。
