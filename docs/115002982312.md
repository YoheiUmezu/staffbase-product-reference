# CSVインポートによるユーザー管理

**URL**: https://support.staffbase.com/hc/ja/articles/115002982312
**取得日**: 2026-05-02 18:48:07 JST

## 内容

CSVインポートで何ができますか？
StaffbaseプラットフォームはCSVファイルを処理できるため、CSVインポートはユーザー管理の簡単な方法です。CSVファイルに関する詳細情報が必要な場合は、例えばWikipedia - カンマ区切り値を参照してください。
CSVインポートはユーザーライフサイクル全体で使用できます。これには以下のユースケースが含まれます：
アプリの展開とユーザーデータの最初のインポート、詳細はCSVファイルの設定を参照してください。
ユーザーのインポートと新しいユーザーへの招待状の送信、詳細はCSVファイルのアップロードを参照してください。
ユーザーの無効化と削除
ユーザーデータの更新
必要に応じてCSVインポートを実行できます：月に1回、週に1回、または毎日でも可能です。
最新のユーザーデータをいつでもエクスポートできます。
従業員がリストから削除されると、そのアカウントは自動的に無効化されます。CSV同期には常にすべてのユーザーデータを含む完全なCSVファイルを使用してください。
システム内で手動で行った変更は、CSVファイルに含まれる情報によって上書きされます。
例：ユーザー「ジョン・ドゥ」が名前を「ドナルド・ダック」に変更した場合、CSVインポートに名前が含まれていると、名前は「ジョン・ドゥ」に戻されます。
どのようなCSVインポートの種類がありますか？
Staffbaseは手動および自動のCSVインポートの両方を提供しています。どちらも同じ機能を果たします。手動インポートはスタジオで実行でき、自動インポートはStaffbaseエンドポイントに接続してスケジュールに従って行われます。詳細についてはこちらを参照してください。
CSVインポートはどのように機能しますか？
CSVファイルをStaffbaseプラットフォームにアップロードすると、ファイルが分析されます。その後、どの列がアプリで提供されるプロファイルフィールドにマッピングされるかを選択します。以前にCSVインポートを使用してユーザーをアップロードしたことがある場合、インポートが行う変更のプレビュー（追加、削除、または更新されたユーザーの数など）が表示されます。
CSVファイルをアップロードして、ユーザーを同期させましょう！
