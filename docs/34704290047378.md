# Staffbaseメール内に「ブラウザで開く」リンクを追加する

**URL**: https://support.staffbase.com/hc/ja/articles/34704290047378
**取得日**: 2026-05-02 18:45:27 JST

## 内容

「ブラウザで開く」リンクを追加すると、受信者はメールをWebページとして表示できます。Webバージョンはパーソナライズや動的コンテンツターゲットをサポートしており、各受信者に合わせたコンテンツが表示されます。また、インライン動画の再生などの追加機能も利用できます。メールが複数の言語で作成されている場合、Webバージョンはユーザーのブラウザ設定に基づいて自動的に正しい言語を表示します。必要に応じて、受信者は他の言語バージョンにも切り替えることができます。
Studioで、メール > 「ブラウザで開く」リンクを追加したいメールに移動します。
Quick Blocksで、Web Versionをクリックします。
「ブラウザで開く」リンクがメールの末尾に追加されます。
必要に応じて、リンクテキストを編集またはさらに形式を整えます。
これで、メールに「ブラウザで開く」リンクが正常に追加されました。受信者は、Webバージョンのメッセージに簡単にアクセスし、印刷したり、ブラウザの印刷機能を使ってPDFとして保存したりできます。
