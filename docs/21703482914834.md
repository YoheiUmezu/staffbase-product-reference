# スクリーン用スライドショーの設定

**URL**: https://support.staffbase.com/hc/ja/articles/21703482914834
**取得日**: 2026-05-02 18:53:30 JST

## 内容

アクティベーション
管理エディター
StaffbaseのScreens機能を使用して、オフィスのラウンジ、カフェテリア、または廊下にニュースを表示します。詳細はデジタルサイネージ用のScreensの概要をご覧ください。
この記事では：
スライドショーの作成
デバイスをスライドショーに接続する
スライドショーのレイアウトを設定する
スライドショーにメディアを追加する
スライドショーを公開する
スライドショーの公開を解除する
スライドショーを削除する
スライドショーの作成
ニュースページまたはニュースチャンネルから作成するスライドショーを選択できます。
スタジオで、Screensに移動します。
スライドショーを作成をクリックします。
新しいスライドショーページが開きます。
スライドショータブで、スライドショーの名前を入力します。
スライドショーのニュースソースとしてチャンネルまたはニュースページを選択します。
オプションで、言語ドロップダウンから、スライドショーを表示する言語を選択します。
スライドショーに表示するニュース投稿の言語バージョンを指定できます。 言語ドロップダウンには、組織で利用可能なすべてのコンテンツ言語が含まれています。選択した言語がニュースコンテンツに利用できない場合、ニュースコンテンツはプラットフォームのデフォルトのコンテンツ言語で表示されます。
保存をクリックします。
スライドショーが作成されました。いつでもスライドショーを編集できます。デバイスをスライドショーに接続したり、レイアウトを設定したり、メディアを追加したり、直接スライドショーを公開してデバイスに表示させることができます。
デバイスをスライドショーに接続する
スライドショーを安全にリンクしたいデバイスを選択できます。また、誰でもURLを使用してスライドショーにアクセスし再生できるようにする公開URLを生成するオプションもあります。
前提条件
デバイス管理で、1つ以上のデバイスを安全にリンクしている必要があります。
スタジオで、Screensに移動します。
デバイスをリンクしたいスライドショーを開きます。
スライドショータブで、デバイスにリンクフィールドをクリックします。
スライドショーを表示したいデバイスを検索して選択します。
オプションで、公開URLを生成をクリックします。
表示デバイスのプレビューを見るために、表示形式の1つを選択します：
横向き
縦向き
保存をクリックします。
デバイスがスライドショーに接続されました。いつでもデバイスを変更できます。レイアウトを設定したり、メディアを追加したり、直接スライドショーを公開してデバイスに表示させることができます。
スライドショーのレイアウトを設定する
システム全体のエディター
レイアウトタブにあるさまざまなオプションを使用して、スライドショーの外観をカスタマイズできます。各スライドの表示時間からロゴの表示まで、要件に応じてスライドショーを調整できます。
タイトル
タイプ
説明
オプション/詳細情報
スライドの表示時間
ドロップダウン
スライドが表示される時間を選択します。
5、10、15、20、25、または30秒。
スライドショーあたりの投稿数
ドロップダウン
スライドショーに表示するニュース投稿の数を選択します。
1から10投稿。
ブランディングを適用
チェックボックス
追加のブランディング設定を開きます。
-
ブランディングを適用：色
セレクター
スライドショーに色の境界線を適用します。
ブランディング: スタジオのブランディング設定で設定されたカラーパレット。
カスタム: お好みのRGBを設定します。
ブランディングの適用: ロゴ
チェックボックス
スライドショーに従業員アプリのロゴを表示します。
これはスタジオのブランディング設定にアップロードされたロゴです。
ブランディングの適用: 日付と時刻
チェックボックス
スライドショーに日付と時刻を表示します。
これはスライドショーが開かれているブラウザの設定に基づいています。
投稿画像のスケーリング
セレクター
スライドショーの画面にティーザー画像をフィットさせるか、フィルします。
画像をフィット: デフォルトで、画像をフレームするぼやけた背景があります。
画像をフィル: ぼやけた境界がないズームインした画像です。境界の一部が切り取られることがあります。
QRコード
チェックボックス
スライドショーの各スライドにQRコードを提供します。
ユーザーはコードをスキャンして、プラットフォーム上で投稿の残りを読むことができます。
スタジオで、スクリーン > 設定したいスライドショー > レイアウトタブ。
スライドの表示時間のドロップダウンから、各スライドが表示される秒数を選択します。
スライドショーあたりの投稿数のドロップダウンから、スライドショーに表示される投稿数を選択します。
ブランディングの適用を選択して、さらなるカスタマイズオプションを開きます:
色の下で、ブランドカラーを確認するか、新しいものを選択します。
ロゴのチェックボックスを選択して、スライドにアプリのロゴを表示します。
日付と時刻のチェックボックスを選択して、スライドに日付と時刻を表示します。
投稿画像のスケーリングの下で、画像をフィットまたは画像をフィルを選択します。
QRコードのチェックボックスを選択して、スライドにコードを表示します。
保存をクリックします。
スライドショーのレイアウトを設定しました。カスタマイズはいつでも変更できます。メディアを追加するか、直接スライドショーを公開してデバイスに表示できます。
スライドショーにメディアを追加する
システム全体のエディター
画像や動画をスライドショーに追加して、組織の重要なメッセージを共有し、拡大します。My Filesから画像や動画を再利用して、スライドショーに追加し、繰り返し表示できます。この静的なカスタムコンテンツと動的なニュースコンテンツの組み合わせにより、メッセージが画面上で最適に表示されます。
メディアはスライドショーのニュース投稿の後に表示されます。画像はレイアウトタブで定義された時間表示され、動画は音声なしでその全長が再生されます。メディアタブでは、スライドショーに表示される順序でファイルを確認できます。
スタジオで、スクリーンに移動します。
メディアを追加したいスライドショーを開きます。
メディアタブで、メディアを追加ボタンをクリックします。
My Filesダイアログが開きます。
1つ以上のファイルを選択します。
オプションで、アップロードをクリックして新しいファイルを追加します。
ファイルを追加をクリックします。
保存をクリックします。
ニュース投稿の後に表示されるメディアを追加しました。スライドショーを公開してデバイスに表示できます。
スライドショーを公開する
システム全体のエディター
スライドショーを設定した後、デバイスで表示できるように公開する必要があります。
スタジオで、スクリーンに移動します。
公開したいスライドショーに移動します。
公開をクリックします。
スライドショーを公開しました。スライドショーはデバイスに表示されるまでに1分かかります。今、インターネットブラウザでスライドショーを再生するためのURLを開くことができます。
スライドショーを非公開にする
システム全体のエディター
特定の画面でスライドショーがもはや関連性がない場合、スライドショーを非公開にしてオフラインにすることができます。たとえば、目標達成時に表示される更新は、日付が過ぎた後はもはや関連性がありません。非公開にすると、スライドショーは表示されていた画面で利用できなくなります。
スタジオで、スクリーンに移動します。
非公開にしたいスライドショーに移動します。
スライドショーページが開きます。
非公開をクリックします。
スライドショーを非公開にするダイアログが開きます。
非公開をクリックしてアクションを確認します。
スライドショーは非公開になり、ユーザーには表示されなくなります。いつでもスライドショーを再公開できます。
スライドショーを削除する
スライドショーがもはや利用可能でない場合は、削除できます。以前に公開されたスライドショーの場合、URLが現在使用中であれば、スライドショーがもはや利用できないというメッセージが表示されます。
スタジオで、スクリーンに移動します。
削除したいスライドショーに移動します。
スライドショーページが開きます。
スライドショーを削除をクリックします。
スライドショーを削除ダイアログが開きます。
スライドショーを削除をクリックして、操作を確認します。
スライドショーは永久に削除されました。
