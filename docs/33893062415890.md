# ニュースチャンネルのソーシャルインタラクションと通知の設定

**URL**: https://support.staffbase.com/hc/ja/articles/33893062415890
**取得日**: 2026-05-02 19:01:17 JST

## 内容

編集責任者
ニュースは、スタッフベースアプリとイントラネットで視聴者ごとに動的なコンテンツを公開できます。これは、ニュースページ、チャンネル、投稿を作成するの3層構造で構成されています。
チャンネルの設定で、ソーシャルインタラクションと公開通知のデフォルトを定義できます。これにより、そのチャンネル内で作成されるすべての投稿を作成するの基準が設定されます。各設定の横には、許可されたとデフォルト設定の2つのオプションがあります。許可されたとして定義された設定は、投稿者以上のロールが個別のニュース投稿記事内で調整できます。
設定をデフォルトで有効にする
Studioでコンテンツ > ニュースに移動します。
デフォルトで有効にしたいオプションのチャンネルを開きます。
歯車アイコンをクリックして設定を開きます。
設定ページが開きます。
チャンネル機能の横で、定義したい設定に移動します。
許可されたのトグルを有効にします。
デフォルト設定のトグルを有効にします。
保存するをクリックします。
選択したオプションがすべてのニュース投稿記事でデフォルトで有効になりました。設定はいつでも調整できます。投稿者は必要に応じて個別のニュース投稿記事でこのオプションをオンにできます。
設定をデフォルトで無効にする
Studioでコンテンツ > ニュースに移動します。
デフォルトで無効にしたいオプションのチャンネルを開きます。
歯車アイコンをクリックして設定を開きます。
設定ページが開きます。
チャンネル機能の横で、定義したい設定に移動します。
許可されたのトグルを有効にします。
デフォルト設定のトグルを無効に調整します。
保存するをクリックします。
選択したオプションがすべてのニュース投稿記事でデフォルトで無効になりました。設定はいつでも調整できます。投稿者は必要に応じて個別のニュース投稿記事でこのオプションをオンにできます。
