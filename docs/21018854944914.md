# ニュースチャンネルタイプの概要

**URL**: https://support.staffbase.com/hc/ja/articles/21018854944914
**取得日**: 2026-05-02 18:51:22 JST

## 内容

Staffbaseプラットフォームは、オーディエンス特有の動的コンテンツの公開を可能にします。ニュースチャンネルは、新聞のニュースセグメントに似ています。たとえば、スポーツセグメントはスポーツ関連のニュースのみを特集しています。同様に、ニュースチャンネルは同じトピックやテーマに関するニュース投稿を持っています。
あなたのプラットフォームは無制限の数のニュースチャンネルをサポートしており、各ニュースチャンネルはニュースページに割り当てられています。 詳細を学ぶ。 チャンネル内では、ニュース投稿は動的で、プラットフォーム内で短期間から中期間の情報を含んでいます。
ニュースチャンネルのタイプは、投稿が表示される形式を定義します。異なるユースケースをサポートするために、3つの異なるチャンネルタイプがあります。以下の記事で各チャンネルタイプについて詳しく学んでください：
記事チャンネル：タイトル、ティーザーテキスト、メディアを含む完全な記事。
更新チャンネル：メディアを含む限られた記事。
メディアチャンネル：キャプション付きの画像または動画。
追加の役立つ情報
ニュースページ、チャンネル、投稿の関係
プラットフォーム内のコンテンツの寿命
