# 更新する投稿を作成する

**URL**: https://support.staffbase.com/hc/ja/articles/35190495229330
**取得日**: 2026-05-02 19:02:25 JST

## 内容

ニュースは、Staffbaseアプリとイントラネットで視聴者ごとに動的なコンテンツを公開することができます。コンテンツを効率的にまとめ、ユーザーが簡単に利用できる形式で配信します。チャンネル内でニュース投稿記事を作成し、公開することができます。コンテンツ形式の外観は、チャンネルタイプによって異なります。この記事では、アップデートチャンネルタイプ内で投稿を作成します。
Studioで、コンテンツ > ニュース > 投稿を追加したいアップデートチャンネルに移動します。
投稿を作成するをクリックします。
アップデートを作成するページが開きます。
テキストの説明を入力します。
必要に応じて、メディアや添付を追加します。
下書きを保存するをクリックします。
必要に応じて、プレビューをクリックして、投稿がユーザーにどのように表示されるかを確認します。
必要に応じて、別の言語でコンテンツを追加することもできます。
下書きの投稿を作成しました。ユーザーに表示するには、まず公開する必要があります。
