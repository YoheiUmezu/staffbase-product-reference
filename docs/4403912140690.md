# Microsoft 365 ドキュメントライブラリウィジェットの設定

**URL**: https://support.staffbase.com/hc/ja/articles/4403912140690
**取得日**: 2026-05-02 18:59:48 JST

## 内容

アクティベーション
システム全体のエディター
従業員アプリやイントラネットで SharePoint ドキュメントライブラリからファイルを見つける手助けをするために、ドキュメントライブラリまたは特定のフォルダーを追加します。Microsoft 365 ドキュメントライブラリウィジェットを使用して開始します。ソート機能により、ユーザーはコンテンツをアルファベット順または最終更新日でソートできます。強力な検索機能により、ユーザーは探しているコンテンツを見つけることができます。
SharePoint では、コンテンツ承認設定により、承認されたコンテンツのみがユーザーに表示されます。表示したいファイルのライブラリに対してコンテンツ承認設定が有効になっている場合、ウィジェットには承認されたファイルのみが含まれます。
前提条件
あなたは Microsoft 365 接続を Staffbase に追加しました。
あなたは アプリまたはイントラネットから Microsoft 365 にサインインしました。
あなたは Microsoft 365 ドキュメントライブラリウィジェットをアクティブにしました。
Microsoft 365 ライブラリウィジェットを追加します。
ウィジェットを設定するためのダイアログが開きます。
オプションで、タイトルを提供します。
タイトルはウィジェットの上に表示され、ユーザーに見えるようになります。
ウィジェットが表示されるデバイスタイプを、対応するアイコンをクリックして選択します。
ウィジェットを表示するユーザーをアプリ内の表示で選択します。
サイト名のドロップダウンから、ライブラリを表示したいサイトを選択します。
ライブラリ名のドロップダウンから、表示したいドキュメントライブラリを選択します。
オプションで、SharePoint ドキュメントライブラリから特定のフォルダーを表示するには、そのフォルダーのパスを入力します。
オプションで、ファイルがウィジェット内でソートされて表示される順序を選択します。
OKをクリックします。
ユーザーがドキュメントライブラリウィジェットを表示する方法
ユーザーがウィジェットと対話すると、まだサインインしていない場合は、従業員アプリまたはイントラネットからMicrosoft 365 アカウントにサインインするように促されます。
