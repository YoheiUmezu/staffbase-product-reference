# アイディエーションの概要

**URL**: https://support.staffbase.com/hc/ja/articles/26065951814546
**取得日**: 2026-05-02 19:01:20 JST

## 内容

システム全体の編集者
この機能はアドオンです。詳細はStaffbase 価格をご参照ください。有効化についてはStaffbaseサポートまたはカスタマーサクセスマネージャーにお問い合わせください。
アイデア創出プラグインは、スタッフベース従業員エクスペリエンス（EX）ブースタープラグインです。
従業員がアイデアを共有し、コメントし、重要なことに投票できるダイナミックなツールです。イノベーションの促進や社内フィードバックの効率化など、組織に構造と柔軟性をもたらします。
カスタマイズ可能なテンプレートで簡単にアイデアを投稿でき、コメント、共有、投票などのコラボレーションツールがチームワークを促進します。投稿は匿名でも可能です。設定可能なステータスシステムにより、各アイデアの段階に応じて進捗を追跡し、機能を制御できます。自動通知で新しいアイデアや更新情報を全員に知らせ、詳細なCSVエクスポートでより深い分析と継続的な改善をサポートします。
