# Staffbaseプラットフォームへのカスタムウィジェットのインストール

**URL**: https://support.staffbase.com/hc/ja/articles/360021196079
**取得日**: 2026-05-02 18:44:33 JST

## 内容

アクティベーション
管理者
カスタムウィジェットは、Staffbaseが提供する標準ウィジェットと非常に似ています。カスタムウィジェットが開発されたら、そのURLをStudioに提供することで、プラットフォームに簡単にインストールできます。カスタムウィジェットの詳細については、こちらをご覧ください。
カスタムウィジェットは、あなたの組織によって開発および所有されています。Staffbaseはプラットフォームとして、カスタムウィジェットに使用されるコンテンツやサービスに対して責任を負いませんが、カスタムウィジェットがStaffbaseの利用規約およびプラットフォームによって定義された適正使用ポリシーに準拠することをお勧めします。
前提条件:
追加したいカスタムウィジェットが開発済みである。
開発チームからカスタムウィジェットのURLを受け取っている。
Studioで、設定 > カスタム ウィジェットに移動します。
カスタムウィジェットページが開きます。
ウィジェットバンドルの下に、カスタムウィジェットのURLを提供します。
インストールをクリックします。
カスタムウィジェットがStaffbaseプラットフォームにインストールされました。
これで、あなたの編集者はウィジェットをページやニュースに追加できます。
