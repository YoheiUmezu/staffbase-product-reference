# プラットフォームでのライブストリーミング

**URL**: https://support.staffbase.com/hc/ja/articles/4409045244946
**取得日**: 2026-05-02 19:00:25 JST

## 内容

タウンホール、記念日、重要な会議をライブでストリーミングすることで、重要なマイルストーンやイベントを皆で祝おう。あなたのアプリやイントラネットでライブストリーミングを行うことで、オンデマンドストリーミングとは異なり、リアルタイムで動画を放送します。これにより、従業員と重要な情報を共有し、フィードバックや質問に直接かつインタラクティブに応答することができます。
定期的なライブストリーミングは、企業文化を強化し、従業員が組織にアイデンティティを持つのに役立ちます - それがオフィスにいる場合でも、リモートで働いている場合でも、移動中であっても。
さらに、定期的なライブストリーミングは:
経営陣と従業員の間のオープンな交流とコミュニケーションを促進します
対面でのインタラクションにより、従業員のエンゲージメントを高めます
従業員が行き来する必要のあるツールやプラットフォームの数を減らします
無制限の参加者数と、対面イベントよりも低コストです
動画ストリームを直接アプリやイントラネットに埋め込むことで、重要な情報が従業員にのみ向けられることを保証します。Staffbaseは、以下のような人気のプラットフォームからの動画やライブストリームの埋め込みをサポートしています:
YouTube
Vimeo
DaCast
特定のユースケースに応じて、プラットフォームにライブストリームや動画を統合する方法は異なります:
埋め込みページ用プラグイン
埋め込みページ用ウィジェット
個別の埋め込みオプションについては、Staffbaseサポートまたはカスタマーサクセスマネージャーにお問い合わせください。
埋め込みページプラグインを使用したライブストリーミング
埋め込みページプラグインは、希望する外部ウェブサイトをプラットフォーム内のページとして埋め込みます。これは、プラットフォームにシームレスに埋め込みたい定期的にスケジュールされたライブストリーム形式に特に便利です。
使用するプラットフォームによっては、動画やライブストリームを正常に埋め込むために特別な埋め込みコードが必要になる場合があります。各プロバイダーのヘルプセンターで、セットアップや埋め込みコードに関する詳細な手順を見つけることができます:
YouTubeヘルプ
Vimeoヘルプセンター
DaCastチームブログ
前提条件:
ライブストリームプラットフォームを選択し、Staffbaseプラットフォームでライブストリーム用のチャンネルを作成しました。
埋め込みコードを生成しました。
埋め込みページプラグインをインストールしました。
スタジオで、埋め込みページプラグインに移動し、ページを追加をクリックします。 埋め込みページを追加するためのダイアログが開きます。
埋め込みページのタイトルを提供します。
ターゲットURLの下に、ストリーミングプラットフォームで生成した埋め込みコードを提供します。
ページを追加をクリックします。
埋め込みページを公開をクリックします。
埋め込みページウィジェットを使用したライブストリーミング
埋め込みページウィジェットを使用すると、既存のページにライブストリームを追加できます。
使用するプラットフォームによっては、動画やライブストリームを正常に埋め込むために特別な埋め込みコードが必要になる場合があります。各プロバイダーのヘルプセンターで、セットアップや埋め込みコードに関する詳細な手順を見つけることができます:
YouTubeヘルプ
Vimeoヘルプセンター
DaCastチームブログ
前提条件:
ライブストリームプラットフォームを選択し、Staffbaseプラットフォームでライブストリーム用のチャンネルを作成しました。
埋め込みコードを生成しました。
スタジオで、埋め込みページウィジェットを追加したいページに移動し、エディタ内で配置したい場所にマウスを置きます。
埋め込みページウィジェットを追加します。
埋め込みページウィジェットを構成するためのダイアログが開きます。
オプションで、タイトルを提供します
タイトルはウィジェットの上に表示され、ユーザーに見えるようになります。
ウィジェットが表示されるデバイスタイプを、対応するアイコンをクリックして選択します。
アプリ内の可視性の下で、ウィジェットが表示されるユーザーを選択します。
ターゲットURLの下に、ストリーミングプラットフォームで生成した埋め込みコードを提供します。
オプションで、デスクトップおよびモバイルデバイス用のウィジェットの高さをカスタマイズします。
ズームやクリックなど、ウィジェットとのユーザーインタラクションを有効にするかどうかを、トグルインタラクションを有効にするを調整して定義します。
ウィジェットのスクロールを有効にするかどうかを、トグルスクロールを有効にするを調整して定義します。
クリックしてOK。
ウィジェットがページに追加されます。
変更を保存します。
