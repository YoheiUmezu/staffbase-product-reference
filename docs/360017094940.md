# 編集者のためのリンクオプション

**URL**: https://support.staffbase.com/hc/ja/articles/360017094940
**取得日**: 2026-05-02 18:44:43 JST

## 内容

リンクを使用することで、コンテンツクリエイターはユーザーの体験を豊かにし、関連する追加コンテンツをワンクリックで提供できます。コンテンツにリンクを配置することで、ユーザーのために文脈に応じたショートカットを作成しています。
編集時には、さまざまなリンクオプションが利用可能です。以下は、エディターを使用してリンクできる一般的なコンテンツタイプのいくつかです：
外部ウェブサイト
プラットフォーム内のコンテンツ（プラグインやページなど）
ページ内の見出しや特定のテキスト
メールアドレス
電話番号
ユーザープロフィール
これらはすべて、インテキストリンクまたはウィジェットを使用して実現できます。
インテキストリンク
インテキストリンクとは、コンテンツの本文に直接表示されるリンクのことで、テキストに表示されるURLまたはハイパーリンクのいずれかです。インテキストリンクは多用途で、さまざまな形式を取ることができます。リンクしたいコンテンツのタイプに応じて、最適なインテキストリンクを選択してください：
テキストに表示されるURL：
コンテンツにリンクする最も簡単な方法は、目的のコンテンツのURLをテキストの本文に直接貼り付けることです。追加の設定は必要ありません。
リンクに使用：
内部コンテンツ
外部ウェブサイト
ハイパーリンク：
ハイパーリンクを使用すると、テキスト、画像、または他のメディアの一部を設定して、クリック時に指定されたURLを開くことができます。ハイパーリンクは、URLが表示されず、テキスト、画像、または他の要素に埋め込まれているため、視覚的にすっきりとした体験を提供します。
リンクに使用：
内部コンテンツ
外部ウェブサイト
メールアドレス
電話番号
アンカーリンク
アンカーリンクは、ページ内の別のポイント（たとえば、見出しや特定の段落）にリンクするために設計されたハイパーリンクの一種です。これは特に長いページで便利で、ユーザーが長い検索やスクロールをせずに、他の関連コンテンツに直接ジャンプできるようにします。
リンクに使用：
ページ内の見出し
ページ内のテキスト
目を引くデザインでのリンク
リンクのグループ化
ウィジェットはリンク機能以上の広範な機能を持つことができますが、コンテンツを整然とした視覚的に魅力的な方法でリンクするために特に設計されたウィジェットがいくつかあります。
どのウィジェットを使用してリンクを作成しても、設定は迅速かつ直感的で、高品質なデザインを提供し、コンテンツにプロフェッショナルな外観をもたらします。自己サービスウィジェットの全範囲は動的ページで利用可能ですが、多くはニュースやジャーニーでも利用可能です。
関連コンテンツへのリンクのグループを表示したい場合（たとえば、オフィスの場所や部門のリソースなど）、さまざまな視覚スタイルでニーズを満たすウィジェットがいくつかあります：
リンクリストウィジェット
リンクタイルウィジェット
ポートフォリオウィジェット
これらのウィジェットを使用して、外部ウェブサイト、プラットフォーム内のコンテンツ、またはその両方をリンクできます。これにより、特定のトピックに関する分散したコンテンツを1か所に簡単に集約できます。
ユーザープロフィールのリンク
さらに、ユーザープロフィールウィジェットを使用すると、コンテンツ内にユーザープロフィール全体（プロフィール写真、職位、連絡先情報など）をリンクできます。電話番号やメールアドレスをハイパーリンクでリンクすることもできますが、ユーザープロフィールウィジェットは名前に親しみやすい顔を与え、ユーザーが同僚とつながるのを容易にします。
