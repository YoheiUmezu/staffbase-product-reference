# Staffbaseメールの添付ファイルの概要

**URL**: https://support.staffbase.com/hc/ja/articles/25485362862610
**取得日**: 2026-05-02 18:42:25 JST

## 内容

エディター
添付ファイルのサイズ
Staffbaseメールの概要ページのコンテンツの下に、最大7.5 MBのファイルを添付できます。
複数の添付ファイルを追加すると、それらの合計サイズが表示されます。
添付ファイルのサイズ制限を超えると、メッセージが表示され、メールを送信できなくなります。
添付ファイルのプレビューまたは削除
詳細オプションメニューを通じて、添付ファイルを削除したり、画像、文書、動画をプレビューしたりできます。音声ファイルはプレビューできません。
