# Microsoft 365 コネクションの概要

**URL**: https://support.staffbase.com/hc/ja/articles/4403677505938
**取得日**: 2026-05-02 18:44:29 JST

## 内容

アクティベーション
管理者
Microsoft 365 コネクションを使用すると、従業員アプリやイントラネット用に Microsoft 365 検索とウィジェットをアクティブ化できます。ユーザーが従業員アプリやイントラネットから Microsoft 365 アカウントにサインインすると、エディターが追加した Microsoft コンテンツにアクセスし、検索機能を使用して Microsoft コンテンツを検索できます。
Microsoft コネクション ワークフロー
Staffbase プラットフォームから Microsoft Azure AD アプリを介して Microsoft 365 ユーザーアカウントに安全に接続し、Microsoft 365 用の検索とウィジェットを使用できます。Microsoft コネクションを設定するには、次の手順に従ってください：
Azure アプリを使用して Microsoft コネクションを追加します。
次のアプリのいずれかを使用して接続を追加できます：
デフォルト Azure アプリ: Staffbase からの既製の Azure AD エンタープライズアプリケーションのオプションを選択します。
Staffbase は、迅速なセットアップのためにデフォルト Azure アプリの使用を推奨します。
カスタム Azure アプリ: カスタムアプリのオプションを選択する場合：
権限リクエストダイアログにカスタムアプリ名、ロゴ、および説明を表示します。
ファーストパーティの Azure エンタープライズアプリケーションを使用するか、同じ権限スコープを持つ独自の Azure アプリがすでに存在し、エンタープライズアプリケーションの数を制限したい場合。
カスタム Azure アプリの接続を追加するには、Azure アプリを作成し、Azure クライアント ID とクライアント シークレットを提供する必要があります。さらに、リダイレクト URI を作成する必要があります。詳細を学ぶ。
Microsoft 365 検索 と ウィジェットを Staffbase プラットフォーム用にアクティブ化します。
従業員アプリまたはイントラネットから Microsoft 365 にサインインし、Staffbase プラットフォームに接続するために Azure アプリに権限を付与します。いつでも権限を確認できます。
