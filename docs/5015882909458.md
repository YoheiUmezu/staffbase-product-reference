# コンテンツレビューを使用したニュース投稿のレビュー

**URL**: https://support.staffbase.com/hc/ja/articles/5015882909458
**取得日**: 2026-05-02 19:03:19 JST

## 内容

アクティベーション
エディター
従業員を引き込む魅力的なニュースを作成するには、時間、労力、チームワークが必要です。すべてのニュースチャンネルタイプで利用可能な新しいコンテンツレビュー機能を使用して、投稿の詳細を簡単に確認したり、意見を求めたりできます。
投稿作成者は、投稿を共有することでコンテンツのレビューをリクエストできます。招待されたコラボレーターは、メールを通じてアクセスを受け取り、その後、ニュース投稿の隣にある新しいレビュータブでコメントを残すことができます。
コンテンツレビューを使用すると、次のことができます：
メールを通じてレビューをリクエストするために投稿を共有する
コメントを残し、他の人のコメントを見る
コメントを通じてフィードバック、レビュー、または承認を共有する
新しいコメントや@メンションに関するメール通知を受け取る
