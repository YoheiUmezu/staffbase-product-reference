# SharePointからStaffbaseニュース投稿を作成する

**URL**: https://support.staffbase.com/hc/ja/articles/11203978226322
**取得日**: 2026-05-02 18:58:07 JST

## 内容

アクティベーション
エディター
SharePoint用のニュースセントラルを使用すると、マウスのクリックでSharePointからニュース記事を作成できます。
Staffbase 365ニュースセントラルを追加したSharePointサイトに移動します。
ニュースメニューを開きます。
+アイコンをクリックします。
スタジオが新しいブラウザタブで開き、ニュース投稿を作成するオプションが表示されます。
ニュース投稿をクリックします。
ニュース投稿が作成されました。これを公開してSharePointでも利用できるようにします。
