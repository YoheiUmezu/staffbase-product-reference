# 従業員メール画像ライブラリの使用

**URL**: https://support.staffbase.com/hc/ja/articles/4412584512914
**取得日**: 2026-05-02 18:49:51 JST

## 内容

個々のメールデザイナードラフトで画像/GIFをアップロードするデフォルトオプションを使用するのとは異なり、画像ライブラリを使用すると、アップロードした画像を複数のメールで再利用できます。
また、組織内のすべてのユーザーが共有画像にアクセスし、メールで使用することができます。
画像ライブラリに画像を追加、整理、削除できるのは管理者および親管理者ユーザーのみですが、組織内のすべての従業員メールユーザーは画像ライブラリの画像を使用できます。
従業員メールウェブアプリを使用して画像ライブラリを管理する必要がありますが、そこに保存された画像は、任意のメールアドインや拡張機能でも使用できます。
画像ライブラリにアップロードできる画像の数に制限はなく、追加できるフォルダーの数にも制限はありません。
画像ライブラリに画像を追加する
管理者
画像ライブラリに画像を追加できるのは管理者および親管理者ユーザーのみですが、組織内のすべての従業員メールユーザーは画像ライブラリの画像を使用できます。
メールドラフトまたはテンプレートを編集している間に、画像要素を選択し、共有ライブラリを開くをクリックします。
新しい画像を追加したいフォルダーを選択または作成します。
フォルダーにアップロードをクリックし、画像を選択します。
複数の画像を一度に選択できます。
Macの場合: Commandキーを押しながら選択します。
Windowsの場合: Ctrlキーを押しながら選択します。
オプションで、画像要素を選択し、編集をクリックして画像を編集します。
現在のメールデザインで行った画像の編集は、画像ライブラリの画像には適用されません。
画像ライブラリから画像を追加する
メールドラフトまたはテンプレートを編集している間に、画像要素を選択し、共有ライブラリを開くをクリックします。
画像ファイルを選択し、選択をクリックします。
オプションで、画像要素を選択し、編集をクリックして画像を編集します。
現在のメールデザインで行った画像の編集は、画像ライブラリの画像には適用されません。
