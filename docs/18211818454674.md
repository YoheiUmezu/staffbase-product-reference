# スマートインサイトダッシュボードの概要

**URL**: https://support.staffbase.com/hc/ja/articles/18211818454674
**取得日**: 2026-05-02 18:49:12 JST

## 内容

アクティベーション
システム全体のエディター
スマートインサイトダッシュボードは、各キャンペーンの詳細な統計を表示します。ダッシュボード内では、キャンペーンに割り当てられたコンテンツの可視性、エンゲージメント、感情に関する詳細なメトリクスを確認できます。ダッシュボードには、分析ダッシュボードのキャンペーンタブからアクセスできます。
スマートインサイトダッシュボードは、スマートインパクト機能の一部です。詳細を学ぶ。 さらなる情報については、カスタマーサクセスマネージャーまたはStaffbaseサポートにお問い合わせください。
トップレベルメトリクス
ダッシュボードの上部には、迅速な概要のための4つのメトリクスがあります。
可視性
エンゲージメント
ネット感情スコア
アライメント
このメトリクスは、キャンペーンに割り当てられたコンテンツを少なくとも1つ表示したユーザーの割合を示します。
このメトリクスは、反応、コメント、共有、またはアライメント調査に回答することによってキャンペーンコンテンツと相互作用したユーザーの割合を示します。ニュース投稿とニュース投稿のコメントの両方に対する相互作用が計算されます。
このメトリクスは、キャンペーンに割り当てられたすべてのニュース投稿に対する全体的なユーザー感情を、コメントに基づいて示します。
視覚的な指標は、現在のスコアが-100から+100の範囲内のどこにあるかを示します。
このメトリクスは、キャンペーンに割り当てられたコンテンツに含まれるすべての調査に対する全体的なユーザーアライメントを示します。
視覚的な指標は、現在のスコアが1から5の範囲内のどこにあるかを示します。
メトリクスの下にあるラベルは、過去7日間のトレンドを示します。たとえば、先週の可視性は50％で、今週は55％です。これは10％のポジティブな増加を示します。
ラベルにカーソルを合わせると、過去7日間に対するメトリクスの追加情報が表示されます。
メトリクスの下にあるラベルは、過去7日間のトレンドを示します。たとえば、先週のエンゲージメントは50％で、今週は55％です。これは10％のポジティブな増加を示します。
ラベルにカーソルを合わせると、過去7日間に対するメトリクスの追加情報が表示されます。
スコアは、ポジティブなコメントの割合からネガティブなコメントの割合を引いたものに基づいて計算されます。
ツールチップ は、スコアの計算方法を詳細に説明します。
スコアは、ユニークなユーザーの応答からの全体的なアライメントスコアに基づいて計算されます。ユーザーからの最新の応答がカウントされます。
投稿の調査には、含まれるために少なくとも5つの応答が必要です。ツールチップ は、スコアの計算方法を詳細に説明します。

トップレベルメトリクスに加えて、トップレベルメトリクスに関するより詳細な統計を提供するセクションがあります。詳細メトリクスセクションについては、各記事で詳しく学んでください：
