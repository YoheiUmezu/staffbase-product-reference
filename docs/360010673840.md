# CSVを使用してオープングループに必須メンバーを追加する

**URL**: https://support.staffbase.com/hc/ja/articles/360010673840
**取得日**: 2026-05-02 18:47:49 JST

## 内容

グループ識別子は、CSVインポートを介してのみユーザープロファイルに追加できます。グループ識別子がユーザープロファイルに追加されると、そのユーザーは自動的にグループに追加され、退会することはできません。
前提条件
オープングループを作成していること。
ユーザーを追加したいオープングループに識別子を追加していること。
CSVファイルに識別子を追加する
オープングループに識別子を追加したら、この識別子を使用してユーザーをグループの必須メンバーとして指定できます。ユーザーCSVにオープングループタグ用の指定列を作成し、各ユーザーに適切な識別子を追加します。
複数の必須グループに同時にユーザーを追加することもできます。その場合、次の形式でセルに複数のグループ識別子を入力してください：
Group_id,Group_id
ユーザーデータを含むCSVファイルを開きます。
必須グループ識別子用の列を1つ作成します。
作成した列に各ユーザーの適切な識別子を追加します。
オープングループ設定に表示される識別子名を使用してください。複数のグループに追加したいユーザーには、識別子をカンマで区切って複数の識別子を追加することが可能です。
CSVファイルへの変更を保存します。
CSVインポートを通じてユーザーをオープングループの必須メンバーとして割り当てる
スタジオで、ユーザー > ユーザーインポートと同期に移動します。
新しいインポートを開始をクリックします。
編集したCSVファイルをアップロードし、続行をクリックします。
ドロップダウンメニューを使用して、各列の値に対応するフィールドを選択します。
グループ識別子用に指定された列には、必須グループ（オープングループのみ）を選択します。。
続行をクリックします。
変更の概要を確認し、インポートを実行をクリックします。
インポートを確認するためのダイアログが表示されます。
インポートを続行するには、インポートを実行をクリックします。
成功したインポート後に変更の概要が表示されるダイアログが開きます。
グループ識別子が対応するユーザープロファイルに追加され、ユーザーがオープングループの必須メンバーとして同期されました。
