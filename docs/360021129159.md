# ページの更新リマインダーの設定

**URL**: https://support.staffbase.com/hc/ja/articles/360021129159
**取得日**: 2026-05-02 19:02:21 JST

## 内容

アクティベーション
エディター
異なるページは、コンテンツに応じて更新が必要な頻度が異なる場合があります。ページの更新リマインダーは、デフォルトでページが6か月間更新されていない場合にエディターに通知するように設定されています。各ページの更新リマインダーの頻度は調整可能で、その特定のページのコンテンツに合ったリズムでリマインダーを受け取ることができます。
ページの更新リマインダーの設定
スタジオで、更新リマインダーを設定したいページに移動します。
歯車アイコンをクリックして設定を開きます。
設定ページが開きます。
更新リマインダーの下に
ページの更新リマインダーをオンにするためにチェックボックスを選択します。
ページの更新リマインダーをオフにするためにチェックボックスの選択を解除します。
保存をクリックします。
ページの更新リマインダーが設定されました。更新リマインダーは、コンテンツが6か月間更新されていない場合にエディターに通知するように設定されています。
ページの更新リマインダーの頻度を変更する
スタジオで、更新リマインダーを調整したいページに移動します。
歯車アイコンをクリックして設定を開きます。
設定ページが開きます。
更新リマインダーのチェックボックスが選択されていることを確認します。
ページの更新リマインダーを受け取る頻度を入力します。
保存をクリックします。
更新リマインダーの頻度が保存されました。
