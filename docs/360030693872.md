# iOSデバイスのリンクコンテンツ用ナビゲーションバーの設定

**URL**: https://support.staffbase.com/hc/ja/articles/360030693872
**取得日**: 2026-05-02 18:59:11 JST

## 内容

iOSデバイスでは、リンクコンテンツに対してネイティブiOSナビゲーションバーを有効または無効にするかを選択できます。この設定は、アプリ内の各リンクに対して指定できます。ナビゲーションバーは、Linksプラグインでデフォルトで有効になっています。
アイデンティティプロバイダーを介してユーザーログインを必要とするウェブサイトに対してiOSナビゲーションバーを有効にすると、iOSデバイスで予期しない動作が発生する可能性があります。そのような場合、Staffbaseはナビゲーションバーを無効にすることを推奨します。アイデンティティプロバイダーを介してログインを必要とするウェブページでナビゲーションバーが必要な場合は、Staffbaseサポートまたはカスタマーサクセスマネージャーにお問い合わせください。
アプリ内リンクのためのiOSナビゲーションバーの設定
iOSナビゲーションバーが有効または無効にされても、Androidまたはデスクトップユーザーには影響ありません。
スタジオで、Linksプラグインに移動します。

プラグイン設定: リンクに移動します。
