# Staffbaseメール内の配信停止オプションの管理

**URL**: https://support.staffbase.com/hc/ja/articles/34411858028562
**取得日**: 2026-05-02 18:48:25 JST

## 内容

フォルダ管理者
メールの配信停止オプションを管理することは、責任あるコミュニケーションの重要な一部です。これにより、メッセージの明確なカテゴリーを定義し、受信者がどの種類のメールを配信停止できるかを決めることができます。一方で、重要なコミュニケーションは影響を受けません。これにより、受信者が受け取る内容をコントロールできる一方で、クリティカルなアップデートを届ける能力を失わずにバランスを取ることができます。また、メール規制への対応、不要な受信トレイの混雑の削減、ユーザーの好みを尊重することで信頼構築にもつながります。
フォルダ単位で配信停止カテゴリーを作成する
デフォルトでは、フォルダごとに配信停止設定は無効化されています。フォルダ管理者はこれを有効にし、フォルダごとに最大10個の配信停止カテゴリーを作成し、各カテゴリーのメールに配信停止オプションを含めるかどうかを指定できます。フォルダ管理者が配信停止カテゴリーを作成した場合、フォルダ編集者はこのフォルダからメールを送信する際にカテゴリーのいずれかを選択する必要があります。たとえば、ニュースレターでは通常配信停止が可能ですが、重要な業務連絡では配信停止ができない場合があります。
Studioでメール > フォルダに移動します。
編集したいフォルダを開きます。
その他のオプションアイコンをクリックします。ドロップダウンメニューが開きます。
設定を選択します。フォルダの設定ページが開きます。
配信停止管理の下で、トグルを有効にしてフォルダの配信停止オプションを許可します。
ダイアログが展開され、カテゴリー作成用のテーブルが表示されます。
カテゴリーを作成をクリックします。カテゴリーを作成ダイアログが開きます。
カテゴリータイトルと説明を追加します。
タイトルと説明は、受信者が配信停止オプションを利用する際に表示されます。
このカテゴリーで配信停止を許可しない場合は、配信停止を許可トグルを無効にします。
保存するをクリックします。
これで、フォルダの配信停止オプションが有効化され、配信停止の許可有無に応じたカテゴリーが作成されました。フォルダの編集者は、このフォルダからメールを送信する際に適切なカテゴリーを選択できるようになります。
配信停止カテゴリーを編集する
カテゴリーの配信停止オプションが無効にされている場合、そのカテゴリーに対する以前の配信停止設定は保持されなくなります。これにより、連絡先が以前に配信停止を選択していたメールを受信する可能性があります。
Studioでメール > フォルダに移動します。
編集したいフォルダを開きます。
その他のオプションアイコンをクリックします。ドロップダウンメニューが開きます。
設定を選択します。フォルダの設定ページが開きます。
配信停止管理の下で、編集したいカテゴリーを選択し、その他のオプションメニューをクリックします。
編集するをクリックします。カテゴリーを編集ダイアログが開きます。
カテゴリーを更新します。
保存するをクリックします。
カテゴリーの編集が完了しました。
配信停止カテゴリーを削除する
サブスクリプションカテゴリーが削除された場合、そのカテゴリーに対する以前の配信停止設定は保持できなくなります。その結果、連絡先が以前に配信停止を選択したメールを受信する可能性があります。
Studioでメール > フォルダに移動します。
編集したいフォルダを開きます。
その他のオプションアイコンをクリックします。ドロップダウンメニューが開きます。
設定を選択します。フォルダの設定ページが開きます。
配信停止管理の下で、削除したいカテゴリーを選択し、その他のオプションメニューをクリックします。
削除するをクリックします。カテゴリーを削除ダイアログが開きます。
削除するをクリックします。
カテゴリーの削除が完了しました。送信済みおよびスケージュールされたメールもこのカテゴリーに表示されます。受信者は引き続きこのカテゴリーの購読を管理できます。
メールの配信停止カテゴリーを選択する
フォルダ編集者
フォルダ管理者がこのフォルダのメールに対して配信停止を許可している場合、メール送信前に配信停止カテゴリーを選択する必要があります。
Studioでメール > 送信またはスケジュールしたいメールの下書きに移動します。
メールを開き、次をクリックします。まとめページが開きます。
配信停止カテゴリーの下で、メールのカテゴリーを選択します。
プレビューをクリックして、受信者に配信停止オプションがどのように表示されるか確認します。
まとめページの残りの項目をすべて入力し、送信するまたはスケジュールをクリックします。
メールに配信停止オプションが追加されました。このカテゴリーで配信停止が許可されている場合、受信者はこのカテゴリーのメールの配信停止ができます。
メールアナリティクスでの配信停止の追跡
単一メールのアナリティクスにあるEmail Funnel Graphでは、送信済みバーにカーソルを合わせると、このメールカテゴリーから配信停止した受信者数が表示されます。Staffbaseメールアナリティクスについてさらに見る。
ユーザー向けの配信停止管理の表示
メールに配信停止オプションが含まれている場合、フッターに購読設定を管理するリンクが表示されます。
受信者がこのリンクをクリックすると、配信停止が許可されているフォルダのすべての配信停止カテゴリーが表示されるページにリダイレクトされます。受信者は一度に1つ以上のカテゴリーの配信停止が可能です。
受信者がカテゴリーの選択を外し、設定を更新をクリックすると、そのカテゴリーからのメールは今後受信しなくなります。
