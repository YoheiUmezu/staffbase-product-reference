# 言語用語集

**URL**: https://support.staffbase.com/hc/ja/articles/9707451301522
**取得日**: 2026-05-02 18:56:55 JST

## 内容

Staffbaseプラットフォームにおける言語管理には異なる用語が使用されています。これらの用語を理解することで、組織の言語要件を効果的に管理することができます。
ローカリゼーション
ローカリゼーションとは、ユーザーの言語選択に基づいてインターフェースとコンテンツの言語を適応させることです。
例えば、あなたが異なる国にオフィスを持つ多国籍企業である場合、組織に関連する言語を追加し、それらの言語でコンテンツを作成することができます。これにより、ユーザーは選択した言語でコンテンツを読むことができます。
言語カスタマイズ
言語カスタマイズとは、インターフェースに自動的に表示される特定のテキストを修正することです。使用ケースに応じて、企業のアイデンティティに合わせて修正することができます。
例えば、あなたのアプリやイントラネットのサインインダイアログのテキストをカスタマイズする必要があります。カスタマイズについてのリクエストや詳細を知りたい場合は、Staffbaseサポートまたはカスタマーサクセスマネージャーにお問い合わせください。
LTR（左から右）言語
LTR言語は左から右に書かれ、読まれます。ほとんどの言語はLTRのカテゴリに入ります。
例えば、英語、フランス語、スペイン語
RTL（右から左）言語
RTL言語は右から左に書かれ、読まれます。
例えば、アラビア語、ヘブライ語、ウルドゥー語
