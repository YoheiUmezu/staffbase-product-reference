## 内容

フィルターの詳細
日付
すべてのタブで利用可。
特定の四半期やコンテンツキャンペーン後のデータを収集し、パフォーマンスを確認できます。
プリセットオプションを使用するか、独自の日付範囲を選択して、データの時間枠を絞り込んだり広げたりできます。
ニュース、コンテンツ、チャット、ユーザー：
プリセットオプションは「今日」「7日間」「昨日」「30日間」「365日間」「今週」「今月」「今年」です。
- 「365日間」は閲覧日から過去365日間をカウントします。
- 「今年」は1月から12月までの現在の暦年です。
デフォルトは7日間に設定されています。
検索：
プリセットオプションは「今月」「前月」「今四半期」「直近の四半期」「今年」「昨年」です。
デフォルトは今月に設定されています。
ページとメール：
プリセットオプションは「今週」「先週」「今月」「前月」「今四半期」「直近の四半期」「今年」「昨年」です。
デフォルトは先週に設定されています。
スペース
ニュース、コンテンツ、ユーザー、メールの各タブで利用可。
アクセス権のある1つ以上のスペースを選択することができます。
デフォルトは7日間に設定されています。
プラットフォーム
ニュース、コンテンツ、ユーザーの各タブで利用可。
Android、iOS、Webから選択できます。
デフォルトはすべてのプラットフォームに設定されています。
ユーザータイプ
有効化
ニュースとコンテンツの各タブで利用可。
登録済みユーザーまたは未登録ユーザーから選択できます。
登録状況に基づいたアナリティクスを確認できます。
未登録ユーザーはすべてのパブリックコンテンツにアクセスできます。これにより、管理者はそのコンテンツへのインタラクションも確認できます。
デフォルトは登録済みユーザーに設定されています。
Cookie、トラッキングピクセル、その他の個別トラッキング方法は使用しません。
このフィルターを有効にするには、Staffbaseサポートまたはカスタマーサポートマネージャーにお問い合わせください。
ユーザーグループ
ニュース、コンテンツ、ユーザー、検索の各タブで利用可。
アクセス権のある1つ以上のユーザーグループを選択できます。
クローズドグループとオープングループの両方が含まれます。
チャンネル
ニュースタブで利用可。
アクセス権のある1つ以上の新しいチャンネルを選択できます。
デフォルトはすべてのチャンネルに設定されています。
コンテンツの種類
コンテンツタブで利用可。
アクセス権のある1つ以上のコンテンツの種類を選択できます。
コンテンツには以下が含まれます：
- フォルダ
- サブコンテンツのない個別プラグイン（ネットワークやチャットなど）
- プラグイン内の個別コンテンツ（単一ページやアンケートなど）
デフォルトはすべてのコンテンツの種類に設定されています。
タイプの作成
ユーザータブで利用可。
ユーザーがプラットフォームに追加された方法を選択します。
プリセットオプション：
- Eメールによる自己登録
- CSVインポート
- 手動での招待
- Active Directory/LDAP
- シングル サインオン
デフォルトはすべての作成タイプに設定されています。
フォルダ
メールタブで利用可。
アクセス権のある1つ以上の特定のフォルダを選択できます。
デフォルトですべてのアクセス可能なフォルダが表示されます。
キャンペーン
メールタブで利用可。
アクセス権のある1つ以上のキャンペーンを選択できます。
デフォルトですべてのアクセス可能なキャンペーンが表示されます。
部門
メールタブで利用可。
受信者の部門を選択します。
部門は、まずプロフィールフィールドとして追加し、各受信者の連絡先情報に入力する必要があります。
デフォルトですべての部門が表示されます。
位置
メールタブで利用可。
受信者の位置を選択します。
位置は、まずプロフィールフィールドとして追加し、各受信者の連絡先情報に入力する必要があります。
デフォルトですべての位置が表示されます。
場所
メールタブで利用可。
受信者の場所を選択します。
場所は、まずプロフィールフィールドとして追加し、各受信者の連絡先情報に入力する必要があります。
デフォルトですべての場所が表示されます。
モード
ナビゲーターで利用可。
ユーザーがナビゲーターと音声またはテキストでやり取りしたかを選択します。
プリセットオプション：
- All Modes
- Text Mode
- Voice Mode
デフォルトですべてのモードが表示されます。
報告された問題
ナビゲーターで利用可。
会話中にユーザーから報告された問題を選択します。
プリセットオプション：
- エラー
- 質問の誤解
- 不正確なインフォメーション
- 関連性がない
- その他
デフォルトですべての問題が表示されます。
言語
ナビゲーターで利用可。
会話の言語を選択します。
デフォルトでAIで利用可能なすべての言語が表示されます。
トピック
ナビゲーターで利用可。
ユーザーが最もよく質問するエリア
プリセットオプション：
- 給与
- 福利厚生
- ITアクセス
- 出張ポリシー
- オンボーディング
- 内部ツール
- 人事プロセス
- 会社ポリシー
デフォルトですべてのトピックが表示されます。
