# ライブ配信イベントへの参加

**URL**: https://support.staffbase.com/hc/ja/articles/32680447432850
**取得日**: 2026-05-02 18:54:53 JST

## 内容

Staffbase Live でライブ配信イベントに参加する場合、モデレーターまたは視聴者として参加できます。それぞれの役割には異なる目的とアクセス方法があります。
ライブイベントに参加する方法
割り当てられた役割に応じてイベントにアクセスできます。
モデレーター
スピーカーの場合、通常はイベント作成者から個別メッセージやメールでモデレーター用リンクが届きます。このリンクを使ってイベントにアクセスしてください。
Studio で、コンテンツ > イベントセンター > スケジュール > プレビューしたい配信設定に移動します。
三点リーダーをクリックしてアクションメニューを開きます。
配信をプレビューをクリックします。
Broadcast Studio タブが開きます。
ライブ配信を開始をクリックします。
ブラウザーにカメラとマイクへのアクセス許可を与える必要がある場合があります。
リンクにアクセスできたら、イベントのホストを開始できます。セッションを終了する際は、ライブ配信を終了をクリックできます。
視聴者
ユーザーとして追加された場合、アプリまたはイントラネットでイベントが配信されるリンクが記載されたメールが届きます。
イベントリンクが記載されたメールを開きます。
イベントに参加をクリックします。
イベントページに移動します。早く参加した場合はイベント開始前であることが表示され、遅れて参加した場合はイベントが終了したことが表示されます。
