# オープングループのユーザーを手動で管理する

**URL**: https://support.staffbase.com/hc/ja/articles/360009728619
**取得日**: 2026-05-02 18:56:09 JST

## 内容

管理者
オープングループを作成すると、ユーザーは希望に応じてグループに参加したり、退会したりできます。ただし、スタジオを通じてオープングループの設定に対してユーザーを手動で追加および削除することもできます。管理者によって実行されますが、オープングループからユーザーを手動で追加または削除することは、そのアクションを永続的にするものではありません。削除されたユーザーは、いつでも削除されたオープングループに再参加できます。
オープングループにユーザーを手動で追加する
スタジオで、ユーザー設定 > ユーザーグループに移動します。
メンバーを追加したいオープングループを選択します。
メンバーを追加をクリックします。
グループに追加したいユーザーの名前を入力します。
追加をクリックします。
ユーザーがオープングループに追加されました。ユーザーのアプリは、自動的に追加されたオープングループの内容を反映するように更新されます。
オープングループからユーザーを手動で削除する
スタジオで、ユーザー > ユーザーグループに移動します。
メンバーを削除したいオープングループを選択します。
削除したいユーザーにカーソルを合わせ、Xをクリックして削除します。
ユーザーグループからの削除を確認するダイアログが開きます。
削除をクリックします。
ユーザーがオープングループから削除されました。削除されたユーザーは、このオープンユーザーグループに公開されたコンテンツをもはや見ることはできませんが、いつでもグループに再参加できます。
