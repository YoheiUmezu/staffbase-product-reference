# 従業員のメールドメインとIPアドレスの許可リスト

**URL**: https://support.staffbase.com/hc/ja/articles/4412581517714
**取得日**: 2026-05-02 18:54:02 JST

## 内容

多くの組織では、インターネット上の攻撃から保護するためにファイアウォールやその他のメールセキュリティツールを導入しています。
従業員メールウェブアプリが追跡および送信に使用するドメインとIPアドレスを、組織のシステムが信頼するように設定します。これにより、メールの配信成功、追跡、およびスパムフォルダーの回避が確保されます。
詳細情報はこちらをクリックしてください。
