# 戦略的計画の概要

**URL**: https://support.staffbase.com/hc/ja/articles/21215820957586
**取得日**: 2026-05-02 18:43:03 JST

## 内容

ベータ
アクティベーション
システム全体のエディター
戦略的計画は、キャンペーンとそれに対応するニュース投稿の概要を構造的かつ動的に把握することを可能にします。これにより、全体的なコミュニケーション戦略を監視し、重複やギャップを特定することができます。これにより、コミュニケーション施策の効率と影響力が向上します。
戦略的計画のタイムラインでは、選択した範囲内のすべてのキャンペーンを確認し、関連するニュース投稿の詳細に深く掘り下げることができます。月、四半期、または年ごとにキャンペーンを明確に把握し、コンテンツと目標の全体像を把握できます。
戦略的計画では、以下のことが可能です：
すべてのキャンペーンとそれに対応するニュース投稿の概要を確認する
フライアウトメニューでキャンペーンとニュース投稿の詳細を表示する
新しいキャンペーンを作成する
月、四半期、年を含む期間を調整する
矢印を使用して前のまたは次の期間に移動する
キャンペーンでフィルタリングする
