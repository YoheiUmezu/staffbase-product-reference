# ニュースでハイライト機能を使用する

**URL**: https://support.staffbase.com/hc/ja/articles/34101598500626
**取得日**: 2026-05-02 19:02:31 JST

## 内容

エディター
ニュースは、Staffbaseアプリとイントラネットで視聴者ごとに動的なコンテンツを公開できます。ニュース投稿記事をハイライト表示すると、ニュースフィード内で目立つようになり、読者が重要な情報をすぐに確認できます。
投稿をハイライト表示する
Studioでコンテンツ > ニュースに移動します。
ハイライトを追加したいニュースチャンネルと投稿を選択します。
オプションで、ハイライト表示のトグルを有効にします。
下書きを保存または保存をクリックします。
投稿をハイライト表示しました。
