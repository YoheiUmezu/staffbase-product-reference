# デジタルサイネージのための画面の概要

**URL**: https://support.staffbase.com/hc/ja/articles/25493796355986
**取得日**: 2026-05-02 18:57:53 JST

## 内容

アクティベーション
管理エディター
オフィスのラウンジ、食堂、バックオフィス、または廊下で、StaffbaseのScreens機能を使用してニュースを表示します。Screensは、組織のニュースを大きなデジタルスクリーンでスライドショー形式のデジタルサイネージに準備します。ニュースチャンネルまたはニュースページを選択するだけで、スライドショーを作成して公開できます。
従業員が携帯電話にアクセスできる場合、QRコードを有効にすることを決定できます。QRコードが有効になっている場合、従業員はそれをスキャンして、プラットフォーム上で投稿の残りを読むことができます。
すべてのデバイスをデバイス管理を使用してリンクしたら、特定のスクリーンにリンクするようにスライドショーを設定するか、生成された公開URLを使用してウェブブラウザでスライドショーを開くことができます。スクリーンとコンテンツのメディア推奨事項について詳しく学びましょう。
更新を表示するために最小1分のリフレッシュ時間が必要で、スライドショーに含まれる他のニュース投稿やメディアを回転させる時間が加わります。更新内容には以下が含まれます：
新しく公開されたニュース投稿
新しく追加されたメディアファイル
スライドショー設定の変更
ユーザーが大画面でスライドショーを見る方法
ユーザーは、最新のニュース投稿、画像、動画を、スライドごとに事前定義された時間で、横向きまたは縦向きモードのスライドショーとして表示できます。QRコードが利用可能な場合、従業員はコードをスキャンして、アプリがすでにデバイスにインストールされている場合は投稿に移動できます。そうでない場合、アプリをダウンロードするか、ウェブブラウザでアプリを開くかを選択する画面に移動します。
