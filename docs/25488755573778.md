# リソースセンターの概要

**URL**: https://support.staffbase.com/hc/ja/articles/25488755573778
**取得日**: 2026-05-02 18:41:57 JST

## 内容

Staffbaseは、プラットフォームを最大限に活用するためのさまざまなリソースを提供しています。リソースセンターは、これらのリソースをスタジオ内で直接利用できるようにします。
あなたは…
新しい機能の設定中にヘルプが必要ですか？
最新のリリースされた機能を探求したいですか？
Staffbaseキャンパスのコースでさらに学びたいですか？
これらすべてのことやその他の情報は、リソースセンターで見つけることができます。
リソースセンターメニューの開き方
スタジオで、上部メニューバーに移動します。
リソースセンターのアイコンをクリックします。
リソースセンターメニューが開きます。
特定のリソースを再オープンする
リソースはリソースセンターに保存されており、いつでも再オープンできます。
スタジオで、上部メニューバーに移動します。
リソースセンターのアイコンをクリックします。
リソースセンターメニューが開きます。
見たいリソースが含まれているメニューに移動します。
たとえば、Q2リリースをクリックして最近のリリースメニューを再オープンし、最新の機能の概要を見つけます。
リソースセンターを通じて最新のStaffbaseアップデートにアクセスする
リソースセンターは、四半期ごとのリリースに関する洞察だけでなく、小規模なアップデートがリリースされたときにも通知します。
マイナーな改善に関する新しいアップデートが利用可能になると、ユーザーはリソースセンターに青いバッジ通知が表示されます。この通知は、ユーザーが製品ニュースを開くか、7日後に消えます。過去7日間にアップデートが公開されていない場合、製品ニュースエリアには最新の四半期リリース通知テキストのみが表示されます。
スタジオで、上部メニューバーに移動します。
新しいアップデートが利用可能な場合、青いバッジ通知が表示されます。
リソースセンターのアイコンをクリックします。
リソースセンターメニューが開きます。
製品ニュースをクリックします。
詳細を学ぶをクリックします。
最新のStaffbaseアップデートページがサポートポータルで開きます。ここで、従業員アプリ、フロントドアイントラネット、従業員メールに対するすべての改善とバグ修正を確認できます。
オプションのポップアップをオプトアウトする
スタジオ内の一部のポップアップ、たとえばイベントやコミュニティの更新を表示しないようにオプトアウトできます。他のポップアップは、プラットフォームの重要な変更について通知し、完全にオフにすることはできません。たとえば、メンテナンス情報や四半期リリース情報などです。
スタジオで、上部メニューバーに移動します。
リソースセンターのアイコンをクリックします。
リソースセンターメニューが開きます。
ポップアップの設定に移動します。
設定ダイアログが開きます。
オプトアウトしたいポップアップを選択します。
オプションのポップアップをオプトアウトしました。再度オプトインしたい場合は、設定ダイアログを再オープンし、設定を変更してください。
なぜリソースセンターが見えないのか
スタジオにアクセスできるが、リソースセンターのアイコンが見えない場合、いくつかの理由が考えられます：
あなたの組織がリソースセンターをオフにしましたか？
あなたの組織がリソースセンターを使用しているかどうかを確認するために、アプリまたはイントラネット管理者に連絡してください。
インターネットブラウザに広告ブロッカーが有効になっていますか？
リソースセンターの技術が一部の広告ブロッカーによって広告として分類される可能性があります。広告ブロッカーが原因であるかどうかを確認し、リソースセンターをスタジオで許可するために設定を調整できるかどうかをIT管理者に確認してください。
