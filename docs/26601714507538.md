# SMSに関するよくある質問（FAQ）

**URL**: https://support.staffbase.com/hc/ja/articles/26601714507538
**取得日**: 2026-05-02 18:45:53 JST

## 内容

Studioで双方向テキスト送信はサポートされていますか？
いいえ、Staffbaseスタジオ内では双方向テキスト送信はサポートされていません。ただし、受信者がSMSに返信した場合、その返信はTwilioメッセージログに記録されます。
受信者はSMSの設定を管理できますか？
はい、管理者はユーザーがSMS通知をオプトアウトできるオプションを有効または無効にすることができます。有効にした場合、ユーザーは自身のデバイスの通知設定でSMSの設定を管理できます。
GDPRやTCPAなどの規制下でSMSの同意はどのように扱われますか？
StaffbaseはSMS通知のオプトイン／オプトアウト機能を提供しています。デフォルトで、ユーザーはSMSの設定を管理できます。管理者は自社のコンプライアンス要件に基づいてこの設定を構成してください。
SMSの配信は追跡・記録されますか？
はい、すべてのSMS配信データはTwilioで追跡・記録されます。TwilioのMonitorタブで詳細なログや配信状況を確認できます。
StudioでSMSのメトリクスは利用できますか？
配信率やクリック率などのSMSメトリクスはStudioでは利用できません。詳細なアナリティクスはTwilioアカウントで確認できます。
Studioでメッセージ配信のリアルタイムステータス更新を取得できますか？
いいえ、Staffbaseスタジオ内でリアルタイムの配信状況を表示することはできません。ただし、Twilioのログで確認できます。Twilioでメッセージ配信の進捗を監視し、問題のトラブルシュートが可能です。
文字数・セグメントカウンターはどのように機能しますか？
文字数・セグメントカウンターはメッセージの長さやコスト管理に役立ちます：
160文字までのメッセージは1セグメントとして送信されます。
160文字を超える場合、153文字ごとに分割されて送信されます。
絵文字を使用した場合：
70文字までのメッセージは1セグメントとして送信されます。
70文字を超える場合、67文字ごとに分割されて送信されます。
長いメッセージは複数のセグメントに分割され、受信者のデバイスで再構成されます。
