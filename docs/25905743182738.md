# ServiceNowチケットウィジェットの概要

**URL**: https://support.staffbase.com/hc/ja/articles/25905743182738
**取得日**: 2026-05-02 18:46:01 JST

## 内容

有効化
ServiceNowチケットウィジェットは、ユーザーがアプリやイントラネットを離れることなく、ServiceNowの有効なチケットの概要を簡単に確認できるようにします。ウィジェット内のリンクをクリックすると、ユーザーはServiceNowの該当ページに移動し、チケットの作成、更新、または返信ができます。
スタッフベースは以下のServiceNowモジュールをサポートし、対応するチケットを一覧表示します：
ITSM: インシデント、リクエスト、タスク
HRSD: ケース、タスク
CSM: ケース
SPM: タスク、デマンド
WSD: ケース、タスク
このウィジェットでできること：
チケットのステータスに関する通知を確認する
有効な未解決チケットや割り当てられたチケットの概要を確認する
チケットを閲覧または素早く検索する方法：
検索：検索フィールドにチケット番号または簡単な説明を入力する
フィルター：必要に応じて以下のフィルターを使用する：
優先度
クローズ済み項目を表示
並び替え： 更新日、作成日、または優先度
最新情報を表示するために更新する
ワンクリックでServiceNowのチケットを開く
プラスアイコンをクリックしてServiceNowで新しいチケットを作成する
このウィジェットは、素早いアクセスと概要確認を目的としています。パワーユーザーは、すべての機能にアクセスするためにServiceNowを直接利用してください。
ServiceNowチケットウィジェットを設定するには、次の手順に従ってください：
スタッフベースにServiceNow接続を追加する。
ServiceNowでServiceNowチケットウィジェットを設定する。
StudioでServiceNow接続を有効にする。
ページにチケットウィジェットを追加する。
イントラネットからServiceNowアカウントにサインインする。
ユーザー向けのServiceNowチケットウィジェットの表示
ServiceNowアカウントにサインインした後、ウィジェットでチケットの概要を確認できます。ウィジェット内のチケットをクリックすると、直接ServiceNowの該当チケットページに移動します。
