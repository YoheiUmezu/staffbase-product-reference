# 受信者の受信トレイに表示されるメールプレビューテキストの設定

**URL**: https://support.staffbase.com/hc/ja/articles/5657586307986
**取得日**: 2026-05-02 18:53:52 JST

## 内容

製品名に関する注意: 2022年以降、BananatagはStaffbaseの従業員メール製品です。
受信者が受信トレイでメールを受け取ると、件名の下に短いプレビューテキストが表示されます。このプレビューテキストは、メールの内容についての追加情報を提供します。
メールデザイナーで作成したメールにテキストがなく、画像や動画コンテンツのみを含む場合、受信者は件名の下に次のテキストを見ることになります:
「プレビューは利用できません。」
稀に、メールデザイナーで作成したメールが画像や動画で始まる場合、そのコンテンツがホストされているソースコードが件名の下に表示されることがあります。
回避策として、または表示されるプレビューテキストをカスタマイズするために、メールデザイナーのドラフトの先頭に目に見えないテキスト要素を含む小さなブロックを追加するオプションがあります。
メールプレビューテキストのカスタマイズ
メールデザイナーで、新しいブロックをドラフトの最上部にドラッグ＆ドロップします。
ブロックにテキスト要素を追加し、カスタムプレビューテキストを入力します。
テキストを選択し、最小のサイズに変更し、テキストの色を背景色に合わせて変更します。
-アイコンをクリックして、テキスト要素の周りのスペースを減らします。
オプションで、テキスト要素の外をクリックして、その周りのブロックを選択し、ブロックスペーシングを減らします。
カスタマイズしたメールプレビューテキストが追加されました。
これで、メールを送信すると、このテキストが受信者の受信トレイの件名の下に表示され、その後にメール内の次のテキスト要素の内容が続きます。
