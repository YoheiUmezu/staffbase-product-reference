# Microsoft 365 Teams 概要ウィジェットの設定

**URL**: https://support.staffbase.com/hc/ja/articles/360021042060
**取得日**: 2026-05-02 18:52:10 JST

## 内容

アクティベーション
システム全体のエディター
ユーザーが所属しているチームとユーザーが表示できるチームの概要を追加するには、Microsoft 365 Teams 概要ウィジェットを使用します。さらに、ウィジェット内の検索機能により、ユーザーはチームを迅速に見つけることができます。
前提条件
あなたは Microsoft 365 接続を Staffbase に追加しました。
あなたは アプリまたはイントラネットから Microsoft 365 にサインインしました。
あなたは Microsoft 365 Teams 概要ウィジェットをアクティブ化しました。
Microsoft 365 Teams 概要ウィジェットを追加します。
ウィジェットを設定するためのダイアログが開きます。
オプションで、タイトルを提供します。
タイトルはウィジェットの上に表示され、ユーザーに見えるようになります。
ウィジェットを表示できるデバイスの種類を、対応するアイコンをクリックして選択します。
ウィジェットを表示したいユーザーをアプリ内の可視性の下で選択します。
OKをクリックします。
ウィジェットがページに追加されます。
変更を保存します。
Microsoft 365 Teams 概要ウィジェットを設定して追加しました。
ユーザーが Microsoft 365 Teams 概要ウィジェットを表示する方法
ウィジェットにアクセスする際、ユーザーはまだサインインしていない場合、Microsoft 365 アカウントにサインインするように促されます。
