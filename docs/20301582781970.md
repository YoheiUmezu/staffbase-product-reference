# スマートインサイトダッシュボードによるコメント感情メトリクスの分析

**取得日**: 2026-05-02 18:55:26 JST

## 内容

アクティベーション
システム全体のエディター
AI
スマートインサイトダッシュボードは、各キャンペーンの詳細な統計を表示します。ダッシュボード内では、キャンペーンに割り当てられたニュース投稿の可視性、エンゲージメント、感情に関する詳細なメトリクスを確認できます。スマートインサイトダッシュボードの詳細を学ぶ。
ダッシュボードには、分析ダッシュボードのキャンペーンタブからアクセスできます。スマートインサイトダッシュボードは、すべてのシステム全体のユーザーロールに対して表示されます。スマートインサイトダッシュボードは、スマートインパクト機能の一部です。スマートインパクトの詳細を学ぶ。
トップレベルメトリクスに関するより詳細な統計を提供するセクションがあります：
可視性
エンゲージメント
コメント感情
アライメント
コメント感情は、ユーザーがキャンペーンに割り当てられたニュース投稿に残したコメントに基づいて、キャンペーンコンテンツに対する感情を示します。これは、ポジティブなコメントとネガティブなコメントの割合の差として計算されます。計算されたスコアは-100から+100の範囲です。
コメントの概要ページでは、各コメントの横にコメント感情ラベルが表示されます。そこで、感情によって概要をフィルタリングしたり、必要に応じて感情ラベルを編集したりできます。コメントの概要ページでの感情ラベルの使用についての詳細を学ぶ。
感情分析は、特定のキャンペーンに割り当てられた投稿のすべてのコメントを処理し、ラベル付けすることによって行われます。この処理は、社内のパイプラインを介して機械学習モデルを使用して行われます。このパイプラインは、各コメントがポジティブ、中立、またはネガティブとしてラベル付けされる確率を提供します。このモデルは複数の言語をサポートしています。完全に社内で実行されるため、Staffbaseは分析を実行するためにサードパーティのプロセッサを必要としません。
感情分析は、アラビア語、英語、フランス語、ドイツ語、ヒンディー語、イタリア語、スペイン語、ポルトガル語のコメントに対して完全に利用可能です。他の言語のコメントに対する分析は、正確性が低い場合があります。
時間経過に伴うコメント感情グラフ
タイムラインは、特定の時点で収集されたコメントの感情の分布を示します。
日付のスタックバーは、キャンペーンに割り当てられたニュース投稿に残されたコメントの分布に対応しています。これらのコメントは、スコアに基づいて3つの領域に分類され、色で示されます：
ポジティブなコメント（緑）
中立的なコメント（灰色）
ネガティブなコメント（オレンジ）
タイムラインのデータポイントにカーソルを合わせると、フライアウトが開きます。フライアウトには、以下の数と正確な割合が含まれます：
ポジティブなコメント
中立的なコメント
ネガティブなコメント
投稿ごとの感情テーブル
このテーブルは、個々のニュース投稿に基づく感情を詳細に示します。これにより、どのニュース投稿がユーザーからどのような反応や感情を引き出しているかを確認できます。
テーブルの最上行には、すべてのニュース投稿のコメントの総数とコメントしたユーザーの総数が表示されます。投稿タイトルをクリックすると、アプリまたはイントラネットのニュース投稿に移動します。
テーブルの行にカーソルを合わせると、少なくとも1つのニュース投稿に関与したユーザーの正確な数が表示されます。テーブルの上部にあるドロップダウンメニューをクリックして、テーブルを感情スコアの高い順または低い順に並べ替えることができます。
