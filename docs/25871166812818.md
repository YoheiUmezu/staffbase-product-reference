# メールをキャンペーンに割り当てる

**URL**: https://support.staffbase.com/hc/ja/articles/25871166812818
**取得日**: 2026-05-02 18:53:18 JST

## 内容

特定のトピックに関するメールをすでに作成している場合は、メール編集者から直接キャンペーンに割り当てることができます。キャンペーン用に新しいメールを作成し、過去のメールも割り当てることで、キャンペーン目標に関する傾向を長期的に把握できます。新しいメールを作成してキャンペーンに割り当てる場合は、こちらの手順に従ってください。
すでに送信済みのメールはキャンペーンに割り当てることができません。
前提条件
キャンペーンが組織で利用可能であること。
少なくとも1つのキャンペーンを作成していること。
既存のメールをキャンペーンに割り当てる
Studioでメールに移動します。
キャンペーンに割り当てたいメールを探します。
その他のオプションアイコン > 編集する をクリックします。
メールデザイナーが開きます。
次をクリックします。
設定のキャンペーンの下で、ドロップダウンメニューからメールを追加したいキャンペーンを選択します。
保存するをクリックします。
このメールはキャンペーンに割り当てられました。
個別のキャンペーン概要で、このメールや他の割り当てられたコンテンツとともに、そのリーチ指標を確認できます。
