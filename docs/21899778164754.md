# StaffbaseでFacebookページを表示するための回避策

**URL**: https://support.staffbase.com/hc/ja/articles/21899778164754
**取得日**: 2026-05-02 18:42:59 JST

## 内容

エディター
Metaは、StaffbaseのFacebookページプラグインに不可欠なモバイルビューを廃止しました。モバイルビューがないと、Facebookページプラグインは機能しません。
したがって、StaffbaseのFacebookページプラグインは2024年10月31日に廃止されます。この更新は、2024年第4四半期リリースノートの10月9日のStaffbaseの変更ログに記載されています。
2024年10月14日、StaffbaseはFacebookページプラグインのすべての現在のインストールを非公開にし、再公開のオプションを無効にしました。これにより、従業員アプリとイントラネットの従業員ビューからプラグインが削除されました。プラグインは2024年10月31日までStaffbase Studioに表示されます。
埋め込みページウィジェットを使用してFacebookページを表示する
Staffbaseの埋め込みページウィジェットを使用して、StaffbaseイントラネットにFacebookページを表示することができます。Facebookページはモバイルでサポートされていないため、モバイルデバイスの埋め込みページウィジェットの表示を無効にすることをお勧めします。
前提条件：
埋め込みたいFacebookページのiFrame URLを生成しています。
スタジオで、Facebookページを追加したいページに移動し、テキストボックス内の配置したい場所にマウスを置きます。
埋め込みページウィジェットを追加します。
埋め込みページウィジェットを構成するためのダイアログが開きます。
オプションで、タイトルを提供します。
タイトルはウィジェットの上に表示され、ユーザーに見えるようになります。
ウィジェットが表示されるデバイスタイプを、対応するアイコンをクリックして選択します。
Staffbaseは、Facebookページがモバイルでサポートされていないため、モバイルデバイスの選択を解除することを推奨します。
ウィジェットが表示されるユーザーをアプリ内の表示の下で選択します。
ターゲットURLの下に、埋め込みたいFacebookページのiFrame URLを貼り付けます。
オプションで、デスクトップおよびモバイルデバイス用のウィジェットの高さをカスタマイズします。
有効にしたい項目を定義します：
ウィジェットとのユーザーインタラクション（ズームやクリックなど）を有効にするには、トグルインタラクションを有効にするを調整します。
ウィジェットのスクロールを有効にするには、トグルスクロールを有効にするを調整します。
9. OKをクリックします。
ウィジェットがページに追加されます。
10. 保存をクリックします。
埋め込みページウィジェットを介して、FacebookページをStaffbaseイントラネットに追加しました。
