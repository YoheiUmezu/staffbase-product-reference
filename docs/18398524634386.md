# CSVファイルを使用したユーザーのインポートと同期

**URL**: https://support.staffbase.com/hc/ja/articles/18398524634386
**取得日**: 2026-05-02 18:53:32 JST

## 内容

ベータ
アクティベーション
Staffbaseは、CSVファイルを使用してプラットフォームにユーザーデータを簡単にインポートおよび同期する方法を提供します。
この方法は、ユーザー管理のさまざまな段階をサポートするために使用できます。具体的には：
新しいユーザーのオンボーディング
既存のユーザーデータの更新
ユーザーのオフボーディング
これにより、Staffbaseが提供する最も人気のあるユーザー管理方法の1つとなっています。
ユーザーインポートと同期の種類
スタジオを介した手動ユーザーインポートと同期
手動CSVインポートオプションにより、管理者はスタジオから直接ユーザーのインポートと同期を実行できます。スタジオのユーザーインターフェースは、ユーザーデータのアップロード、設定、プレビュー、およびインポートを簡単に行うオプションを提供します。詳細な概要ページは、インポートプロセスのさまざまな段階でのインポートの良い概要を管理者に提供します。詳細を学ぶ。
CSVインポートAPIを介した自動ユーザーインポートと同期
この方法により、ITチームはAPIを使用して自動データ同期をスケジュールできます。これにより、スタジオを介した頻繁な手動作業が不要になり、インポートは予め決められた時間に実行されるようにスケジュールできます。APIの機能は、複雑なユーザー同期シナリオを容易にするための柔軟性を提供します。詳細を学ぶ。
