# ファイルマネージャーでのファイルの置き換え

**URL**: https://support.staffbase.com/hc/ja/articles/21683494928786
**取得日**: 2026-05-02 18:58:51 JST

## 内容

ベータ
アクティベーション
重要なポリシーや文書は常に進化しており、Staffbaseはファイルマネージャーからファイルを中央で置き換えるオプションを提供し、ファイルがどこにあってもすべてのインスタンスを更新できます。FAQ文書、組織図、ポリシーおよびガイドラインの更新など、最新のバージョンで既存のファイルを置き換えることで文書を更新できます。
置き換えられたファイルのURLは元のファイルと同じままで、投稿、ページ、メール、その他のファイルがリンクされている場所でのリンク切れを避けます。これにより、すでに文書へのリンクが提供されているユーザーは、最新のファイルにアクセスできます。
ファイル置き換えガイドライン
メディアファイル: 同じメディア形式の他のファイルで画像、動画、音声ファイルを置き換えることができます。同じファイルタイプでなくても構いません。たとえば、JPGをPNGやHEICに置き換えることができます。なぜなら、同じファイルタイプではないものの、どちらも画像だからです。動画や音声ファイルにも同様のことが当てはまります。
画像を置き換えると、変更が反映されるまで最大15分かかることがあります。
文書およびその他の一般的なファイル: 同じファイルタイプの文書およびその他の一般的なファイルのみを置き換えることができます。たとえば、PDFはPDF、Word文書は別のWord文書、zipはzip、またはcsvはcsvファイルです。
役割
すべてのユーザーは、自分がアップロードしたファイルを置き換えることができます。
システム全体の管理者は、すべてのユーザーがアップロードしたファイルを置き換えることができます。
ファイル情報
プレビュー プレビュー画像を表示して、正しいファイルにいることを確認します。
一部のファイルは、画像がない場合やPDFでない場合、プレビュー画像が表示されません。
メタデータは、サイズ、使用されている場所、アップロード日、アップローダー情報などのファイルの詳細を提供します。
新しいファイルは、同じメディア形式またはファイルタイプのファイルで置き換えることを可能にします。
更新ノートは、ファイルを置き換える理由を追加でき、これはファイルのバージョン履歴に記録されます。
バージョン履歴は、誰がファイルを更新したか、いつ更新されたか、なぜ更新されたかを示します。
ファイルマネージャーでのファイルの更新
スタジオで、ファイルに移動します。
ファイルマネージャーページが開きます。
置き換えたいファイルの三点リーダーをクリックします。
ファイルを置き換えるをクリックします。
ファイルを置き換えるダイアログが開きます。
新しいファイルフィールドで、コンピューターから選択をクリックして、デバイスからファイルを選択します。
オプションで、フィールドにファイルをドラッグアンドドロップします。
新しい名前の異なる文書をアップロードしても、文書名はそのままです。したがって、新しい文書をアップロードする際に、document_v2やdocument_Jan2024のようにバージョン番号や日付を追加しても、文書名には影響しません。
オプションで、間違った文書をアップロードした場合はゴミ箱アイコンを使用し、ステップ4を繰り返します。
更新ノートフィールドに、ファイルの変更内容を追加します。
置き換えるをクリックします。
ファイルが置き換えられましたページが表示され、ファイルが置き換えられたことを確認します。
ファイルを正常に置き換えました。
