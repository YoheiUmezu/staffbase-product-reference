# 従業員メールアドインまたは拡張機能の始め方

**URL**: https://support.staffbase.com/hc/ja/articles/4412581481874
**取得日**: 2026-05-02 18:44:12 JST

## 内容

ようこそ、従業員メールへ！私たちの目標は、あなたのようなチームがメールコミュニケーションの効果と影響を高める手助けをすることです。
従業員メールは、美しい従業員メールを作成し、送信時にトラッキングを追加し、エンゲージメントを測定することで、内部コミュニケーション戦略を改善し、測定可能な影響を与えることができます。
Power/BusinessまたはEnterpriseプランを利用することで、さらに強力な機能にアクセスするには、Staffbaseは従業員メールのWebアプリから直接メールを送信することを推奨します。
Webアプリの送信ワークフローを設定する方法をITチームの助けを借りて学びましょう。
最初のステップ
以下の手順では、メールクライアント用のアドインまたは拡張機能をダウンロードしてインストールし、メールを作成してトラッキングし、コミュニケーションの影響を理解する方法を説明します。
従業員メールのWebアプリにサインインする
Outlook用のアドインまたはGmail用の拡張機能をインストールする
メールデザイナーでドラフトとテンプレートを作成する
メールをトラッキングする
結果を読み解く
