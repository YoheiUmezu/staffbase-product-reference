# グループ管理の概要

**URL**: https://support.staffbase.com/hc/ja/articles/27820456673554
**取得日**: 2026-05-02 18:41:28 JST

## 内容

ユーザーグループはStaffbaseプラットフォームの中心です。同じオフィスの場所、部門、または似た興味を持つユーザーをクラスタリングし、これらの類似性に特化したコンテンツをターゲットにすることができます。
Studioのグループ概要ページは、管理者とグループ管理者にプラットフォーム上のすべてのグループの集中化された整理されたビューを提供します。グループはタイプ別に分類されて、ナビゲーションと管理が容易になります。
Staffbaseは、既存のグループ管理に代わる強化されたグループ管理を導入します。この記事では、新しいグループ管理の概要、その利点、および開始方法について説明します。

新しいグループリリース
新しいグループ管理は現在ベータ版として利用可能です。自動的に展開されますが、2025年12月11日にQ4 2025リリースと共に行われます。新しいグループ管理に慣れるために、グループ管理の開始方法を参照してください。
公式リリース前に新しいグループ管理を試したい場合は、Staffbaseサポートまたはカスタマーサクセスマネージャーに連絡して事前に有効にしてください。
移行の準備にもっと時間が必要な場合は、2026年1月31日まで自動展開を延期できます。
グループ管理の利点
直感的なグループビルダー
効率を重視した直感的なグループビルダーでグループを簡単に作成および編集できます。その合理化されたインターフェースは、最も複雑なグループ構成でも自信を持って迅速に管理するのに役立ちます。
高度なロジック
柔軟な含む・除外ロジックを使用して、正確なルールベースのグループを構築します。また、条件付きグループから個々のユーザーを手動で追加または削除することもでき、定義された条件に合わない場合でもグループメンバーシップを完全に制御できます。
簡素化された管理
すべてのグループタスクを単一の概要ページから管理して、時間を節約し手動作業を減らします。グループの編集、削除、複製、およびメンバーシップの直接管理が可能です。さらに、CSVを介したメンバーの追加などの一括アクションにより、大規模なチームのオンボーディングやオープンおよび手動グループの大規模な更新が容易になります。

既存のグループやメンバーに影響はありますか？
いいえ。新しいグループ管理が自動的に展開されると、現在のすべてのグループ、設定、および権限はそのまま維持されます。
既存のグループで新機能を使用できますか？
はい。すべての新機能は既存のグループで動作します。追加の設定は必要ありません。
古いグループ管理の体験にアクセスできますか？
いいえ。このリリースにより、新しいグループ体験が自動的に提供されます。移行にもっと時間が必要な場合は、2026年1月31日までオプトアウトできます。
