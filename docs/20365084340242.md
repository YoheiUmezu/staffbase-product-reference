# 高度なコンテンツレビューの概要

**URL**: https://support.staffbase.com/hc/ja/articles/20365084340242
**取得日**: 2026-05-02 18:43:08 JST

## 内容

アクティベーション
従業員を引き込む魅力的なニュースを作成するには、時間、労力、チームワークが必要です。すべてのニュースチャンネルタイプで利用可能なコンテンツレビュー機能を使用して、投稿の詳細を簡単に確認したり、意見を受け取ったりできます。
スタジオにアクセスできるコンテンツ作成者は、個々のユーザーと投稿を共有することで、コンテンツのレビューをリクエストできます。レビューに招待されたユーザーは、投稿のレビュアーとして追加され、投稿へのリンクがメールで送信されます。ユーザーは、プラットフォーム内のレビュー専用ページに移動し、レビュータブでコメントを残すことができます。スタジオへのアクセスは必要ありません。
高度なコンテンツレビューを使用して、一般的なフィードバックを提供したり、公開の承認をリクエストしたりできます。投稿作成者はすべての新しいコメントについてメール通知を受け取るため、何も見逃すことはありません。
高度なコンテンツレビューを使用すると、次のことができます：
メールを介してレビューをリクエストするために投稿を共有する
コメントを残し、他の人のコメントを見る
コメントを編集する
コメントを削除する
ユーザー名の前に@を入力してユーザーに言及する
ユーザーにタスクを割り当てる
ユーザーから承認をリクエストする
利用可能な場合は複数の言語間で切り替える
新しいコメントに関するメール通知を受け取る
追加の役立つ情報
高度なコンテンツレビューにおけるレビューページアクセスの管理
高度なコンテンツレビューを使用したニュース投稿のレビュー
