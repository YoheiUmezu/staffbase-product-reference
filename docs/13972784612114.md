# APIトークンの管理

## 内容

管理者
APIトークンは、サードパーティのサービスがStaffbaseプラットフォームと通信を確立することを可能にします。APIトークンを監査し、不要になったものを無効化または削除することは良い習慣です。
APIトークンを無効にする
APIトークンを無効にしてアクセスを一時的に停止することができます。
APIトークンを無効にすると、そのAPIトークンを使用しているサードパーティの統合が切断されます。進める前にトークンを無効にできることを確認してください。
Studioで、歯車アイコンをクリックして設定を開きます。
システムの下で、APIアクセスをクリックします。
APIトークンページが開きます。
アクションの下で、3つのドットをクリックしてアクションメニューを開きます。
アクションメニューが開きます。
無効をクリックします。
確認ダイアログが開きます。
無効をクリックします。
APIトークンを無効にしました。いつでも再度有効にすることができます。
APIトークンを有効にする
以前に無効にしたAPIトークンを再度有効にすることができます。
Studioで、歯車アイコンをクリックして設定を開きます。
システムの下で、APIアクセスをクリックします。
APIトークンページが開きます。
アクションの下で、3つのドットをクリックしてアクションメニューを開きます。
アクションメニューが開きます。
有効をクリックします。
確認ダイアログが開きます。
有効をクリックします。
APIトークンを再度有効にしました。これで、APIトークンは統合に使用できます。
APIトークンを削除する
プラットフォームからAPIトークンを永久に削除することができます。
APIトークンを削除すると、そのAPIトークンを使用しているサードパーティの統合が切断されます。統合が不要になったか、新しいAPIトークンに置き換えたことを確認してから進めてください。
Studioで、歯車アイコンをクリックして設定を開きます。
システムの下で、APIアクセスをクリックします。
APIトークンページが開きます。
アクションの下で、3つのドットをクリックしてアクションメニューを開きます。
アクションメニューが開きます。
削除をクリックします。
確認ダイアログが開きます。
削除をクリックします。
APIトークンを永久に削除しました。
