# SharePointからStaffbaseニュース投稿を編集する

**URL**: https://support.staffbase.com/hc/ja/articles/11171247858578
**取得日**: 2026-05-02 18:58:55 JST

## 内容

アクティベーション
エディター
SharePoint用のニュースセントラルでは、ボタンをクリックするだけでSharePointからニュース投稿を編集できます。
Staffbase 365ニュースセントラルを追加したSharePointサイトに移動します。
編集したいニュース投稿を開きます。
スタジオで編集をクリックします。
ニュース投稿の編集ページにリダイレクトされ、必要な変更を行うことができます。
保存をクリックします。
投稿を編集しました。
スタジオで行ったすべての変更は、自動的にSharePointに反映されます。
