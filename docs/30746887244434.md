# ジャーニーに関するよくある質問

**URL**: https://support.staffbase.com/hc/ja/articles/30746887244434
**取得日**: 2026-05-02 18:48:32 JST

## 内容

ジャーニーのステップの日数はどのように計算されますか？
ユーザーがジャーニーを開始した日は1日目としてカウントされます。
各ステップが表示されるタイミングを、ユーザーがジャーニーに参加した後の日数で指定できます。日数の値は1から任意の数まで設定でき、配信の時間も指定できます。
ジャーニーの通知はカレンダーの日ではなく、絶対的なタイムスタンプに基づいて送信されます。つまり、ジャーニーの日は午前0時から午前0時（00:00–24:00）までではなく、ユーザーがジャーニーに参加した時間と各ジャーニーステップのスケジュールされた時間によって決まります。

ジャーニーステップを送信する際に個々のユーザーのタイムゾーンは考慮されますか？
いいえ、ジャーニーステップを送信する際に個々のユーザーのタイムゾーンは考慮されません。スケジュールされた日付と時間は、ジャーニー設定で選択されたタイムゾーンを指します。
ユーザーはジャーニー関連の通知を管理できますか？
いいえ、ユーザーはジャーニーに関連する通知を管理できません。ジャーニーは、オンボーディングの指示やコンプライアンス研修など、重要または時間に敏感な情報を提供するために使用されることが多いため、すべての対象従業員がこれらのコミュニケーションを受け取ることを保証するために、ジャーニーの通知設定は個々のユーザーによって管理することはできません。
特定のジャーニーステップに対して通知が送信されるかどうかは、そのステップ内で設定された通知設定に依存します。Studioと該当するジャーニーにアクセスできるユーザーのみがこれらの設定を調整できます。
ユーザーはいつジャーニーを退出し、再参加できますか？
ユーザーがジャーニーにリンクされたユーザーグループを離れると、自動的にジャーニーから削除されます。ジャーニーがまだ有効な間に同じユーザーグループに再参加すると、自動的にジャーニーにも再参加します。
これは、特定のグループにユーザーをオンボーディングする設定があるジャーニーにのみ適用されます。
ジャーニーに再参加することは、ユーザーが受け取る通知にどのように影響しますか？
これは、選択されたジャーニーの再参加設定に依存します：
ユーザーがジャーニーを続行する
ユーザーはジャーニーに再参加し、ジャーニーの現在のタイムラインを続行します。
ジャーニーに参加していなかった間に見逃した通知は受け取りません。
進行中のタイムラインに一致するステップを続行します。
ユーザーがジャーニーを再開する
ユーザーはステップ1から再開します。
すべての通知と投稿は、ユーザーがジャーニーに新しく参加したかのように、ジャーニーのタイムラインに従って送信されます。
以前に完了したステップは考慮されません。
