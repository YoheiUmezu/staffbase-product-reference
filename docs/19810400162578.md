# CSVファイルのアップロード

**URL**: https://support.staffbase.com/hc/ja/articles/19810400162578
**取得日**: 2026-05-02 18:58:52 JST

## 内容

ベータ
アクティベーション
管理者
ユーザーデータのインポートと同期の準備として、CSVファイルをアップロードできます。複数の管理者が同時にこのページで作業できます。
前提条件:
アップロード用のCSVファイルを準備しています。詳細はこちら。
CSVファイルに定義された各ユーザーの識別子を持っています。
プラットフォームに既存のユーザーがいる場合は、識別子が設定され、CSVファイルの識別子と一致していることを確認してください。詳細はこちら。
スタジオで、ユーザー > ユーザーインポートと同期に移動します。
新しいインポートをクリックします。
アップロードしたいCSVファイルを選択します。
ファイルがアップロードされたことを示すメッセージが表示されます。
ファイルをアップロードしました。これはユーザーインポートと同期ページにドラフトとして表示されます。これで、CSVファイルを設定できます。
