# 従業員メールアドインのバージョン番号を見つける

**URL**: https://support.staffbase.com/hc/ja/articles/4415102151058
**取得日**: 2026-05-02 18:44:05 JST

## 内容

製品名に関する注意: 2022年以降、BananatagはStaffbaseの従業員メール製品です。
従業員メールアドインのバージョン番号を見つける方法:
Outlook Classic
Outlookリボンで、ヘルプを取得ボタンをクリックします。Staffbaseで作成の隣にあります。
新しいメールドラフトが自動的に作成されます。
メールドラフトに含まれている現在のアドインバージョンを見つけます。
Outlook 365
Outlook 365アドインは常に最新の状態であり、手動での更新や再インストールは必要ありません。Macユーザーも同様です。
Gmail
設定をクリックします。
下部に拡張機能のバージョンが表示されます。
Outlook for Mac
Outlook 365アドインはMacでも常に最新の状態であり、手動での更新や再インストールは必要ありません。
