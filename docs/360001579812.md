# メニューにおけるニュースページとフォルダーの多言語表示

**URL**: https://support.staffbase.com/hc/ja/articles/360001579812
**取得日**: 2026-05-02 19:03:16 JST

## 内容

複数の言語でコンテンツを作成するには、アプリがそのように設定されている必要があります。このオプションが必要な場合は、管理者にお問い合わせください。
スタジオで、コンテンツに移動します。
メニューのシステムセクションで、メニューをクリックします。
ニュースページまたはフォルダーを選択します。
このメニュー項目の設定のためにギアアイコンをクリックします。
言語のドロップダウンメニューを開きます。
追加をクリックします。
コンテンツ言語を選択します。
ドロップダウンメニューには、アプリで有効になっているすべてのコンテンツ言語が含まれています。追加の言語が必要な場合は、管理者に有効にするよう依頼してください。
OKをクリックします。
新しい言語でフォルダー名またはタイトルを入力します。
フォルダー名またはタイトルは、メニューでの名前として使用されます。
保存をクリックします。
