# 投稿の著者を変更する

**URL**: https://support.staffbase.com/hc/ja/articles/4403630707346
**取得日**: 2026-05-02 19:03:15 JST

## 内容

システム全体のエディター
コンテンツチームは、他のユーザーの名義で公開できるニュース記事をドラフトすることができます。これは、部門長やCEOの名義で会社全体のニュースを公開する際に特に便利です。
RSSフィードの投稿には公開著者が表示されないため、ゴーストライティングは不可能です。
前提条件
著者として設定したい人は、投稿するチャンネルのエディターでなければなりません。
投稿の著者を変更する
スタジオで、コンテンツ > ニュースに移動します。
著者を変更したいニュース投稿を開きます。
ニュースエディターが開きます。
サイドメニューのオプション > 著者として公開の下から、ドロップダウンメニューで著者として設定したいユーザーを選択します。
保存をクリックします。
投稿の著者を変更しました。システム全体のアクセス権を持たない他のエディターは著者を変更することはできませんが、常に自分が代筆している著者を確認できます。
