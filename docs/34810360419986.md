# AIページ生成を使用したページの作成

**URL**: https://support.staffbase.com/hc/ja/articles/34810360419986
**取得日**: 2026-05-02 18:52:54 JST

## 内容

AI
編集者
AIページ生成は、Content Designer使用時に利用できます。プロンプトに基づいて構造化されたページを素早く作成するのに役立ちます。手動作業を減らしながら、最終的なコンテンツを完全にコントロールできます。
ページを生成するには、コンテンツの目的を説明するプロンプトを入力します。出力を強化するために、追加コンテキスト用のドキュメント、コンテンツを強化するためのメディアファイル、またはテンプレートとして既存のページを追加できます。あなたの入力に基づいて、AIが事前定義されたセクション、ブロック、おすすめコンテンツを含む下書きページを生成します。
最適な結果を得るために、プロンプト作成のベストプラクティスに従ってください。
対応ファイルと制限
生成される出力を向上させるために、追加コンテキストやメディアを提供できます。
ドキュメント
最大数：1
対応形式：PDF、DOCX
複数ページのドキュメントの場合、最初の20ページのみ処理されます
メディア
最大数：5
最大サイズ
ドキュメントおよびメディアの最大サイズ：25MB
前提条件
AIページ生成は、AI設定で管理者によって有効化されています。
AIページ生成は、あなたのユーザーアカウントで有効化されています。
AIページ生成は、Content Designerで新規ページを作成する場合のみ利用できます。既存のページを編集する場合は利用できません。
Content Designerを使用してページを作成する。
ページ概要ページが開きます。
+アイコンをクリックします。
アップロードまたは既存ページを添付するオプションが開きます。
次のいずれかのオプションを選択し、ドキュメントと最大5つのメディアファイルをアップロードします：
コンピュータ
マイファイル
任意で、+アイコンをクリックし、既存ページを添付を選択して、ページをテンプレートとして使用します。
テキストフィールドに、生成したいコンテンツを説明するプロンプトを入力します。
送信アイコンをクリックします。
構造化されたコンテンツでページが生成されます。公開前に下書きをレビューおよび修正できます。
