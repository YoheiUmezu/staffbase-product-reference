# スマートキャンペーンの使用

**URL**: https://support.staffbase.com/hc/ja/articles/19675063543442
**取得日**: 2026-05-02 18:46:36 JST

## 内容

アクティベーション
システム全体のエディター
スマートキャンペーンを使用すると、コンテンツ戦略を監視できます。各スマートキャンペーンに対して、コンテンツ作成を導くための戦略的目標を設定し、開始日と終了日、キャンペーンの色を追加できます。キャンペーンは、特定のトピックに関するニュース投稿など、戦略的に整合したコンテンツをまとめます。コンテンツ戦略を実行に移し、キャンペーンを作成しましょう。
キャンペーンの作成
スタジオで、プランニング > キャンペーンに移動します。
キャンペーンの作成をクリックします。
キャンペーンの作成ページが開きます。
キャンペーンのタイトルを入力します。
オプションで、キャンペーンの目標を入力します。
オプションで、キャンペーンの開始日と終了日を入力します。
キャンペーンの色を選択します。
オプションで、キャンペーンの整合性の下で、トグルを有効にして整合性調査を作成します。
作成をクリックします。
キャンペーンを作成しました。これで、既存のニュース投稿をキャンペーンに割り当てるか、新しい投稿を作成することができます。
キャンペーンの編集
キャンペーンを作成した後は、いつでもニーズに合わせて編集できます。
スタジオで、プランニング > キャンペーン > 編集したいキャンペーンに移動します。
三点メニューをクリックします。
編集をクリックします。
キャンペーンの編集ダイアログが開きます。
キャンペーンの任意のセクションを編集します。
保存をクリックします。
キャンペーンを編集しました。変更はすぐに反映されます。
キャンペーンの削除
キャンペーンを削除すると、すべてのニュース投稿がキャンペーンから解除されます。投稿とその統計は、それぞれのニュースチャンネルで利用可能なままです。削除されたキャンペーンは復元できません。
スタジオで、プランニング > キャンペーン > 削除したいキャンペーンに移動します。
三点メニューをクリックします。
削除をクリックします。
キャンペーンの削除ダイアログが開きます。
削除をクリックして確認するか、キャンセルをクリックして戻ります。
キャンペーンを正常に削除しました。このキャンペーンに以前割り当てられていた投稿は解除されました。
