# Microsoft Teams にスタッフベースのURLを追加する

**URL**: https://support.staffbase.com/hc/ja/articles/29415750520850
**取得日**: 2026-05-02 18:57:49 JST

## 内容

この記事はAIを使用して翻訳されており、不正確な情報が含まれている可能性があります。情報に不安がある場合は、正確性のために英語の原文を参照してください。
アクティベーション
管理者
管理者の同意を与えた後、ユーザーがアクセスできるようにするために、Microsoft TeamsのStaffbase ConnectアプリでStaffbase URLを設定する必要があります。
前提条件
Microsoft 365でグローバル管理者またはTeamsサービス管理者の役割を持っていること。
Microsoft Teamsの左側のナビゲーションペインで、Staffbase Connect Appをクリックします。
アプリ名は組織の設定に基づいて異なる場合があります。
Staffbase 設定の下に、StaffbaseウェブアプリのURLを入力します。
適用するをクリックします。
Microsoft TeamsにStaffbase URLを追加しました。選択されたユーザーは、Microsoft TeamsでStaffbase Connect Appにアクセスできるようになりました。
URLが設定されると、グローバル管理者またはTeamsサービス管理者は、アプリを開くたびにURLを再設定できる画面が表示されます。
この画面は、Microsoft Teamsの管理者が常にアプリの設定にアクセスでき、ロックアウトを回避できるようにします。アプリが適切に設定されている場合でも表示されます。
この動作は、Connectアプリのストアバージョンにのみ適用されます。
