# ページとニュースでのウィジェットの使用

**URL**: https://support.staffbase.com/hc/ja/articles/360010074960
**取得日**: 2026-05-02 18:50:18 JST

## 内容

価格
アクティベーション
エディター
使いやすいウィジェットを使用して、魅力的で情報豊かでエンゲージングなページや投稿を作成し、ページやニュースを強化します。
ウィジェットの追加
スタジオで、ウィジェットを追加したいページまたは投稿に移動します。
エディターで、ウィジェットを追加したい場所にカーソルを置きます。
エディターツールバーで、ウィジェットを追加のプラスアイコンをクリックします。
ウィジェットを追加ダイアログが開きます。
追加したいウィジェットをクリックします。
ウィジェットを設定します。
各ウィジェットには異なる詳細と設定が必要です。
OKをクリックします。
ウィジェットがページまたは投稿に追加されます。
簡単なドラッグアンドドロップ操作で、ページのセクションにウィジェットを追加することもできます。
ウィジェットの詳細のカスタマイズ
ウィジェットを追加した後、その詳細を編集してカスタマイズできます。
スタジオで、ウィジェットを編集したいページまたは投稿に移動します。
エディターで、編集したいウィジェットにカーソルを合わせます。
ウィジェットを設定するためのアイコンが表示されます。
ギアアイコンをクリックして設定を開きます。
または、ウィジェットをダブルクリックします。
ウィジェットを設定するためのダイアログが開きます。
変更したい詳細を編集します。
詳細は、編集しているウィジェットによって異なります。
OKをクリックします。
ウィジェットのレイアウトスタイルの設定
ウィジェットのレイアウトスタイルをカードスタイルありまたはなしで設定できます。カードスタイルは、ウィジェットを境界線のある白いカード上に表示します。
スタジオで、ウィジェットのレイアウトを調整したいページまたは投稿に移動します。
エディターで、削除したいウィジェットまたはセクションにカーソルを合わせます。
ウィジェットを設定するためのアイコンが表示されます。
レイアウトアイコンをクリックします。
ウィジェットのレイアウトが切り替わります。変更されたレイアウトを直接見ることができます。
再度クリックすると、以前のレイアウトに戻ります。
カードレイアウトあり
カードレイアウトなし
ウィジェットの削除
スタジオで、ウィジェットを削除したいページまたは投稿に移動します。
エディターで、削除したいウィジェットにカーソルを合わせます。
ウィジェットを設定するためのアイコンが表示されます。
削除アイコンをクリックします。
ウィジェットが削除されます。
削除後は復元できません。新しいウィジェットを追加してください。
保存をクリックします。
