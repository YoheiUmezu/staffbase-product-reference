# チャットの既読通知を無効にする

**URL**: https://support.staffbase.com/hc/ja/articles/9678601421458
**取得日**: 2026-05-02 18:58:09 JST

## 内容

既読通知は、チャットの1対1の会話で利用可能です。メッセージを送信し、同僚が会話を開くと、メッセージの下に目のアイコンが表示され、既読であることを示します。ユーザー設定で既読通知を完全に無効にすることができます。これにより、他のユーザーはあなたがメッセージを読んだことを確認できず、あなたも相手がメッセージを読んだかどうかを確認できなくなります。
iOS
アプリメニューを開きます。
設定を開くためにギアアイコンをタップします。
設定ページが開きます。
アカウントをタップします。
アカウント設定ページが開きます。
既読通知のトグルを調整して、チャットメッセージの既読通知を無効にします。
チャットの既読通知が無効になりました。メッセージを読んだときに表示されなくなり、あなたのメッセージが読まれたかどうかも確認できなくなります。
Android
プロフィールメニューを開きます。
設定をタップします。
設定ページが開きます。
アカウントをタップします。
アカウント設定ページが開きます。
既読通知のトグルを調整して、チャットメッセージの既読通知を無効にします。
チャットの既読通知が無効になりました。メッセージを読んだときに表示されなくなり、あなたのメッセージが読まれたかどうかも確認できなくなります。
Webアプリ
プロフィールメニューを開きます。
設定をクリックします。
アカウントをクリックします。
既読通知のトグルを調整して、チャットメッセージの既読通知を無効にします。
チャットの既読通知が無効になりました。メッセージを読んだときに表示されなくなり、あなたのメッセージが読まれたかどうかも確認できなくなります。
既読通知オプションを無効にすると、オプションが有効だった間に読んだメッセージの下に目のアイコンが残ります。
