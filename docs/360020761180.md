# Microsoft 365ファイルウィジェットの設定

**URL**: https://support.staffbase.com/hc/ja/articles/360020761180
**取得日**: 2026-05-02 18:57:19 JST

## 内容

アクティベーション
システム全体のエディター
Staffbaseプラットフォームのページやニュース投稿にMicrosoftファイルのリストを追加するには、Microsoft 365ファイルウィジェットを使用します。エディターは、このウィジェットを追加して、トレンドや最近のファイル、特定のライブラリからのファイルなど、Microsoft 365のさまざまなタイプのファイルリストを表示できます。ユーザーは、ファイルをリストビューまたはタイルビューのいずれかで表示することを選択できます。さらに、ウィジェット内の検索機能により、ユーザーは探しているファイルを迅速に見つけることができます。
SharePointでは、コンテンツ承認設定により、承認されたコンテンツのみがユーザーに表示されます。表示したいファイルのライブラリに対してコンテンツ承認設定が有効になっている場合、ウィジェットには承認されたファイルのみが含まれます。また、ウィジェットに表示したいファイルに対するユーザーのアクセス権があることを確認してください。
前提条件
あなたは Microsoft 365接続をStaffbaseに追加しました。
あなたは アプリまたはイントラネットからMicrosoft 365にサインインしました。
あなたは Microsoft 365ファイルウィジェットを有効化しました。
Microsoft 365ファイルウィジェットを追加します。
ウィジェットを設定するためのダイアログが開きます。
オプションで、タイトルを提供します。
タイトルはウィジェットの上に表示され、ユーザーに見えるようになります。
ウィジェットを表示できるデバイスの種類を、対応するアイコンをクリックして選択します。
ウィジェットを表示したいユーザーをアプリ内の表示で選択します。
ドロップダウンメニューリストタイプで、ウィジェットに追加したいリストタイプを選択します:
トレンドファイル
SharePointまたはOneDriveで現在最も人気のあるファイルを表示します。このファイルリストは、すべてのユーザーのアクティビティに基づいて、最も表示されたり変更されたファイルを表示します。
最近のファイル
SharePoint OnlineおよびOneDriveで最近変更または表示されたファイルを表示します。ファイルリストは、各ユーザーに対して、最も最近にやり取りしたファイルを表示するように適応します。
特定のライブラリ
任意のSharePoint OnlineサイトのドキュメントライブラリからのSharePoint Onlineファイルを表示します。ライブラリは、特定の順序で特定のドキュメントを表示します。
検索ベース
提供された検索クエリに基づいて、任意のSharePoint OnlineサイトのドキュメントライブラリからのSharePoint Onlineファイルを表示します。
特定のライブラリの場合、表示したいファイルのサイトとライブラリを選択します。
検索ベースリストタイプの場合、次の情報を提供します:
クエリテキスト: キーワードクエリ言語 (KQL)で検索クエリテキストを入力します。
ソートプロパティ: 検索をソートするプロパティを入力します。
ソート順: 検索をソートする順序を選択します。
表示するファイルの数を選択します。
この数のファイルは、ユーザーのファイルリストに自動的に表示されます。さらにファイルは、スクロールすることで表示されます。
OK.をクリックします。
ウィジェットがページに追加されます。
変更を保存します。
Microsoft 365ファイルウィジェットを設定して追加しました。
ユーザーがMicrosoft 365ファイルウィジェットを表示する方法
ウィジェットにアクセスする際、ユーザーはMicrosoft 365アカウントにサインインするように促されます（まだサインインしていない場合）。
