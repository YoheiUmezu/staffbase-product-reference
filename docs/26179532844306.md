# パスキーの管理

**URL**: https://support.staffbase.com/hc/ja/articles/26179532844306
**取得日**: 2026-05-02 18:53:17 JST

## 内容

有効化
新しいパスキーの作成や、不要になったパスキーの削除によって、パスキーを管理できます。
パスキーを作成する
パスワードを入力せずにアプリやイントラネットにサインインするためのパスキーを作成できます。
ご利用のエコシステムによっては、1つまたは2つのパスキーを作成する必要があります。パスキーとその仕組みについて詳しくはこちらをご覧ください。
パスキーを作成する方法は2つあります。
登録時
プラットフォームに初めて登録する際にパスキーを作成できます。最初のサインインが成功すると、パスキーの作成を促すメッセージが表示されますが、次回以降のサインイン時には表示されません。このメッセージをスキップした場合や新しいパスキーを作成したい場合は、アカウントの設定で作成する必要があります。
ウェブブラウザで、アプリまたはイントラネットのドメインURLを入力します。例：https://mycompany.com
サインインページが開きます。
サインイン方法（例：ユーザーネームとパスワード、またはSSO）でアカウントにサインインします。
+ パスキーを作成するをクリックします。
パスキー作成の案内が表示されます。
今はスキップをクリックした場合、アカウントの設定でのみパスキーを作成できます。
デバイスの案内に従ってパスキーを作成・保存します。例：macOSではTouch IDで指紋認証を求められます。
パスキーが作成されました。今後のサインイン時に使用できます。新しいパスキーを作成する場合は、アカウントの設定で作成してください。
アカウントの設定で
すでに登録済みのユーザーで、まだパスキーを作成していない場合や新しいパスキーを作成したい場合に利用します。
サインインダイアログにはパスキーでサインインオプションがありますが、このボタンはすでにアプリやイントラネット用のパスキーを作成している場合のみ機能します。新しいパスキーの作成には使用できません。
アプリまたはイントラネットで、設定 アカウント パスキーに移動します。
パスキーの一覧が開きます。
+ パスキーを作成するをクリックします。
パスキー作成のダイアログが開きます。
デバイスの案内に従ってパスキーを作成・保存します。例：macOSではTouch IDで指紋認証を求められます。
案内が完了すると、プライベートキーがデバイスのパスワードマネージャーに保存され、アプリやイントラネットの設定 アカウント パスキーにパスキーが表示されます。
パスキーでサインインする
パスキーを作成した後は、パスキーでサインインできます。
ウェブブラウザで、アプリまたはイントラネットのドメインURLを入力します。例：https://mycompany.com
サインインページが開きます。
パスキーでサインインをクリックします。
指紋認証、顔認証、またはデバイスロックの入力を求められます。
アカウントの設定でパスキーを作成していない場合は利用できません。
選択した認証方法を提供します。
アプリまたはイントラネットにサインインできます。
パスキーを削除する
不要になった場合やデバイスを廃棄する場合は、アカウントからパスキーを削除できます。
サインインの問題を防ぐため、スタッフベースのプラットフォームとデバイスのパスワードマネージャーの両方からパスキーを必ず削除してください。パスワードマネージャーからプライベートキーを削除する方法は、各デバイスのパスワードマネージャーのマニュアルをご確認ください。
アプリまたはイントラネットで、設定 アカウント パスキーに移動します。
パスキーの一覧が開きます。
削除したいパスキーに対応するXをクリックします。
パスキー削除のダイアログが開きます。
削除するをクリックします。
スタッフベースのプラットフォームからパスキーが削除されます。
デバイスの設定で、パスワードマネージャーからパスキーのプライベートキーを削除します。
保存されたキーや認証情報の削除手順は、パスワードマネージャーのマニュアルをご参照ください。
スタッフベースアカウントのパスキーを完全に削除できました。
