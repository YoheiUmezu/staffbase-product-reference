# コミュニティ発見ページの利用方法

**URL**: https://support.staffbase.com/hc/ja/articles/35134258148882
**取得日**: 2026-05-02 18:54:45 JST

## 内容

コミュニティは、ディスカッションや共有のための専用エリアで、より強い接続や主要な取り組みへの連携をサポートします。コミュニティ発見ページを使って、いつでも関連するコミュニティを見つけて参加し、より接続されたエンゲージメントの高い職場を作ることができます。
発見ページでは、あなたがアクセスできるすべてのコミュニティを表示し、参加してそのコンテンツを閲覧できます。コミュニティ内では、投稿を作成したり、他の投稿にリアクションや返信をしたり、新しいコンテンツのお知らせを受け取ることができます。お知らせはいつでも調整できます。
コミュニティに参加する
アプリまたはイントラネット内で、コミュニティに移動します。
あなたがアクセスできるすべてのコミュニティが表示されます。
興味のあるコミュニティに移動します。
参加をクリックします。
コミュニティページが開き、既存の投稿が表示されます。
コミュニティに参加しました。これで投稿や他のメンバーとの交流ができます。いつでもコミュニティから退出できます。
