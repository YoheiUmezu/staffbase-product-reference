# Internet Explorer 11のサポート終了が従業員メールに与える影響

**URL**: https://support.staffbase.com/hc/ja/articles/4412584518674
**取得日**: 2026-05-02 18:51:51 JST

## 内容

製品名に関する注意: 2022年以降、BananatagはStaffbaseの従業員メール製品です。
何が起こっているのか？
Microsoftは2021年8月にInternet Explorer 11（IE11）のサポートを終了しました。これにより、すべてのIE11ブラウザーが突然動作しなくなるわけではありませんが、アプリやウェブサイトがブラウザーのサポートを停止するにつれて、体験が徐々に劣化することを意味します。IE11の廃止については、Microsoftのサポートウェブサイトでこちらで読むことができます。
従業員メールの顧客に与える影響は？
MicrosoftのIE11サポート終了は、2つのことを意味します：
IE11を使用してapp.bananatag.comにアクセスしている場合は、できるだけ早く別のウェブブラウザーに切り替える必要があります。
2021年8月以降、IE11を通じて製品のサポートを完全に停止しました。これは、バグの修正や新機能のテストを停止したことを意味します。
2021年9月中旬以降、app.bananatag.comはIE11から完全にアクセスできなくなります。
Outlookアドインを使用している場合は、Outlook 365またはOutlook Classic 3製品にアップグレードする必要があります。
IE11をウェブブラウザーとして使用していない場合、Outlookアドインを使用していない場合、またはすでにOutlook 365またはOutlook Classic 3製品を使用している場合は、特にアクションは必要ありません。
1. IE11をウェブブラウザーとして使用しています
MicrosoftがIE11のサポートを終了することがわかっているため、従業員メールウェブアプリがこのブラウザーを通じてアクセスされるときにのみ発生する問題の修正に割く時間とリソースを減らしています。
そのため、最良の体験を得るためには、できるだけ早くGoogle ChromeまたはMicrosoft Edgeに切り替えることをお勧めします。
2021年9月中旬以降、app.bananatag.comはIE11から完全にアクセスできなくなります。
2. Outlookアドインを使用しています
私たちのOutlook Classic 2アドインはIE11によって動作しているため、残念ながら2021年8月にサポートを終了しました。
できるだけ早く以下のいずれかにアップグレードすることをお勧めします：
Outlook 365拡張機能（組織がMicrosoft 365を使用している場合は推奨）
Outlook Classic 3アドイン
幸いなことに、どちらのオプションにも多くの改善が含まれており、すべての従業員メールプランに含まれています。
どのバージョンのOutlookアドインを使用しているかわからない
心配しないでください！どのアドインバージョンを使用しているか確認できます。
Outlookの使用に疲れたらどうすればいいですか？
メールは従業員メールウェブアプリから直接送信できます。Outlookに依存しなくなるだけでなく、独自のカスタム配信リストを迅速に作成したり、従業員属性に基づいてメトリクスをセグメント化したりすることができます。このオプションに興味がある場合は、アカウントマネージャーにお問い合わせください。
新しいブラウザーを入手するにはどうすればよいですか？
従業員メールで最良の体験を得るための選択肢はいくつかあります。
推奨トップ:
- Google Chrome
その他の推奨:
- Microsoft Edge
- Apple Safari
