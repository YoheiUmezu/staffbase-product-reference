# メールデザイナーからのメールのエクスポート

**URL**: https://support.staffbase.com/hc/ja/articles/4412584493970
**取得日**: 2026-05-02 18:55:55 JST

## 内容

Employee Emailのメールデザイナーで作成されたメールは、ニーズに応じてHTMLまたはPDFファイルとしてダウンロードできます。
HTMLエクスポートは、イントラネットなどのウェブページとしてホスティングするのに便利で、ユーザーがインターネットブラウザーで閲覧できるようになります。開く場所によって異なる表示になる可能性があるため、常に同じように見えるとは限りません。
HTMLエクスポートは、校正またはアーカイブ目的以外には推奨されません。HTMLファイルをメールとして送信しようとすると、フォーマットに悪影響を及ぼし、追跡されません。
PDFエクスポートは、印刷およびアーカイブ目的に便利です。事前にレンダリングされているため、どのソフトウェアやデバイスで開いても同じように見えるはずです。
メールのドラフトまたはテンプレートをダウンロードする
Employee Emailウェブアプリでドラフトまたはテンプレートを編集している間に、ダウンロードオプションアイコンをクリックします。
開いたドロップダウンメニューで、PDFまたはHTMLファイルをダウンロードするかを選択します。
送信済みメールをダウンロードする
送信済みメールのコピーをダウンロードするには、まずそれからドラフトを作成する必要があります。
メールを送信したアカウントでサインインします。
誰でも通常の方法でサインインでき、共有メールアドレスに対応するアカウントなどの共有資格情報を使用できます。
もしあなたがマネージャー権限以上のEmployee Emailアカウントを持っている場合、組織内の他のユーザーとしてサインインすることもできます。
メールが作成されたアカウントとは異なるアカウントから送信された場合、ダウンロードする権限がない可能性があります。権限の高いアカウントでサインインするか、Staffbaseサポートまたはカスタマーサクセスマネージャーに連絡して回復してもらうことができます。
Email>送信済みメールに移動します。
ダウンロードしたい送信済みメールの詳細を表示するためにクリックします。
ドラフトを作成をクリックします。
ドラフトに名前を付けて次へをクリックします。
ダウンロードオプションアイコンをクリックします。
開いたドロップダウンメニューで、PDFまたはHTMLファイルをダウンロードするかを選択します。
