# グループコンテンツへの直接リンクと@メンション

**URL**: https://support.staffbase.com/hc/ja/articles/360019390359
**取得日**: 2026-05-02 19:01:57 JST

## 内容

アクティベーション
オープングループコンテンツは、グループメンバーシップに関係なくすべてのユーザーがアクセスできます。直接リンクを使用してオープングループコンテンツを共有したり、ニュース、ページ、プラグイン、ウィジェットにリンクしたりできます。オープングループコンテンツの下のコメントでユーザーを@メンションすることで、そのコンテンツが関連していることを直接通知できます。
オープングループコンテンツへのリンクと@メンションを使用することで、アプリやイントラネットを情報ハブに変え、クリーンなメニューと構造を提供しつつ、必要に応じて豊富なコンテンツを利用できるようにします。
追加の有用な情報
オープングループコンテンツは検索結果にも表示されます。詳細はユーザーがコンテンツを発見する手助けをご覧ください。
