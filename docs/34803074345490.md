# Navigatorのブランディングの設定

**URL**: https://support.staffbase.com/hc/ja/articles/34803074345490
**取得日**: 2026-05-02 18:40:29 JST

## 内容

アクティベーション
AI
管理者
NavigatorはStaffbaseアプリとイントラネットをインテリジェントなワークプレイスアシスタントに変えます。ユーザーは音声またはテキストで設定する言語を使って質問し、企業の知識に基づいたインスタントな回答を受け取ることができます。
Navigatorの外観をカスタマイズして、あなたのブランドに合わせたり、ユーザーの体験を向上させたりできます。すべてのブランディングカスタマイズは保存前にプレビューできます。
1. Studioで設定 > アプリとイントラネットに移動します。
アプリとイントラネットのブランディングページが開きます。
2. アプリ > Navigatorに移動します。
3. ロゴの下で、Navigatorアイコンとして使用する画像をアップロードします。
4. メッセージの色の下で、次の設定を行います。
テキスト：チャットテキストのカラーを選択する。
バックグラウンド：メッセージのバックグラウンドカラーを選択する。
5. ボイスオーブの色の下で、ユーザーが音声入力を使用する際に表示されるアクセントカラーを選択します。
6. 保存するをクリックします。
Navigatorのブランディングが設定されました。
