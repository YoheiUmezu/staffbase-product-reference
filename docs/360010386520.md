# スペースのターゲットグループの設定

**URL**: https://support.staffbase.com/hc/ja/articles/360010386520
**取得日**: 2026-05-02 18:57:26 JST

## 内容

可視性設定を使用して、スペースのターゲットグループを定義できます。可視性設定では、スペース内のコンテンツが全社に公開されるか、特定のユーザーグループに公開されるか、または特定のユーザーに公開されるかを設定できます。スペースはしばしばユーザーグループを使用してコンテンツをターゲットにするため、スペースを考慮してユーザーグループを設定してください。
全従業員スペースは、すべての登録ユーザーが利用できるすべてのコンテンツを含むデフォルトのスペースです。このスペースの可視性設定は変更できないため、コンテンツのターゲティングには使用できません。
スタジオで、設定を変更したいスペースに移動します。
スペースを編集をクリックします。
スペースを編集ダイアログが開きます。
ドロップダウンメニューからオプションを選択して、スペースの可視性を設定します。
保存をクリックします。
