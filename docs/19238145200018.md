# Microsoft TeamsにおけるStaffbaseプラットフォームの概要

**URL**: https://support.staffbase.com/hc/ja/articles/19238145200018
**取得日**: 2026-05-02 18:43:16 JST

## 内容

アクティベーション
あなたの組織が従業員アプリまたはイントラネットとして使用している可能性のあるStaffbaseプラットフォームをMicrosoft Teams環境内で利用できるようにすることができます。これにより、組織はMicrosoft TeamsとStaffbaseプラットフォームをユーザーにとって一体として機能させることができます。組織内の一部のチームが生産性とコラボレーションのためにMicrosoft Teamsを多く使用している場合、これにより彼らはMicrosoft Teamsから直接Staffbaseプラットフォームを使用する柔軟性を得ることができます。この統合により、StaffbaseプラットフォームがMicrosoft Teamsに埋め込まれ、ユーザーは重要な内部および企業コミュニケーションを見逃すことがありません。
ユーザーがMicrosoft Teams内でStaffbaseプラットフォームをどのように表示するか
初めてMicrosoft TeamsからStaffbaseアプリにアクセスする際、ユーザーは自分の資格情報を使用してサインインするように指示されます。サインイン後、ユーザーはアプリにアクセスできます。さらに、ユーザーは自分のプロフィール写真の隣にあるナビゲーションボタンを使用して、現在のページと前のページの間を行き来することができます。
追加の役立つ情報:
