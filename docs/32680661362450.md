# Staffbase Liveでの参加者のモデレーションと管理

**URL**: https://support.staffbase.com/hc/ja/articles/32680661362450
**取得日**: 2026-05-02 18:56:27 JST

## 内容

有効化
モデレーター
Staffbase Liveのモデレーターとして、ライブセッション中に参加者のやり取りを管理します。これには、参加者の行動の監視、チャットでの会話への対応、セッションを円滑に進行するために必要な対応を行うことが含まれます。
モデレーションや管理機能を活用して、ディスカッションが敬意を持って行われ、組織の行動規範に沿うようにしましょう。積極的なモデレーションは、登壇者がコンテンツの提供に集中でき、参加者が建設的に参加できる安全で包括的な環境を作ります。
参加者を制限する
参加者が行動規範に従わない場合やイベントを妨害する場合は、ライブイベントからブロックまたは削除して制限できます。
Broadcast Studioで、チャットに移動します。
チャットの横にあるドロップダウンをクリックします。
参加者をクリックします。
イベントから削除したい参加者の横にある三点リーダーをクリックします。
ドロップダウンが開きます。
次のいずれかのオプションを選択します。
参加者を削除する
参加者がイベントを妨害したり、期待される行動に違反した場合、警告としてライブ配信から削除できます。参加者には通知が届き、ブラウザを更新することでいつでもイベントに再参加できます。
参加者を削除をクリックします。
参加者を削除しました。参加者にはイベントに再参加できる旨のメッセージが表示されます。
参加者をブロックする
参加者が引き続きイベントを妨害したり、重大な行動規範違反をした場合は、ライブ配信からブロックできます。参加者には通知が届き、イベントに再参加できなくなります。
参加者をブロックをクリックします。
参加者をブロックの確認ダイアログが開きます。
ブロックをクリックします。
参加者をブロックしました。参加者にはイベントからブロックされた旨のメッセージが表示されます。
