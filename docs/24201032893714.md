# Staffbase Emailでのユーザーグループの作成

**URL**: https://support.staffbase.com/hc/ja/articles/24201032893714
**取得日**: 2026-05-02 18:56:43 JST

## 内容

管理者
Staffbase EmailをStaffbase IntranetまたはEmployee Appと一緒に使用する場合は、グループの種類を参照してください。作成したいユーザーグループの種類に応じて、対応するドキュメントを参照してください。以下の情報は、Staffbase Emailがスタンドアロン製品として使用される場合にのみ適用されます。
条件に基づいて、またはスタジオから手動でユーザーグループを作成できます。
手動
個々のユーザーを手動で選択することによってユーザーグループを作成できます。この方法はより多くの手間がかかりますが、ユーザーグループを迅速に作成または変更する必要がある場合に完全な柔軟性を提供します。このアプローチは、一度限りまたは臨時のグループ設定に最適です。
グループを手動で作成する
スタジオで、ユーザー > グループに移動します。
グループを作成をクリックします。
グループを作成ダイアログが開きます。
グループの名前と短い説明を入力します。
ユーザーを追加の下で、手動を選択します。
作成をクリックします。
グループが作成されました。
手動で作成したグループにユーザーを追加する
スタジオで、ユーザー > グループに移動します。
作成したグループを見つけて開きます。
メンバーを追加をクリックします。
メンバーを追加ダイアログが開きます。
メンバーのドロップダウンから、追加したいユーザーを選択します。
メンバーを追加をクリックします。
グループにユーザーを追加しました。グループはいつでも編集または削除できます。
グループを削除すると、ユーザーグループに基づいてターゲットオーディエンスに送信予定のメールに影響を与える可能性があります。
条件に基づいて
条件を定義することによってユーザーグループを作成できます。グループは、次のロジックを使用して1つまたは複数の条件で設定できます：
AND: 指定されたすべての条件を満たすユーザーのみがグループの一部となります。
OR: 指定された条件のいずれかを満たすすべてのユーザーがグループの一部となります。
例を表示するには、Staffbase Emailにおけるユーザーグループの概要を参照してください。
スタジオで、ユーザー > グループに移動します。
グループを作成をクリックします。
グループを作成ダイアログが開きます。
グループの名前と短い説明を入力します。
ユーザーを追加の下で、条件に基づいてを選択します。
グループ条件を追加をクリックします。
条件のドロップダウンから、条件に基づくプロファイルフィールドと対応する値を選択します。
オプションで、次のいずれかをクリックし、プロファイルフィールドと対応する値を選択します：
条件を追加: ORロジックを追加します。
グループ条件を追加: ANDロジックを追加します。
作成をクリックします。
グループが作成されました。
グループはいつでも編集または削除できます。
グループを削除すると、ユーザーグループに基づいてターゲットオーディエンスに送信予定のメールに影響を与える可能性があります。
