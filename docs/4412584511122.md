# 従業員メールデザイナードラフトにカレンダー招待を追加する

**URL**: https://support.staffbase.com/hc/ja/articles/4412584511122
**取得日**: 2026-05-02 18:57:07 JST

## 内容

従業員メールを使用すると、メール内でさまざまな方法でカレンダー招待を.icsファイルの形式で提供できます：
カレンダー招待へのクリック可能なリンクを作成する
カレンダー招待を含む.icsファイルをメールに直接添付する
カレンダー招待へのクリック可能なリンクを作成する
イベントのための.icsファイルを作成します。これは、Outlookカレンダーから招待を選択し、iCalendarとして転送のオプションを選択して、生成された添付ファイルをダウンロードすることで行えます。
Macコンピューターでは、カレンダーイベントをデスクトップにドラッグするオプションもあります。
受信者が.icsファイルをダウンロードすると、自動的にそのイベントがOutlookカレンダーに追加されます。
このファイルをローカルイントラネットまたはファイルホスティングサービスにアップロードし、URLをメモします。
イベントのURLができたら、それをEmail Designerのドラフトにリンクとして追加できます。
リンクはメール本文に直接追加することも、ボタンに追加することもできます。
メール内にイベント招待へのリンクを正常に作成しました。
カレンダー招待を直接添付する
.icsファイルへのリンクを提供する代わりに、.icsファイルをメールに直接添付できます：
上記の手順でイベントのための.icsファイルを作成します。
.icsファイルをメールドラフトに直接添付します。
受信者がメールを開くと、.icsファイルが添付ファイルとして表示されます。受信者は添付された.icsファイルをクリックすることで、イベントをダウンロードしてOutlookカレンダーに追加できます。
カレンダー招待をメールに正常に追加しました。
