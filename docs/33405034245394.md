# インタラクティブニュースチャンネルの作成

**URL**: https://support.staffbase.com/hc/ja/articles/33405034245394
**取得日**: 2026-05-02 19:02:48 JST

## 内容

インタラクティブニュースチャンネルを使用すると、ユーザーはニュースを閲覧したり、意見を交換したり、同僚との会議を設定したりできます。ユーザーが最近の生活の更新を投稿したり、チームの瞬間を共有したりするためのソーシャルウォールを作成します。また、特定のオフィスやエリアのためにインタラクティブニュースチャンネルを使用して掲示板を構築することもできます。詳細については、インタラクティブニュースチャンネルの概要をご覧ください。
インタラクティブニュースチャンネルの作成
スタジオで、コンテンツ > ニュースに移動します。
チャンネルを追加をクリックします。
チャンネルを作成 ダイアログが開きます。
名前を入力します。
チャンネルタイプの下で、 更新を選択します。
このチャンネルタイプは、短い記事とメディアを含めるオプションがあるため、インタラクティブニュースチャンネルに最適です。ニュースチャンネルのタイプについて詳しく学ぶ。
保存をクリックします。
チャンネルが作成され、チャンネル設定が開きます。
投稿者のドロップダウンの下で、インタラクティブニュースチャンネルに投稿できるユーザーを選択します。
保存をクリックします。
ニュースチャンネルを作成し、ユーザーが投稿できるようにしました。ニュースチャンネルを公開すると、ユーザーが利用できるようになります。
インタラクティブニュースチャンネルがユーザーにどのように見えるか
ユーザーは今、投稿を作成し、ニュースチャンネルで意味のある対話を行うことができます。

投稿ウィジェットの設定
