# キャンペーンの作成

**URL**: https://support.staffbase.com/hc/ja/articles/360021083120
**取得日**: 2026-05-02 18:47:33 JST

## 内容

アクティベーション
システム全体のエディター
キャンペーンを使用すると、コンテンツ戦略を監視できます。各キャンペーンに対して、コンテンツ作成を導く戦略的目標を設定できます。コンテンツ戦略を実行に移し、キャンペーンを作成しましょう。
スタジオで、プランニング > キャンペーンに移動します。
キャンペーンの作成をクリックします。
キャンペーンの作成ページが開きます。
キャンペーンのタイトルを入力します。
オプションで、キャンペーンの目標を入力します。
オプションで、キャンペーンの開始日と終了日を入力します。
キャンペーンの色を選択します。
オプションで、キャンペーンの整合性の下で、整合性調査を作成するためのトグルを有効にします。
作成をクリックします。
キャンペーンが作成されました。これで、既存のニュース投稿をキャンペーンに割り当てるか、新しい投稿を作成できます。
追加の役立つ情報
既存のニュース投稿をキャンペーンに割り当てる
キャンペーンからニュース投稿を作成する
