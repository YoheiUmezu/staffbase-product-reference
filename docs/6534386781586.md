# Microsoft 365ファイルビューワウィジェットの設定

**URL**: https://support.staffbase.com/hc/ja/articles/6534386781586
**取得日**: 2026-05-02 18:58:12 JST

## 内容

アクティベーション
システム全体のエディタ
Microsoft 365ファイルビューワウィジェットを使用して、Staffbaseプラットフォームのページやニュース投稿でSharePointから直接ファイルを表示できるようにします。任意のファイルタイプをページや投稿に埋め込むことができ、ウィジェットはファイルを中心に表示します。たとえば、会社のロードマッププレゼンテーションをウィジェットに追加し、従業員がページ上でプレゼンテーションを簡単に見つけて各スライドを確認できるようにします。また、CEOからのビデオメッセージを追加し、従業員にビデオプレーヤーで再生させることもできます。
前提条件
あなたは Microsoft 365接続をStaffbaseに追加しました。
あなたは アプリまたはイントラネットからMicrosoft 365にサインインしました。
あなたは Microsoft 365ファイルビューワウィジェットをアクティブにしました。
ファイルビューワウィジェットの追加
ファイルビューワウィジェットを追加する方法は2つあります。
ウィジェットアイコンを追加
スタジオで、ウィジェットを追加したいページまたは投稿に移動します。
コンテンツエディタで、ウィジェットを追加したい場所にカーソルを置きます。
コンテンツエディタツールバーで、ウィジェットを追加のプラスアイコンをクリックします。
ウィジェットを追加ダイアログが開きます。
ファイルビューワをクリックします。
Microsoft 365オプション
スタジオで、ウィジェットを追加したいページまたは投稿に移動します。
コンテンツエディタで、ウィジェットを追加したい場所にカーソルを置きます。
コンテンツエディタツールバーで、メディアを追加のドロップダウンをクリックします。
Microsoft 365をクリックします。
Microsoft 365オプションが見つからない場合は、カスタマーサクセスマネージャーまたはStaffbaseサポートにお問い合わせください。
Microsoft 365ファイルビューワウィジェットを追加しました。次に、設定を行う必要があります。
ファイルビューワウィジェットの設定
Microsoft 365ファイルビューワウィジェットを追加します。
ファイルビューワダイアログで、オプションでタイトルを提供します。
タイトルはウィジェットの上に表示され、ユーザーに見えるようになります。
ウィジェットを表示するデバイスタイプを選択します。
アプリ内の可視性で、ウィジェットを利用可能にしたいユーザーを選択します。
ウィジェットに表示したいファイルを検索して選択します。
ファイルドロップダウンは、StaffbaseプラットフォームからMicrosoftアカウントにサインインしている場合のみファイルを表示します。サインインしていない場合は、サインインを促されます。表示されるファイルは、あなたの役割と権限に基づいてMicrosoftでアクセスできるファイルであり、Microsoftによって決定された関連性の順序で表示されます。
OKをクリックします。
ファイルを含むウィジェットがページまたは投稿に追加されます。
ページまたは投稿を保存します。
ウィジェットを設定し、Microsoft 365ファイルビューワウィジェットをページに追加しました。
ユーザーにとってのMicrosoft 365ファイルビューワウィジェットの見た目
ウィジェットにアクセスする際、ユーザーは Microsoft 365アカウントにサインインするように促されます、まだサインインしていない場合は、従業員アプリまたはイントラネットで。
