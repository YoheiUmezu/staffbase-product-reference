# 法的文書の確認の設定

**URL**: https://support.staffbase.com/hc/ja/articles/115002233152
**取得日**: 2026-05-02 18:52:46 JST

## 内容

登録プロセス中に、新しいユーザーに法的文書の確認を求めることができます。
登録およびサインインの方法に応じて、確認ダイアログは一度だけ表示されるか、毎回サインイン時に表示されます：
ユーザーがシングルサインオン（SSO）を介してアプリにアクセスする場合、アプリにサインインするたびに法的文書の確認を行う必要があります。
メールとパスワードでのサインインなど、他のすべてのアクセス方法では、ユーザーが初めてサインインしたときにのみ確認が表示されます。
例：
法的文書の確認を伴うサインアッププロセス。
前提条件
法的文書を作成しました。
確認の有効化
スタジオで、設定 > 法的文書。
確認設定をクリックします。
有効に切り替えます。
ユーザーに法的文書の確認を求めるメッセージを提供します。
確認ダイアログ内で法的文書へのリンクを提供できます。これを行う方法については、関連するヘルプトピックを参照してください。
保存をクリックします。
