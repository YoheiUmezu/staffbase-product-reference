# コンテンツデザイナー内の表示設定の使い方

**URL**: https://support.staffbase.com/hc/ja/articles/32702253432594
**取得日**: 2026-05-02 19:00:05 JST

## 内容

編集者
Content Designer の表示設定では、ページ上のコンテンツがいつ、どこで、誰に表示されるかを管理できます。コンテンツを特定の視聴者のみに表示したり、他のユーザーから非表示にしたりできます。これにより、ページを複製することなく、適切なデバイスに適切なコンテンツを表示できます。
検索、ナビゲーター、AIの回答は、ブロック、コンテナ、セクションレベルの表示制限を考慮しません。コンテンツの安全性を保つために、Staffbase では適切なページレベルの表示設定を持つ別々のページを作成することを推奨します。
表示ルールは、次の条件に基づいてセクション、コンテナ、個々のブロックに適用できます：
デバイスタイプ： デスクトップ、タブレット、モバイルから選択
ユーザーグループ： コンテンツを選択されたグループのみに表示、または特定の視聴者から非表示にする
これらのオプションは組み合わせて使用できます。たとえば、デスクトップで特定のユーザーグループのみにコンテンツを表示したり、特定のグループからコンテンツを非表示にしつつ、他の全員には表示したままにすることができます。
Studio で、コンテンツ > ページ > 編集または作成したいページに移動します。
任意で、アプリ内で編集アイコンをクリックしてページを更新します。
設定したいセクション、コンテナ、またはブロックを選択します。
コンテキストツールバーで鉛筆アイコンをクリックして編集します。
表示 タブで、次のオプションを設定します：
表示： コンテンツが表示されるデバイス。
公開範囲： コンテンツを閲覧できるユーザー。
非表示： コンテンツを閲覧できないユーザー。
要素の表示設定が完了しました。
