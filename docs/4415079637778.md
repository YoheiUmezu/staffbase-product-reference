# 従業員メールをGmailのInboxで使用できますか？

**URL**: https://support.staffbase.com/hc/ja/articles/4415079637778
**取得日**: 2026-05-02 18:55:52 JST

## 内容

従業員メールは、Google Chromeのブラウザ拡張機能を通じてGmailとシームレスに統合され、メールの追跡、スケジュール設定、スヌーズを迅速に行うことができます。
GmailのInbox
現在、従業員メールの拡張機能は、ウェブベースのGmailクライアントのみと互換性があります。ウェブ上のGmailとGmailのInboxとの間にはさまざまな違いがあるため、この拡張機能はGmailのInboxにインストールまたは使用することはできません。
