# 確認ダイアログから法的文書へのリンク

**URL**: https://support.staffbase.com/hc/ja/articles/115002711912
**取得日**: 2026-05-02 18:54:41 JST

## 内容

法的文書の確認を有効にすると、確認ダイアログから直接法的文書にリンクできます。これにより、ユーザーは新しいおよび更新された法的文書を検索する必要がなくなり、確認ダイアログから直接簡単にアクセスできるようになります。
新しいおよび更新された法的文書だけでなく、ユーザーごとに関連する法的文書も表示される法的文書のターゲティングのベータ版が利用可能です。
前提条件
ショートネームを持つ法的文書を作成していること
法的文書の確認を有効にしていること。
法的文書へのリンク
スタジオで、設定 > 法的文書に移動します。
確認設定をクリックします。
以前に作成した確認ダイアログのメッセージが開きます。
メッセージに法的文書のショートネームを波括弧で入力します：
{shortname}、例えば {privacy-policy}
ショートネームは各法的文書の設定で構成されています。すべての利用可能なショートネームとそれに対応する法的文書は、メッセージエディタの下に表示されます。さらに法的文書にリンクする必要がある場合は、最初にそれらを法的文書として設定してください。
確認のプレビューリンクをクリックして、確認ダイアログのプレビューを表示します。
リンクをクリックすると、プレビューが別のダイアログで開きます。
受け入れるをクリックしてプレビューを閉じます。
保存をクリックします。
