# 埋め込みページウィジェットの設定

**URL**: https://support.staffbase.com/hc/ja/articles/25488187262738
**取得日**: 2026-05-02 18:58:44 JST

## 内容

エディター
埋め込みページウィジェットを使用すると、外部のウェブページやアプリの別の部分からのコンテンツをページに直接埋め込むことができます。これにより、関連するトピックや重要な外部コンテンツを1つの中央ページに明確にまとめることができます。
スタジオで、埋め込みページウィジェットを追加したいページに移動し、エディター内で配置したい場所にマウスを置きます。
埋め込みページウィジェットを追加します。
埋め込みページウィジェットを設定するためのダイアログが開きます。
オプションで、タイトルを提供します。
タイトルはウィジェットの上に表示され、ユーザーに見えるようになります。
ウィジェットが表示されるデバイスタイプを、対応するアイコンをクリックして選択します。
ウィジェットが表示されるユーザーをアプリ内の表示設定で選択します。
ターゲットURLの下に、ページに埋め込みたいウェブサイトのURLを貼り付けます。
事前に、希望するページが埋め込みを許可しているか確認してください。
オプションで、デスクトップおよびモバイルデバイス用にウィジェットの高さをカスタマイズします。
ズームやクリックなど、ウィジェットとのユーザーインタラクションを有効にするかどうかをインタラクションを有効にするのトグルを調整して定義します。
ウィジェットのスクロールを有効にするかどうかをスクロールを有効にするのトグルを調整して定義します。
OKをクリックします。ウィジェットがページに追加されます。
変更を保存します。
