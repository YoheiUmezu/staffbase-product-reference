# Microsoft 365 Viva Communitiesウィジェットの設定

**URL**: https://support.staffbase.com/hc/ja/articles/16226543431058
**取得日**: 2026-05-02 18:46:48 JST

## 内容

従業員のコミュニティへの参加を促進し、Viva Engageコミュニティカードをページやニュース投稿に追加することで、従業員のエンゲージメントを高めましょう。従業員は、ページやニュース投稿のコミュニティカードをクリックすることで、Viva Engageアプリ内のコミュニティスペースに簡単にアクセスできます。Staffbaseでページやニュース投稿にViva Engageコミュニティを追加するには、Microsoft 365 Viva Communitiesウィジェットを使用します。
前提条件
Microsoft 365 ConnectionsをStaffbaseに追加している。
アプリまたはイントラネットからMicrosoft 365にサインインしている。
Microsoft 365 Viva Communitiesウィジェットを有効化している。
Microsoft 365 Viva Communitiesウィジェットを追加します。
ウィジェットの設定ダイアログが開きます。
オプションで、タイトルを提供します。
タイトルはウィジェットの上に表示され、ユーザーに見えるようになります。
ウィジェットが表示されるデバイスタイプを、対応するアイコンをクリックして選択します。
アプリ内の表示の下で、ウィジェットを表示したいユーザーを選択します。
コミュニティのドロップダウンから、カードとして表示したいコミュニティを選択します。
新しく作成されたViva EngageコミュニティがViva Communitiesウィジェットに表示されるまでに数分の遅延がある場合があります。
オプションで、+アイコンをクリックして別のコミュニティカードを追加します。
+アイコンをクリックすることで、好きなだけコミュニティを追加できます。
OKをクリックします。
Viva Communitiesウィジェットをページまたは投稿に追加しました。ページまたは投稿が公開されると、ウィジェットはユーザーに利用可能になります。
Microsoft 365 Viva Communitiesウィジェットの編集
Microsoft 365 Viva Communitiesウィジェットを編集して、一般設定を変更したり、追加したコミュニティの順序を更新したり、コミュニティの1つを削除したりできます。
スタジオで、Microsoft 365 Viva Communitiesウィジェットを編集したいページまたは投稿に移動します。
コンテンツエディタで、Microsoft 365 Viva Communitiesウィジェットをクリックします。
順序を変更
メニューアイコンをクリックして、コミュニティをドラッグ＆ドロップして順序を再配置します。
クリックしてOK。
コミュニティの表示順序を編集しました。
コミュニティを削除
Xアイコンをクリックしてコミュニティを削除します。
クリックしてOK。
Viva Communitiesウィジェットからコミュニティを削除しました。
Microsoft 365 Viva Communitiesウィジェットがユーザーにどのように表示されるか
ユーザーがウィジェットのあるページまたは投稿に移動すると、Microsoft 365アカウントにサインインするように促されます（まだサインインしていない場合）。コミュニティカードをクリックすると、Viva Engageアプリ内のコミュニティに移動します。
