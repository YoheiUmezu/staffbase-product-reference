# ユーザー名とパスワードを使用して手動でユーザーを追加する

**URL**: https://support.staffbase.com/hc/ja/articles/360010058340
**取得日**: 2026-05-02 18:47:52 JST

## 内容

各ユーザーのユーザー名とパスワードを作成して、アプリに手動でユーザーを追加できます。この方法は、ユーザーのメールアドレスや連絡先情報がない場合に推奨されます。その場合、ユーザーの資格情報は対面、郵送、またはお好みの別の方法で配布できます。
スタジオで、ユーザー設定に移動します。
ユーザーを追加をクリックします。
新しいユーザーを招待するためのダイアログが開きます。
ユーザー名招待をクリックします。
ユーザーの名前とユーザー名を入力します。
ドロップダウンメニューから、アプリ内のユーザーのアクセス権を設定します。
オプションで、一時的なパスワードを提供します。このフィールドを空のままにすると、一時的なパスワードがランダムに生成されます。
ユーザーを追加をクリックします。
新しいユーザーが正常に作成されました。
自動的に 保留中のユーザータブにリダイレクトされます。
追加の役立つ情報
ダイアログの 詳細設定の下で、ユーザーの識別子、ユーザーがパスワードをリセットするための回復コード、またはユーザーグループに事前にユーザーを追加するオプションもあります。
