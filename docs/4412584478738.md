# メールデザインにおける高度なブロックコントロールの使用

**URL**: https://support.staffbase.com/hc/ja/articles/4412584478738
**取得日**: 2026-05-02 18:49:55 JST

## 内容

Employee Emailでデザインを編集している間、ブロックコントロールを使用すると、ブロック全体を選択し、編集し、他のブロックの中で再配置することができます。この記事では、ブロック自体にフォーマットを適用するためのブロック設定メニューの概要を提供します。

これらのシンプルなブロックコントロールを使用すると、メールのデザインが簡単に行えます。
追加
メールデザインに追加できるブロックには2種類あります：
ブロックレイアウトは、選択した要素を追加できる空のブロックデザインです。
クイックブロックは、メールを迅速にデザインできる事前設計されたブロックです。
メールにブロックを追加するには：
ブロックタイプをクリックしてドラッグし、メールデザインに追加します。
ブロックレイアウトから追加する場合は、レイアウトに1つ以上の要素をドラッグ＆ドロップする必要があります。
選択
ブロック全体を制御する前に、まずブロック自体が選択されていることを確認する必要があります。ブロックをクリックして選択します。
ブロック自体を常にクリックする最も簡単な方法は、ブロックの最右端または最左端の空のブロックスペースをクリックすることです。デザインに4列の要素があっても、このスペースは常に存在し、ブロック自体をクリックして選択できます。
ブロックを選択したら、ブロックの右上隅に表示されるブロックレイアウトメニューにアクセスできます。
デザイン内のすべてのブロックを選択するには、ブロック設定の下にある画面右上隅のすべて選択チェックボックスをクリックします。
ここから、すべてのブロックをロックしたり、すべての色を調整したり、すべての間隔を調整したり、すべてを削除したりできます。
削除
削除したいブロックのアイコンをクリックすると、そのブロックが削除されます。削除されたブロックの下にあるブロックは、その位置に移動します。
入れ替え
要素の入れ替え機能と同様に、この機能は、上または下のブロックとブロックを入れ替えます。
またはアイコンをクリックすると、選択したブロックが上または下のブロックと入れ替わります。
ドラッグ＆ドロップ
ブロックは、異なる場所にドラッグ＆ドロップすることで再配置できます。これは、ブロックを選択した後にクリックし、マウスボタンを押し続けて、メールデザイン内の希望の位置にドラッグすることで行えます。
クローン
要素のクローン機能と同様に、この機能はブロックをクローンし、選択したブロックの下に挿入します。クローンされたブロックは、クローン元のブロックと同じ要素をすべて含みます。
ブロックをクローンするには、アイコンをクリックすると、選択したブロックの下に表示されます。
ブロックデザインと設定の編集
ブロックが選択されると、ブロック設定メニューがメールデザイナーの右側に表示されます。このメニューでは、ブロック内および全体のデザイン内のいくつかの設定を編集できます。
ブロックの色
ブロック設定メニューには、ブロックの色を変更するための2つのフィールドがあります。
外側の色フィールドでは、要素が存在しないブロックの領域の色を変更できます。
内側の色フィールドでは、要素が存在するブロックの領域の色を変更できます。
両方のフィールドで、フィールド内の色を選択するか、16進数コードを入力するか、RGBカラー値を入力することで色を変更できます。
ブロックの間隔
ブロックの間隔設定では、選択したブロックとその上または下のブロックとの間隔を調整できます。
ブロックの間隔設定には、調整可能な2つのフィールドがあります。
上のフィールドは、選択したブロックとその上のブロックとの間隔を制御します。
下のフィールドは、選択したブロックとその下のブロックとの間隔を制御します。
各フィールドで、またはアイコンをクリックすると、ブロック間の間隔を増減できます。
間隔を均等に調整するには、2つのブロックの間隔フィールドの間にあるロックアイコンをクリックします。次に、いずれかのフィールドで間隔を増減すると、反対のフィールドでも間隔調整が反映されます。
ブロックのロック
ブロック設定メニューの上部にロックアイコンが表示されます。この機能を使用すると、ブロックをロックして、ロックが解除されるまで編集できなくなります。
ロックは、複数のコラボレーターとデザインを編集する際のベストプラクティスであり、他の人によってすでに編集されたブロックを誤って変更することを防ぎます。
