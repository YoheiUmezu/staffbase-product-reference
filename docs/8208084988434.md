# エディトリアルカレンダーでのブロッカーの使用

**URL**: https://support.staffbase.com/hc/ja/articles/8208084988434
**取得日**: 2026-05-02 18:53:49 JST

## 内容

アクティベーション
システム全体のエディター
エディトリアルカレンダー内にブロッカーを追加することで、重要な会社の計画やコミュニケーションを含めることができます。ブロッカーはすべてのエディターに表示されます。また、期間中の出版の凍結を設定することもできます。この出版の凍結は、エディターに対して投稿の再スケジュールを検討することを提案します。エディトリアルカレンダーとスタジオニュースエディターの両方で行われます。エディターは、最大のリーチを得るために、その時間外に投稿を計画することができ、組織のコンテンツ計画を次のレベルに引き上げるのに役立ちます。
ブロッカーの追加
スタジオで、プランニング > エディトリアルカレンダーに移動します。
作成をクリックし、ドロップダウンメニューからブロッカーを選択します。
タイトル、期間、およびオプションで説明を入力します。
オプションで、ブロッカーとともに出版の凍結を設定し、異なる時間に投稿を再スケジュールすることを提案します。
作成をクリックします。
ブロッカーが作成されました。これで、エディトリアルカレンダー内のすべてのエディターに表示されます。
