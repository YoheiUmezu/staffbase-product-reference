# コンテンツレビューにおけるレビューページアクセスの管理

**URL**: https://support.staffbase.com/hc/ja/articles/21558340012818
**取得日**: 2026-05-02 19:03:20 JST

## 内容

アクティベーション
エディター
従業員を引き込む魅力的なニュースを作成するには、時間、労力、チームワークが必要です。すべてのニュースチャンネルタイプで利用可能なコンテンツレビュー機能を使用して、投稿の詳細を簡単に確認したり、フィードバックを受け取ったりできます。 詳細を学ぶ。
ニュース投稿に対するフィードバックを受け取るには、まずユーザーをレビューページに招待する必要があります。ユーザーを招待するには、メール招待を介して個別のニュース投稿を共有します。共有ボタンを使用して、招待を受け取る個々のプラットフォームユーザーを選択します。
新たに追加されたレビュアーは、メール内のリンクを開いて、アプリ内で投稿とレビュータブを直接確認できます。招待された後、彼らはメール内のリンクを開いてレビューページを見ることができます。レビューページでは、レビュータブを使用してコメントを通じてフィードバックや承認を行うことができます。
すべてのユーザーは、ユーザーの役割に関係なく、レビューページを表示および使用するために招待できます。
レビューページへのアクセスを許可する
前提条件
ユーザーは、Staffbaseアカウントに関連付けられたメールアドレスを持っている必要があります。フィードバックを残すための招待は、メールのみで送信されます。
スタジオで、コンテンツ > ニュースに移動します。
レビュアーを追加したい投稿を開きます。
共有をクリックします。
共有ダイアログが開きます。
名前を入力し、レビュアーとして追加したいユーザーを選択します。
招待を送信をクリックします。
ユーザーをレビューページに招待しました。彼らはニュース投稿をレビューするように求めるメールと、ユーザー表示の投稿へのリンクを受け取ります。
レビューページへのアクセスを削除する
スタジオで、コンテンツ> ニュースに移動します。
すでにレビュアーを追加した投稿を開きます。
共有をクリックします。
共有ダイアログが開きます。
削除したいユーザーの隣にあるXをクリックします。
変更は自動的に保存されます。ユーザーはもはやレビューページにアクセスできず、フィードバックを残すことができません。
