# Confluence検索統合の概要

**URL**: https://support.staffbase.com/hc/ja/articles/20768171201938
**取得日**: 2026-05-02 18:43:05 JST

## 内容

ベータ
アクティベーション
ConfluenceとStaffbaseの統合により、プラットフォーム間でのコンテンツ検索がこれまで以上に簡単になります。このFront Door Intranet用の統合を使用すると、Staffbaseプラットフォームに存在するコンテンツだけでなく、Confluenceのコンテンツにもアクセスできます。これにより、別のプラットフォームで再度コンテンツを検索する手間や、コンテンツ作成者がコンテンツを重複して作成することを避けることができます。Confluenceからの検索結果は、Staffbaseプラットフォームの別の検索タブに表示されます。
ユーザーが検索を利用できるようにするには、Atlassian接続をStaffbaseに追加する必要があります。詳細はこちら。
ユーザーにとってのConfluence統合の見え方
ユーザーがAtlassianアカウントにサインインすると、Confluence検索タブでコンテンツを検索できます。詳細はこちら。検索タブの結果をクリックすると、ユーザーは直接Confluenceの目的のコンテンツに移動します。
