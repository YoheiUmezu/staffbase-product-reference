# クイズカレンダーの設定

**URL**: https://support.staffbase.com/hc/ja/articles/4403494619282
**取得日**: 2026-05-02 18:50:03 JST

## 内容

クイズカレンダーは、重要な会社のイベント、マイルストーン、または休日に向けてユーザーを楽しませるための楽しい方法です。カレンダーの各日ごとに、ユーザーがアンロックできるカスタムクイズ質問、画像、またはテキストを設定できます。これにより、ユーザーは毎日戻ってくる理由ができ、プラットフォームを楽しむことができます。
クイズカレンダーには、カレンダーを公開したい時期に応じて2つのオプションがあります。
クリスマスカレンダー は、事前に設定された日付範囲を持っています。12月1日に始まり、12月24日に終了します。
カスタムカレンダー は、任意の時期に開始できるカスタマイズ可能な期間を持っています。このクイズタイプは最小2日、最大40日まで設定できます。
前提条件
クイズカレンダー プラグインをインストールしています。
最初のクイズカレンダーの設定
スタジオで、コンテンツ  > クイズカレンダー に移動します。
インストール時に追加されたクイズカレンダーが表示されます。
追加したいカレンダーのタイプを選択します。
カスタムカレンダーの場合は、開始日 と 終了日 を入力します。
追加 をクリックします。
最初のクイズカレンダーを追加しました。これで、コンテンツをカスタマイズするための質問を追加したり、設定を構成したりできます。
新しいクイズカレンダーの追加
スタジオで、コンテンツ  > クイズカレンダー に移動します。
クイズカレンダーを追加 をクリックします。
新しいクイズカレンダーを追加するためのダイアログが開きます。
クイズカレンダーのタイトルを入力します。
ドロップダウンメニューからクイズカレンダーの可視性を選択します。
クイズカレンダーを追加 をクリックします。
新しいクイズカレンダーが開きます。
追加したいカレンダーのタイプを選択します。
カスタムカレンダーの場合は、開始日 と 終了日 を入力します。
追加 をクリックします。
新しいクイズカレンダーを追加しました。これで、コンテンツをカスタマイズするための質問を追加したり、設定を構成したりできます。
クイズ質問の追加
スタジオで、コンテンツ  > クイズカレンダー に移動し、コンテンツを追加したいクイズカレンダーに移動します。
コンテンツを追加したい日をクリックします。
任意で、以下のコンテンツタイプを追加します：
タイトル
これは、カレンダーの日のコンテンツの上に太字で表示されます。
画像
例えば、記念または祝賀の画像を追加します。
テキスト
イニシアチブに関する面白い事実、異なる休日の伝統、またはイベントに対する期待感を高めるためのテキストを追加します。
質問を追加 をクリックします。
新しい質問を作成するためのフィールドが表示されます。
クイズの質問を入力します。
回答選択肢 の下に、質問に対する最大4つの可能な回答を入力します。
正しい回答 の下に、クイズの質問に対する正しい回答を選択します。
クイズカレンダーの追加日にもコンテンツを追加するために繰り返します。
保存 をクリックします。
クイズカレンダーに質問を追加しました。いつでもさらに追加したり、クイズの質問をプレビューしたりできます。
クイズカレンダーコンテンツのプレビュー
クイズの質問を簡単にテストし、クイズカレンダーを公開する前にユーザーがコンテンツをどのように見るかをプレビューできます。
クイズの編集権限を持つ人は、公開後も参加できません。ただし、アプリ内やスタジオのプレビューからいつでもクイズの質問に回答できますが、ランキングには影響しません。
スタジオで、コンテンツ  > クイズカレンダー に移動し、プレビューしたいクイズカレンダーに移動します。
プレビュー をクリックします。
プレビューダイアログが開きます。
どのデバイスタイプでクイズカレンダーのコンテンツをプレビューするかを選択します。
クイズカレンダーの日をクリックして、そのコンテンツをプレビューします。
クイズカレンダーの公開
ユーザーがクイズカレンダーに参加するためには、まずそれを公開してアクセス可能にする必要があります。
スタジオで、コンテンツ  > クイズカレンダー に移動します。
公開したいクイズカレンダーを選択します。
クイズカレンダーを公開 をクリックします。
ユーザーにとってのクイズカレンダーの見た目
これは、カスタム画像の背景を持つクイズカレンダーがユーザーにどのように見えるかの例です。
