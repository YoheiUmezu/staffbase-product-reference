# ニュースチャンネルの投稿作成者の表示設定

**URL**: https://support.staffbase.com/hc/ja/articles/33742679334034
**取得日**: 2026-05-02 19:00:37 JST

## 内容

編集責任者
ニュースは、スタッフベースアプリとイントラネットで視聴者ごとに動的なコンテンツを公開できます。これは、ニュースページ、チャンネル、投稿を作成するの3層構造で構成されています。
記事およびメディアのニュースチャンネルのチャンネル設定で、ニュース投稿の末尾に作成者の名前を表示するかどうかを決めることができます。
Studioで、設定を変更したいチャンネルに移動します。
歯車アイコンをクリックして設定を開きます。
設定ページが開きます。
高度な機能の横で、作成者を表示に移動します。
トグルを調整します。
保存するをクリックします。
投稿を作成する作成者の表示設定が完了しました。設定はいつでも変更できます。
