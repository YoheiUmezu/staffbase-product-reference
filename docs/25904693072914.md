# ServiceNowでKnowledge APIを有効にする

**URL**: https://support.staffbase.com/hc/ja/articles/25904693072914
**取得日**: 2026-05-02 18:51:05 JST

## 内容

有効化
管理者
ServiceNow ナレッジ検索 API はデフォルトでは有効になっていません。連携を始める前に ServiceNow の設定を構成する必要があります。次の作業が必要です：
ServiceNow 接続を Staffbase に追加する
ServiceNow で Knowledge API をインストールする
Studio で ServiceNow を有効にしてユーザーが利用できるようにする
前提条件
ServiceNow 接続を Staffbase に追加しています。
Staffbase で ServiceNow インスタンスへのエンドポイント設定後にクライアント ID またはインスタンス名を編集した場合は、ServiceNow のパスワードを更新する必要があります。
ServiceNow で、All > System Definition > Plugins に移動します。
ServiceNow ストアが開きます。
Knowledge API を検索する。
検索結果ページが開きます。
Knowledge API の下で、Install をクリックします。
Knowledge API のインストールが完了し、ServiceNow プラットフォームで有効になりました。次に、Studio で ServiceNow を有効にする手順に進むことができます。
