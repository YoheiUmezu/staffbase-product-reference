# 分析ダッシュボードを使用したニュースメトリクスの分析

**URL**: https://support.staffbase.com/hc/ja/articles/12414610410514
**取得日**: 2026-05-02 18:53:41 JST

## 内容

エディター
アナリティクスダッシュボードは、ユーザーがStaffbaseプラットフォームをどのように、いつ、どこでエンゲージメントしているかについて、より深いインサイトを提供します。アナリティクスダッシュボードは、さまざまなカテゴリに分かれており、詳細な指標を確認できます。
この記事では、ニュースタブの詳細、グラフや分析データについて説明します。アナリティクスダッシュボード全体やユーザー権限については、アナリティクスダッシュボードの概要をご覧ください。
フィルター
ダッシュボードは自動的にニュースタブが開き、デフォルトで今日の指標が表示されます。日付範囲を変更したり、さらにフィルターを使って結果を絞り込むことができます。
チャンネルフィルターはニュースタブ特有のものです。組織のニュースチャンネルや、そのチャンネルへのアクセス権に基づいてフィルターできます。フィルターを選択すると、ニュースタブ内のすべてのグラフに適用されます。
フィルターの詳細はこちら。
主な指標
タブの上部には、クイック概要用の8つの指標があります。ラベルにカーソルを合わせると、前月と比較した指標の情報が表示されます。指標の下のラベルは、前月と比べて指標が増加したか減少したかを示します。たとえば、先月の合計閲覧数が100で今月が50の場合、50%減少と表示され、下降傾向を示します。

ニュースエンゲージメントの推移の折れ線グラフでは、さまざまなニュース指標の詳細な統計が表示されます。各指標にはグラフ内で対応する色が割り当てられています。デフォルトではすべてのニュース指標が表示され、オプションの重なりをグラフ全体で確認できます。1つ以上の指標を非表示にしてグラフをカスタマイズすることも可能です。これにより、たとえば投稿記事といいねなど、特定の項目同士を比較できます。
グラフ上の日付にカーソルを合わせると、選択した指標に応じてその日の具体的なデータが表示されます。さらにフィルターを適用したり、指標を外すことで結果を絞り込むことができます。
グラフ右側の値も、選択した指標やフィルターによって自動的に最大値が調整されます。たとえば、訪問数の指標が高い場合にそれを外すと、次に高い値に自動的に調整されます。
グラフ下部の値も、選択した期間によって自動的に調整されます。1年など長期間の場合は月単位、1週間など短期間の場合は日単位で表示されます。

その指標はグレー表示となり、グラフに表示されなくなります。
ニュースチャンネルのエンゲージメント
ニュースチャンネルのエンゲージメントテーブルでは、訪問数、いいね、コメント、共有などのエンゲージメント指標に基づいて、プラットフォーム内で最もパフォーマンスの高いニュースチャンネルが表示されます。これにより、管理者はどのチャンネルがどのように注目されているかを確認できます。チャンネルは横棒グラフで表示されます。

アナリティクス用語の詳細は、アナリティクス用語集をご覧ください。
デフォルトでは、リストは訪問数で並べ替えられています。ヘッダー行の他のタイトルをクリックすると、タイトルのアルファベット順やコメント数など、任意の列で並べ替えることができます。このリストはすべての投稿を一覧で確認でき、直接比較するのに便利です。
訪問数が訪問者数より多い場合、同じユーザーが記事を複数回読んでいることを意味します。また、いいねの数が訪問数より多い場合もあります。これは、ユーザーが投稿を開かずにニュースフィード内からいいねできるためです。
