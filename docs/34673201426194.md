# AI ページ生成の概要

**URL**: https://support.staffbase.com/hc/ja/articles/34673201426194
**取得日**: 2026-05-02 18:48:21 JST

## 内容

AI
ページは、ガイドライン、リソース、または長期的に関連性のあるコンテンツなどのインフォメーションを共有するのに最適です。スタジオでは、クラシックエディターまたは最新のContent Designerを使用してページを作成することができます。Content Designerは、柔軟なレイアウトと使いやすさを備えた最新のWYSIWYG編集体験を提供します。
AIページ生成はContent Designerで利用可です。プロンプトに基づいて構造化されたページを素早く作成するのに役立ちます。手動作業を減らしながら、最終的なコンテンツを完全にコントロールできます。
ページを生成するには、コンテンツの目的を説明してください。任意で、コンテキスト用にドキュメントをアップロードしたり、画像を挿入するためにメディアを追加したり、既存の最新ページをテンプレートとして使用して出力をガイドすることもできます。この入力に基づいて、AIがあらかじめ定義されたセクション、ブロック、および提案されたコンテンツを含む下書きページを生成します。
生成されたページはスタートポイントとして使用し、必要に応じて調整できます。これにより、コンテンツを効率的に作成・構成・改善し、あなたのorganizationの標準に合わせることができます。
AIページ生成のメリット
コンテンツ作成の高速化
数秒でページの下書きを生成し、新規コンテンツ作成にかかる時間を短縮します。
コンテンツ作成のハードルを下げる
専門知識がなくても構造化されたページを作成でき、より多くのチームがコンテンツ作成に参加しやすくなります。
フルエディトリアルコントロール
最終的なコンテンツを完全にコントロールできます。AIが生成した下書きをレビュー・編集・調整し、Accuracyやorganizationの標準に合わせることができます。
追加のヘルプ情報
AIページ生成を使用してページを作成する
AIページ生成をユーザーまたはユーザーグループに展開する
