# Staffbase Liveで最初のイベントを作成する

**URL**: https://support.staffbase.com/hc/ja/articles/33620284003602
**取得日**: 2026-05-02 18:45:33 JST

## 内容

アクティベーション
編集責任者
Staffbase Live を使って、セッションをリアルタイムで配信するイベントを作成できます。イベントの設定時に、事前にスケジュールを設定したり、モデレーターをスピーカーやホストとして割り当てたり、どのユーザーやユーザーグループが視聴者として参加できるかを指定できます。
Staffbase Live は最大 100,000 人の視聴者に対応しています。20,000 人を超える視聴者が参加するイベントを計画している場合は、事前に Staffbase サポートまたはカスタマーサクセスマネージャーにご連絡ください。
Events Center プラグインをインストールする
Staffbase Live を使って、セッションをリアルタイムで配信するイベントを作成できます。Staffbase Live 用のイベントを作成するには、Events Center プラグインをインストールする必要があります。
Studio で コンテンツ に移動します。
コンテンツ メニューで、プラグインを追加する をクリックします。
Events Center に移動し、イベントを追加する をクリックします。イベント情報 ページが開きます。
スペース ドロップダウンで、プラグインを追加したいスペースを選択します。
タイトル フィールドに、ライブ配信したいイベントのタイトルを入力します。
短い説明 フィールドに、イベントの簡単な説明を入力します。
イベントをスケジュールするために、次の情報を選択します。
開始日
開始時刻
終了日
終了時刻
タイムゾーン
次 をクリックします。
イベント情報が追加されました。次に設定を構成できます。
イベント設定を構成する
設定 で、イベントタイプ はデフォルトで ライブ配信 に設定されています。
視聴者を選択する で、ユーザーグループ または ユーザー ドロップダウンを使って、イベントに参加できる対象を選択します。
リマインダーを設定する で、メールで を選択して、ライブ配信の 1 時間前にメールリマインダーを送信します。
モデレーターを選択する で、イベントで話すまたはホストする必要がある管理者を選択します。モデレーターは、Staffbase Live に関連するすべての作業を行うために管理者ロールが必要です。
作成する をクリックします。
イベントが作成されました。モデレーターと視聴者には、イベントの閲覧またはモデレート用のロール別リンクとカレンダー招待が記載されたメールが届きます。たとえば、Google カレンダーのユーザーには、イベントをカレンダーに追加するよう促すメールが届きます。
イベント作成者にはカレンダー招待は送信されません。
