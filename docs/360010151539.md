# ユーザー資格情報の印刷と配布

**URL**: https://support.staffbase.com/hc/ja/articles/360010151539
**取得日**: 2026-05-02 18:52:22 JST

## 内容

新しいユーザーをユーザー名またはアクセスコードで手動で設定した後、これらの資格情報をユーザーに配布する必要があります。Staffbaseは、配布を支援するためにアプリ内に便利な印刷機能を提供しています。印刷物は事前にフォーマットされており、ユーザーの初回ログイン情報とアプリをダウンロードするためのリンクおよびQRコードが含まれています。
スタジオで、ユーザー設定 > 保留中のユーザーに移動します。
エクスポート / 印刷をクリックします。
ドロップダウンメニューから招待状の印刷をクリックします。
ユーザー資格情報を印刷します。
資格情報を安全な方法でユーザーに配布します。たとえば、対面または給与明細と一緒に配布します。
