# アクセシブルな画像のための代替テキストの追加

**URL**: https://support.staffbase.com/hc/ja/articles/21398307356050
**取得日**: 2026-05-02 18:43:00 JST

## 内容

altテキスト機能を使用すると、編集者はニュース投稿やページの画像に代替テキストを追加できます。スクリーンリーダーはこのテキストを使用して、視覚障害のあるユーザーがこれらの画像を理解するのを助けます。altテキストは、画像が読み込まれない弱いインターネット接続でも役立ちます。その場合、代替テキストが表示されます。altテキストは、すべての従業員にとって、よりアクセシブルで効率的かつ公平な作業環境を作ります。altテキストの使用に関するベストプラクティスについて詳しく学びましょう。
画像にaltテキストを追加する
ページやニュース投稿の画像にaltテキストを追加できますが、ヒーロー画像ウィジェット、チャット、または他のプラグインには利用できません。
スタジオでコンテンツに移動します。
altテキストを追加したいページまたは投稿に移動します。
altテキストを追加したい画像をクリックします。
赤いマークは、画像にaltテキストが追加されていないことを示します。
ALTアイコンをクリックします。
詳細を編集ダイアログが開きます。
画像のAlt Textを入力します。
適用をクリックします。
画像にaltテキストが追加されました。画像をクリックすると、緑のチェックマークがaltテキストが追加されたことを示します。上にカーソルを合わせるとaltテキストを読むことができます。
ニュース投稿やページが複数の言語で利用可能な場合は、各言語インスタンスに対してaltテキストを提供することを忘れないでください。
保存をクリックします。
画像にaltテキストを追加しました。いつでもaltテキストを編集するために戻ることができます。
