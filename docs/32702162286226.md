# コンテンツデザイナーで背景色や画像を追加する

**URL**: https://support.staffbase.com/hc/ja/articles/32702162286226
**取得日**: 2026-05-02 18:59:23 JST

## 内容

編集者
Content Designer では、セクションやコンテナにカスタムの背景色や画像を設定できます。さらに、背景画像にカラーオーバーレイを適用して、画像を控えめにしたりコントラストを調整したりできます。これにより、コンテンツを強調したり、コントラストを向上させたり、ページデザインをブランドガイドラインに合わせたりすることができます。
利用できるのは、アプリとイントラネットの設定で定義されたブランドカラーのみです。追加の色が必要な場合は、管理者にアプリとイントラネットの設定で追加してもらってください。
Studio で コンテンツ > ページ > 編集したいページに移動します。
更新したいセクションまたはコンテナを選択する。
コンテキストツールバーが開きます。
デザイン で、次のいずれかのオプションを選択します。
背景色
背景色 オプションを選択します。
ブランドカラーのドロップダウンから色を選択します。
セクションまたはコンテナに背景色が追加されました。
背景画像
背景画像 オプションを選択します。
画像を追加 をクリックします。
マイファイル ダイアログが開きます。
画像を選択またはアップロードします。
追加する をクリックします。
セクションまたはコンテナに背景画像が追加されました。
テキストの読みやすさやビジュアルの一貫性を高めるために、背景画像にオーバーレイを追加できます。
必要に応じて、オーバーレイ のトグルを調整してオーバーレイを有効にします。
オーバーレイ で、ブランドカラードロップダウンから色を選択します。
不透明度を設定 で、スライダーを調整するか数値を入力します。
セクションまたはコンテナの背景画像が設定されました。
