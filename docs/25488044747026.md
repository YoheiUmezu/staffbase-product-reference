# ウィジェットの概要

**URL**: https://support.staffbase.com/hc/ja/articles/25488044747026
**取得日**: 2026-05-02 18:42:05 JST

## 内容

編集者
ウィジェットは、任意のページやニュースで特定のコンテンツを表示するデザイン要素のようなものです。一部のウィジェットは静的なコンテンツを表示し、他のウィジェットはニュースのような動的な情報を表示します。スタッフベースのプラットフォームでは、作成者や編集者として、使いやすく、コンテンツを最適な形で提示できる便利なウィジェットを選択できます。
ウィジェットによる高度なコンテンツ編集オプション
デバイスの指定：一部のコンテンツはモバイルでより効果的に表示され、他のコンテンツはデスクトップユーザー向けに適しています。ウィジェットを使うことで、どのデバイスでユーザーがウィジェットのコンテンツを見るかを指定でき、最適なユーザー体験をサポートします。
複数のウィジェットを組み合わせて使用：1つまたは複数のウィジェットを組み合わせて、ユーザーにとって意味のある形でコンテンツを提示できます。たとえば、ヒーロー画像ウィジェット内にボタンウィジェットを配置するなどです。
レイアウトスタイルの指定：ウィジェットのレイアウトスタイルを、カードスタイルありまたはなしで設定できます。カードスタイルでは、ウィジェットが白いカードと枠線で表示されます。
ウィジェットの数が多いと、ページの読み込み時間に影響する場合があります。多くのウィジェットを使ってページを作成した際に読み込み時間が長くなった場合は、スタッフベースではウィジェットの数を減らすことを推奨しています。
静的コンテンツ ウィジェット
編集者
他のウィジェットと同じデザインやビジュアルスタイルで、あらゆる種類の静的コンテンツを追加するには、静的コンテンツ ウィジェットを使用します。これにより、静的コンテンツ内のコンテンツも他のウィジェットと同じレイアウトスタイルのオプションを利用できます。このウィジェットの背景色を変更したり、アプリのブランディングカラーを選択したりすることも可能です。
ユーザープロフィール ウィジェット
編集者
従業員ディレクトリ以外で連絡先情報を追加するには、ユーザープロフィール ウィジェットを使用します。たとえば、「お問い合わせ」ページに連絡先を表示したい場合に便利です。ユーザープロフィールに場所、部署、役職などの詳細が含まれている場合、それらの情報が表示されます。
ファイルリスト ウィジェット
編集者
ファイルを選択して追加するには、ファイルリスト ウィジェットを使用してユーザーがファイルを閲覧できるようにします。たとえば、企業の社会的責任に関するページがある場合、ガイドラインや年間計画など関連するすべてのファイルをファイルリストにまとめて配置できます。
従業員アプリでは、ファイルはファイルリスト ウィジェットで閲覧できますが、ダウンロードはできません。
イントラネットでは、ファイルの閲覧とダウンロードが可能です。
ファイルリスト ウィジェットをコンテンツに追加する際、ウィジェットの表示対象をすべてのユーザーまたは選択したグループに設定できます。この表示設定はウィジェットにのみ適用され、ファイル全体の検索性には影響しません。ファイルを含むページにアクセスできるユーザーは、引き続きそのファイルを見つけて閲覧できます。
情報ボックス ウィジェット
編集者
特定の情報や注意、ヒントなどに注目を集めるためのボックスをページに追加するには、情報ボックス ウィジェットを使用します。色付きのボックスがユーザーの注意をコンテンツに引き付けます。
ヒーロー画像 ウィジェット
編集者
読者の注目を集めるワイド画像を追加するには、ヒーロー画像 ウィジェットを使用します。画像上に見出しやサブ見出し、ボタン ウィジェットを追加することもできます。このウィジェットでは、画像のどの部分を小さいモバイル端末で表示するかを選択できるため、どちらのデバイスでも画像が美しく表示されます。
ヒーロー画像の理想的なサイズは1300 x 433ピクセルです。ただし、ヒーロー画像を追加するセクションのサイズが100%未満の場合、画像は自動的に各側から一定の割合でトリミングされます。トリミングされる範囲はセクションの幅やページカードモードが有効かどうかによって決まります。
アコーディオン ウィジェット
編集者
アコーディオンメニューを追加するには、アコーディオン ウィジェットを使用します。アコーディオンメニューは、クリックすると各見出しの下にコンテンツが表示される積み重ねられたヘッダーです。長いコンテンツを構造的に表示したいページに非常に便利です。ユーザーはページの概要を把握し、展開可能なアコーディオンヘッダーをクリックして中のコンテンツを読むことができます。
ボタン ウィジェット
編集者
ボタンを追加するには、ボタン ウィジェットを使用します。ボタンにはウェブサイト、メールアドレス、電話番号をリンクできます。ユーザーがボタン ウィジェットをクリックすると、関連する機能が実行されます。これにより、ユーザーはすぐにアクションを起こすことができます。たとえば、メールアドレスをボタンに設定した場合、ユーザーがクリックすると、その端末にインストールされているメールクライアントが開き、指定されたアドレス宛てのメール作成画面が表示されます。
ナビゲーション ウィジェット
ベータ
有効にする
編集者
ページ上にメニュー構造を表示するには、ナビゲーション ウィジェットを使用します。このウィジェットでは、メニューツリー全体または特定のブランチを表示できます。特定のブランチを表示するには、ウィジェットの設定で親アイテムのメニューIDを入力します。すると、その親アイテムのすべての子メニューアイテムが一覧表示され、ユーザーが階層的にコンテンツをナビゲートできるようになります。ナビゲーション ウィジェットは、メニューアイテムの追加・削除・並べ替えが行われると自動的に更新されます。
ニュースステージ ウィジェット
編集者
ページ上でニュースを目立たせて表示するには、ニュースステージ ウィジェットを使用します。さまざまなレイアウトやスライダーのトランジションスタイルを選択できます。動画もニュースステージから直接再生できます。スライダーに含める投稿数も設定可能です。さらに、ティーザーテキストやコメント、いいね、ブックマーク、共有などのソーシャルインタラクションも追加できます。
ニュースステージ ウィジェットには、記事やメディアからの投稿のみ追加できます。
ニュースフィード ウィジェット
編集者
Staffbaseのニュースフィードをページに追加するには、ニュースフィード ウィジェットを使用します。ページやページのセクションがニュースセグメントとして機能し、ニュースページやチャンネルからの投稿が一覧表示されます。ニュースフィードに含める投稿数を設定できます。ユーザーは、ニュースページやチャンネルの事前設定された並び順に応じて、最新または最も反応の多いニュース投稿をフィードで確認できます。
リンクリスト ウィジェット
編集者
便利なウェブサイトのリンク一覧を追加するには、リンクリスト ウィジェットを使用します。たとえば、ページに過去の活動の写真や動画ギャラリーへのリンク、来年の計画活動へのリンクなど、関連するリンクを追加できます。
リンクタイル ウィジェット
編集者
タイル形式でユーザーにリンクを表示するには、リンクタイル ウィジェットを使用します。ユーザーがタイルをクリックすると、関連付けられたリンクが開きます。さらに、アイコンの追加や背景色の変更など、レイアウトをカスタマイズできます。
ポートフォリオ ウィジェット
編集者
ポートフォリオ ウィジェットは、異なる部署や拠点、製品ラインなど、類似したコンテンツの異なるブランチをリンクするのに最適です。各リンクアイテムに画像や短いティーザーを追加することで、ユーザー向けにリンクの内容を分かりやすくし、ナビゲーションをさらに直感的にできます。
ポートフォリオ ウィジェット内のすべての画像は、16:9形式で、サイズは最低1000 x 562ピクセルである必要があります。
投稿ウィジェット
編集者
ニュースフィードに直接コンテンツを投稿するには、投稿ウィジェットを使用します。投稿ウィジェットは入力フィールドとして表示され、ユーザーはクリックして直感的に投稿を作成できます。モバイルアプリでは、入力フィールドを画面端に浮かせて表示するボタンとして設定することもでき、ページ内のどこからでも新しいコンテンツを追加できます。投稿ウィジェットは、ソーシャルウォールやFAQページ、マーケットプレイスページなど、インタラクションが多いページで特に便利です。
プラグイン ウィジェット
編集者
ページにプラグインコンテンツを埋め込むには、プラグイン ウィジェットを使用します。フォームや献立表、イベントカレンダーなどのプラグインコンテンツでページを充実させましょう。ユーザーインタラクションも有効にでき、ユーザーはページを離れることなくフォームの送信や週間の食堂メニューの閲覧が可能です。
アニメーション ウィジェット
編集者
ニュース投稿にお祝いのアニメーションを追加して、楽しく魅力的な演出を加えるには、アニメーション ウィジェットを使用します。投稿に紙吹雪や花火、風船、雪などを加え、良いニュースや特別なお知らせを従業員と一緒に祝うことができます。
グループ購読 ウィジェット
編集者
ユーザーがページ上でオープングループに直接参加・退会できるようにするには、グループ購読 ウィジェットを使用します。これにより、オープングループがユーザーにとってより見つけやすく、アクセスしやすくなります。ウィジェットは、ユーザーがすでに関連グループのメンバーかどうかを認識し、それに応じて表示されます。メンバーでない場合、ウィジェットのボタンをクリックしてすぐにそのグループのコンテンツを閲覧できます。すでに購読している場合は、購読解除してグループを退会し、コンテンツを非表示にできます。さらに、ボタンのテキストや色もカスタマイズ可能です。
埋め込みページ ウィジェット
編集者
外部ウェブサイトをページ上にシームレスに表示するには、埋め込みページ ウィジェットを使用します。これにより、外部ウェブサイトをページ内に戦略的に表示でき、複数の外部コンテンツストリームを1か所にまとめることができます。デスクトップやモバイルで最適に表示されるよう高さをカスタマイズでき、スクロールやインタラクションを有効にして、ユーザーがページを離れることなく外部ウェブサイトをナビゲート・クリックできます。
タスク管理ハブ ウィジェット
編集者
タスク管理ハブウィジェットは、従業員が割り当てられているすべてのタスクプロジェクトへの中央エントリーポイントを提供します。複数のチーム、プロジェクト、または拠点で作業するユーザーが、メニューを移動せずにプロジェクト間を迅速かつ確実に切り替えてアクセスできるようにします。
インテグレーションウィジェット
Microsoft 365 ウィジェット
有効化
編集者
他の標準ウィジェットと同じデザインやビジュアルスタイルで Microsoft 365 連携を有効にするには、Microsoft 365 ウィジェットを使用してください。これらのウィジェットは、Microsoft プラットフォームのコンテンツやサービスをページやニュース投稿記事でユーザーに提供します。Microsoft 365 ウィジェットの詳細は、こちらをご覧ください。
Google ウィジェット
ベータ
有効化
編集者
Google ウィジェットを使って、Google Workspace をイントラネットに取り入れましょう。Google ドライブウィジェットやGoogle カレンダーウィジェットにより、ファイルやカレンダーへの個別アクセスをページ上で直接提供し、従業員が自分のドキュメントや会議をリアルタイムでシームレスに確認できます。
ServiceNow ウィジェット
ベータ
有効化
編集者
従業員がページ上で自分の ServiceNow チケットの概要を確認できるようにするには、ServiceNow チケットウィジェットを使用してください。このウィジェットでは、未解決および割り当て済みのチケットが表示されます。従業員は ServiceNow チケットウィジェット内のリンクから、チケットの検索、新規作成、既存チケットの開く操作を直接行うことができます。
HR ウィジェット
ベータ
有効化
HR ウィジェットを使うことで、組織は従業員がページ内で自分の人事情報に直接アクセスできるようにします。統合された HR システムから安全にデータを取得し、従業員はアプリやイントラネットから給与明細や欠勤記録を簡単に確認できます。
