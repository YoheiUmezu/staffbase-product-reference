# 開発者コンソールの開き方

**URL**: https://support.staffbase.com/hc/ja/articles/12354166951954
**取得日**: 2026-05-02 18:43:35 JST

## 内容

お客様サポートチームを通じてリクエストを管理するプロセスの一環として、スクリーンショットを求められることがあります。開発者コンソールの情報は、お客様サポートチームがエラーや問題を特定し、迅速に解決策を見つけるのに役立ちます。
Chrome
Chromeブラウザで、三つの点をクリックします。
メニューが開きます。
クリック その他のツール。
クリック 開発者ツール。
開発者コンソールが開きます。
または、Windowsの場合はキーボードショートカット [Ctrl]+[Shift]+[J]、Macの場合は [⌘]+[Option]+[J] を使用します。
Mozilla Firefox
Firefoxブラウザで、三つの線をクリックします。
メニューが開きます。
クリック その他のツール。
クリック ウェブ開発者ツール。
開発者コンソールが開きます。
または、Windowsの場合はキーボードショートカット [Ctrl]+[Shift]+[J]、Macの場合は [⌘]+[Option]+[J] を使用します。
Safari
Safariブラウザで、メニューバーの Safari をクリックします。
クリック 設定。
設定が開きます。
「詳細設定」に移動します。
チェックボックス メニューバーに開発メニューを表示 を選択します。
Macのメニューバーが 開発 アイテムを含むように拡張されます。
メニューバーの 開発をクリックします。
クリック JavaScriptコンソールを表示。
開発者コンソールが開きます。
または、キーボードショートカット [Option]+[Cmd]+[C] を使用します。これには、Safariの設定で開発メニューが有効になっている必要があります。
Microsoft Edge
Edgeブラウザで、三つの点をクリックします。
メニューが開きます。
クリック その他のツール。
クリック 開発者ツール。
開発者コンソールが開きます。
または、Windowsの場合はキーボードショートカット [Ctrl]+[Shift]+[J]、Macの場合は [⌘]+[Option]+[J] を使用します。
