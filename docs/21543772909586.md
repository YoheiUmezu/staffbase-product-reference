# エディトリアルカレンダーの週次ビューの使用

**URL**: https://support.staffbase.com/hc/ja/articles/21543772909586
**取得日**: 2026-05-02 18:46:24 JST

## 内容

エディター
エディトリアルカレンダーの週次ビューは、プラットフォーム内のニュース投稿とブロッカーの概要を提供し、デザインが更新されています。スペースとチャンネルのフィルターを使用して、特定のコンテンツに絞り込むことができます。
週次ビューから直接、便利に以下の操作ができます：
ニュース投稿とブロッカーを表示
ニュース投稿とブロッカーを作成
ニュース投稿とブロッカーをドラッグ＆ドロップ
ドラフト投稿をカレンダーに追加
ニュース投稿またはブロッカーをクリックして、主要情報を含むフライアウトを表示
フライアウト内のニュース投稿タイトルをクリックして、ニュース投稿を開く
フライアウト内の記事およびメディア投稿の主要情報を編集
フライアウト内のブロッカーを編集
組織で利用可能な場合、スペース、チャンネル、キャンペーンでビューをフィルタリング
オプションのフル透明モードを使用して、通常アクセスできないコンテンツを表示（表示専用形式）
ニュース投稿のフライアウトでは、以下のような有用な情報を表示できます：
ターゲットオーディエンス
有効化された通知タイプ
作成者
投稿のステータス
割り当てられたキャンペーン（組織で利用可能な場合）
投稿のステータスが公開されている場合の投稿統計
ブロッカーのフライアウトでは、以下のような有用な情報を表示できます：
開始時刻と終了時刻
説明
作成者
エディトリアルカレンダーにドラフト投稿を追加する
まだ公開日が割り当てられていないドラフト投稿をエディトリアルカレンダーの週次ビューに追加できます。
カレンダーにドラフト投稿を追加することで、コンテンツ計画を視覚化できますが、公開のスケジュールは設定されません。準備ができたら、各投稿を個別にスケジュールすることを忘れないでください。
スタジオで、計画 > エディトリアルカレンダーに移動します。
フィルターをクリックします。
フィルターオプションが開きます。
表示オプションのドロップダウンから、ドラフトを選択します。
適用をクリックします。
ドラフト投稿のサイドバーが開きます。
注意：サイドバーには、まだ計画またはスケジュールされていないすべての未スケジュールのドラフト投稿が、最終編集日付の降順で表示されます。
計画したい投稿に移動し、カレンダーの希望の日付と時刻にドラッグします。
注意：ドラフト投稿は未来の日付にのみドラッグ＆ドロップできます。
ドラフトがカレンダーに追加されました。今すぐクイック編集を使用してドラフトをスケジュールできます。
エディトリアルカレンダーから投稿を削除するには、エディトリアルカレンダーからドラフトサイドバーに投稿をドラッグ＆ドロップします。投稿はサイドバーの上部に表示されます。
サイドバーを閉じるには、フィルターで表示オプションドラフトの選択を解除するか、サイドバーのXをクリックします。
