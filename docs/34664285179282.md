# ローカル視聴者を管理するためのSpaceグループの概要

**URL**: https://support.staffbase.com/hc/ja/articles/34664285179282
**取得日**: 2026-05-02 18:59:18 JST

## 内容

アクティベーション
ユーザーグループは、Staffbase プラットフォームのコアコンセプトです。ユーザーをオフィスの場所、部門、興味などの共通属性に基づいてグループ化できます。その後、各グループにコンテンツをターゲットできます。
スペースは、Studio 内でスペース管理者が特定の視聴者向けにコンテンツを整理・管理するための指定エリアです。通常、部門、国、ブランドなど、組織の構造を反映しています。
ユーザーグループとスペースのコンセプトが組み合わさることで、Space グループが形成されます。これにより、スペース管理者はスペース固有のグループを独自に作成・管理でき、管理者は全体のガバナンスを維持します。この分散化により、より迅速なコンテンツターゲティングとスケーラブルなコラボレーションが可能になります。
Space グループを使用するメリット
分散型の視聴者管理
管理者への依存が減ることで、スペース管理者はスペース中心のニーズにより迅速に対応できます。
より迅速かつターゲット対象のコンテンツ配信
スペース固有のグループを管理することで、チームは関連する視聴者を定義し、遅延なくコンテンツを公開できます。これにより、内部コミュニケーションのスピードと精度が向上します。
大規模な組織におけるスケーラビリティの向上
Space グループは、複数の場所、ブランド、部門を持つ成長中の組織をサポートします。チームは、中央管理を複雑にすることなくオンボーディングや管理が可能です。
ローカルの自律性とバランスの取れたガバナンス
チームは定められた範囲内で独立して運用でき、組織全体でコントロールと柔軟性の両立を実現します。
