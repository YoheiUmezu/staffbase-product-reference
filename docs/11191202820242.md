# ビデオインサイトの表示

**URL**: https://support.staffbase.com/hc/ja/articles/11191202820242
**取得日**: 2026-05-02 18:46:52 JST

## 内容

アクティベーション
エディター
ビデオインサイトは、コンテンツの可視性とエンゲージメント率をよりよく理解するための指標を提供します。従業員アプリとイントラネットの以下の領域でビデオインサイトを見つけることができます。
ビデオインサイトは、Staffbaseプラットフォームにアップロードされたビデオにのみ利用可能です。YouTubeやVimeoなどの外部プラットフォームはサポートされていません。
ニュース投稿
スタジオ > ニュースに移動します。
ビデオを含むニュース投稿を開きます。
ビデオインサイトアイコンをクリックします。
ファイルインサイトダイアログが開き、メタデータと指標が表示されます。 メタデータと取得した指標の簡単な説明を表示するには、(?)アイコンにカーソルを合わせてください。
Xをクリックして閉じます。
ページ
スタジオ > ページに移動します。
ビデオを含むページを開きます。
ビデオインサイトアイコンをクリックします。
ファイルインサイトダイアログが開き、メタデータと指標が表示されます。 メタデータと取得した指標の簡単な説明を表示するには、(?)アイコンにカーソルを合わせてください。
Xをクリックして閉じます。
ページ内からビデオインサイトを表示しました。
ファイルマネージャー
マイファイルビューおよび中央ファイルマネージャーからビデオインサイトを表示できます。 詳細を学ぶ。
