# Staffbaseスタジオとダッシュボード

**URL**: https://support.staffbase.com/hc/ja/articles/4958967494802
**取得日**: 2026-05-02 18:49:40 JST

## 内容

すべての内部コミュニケーションチャネルを一元管理します。スタジオでは、ネイティブアプリ、イントラネット、またはメールなど、さまざまなチャネルを簡単に見つけ、管理し、切り替えることができます。ダッシュボードは、内部コミュニケーションタスクへの入り口であり、プラットフォーム上のエンゲージメントと活動の全体像を提供します。
最適な視覚体験とユーザー体験のために、Staffbaseはスタジオを最小解像度1440 pxの画面で使用することを推奨します。ただし、Staffbase Emailでは、最小解像度1100 pxがサポートされています。
スタジオとダッシュボードの概要
ダッシュボードは、内部コミュニケーションコンテンツの分析、計画、作成を開始する場所です。
プラットフォームのエンゲージメントと活動の全体像を把握し、最新のトップパフォーマンスコンテンツに関する非常に関連性の高い分析を得ることができます。ダッシュボードからは、クイックアクションを通じて最も重要なタスクにワンクリックでアクセスできます。
水平スタジオナビゲーションでは、ダッシュボードや、プラン、役割、権限に応じた追加のメニュー項目が提供されます。計画、コンテンツ、メール、分析、ユーザー管理、および設定の間で迅速に切り替えることができます。
編集カレンダーとキャンペーンは、計画タブの下にあります。
これは、すべての計画ニーズのためのワンストップデスティネーションです。コミュニケーションの計画を次のレベルに引き上げましょう。
ニュース、ページ、スペース、メニュー、コメント、およびその他のすべてのプラグインは、現在コンテンツの下にまとめられています。
コンテンツセクションでは、コンテンツの作成と管理の最良の概要を提供します。スペースチューザーを使用して、常に正しいオーディエンスをターゲットにしてください。
従業員メールはスタジオに統合されています。
ワンクリックで、Staffbase Emailのすべてのお気に入り機能に切り替えます。統合されたナビゲーションを使用して、チームと一緒にメールをデザイン、送信、追跡、共有します。
アプリとイントラネットの分析にトップナビゲーションからアクセスします。
分析データからユーザーエンゲージメントについて学び、それに応じてコミュニケーションを計画します。
Staffbase Emailの分析はメールセクションにあります。
ユーザー管理と設定にトップナビゲーションバーから直接アクセスします。
アプリとイントラネットのユーザー、メール連絡先、さまざまなメール権限と役割を一か所で見つけます。ギアホイールをクリックすると、プラットフォーム設定が開きます。

FAQ
新しいスタジオでロゴが見えないのはなぜですか？
旧スタジオでは、組織のロゴでスタジオをブランディングすることが可能でした。新しいスタジオでは、このオプションが削除され、常にStaffbaseのロゴが表示されます。ロゴ設定はスタジオにのみ影響します。アプリやイントラネットでは、ユーザーはブランドのロゴと色を確認できます。
スマートフォンのプレビューが見えなくなったのはなぜですか？
旧スタジオのダッシュボードでは、スマートフォンフレーム内でコンテンツのプレビューが表示されていました。このスマートフォンプレビューは新しいスタジオでは利用できなくなり、削除されました。
ダッシュボードタイルにアクセスする際に「何かがうまくいかなかった」というエラーメッセージが表示されるのはなぜですか？
これは、ブラウザに広告ブロッカーがインストールされている場合に発生する可能性があります。その場合、広告ブロッカーは分析の表示もブロックします。問題を解決する方法を学んでください。
