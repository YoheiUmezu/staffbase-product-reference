# 従業員メールのチームページにアクセスする

**URL**: https://support.staffbase.com/hc/ja/articles/4578694923538
**取得日**: 2026-05-02 18:47:04 JST

## 内容

製品名に関する注意: 2022年以降、BananatagはStaffbaseの従業員メール製品です。
マネージャー
組織内の従業員メールユーザーのさまざまな設定を管理するためにチームページにアクセスするか、他のユーザーとしてサインインして、彼らが見るのと同じウェブアプリのビューを確認します。
現在、Staffbaseのアプリおよび/またはイントラネットが従業員メールと統合されている場合:
あなたのスタジオを使用して:
ユーザーを追加
ユーザーを削除
ユーザー情報を更新
チームページを使用して:
メール機能へのユーザーのアクセスを管理します。たとえば、送信、スタイリング、レポートの生成など
メールコラボレーションのためのグループメンバーシップを編集します
他のユーザーとしてサインイン (たとえば、共有メールアドレス用に作成された共有ユーザーアカウント)
手順
ウェブアプリ
クイックリンク
北米のお客様 - https://app.bananatag.com/team
ヨーロッパのお客様 - https://app.de.bananatag.com/team
ナビゲーション手順
従業員メールウェブアプリで、イニシャルが入ったアカウントアイコンをクリックします。
開いたドロップダウンメニューでチームをクリックします。
チームページが開きます。
スタジオ
最新の スタジオインターフェース について詳しく学びます。
スタジオには従業員メールの統合も含まれています！
ナビゲーション
スタジオで、ユーザー設定 > メールチームと役割に移動します。
チームページが開きます。
