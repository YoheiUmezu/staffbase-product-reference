# ニュース投稿を別のチャンネルにコピーする

**URL**: https://support.staffbase.com/hc/ja/articles/33943264091410
**取得日**: 2026-05-02 19:02:38 JST

## 内容

編集者
ニュースを使用すると、スタッフベースアプリやイントラネットで視聴者ごとに特化したコンテンツを公開できます。ニュースチャンネル内でニュース投稿記事を作成することができます。また、同じ種類の別のチャンネルに投稿をコピーすることも可能です。コピーされた投稿は、元の投稿の公開状況に関わらず、新しいチャンネルで下書きとして表示されます。
Studioでコンテンツ > ニュースに移動します。
コピーしたいチャンネルと投稿を探します。
オプションで投稿をコピーするをクリックします。
投稿をコピーするダイアログが開きます。
ドロップダウンから、投稿をコピーしたいチャンネルを選択します。
コピーするをクリックします。
コピーした投稿のチャンネルが開きます。
これで、投稿を新しいチャンネルの下書きとしてコピーできました。次に、投稿を公開することができます。
