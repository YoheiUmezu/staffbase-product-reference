# ページの作成

**URL**: https://support.staffbase.com/hc/ja/articles/206435159
**取得日**: 2026-05-02 18:57:41 JST

## 内容

編集者
ページは、ガイドライン、リソース、または長期間有効なコンテンツなど、長期的な関連性のあるインフォメーションを共有するのに最適です。Studioでは、クラシックエディターまたは最新の新エディターを使用してページを作成できます。新エディターは、柔軟なレイアウトと使いやすさを備えた最新のWYSIWYG編集体験を提供します。
ページを作成する
Studio内でコンテンツ > ページに移動します。
ページを作成するをクリックします。
ページを作成するダイアログが開きます。
次のオプションのいずれかを選択します：
新エディターで作成
ベータ
ページのタイトルを入力します。
任意で説明を追加します。
スペースドロップダウンからスペースを選択します。
ページの表示を設定します。
ページの所有者となるユーザーを選択します。
作成するをクリックします。
新エディターが開きます。
ページが作成されました。ここからコンテンツの作成方法を選択できます：
AIページ生成を利用できる場合は、下書きを生成します。
最初から新しく始めるを選択し、ページコンテンツとレイアウトを手動で作成します。
クラシックエディターで継続する
ページのタイトルを入力します。
任意で説明を追加します。
スペースドロップダウンからスペースを選択します。
ページの表示を設定します。
ページの所有者となるユーザーを選択します。
テンプレートドロップダウンメニューから使用するテンプレートを選択します。
作成するをクリックします。
ページ編集者が開きます。
ページが作成されました。ページをカスタマイズして公開できます。
