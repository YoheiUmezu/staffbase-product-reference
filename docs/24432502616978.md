# Microsoft 365 Copilot用のStaffbaseエージェントの追加

**URL**: https://support.staffbase.com/hc/ja/articles/24432502616978
**取得日**: 2026-05-02 18:58:47 JST

## 内容

ベータ
アクティベーション
管理者
Microsoft Teamsの管理センターでカスタムアプリとしてアップロードすることで、Microsoft 365 Copilot用のStaffbaseエージェントを簡単にインストールできます。アプリパッケージはStaffbase Studioから.zipファイルとしてダウンロードできます。Microsoft 365 Copilot用のStaffbaseエージェントは、共有テナントデータを利用します。
前提条件
Microsoft 365 Copilotライセンスを持っているか、メーター消費を使用しています。
Microsoft 365でグローバル管理者またはTeamsサービス管理者の役割を持っています。
Microsoft Entra IDとのシングルサインオン（SSO）を設定しています。
Microsoft 365 Copilot用のStaffbaseエージェントのダウンロード
Studioで、設定に移動します。
拡張機能の下で、Microsoft 365をクリックします。
Microsoft 365 Copilot用のStaffbaseエージェントに移動し、ダウンロードをクリックします。
Microsoft Teams用のMicrosoft 365 Copilot用のStaffbaseエージェントをダウンロードしました。
Microsoft 365 Copilot用のStaffbaseエージェントのインストール
前提条件
Studioからダウンロードしたアプリパッケージを持っています。
Microsoft 365でグローバル管理者の役割を持っている必要があります。
Staffbase StudioからダウンロードしたMicrosoft 365 Copilot用のStaffbaseエージェントアプリパッケージをインストールできます。Microsoft Learnに記載されている手順に従って、Microsoft 365管理センターにカスタムアプリとしてパッケージをアップロードします。
Microsoft 365 Copilot用のStaffbaseエージェントへの同意の付与
Microsoft 365 Copilot用のStaffbaseエージェントは、Microsoftシングルサインオン（SSO）を使用してStaffbaseにサインインします。Microsoft Entra IDでStaffbaseエージェント用のMicrosoft 365 Copilot SSOアプリケーションに同意を付与する必要があります。
前提条件
Microsoft 365管理センターにカスタムアプリとしてアプリパッケージをアップロードしています。
新しいエンタープライズアプリケーションを追加および構成するための必要な権限を持って、組織のMicrosoft Entra IDにアクセスできます。
Studioで、設定に移動します。
拡張機能の下で、Microsoft 365をクリックします。
Microsoft 365 Copilot用のStaffbaseエージェントに移動します。
Microsoft 365管理者アカウントで同意を付与リンクを開きます。
権限ダイアログが開きます。
受け入れるをクリックします。
Microsoft Entra IDでStaffbaseエージェント用のMicrosoft 365 Copilot SSOアプリケーションに対する権限を付与しました。
