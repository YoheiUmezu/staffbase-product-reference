# コメントでの@メンションの使用

**URL**: https://support.staffbase.com/hc/ja/articles/360011564220
**取得日**: 2026-05-02 18:57:25 JST

## 内容

@メンションは、従業員アプリやイントラネットでユーザーをタグ付けし、彼らの注意を引く方法です。@メンションは、Staffbaseプラットフォームの複数の場所で利用可能です。コメントでは、興味を持ちそうな投稿を知らせたり、会話に参加させたりするためにユーザーをタグ付けできます。
同僚を通知するには、@記号の後にそのユーザーの名前を入力してタグ付けします。入力を続けると、ドロップダウンに一致するリストが表示され、自動的に絞り込まれます。投稿に適切なアクセス権を持つユーザーのみがドロップダウンに表示されます。コメントを送信すると、@メンションをクリックすることでそのユーザーのプロフィールに移動します。
タグ付けされた同僚は、通知センターで通知を受け取ります。また、プッシュ通知が有効になっている場合は、モバイルデバイスにプッシュ通知も届きます。
ユーザーの@メンション
従業員アプリまたはイントラネットで、ニュース投稿に移動します。
コメントセクションに移動します。
コメントフィールドに@記号とタグ付けしたいユーザーの名前を入力します。
タグ付けしたいユーザーを選択します。
あなたは@メンションを使用して従業員をタグ付けしました。コメントを送信すると、そのタグはユーザーの通知またはモバイルデバイスのプッシュ通知で表示されます。
@メンションのプッシュ通知を有効にする
従業員アプリまたはイントラネットで、設定 > 通知に移動します。
メンションの下で、プッシュ通知のトグルを有効にします。
