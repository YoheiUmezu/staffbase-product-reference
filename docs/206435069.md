# チャンネルの作成

**URL**: https://support.staffbase.com/hc/ja/articles/206435069
**取得日**: 2026-05-02 18:59:17 JST

## 内容

チャンネルを使用して情報を公開し、すべての従業員に最新情報を提供します。
チャンネルの作成に関する詳細情報は、この記事のニュースチャンネルの概要をご覧ください。
スタジオで、コンテンツ > ニュースに移動します。
チャンネルを追加をクリックします。
チャンネルの作成ダイアログが開きます。
チャンネル名を入力します。
チャンネルの種類を選択します。
記事、メディア、または更新のチャンネルタイプのいずれかを選択します。チャンネルが作成されると、チャンネルタイプを変更することはできません。
保存をクリックします。
