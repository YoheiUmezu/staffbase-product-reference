# 投稿統計によるニュース分析

**URL**: https://support.staffbase.com/hc/ja/articles/360015278180
**取得日**: 2026-05-02 18:44:47 JST

## 内容

エディター
投稿統計とリーチ分析は、コンテンツ戦略に沿った改善を行い、内部コミュニケーションプラットフォームでのエンゲージメントを向上させるための力を与えます。投稿統計は、チャネル内の各投稿のパフォーマンスを測定し、他の投稿と比較するのに役立ちます。
これらの数値は視覚的に魅力的で、消費しやすい形で提示され、データに基づいた意思決定を行うのに役立ちます。詳細なトラフィックソースは、各ユーザーが投稿にどのようにアクセスしたかを示します。また、投稿にアクセスするための最も一般的な5つのナビゲーションパスも示します。詳細については、ニュースの投稿統計におけるトラフィックソースの記事をご覧ください。
分析用語集では、Staffbaseプラットフォーム全体の特定の指標についての詳細が提供されています。
投稿統計の表示
スタジオ
スタジオでは、投稿のパフォーマンスに関する詳細な洞察を提供します。ニュースの概要で、1クリックで投稿のエンゲージメントに関する関連数値や最も重要なトラフィックソースを確認できます。
スタジオで、コンテンツ > ニュースに移動します。
統計を表示したい投稿に移動します。
リーチボタンにカーソルを合わせて、簡単に概要を確認します。
リーチボタンをクリックします。
投稿統計が表示されます。
アプリ または イントラネット
投稿統計は、アプリまたはイントラネットの投稿から直接、統計ボタンを使用しても利用できます。この簡単な概要は、スタジオビューにアクセスできない貢献者にとって特に便利です。
アプリまたはイントラネットで、統計を表示したい投稿に移動します。
投稿を表示しているときに、ヘッダーの統計をクリックします。
投稿統計ダイアログが開きます。
