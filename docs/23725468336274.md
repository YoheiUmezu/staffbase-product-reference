# 2025年第1四半期リリース

**URL**: https://support.staffbase.com/hc/ja/articles/23725468336274
**取得日**: 2026-05-02 18:56:45 JST

## 内容

今年は、接続、創造、コラボレーションの方法を再定義する革新的な機能で、イノベーションを促進しましょう。改良されたアイデアポータルはフィードバック管理を効率化し、新しいオフラインモードは、チームがいつでもどこでも情報を得られるようにします。ブリーフィングとMicrosoft 365 Copilot用のStaffbaseエージェントを使用して、次世代の生産性を体験し、よりスマートなワークフローとシームレスなコミュニケーションを実現します。そして今、Screensはベータ版を卒業し、従業員アプリまたはイントラネットビジネスパッケージ以上のアドオンとして利用可能です。興奮はここで止まりません—Staffbase Emailが間もなく登場します！詳細については、ローンチ情報をチェックしてください。
2025年2月20日以降に登場するすべての情報を発見するために読み進めましょう。一緒にイノベーションを点火しましょう！
改良されたアイデアポータル
自動展開
従業員アプリ イントラネット
改良されたアイデアポータルは、アイデアの提出と管理を効率化し、Staffbaseがあなたのフィードバックに迅速に対応できるようにします。管理者や上位の役割は、フィードバックを提供するボタンを使用して簡単に意見を共有でき、新しいフィードバック管理ポリシーにより、あなたの意見の処理の透明性が確保されます。また、公開されたアイデアに投票して、あなたにとって最も重要なものを優先することもできます。一緒に、プラットフォームをあなたのニーズに合わせてカスタマイズしましょう。
ユーザープロファイルの組織図
アクティベーション
従業員アプリビジネス イントラネットビジネス
ベータ版を正式に卒業したユーザープロファイルの組織図は、組織の構造を包括的に把握することができます。ユーザーのプロファイルからマネージャー、直接の同僚、部下を特定したり、表示されたプロファイルをクリックすることで、あるユーザーの組織図から別のユーザーの組織図にジャンプすることができます。さらに、ボタンをクリックするだけで、組織図に追加の従業員のバッチをシームレスに読み込むことができます。
Screens
アクティベーション
従業員アプリビジネス イントラネットビジネス アドオン
前回のリリースでは、古いスクリーン埋め込みをさようならし、現在ベータ版を卒業したScreensを歓迎しました！このデジタルサイネージのためのアップグレードされたソリューションは、上部ナビゲーションから簡単にアクセスできます。安全なデバイス管理、検索機能付きのニュースソースセレクター、マイファイル経由のメディアアップロード、高度なカスタマイズを提供します。投稿の制限、期間、ブランディング、画像のスケーリングを調整して、観客を魅了する洗練されたプロフェッショナルなディスプレイを作成しましょう。
改善された編集カレンダーの週間ビュー
自動展開
従業員アプリビジネス イントラネットビジネス
改善された編集カレンダーの週間ビューで計画を強化しましょう。ベータ版を卒業し、時間単位のグリッドを備えたこの機能は、編集者に現在の週のすべての計画された投稿のスナップショットを提供し、次の週の戦略的計画を促進します。ワークフローを効率化し、週をスムーズに進行させるための完璧なツールです。よりスマートに計画し、整理された状態を保ち、週を順調に進めましょう！

ベータ

改善点
自動展開
サポートされているiOSモバイルアプリバージョン
サポートされているiOSモバイルアプリの最小バージョンは現在16.0です。
ニュース投稿概要ページの改善
チャンネルのニュース投稿概要ページは、エディターロール以上のユーザー向けに多くの発見性の改善が施され、UXが向上しました。これらの改善により、特定の投稿を見つけることが簡素化され、迅速化され、タイトルやコンテンツのテキスト検索が可能になりました。投稿の状態でフィルタリングして、ドラフト、スケジュール済み、または公開された記事を表示できます。ソースでフィルタリングして、Staffbaseで作成されたコンテンツやRSSフィードなどの外部コンテンツを表示できます。最後に、最近更新されたもの、作成日、または公開日で概要をソートできます。
ニュース投稿の共有の強化
従業員は、興味深いニュース投稿を同僚と2つの新しい方法で共有できるようになりました。ニュース共有が有効になっていると、ユーザーはニュース投稿をメールやデスクトップまたはモバイルデバイスのアプリに簡単に共有できます。これらの改善により、チームがしっかりとつながり、情報を得ることができます。
アクティベーション
ジャーニーズ投稿の自動翻訳
ジャーニーズは、自動翻訳をサポートし、エディターが効率的に多言語のジャーニーズコンテンツを作成できるようになりました。
個人メールアドレスの編集
個人メールアドレスの変更機能がベータを終了します。従業員アプリまたはイントラネットアカウント設定から個人メールアドレスを変更できるようになりました。サポートに連絡する必要はありません。この機能を有効にするには、Staffbaseサポートまたはカスタマーサクセスマネージャーに連絡してください。
メニュー項目の可視性
メニュー項目の可視性のデフォルト設定がベータを終了します。新しく作成されたインストール（ページ、プラグインインスタンス、フォルダー、ニュースページなど）は、デフォルトのデバイス可視性設定がデスクトップとモバイルでメニューに隠されるように設定されています。これにより、設定で有効にしない限りメニュー項目が隠され、混雑したメニューを防ぎ、クリーンで整然としたインターフェースを保ちます。
アプリヘッダーと水平ナビゲーションの再設計
デスクトップのアプリヘッダーと水平ナビゲーションが再設計され、よりモダンな外観と感触、改善されたブラウジング体験を提供します。この更新により、使いやすさが向上し、ユーザーにとってナビゲーションがよりスムーズで直感的になります。
コメント概要ページの感情ラベル
コメント概要ページのコメント感情ラベルを使用して、オーディエンスのトーンを簡単に把握できます。コメントがポジティブ、ニュートラル、またはネガティブかを一目で確認できます。エディターロール以上のユーザーは、感情でフィルタリングしたり、必要に応じて個々のコメントの感情を手動で編集したりできます。すべての更新は、Smart Insightsダッシュボードのコメント感情セクションにシームレスに反映され、分析と戦略的な対応を行うための明確さを提供します。
オープンベータとアクティベーション
タブレットの使いやすさの向上
タブレット専用に設計された、より広々としたユーザーフレンドリーなインターフェースで、従業員のタブレット体験を向上させます。この更新により、タブレットの画面サイズを最大限に活用し、より大きなディスプレイに完全に合わせたシームレスで向上した体験を提供します。
チャット会話でのファイルアップロード
移動中にドキュメントをシームレスに共有できます。従業員アプリのユーザーは、PDF、XLS、DOCなどのファイルをモバイルフォンからチャット会話に直接アップロードできるようになりました。フロントラインの従業員が、従業員アプリ内で重要なドキュメントを共有または受信できるようにし、コミュニケーションを効率的に保ち、ワークフローをスムーズにします。
モバイルクイックリンクの改善
モバイルクイックリンクの管理は、メニュー管理から完全に独立しており、より大きな柔軟性と精度を提供します。モバイルメニュー内のレベルや表示設定に関係なく、任意のメニュー項目をモバイルクイックリンクとして指定できるようになりました。
スケーラビリティの向上したユーザーデータエクスポート
ユーザーデータをCSVとしてエクスポートする機能が、より強力で効率的になりました。管理者は、任意の数のユーザーのCSVファイルをエクスポートでき、プロセスは最適なパフォーマンスのために非同期で処理されます。洗練されたユーザーインターフェースにより、大規模なデータセットを簡単に管理し、作業できるシームレスで直感的な体験が提供されます。
インフラストラクチャ
新しいAUホスティングインフラストラクチャ
Staffbaseは、オーストラリアのお客様により良いサービスを提供するためにホスティングインフラストラクチャを拡張しています！米国およびEUのホスティングインフラストラクチャに加えて、AUホスティングインフラストラクチャの立ち上げを発表できることを嬉しく思います。オーストラリア東部（NSW）のMicrosoft Azureによってホストされており、この強化により、データが国内に安全に保たれ、地域のコンプライアンスおよびセキュリティ要件を自信を持って満たすことができます。
Staffbase Email
あなたのメールはクリックされていますが、従業員は本当にメッセージを受け取っていますか？今こそ、問題に取り組む時です — Staffbase Emailで内部コミュニケーションを変革しましょう！
内部メールをより効果的かつ簡単にするために設計された新しい直感的なオールインワンプラットフォームで、メールワークフローを合理化します。従業員と真に繋がるインスピレーションを与えるメールキャンペーンを作成し、その効果に関する強力な洞察を得ましょう。
より良いコミュニケーション管理への旅は、Staffbase Emailの立ち上げから始まります — 近日公開予定！続報をお待ちください！

ウェブアプリ/フロントドアイントラネット
アプリとイントラネットの改善
ニュースでは、内部共有のために、ユーザーがニュース投稿をメールまたはデスクトップやモバイルデバイスのアプリケーションに共有できるようになりました。詳細を学ぶ。
エディトリアルカレンダーでは、記事およびメディア投稿のクイック編集オプションが月間および改善された週間ビューに追加されました。ニュースチャンネルの種類に応じて、クイック編集ではタイトル、日付と時間、ティーザーテキスト、説明を編集し、スケジュールを無効または有効にすることができます。詳細を学ぶ。
スタジオのコンテンツ言語ページでは、デザインとレイアウトが更新され、ユーザーエクスペリエンスが向上しました。
ファイルマネージャーでは、コレクションやファイルにマウスをホバーすると、そのフルネームが表示されます。
アプリとイントラネットのバグ修正
チェコ語およびスロバキア語の日時形式が正しくありませんでした。これが修正されました。
モバイルアプリ
モバイルアプリの改善
設定セクションでは、法的メニュー項目の配置が最適化されました。これにより、読み込み時間が短縮され、API応答時間による遅延が防止されます。
2024年11月6日
アプリとイントラネットの改善
フォームプラグインでは、提出日が米国の日付形式（月/日/年）のみ表示されていました。これが改善され、ユーザーのデバイスまたはブラウザの言語に基づいて適応されるようになりました。
戦略的計画では、ステータスが「計画中」のニュース投稿が、割り当てられたキャンペーンの下にタイムラインビューで表示されるようになりました。
ページでは、アニメーションウィジェットが利用可能になりました。4つの祝賀オーバーレイ（紙吹雪、花火、風船、雪）から選択できます。アニメーションウィジェットはニュースでも利用可能です。
外部コンテンツでは、カスタムチャンネルと割り当てられたアカウントの名前を編集できるようになりました。さらに、チャンネルアイコンを調整できます。
ファイルマネージャーのファイル置換機能では、バージョン履歴が利用可能になりました。この機能では、ファイルが置き換えられた回数、誰によって置き換えられたか、理由を説明する更新ノートが表示されます。
アプリとイントラネットのバグ修正
ニュースでは、場合によってはポーランド語およびジョージア語の投稿がHTMLハイパーリンクがある場合に正しく翻訳されませんでした。これが修正されました。
アプリとイントラネットの廃止
Facebookページプラグインは2024年10月31日に廃止されました。このプラグインはStaffbase Studioでは利用できなくなりました。
2024年10月23日
ウェブアプリ/フロントドアイントラネット
アプリとイントラネットの改善
コンテンツレビュー機能がレスポンシブになり、モバイルデバイス（タブレットやモバイルブラウザを含む）での最適な使いやすさと機能性を提供します。
検索結果が表示されたとき、ソートオプションのラベルが正しく表示されていませんでした。これが修正されました。
ファイルマネージャーでは、ファイルとコレクションが新しいタブに整理されて表示されます。すべてのコレクションに移動してコレクションを見つけ、すべてのファイルに移動してファイルを見つけてください。
ミッションコントロール
ミッションコントロールの改善
スマートインサイトダッシュボードのタイルユーザーグループ別の可視性およびユーザーグループ別のエンゲージメントでは、匿名性を確保するために、5人以上のメンバーを持つユーザーグループのみが表示されます。
ミッションコントロールのバグ修正
ニュース投稿で、詳細なコンテンツレビュー機能を使用する際、ユーザー役割のラベルがStaffbaseで使用されている実際のユーザー役割を表示していませんでした。これが修正されました。
