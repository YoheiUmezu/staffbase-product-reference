# アナリティクスダッシュボードでユーザーメトリクスを分析する

**URL**: https://support.staffbase.com/hc/ja/articles/12216791121554
**取得日**: 2026-05-02 19:00:18 JST

## 内容

スペース管理者
アナリティクスダッシュボードは、ユーザーがStaffbaseプラットフォームとどのように、いつ、どこで関与しているかについてのより深い洞察を提供します。アナリティクスダッシュボードは、メトリクスを深く掘り下げることができるように、さまざまなカテゴリに分かれています。
この記事では、ユーザータブの詳細、グラフやその中にある分析データについて説明します。ユーザーの役割アクセスを含むアナリティクスダッシュボード全体については、アナリティクスダッシュボードの概要をご覧ください。
フィルター
ユーザータブは、デフォルトで過去7日間のデータを表示します。日付範囲を変更し、さらにフィルターを使用して結果を絞り込むことができます。
作成タイプフィルターは、ユーザータブに特有のものです。このフィルターを使用すると、グラフに表示したいアカウント作成の種類を選択できます。これは、事前設定されたフィルターオプションのリストであり、組織のユーザー管理に基づいて動的ではありません。
フィルターには以下が含まれます：
メールによる自己登録
CSVインポート
手動招待
SCIM
Active Directory/LDAP
シングル サインオン
フィルタリングについて詳しく学ぶ。
グラフ
ユーザータブには、ユーザーに関する詳細なメトリクスを持つ4つのグラフがあります。
異なるユーザータイプのメトリクスは、技術的な理由で異なる時間に更新されます。合計および登録済みユーザーのメトリクスは、毎日午前2時頃（CEST）に更新されます。アクティブおよびエンゲージドユーザーのメトリクスは、リアルタイムで、日中継続的に更新されます。その結果、レートが完全に一致しない場合があります。
登録済みおよび合計ユーザーのデータは、2023年6月26日まで遡ります。それ以前の期間が選択された場合、これら2つのメトリクスのデータは利用できません。
ユーザーアクティビティ
ユーザーアクティビティグラフは、プラットフォーム上のアクティブ、エンゲージメント、登録済み、および合計ユーザーを表示します。
アクティブユーザー: 少なくともアプリを開いたユーザー
エンゲージメントユーザー: コメント、いいね、または共有で参加したユーザー
登録済みユーザー: プラットフォームにログインしたユーザー
合計ユーザー: すべての登録済み、保留中、および招待されたユーザー
アナリティクス用語について詳しくは、アナリティクス用語集をご覧ください。
ユーザーアクティビティの折れ線グラフは、さまざまなユーザータイプの詳細な統計を表示します。デフォルトでは、すべてのユーザーメトリクスを表示します。各メトリクスにはグラフ内で対応する色があります。このグラフはカスタマイズ可能です。特定のユーザーメトリクスを削除したり、フィルターを適用することで結果を絞り込むことができます。
プラットフォームフィルターは、登録のために収集されないため、合計および登録済みユーザーメトリクスには適用されません。
異なるメトリクスの下にある色分けされたパーセンテージは、前の期間からそのメトリクスがどのように変化したかを示します。緑の正の数値はメトリクスが増加したことを示し、赤の負の数値はメトリクスが減少したことを示します。デフォルトでは、過去7日間の比較を表示します。これは日付範囲フィルターに基づいて自動的に更新されます。
特定のユーザーメトリクスを見つける
Studioで、アナリティクス > ユーザーに移動します。
ユーザー採用グラフに移動します。
日付の上にカーソルを合わせて、正確なユーザーメトリクスを確認します。
ユーザーメトリクスの選択解除
Studioで、アナリティクス > ユーザーに移動します。
ユーザー採用グラフに移動します。
グラフに表示したくないユーザーメトリクスをクリックします。
ユーザーメトリクスはグレーアウトされ、グラフに表示されなくなります。
ユーザーメトリクスの選択を解除し、グラフから削除しました。再度グラフに含めるには、メトリクスを再度クリックしてください。
ユーザーファネル
ユーザーファネルは、プラットフォームとのユーザーのインタラクションの進行を示す理解しやすいビジュアルです。ファネルチャートは、ユーザーコンバージョンのどの段階で大幅な減少があるかを理解するのに役立ちます。メトリクスに基づいて、コンテンツ戦略を調整し、より良いユーザー体験を提供できます。
ファネルは、4つの段階の各段階でのユーザー数を示します: 合計ユーザー > 登録済みユーザー > アクティブユーザー > エンゲージメントユーザー。各ユーザータイプの上のパーセンテージは、前の段階からの比率を表示します。例えば、アクティブユーザーの26.32%がエンゲージメントユーザーでもあります。ファネルのコンバージョン率は、2つの連続する段階間の比率に基づいて計算されます。
ユーザーインタラクション
ユーザーインタラクションのドーナツグラフは、コンテンツに対するユーザーの関与方法によってエンゲージされたユーザーを分けます。
メトリクスは、ユーザーが以下をどのように使用したかに基づいています：
コメント
いいね
共有
グラフの下や異なるセクションにカーソルを合わせると、正確なデータを簡単に確認できます。
時間別アクティブユーザー
時間別ユーザーアクティビティグラフは、曜日と時間帯にわたるユーザーアクティビティを表示します。異なる青の濃淡により、平均アクティビティの高低を簡単に見分けることができます。これらのメトリクスにより、投稿の公開やメンテナンス作業の計画など、コンテンツ管理活動をより良くスケジュールできます。時間ブロックにカーソルを合わせると、その時間帯のプラットフォーム上のアクティブユーザーの正確な数が表示されます。平均データは最も近い整数に切り上げられます。
グラフは常に7日間を表示しますが、日付範囲フィルターを使用すると、選択した期間の平均アクティビティを確認できます。例えば、7日間の期間の場合、その週の各日のアクティビティデータが表示されます。7日以上の期間の場合、各時間枠のアクティビティデータが平均化されます。
現在の7日間を表示している場合、データはグラフを表示している時点で終了します。木曜日の午前10時に確認している場合、それ以降のデータは表示されません。
CSVエクスポート
ユーザータブからCSVダウンロードを通じてユーザーデータをダウンロードできます。
ユーザー時系列データのダウンロード
Studioで、アナリティクス > ユーザーに移動します。
タブの上部に移動します。
必要に応じて、データを絞り込むためのフィルターを実装します。
CSVエクスポートをクリックします。
CSVエクスポートダイアログが開きます。
ユーザー時系列CSVをクリックします。
CSVのダウンロードが自動的に始まります。
ユーザータブからデータをダウンロードしました。メトリクスは時系列の形式でダウンロードされます。
