# Q&Aページの設定

**URL**: https://support.staffbase.com/hc/ja/articles/26066094019730
**取得日**: 2026-05-02 18:48:39 JST

## 内容

システム全体の編集者
この機能はアドオンです。詳細はStaffbase 価格をご参照ください。有効化についてはStaffbaseサポートまたはカスタマーサクセスマネージャーにお問い合わせください。
Q&AプラグインはStaffbase Employee Experience (EX) Boosterプラグインです。従業員アプリやイントラネットを、質問ができ、回答され、簡単に見つけられるインタラクティブなナレッジハブに変えます。Q&Aプラグインをインストールした後、さらに設定を行い、最初のQ&Aページを公開してユーザーに表示できるようにします。
Q&Aページを設定する
Studioで、コンテンツ > Q&Aに移動します。
設定したいQ&Aページに移動します。
ページ下部のマニュアルをクリックして設定ガイドを開きます。
マニュアルを使って、ページをニーズに合わせて設定します。
保存するをクリックします。
Q&Aページの設定が完了しました。ページを公開してユーザーに表示できるようになります。
Q&Aページを公開する
Studioで、コンテンツ > Q&Aに移動します。
公開したいQ&Aページに移動します。
Q&Aを公開するをクリックします。
Q&Aページが公開されました。ユーザーに表示されるようになりました。
