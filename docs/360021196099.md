# Staffbaseプラットフォームからカスタムウィジェットをアンインストールする

**URL**: https://support.staffbase.com/hc/ja/articles/360021196099
**取得日**: 2026-05-02 18:47:32 JST

## 内容

アクティベーション
管理者
プラットフォームに以前インストールしたカスタムウィジェットをアンインストールできます。カスタムウィジェットをアンインストールすると、ニュースやページの編集者にはもはや利用できなくなります。
アンインストールしたいウィジェットを投稿やページから削除してください。ウィジェットをアンインストールすると、すでに使用している投稿やページのレイアウトが崩れる可能性があります。
スタジオで、設定 > カスタム ウィジェットに移動します。
カスタムウィジェットページが開きます。
インストール済みウィジェットバンドルの下で、アンインストールしたいカスタムウィジェットに移動します。
アンインストールアイコンをクリックします。
ウィジェットをアンインストールダイアログが開きます。
アンインストールをクリックします。
カスタムウィジェットをStaffbaseプラットフォームからアンインストールしました。これは、ウィジェットとその内容が使用されていた投稿やページから削除されたことを意味します。
