# CSVを使用したグループメンバーの管理

**URL**: https://support.staffbase.com/hc/ja/articles/16607509470354
**取得日**: 2026-05-02 19:01:06 JST

## 内容

グループ管理者
スタッフベースは、コンテンツを効果的にターゲットするためにさまざまな種類のユーザーグループを提供しています。内部手動グループおよびオープングループのユーザーを手動で管理することができます。より迅速な一括管理のために、CSVファイルをインポートしてグループ内のユーザーを追加または置き換えることができます。
CSVインポート機能は条件付きグループには利用できません。これらは設定した条件に基づいて自動的に管理されます。
前提条件:
識別子の値のみを含むCSVファイルを持っていること。
追加したいすべてのユーザーが、すでに連絡先またはユーザーとしてプラットフォームに存在していること。
Studioで、ユーザー > グループ > ユーザーを追加したいオープンまたは内部手動グループに移動します。
3つのドットをクリックして、その他のオプションメニューを開きます。
編集するをクリックします。
メンバータブの下で、CSVをインポートをクリックします。
CSVをインポートダイアログが開きます。
インポートタイプの下で、次のいずれかを選択します:
ファイルから新しいメンバーを追加する
すべてのメンバーを置き換える
CSVをインポートをクリックします。
コンピュータからCSVファイルを選択し、開くをクリックします。
グループにメンバーをインポートしました。
アップロードに失敗した場合、概要ページからエラーレポートをダウンロードできます。エラーレポートには、追加できなかったユーザーが記載されています。
