# ユーザー報告コメント

**URL**: https://support.staffbase.com/hc/ja/articles/8196495575570
**取得日**: 2026-05-02 18:47:00 JST

## 内容

エディター
コメントは、あなたのプラットフォームでのコミュニケーションの中心的な部分であり、アイデア、意見、知識を共有するのに最適な場所です。これは、あなたのStaffbaseプラットフォームにおけるインタラクションの一種であり、従業員とあなたが作成したコンテンツをつなげるのに役立ちます。
ユーザーは、コメントを報告してあなたのモデレーションチームによるレビューを受けることができます。報告者の身元を保護するために、これらのコメント報告は匿名で受け取ります。
報告されたコメントには以下が含まれます：
側面にある色付きバー
🟥 赤バー：報告されたコメント - 対応が必要
🟦 青バー：モデレーションのために非表示のコメント - 対応が必要
報告の数を表示するフラグ
アプリ/イントラネット
ユーザー報告コメントは、ニュース投稿の下のコメントスレッドに表示されます。これらはフラグでマークされており、エディターの役割以上のユーザーのみが見ることができます。このフラグは、コメントが報告された回数を示します。
報告されたコメントから直接、次の操作を行うことができます：
コメントを承認
モデレーションのために非表示
削除
報告
スタジオ
スタジオのコメント概要は、プラットフォーム全体のコメントを表示します。スタジオのコメントについて詳しく学ぶ。
コメント概要では、ユーザー報告コメントは「すべて」と「報告済み」タブにあります。さらなる調査のためにすでに非表示にされたコメントは「モデレーションのために非表示」タブにあります。削除またはゴミ箱に移動されたコメントは「削除済み」タブにあります。
スタジオから、次の操作を行うことができます：
すべてのコメントを表示
モデレーションのために報告されたコメントを非表示にする
モデレーションのために非表示にされたコメントを承認する
モデレーションのために非表示にされたコメントを削除する
すべての削除されたコメントを表示
