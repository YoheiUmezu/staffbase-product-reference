# StaffbaseへのMicrosoft 365接続の追加

**URL**: https://support.staffbase.com/hc/ja/articles/4410098659986
**取得日**: 2026-05-02 18:47:25 JST

## 内容

管理者
Microsoft 365接続を追加し、ユーザーがウィジェットや検索を通じてMicrosoftコンテンツにアクセスできるようにします。
前提条件
カスタムAzureアプリで接続を追加する場合は、AzureアプリのクライアントIDとクライアントシークレットが必要です。
スタジオで、設定 > サードパーティ接続に移動します。
Microsoft 365の下で、追加をクリックします。
Microsoft接続を追加するためのダイアログが開きます。
サインインフローに使用するMicrosoftコネクタアプリのタイプを選択します:
デフォルトアプリ
Staffbaseの既製のAzure ADアプリを使用して、プラットフォームのMicrosoft接続を迅速に設定します。
カスタムアプリ
Microsoft接続のために独自のカスタムAzureアプリを使用します。始めるには、カスタムAzureアプリのクライアントIDとシークレットが必要です。
Microsoft 365接続を介してコンテンツが表示されるユーザーをMicrosoft 365検索の可視性の下で選択します。
テストのために、最初は可視性を制限し、より大きなグループにアクセス可能にする前に確認することをお勧めします。
追加をクリックします。
Microsoft 365への接続が構成されました。
