# 必須グループ割り当てのためのグループ識別子の作成

**URL**: https://support.staffbase.com/hc/ja/articles/360010785539
**取得日**: 2026-05-02 18:44:55 JST

## 内容

必須グループを使用すると、特定のユーザーに対してコンテンツを必須に設定でき、これらのユーザーは購読解除できません。グループコンテンツを表示できるが必須としてマークされていないユーザーにとって、このグループは通常のオープングループのように機能し、希望に応じてコンテンツに購読したり購読解除したりできます。これは、たとえば、会社の透明性を促進したり、出張中の労働者に位置情報を提供したりするのに役立ちます。
グループ識別子は、グループで設定し、その後このグループにユーザーを自動的に追加するために再利用する属性です。これが必須グループの設定を開始する方法です。
前提条件
オープングループを作成していること。
スタジオで、ユーザー設定 > ユーザーグループに移動します。
オープングループタブをクリックします。
必須メンバーを追加したいオープングループに移動します。
グループを編集をクリックします。
グループを編集するためのダイアログが開きます。
識別子フィールドに、このグループのための一意の識別子を提供します。
保存をクリックします。
このグループ識別子がプロフィールに関連付けられているすべてのユーザーは、自動的にこのグループに必須メンバーとして追加されます。
