# APIトークンの管理

**URL**: https://support.staffbase.com/hc/ja/articles/25487807954322
**取得日**: 2026-05-02 18:42:06 JST

## 内容

管理者
APIトークンは、サードパーティサービスがStaffbaseプラットフォームと通信を確立することを可能にします。APIトークンを監査し、不要なものは無効化または削除することが良いプラクティスです。
APIトークンの無効化
APIトークンを無効化することで、一時的にアクセスを停止できます。
APIトークンを無効化すると、そのAPIトークンを使用しているサードパーティの統合が切断されます。進む前に、トークンが無効化できることを確認してください。
スタジオで、歯車アイコンをクリックして設定を開きます。
システムの下で、APIアクセスをクリックします。
APIトークンページが開きます。
アクションの下で、三つの点をクリックしてアクションメニューを開きます。
アクションメニューが開きます。
無効化をクリックします。
確認ダイアログが開きます。
無効化をクリックします。
APIトークンを無効化しました。いつでも再度有効化できます。
APIトークンの有効化
以前に無効化したAPIトークンを再度有効化できます。
スタジオで、歯車アイコンをクリックして設定を開きます。
システムの下で、APIアクセスをクリックします。
APIトークンページが開きます。
アクションの下で、三つの点をクリックしてアクションメニューを開きます。
アクションメニューが開きます。
有効化をクリックします。
確認ダイアログが開きます。
有効化をクリックします。
APIトークンを再度有効化しました。これで、APIトークンは統合に使用できます。
APIトークンの削除
プラットフォームからAPIトークンを永久に削除できます。
APIトークンを削除すると、そのAPIトークンを使用しているサードパーティの統合が切断されます。進む前に、統合が不要であることを確認するか、新しいAPIトークンに置き換えてください。
スタジオで、歯車アイコンをクリックして設定を開きます。
システムの下で、APIアクセスをクリックします。
APIトークンページが開きます。
アクションの下で、三つの点をクリックしてアクションメニューを開きます。
アクションメニューが開きます。
削除をクリックします。
確認ダイアログが開きます。
削除をクリックします。
APIトークンを永久に削除しました。
