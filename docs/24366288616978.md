# Staffbase Emailのスタジオへのアクセス

**URL**: https://support.staffbase.com/hc/ja/articles/24366288616978
**取得日**: 2026-05-02 18:42:45 JST

## 内容

Staffbase EmailをStaffbase IntranetまたはEmployee Appと一緒に使用する場合は、スタジオへのアクセスを参照してください。以下の情報は、Staffbase Emailがスタンドアロン製品として使用される場合にのみ適用されます。
スタジオユーザーである場合、Staffbase Emailにアカウントを登録するための指示が記載されたメールが届きます。アカウントを登録した後、スタジオにアクセスできます。
ウェブブラウザーで、Staffbase EmailのドメインURLを入力します。例えば、https://mycompany.com
サインインページが開きます。
組織の認証方法（SSOや資格情報など）を使用してサインインします。
スタジオが開きます。
スタジオに正常にアクセスしました。表示および使用できる機能や機能は、あなたの権限と役割に依存します。
