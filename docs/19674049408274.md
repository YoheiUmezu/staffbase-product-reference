# スマートキャンペーンの概要

**URL**: https://support.staffbase.com/hc/ja/articles/19674049408274
**取得日**: 2026-05-02 18:43:12 JST

## 内容

アクティベーション
システム全体のエディター
スマートキャンペーンは、戦略的なコンテンツ目標を設定、追跡、分析する方法です。ゼロからコンテンツ戦略を作成するのは daunting ですが、スマートキャンペーンを使えば簡単に始められます。すでに戦略がある場合やデータを統合する方法を探している場合、スマートキャンペーンはコンテンツとメトリクスへのアクセスを1つの簡単なダッシュボードにまとめます。詳細については、Staffbaseサポートまたはカスタマーサクセスマネージャーにお問い合わせください。
スマートキャンペーンの利点
コンテンツを戦略に合わせ、スマートキャンペーンを使用してコミュニケーションがビジネス目標に与える影響を理解します。
戦略を立てる
コンテンツチームのために戦略的目標を明確かつアクセス可能にします。作成するキャンペーンごとにキャンペーン目標を追加できます。この目標は、キャンペーンの概要および詳細ビューに表示されるため、システム全体のエディターや管理者はコンテンツの管理や評価を行う際に常に念頭に置くことができます。
タイムラインを提供する
各キャンペーンに開始日と終了日を追加することで、キャンペーン計画の透明性を高めます。これらの日付はキャンペーンの詳細ビューに表示されます。キャンペーンの概要では、ステータスラベルがキャンペーンが今後のもの、アクティブなもの、または完了したものかを示します。
コンテンツを作成および管理する
投稿の編集タイムラインを一目で確認できます。この便利な概要を超えて、キャンペーンの詳細ビューから新しい投稿を直接作成したり、ドラフトを編集したりできます。各割り当てられた投稿の公開ステータスを示すラベルにより、公開済み、スケジュール済み、またはパイプラインにある投稿の数をすぐに確認できます。
分析する
キャンペーンは、マクロおよびマイクロレベルでの分析を可能にします。キャンペーン内のどの投稿が最も良い結果を出したか、また、より広いスケールでどのキャンペーンが読者の間で最もエンゲージメントを生んでいるかを並べて見ることができます。各投稿およびキャンペーン全体のいいね、コメント、訪問数などの統計がすぐに利用可能です。
キャンペーンステータスの説明
キャンペーンステータスは、次に実施されるキャンペーン、現在実施中のキャンペーン、またはすでに完了したキャンペーンを追跡するのに役立ちます。キャンペーンを作成し、開始日と終了日を割り当てると、ステータスが自動的に割り当てられます。
アクティブ: キャンペーンは現在実施中です。
今後: キャンペーンは将来に予定されており、まだ開始されていません。
完了: キャンペーンは終了しました。
