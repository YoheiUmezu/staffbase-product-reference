# 高度な編集カレンダーの概要

**URL**: https://support.staffbase.com/hc/ja/articles/20176517041554
**取得日**: 2026-05-02 18:43:09 JST

## 内容

アクティベーション
エディター
編集カレンダーは、エディターがアクセスできるスペースやチャネルにおける計画されたコンテンツを一目で確認できるようにします。これにより、エディターはコンテンツを視覚化し、特定の日に計画されているコンテンツが多すぎるかどうかを確認することが容易になります。その後、リーチを最大化するためにコンテンツを再配分できます。編集カレンダーは、エディターに圧倒されることなく包括的な概要を提供します。
編集カレンダーから直接、便利に以下の操作ができます：
ニュース投稿の追加、編集、スケジュール設定
ブロッカーの追加
キャンペーンの表示
ニュース投稿、キャンペーン、ブロッカーの最も重要な詳細をフライアウトで表示
フライアウトからニュース投稿およびキャンペーンの詳細に移動
フライアウトでブロッカーの詳細を編集
ニュース投稿、キャンペーン、ブロッカーの月次概要を表示
キャンペーン、スペース、チャネルでコンテンツをフィルタリング
クイック分析を表示
デフォルトでは、エディターはアクセスできるスペースやチャネルのニュースを表示および編集できます。ただし、オプションのフル透明モードを使用すると、すべてのスペースおよびチャネルのコンテンツがカレンダーに表示され、表示専用形式になります。フル透明モードの詳細を学ぶ。
