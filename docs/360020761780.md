# Microsoft 365 タスクウィジェットの設定

**URL**: https://support.staffbase.com/hc/ja/articles/360020761780
**取得日**: 2026-05-02 18:56:03 JST

## 内容

アクティベーション
システム全体のエディター
Microsoft 365 タスクウィジェットを使用すると、Staffbase プラットフォームのページやニュース投稿に直接 Microsoft Planner のタスクを表示できます。このウィジェットは、開始されていないタスクや進行中のタスクを含むすべてのタスクの概要をユーザーに提供するために、3つのタブを表示します。さらに、ウィジェット内の検索およびフィルター機能により、ユーザーは探しているタスクを迅速に見つけることができます。
前提条件
あなたは Microsoft 365 接続を Staffbase に追加しました。
あなたは アプリまたはイントラネットから Microsoft 365 にサインインしました。
あなたは Microsoft 365 タスクウィジェットをアクティブにしました。
Microsoft 365 タスクウィジェットを追加します。
ウィジェットを設定するためのダイアログが開きます。
オプションで、タイトルを提供します。
タイトルはウィジェットの上に表示され、ユーザーに見えるようになります。
ウィジェットを表示できるデバイスタイプを、対応するアイコンをクリックして選択します。
アプリ内の可視性の下で、ウィジェットを表示したいユーザーを選択します。
ウィジェットに表示するタスクの数を選択します。この数のタスクが自動的にウィジェットに表示されます。さらにタスクはスクロールすることで読み込まれます。
OKをクリックします。
ウィジェットがページに追加されます。
変更を保存します。
Microsoft 365 タスクウィジェットを設定して追加しました。
ユーザーが Microsoft 365 タスクウィジェットを表示する方法
ウィジェットにアクセスする際、ユーザーはまだサインインしていない場合、従業員アプリまたはイントラネットでMicrosoft 365 アカウントにサインインするように促されます。
