# コレクションの使用

**URL**: https://support.staffbase.com/hc/ja/articles/18566030385170
**取得日**: 2026-05-02 18:55:30 JST

## 内容

コレクションを使用して、トピックや目的別にファイルを整理します。各ファイルにホームを与え、誰がアクセスできるかを決定します。コレクションを自分だけのものにするか、次の相手と共有します:
選択したユーザー: プロジェクトやキャンペーンで複数の貢献者と作業しながら、メディアファイルの整理と共有の課題に取り組みます。
組織: 組織のすべてのユーザーとコレクションを共有できます。スタジオユーザーは、これらの事前承認されたコレクションから選択し、コンテンツで使用できます。たとえば、会社のロゴや画像などです。
コレクションにアクセスするには:
マイファイル
ニュースやページのコンテンツを作成する際に、簡単にナビゲートしファイルを再利用するために、マイファイルでコレクションを見つけることができます。ここでは:
コレクションを表示
共有されたコレクションを表示
コレクションにファイルをアップロード
これらのコレクションからファイルを再利用
ファイルマネージャー
ファイルとコレクションを効率的に管理するために、ファイルマネージャーにアクセスします。次のことができます:
コレクションを表示
ファイルを表示または管理するための適切な権限でコレクションを作成し、共有
新しいファイルや既にアップロードされたファイルをコレクションに追加
コレクション内のファイルを置き換え
コレクションからファイルを削除
コレクションを削除
コレクションでさらに多くのことを行う:
コレクションの作成と共有
コンテンツクリエイターとして、組織のニーズに基づいてコレクションを作成し、共有権限を設定できます。たとえば、プロジェクトやキャンペーンなどです。
スタジオで、ファイルに移動します。
ファイルマネージャーページが開きます。
コレクションをクリックします。
コレクションの作成と共有ダイアログが開きます。
コレクションの名前を追加します。
表示の下で、コレクションを表示できる人を選択します:
なし: あなたのみがコレクションを表示できます。
すべてのユーザー: すべてのスタジオユーザーがコレクションを表示できます。
選択したユーザーまたはユーザーグループ: 選択したユーザーのみがコレクションを表示できます。
管理の下で、コレクションを管理できる人を選択します:
なし: あなたのみがコレクションを管理できます。
すべてのユーザー: すべてのスタジオユーザーがコレクションを管理できます。
選択したユーザーまたはユーザーグループ: あなたのチームの特定のメンバーおよび/または特定のグループのみがコレクションを管理できます。
管理者は、表示および管理の共有権限でなしが選択されている場合でも、中央の場所からすべてのコレクションを表示および管理できます。
共有はスタジオ内のユーザーにのみ適用され、アプリユーザーの可視性やアクセス権限には影響しません。
作成をクリックします。
コレクションが正常に作成されました。
コレクションにファイルを追加する
コレクションを作成し、共有権限を設定した後、次のものを追加できます:
新しいファイル
スタジオで、ファイルに移動します。
ファイルマネージャーページが開きます。
コレクションには、マイファイルからもアクセスできます。
ファイルを追加したいコレクションをダブルクリックします。
コレクションが開きます。
アップロードをクリックします。
コンピューターからアップロードしたいファイルを選択します。
開くをクリックします。
コレクションにファイルを追加しました。
使用済みファイル
ファイルマネージャーに以前アップロードされたファイルをコレクションに追加できます。
スタジオで、ファイルに移動します。
ファイルマネージャーページが開きます。
すべてのファイルから、コレクションに追加したい画像を選択します。
コレクションに追加をクリックします。
選択したファイルを追加したいコレクションを選択します。
追加をクリックします。
コレクションにファイルを追加しました。
コレクションの編集
コレクションの名前を変更したり、アクセス権限を変更する必要がある場合は、設定を編集できます。
スタジオで、ファイルに移動します。
ファイルマネージャーページが開きます。
更新したいコレクションの隣にあるその他のオプションアイコンをクリックします。
共有をクリックします。
コレクションを共有ダイアログが開きます。
必要な編集を行います。
保存をクリックします。
コレクションを正常に編集しました。
コレクションからファイルを削除する コレクションに戻って、もはやそこに属さないファイルを削除することができます。 スタジオで、ファイルに移動します。
ファイルマネージャーページが開きます。 ファイルがあるコレクションをダブルクリックします。
コレクションが開きます。
削除したいファイルを選択します。 コレクションから削除をクリックします。
確認ダイアログが開き、選択したファイルとそれが使用されている場所の数が表示されます。
選択内容を確認した後、削除をクリックします。 コレクションからファイルを正常に削除しました。 コレクションからファイルを削除しても、すべてのファイルや追加された他のコレクションではまだ利用可能です。 コレクションを削除する もはや必要のないコレクションを削除できます。コレクション内のファイルは、ファイルマネージャーのファイルセクションで引き続き利用可能です。 スタジオで、ファイルに移動します。
ファイルマネージャーページが開きます。 すべてのコレクションをクリックします。 削除したいコレクションを選択します。
コレクションを削除をクリックします。
コレクションを削除ダイアログが開きます。
ダイアログには、選択したコレクションの数と、それが使用されている場所の数が表示されます。
選択内容を確認した後、削除をクリックします。
コレクションを正常に削除しました。コレクションは、あなたやコレクションを共有したコラボレーターには表示されなくなります。
コレクションを削除しても、このコレクション内のファイルは、すべてのファイルや他のコレクションで引き続き利用可能です。
