# 記事チャンネルのニュース投稿記事のレイアウトを選択する

**URL**: https://support.staffbase.com/hc/ja/articles/33774973050002
**取得日**: 2026-05-02 18:54:48 JST

## 内容

編集責任者
ニュースは、スタッフベースアプリとイントラネットで視聴者ごとに動的なコンテンツを公開できます。ニュースは、ニュースページ、チャンネル、投稿を作成するの3層構造で構成されています。ニュースチャンネルには、記事、更新、メディアのタイプがあります。記事チャンネルでは、ニュースフィード内の記事の表示レイアウトを3種類から選択できます。それぞれのレイアウトは異なる用途に対応しています。
拡張
拡張ニュースフィードレイアウトでは、メインの記事画像が大きく表示されます。これにより、各記事が目立ち、ユーザーが個々の投稿を作成するに引き付けられます。
標準
標準ニュースフィードレイアウトでは、最新の記事画像が大きく表示され、それ以前の記事はコンパクトに表示されます。このレイアウトは、最新の投稿を作成するを優先したい場合に適しています。
コンパクト
コンパクトニュースフィードレイアウトでは、すべての記事がコンパクトに一覧表示されます。フィード内のすべての投稿を作成するに均等に注目が集まります。
記事のレイアウトを選択する
Studioでコンテンツ > ニュースに移動します。
レイアウトを設定したい記事チャンネルを開きます。
歯車アイコンをクリックして設定を開きます。
設定ページが開きます。
レイアウトに移動します。
レイアウトオプションを選択します。
保存するをクリックします。
ニュースフィード内の記事投稿を作成するのレイアウトを選択しました。設定はいつでも更新することができます。
