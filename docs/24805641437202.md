# スタッフベースメールをエディトリアルカレンダーで使用する

**URL**: https://support.staffbase.com/hc/ja/articles/24805641437202
**取得日**: 2026-05-02 19:01:02 JST

## 内容

エディター
マルチチャネルエディトリアルカレンダーは、異なるスタッフベースチャネル間の透明性を提供し、真にシームレスな体験を実現します。ニュースのためのすべての素晴らしい機能に加えて、エディトリアルカレンダーを使用することで、エディターに内部コミュニケーションコンテンツの一貫した概要を提供できます。従業員メールが従業員アプリまたはイントラネットのスタジオで利用可能な場合、エディターは自分のスケジュールされたメールや送信済みのメールをすぐに確認し、エディトリアルカレンダー内で新しいメールを直接作成することもできます。
エディトリアルカレンダーから、便利に以下の操作ができます：
新しいメールを作成する
自分のスケジュールされたメールと送信済みのメールを表示する
メールのステータスを確認する
メールデザイナーに移動する
メールが送信された後にメール統計と詳細にアクセスする
エディトリアルカレンダーでのメール作成
スタジオで、プランニング > エディトリアルカレンダーに移動します。
作成 > メールをクリックします。
内部タイトル、フォルダー、オプションでターゲットグループとキャンペーンを入力します。
次へをクリックします。
メールを作成する方法を選択の下で、オプションを選択します。
テンプレートを使用を選択した場合は、テンプレートを選択します。
次へをクリックします。
メールデザイナーが開きます。
メールを作成する。
メールを作成しました。これで、エディトリアルカレンダー内のすべてのエディターに表示されます。
エディトリアルカレンダーからメールデザイナーに移動する
スケジュールされたがまだ送信されていないメールを編集するために、メールデザイナーに移動できます。メールが送信されると、エディトリアルカレンダーからは開けなくなります。
エディトリアルカレンダーで、開きたいスケジュールされたメールに移動します。
メールを選択します。
フライアウトが開きます。
メールのタイトルをクリックします。
メールデザイナーが新しいタブで開きます。
メールデザイナーを開き、送信前にスケジュールされたメールを編集できます。
エディトリアルカレンダーからメール統計と詳細にアクセスする
メールが送信された後、エディトリアルカレンダーから直接メール統計と詳細のダッシュボードを開くことができます。
統計タブには以下の指標が含まれます：
受信者
開封数
クリック数
メールファunnel
トップリンク
総活動の分布
ユニーク活動の進行状況
詳細タブには：
送信日
ターゲットオーディエンス
送信者アドレス
送信者名
返信オプション
キャンペーン
件名
メールのプレビュー
エディトリアルカレンダーで、統計または詳細を表示したいメールに移動します。
メールを選択します。
フライアウトが開きます。
メールのタイトルをクリックします。
メールの統計が新しいタブで開きます。
詳細タブをクリックして、メールの詳細を表示します。
メールの統計と詳細にアクセスし、指標を確認できます。
