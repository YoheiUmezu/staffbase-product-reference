# 通知の設定

**URL**: https://support.staffbase.com/hc/ja/articles/207179685
**取得日**: 2026-05-02 18:45:18 JST

## 内容

お知らせは、新しいコンテンツが利用可能になったときに通知します。モバイルデバイスのプッシュ通知を有効または無効にすることができます。プッシュ通知に加えて、新しいコンテンツについてメールでお知らせを受け取ることも選択できます。
メール通知の有効化または無効化のオプションは、登録時にログイン用メールアドレスを設定した場合のみ、設定内で利用できます。
利用可能な機能によって、次の内容のお知らせを受け取ることができます：
新しい投稿
あなたの投稿へのコメントやコメントへの返信
@メンションによる直接的な記述
新しいチャットメッセージ
新しい給与明細
新しいコミュニティメッセージ、コメント、または返信
SMSによるクリティカルなお知らせ
お知らせを設定する
アプリ
アプリ内でメニューアイコンをタップします。
メニューが開きます。
歯車アイコンをタップして設定を開きます。
設定ページが開きます。
お知らせをタップします。
設定したいお知らせまで移動します。
トグルをスライドしてお知らせを有効または無効にします。
イントラネット
イントラネット内でプロフィールメニューに移動します。
設定をクリックします。
設定ページが開きます。
お知らせをクリックします。
設定したいお知らせまで移動します。
トグルをスライドしてお知らせを有効または無効にします。
お知らせの設定が完了しました。変更内容は自動的に保存されます。
