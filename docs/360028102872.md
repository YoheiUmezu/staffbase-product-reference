# 投稿の設定をカスタマイズする

**URL**: https://support.staffbase.com/hc/ja/articles/360028102872
**取得日**: 2026-05-02 19:02:35 JST

## 内容

編集者
ニュースは、スタッフベースアプリとイントラネットで、視聴者ごとに動的なコンテンツを公開できます。これは、ニュースページ、チャンネル、投稿を作成するの3層構造で構成されています。
ニュースチャンネルでは、編集責任者が新しく作成されるすべてのニュース投稿記事のデフォルト設定を決定します。各投稿記事ごとに、有効にした設定を個別にカスタマイズできます。これにより、公開やその他のオプションを各投稿記事ごとに調整できます。
投稿記事でカスタマイズできる設定：
出版のお知らせ
コメント
いいね
既読確認
ハイライト
ゴーストライティング
投稿記事の設定をカスタマイズする
投稿記事のオプションセクションには、さまざまなトグルがあります。トグルが右側にある場合は有効、左側にある場合は無効です。
Studio
Studioでコンテンツ > ニュース > 設定をカスタマイズしたいニュースチャンネルと投稿記事に移動します。
公開サイドパネルのオプションで、選択したオプションの横にあるトグルを調整します。
下書きを保存または保存をクリックします。
ニュース投稿記事の個別設定をカスタマイズしました。設定はいつでも調整できます。
アプリ/イントラネット
アプリまたはイントラネットで、チャンネルをカスタマイズしたいニュース投稿記事に移動します。
詳細設定をクリックします。
選択したオプションの横にあるトグルを調整します。
公開または保存をクリックします。
ニュース投稿記事の個別設定をカスタマイズしました。設定はいつでも調整できます。
