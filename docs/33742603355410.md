# ニュースチャンネルに投稿者または編集者を割り当てる

**URL**: https://support.staffbase.com/hc/ja/articles/33742603355410
**取得日**: 2026-05-02 19:00:03 JST

## 内容

編集責任者
ニュースは、スタッフベースアプリとイントラネットで視聴者ごとに動的なコンテンツを公開できます。これは、ニュースページ、チャンネル、投稿を作成するの3層構造で構成されています。チャンネルの設定で、ユーザーやユーザーグループにコンテンツ固有の権限を付与できます。これにより、チャンネル内でコンテンツを作成・編集できます。チャンネルの設定で作成者や編集者を設定できます。チャンネル内のすべての投稿は同じ作成者と編集者を持ちます。
編集者は、チャンネル内のすべての投稿を作成、編集、管理できる上位の役割です。投稿者（作成者）は自分のコンテンツのみ作成・編集できます。他の投稿者の投稿を編集または削除することはできません。
ニュースチャンネルに投稿者を割り当てる
Studioでコンテンツ > ニュースに移動します。
編集者または投稿者を割り当てたいチャンネルを開きます。
歯車アイコンをクリックして設定を開きます。
設定ページが開きます。
投稿者のドロップダウンから選択したユーザー、グループ、またはAPIトークンを選択します。
個別のユーザー、グループ、またはAPIトークンを検索します。
保存するをクリックします。
ニュースチャンネルに投稿者を割り当てました。投稿者はすぐにコンテンツを作成できます。投稿者はいつでも調整できます。
ニュースチャンネルに編集者を割り当てる
Studioでコンテンツ > ニュースに移動します。
編集者または投稿者を割り当てたいチャンネルを開きます。
歯車アイコンをクリックして設定を開きます。
設定ページが開きます。
個別のユーザー、グループ、またはAPIトークンを検索します。
保存するをクリックします。
ニュースチャンネルに編集者を割り当てました。編集者はすぐにチャンネル内のすべてのコンテンツを作成・管理できます。編集者はいつでも調整できます。
