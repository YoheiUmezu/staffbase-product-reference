# ショートウィジェットの設定方法

**URL**: https://support.staffbase.com/hc/ja/articles/31026117928466
**取得日**: 2026-05-02 18:41:03 JST

## 内容

編集者
ショート動画は、メディアニュースチャネルの投稿を基にした動画コンテンツを表示する魅力的な方法です。ショート動画は、プラットフォームに親しみやすいソーシャルメディア風の体験をもたらします。ページ用ショート動画ウィジェットを使うと、メディアニュースチャネルを基に、アプリやイントラネット内でこれらの動画を表示できます。
ウィジェットから公開済み・未公開のニュースチャネルの両方を選択できます。未公開のニュースチャネルに接続されたウィジェットはページに表示されません。これにより、チャンネルをLIVEにする前に複数のショート動画を作成・アップロードできます。
Staffbaseでは、ショート動画専用のメディアチャネルまたはニュースページを作成することを推奨しています。これにより、この特定の形式に合わせて設定やコンテンツを調整し、動画が正しい形式で表示されるようにできます。
前提条件：
ショート動画用メディアチャネルの編集者であること。
ウィジェットを追加したいページの編集者であること。
ページにショート動画ウィジェットを追加する。
プラグイン ウィジェットを設定するダイアログが開く。
ウィジェットを表示できるデバイスのタイプを選択する。
アプリ内での表示の下で、ウィジェットを表示したいユーザーを選択する。
ドロップダウンからウィジェットのコンテンツソースとしてニュースページまたはメディアチャネルを選択する。
保存するをクリックする。
ショート動画ウィジェットがページに追加されました。メディアチャネル内の既存の動画は自動的に表示されます。または、ショート動画を作成することもできます。
