# スペースのメニューのエクスポート

**URL**: https://support.staffbase.com/hc/ja/articles/17890519215506
**取得日**: 2026-05-02 19:00:47 JST

## 内容

アクティベーション
エディター
アプリとイントラネットのメニュー構造をCSVファイルとしてエクスポートできます。エクスポートは、特定のスペース内のメニュー項目またはすべてのスペースを一度に対象にすることができます。メニューをエクスポートすることで、モバイルおよびウェブアプリのメニュー構造を包括的に把握するのに特に役立ちます。
さらに、モバイルアプリで使用されているクイックリンクを確認することもできます。この機能について詳しくは、Staffbaseサポートまたはカスタマーサクセスマネージャーにお問い合わせください。
単一スペースのメニューのエクスポート
スタジオで、コンテンツ > システム > メニューに移動します。
メニューをエクスポートしたいスペースを選択します。
メニューが開きます。
CSVとしてエクスポートをクリックします。
CSVとしてエクスポートしますか？ダイアログが開きます。
エクスポートをクリックします。
メニューをCSVとして正常にエクスポートしました。ファイルはダウンロードフォルダーにあります。
すべてのスペースのメニューのエクスポート
スタジオで、コンテンツ > システム > メニューに移動します。
スペースのドロップダウンからすべてのコンテンツを選択します。
メニュー構造のエクスポートの下で、CSVとしてエクスポートをクリックします。
メニュー構造をCSVとして正常にエクスポートしました。ファイルはダウンロードフォルダーにあります。
