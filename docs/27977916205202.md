# 調査結果の概要

**URL**: https://support.staffbase.com/hc/ja/articles/27977916205202
**取得日**: 2026-05-02 18:55:00 JST

## 内容

エディター
調査結果は、各質問の詳細な内訳と、個別の調査における時間経過に伴う結果の概要を提供します。結果は、回答が提出されるとリアルタイムで更新されます。スタジオの調査プラグインで結果を表示できます。
調査を機密に保つため、調査結果は最低5件のユーザー提出があった後にのみ表示されます。同様に、各質問の調査結果は、各質問に対して5件の回答提出があった後にのみ表示されます。これにより、個々の回答が匿名のままとなり、ユーザーのプライバシーが保護されます。
調査結果の理解
調査結果は、スタジオの調査プラグインで特定の調査に対して利用可能です。結果は機密であり、個々の回答には識別情報は表示されません。結果には、調査の開始日と終了日、質問の数、参加者数が含まれます。調査の回答は平均化され、棒グラフで詳細な内訳が表示されます。
調査が定期的に行われる場合、これまでに何回のラウンドが行われたかを確認し、特定のラウンドの結果を表示することもできます。また、時間経過に伴う結果を折れ線グラフで確認することもできます。
結果フィルターの使用
調査結果は、各調査に対してフィルターを使用してソートできます。フィルタリングは、データを絞り込み、特定し、必要に応じて分析するのに役立ちます。フィルターはプロファイルフィールドに基づいてカスタマイズ可能で、最初にアプリの設定セクションで有効化する必要があります。複数のフィルターカテゴリを組み合わせて、結果をさらに絞り込むことができます。
結果の共有
従業員が自分の回答を提出した後に、調査結果の平均値と詳細な内訳を確認できるオプションを提供することで、エンゲージメントと透明性を高めます。回答は機密のままであり、ユーザーによるフィルタリングはできません。この設定は、各調査ごとに個別に有効化およびカスタマイズでき、いつでもオンまたはオフにできます。結果は、調査が公開されている間のみユーザーに表示されます。
