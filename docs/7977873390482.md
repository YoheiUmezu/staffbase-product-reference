# チャットメッセージの引用

**URL**: https://support.staffbase.com/hc/ja/articles/7977873390482
**取得日**: 2026-05-02 18:51:41 JST

## 内容

時には、返信する頃には会話の中でメッセージが埋もれてしまうことがあります。スレッド内で返信したい正確なチャットメッセージを引用することで、あなたの返信に適切な文脈と背景を提供します。
モバイルアプリ
チャットに移動します。
編集したい会話を開き、引用したいメッセージに移動します。
チャットメッセージをタップして長押しします。
アクションメニューが開きます。
メッセージに返信をタップします。
返信メッセージを入力します。
元のチャットメッセージがあなたの返信に引用されます。
紙飛行機のアイコンをタップして送信します。
チャットメッセージを引用して返信しました。
ウェブアプリ
チャットに移動します。
編集したい会話を開き、引用したいメッセージに移動します。
矢印アイコンをクリックします。
返信メッセージを入力します。
元のチャットメッセージがあなたの返信に引用されます。
紙飛行機のアイコンをクリックして送信します。
チャットメッセージを引用して返信しました。
