# Staffbase PnP モダン検索拡張ライブラリの構成

**URL**: https://support.staffbase.com/hc/ja/articles/11182594050450
**取得日**: 2026-05-02 18:46:54 JST

## 内容

アクティベーション
Staffbase PnP モダン検索拡張ライブラリを使用すると、PnP モダン検索 Web パーツを利用して SharePoint で Staffbase ニュースを検索できます。詳細を学ぶ。Staffbase PnP モダン検索拡張ライブラリは、設定が必要な拡張機能です。
この記事の内容
以下の手順で Staffbase PnP モダン拡張ライブラリを設定する方法を学びます：
Staffbase PnP モダン検索拡張ライブラリ拡張をインストールする
PnP 検索結果 Web パーツをページに追加する
PnP 検索結果 Web パーツを設定して Staffbase ニュースを追加する
Staffbase ニュースで PnP 検索結果 Web パーツを設定する
前提条件
Studio から Staffbase PnP モダン検索拡張ライブラリ拡張パッケージをダウンロードしている。
シングルサインオン (SSO) を設定して、Microsoft Entra ID (旧 Azure Active Directory) を使用して Staffbase と SharePoint の間でユーザーを認証している。
Microsoft SharePoint Online に次のオプションがある：
アプリカタログが設定された SharePoint 環境。
編集権限のある SharePoint のサイトコレクション。
Staffbase は、SharePoint のコミュニケーションサイトの使用を推奨しています。
テナントのサイトコレクションアプリカタログに対する貢献権限。
モダンページが SharePoint にある。
PnP モダン検索 4.8.0 Web パーツがサイトアプリカタログにダウンロードされ、インストールされている。
Staffbase PnP モダン検索拡張ライブラリのインストール
前提条件
Staffbase PnP モダン検索拡張ライブラリを設定したい SharePoint サイトに PnP モダン検索 Web パーツをインストールし、設定している。
SharePoint 管理センター > その他の機能に移動します。
その他の機能ページが開きます。
アプリの下で、開くをクリックします。
アプリカタログ > SharePoint 用アプリに移動します。
アップロードをクリックし、Staffbase PnP モダン検索拡張ライブラリファイルを選択します。
OKをクリックします。
デプロイの確認ダイアログが開きます。
オプションで、このソリューションを組織内のすべてのサイトで利用可能にするのチェックボックスを選択します。
このオプションが選択されていない場合、Staffbase PnP モダン検索拡張ライブラリはサイトアプリカタログでのみ利用可能になります。さらに、サイトアプリカタログに手動で追加する必要があります。Staffbase PnP モダン検索拡張ライブラリをサイトアプリカタログに追加する方法については、SharePoint サイトアプリカタログにアプリを追加するを参照してください。
デプロイをクリックします。
PnP 検索結果 Web パーツの追加
前提条件：
Web パーツを追加する権限を持つ SharePoint サイトメンバー以上の役割を持っている。
Staffbase PnP モダン検索拡張ライブラリを SharePoint サイトにインストールしている。
Staffbase PnP モダン検索拡張ライブラリをインストールした SharePoint サイトの任意のページに移動します。
ページの右上にある 編集 をクリックします。
ページの上にマウスをホバーします。
新しい Web パーツを追加するための + アイコンが開きます。
+ アイコンをクリックします。
利用可能な Web パーツのダイアログが開きます。
PnP 検索結果を検索して選択します。
最適な表示のために、Staffbase は Web パーツを1カラムセクションに追加することを推奨しています。
設定ダイアログが開きます。
PnP 検索結果をページに追加しました。次に、Web パーツを設定して Staffbase ニュースを追加することができます。
Staffbase ニュースを追加するための PnP 検索結果 Web パーツの設定
これで、PnP 検索結果 Web パーツを設定して Staffbase ニュースを追加できます。
PnP 検索結果 Web パーツを追加した後に開く設定ダイアログで、設定をクリックします。
PnP - 検索結果ダイアログが開きます。
次へをクリックします。
拡張機能の設定の下で、設定をクリックします。
拡張ライブラリを設定するためのダイアログが開きます。
名前/目的フィールドに「Staffbase ニュース」と入力します。
マニフェスト GUIDフィールドに「3b164641-fa65-4279-9cce-c8981bb32f03」と入力します。
有効/無効のトグルを オン にします。
追加して保存をクリックします。
保存をクリックしてページを保存します。
Staffbase ニュースを追加するために PnP 検索結果 Web パーツを設定しました。
Staffbase NewsをデータソースとしてPnP検索結果Webパーツを構成する
これで、Staffbase NewsをデータソースとしてPnP検索結果Webパーツを構成できます。
以前にPnP検索結果Webパーツを構成したSharePointサイトのページに移動します。
ページの右上にある編集をクリックします。
以前に追加したPnP検索結果Webパーツの編集をクリックします。
PnP - 検索結果ダイアログが開きます。
Staffbase Newsを選択します。
Staffbase web-app URLフィールドに、Staffbase webアプリのURLを入力します。
次へをクリックします。
レイアウトとしてStaffbaseニュースカードを選択します。
保存をクリックしてページを保存します。
ページが保存され、PnP検索結果WebパーツがStaffbase Newsで正常に構成されました。構成されたWebパーツにStaffbase Newsが表示されます。
