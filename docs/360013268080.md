# 分析データのCSVへのエクスポート

**URL**: https://support.staffbase.com/hc/ja/articles/360013268080
**取得日**: 2026-05-02 19:02:05 JST

## 内容

分析ダッシュボードは貴重な指標を提供します。エクスポートしたデータを使用して、編集者や著者と共有するレポートを作成し、コンテンツの成功を評価できます。管理者はフィルターを使用して、組織のさまざまなメンバー向けに異なるデータセットを作成できます。これらのフィルターにより、ユーザーインターフェイスで現在表示されているデータのみを抽出できます。
データをCSVにエクスポートする
スタジオで、分析に移動します。
目的のタブに移動し、必要に応じてフィルターを適用します。
CSVエクスポートをクリックしてダウンロードを完了します。
タブの指標をCSV経由でダウンロードしました。新しい指標をダウンロードするために、いつでもフィルターを追加または削除できます。
