# Navigator にサービスナウ フォームを追加する

**URL**: https://support.staffbase.com/hc/ja/articles/34808607485202
**取得日**: 2026-05-02 18:48:18 JST

## 内容

アクティベーション
管理者
AI
Navigator の ServiceNow 連携により、ユーザーは ServiceNow サービスへアクセスし、チケットを作成できます。検索、チケットの表示、チケット作成を 1 つのシンプルな体験にまとめています。従業員はどの言語でも音声またはテキストで質問でき、ServiceNow ナレッジベース、ServiceNow カタログ、ServiceNow チケットの許可に基づいてインスタントな回答を受け取ることができます。
Navigator の管理アクセスを持つユーザーは、管理者の役割がなくてもフォームを設定できます。
Studio で コンテンツ > システム > Navigator に移動します。
ナレッジ タブに移動します。
ServiceNow の下で、フォーム追加 をクリックします。
ServiceNow 追加ダイアログが開きます。
名前 に、フォームの名前を入力します。
フォーム URL に、ユーザーがリダイレクトされる URL を入力します。
手順 で、Navigator がフォームをどのように使用するかを定義します。
保存する をクリックします。
フォームが追加されました。これでユーザーは、あなたが用意したフォームを使ってチケットを作成できます。フォームはいくつでも追加できます。また、いつでも編集や削除が可能です。
