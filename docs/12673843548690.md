# プロフィールに画像を追加する

**URL**: https://support.staffbase.com/hc/ja/articles/12673843548690
**取得日**: 2026-05-02 18:51:31 JST

## 内容

プロフィールに画像を追加することで、ページ訪問者との視覚的なつながりが生まれ、彼らがあなたを認識し、より良くつながることが容易になります。
ユーザープロフィールにアップロードできる画像は2種類あります：
アバター画像：ユーザープロフィールに自分の写真をアップロードすることで、他の人があなたを簡単に認識できるようになります。似た名前の他の人と区別され、投稿やコメントを追加した際の可視性も向上します。アバター画像はプロフィールだけでなく、コメントやStaffbase Studioでも表示されます。
ヘッダー画像：ユーザープロフィールにヘッダー画像をアップロードすることで、プロフィールをパーソナライズし、目立たせることができます。また、あなたの個性、興味、またはストーリーを伝えるのにも役立ちます。
ヘッダー画像には固定の解像度はなく、ブラウザウィンドウのサイズやディスプレイサイズによって異なる場合があります。

画像が推奨サイズに合わない場合、自動的に最適なサイズにリサイズされます。アスペクト比は保持され、追加で画像をトリミングすることもできます。
アバター画像の追加
Webアプリまたはモバイルで以下の手順を使用してアバター画像を追加します。
Webアプリ
プロフィールメニューを開きます。
自分の名前をクリックします。
プロフィールが開きます。
プロフィールを編集をクリックします。
編集モードが開きます。
アバター画像の下にあるアップロードをクリックします。
デバイスから画像を選択します。
アバター画像のフォーカスポイントを設定し、アップロードをクリックします。
アバター画像があなたの名前の上に表示されます。
変更を保存をクリックします。
ユーザープロフィールにアバター画像を追加しました。
モバイルアプリ
アプリメニューを開きます。
自分の名前をタップします。
プロフィールが開きます。
編集をタップします。
編集モードが開きます。
アップロードをタップします。
デバイスからアバター画像を選択します。
ヘッダー画像のフォーカスポイントを設定し、アップロードをタップします。
アバター画像があなたの名前の上に表示されます。
保存をタップします。
ユーザープロフィールにアバター画像を追加しました。
ヘッダー画像の追加
Webアプリまたはモバイルで以下の手順を使用してヘッダー画像を追加します。
Webアプリ
プロフィールメニューを開きます。
自分の名前をクリックします。
プロフィールが開きます。
プロフィールを編集をクリックします。
編集モードが開きます。
ヘッダー画像をアップロードをクリックします。
デバイスからヘッダー画像を選択します。
ヘッダー画像のフォーカスポイントを設定し、アップロードをクリックします。
ヘッダー画像がプロフィール画像の上に表示されます。
変更を保存をクリックします。
ユーザープロフィールにヘッダー画像を追加しました。
モバイルアプリ
アプリメニューを開きます。
自分の名前をタップします。
プロフィールが開きます。
編集をタップします。
編集モードが開きます。
アップロードをタップします。
デバイスからヘッダー画像を選択します。
ヘッダー画像のフォーカスポイントを設定し、アップロードをタップします。
ヘッダー画像がプロフィール画像の上に表示されます。
