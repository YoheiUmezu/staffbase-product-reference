# メールデザインにカスタムブロックを構築する

**URL**: https://support.staffbase.com/hc/ja/articles/4412592726418
**取得日**: 2026-05-02 18:47:15 JST

## 内容

メールデザインにカスタマイズされたブロックを追加します。カスタムブロックは、特定のデザイン要件を満たすために調整されたレイアウトやデザインを構築するための優れた方法です。
この記事の内容
新しいカスタムブロックレイアウトを作成する
既存のブロックレイアウトをカスタマイズする
新しいカスタムブロックレイアウトを作成する
ブロックレイアウトは、異なるコンテンツ（テキスト、画像、動画など）を要素の形で挿入できる列のフレームです。ブロックレイアウトは、メールデザイナーの画面左側にあります。
希望のブロックレイアウトを見つけたら、それをデザイン内の希望の位置にドラッグ＆ドロップして追加できます。
既存のブロックレイアウトをカスタマイズする
ブロックをカスタマイズして、オーディエンスを引き付けます。コンテンツを最も意味のある方法で配置できるように、ブロックをカスタマイズできます。
要素を追加
希望のブロックレイアウトをデザインに追加したら、画面左側の要素セクションから要素を追加できます。
要素は、デザインに追加できるさまざまな種類のコンテンツです。これには、テキスト、画像、動画、ボタン、アンケート、スペーサー、ソーシャルリンクが含まれます。
希望の要素をクリックして、デザインにドラッグ＆ドロップします。
行を追加
希望の要素をデザインに追加した後、ブロックレイアウトに行を追加できます。行は、同じ列内に複数の要素を作成して追加する方法です。
たとえば、各列に画像要素、テキスト要素、ボタンを持つ3つの列を作成したい場合があります。
列に行を追加するには、希望の要素を選択し、列内の既存の要素の上にドラッグします。
要素をクローン
要素は、クローン機能を使用して列に自動的に追加できます。これにより、要素が自動的にクローンされ、現在の要素の下の行に挿入されます。
クローン機能は、クローンしたい要素をクリックし、要素の右上に表示されるアイコンをクリックすることで使用できます。これにより、要素がクローンされ、現在の要素の下に挿入されます。
要素を削除
削除機能は、行内の要素を削除します。これは、要素をクリックし、要素の右上に表示されるアイコンをクリックすることで使用できます。これにより、選択した要素が削除されます。
削除された要素の下に要素がある場合、それらは1行上に移動し、ブロックの最上行に要素が配置されます。
