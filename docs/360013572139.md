# デスクトップおよびモバイルデバイスのカスタムメニューの設定

**URL**: https://support.staffbase.com/hc/ja/articles/360013572139
**取得日**: 2026-05-02 18:59:55 JST

## 内容

アクティベーション
システム全体のエディター
従業員アプリは、デスクトップおよびモバイルデバイスで使用される際に異なるユースケースに対応します。メニュー項目の表示設定を使用することで、これらのユースケースに対応する異なるメニューを作成することが可能です：
モバイルデバイスでは、小さな画面のため、コンテンツに迅速にアクセスするためのシンプルで短いメニュー構造が必要です。より深いレベルのコンテンツには、検索や直接リンクを介してアクセスできます。
デスクトップデバイスでは、従業員アプリがフロントドアイントラネットとして機能し、より深いレベルのコンテンツにアクセスできます。大きなデバイス画面では、フライアウトを含む完全なコンテンツメニューにより、ユーザーはコンテンツをより早く見つけることができます。
メニュー項目の表示設定を変更しても、コンテンツへのアクセスには影響しません。このメニュー項目のコンテンツは、検索や直接リンクを介して引き続きアクセス可能です。
メニュー項目へのナビゲーション
メニュー項目の表示設定は、以下の方法で構成できます：
メニュー設定
スタジオで、メニューに移動します。
表示設定を調整したいメニュー項目のギアアイコンをクリックします。
メニュー項目を編集するためのダイアログが開きます。
特定のニュース、ページ、またはプラグインの設定
スタジオで、表示設定を調整したいニュース、ページ、またはプラグインに移動します。
ギアアイコンをクリックします。
設定を編集するためのページが開きます。
メニュー項目の表示設定
設定を編集するページまたはダイアログで、メニューに表示の下に、メニュー項目を表示するチェックボックスを選択します。
クリックして OK または 保存します。
ユーザーが異なるメニューを表示する方法
デスクトップのメニュー：
メニュー項目 Partnerlink
は表示設定されています
モバイルのメニュー：
メニュー項目 Partnerlink
は非表示に設定されています
