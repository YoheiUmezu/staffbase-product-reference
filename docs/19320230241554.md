# ニュース投稿での@メンションの使用

**URL**: https://support.staffbase.com/hc/ja/articles/19320230241554
**取得日**: 2026-05-02 19:03:05 JST

## 内容

貢献者
@メンションは、従業員アプリやイントラネットでユーザーをタグ付けし、彼らの注意を引く方法です。@メンションは、Staffbaseプラットフォームの複数の場所で利用可能です。ニュースでは、投稿の中で直接ユーザーをタグ付けできます。たとえば、大きなプロジェクトを成功裏に完了したチームをタグ付けしたり、大きな勤続年数を祝う同僚を称賛したりできます。
@記号の後に名前を入力することで、同僚に通知を送ることができます。入力を続けると、マッチのリストがドロップダウンに表示され、自動的に絞り込まれます。ドロップダウンには、投稿に適切なアクセス権を持つユーザーのみが表示されます。投稿を公開した後、@メンションをクリックすると、そのユーザーのプロフィールに移動します。
タグ付けされた同僚は、通知センターで通知を受け取ります。また、プッシュ通知が有効になっている場合は、モバイルデバイスにプッシュ通知も届きます。公開された投稿にメンションが追加された場合や、投稿の著者をメンションした場合には通知は送信されません。
タグ付けされたユーザーの名前は、アプリやイントラネットの公開エリアには表示されません。その場合、名前はアスタリスク*で匿名化されます。たとえば、@JohnSmithは@***に匿名化されます。
ユーザーを@メンションする
従業員アプリまたはイントラネットでニュース投稿に移動します。
コンテンツエディタで、@記号とタグ付けしたいユーザーの名前を入力します。
タグ付けしたいユーザーを選択します。
あなたは@メンションを使用して従業員をタグ付けしました。投稿を公開すると、そのタグはユーザーの通知またはモバイルデバイスのプッシュ通知で表示されます。投稿を読んでいるユーザーは、そのタグをクリックしてその人のユーザープロフィールに移動できます。
