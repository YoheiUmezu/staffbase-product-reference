# 分析ダッシュボードの概要

**URL**: https://support.staffbase.com/hc/ja/articles/360013327939
**取得日**: 2026-05-02 18:47:45 JST

## 内容

アナリティクスダッシュボードは、ユーザーのエンゲージメントやコンテンツの成功を測定するための強力なツールです。Studioのアナリティクスからアクセスでき、追加の設定や構成は不要で、常に有用なデータを利用できます。Staffbaseは、さまざまなカテゴリーごとに幅広い分析データを提供しており、詳細なメトリクスを深く掘り下げることができます。
従業員アプリとイントラネットで利用可：
ニュース
ページ
コンテンツ
チャット
ユーザー
ナビゲーター
ハッシュタグ
検索する
キャンペーン
従業員アプリ、イントラネット、メールの組み合わせ、またはメール単体で利用可：
キャンペーン
メール
各トピックには、それぞれ独自のメトリクスやフィルターを持つタブがあります。このようにして、Staffbaseはあなたの企業やさまざまな関係者のニーズに合わせて統計を簡単にフィルターできます。アプリ全体のパフォーマンスから詳細なエンゲージメントまで、アナリティクスはコンテンツに関する意思決定をサポートする確かなデータを提供します。ダッシュボード内のすべてのデータはGDPRに準拠しており、集計された形で保存されます。そのため、個々のユーザーに関する情報は表示されません。
権限と役割
分析ダッシュボードは、追加の設定なしに、ユーザーの役割に応じて細かい権限を自動的に使用します。管理者、管理エディター、エディターを含むStaffbaseプラットフォームへのシステム全体のアクセス権を持つユーザーは、すべての4つのタブを見ることができます。
スペース管理者とコンテンツエディターは、特定のスペースまたはコンテンツのニュースおよびコンテンツタブを見ることができます。これらの権限により、各役割はスタジオから必要な正確なデータを表示およびダウンロードできます。
