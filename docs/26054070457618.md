# Q&Aプラグインのインストール

**URL**: https://support.staffbase.com/hc/ja/articles/26054070457618
**取得日**: 2026-05-02 18:45:55 JST

## 内容

この機能はアドオンです。詳細はStaffbaseの価格をご参照ください。有効化についてはStaffbaseサポートまたはカスタマーサクセスマネージャーにお問い合わせください。
Q&Aプラグインは、Staffbase従業員エクスペリエンス（EX）ブースタープラグインです。
従業員アプリやイントラネットを、質問ができ、回答され、簡単に見つけられるインタラクティブなナレッジハブに変えます。
前提条件
APIトークンを作成し、読み取り専用アクセス権を持っていること。
APIトークンを暗号化してsupport@staffbase.comに送信したこと。
APIトークンは、ユーザーAPIを通じてユーザープロフィール情報にアクセスするために必要です。APIトークンにより、プラグインはユーザープロフィール情報（プロフィール画像や権限など）にアクセスできます。これにより、認可されたユーザーのみが質問を投稿でき、他のユーザーは新しい質問が共有された際に通知を受け取ることができます。
Studioで、コンテンツ > コンテンツメニューに移動します。
メニューの一番下までスクロールし、プラグインを追加するをクリックします。
プラグインページが開きます。
Q&Aに移動します。
インストールをクリックします。
Q&Aを追加するダイアログが開きます。
Q&Aページのタイトルを入力します。
タイトルはプラグイン設定でいつでも変更できます。
表示ドロップダウンメニューから、Q&Aページを表示するユーザーを選択します。
Q&Aを追加するをクリックします。
Q&Aプラグインをインストールし、Q&Aページを追加しました。プラグインは左側のコンテンツセクションに表示されます。ページを設定して公開することで、ユーザーに表示できるようになります。
