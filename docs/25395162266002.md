# スタッフベースサポートポータルの概要

**URL**: https://support.staffbase.com/hc/ja/articles/25395162266002
**取得日**: 2026-05-02 18:42:30 JST

## 内容

サポートポータルは、Staffbase のすべての製品に対する体系的なガイダンスと迅速なトラブルシューティングを提供します。ステップバイステップのガイド、更新、有効、ベストプラクティスなどへのアクセスを確保します。直感的なナビゲーション、強力な検索、戦略的に配置されたコンテンツで、必要な情報を簡単に見つけることができます。また、キャンパスや開発者ポータルへの直接リンクを通じて、Staffbase の追加の有効化リソースにアクセスし、包括的な学習とサポートを受けることができます。

ホームページナビゲーション
トップナビゲーション
サポートポータルのトップナビゲーションは、重要なコンテンツへのクイックリンクを提供します。
ホーム: サポートポータルのどこからでもすぐにホームページに戻ります。
製品: 主な製品エリア、つまりアプリ/イントラネットとStaffbaseメールを探索します。
新着情報: 四半期ごとおよび隔週でリリースされた最近の製品開発を確認します。
言語: ドロップダウンから利用可能な言語を選択します。
高度な検索
外部リンク
特定の用語や機能がすでに頭にある場合は、
検索を使用して直接探しているものを入力してください。
また、質問を入力して、クイックアンサーを通じて問題に関連する指示を直接受け取ることもできます。
提供されたリンクを通じてキャンパスコースや開発者ポータルを探索し、製品の包括的な理解を深めてください。
外部リンクアイコン  は、このリンクがサポートポータルの外部にナビゲートすることを示します。
システムステータス
サービスの中断がまれに発生した場合、信号機のインジケーターがシステムステータスを通知します。
直感的なナビゲーション
スタートガイド、機能概要、統合サポートの直接エントリーポイントにより、目的のコンテンツをより早く見つけることができます。
ユースケース: 内部コミュニケーションを強化するためのアイデアや機能を提供します。各ユースケースには、ステップバイステップの指示、ウォークスルー、ベストプラクティス、追加のリソースを含むエンドツーエンドのガイドが含まれており、Staffbaseを使用して成功裏に実装するのに役立ちます。
スタートガイド: プラットフォームのセットアップ、主要機能の構成、スムーズな稼働のための基本的な手順を案内します。
トピック: Staffbaseの機能を包括的にリストアップし、専用のガイドとFAQを備えた簡単にナビゲートできる形式で探索できます。
統合: カスタム統合のためのすべての重要な情報を提供し、Microsoft 365統合などの他のサードパーティ統合を中央ハブで提供します。
複数セクションのカテゴリ化により、関連する記事が複数のセクションに表示され、期待通りの場所でトピックを見つけることができます。
たとえば、「アンケートのユースケースとベストプラクティス」は次の両方にあります:
スタートガイド > 製品オリエンテーション > ユースケースとベストプラクティス
機能 > コンテンツ > アンケートプラグイン
この複数の配置により、必要な情報を見つけやすくなります。
Staffbaseの新着情報
小規模な更新: 機能の改善、バグ修正、その他の最新の更新情報を確認します。
四半期リリース: リリースノートで四半期ごとのリリースについて学びます。
API更新: 開発者ポータルで最新のAPI開発情報を入手します。
動画セクションでは、製品ニュースと機能のハイライトを紹介しています。
情報とヘルプ
トレンドトピック: このスペースでトレンドになっているものを発見し、コミュニケーションを強化するために活用できるものを見つけます。
お問い合わせ: サポートチームに連絡してください。私たちはあなたのためにここにいます。
デモを予約する
サポートポータルで見たものが気に入ったが、まだ製品を使用していない場合は、コミュニケーションを効果的に管理するためのデモを予約してください。
クイックアンサー
クイックアンサーは、サポートポータルでの質問に対して即座にAIによる回答を提供します。検索バーに質問を入力するだけで、クイックアンサーがステップバイステップのガイダンスを提案し、最も関連性の高い記事を指し示します。短いまたは不成功な検索はクイックアンサーをトリガーしません。フィードバックを提供して、回答の質を継続的に向上させることもできます。
検索結果ページ
サポートポータルで検索すると、サポートポータルと開発者ポータルの両方からのコンテンツが結果に含まれます。検索結果ページでは、ソースでフィルタリングして、最も関連性の高い情報をすばやく見つけることができます。
強化されたナビゲーション
直感的な情報アーキテクチャとナビゲーションにより、サポートポータル内で常に自分の位置を把握できます。
カテゴリとセクションナビゲーション
記事内ナビゲーション
記事の上部にあるパンくずリストに加えて、左側のナビゲーションは、記事を読んでいる間にどのカテゴリにいるかを明確に表示します。左側のナビゲーションでカテゴリを選択すると、すべてのセクションとその記事のリストが表示されます。特定のトピックに関する利用可能な情報を包括的に把握するために、カテゴリ内の複数のセクションを探索してください。
記事内では、左側のメニューが現在のカテゴリを表示してナビゲーションを構造化し、右側のパネルがセクション内の他の記事の概要を表示します。記事の下部にある2つのボタンで、セクション内の前後の記事に移動し、コンテンツの流れを途切れさせません。関連する記事間のクロスリンクに加え、キャンパスコースや開発者ポータルのコンテンツが、関連する有効化リソースに接続することで学習体験を向上させます。

サポートポータルを使用するためのベストプラクティス
ユーザージャーニーに従う
はじめにでは、次のことを探求します:
製品オリエンテーションで、製品と利用可能なサポートリソースに慣れ親しむ。
プラットフォーム設定で、管理者が設定の旅をナビゲートするためのステップバイステップガイダンス。
コンテンツ管理で、プラグイン、ウィジェット、コンテンツターゲティングなどの機能を使用してプラットフォームコンテンツを整理する方法を理解する。
ユーザー管理で、サインイン方法、ユーザーグループ、ユーザーアクセスと組織を効率化するための権限について学ぶ。
Staffbaseメール専用設定で、Staffbaseメール製品について学ぶ。
トピックでは、各機能を詳細に探求し、包括的な理解を得る。
統合では、Microsoft 365、Box、Confluence、その他のStaffbaseから利用可能な統合の可能性を発見する。
検索とナビゲーションを使用する
検索機能を使用して、必要な情報を簡単に見つける。
検索バーに質問を入力して、クイックアンサーからステップバイステップの指示をすぐに受け取る。
記事のパンくずリストとサイドナビゲーションを使用して効率的にナビゲートする。
ブックマークとフォロー
頻繁に使用する記事をブックマークして簡単に参照する。
製品リリースセクションをフォローして、主要または小規模なリリースがあるたびにメールを受け取ることで最新情報を入手する。
サポートを改善するためのフィードバックを提供する
StudioでStaffbaseプラットフォームに関するアイデアを共有する。アイデアポータルの使用について詳しく学ぶ。
チケットを作成することで、製品に関するフィードバックを共有する。
記事の下部にあるはいまたはいいえボタンをクリックして、コンテンツが役立ったかどうかを知らせ、改善に役立てる。
