# ページのコンテンツガバナンスの概要

**URL**: https://support.staffbase.com/hc/ja/articles/32399064803858
**取得日**: 2026-05-02 18:57:47 JST

## 内容

有効化
編集者
AI
コンテンツ ガバナンスは、編集者がページ プラグイン内で公開済みページを大規模に管理できるようにサポートします。ページ概要の隣にある専用タブでは、公開済みページを継続的に監視し、古い、未使用、または構造が不十分なコンテンツなどの問題を特定します。これらのインサイトにより、チームはガバナンスの実践を改善し、ページ関連のコンテンツ負債を減らすために必要な情報を得ることができます。概要は、成長するイントラネットの拡張や、チームが変化する組織のニーズに適応するのを支援します。
コンテンツ ガバナンス タブでは、主要なメトリクスが一目で確認でき、リストにはあなたがアクセスできる各ページのタイトル、問題、閲覧数、所有者が表示されます。
各ページのアクションメニューから、問題の詳細と推奨されるアクションを表示できます。また、編集、公開ステータスの調整、リマインダーの変更も可能です。コンテンツ ガバナンスにより、ページ管理がよりスマートになり、手動レビューへの依存が減ります。
ページに以下の状態がある場合、問題が表示されます：
1年以上更新されていない
ページのリマインダーが期限切れ
過去3か月間に閲覧数がゼロ
公開されていない、または削除された従業員アプリやイントラネット コンテンツへの無効なリンク
無効なユーザーにリンクされたユーザーウィジェット
スペース内の他のページと重複するコンテンツ（AIによる検出）
スペース内の他のページと矛盾するコンテンツ（AIによる検出）
