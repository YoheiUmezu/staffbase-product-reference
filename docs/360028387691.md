# チャンネルの公開解除

**URL**: https://support.staffbase.com/hc/ja/articles/360028387691
**取得日**: 2026-05-02 19:02:12 JST

## 内容

すでに公開されているチャンネルを削除するか、公開解除することができます。公開解除を行うことで、後で再度公開することが可能です。
スタジオで、公開解除したいチャンネルに移動します。
設定を開くためにギアアイコンをクリックします。
チャンネルの設定ページが開きます。
チャンネルの公開解除をクリックします。
このチャンネル内で公開されていたすべての投稿は、アプリ内で表示されなくなります。
