# SMSの概要

## 内容

ベータ
アクティベーション
管理者
Staffbase SMSは、重要な更新情報や安全警告をモバイルフォンのテキストメッセージとして迅速に配信することを可能にします。信頼性の高いコミュニケーション、高い開封率、簡単なアクセスを確保し、分散した労働力への全体的なコミュニケーションを改善します。SMS機能は、Studioから直接SMSでデスクレスおよびフロントラインの従業員に到達するために、Twilioのメッセージングプラットフォームと直接接続します。
Staffbase SMSの利点
信頼性の高い即時コミュニケーション
従業員を情報で常に更新するために、時間に敏感なメッセージを迅速に配信します。緊急時や時間に敏感な状況では、毎瞬が重要です。SMS統合により、重要な更新情報や安全警告を従業員に即座に配信できます。重要な情報はリアルタイムで伝達され、安全でつながりのある職場を促進します。
広範なリーチ
地理的な場所や職務に関係なく、メッセージを効率的に配信でき、包括性を促進します。
高い開封率
労働力の大部分がメッセージを確認し、全体的な応答性を向上させます。
Twilioの活用
Twilioは、従業員に世界中で到達できるグローバルサーバーネットワークを提供します。Twilioは非常にスケーラブルで、幅広いSMSトラフィックを処理する能力があります。
メッセージを送信する前に考慮すべきこと
アメリカ合衆国へのメッセージ: アメリカ合衆国にSMSメッセージを送信する予定がある場合、特定の登録が必要です: アプリケーション・トゥ・パーソン (A2P) 登録。A2Pはメッセージの配信を改善し、メッセージのフィルタリングやブロックを防ぎます。Twilioの設定時に登録できます。 詳細を学ぶ
メッセージのセグメンテーション: 1 SMSは1セグメントに相当し、標準テキストでは最大160文字を許可します。絵文字や中国語のようなUnicode文字は、セグメントごとの制限を70文字に減少させます。この制限を超えると、メッセージは複数のセグメントに分割されます。詳細を学ぶ
従業員を追加する際に考慮すべきこと
従業員の同意: テキストメッセージを送信する前に、従業員からの同意を得ることを確認してください。詳細を学ぶ
手動でインポートしたデータ: 既存の従業員の電話番号やメールアドレス、名前などのデータをインポートします。
ユーザー提供のデータ: 従業員がユーザープロファイルに電話番号を追加するよう促し、シームレスなコミュニケーション体験を提供します。
ユーザーの電話番号形式: 形式はE.164でなければなりません。これは国際的に認識された標準です。詳細を学ぶ
