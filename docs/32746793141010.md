# ページ用コンテンツデザイナーの概要

**URL**: https://support.staffbase.com/hc/ja/articles/32746793141010
**取得日**: 2026-05-02 18:54:50 JST

## 内容

ページ用コンテンツデザイナーを使うと、直感的なWYSIWYG（見たまま編集）体験で、モダンで柔軟なページを作成・管理できます。編集者で見える内容は、ページが公開された後に従業員が見る内容とほぼ同じです。
コンテンツデザイナーは、構造化されたページレイアウトと直感的な編集者、サイドパネルツールを組み合わせています。これにより、ページ作成がより速くなり、レイアウトの一貫性を保つことができます。
トップメニュー
コンテンツデザイナーのトップメニューでは、重要なページアクションに素早くアクセスできます。コンテンツの編集、設定の管理、公開のコントロールを一箇所で行えます。
タイトルと概要を編集：ページタイトルと概要はいつでも更新できます。
言語を選択する：多言語ページ用に言語バージョンを追加または削除します。
編集者 と 設定 の切り替え：編集とページ設定（言語、表示、編集者アクセス、その他のページレベル設定）の管理を切り替えます。
プレビュー：公開前にデスクトップ、タブレット、モバイルでページをプレビューできます。
保存する：下書きや公開済みページの作業中に変更内容を保存します。
公開 / 更新する：新規ページの公開や、公開済みページへの変更を従業員に反映する前に適用できます。
アクションメニュー：三点メニューから追加のページレベルアクションにアクセスできます：
ページを開く
ページのバージョンを表示・復元する
ページを公開または非公開にする
ページリンクをコピーする
スペース内でページを移動する
ページを削除する
編集者
編集者は、ページを作成・構成するためのメインキャンバスです。ビルディングブロックを使った本格的なWYSIWYG体験で、編集内容が最終的なページレイアウトに反映されます。ページはセクション、コンテナ、コラムで構成され、明確で柔軟なレイアウトを作成できます。この構造化されたアプローチにより、きれいで予測可能なレイアウトを柔軟に作成できます。
サイドパネル
サイドパネルでは、ページ作成中に必要なツールへ編集者を離れずに素早くアクセスできます。
サイドパネルでできること：
コンテンツを追加：ブロックを使ってテキスト、メディア、ニュース、ボタン、その他の要素を追加します。ブロックはコンテナやコラムにクリックまたはドラッグで直接追加できます。
テンプレートを使用する：あらかじめデザインされたセクションレイアウトを素早くページに追加できます。セクションテンプレートは、コンテナやブロックを含む構造を提供し、ゼロから作成せずに一貫性のあるセクションを構築できます。テンプレートを挿入後、コンテンツをカスタマイズできます。
ページをナビゲーション：ナビゲーションを使って、セクション、コンテナ、ブロック間を移動します。ページ全体の構造が表示されます。
メディアを管理：マイファイルを使って画像、動画、その他の対応メディアファイルをアップロード・管理・コピーできます。マイファイルからページへ直接ドラッグできます。
元に戻す・やり直す 変更：矢印アイコンで編集内容を簡単に元に戻したりやり直したりできます。
キーボードで作業：テキスト編集、ナビゲーション、選択などのキーボードショートカットを確認し、マウスなしで効率的に作業できます。
ガイドを利用：本のアイコンを使って、コンテンツデザイナーのツアーやウォークスルーを利用できます。
フィードバックを送信：星付きの吹き出しアイコンからコンテンツデザイナーへのフィードバックを送信できます。フォームが開き、良かった点や改善案を共有できます。あなたのフィードバックは編集体験の向上や今後のアップデートに役立ちます。
以下のステップは、コンテンツデザイナーでページを作成・公開する一般的な流れです。いくつかのステップは後で実施したり、不要な場合はスキップできます。
ページを作成する
タイトルと概要を更新する
ページを構築する
追加コンテンツ言語を追加する
ページ設定を更新する
ページ作成をよりスマートかつ効率的にするには、ページ作成を高速化するツールを活用してください。
コンテンツデザイナーのメリット
直感的なコンテンツ作成
テキスト、メディア、ニュースなどのドラッグ＆ドロップブロックを使って、コンテンツを追加・アレンジできます。ツールを切り替えたりワークフローを中断せずに、リッチなページを構築できます。
ターゲット対象のコンテンツ表示
表示ルールを適用して、特定のコンテンツを誰が閲覧できるかをコントロールできます。1つのページから異なる視聴者向けにページを調整しながら管理できます。
マルチデバイスプレビュー
編集者内でデスクトップ、タブレット、モバイル向けにページをプレビューできます。公開前にレイアウトやコンテンツを確認できます。
アクセシビリティサポート
これらの機能はアクセシビリティのベストプラクティスをサポートしますが、ページがアクセシビリティやコンプライアンス要件を満たすかどうかは編集者の責任です。
