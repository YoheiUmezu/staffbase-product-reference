# CSVファイルのサンプル例

**URL**: https://support.staffbase.com/hc/ja/articles/360007108391
**取得日**: 2026-05-02 18:57:36 JST

## 内容

CSV同期でユーザーデータを管理する予定の場合、最初のステップのひとつはCSVファイルの設定です。CSVファイルやユーザー管理での利用方法についてさらに詳しく知りたい場合は、このセクションの他の記事もご覧ください。
添付のCSVサンプルをダウンロードして、ご自身のCSVファイル作成の出発点としてご利用いただけます。ここではいくつかの異なる例を紹介しています。ご利用のオンボーディング方法に合ったものを選択してください：

メールによるオンボーディングの例
アクセスコードによるオンボーディングの例
