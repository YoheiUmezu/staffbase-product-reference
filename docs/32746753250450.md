# タスクの概要

**URL**: https://support.staffbase.com/hc/ja/articles/32746753250450
**取得日**: 2026-05-02 18:57:45 JST

## 内容

アクティベーション
タスクは、あなたの組織内で作業を整理し、割り当て、追跡する方法を一元化します。Staffbaseプラットフォーム内でプロジェクト、リスト、タスクを管理するためのシンプルで柔軟な方法を提供します。日常業務、支店のアクティビティ、プロジェクトベースの作業などに利用できます。上司はチームの進捗状況を明確に把握できます。
タスクは、プロジェクト、リスト、タスクという明確な構造に従っています。この構造により、あらゆる規模の作業を計画し、追跡しやすくなります。
プロジェクト：関連する作業を整理し、どのユーザーまたはユーザーグループがアクセスできるかを管理します。
リスト：プロジェクト内の作業を分類します（例：在庫タスクや日常業務など）。
タスク：個々のアクションアイテムを割り当て、スケジュールし、優先順位を付け、完了としてマークします。
各プロジェクトはスペースに属し、選択されたユーザーまたはユーザーグループに公開できます。1つのプロジェクトには最大50個のリストを含めることができ、1つのリストには最大1,000個のタスクを含めることができます。プロジェクトは、誰が閲覧できるかやプラットフォーム内での表示場所など、チームのニーズに合わせてカスタマイズ可能です。公開されると、割り当てられたスペース内のユーザーがプロジェクトを利用できるようになります。
フィルタ：
タスクは以下の条件でフィルターできます：
今日：このリストには、その日に割り当てられたすべてのタスクが表示されます。
あなたのタスク：このリストには、あなたに割り当てられたタスクが表示されます。
リスト：
デフォルトのTo-doリストに加えて、ユーザーは自分のタスクをグループ化するカスタムリストを作成できます。
管理者および編集責任者はStudioでプロジェクトを作成・編集でき、すべてのユーザーは自分のアプリまたはイントラネットでリストやタスクを管理します。
添付：
タスク内で、ユーザーは画像、動画、PDFなどの添付ファイルを追加して、より良いコンテキストを提供できます。
タスクプラグインのメリット
作業の一元管理
すべてのリストとタスクを1か所で管理できます。複数のツールを使う必要がなくなり、全員の足並みが揃います。
チームコラボレーションの効率化
チームやスペース内でタスクを割り当て、追跡し、完了できます。明確な担当により、責任と透明性が向上します。
可視性と進捗の追跡
上司やチームメンバーに、タスクの進捗状況をリアルタイムで提供します。これにより、優先事項やボトルネックを早期に特定できます。
柔軟で拡張性のある管理
日々のチェックリストから大規模なプロジェクトまで、幅広いユースケースに対応し、整理された直感的な構造を維持できます。
