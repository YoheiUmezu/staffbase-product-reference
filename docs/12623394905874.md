# マイファイルからのインサイトの表示

**URL**: https://support.staffbase.com/hc/ja/articles/12623394905874
**取得日**: 2026-05-02 18:49:22 JST

## 内容

マイファイルビューおよび中央ファイルマネージャーからビデオインサイトを表示できます。ファイルマネージャーについて詳しく学びましょう。
中央ファイルマネージャーからのインサイトの表示
管理者
プラットフォームにアップロードされたビデオのインサイトを表示して、最も魅力的なビデオを分析します。
スタジオで、コンテンツ > ファイルマネージャーに移動します。
ファイルマネージャーが開きます。
ビデオのオプションをさらに開くために三点リーダーをクリックします。
ドロップダウンメニューが開きます。
インサイトをクリックします。
ファイルインサイトダイアログが開きます。メタデータとメトリクスが表示されます。(?)アイコンにカーソルを合わせると、取得されたメタデータとメトリクスの簡単な説明が表示されます。
Xをクリックして閉じます。
ページ内からビデオインサイトを表示しました。
マイファイルからのインサイトの表示
エディター
プラットフォームにアップロードしたビデオコンテンツのインサイトを表示します。
スタジオビュー
スタジオで、ページまたは投稿に移動してマイファイルにアクセスします。
コンテンツエディターで、メディアアイコンをクリックしてメニューを開きます。
マイファイルをクリックします。
ビデオのオプションをさらに開くために三点リーダーをクリックします。
ドロップダウンメニューが開きます。
ファイル インサイトをクリックします。
ファイル インサイトダイアログが開きます。メタデータとメトリクスが表示されます。(?)アイコンにカーソルを合わせると、取得されたメタデータとメトリクスの簡単な説明が表示されます。
Xをクリックして閉じます。
あなたはマイファイルからビデオのインサイトを表示しました。
ユーザービュー
ユーザービューで、任意のニュース投稿またはページに移動します。
編集をクリックしてコンテンツエディターを開きます。
コンテンツエディターで、メディアアイコンをクリックしてメニューを開きます。
マイファイルをクリックします。
ビデオのオプションをさらに開くために三点リーダーをクリックします。
ドロップダウンメニューが開きます。
インサイトをクリックします。
ファイル インサイトダイアログが開きます。メタデータとメトリクスが表示されます。(?)アイコンにカーソルを合わせると、取得されたメタデータとメトリクスの簡単な説明が表示されます。
Xをクリックして閉じます。
あなたはマイファイルからビデオのインサイトを表示しました。
