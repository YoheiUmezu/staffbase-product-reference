# プロファイルフィールドのユーザー参照入力タイプの概要

**URL**: https://support.staffbase.com/hc/ja/articles/13982885359122
**取得日**: 2026-05-02 18:46:50 JST

## 内容

管理者
プロファイルフィールドはユーザーに関する情報を保存します。ユーザー参照はプロファイルフィールドで使用される入力タイプの一つです。異なる入力フィールドについて詳しく学びましょう。
ユーザー参照入力タイプは、プロファイルフィールドに参照されたユーザーの名前を表示することを可能にします。このフィールドはユーザープロファイル内でハイパーリンクとして表示され、クリックすると参照されたユーザーのプロファイルページに移動します。あなたのプラットフォームには、システムプロファイルフィールドとしてユーザー参照が含まれています。このフィールドは、ユーザープロファイルページにユーザーのマネージャーを参照して表示するのに最適です。
このフィールドを組織図と併用することで、ユーザーが組織の階層内でどこに位置しているかを表示できます。これにより、ユーザーのマネージャー、部下（いる場合）、および同じチームの同じレベルの同僚を確認できます。
特定の組織では、ユーザーが2人のマネージャーを持つことがあります - 部門マネージャーとプロジェクトマネージャーです。プロジェクトマネージャーがいる場合、部下は点線で報告することがあります。両方のマネージャーを表示したい場合は、ユーザー参照入力タイプを使用してカスタムプロファイルフィールドを作成し、ユーザープロファイルに2人目のマネージャーを表示できます。
カスタムプロファイルフィールドを使用して2人目のマネージャーを追加しても、組織図には影響しません。
ユーザーデータがユーザー参照入力タイプフィールドに同期される方法
ユーザー参照入力タイプを使用するには、次のいずれかのユーザー管理方法を使用してデータを同期する必要があります：
SCIM
CSVインポート
ユーザーAPI
データの設定を手伝ってもらうために、HR部門とIT管理者に連絡する必要があります。
ユーザーにとってのユーザー参照入力タイプを持つプロファイルフィールドの見え方
ユーザープロファイルページを開くと、ユーザー参照入力タイプを持つプロファイルフィールドはハイパーリンクとして表示されます。以下の例では、ユーザーのマネージャーが表示されています。ハイパーリンクをクリックすると、マネージャーのユーザープロファイルページに移動します。
