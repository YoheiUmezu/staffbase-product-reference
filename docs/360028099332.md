# ニュースページの削除

**URL**: https://support.staffbase.com/hc/ja/articles/360028099332
**取得日**: 2026-05-02 19:03:07 JST

## 内容

アプリ内でニュースページが不要になった場合は、それを削除できます。ただし、ニュースページは削除するために空である必要があります。チャンネルがまだ含まれているニュースページは削除できません。
スタジオで、コンテンツに移動します。
メニューのシステムセクションの下で、メニューをクリックします。
削除したいニュースページに移動します。
メニュー項目の概要では、次のアイコンでニュースページを識別できます:
削除したいニュースページに対応するギアアイコンをクリックします。
ニュースページの編集ダイアログが開きます。
ニュースページを削除をクリックします。
このニュースページに関連付けられたチャンネルがまだある場合、それらのチャンネルのリストがダイアログに表示されます。これらのチャンネルが別のニュースページに割り当てられるまで、ニュースページを削除することはできません。
保存をクリックします。
