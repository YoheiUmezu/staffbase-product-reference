# イントラネットからBoxアカウントにサインインする

**URL**: https://support.staffbase.com/hc/ja/articles/20499826724242
**取得日**: 2026-05-02 18:51:25 JST

## 内容

アクティベーション
イントラネットからBoxアカウントにサインインして、Boxのコンテンツを検索およびアクセスできます。
イントラネットで、プロフィールメニューに移動します。
設定 > アカウントに移動します。
接続されたアカウントの下で、Boxのサインインをクリックします。
Boxのサインイン画面にリダイレクトされます。
Boxのサインイン画面で、メールアドレスとパスワードを入力します。
承認をクリックします。
Boxへのアクセスを許可をクリックして、StaffbaseがあなたのBoxアカウントに接続するための権限を付与します。
イントラネットからBoxアカウントにサインインしました。サインアウトをクリックすることで、同じセクションからBoxアカウントからサインアウトできます。
また、従業員アプリの Box 検索タブまたはイントラネットの検索機能から直接Boxアカウントにサインインすることもできます。
