# ユーザープロファイルの概要

**URL**: https://support.staffbase.com/hc/ja/articles/25487287700882
**取得日**: 2026-05-02 18:48:46 JST

## 内容

ユーザープロファイルは、最先端の外観と感触を持っています。これは、連絡先の詳細、チーム情報、プロファイルの見出しなど、組織の表示設定に基づいてユーザーの概要を迅速に提供します。プロファイル。
情報を見つける
ユーザープロファイルには、HRまたはITシステムから自動的に生成された基本的な従業員およびチーム情報が含まれています。例えば：
名
姓
役職
部署
公開メールアドレス
所在地
マネージャー
情報を追加する
情報を簡単に共有し、ユーザープロファイルに個人的なタッチを加えます。ユーザープロファイルページでは、情報を追加する例は以下の通りです：
ユーザープロファイルフィールド
例
ヘッダー画像： プロファイルに個人的なタッチを加え、目立たせるためのものです。ヘッダー画像を追加しない場合、ヘッダーエリアは自動的にサイズが縮小されます。
アイデンティティ詳細： 名前に顔を合わせ、希望する代名詞を指定します。
個人情報： プロファイルの見出しに強調したい内容を追加します。
ボタン： ボタン形式でリンクを追加します。
URL
LinkedIn
連絡先情報： どこで連絡を取れるかの情報を追加します。
オフィス電話
携帯電話
その他のカスタマイズ可能なフィールド
組織の設定に基づいて：
Xingなどのソーシャルメディアプロファイル
追加のマネージャー
プラットフォームの設定に応じて、ユーザープロファイルの右側には、従業員番号、ペット、趣味、言語などのカスタマイズ可能なさまざまな詳細が含まれる場合があります。
情報を編集する
いつでもプロファイル情報を編集し、他の人にどのように見えるかをプレビューできます。
同僚と交流する
組織がチャットプラグインを有効にしている場合、ユーザープロファイルのメッセージを送信ボタンをクリックすることで、誰かと会話を始めることができます。
ユーザーはMicrosoft Teamsを使用している場合、ユーザープロファイルにMicrosoft Teamsのドロップダウンが表示され、同僚とチャット、通話、ビデオ通話ができます。
チームの詳細を表示する
新しい従業員が組織の報告構造を把握しようとしている場合、マネージャーがワークフローを構築している場合、または従業員が不在の際にチームの誰かに連絡を取りたい場合、チームの詳細は透明性と責任を追加します。
マネージャー： 従業員が所属するチームに関する情報を提供します。
組織図： 従業員の会社構造における階層的な位置を表示し、自動的に生成されます。プロファイルのカードは常に左側に表示され、強調表示されます。
最近の投稿を見る
アクティベーション
組織の設定に基づいて、あなたまたは他のユーザープロファイルの最近の投稿にアクセスできます。
最近のコメントを見る
組織の設定に基づいて、あなたまたは他のユーザープロファイルの最近のコメントにアクセスできます。
ユーザープロファイルに最近のコメントを表示しないことが好ましい場合、管理者はカスタマーサクセスマネージャーまたはStaffbaseサポートに連絡してオプションを無効にすることができます。
