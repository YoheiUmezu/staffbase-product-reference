# ページの設定を使用する

**URL**: https://support.staffbase.com/hc/ja/articles/27895335171858
**取得日**: 2026-05-02 18:55:03 JST

## 内容

ページは、長期間有効であったり、時々のみ更新が必要な静的なコンテンツに最適です。ページをカスタマイズすることで、特定のコミュニケーション目標や視聴者のニーズに対応できます。ページの設定をカスタマイズすることで、ページの表示場所、閲覧できるユーザー、コンテンツを管理できるユーザーを制御できます。
ページ設定を開く方法は2つあります：
ページ概要から
Studioでコンテンツ > ページに移動します。
ページの概要が開きます。
設定を表示したいページに移動します。
3つのドットをクリックしてアクションメニューを開きます。
設定をクリックします。
コンテンツエディターから直接
Studioでコンテンツ > ページに移動します。
ページの概要が開きます。
設定を表示したいページに移動します。
歯車アイコンをクリックします。
ページ設定にアクセスできました。これでページのさまざまな設定を構成できます。
言語設定を構成する
ページの言語別バージョンを作成または管理し、異なる地域のユーザー向けに一貫性があり、分かりやすく、最適化されたページを提供できます。ページのタイトルや説明を複数の言語で設定できます。多言語ページを追加する場合、組織で有効化されている言語を選択できます。追加の言語が必要な場合は、管理者が有効化できます。
複数言語でページを作成するには、組織でこの機能が設定されている必要があります。このオプションが必要な場合は、管理者にお問い合わせください。
ページ設定で、+記号をクリックして言語を追加します。
言語を追加ダイアログが開きます。
追加したい言語を選択します。
適用するをクリックします。
追加するをクリックします。
各言語の名前欄にページのタイトルを入力します。
必要に応じて、説明欄にページの説明を入力します。
追加した各言語について、手順5と6を繰り返します。
保存するをクリックします。
ページの言語設定が構成されました。
共通設定を構成する
共通設定では、ページの作成者やスペース、ページで使用されているアイコンが表示されます。
アイコンはメニュー設定でカスタマイズできます。
共通設定で構成できる内容：
ページを表示するデバイス
パンくずリスト設定
ページ設定で共通設定に移動します。
オーナードロップダウンから、ページの担当ユーザーを選択します。
メニューに表示するデバイスで、ページをメニューに表示したいデバイスの種類を選択します。
パンくずリスト設定で、ページのパンくずリストでの表示方法を選択します：
デスクトップメニューでのみ表示
常に表示
表示しない
保存するをクリックします。
ページの共通設定が構成されました。
アプリでの表示設定を構成する
アプリやイントラネットでページを誰が閲覧できるかを、作成時または後からページ設定で制御できます。これにより、適切なユーザーが適切なコンテンツを閲覧できるようになります。ページの表示設定は、特定のユーザー、グループ、APIトークンごとにカスタマイズできます。すでに作成したページの表示設定を変更できます。
ページ設定でアプリでの表示に移動します。
ドロップダウンからページの表示設定を選択します。
選択したスペースのユーザーとグループを選択した場合、さまざまなオプションが表示されます。
アプリでの表示設定
ページに関連するオプションを選択します。
選択したオプションに応じて、対応するドロップダウンで対象の視聴者をさらに絞り込みます。
公開エリアで表示オプションを選択し、ページを公開エリアで表示するかどうかを決定します。
保存するをクリックします。
アプリやイントラネットでのページの表示設定が構成されました。
編集者を構成する
ページを編集できるユーザーを指定できます。これにより、コンテンツ固有の権限を持つユーザーのみがページのコンテンツ品質を管理できるようになります。
ページ設定で編集者に移動します。
ページの編集者を絞り込むために、1つ以上のオプションを選択します。
選択したオプションに応じて、対応するドロップダウンで編集者を指定します。
保存するをクリックします。
このページへのアクセスと編集権限を適切な編集者に割り当てました。
プラグイン設定を構成する
プラグイン設定では、ページが一定期間更新されていない場合にアラートを出して編集者にコンテンツの正確性と最新性を保つよう促す、更新リマインダーを設定できます。
更新リマインダーは、ページが一定期間更新されていない場合に編集者に通知します。ページはデフォルトで6か月後にリマインダーが表示されるよう設定されていますが、コンテンツの見直し頻度に応じて各ページごとに調整できます。更新リマインダーは、コンテンツエディターとページ概要の両方に表示されます。これらのリマインダーは定期的なコンテンツのメンテナンスを促し、古い情報が見落とされるのを防ぎます。
ページ設定でプラグイン設定に移動します。
更新リマインダーで、更新リマインダーを表示オプションを選択して、ページの更新リマインダーを有効にします。
ドロップダウンから、ページの更新リマインダーの頻度を選択します。
保存するをクリックします。
ページの更新リマインダーが構成されました。リマインダーはいつでも削除できます。
