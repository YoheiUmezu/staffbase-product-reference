# コメントの削除

**URL**: https://support.staffbase.com/hc/ja/articles/4408409900690
**取得日**: 2026-05-02 18:54:13 JST

## 内容

エディター
コメントのモデレーションは、コンテンツがプラットフォームのガイドラインに準拠していることを確認し、すべてのユーザーが安全に感じられるようにするのに役立ちます。このプロセスができるだけ透明であり、重要な情報が保持されることが不可欠です。
コメントを削除する際には、2つのオプションから選択できます：
コメントを削除
コメントを削除すると、コメントの代わりにプレースホルダーが表示され、ユーザーにコメントが削除されたことを通知します。この選択肢では、削除されたコメントのスレッドは保持されます。他のユーザーは削除されたコメントに返信できますが、誰が書いたのかやいいねの情報は表示されなくなります。元のコメントはチャンネルのエディターのみが見ることができ、いつでも復元可能です。
コメントをゴミ箱に移動
コメントをゴミ箱に移動すると、そのコメントとスレッド内の他のすべてのコメントが削除されます。この場合、削除されたことを示すプレースホルダーは表示されません。30日後、コメントはゴミ箱から永久に削除され、復元できなくなります。
コメントは以下から削除できます：
スタジオのコメント概要
ニュース投稿内
コメント概要からのコメントの削除
スタジオで、コンテンツ > システム > コメント.
コメント概要で、削除したいコメントに移動します。
削除をクリックします。
コメントを削除するための確認ダイアログが開きます。
コメントを削除するか、ゴミ箱に移動するかを選択します。
確認をクリックします。
コメントが削除されました。コメントはユーザーには表示されなくなります。チャンネルのエディターは削除されたコメントを引き続き見ることができ、復元できます。
投稿ビューからのコメントの直接削除
ニュース投稿で、削除したいコメントに移動します。
コメントのための3つのドットをクリックしてアクションメニューを開きます。
削除をクリックします。
コメントを削除するための確認ダイアログが開きます。
コメントを削除するか、ゴミ箱に移動するかを選択します。
確認をクリックします。
コメントが削除されました。コメントはユーザーには表示されなくなります。チャンネルのエディターは削除されたコメントを引き続き見ることができ、復元できます。
