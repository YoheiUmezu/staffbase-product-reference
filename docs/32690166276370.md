# コンテンツデザイナーでページのタイトルと説明を更新する

**URL**: https://support.staffbase.com/hc/ja/articles/32690166276370
**取得日**: 2026-05-02 19:00:58 JST

## 内容

ページのタイトルと説明は、コンテンツデザイナーで作成後いつでも編集できます。
Studioで、コンテンツ > ページに移動します。
更新したいページを開きます。
上部メニューで、ページのタイトルに移動します。
鉛筆アイコンをクリックします。
タイトルと説明を編集ダイアログが開きます。
ページのタイトルと説明を編集します。
保存するをクリックします。
ページのタイトルと説明が更新されました。
