# Microsoft 365 ファイルプラグインのインストールと構成

**URL**: https://support.staffbase.com/hc/ja/articles/360014012959
**取得日**: 2026-05-02 19:01:59 JST

## 内容

StaffbaseのMicrosoft 365ファイルプラグインは、Microsoft 365のドキュメントをあなたのアプリまたはイントラネットに表示します。
StaffbaseのMicrosoft 365ファイルプラグインは、現在アクティブな開発が行われておらず、セキュリティ関連のメンテナンスのみが行われています。Staffbaseは、代わりにウィジェットの使用を推奨しています。ページにMicrosoft 365ファイルウィジェットを追加することで、ユーザーにさらに良い体験を提供し、将来の更新や改善を受けることができます。
❯❯ Microsoft 365ファイルウィジェットの詳細を学ぶ。
Staffbaseプラットフォームのユーザーは、ファイルを表示およびアクセスするためにMicrosoft 365ライセンスが必要です。

前提条件:

Microsoft 365ファイルプラグインの設定
Microsoft 365からのドキュメントリストを従業員アプリに表示するように構成でき、ユーザーはMicrosoft 365内の任意のSharePoint Onlineサイトからファイルを表示できます。SharePoint Onlineのファイルがアプリに表示される方法を構成できます。各ファイルリストについて、以下のいずれかを選択できます:
特定のライブラリ
任意のSharePoint OnlineサイトのライブラリからSharePoint Onlineファイルを表示します。このオプションを使用すると、特定のドキュメントを特定の順序で表示できます。
最近のファイル
SharePoint OnlineおよびOneDriveで最近変更または表示されたファイルを表示します。このオプションを使用すると、ファイルリストは各ユーザーに合わせて調整され、最近やり取りしたファイルが表示されます。
トレンドファイル
SharePointまたはOneDriveで現在最も人気のあるファイルを表示します。このファイルリストは、すべてのユーザーのアクティビティに基づいて調整され、最も表示され、変更されたファイルが表示されます。
