# スタッフベース コネクトアプリへの管理者同意の付与

**URL**: https://support.staffbase.com/hc/ja/articles/29414627040274
**取得日**: 2026-05-02 18:53:07 JST

## 内容

有効化
管理者
従業員がMicrosoft Teamsに追加されたStaffbase Connectアプリへアクセスできるようにするには、Staffbase Connectアプリの表示に管理者の同意を与える必要があります。権限を付与することで、StaffbaseアプリがSSOを利用できるようになり、Microsoft 365の管理者のみがStaffbaseアプリの設定にアクセスできることが保証されます。
前提条件
Microsoft 365でグローバル管理者またはTeamsサービス管理者の役割を持っていること。
Microsoft Teams管理センターで、Teamsアプリ > アプリの管理に移動します。
または、Microsoft 365管理センターからMicrosoft Teamsにアクセスします。
名前で検索するフィールドで、Staffbase Connectを検索します。
アプリへのリンクをクリックします。
アプリの詳細が開きます。
権限タブで、管理者の同意を付与するをクリックします。
Microsoft管理者アカウントへのサインインが求められます。
Microsoftアカウントにサインインします。
詳細を確認し、承諾をクリックします。
これでMicrosoft TeamsのStaffbase Connectアプリに権限が付与されました。次に、組織のStaffbaseウェブアプリのURLを追加できます。
