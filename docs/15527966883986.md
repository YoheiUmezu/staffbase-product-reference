# ServiceNow検索統合の概要

**URL**: https://support.staffbase.com/hc/ja/articles/15527966883986
**取得日**: 2026-05-02 18:43:28 JST

## 内容

ベータ
アクティベーション
ServiceNowとStaffbaseの統合により、ユーザーはプラットフォーム全体でコンテンツを見つけるのがこれまで以上に簡単になります。この統合により、ユーザーはStaffbaseプラットフォーム内のコンテンツだけでなく、ServiceNowのナレッジベースリソースにもアクセスできます。これにより、ユーザーは別のプラットフォームで再度コンテンツを検索する手間が省け、同時にコンテンツ作成者はコンテンツの重複を避けることができます。ServiceNowからの検索結果は、Staffbaseプラットフォームの別の検索タブに表示されます。詳細については、Staffbaseサポートまたはカスタマーサクセスマネージャーにお問い合わせください。
現在、この統合はデスクトップでのみ利用可能です。モバイルでの統合は後日提供される予定です。
ユーザーにServiceNow検索を利用可能にするには、ServiceNow接続をStaffbaseに追加する必要があります。
ユーザーにとってのServiceNow統合の見え方
ユーザーがServiceNowアカウントにサインインすると、ServiceNow検索タブでコンテンツを検索できます。検索タブの結果をクリックすると、ユーザーはServiceNowのナレッジベース内の目的のコンテンツに直接移動します。
