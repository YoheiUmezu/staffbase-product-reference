# チャンネルの削除

**URL**: https://support.staffbase.com/hc/ja/articles/360028213811
**取得日**: 2026-05-02 19:03:08 JST

## 内容

チャンネルは、公開した後でも削除できます。削除後、チャンネルおよびそのチャンネル内のすべての投稿は、アプリのユーザーやスタジオの管理者には表示されなくなります。削除されたチャンネルは、削除から30日以内であれば、ゴミ箱機能を使用して復元できます。
スタジオで、削除したいチャンネルに移動します。
設定を開くためにギアアイコンをクリックします。
設定ページが開きます。
チャンネルを削除をクリックします。
チャンネルを削除の確認ダイアログが開きます。
ゴミ箱に移動をクリックします。
