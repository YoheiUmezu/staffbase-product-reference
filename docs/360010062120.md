# アクセスコードを使用してユーザーを手動で追加する

**URL**: https://support.staffbase.com/hc/ja/articles/360010062120
**取得日**: 2026-05-02 18:50:19 JST

## 内容

アプリにユーザーを手動で追加するためのアクセスコードを生成できます。この方法は、ユーザーのメールアドレスや連絡先情報がない場合にのみ推奨されます。そのような場合、資格情報は対面、郵送、またはお好みの別の方法で配布できます。
スタジオで、ユーザー設定に移動します。
ユーザーを追加をクリックします。
新しいユーザーを招待するためのダイアログが開きます。
アクセスコードを作成をクリックします。
生成したいアクセスコードの数を入力します。
コードを作成をクリックします。
アクセスコードが生成されました。これにより、アプリ内にコードごとに1人の新しいユーザーが作成されました。
自動的に保留中のユーザータブにリダイレクトされ、各ユーザーの詳細を編集できるようになります。
