# コメントモデレーションの概要

**URL**: https://support.staffbase.com/hc/ja/articles/5166211845266
**取得日**: 2026-05-02 18:43:55 JST

## 内容

エディター
コメントは、あなたのプラットフォームにおけるコミュニケーションの中心的な部分であり、アイデア、意見、知識を共有するための完璧な場所です。すべてのユーザーが自分の意見を自由に安心して共有できるようにするために、あなたのプラットフォームは差別、嫌がらせ、スパムの余地がない安全な空間であるべきです。
コメントモデレーションは、ユーザー生成コンテンツがあなたのプラットフォームのガイドラインに準拠していることを確認するのに役立ちます。コメントモデレーション機能を使用して：
コメントを非表示にする
潜在的に不適切なコメントを非表示にして、モデレーションチームがそのコメントがプラットフォームのポリシーに違反しているかどうかを判断し、削除すべきか、再承認できるかを決定できるようにします。
コメントを承認する
非表示にされたコメントがプラットフォームのポリシーに違反していない場合や、誤って非表示にされた場合、モデレーションチームはそのコメントを簡単に再承認できます。
コメントを削除する
不適切なコメントを迅速かつ透明に削除します。コメントのみを削除するか、その下のスレッドとともに削除するかを選択できます。ユーザーは削除されたコメントに対しても返信でき、重要な情報が失われることはありません。
モデレーションプロセスのすべてのステップでの完全な透明性
コメントモデレーションは、あなたのプラットフォームを自由な交流のための安全な場所にすることを目的としています。最大限の透明性を確保し、重要な情報や魅力的な議論が失われないようにすることが重要です。したがって、コメントがレビューのために非表示にされたり削除された場合、ユーザーはそのコメントの代わりにプレースホルダーを見て、コメントがモデレーションのために非表示にされたか、削除されたことを通知されます。
