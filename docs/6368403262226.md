# 分析ダッシュボード FAQ

**URL**: https://support.staffbase.com/hc/ja/articles/6368403262226
**取得日**: 2026-05-02 19:02:10 JST

## 内容

エディター
新しいアクティビティが分析ダッシュボードに表示されるまでにどのくらいの時間がかかりますか？
現在、分析データは毎時更新されます。したがって、ニュース投稿が9:35に公開された場合、この投稿へのインタラクションは10:00まで分析ダッシュボードに表示されません。
分析ダッシュボードではコメントといいねはどのようにカウントされますか？
通常のコメントに加えて、コメントへの返信や削除されたコメントも総コメントインタラクションにカウントされます。同様に、ユーザーによって削除されたいいねも総いいね数に残ります。これが、アプリ内の投稿のコメント数やいいね数が分析ダッシュボードに表示される数と異なる理由です。
分析は削除されたコンテンツのインタラクションをどのように扱いますか？
削除されたニュース投稿の統計は分析ダッシュボードに含まれますが、タイトルは<削除されたアイテム>として表示されます。
異なるタイムゾーンの分析をどのように解釈すればよいですか？
すべてのデータは、データを表示している管理者のタイムゾーンで表示されます。
分析ダッシュボードにアクセスする際に「何かがうまくいかなかった」というエラーメッセージが表示されるのはなぜですか？
これは、ブラウザに広告ブロッカーがインストールされている場合に発生する可能性があります。その場合、広告ブロッカーは分析の表示もブロックします。問題を解決する方法を学んでください。
ニュース投稿が別のニュースチャンネルに移動された場合、統計はどのように変わりますか？いいね、訪問、コメントは元のニュースチャンネルでまだカウントされますか？
ニュース投稿の分析は、アクティビティが最初に発生したニュースチャンネルに基づいて収集されます。投稿が元のニュースチャンネルで持っていたインタラクションは元のニュースチャンネルに残ります。新しいチャンネルで受け取るインタラクションは新しいチャンネルに反映されます。ニュースリストで投稿が移動された後のメトリクスの概要を取得するには、チャンネルフィルタードロップダウンから現在のニュースチャンネルと以前のニュースチャンネルの両方を選択できます。
