# Staffbaseメールにおけるブランディングの概要

**URL**: https://support.staffbase.com/hc/ja/articles/24341952179730
**取得日**: 2026-05-02 18:42:46 JST

## 内容

あなたのメールは、組織の価値観や文化を反映する必要があります。Staffbaseは、プラットフォームから送信するメールにブランドアイデンティティを注入するためのさまざまな可能性を提供し、すべてのメッセージが組織の独自のスタイルと声に合致するようにします。
フォント
さまざまなフォントから選択するか、組織のカスタムフォントをアップロードして、すべてのコミュニケーションで一貫性を保つことができます。利用可能なフォントオプションの種類は以下の通りです：
デフォルトフォントとセカンダリフォント
デフォルトフォント: メール作成時に特定のフォントが選択されていない場合に自動的に適用されます。メールブランディングページでデフォルトラベルが付けられています。選択したフォントのオプションに応じて、太字、斜体、または太字斜体などのフォントスタイルを選択することもできます。
セカンダリフォント: あなたの第二のお気に入りのフォントです。メールブランディングページでセカンダリラベルが付けられ、メールデザイナーでメールをデザインする際のフォントドロップダウンメニューの第二のオプションとして表示されます。
利用可能なフォントオプション
以下は、あなたが利用できるフォントです：
Arial
Arial Black
Book Antiqua
Comic Sans MS
Courier New
Georgia
Helvetica
Lucida Sans Unicode
Tahoma
Terminal
Times New Roman
Trebuchet MS
Verdana
カスタムフォント
カスタムフォントは、プラットフォーム上で簡単に利用できないフォントです。メールをさらにパーソナライズするために、以下の形式でカスタムフォントをアップロードできます：
ファイルサイズ: 最大5MB
ファイルタイプ: .TTF, .OTF, .WOFF, .WOFF2
Staffbaseは可変フォントをサポートしていません。
フォールバックフォント
メールでカスタムフォントを使用する場合、フォールバックフォントを定義できます。フォールバックフォントを定義しない場合、デフォルトでは、カスタムフォントがユーザーのメールクライアントで利用できないときに、メールクライアントはArialフォントにフォールバックします。
スタイル
メール内の以下のスタイルをカスタマイズできます：
リンク
ボタン
色
色のページからブランドカラーを追加できます。色のページは中央リポジトリとして機能し、ボタンやリンクスタイルを定義するための色を管理および再利用できます。既存の色をカスタマイズしたり、追加のブランドカラーをHEXコードやカラーピッカーを使用して簡単に追加できます。追加できる色の数に制限はありません。
