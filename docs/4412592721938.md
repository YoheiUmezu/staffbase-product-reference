# 従業員メールWebアプリの始め方

**URL**: https://support.staffbase.com/hc/ja/articles/4412592721938
**取得日**: 2026-05-02 18:44:16 JST

## 内容

製品名に関する注意: 2022年以降、BananatagはStaffbaseの従業員メール製品です。
従業員メールは、美しい従業員メールを作成し、カスタマイズされた従業員リストに配信し、エンゲージメントを測定することで、内部コミュニケーション戦略を改善し、測定可能な影響を与えるのに役立ちます。
Power/BusinessまたはEnterpriseプランを利用すると、従業員メール Webアプリから直接メールを送信することで、以下のことが可能になります:
ITの助けを借りずにカスタム配信リストを作成、編集、共有する
部門、場所、職務など、保存している任意の従業員属性に基づいてリストにターゲットを絞ったメールを送信する
メールを開かなかった受信者のみをフォローアップする
ユニークな受信者追跡を使用して詳細なエンゲージメントデータをキャプチャする
従業員属性別にメールエンゲージメントメトリクスを分解する
組織のためのWebアプリの設定
ITチームの支援を受けて、以下の手順に従ってWebアプリの送信ワークフローを有効にします。
Staffbaseは、この設定を行うためにAdminまたはParent Adminアカウントを使用することを推奨します。
組織のメールドメインを確認し、接続する
初期のドメイン確認プロセスにはITチームの支援が必要であり、これにより従業員メールがあなたのドメインを代表して当社のシステムを使用して送信することが許可されます。
連絡先をアップロードまたは同期する
受信者リストを管理するために、好みに応じていくつかの異なるアプローチを取ることができます。
複数のシステムを使用している場合は、以下のオプションを組み合わせることもできます:
カンマ区切り値（CSV）ファイルをアップロードする
Microsoft Entra ID（以前のAzure AD）および/またはWorkdayとの自動同期を設定する
あなたのアプリおよび/またはイントラネットからの自動同期を設定する
設定後の最初のステップ
ドメインと連絡先が設定されると、従業員メールユーザーアカウントを持つ組織内の誰でも、以下に示す最初のステップに従ってWebアプリの送信ワークフローを試すことができます。
従業員メールWebアプリにサインインする
カスタム配信リストを作成する
メールデザイナーでドラフトとテンプレートを作成する
Webアプリからメールを送信する
デフォルトでは、この機能はBase Userアカウントには利用できません。
Enterpriseプランでは、Parent Adminがそれを有効にできます。
結果を読み取り、解釈する
