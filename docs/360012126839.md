# 部分CSVインポートの概要

**URL**: https://support.staffbase.com/hc/ja/articles/360012126839
**取得日**: 2026-05-02 19:01:40 JST

## 内容

このインポート方法はレガシーと見なされており、将来的に廃止される予定です。その代わりに、Staffbaseはできるだけ早く[新しい] CSVインポート方法に切り替えることを推奨します。[新しい] CSVインポート方法は、先進的な機能とより良いユーザーエクスペリエンスを備えたベータ機能です。
部分CSVインポートとは何ですか？
部分CSVインポートは、管理者が全体のユーザーの一部にのみ影響を与えるCSVインポートを実行できるようにします。部分CSVタグを作成することで、更新すべきユーザーのサブセットを示すことができます。これは、異なる支店や場所を持つ組織にとって非常に便利です。各支店や場所には特定の部分CSVタグがあり、中央管理者が各支店のユーザーを個別に更新したり、分散した場所の管理者が独立してユーザーを管理したりすることが容易になります。
部分CSVインポートは私の組織に適していますか？
Staffbaseは、中央集権的な従業員管理システムを持たない組織にのみ部分CSVインポートを推奨しています。組織の各支店や場所の人員が現地で自律的に管理されている場合、部分CSVインポートは大いに役立ちます。
分散した支店を持つ組織は、特定のユーザーグループに対して更新を行うことができるため、部分CSVインポートの恩恵を受けます。これにより、部分CSVインポート中に全ユーザーベースのサブセットのみが更新されるため、エラーを抑制することができます。
どのユーザー管理方法が組織に最適かについての個別相談は、Staffbaseサポートまたはカスタマーサクセスマネージャーにお問い合わせください。
管理者の任命
ユーザーが部分CSVインポートを実行できるようにするには、そのユーザーをシステム全体の管理者にする必要があります。これにより、そのユーザーは他のアプリ全体の設定や他の部分CSVタグにもアクセスできるようになります。
管理者がスタジオの使い方を学ぶことが重要です。部分CSVインポートの権限は、管理者にアプリ設定、すべてのユーザーデータ、およびすべての部分CSVタグへの完全なアクセスを与えます。
アプリ内で同時に実行できるCSVインポートは1つだけであることを念頭に置いておくことも重要です。管理者がCSVインポートをすぐに完了しない場合、他のすべての管理者は、そのインポートが完了するか、開始した管理者によって削除されるまで新しいインポートを開始することができません。Staffbaseは、できるだけ少ない管理者を追加することを一般的に推奨しています。
