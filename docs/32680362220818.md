# Staffbase Live のイベントを管理する

**URL**: https://support.staffbase.com/hc/ja/articles/32680362220818
**取得日**: 2026-05-02 18:53:00 JST

## 内容

アクティベーション
作成したライブイベントは、Studio で直接管理できます。イベントの概要から、次のことができます。
イベントの詳細を表示または編集する
イベントをコピーして、既存のイベントをもとに新しいイベントを作成する
ライブ配信前にブロードキャストをプレビューする
モデレーターまたは視聴者用リンクをコピーして、役割ごとのアクセスを共有する
不要になったイベントを削除する
これらのオプションにより、ライブイベントの更新、再利用、共有を行いながら、アクセスや設定を完全に管理できます。
イベントの詳細を表示または編集する
管理者
編集できるのはスケジュールされたイベントのみです。過去のイベントは編集できませんが、詳細を表示できます。
Studio で、コンテンツ > イベントセンター > スケジュール済み > 表示または編集したいイベントに移動します。
過去のイベントを表示するには、過去のイベント で、表示したいイベントに移動します。
三点リーダーをクリックしてアクションメニューを開きます。
表示・編集 をクリックします。イベントページの 概要 タブが開きます。
イベント詳細 に移動します。
詳細を表示するか、必要に応じて編集します。
保存する をクリックします。
イベントの詳細を表示または編集しました。
イベントをコピーする
管理者
イベントをコピーして、既存のイベント詳細をもとに新しいイベントを作成できます。これは、定期的または類似のセッションを簡単に設定したい場合に便利です。たとえば、毎月の部門全体会議などです。
Studio で、コンテンツ > イベントセンター > スケジュール済み > コピーしたいイベントに移動します。
三点リーダーをクリックしてアクションメニューを開きます。
イベントをコピー をクリックします。
イベントをコピーしますか？ ダイアログが開きます。
コピーする をクリックします。
イベント情報 で、日付と時刻を入力します。
次 をクリックします。
設定 で、必要な変更を行います。
作成する をクリックします。
既存のイベントをコピーして新しいイベントを作成しました。
ブロードキャストをプレビューする
モデレーター
ライブ配信イベントの前に、Broadcast Studio でブロードキャストの設定をプレビューできます。
Studio で、コンテンツ > イベントセンター > スケジュール済み > プレビューしたいブロードキャスト設定に移動します。
三点リーダーをクリックしてアクションメニューを開きます。
ブロードキャストをプレビュー をクリックします。
Broadcast Studio タブが開きます。
設定を確認し、見た目や動作をチェックします。
イベントのブロードキャスト設定をプレビューしました。
ライブ配信リンクをコピーする
モデレーター
スケジュールされたイベントのモデレーター用および視聴者用のリンクをコピーできます。これらのリンクは、ニュース投稿記事、ページ、または個人メッセージで共有できます。
たとえば、視聴者用リンクをニュース投稿記事で共有し、モデレーターや Broadcast Studio のリンクを CEO にメールで送信できます。
ユーザーは、視聴者またはモデレーターとして追加され、専用のイベントリンクを使用した場合のみライブ配信に参加できます。リンクだけではアクセス権は付与されません。
Studio で、コンテンツ > イベントセンター > スケジュール済み > コピーしたいイベントリンクに移動します。
表示・編集 をクリックします。
イベントページの 概要 タブが開きます。
スピーカーとモデレーター用 Broadcast Studio の横にある リンクをコピー をクリックすると、モデレーター用リンクをコピーできます。
視聴者ページ の横にある リンクをコピー をクリックすると、参加者用リンクをコピーできます。
リンクをコピーし、個人メッセージや投稿で使用できます。
