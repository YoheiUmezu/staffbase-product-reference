# リンクリストウィジェットの設定

**URL**: https://support.staffbase.com/hc/ja/articles/25488281324178
**取得日**: 2026-05-02 18:51:07 JST

## 内容

エディター
リンクリストウィジェットを使用すると、モバイルおよびウェブアプリにウェブサイトのリンクを追加できます。
追加されると、これらのウェブサイトのリンクはリストとして表示されます。ユーザーがリスト内のリンクをクリックすると、それに関連付けられたリンクが開きます。
リンクリストウィジェットを設定するには、以下の項目を設定する必要があります：
一般設定
リンクリスト
一般設定の構成
リンクリストウィジェットを追加します。
リンクリストウィジェットのダイアログが開きます。
オプションで、タイトルを提供します。
タイトルはウィジェットの上に表示され、ユーザーに見えます。
ウィジェットが表示されるデバイスタイプを、対応するアイコンをクリックして選択します。
アプリ内の表示設定の下で、ウィジェットを表示するユーザーを選択します。
オプションで、モバイルブラウザーで開くのトグルを有効にします。
有効にすると、モバイルデバイスで定義されたターゲットURLが外部ブラウザーで開きます。
リンクリストウィジェットの一般設定が構成されました。次に、リンクのリストを作成できます。
リンクタイルの設定
リンクのタイトルを提供します。
タイトルはユーザーに見えます。リンクの内容を補完するタイトルを使用してください。
リンクしたいウェブサイトのURLを追加します。
オプションで、アイコンを追加します。
標準アイコンライブラリからアイコンを選択するか、カスタムアイコンをアップロードするか、アイコンを含めないこともできます。
オプションで、ステップ1-3を繰り返して、リストにさらにリンクを追加します。
すべてのリンクをリストに追加したら、OKをクリックします。リンクはいつでも調整できます。
リンクリストウィジェットがページに追加されました。
