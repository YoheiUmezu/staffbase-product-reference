# ページまたは投稿を作成するXX内で動画を埋め込む

**URL**: https://support.staffbase.com/hc/ja/articles/20006592096146
**取得日**: 2026-05-02 18:55:27 JST

## 内容

編集者
さまざまな動画プラットフォームから関連する動画コンテンツを提供して、読者の関心を引きましょう。動画はページや投稿に直接埋め込むことができます。
パブリックにアクセスできる動画のみ、サポートされている動画プラットフォームのリンクを使って埋め込むことができます。プライベート動画は埋め込むことができません。
動画を埋め込む前に、プライバシー設定が次のようになっていることを確認してください：
パブリック：完全にサポートされています
限定公開：サポートされています。
動画は検索結果に表示されませんが、直接リンクからアクセスできます。
ページまたは投稿に動画を埋め込む
スタジオで、動画を追加したいページまたは投稿に移動します。
コンテンツエディターで、動画を追加したい場所をクリックします。
コンテンツエディターツールバーで、メディアを追加するアイコンをクリックしてメニューを開きます。
ウェブサイトから埋め込むをクリックします。
動画の直接リンクを追加します。
保存するをクリックします。
動画が埋め込まれました。今、ページまたは投稿を保存してください。
