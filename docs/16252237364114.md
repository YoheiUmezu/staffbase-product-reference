# Staffbase Protectの概要

**URL**: https://support.staffbase.com/hc/ja/articles/16252237364114
**取得日**: 2026-05-02 18:49:18 JST

## 内容

Staffbase Protectは、潜在的なサイバー脅威からStaffbaseプラットフォームを保護するために設計された機能です。これは、クラウドベースのセキュリティソリューションの主要プロバイダーであるCloudflareプラットフォームを使用して、この追加の保護層を提供します。
Staffbase Protectの利点
Staffbaseインフラストラクチャへのトラフィックフローをフィルタリングします
有害なリクエストをフィルタリングします
正当なリクエストのみを許可します
分散型サービス拒否攻撃（DDoS）などの悪意のある攻撃をブロックします
ウェブアプリケーションファイアウォール（WAF）として機能します
セキュリティ保護を強化します
Staffbase APIの信頼性のある運用を確保します
ユーザーとStaffbaseプラットフォーム間のデータの安全な転送を確保します
