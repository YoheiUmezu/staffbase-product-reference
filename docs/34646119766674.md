# Staffbaseプラットフォーム内でのAIへのプロンプト入力

**URL**: https://support.staffbase.com/hc/ja/articles/34646119766674
**取得日**: 2026-05-02 18:48:23 JST

## 内容

効果的なプロンプトは、StaffbaseプラットフォームのAI機能から質の高い出力を得るための鍵です。明確なプロンプトは曖昧さを減らし、AIがあなたの意図、トーン、望む結果をよりよく理解するのに役立ちます。構造化された指示は一貫性を高め、繰り返し使用する際の結果の信頼性を向上させます。
StaffbaseのAI搭載機能では、目的やワークフローに応じてさまざまなタイプのプロンプトが使用されています。すべてのAI搭載機能で直接プロンプトを入力できるわけではなく、一部はバックグラウンドで自動的に動作します。プロンプト入力に対応している機能では、具体的な入力内容は異なる場合がありますが、以下の基本原則が適用されます。
ベストプラクティス
Staffbaseは、ユーザーに最適な体験を提供するためにいくつかの推奨事項を用意しています。
具体的かつ簡潔にする
コンテキスト、トーン、形式、望む結果など、明確な詳細を含めてください。
複雑なリクエストは、精度を高めるために小さく明確な指示に分割してください。
可能であれば、PDFや画像など、以前に作成したサポートドキュメントをアップロードしてください。
視聴者と制約を定義する
コンテンツの対象となる視聴者を明確に記載してください。
AIが知っておくべき視聴者に関する具体的な情報を含めてください。
文字数制限や表現方法など、制約があれば記載してください。
繰り返し改善する
得られた結果に基づいてプロンプトを調整・改善してください。
具体的なユースケースやチームのニーズに応じてプロンプトのアプローチを調整してください。
プロンプトをドキュメント化し共有する
効果的なプロンプトは、今後のコンテンツ作成のためにドキュメント化してコピーするようにしてください。
実績のあるプロンプトはチーム内で共有し、チーム全体でコピーできるようにしてください。
