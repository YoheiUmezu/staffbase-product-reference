# クイズページの設定

**URL**: https://support.staffbase.com/hc/ja/articles/26066018496786
**取得日**: 2026-05-02 18:48:42 JST

## 内容

システム全体の編集者
この機能はアドオンです。詳細はStaffbaseの価格をご参照ください。有効化についてはStaffbaseサポートまたはカスタマーサクセスマネージャーにお問い合わせください。
クイズプラグインはStaffbase従業員エクスペリエンス（EX）ブースタープラグインです。フレンドリーなコンペティションを実施する場合やeラーニングコンテンツを提供する場合でも、クイズプラグインは参加を促す効果的な方法です。クイズプラグインをインストールした後、さらに設定を行い、最初のクイズページを公開してユーザーに表示できるようにします。
クイズページを設定する
Studioで、コンテンツ > クイズに移動します。
設定したいクイズページに移動します。
ページ下部のマニュアルをクリックして設定ガイドを開きます。
マニュアルを使って、ページをニーズに合わせて設定します。
保存するをクリックします。
クイズページの設定が完了しました。ページを公開してユーザーに表示できるようになりました。
クイズページを公開する
Studioで、コンテンツ > クイズに移動します。
公開したいクイズページに移動します。
クイズを公開するをクリックします。
クイズページが公開されました。ユーザーに表示されるようになりました。
