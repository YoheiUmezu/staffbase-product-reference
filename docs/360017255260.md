# ページテンプレートの追加

**URL**: https://support.staffbase.com/hc/ja/articles/360017255260
**取得日**: 2026-05-02 18:59:52 JST

## 内容

アクティベーション
管理者
ページテンプレートを使用することで、プラットフォーム上の一貫性と共通の構造を確保できます。ページテンプレートを個別化したり、組織のために新しいテンプレートを作成したりできます。
スタジオで、設定に移動します。
ページテンプレートをクリックします。
ページテンプレートの概要が表示されます。
テンプレートの追加をクリックします。
ページテンプレートを作成するためのエディタが開きます。
テンプレートのタイトルを入力します。
ウィジェットやセクションなど、必要なデザイン要素をテンプレートに追加します。
保存をクリックします。
新しいページテンプレートが組織に追加されました。このページテンプレートは、ページを作成する際にテンプレートのドロップダウンメニューでエディタが利用できるようになります。いつでもページテンプレートを変更したり、削除したりできます。
