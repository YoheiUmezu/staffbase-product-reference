# なぜ公共エリアの表示を切り替えるスイッチが見えないのですか？

**URL**: https://support.staffbase.com/hc/ja/articles/360001314671
**取得日**: 2026-05-02 18:52:37 JST

## 内容

最初に確認すること：
もし 公共エリアの表示 スイッチが見えない場合、あなたはおそらく完全にブランディングされたアプリを持っていない—まだです。例えば、営業デモ中にお見せするアプリは完全にブランディングされていません。
すべての顧客は、カスタマーサクセスマネージャーによって完全にブランディングされたアプリを設定してもらいます。設定が完了すると、あなたは自分のブランディングされたアプリ内でのみ作業することになります。質問がある場合は、Staffbaseサポートまたはカスタマーサクセスマネージャーにお気軽にお問い合わせください。
スペースを使用している場合：
現時点では、デフォルトのスペース（全従業員）からニュースチャンネルやページを公開することのみが可能です。
