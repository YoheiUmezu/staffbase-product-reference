# 不適切なコメントの報告

**URL**: https://support.staffbase.com/hc/ja/articles/8196822852754
**取得日**: 2026-05-02 18:55:39 JST

## 内容

ベータ
アクティベーション
攻撃的または不適切なコメントをモデレーターチームに報告できます。モデレーターはコメントの報告を匿名で受け取ります。不適切なコメントを報告することで、誰もが安全に参加できるコミュニティを作る手助けになります。
アプリまたはイントラネットで、不適切なコメントに移動します。
アクションメニューを開くために、3つのドットをクリックします。
報告をクリックします。
確認ダイアログが開きます。
報告をクリックします。
コメントが報告されました。
コメントは報告後も表示されます。報告者であるあなたや他のユーザーは、コメントが報告されたことを示すインジケーターを見ることはありません。報告されたコメントのフラグはモデレーションチームのみが確認できます。
