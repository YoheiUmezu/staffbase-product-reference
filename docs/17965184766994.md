# 分析キャンペーンメトリクスと分析ダッシュボード

**URL**: https://support.staffbase.com/hc/ja/articles/17965184766994
**取得日**: 2026-05-02 18:46:43 JST

## 内容

アクティベーション
システム全体のエディター
分析ダッシュボードは、ユーザーがStaffbaseプラットフォームとどのように、いつ、どこで関わっているかについての深い洞察を提供します。
この記事では、キャンペーンタブについて詳細に説明し、その中のグラフや分析データを含みます。このタブはSmart Impact機能の一部であり、Smart Impactについて詳しく学ぶ。 ユーザーの役割アクセスを含む全体の分析ダッシュボードについては、分析ダッシュボードの概要で学んでください。
キャンペーンタブは、デフォルトで過去7日間のデータを表示します。日付範囲を変更し、結果を絞り込むためにさらにフィルターを使用できます。フィルタリングについて詳しく学ぶ。
キャンペーンリスト
キャンペーンタブには、コンテンツに関する詳細なメトリクスを持つリストがあります。キャンペーンについては、キャンペーンの概要記事で詳しく学んでください。
キャンペーンリストは、プラットフォーム内のキャンペーンの広範な概要です。リストには、キャンペーンのタイトル、可視性、エンゲージメント、投稿数が表示されます。
リストは毎日更新されます。メトリクスは、処理と表示に最大24時間かかる場合があります。
異なるテーブルヘッダーは次のとおりです：
ヘッダー
説明
キャンペーン
キャンペーンの名前
可視性
キャンペーン内の少なくとも1つの投稿を表示したユーザーの割合
エンゲージメント
少なくとも1つのキャンペーン投稿に反応、コメント、または共有した訪問者の割合
アライメント
そのキャンペーンのために作成されたアライメント調査の平均スコア
投稿
そのキャンペーンに割り当てられた投稿の数

Smart Insightsダッシュボードへのナビゲーション
デフォルトでは、テーブルはキャンペーンタイトルのアルファベット順にソートされています。特定のヘッダーをクリックすることで、テーブルをさらにソートできます。
キャンペーンリストのソート
テーブルヘッダーによってキャンペーンリストをソートできます。
スタジオで、分析 > キャンペーンに移動します。
キャンペーンリストが表示されます。
リストをソートしたいヘッダーをクリックします。
リストは自動的に選択したヘッダーによって降順にソートされます。
オプションで、再度クリックしてリストを昇順にソートします。
特定のヘッダーによってキャンペーンリストをソートしました。キャンペーンタブから移動すると、結果は自動的にキャンペーンタイトルで表示されるようにリセットされます。
