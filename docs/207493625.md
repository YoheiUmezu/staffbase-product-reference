# 画像ギャラリーの画像の順序を変更する

**URL**: https://support.staffbase.com/hc/ja/articles/207493625
**取得日**: 2026-05-02 19:01:45 JST

## 内容

画像ギャラリーを作成すると、画像ギャラリー内の画像の順序を簡単に変更できます。また、既存の画像ギャラリーの画像の順序も変更できます。
ページ内にも画像ギャラリーを追加できます。
スタジオで、コンテンツ > ニュースに移動し、画像ギャラリーを追加または変更したい記事を選択します。
編集をクリックして記事エディターを開きます。
記事エディターが開きます。
記事テキスト内で画像ギャラリーを追加または編集します：
新しい画像ギャラリーを追加するには、画像ギャラリーの追加/編集をクリックします。
既存の画像ギャラリーを変更するには、まずエディターでギャラリーを選択し、次に画像ギャラリーの追加/編集をクリックします。
画像ギャラリーの追加/編集ダイアログが開きます。
新しい画像ギャラリーの場合、画像を追加します。
ハンドルを使用して、画像を希望の順序にドラッグアンドドロップします。
保存をクリックします。
