# 既存のニュース投稿をキャンペーンに割り当てる

**URL**: https://support.staffbase.com/hc/ja/articles/360021114779
**取得日**: 2026-05-02 18:52:09 JST

## 内容

システム全体のエディター
特定のトピックに関するニュース投稿をすでに公開している場合、コンテンツエディターから直接キャンペーンに割り当てることができます。キャンペーンのために新しい投稿を作成し、古い投稿を割り当てることで、キャンペーンの目標に関するトレンドを時間の経過とともに把握できます。
前提条件
少なくとも1つのキャンペーンを作成している必要があります。
ニュース投稿をキャンペーンに割り当てる
スタジオで、コンテンツ > ニュースに移動します。
キャンペーンに割り当てたいニュース記事、更新、またはメディア投稿を開きます。
コンテンツエディターが開きます。
サイドメニューのキャンペーンの下から、ドロップダウンメニューから投稿を追加したいキャンペーンを選択します。
保存をクリックします。
投稿は現在キャンペーンに割り当てられています。
個々のキャンペーンの概要で、この投稿とそのリーチメトリクスを、キャンペーンに割り当てられた他の投稿と一緒に表示できます。
