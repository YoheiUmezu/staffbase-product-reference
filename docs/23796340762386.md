# ニュース投稿の概要ページの概要

**URL**: https://support.staffbase.com/hc/ja/articles/23796340762386
**取得日**: 2026-05-02 19:02:18 JST

## 内容

エディター
Staffbaseプラットフォームは、オーディエンス特有の動的コンテンツをニュースチャンネルで公開することを可能にします。 ニュースチャンネルの種類について詳しく学ぶ。スタジオ内の各ニュースチャンネルには、作成、スケジュール、または公開された個々の投稿を一覧表示する概要ページがあります。この概要は、あなたがアクセス可能なニュースチャンネルを選択すると自動的に開きます。
概要に表示されるニュース投稿を絞り込むことが可能です。フィルタリングおよびソートオプションを使用することで、特定のニュース投稿を見つけるのがより簡単かつ迅速になります。あなたは：
デフォルトでは、投稿は最終更新日時の順にリストされます。2か月前に公開された投稿が昨日変更された場合、その投稿は概要ページの最上部に表示されます。
公開状況でフィルタリングすると、新しい投稿の日付がその選択を反映するように更新されます。たとえば、CEOの更新は2024年7月3日に最終更新されました。公開でソートすると、その投稿の日付は2024年7月2日に変更されます。これは、その投稿が公開された日です。
