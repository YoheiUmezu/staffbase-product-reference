# 最初のジャーニーステップの作成とスケジュール設定

**URL**: https://support.staffbase.com/hc/ja/articles/8236645789842
**取得日**: 2026-05-02 18:53:47 JST

## 内容

アクティベーション
貢献者
ジャーニーステップは、ユーザーに適切なタイミングで適切なコンテンツを提供します。ジャーニーステップには2種類あります：
インスタントステップ：インスタントステップは、挨拶メッセージのように考えることができます。設定されている場合、ユーザーはジャーニーに参加するとすぐにインスタントステップを受け取ります。
スケジュールステップ：スケジュールステップは、ユーザーのジャーニーの特定の日数に設定できます。
ユーザーのニーズを予測し、ジャーニーステップの作成を開始しましょう。
スタジオで、コンテンツ > ジャーニーに移動します。
コンテンツを追加したいジャーニーに移動します。
最初のステップを作成をクリックします。
ジャーニーステップコンテンツエディターが開きます。
ステップのタイトルを提供します。
ステップコンテンツの下に、希望するコンテンツを追加します。テキスト、メディア、リンク、ウィジェットなどを追加することで、コンテンツを充実させることができます。
スケジュールの下で、ジャーニーのユーザーが最初のジャーニーステップを即時に受け取るか、スケジュールされた時間に受け取るかを選択します。
インスタント：ユーザーはジャーニーに参加した瞬間にのみインスタント投稿を受け取ります。アクティブなジャーニーのためにインスタント投稿を作成し続けることができますが、既存のジャーニーユーザーはもはやインスタント投稿を受け取らず、新しいユーザーのみが受け取ります。たとえば、ユーザーがアプリに登録するとすぐにジャーニーを開始してほしい場合です。
スケジュール：スケジュールステップは、ユーザーがジャーニーを開始した後の特定の時間と日に表示されます。ジャーニーステップをスケジュールするには、ジャーニーの日の下でステップがユーザーに届くジャーニーの日を選択し、時間の下で特定の時間を選択します。
保存をクリックします。
ジャーニーステップが保存されました。
いつでもジャーニーの概要ページからステップを編集または削除したり、追加のステップを追加したりできます。
