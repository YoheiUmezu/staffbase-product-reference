# 従業員メールにおける連絡先属性をマッピングするためのフィールドの作成

**URL**: https://support.staffbase.com/hc/ja/articles/6328523121554
**取得日**: 2026-05-02 18:53:51 JST

## 内容

マネージャー
カスタム配信リストを作成するために、従業員メールウェブアプリで、以下のいずれかの方法を使用して連絡先を追加できます：
カンマ区切り値（CSV）ファイルのアップロード
Microsoft Entra ID（以前のAzure AD）との自動同期の設定
Workdayとの自動同期の設定
Staffbase アプリ/イントラネットからの自動同期の設定
すべての方法では、データソースから連絡先の属性をウェブアプリの対応するフィールドにマッピングする必要があります。
属性は、受信者を区別する特性であり、オーディエンスをセグメント化し、特定のグループに対してコミュニケーションをターゲットにすることを可能にします。
使用したい属性に一致するオプションがドロップダウンにない場合は、マッピングをスキップするか、この情報を保存するための新しいフィールドを作成できます。
注意：この機能は、従業員メールウェブアプリから直接メールを送信する場合にのみ利用可能です。アドインや拡張機能をインストールし、メールクライアントから追跡されたメールを送信する場合は、この文書は関連しません。
新しいフィールドの作成
新しいフィールドを作成をクリックします。
ダイアログが開きます。
新しいフィールド名を入力します。
以下のいずれかをデータタイプとして選択します：
テキスト
文字、数字、スペース、句読点、およびほとんどの記号を受け入れます。
数値
数字とドットのみを受け入れます。
日付（YYYY-MM-DD）
有効な日付形式である必要があります。
フィールドを作成をクリックします。
