# Staffbaseメールのデフォルトブランドカラーのカスタマイズ

**URL**: https://support.staffbase.com/hc/ja/articles/26307820295698
**取得日**: 2026-05-02 18:41:38 JST

## 内容

管理者
Staffbaseプラットフォームには、プラットフォームのデザインに合わせた2つのデフォルトカラーが含まれています。これらのカラーは、ブランドに合わせてカスタマイズできます。ただし、これらの最初の2つのデフォルトカラーは削除できないことに注意してください。
スタジオで、設定 > メールに移動します。
カラーをクリックします。
カラーページが開きます。
カラー名の下で、最初のフィールドをクリックしてカラーの名前を編集します。
対応する値フィールドで、カラーの16進コードを更新します。
2番目のカラーについても、ステップ3と4を繰り返します。
保存をクリックします。
Staffbaseメールのデフォルトブランドカラーを正常にカスタマイズしました。これらの2つのカラーは削除できませんが、いつでも更新できます。また、追加のデフォルトカラーを追加することもできます。
