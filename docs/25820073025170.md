# ファイル使用状況でファイルの場所を確認する

**URL**: https://support.staffbase.com/hc/ja/articles/25820073025170
**取得日**: 2026-05-02 19:00:07 JST

## 内容

ファイルマネージャーのファイル使用状況機能では、各ファイルがページやニュース投稿記事のどこで使用されているかを一元的に確認できます。これにより、ファイルの場所を手動で検索する必要がなくなり、ファイル管理や更新が簡単になります。ファイル使用状況を活用することで、正確な配信、デジタル資産全体でのコンテンツの一貫性維持、そしてコンテンツの最新化と整理を効率的に行うことができます。
ファイル使用状況は、ニュース投稿記事やアクセス権のあるページで実際に使用されているファイルに対して利用できます。各コンテンツはリスト形式で表示され、Studio内の正確な場所へのリンクが付いています。これにより、プラットフォーム内でコンテンツを簡単に確認し、更新することができます。コメントやチャットメッセージなど、他の場所でアップロードされたファイルは追跡されません。
ファイル使用状況にアクセスする方法
Studioでファイルに移動します。
ファイルマネージャーページが開きます。
使用状況を確認したいファイルの三点リーダーをクリックします。
インサイトをクリックします。
インサイトダイアログが開きます。
ファイル使用状況に移動します。
必要に応じて、リンクをクリックしてStudio内のコンテンツに移動します。
ファイルが使用されている場所を確認できました。
