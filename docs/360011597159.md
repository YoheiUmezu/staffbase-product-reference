# サインイン保護されたコンテンツの埋め込み

**URL**: https://support.staffbase.com/hc/ja/articles/360011597159
**取得日**: 2026-05-02 18:56:06 JST

## 内容

プラットフォーム外からサインイン保護されたコンテンツやサービスをリンクまたは埋め込む場合、シームレスなユーザー体験を提供するオプションの一つとして、SAML統合プラグインがあります。IT部門がSAML統合プラグインで接続を設定すると、Staffbaseプラットフォームにサインインしているアプリユーザーは、自動的にサインイン保護されたコンテンツやサービスにアクセスできます。SAML統合プラグインは、業界標準のSAMLプロトコルを使用して、2つのシステム間でユーザーのアイデンティティを認証します。これは、機密性の高い会社データをStaffbaseプラットフォーム内でアクセス可能にしたり、第三者サービスに安全にアクセスできるようにする貴重な機能です。
SAML統合プラグインを使用したサインイン保護されたコンテンツとサービスの埋め込み
IT部門からサポートを受けて、SSOサインインのためにSAML統合プラグインを設定します。
接続が設定されたら、公開前に最終調整を行うことができます。
スタジオでSAML統合に移動します。
埋め込みたいSAML接続をクリックします。
オプションで、ライブプレビューを使用して、埋め込まれたコンテンツやサービスがユーザーにどのように表示されるかを確認します。
オプションで、ギアアイコンをクリックして設定を開き、このSAML統合の可視性、編集者、またはその他の設定を構成し、保存します。
公開をクリックして、サインイン保護されたコンテンツやサービスを埋め込みます。
埋め込まれたサービスを含む新しいメニュー項目がプラットフォームに追加されます。
外部の埋め込まれたコンテンツがStaffbaseプラットフォームで正しく表示されない場合は、正しいクッキー設定がされていることを確認してください。
2021年末から、多くのブラウザはクッキーに対して以下の設定を要求します：
Set-Cookie: session=a-session; SameSite=None; Secure
サインイン保護されたサービスのサブページへのディープリンク
SAML統合プラグインを使用して埋め込む大きな利点の一つは、外部コンテンツやサービスのサブページに直接リンクできることです。これをディープリンクと呼びます。外部コンテンツやサービスのサブページはプラットフォーム内に埋め込まれ、アプリ内の通常のページのように表示されます。ユーザーはサインインを認識せず、コンテンツやサービスが外部ソースから提供されていることにも気づきません。
前提条件
外部コンテンツやサービスのサブページへの完全なリンクを持っていること。
外部コンテンツの提供者がSAML RelayStateをサポートしていること。SAML RelayStateに関する詳細は、公式のSAMLドキュメントを参照してください。
SAML接続を構成し、IT部門の助けを借りて設定します。
リンクしたいサブページへのアクセスを提供するSSO接続に移動します。
ギアアイコンをクリックして設定を開きます。
アプリ内の可視性について、SAML接続をすべてのユーザーのために構成します。
メニューに表示について、デスクトップとモバイルのアイコンの選択を外します。
保存をクリックします。
公開をクリックします。
SAML接続は機能しますが、メニュー内でユーザーには表示されません。サブページへのアクセスをユーザーに提供するために、ディープリンクを構成します。
Pレビューをクリックします。
リンクをコピーします。
リンクをテキスト編集ツールに貼り付けます。
リンクにはスペースや改行が含まれてはいけません。テキストエディタでリンクが新しい行に続く場合は問題ありません。
リンクを次のように調整します：
リンクをアプリ内で使用する場合は、サブページへのパスを末尾に追加します。例：
https://app.yourcompany.com/content/saml/contentID123456/path-to-sub-page
リンクをアプリ外で使用する場合は、パスを追加し、さらにカスタムドメインの後にオープンリンクパラメータを追加します。例：
https://app.yourcompany.com/openlink/content/saml/contentID123456/path-to-sub-page
これで、ディープリンクを使用できるようになりました。例えば、リンクプラグインやリンクウィジェットで使用できます。作成したディープリンクをターゲットURLとして貼り付けてください。
