# 投稿の作成

**URL**: https://support.staffbase.com/hc/ja/articles/207136695
**取得日**: 2026-05-02 19:02:24 JST

## 内容

ニュースは、Staffbaseアプリとイントラネットで視聴者ごとに動的なコンテンツを公開できます。コンテンツを効率的に整理し、従業員に分かりやすい形式で届けます。チャンネル内でニュース投稿記事を作成して公開できます。コンテンツ形式の外観は、チャンネルタイプによって異なります。本記事では、記事チャンネルタイプ内で投稿を作成します。
記事のテキスト、タイトル、ティーザー テキストには、任意でAI搭載のコンパニオンエディターを使用できます。
Studioで、コンテンツ > ニュース > 投稿を追加したい記事チャンネルに移動します。
投稿を作成するをクリックします。
記事を作成するページが開きます。
記事のタイトルを入力します。
記事のテキストを入力します。
記事の画像または動画を選択またはアップロードします。
任意で、ティーザー テキスト、メディア、添付を追加します。
下書きを保存するをクリックします。
任意で、プレビューをクリックして、ユーザーにどのように表示されるか確認します。
任意で、他の言語でコンテンツを追加することもできます。
下書きの投稿を作成しました。ユーザーに表示するには、まず公開する必要があります。
