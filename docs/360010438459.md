# ユーザーグループに管理者を追加する

**URL**: https://support.staffbase.com/hc/ja/articles/360010438459
**取得日**: 2026-05-02 18:54:25 JST

## 内容

管理者
ユーザーグループの設定、メンバー、およびチャットを管理するために、任意のユーザーグループに1人以上のグループ管理者を割り当てることができます。別のユーザーグループの管理者として、ユーザーグループ全体を割り当てることも可能です。管理者は、管理者として割り当てられるためにグループのメンバーである必要はありません。ユーザーグループが最新の状態に保たれるように、管理者を割り当ててください。
スタジオで、ユーザー設定 > ユーザーグループに移動します。
管理者を追加したいグループを選択します。
グループを編集をクリックします。
グループを編集するためのダイアログが開きます。
管理者フィールドで、管理者に任命したいユーザーを1人以上選択します。
クリックして 保存します。
ユーザーグループに管理者を追加しました。
