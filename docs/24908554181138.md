# スペースの編集

**URL**: https://support.staffbase.com/hc/ja/articles/24908554181138
**取得日**: 2026-05-02 18:55:19 JST

## 内容

この記事の情報は、従業員アプリまたはStaffbaseイントラネットを使用している場合、及びStaffbaseメールが従業員アプリまたはStaffbaseイントラネットと一緒に使用されている場合に適用されます。Staffbaseメールをスタンドアロン製品として使用している場合は、Staffbaseメールでのスペースの編集を参照してください。
アプリおよびイントラネットユーザーとして
スタジオで、コンテンツ > スペースに移動します。
スペースページが開きます。
編集したいスペースに移動し、スペースを編集をクリックします。
スペースを編集ダイアログが開きます。
スペースの詳細を編集します。
保存をクリックします。
スペースを正常に編集しました。
アプリおよびイントラネットおよびStaffbaseメールユーザーとして
管理者
スタジオで、設定 > スペースの概要に移動します。
スペースの概要ページが開きます。
編集したいスペースに移動します。
オプションメニューを開くために3つのドットをクリックします。
スペースを編集をクリックします。
スペースを編集ページが開きます。
スペースの詳細を編集します。
保存をクリックします。
スペースを正常に編集しました。スペースの概要ページに自動的にリダイレクトされます。
スペース管理者として、スペースの選択した詳細を編集できます。コンテンツ > スペース > 編集したいスペースに移動します。スペースを編集をクリックします。スペースを編集ページが開きます。ここで、スペース名を編集したり、スペース管理者を追加または削除したりできます。
