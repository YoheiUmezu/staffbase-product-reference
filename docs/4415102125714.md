# 従業員メールへの新規ユーザーの追加

**URL**: https://support.staffbase.com/hc/ja/articles/4415102125714
**取得日**: 2026-05-02 18:51:50 JST

## 内容

管理者
組織内のコミュニケーターが 従業員メール を使用したい場合や、送信および追跡したい共有コミュニケーション用のメール アドレスのために新しいユーザーアカウントを作成します。
統合された アプリ/イントラネット および メール を持つ組織でユーザーを管理する際、スタジオのメインユーザー設定メニューとこのメニューに埋め込まれたチームページは異なる機能を果たします。
スタジオ を使用して：
ユーザーを追加
ユーザーを削除
ユーザー情報を更新
チーム ページを使用して：
メール機能へのユーザーのアクセスを管理します。たとえば、送信、スタイリング、レポートの生成など
メールコラボレーションのためのグループメンバーシップを編集
他のユーザーとしてサインインします。たとえば、共有メールアドレス用に作成された共有ユーザーアカウント
新しい個別ユーザー
チーム ページで 新しいユーザー をクリックします。
新しいユーザーの 名、姓、および メールアドレス を入力します。
このアカウントは、組織内の別の人のためのものか、送信および追跡したい別のメールアドレスのためのものです。
注: 無効化された 従業員メール ユーザーアカウントのメールアドレスは、新しいアカウントを作成するために使用できません。ユーザーアカウントを再アクティブ化するには、Staffbaseサポートに連絡してください。
ドロップダウンメニューから 部門 を選択します。
既存の グループ を選択します。
ユーザーが特定の既存グループに所属する必要がない場合は、デフォルト グループを選択します。
共有ビュー が有効になっている場合、グループのすべてのメンバーは、グループのメンバーによって送信されたメールや、そのグループのユーザーによって作成されたドラフトにアクセスできます。
マネージャーは、同じグループに所属するマネージャーまたはベースユーザーの特定の設定を管理できるため、特定のグループ構成が必要な場合があります。
ユーザーを新しいグループに追加する必要がある場合は、新しいグループを作成 をクリックし、グループ名を入力してチェックマークをクリックして保存します。
適切な権限を付与するために アカウントタイプ を選択します。
ユーザー権限について詳しく学ぶ。
ユーザーを作成 をクリックします。
新しいユーザーのリスト
新しいユーザーのリストを含むカンマ区切り値 (CSV) ファイルを作成します。 従業員メール の標準化された形式で。
ほとんどのスプレッドシートアプリケーションでこのタイプのファイルを作成および編集できます。
たとえば: John, Smith, john.smith@yourcompany.com, BASE_USER。
新しいユーザー ページのドロップダウンメニューから1つ以上の 部門 名をコピーし、リストされた各新しいユーザーの最終スプレッドシート列に追加します。
チーム ページで ユーザーをインポート をクリックします。
プロンプトが表示されたら、CSVファイルを選択するか、ドラッグアンドドロップします。
追加されるすべてのユーザーの既存の グループ を選択します。
特定の既存グループに所属する必要がない場合は、デフォルト グループを選択します。
確認 をクリックします。
ユーザーのリストを追加すると、その新しいプロファイルが チーム ページに表示されます。
誰かがすぐに表示されない場合、そのメールアドレスに対応するアカウントが以前に削除されている可能性があります。
注: 無効化された 従業員メール ユーザーアカウントのメールアドレスは、新しいアカウントを作成するために使用できません。ユーザーアカウントを再アクティブ化するには、Staffbaseサポートに連絡してください。
