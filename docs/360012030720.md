# 部分CSVタグの作成

**URL**: https://support.staffbase.com/hc/ja/articles/360012030720
**取得日**: 2026-05-02 19:02:00 JST

## 内容

アクティベーション
このインポート方法はレガシーと見なされており、将来的に廃止される予定です。その代わりに、Staffbaseはできるだけ早く[新しい] CSVインポート方法への切り替えを推奨しています。[新しい] CSVインポート方法は、先進的な機能とより良いユーザーエクスペリエンスを備えたベータ機能です。
部分CSVタグは、部分CSVインポートを実行する際にユーザーのサブセットを区別するものです。各支店が個別に部分CSVインポートを実行する場合、各支店は更新するユーザーのサブセットを識別するために別々のタグが必要です。各管理者は、部分CSVインポートの開始時に対応するタグを選択します。管理者は常に自分のタグを使用し、毎回の部分CSVインポートで新しいタグを作成しないことが重要です。タグが明確かつ一貫して命名されるようにするために、Staffbaseはすべての管理者が遵守できる命名規則を確立することを推奨しています。
スタジオで、ユーザー設定 > ユーザーのインポートと同期に移動します。
新しいインポートをクリックします。
このタグに関連付けたいユーザー情報を含むCSVファイルをアップロードします。
詳細設定 > これらのユーザーに適用の下で、ドロップダウンメニューを使用して新しいタグを使用…を選択します。
新しいタグを作成するためのダイアログが開きます。
新しい部分インポートタグの名前を提供します。
このタグには認識可能な名前を付けて、他の管理者が誤って使用しないようにしてください。誤ったタグを使用すると、ユーザーが無効化されたり重複したりする可能性があります。
OKをクリックします。
続行をクリックし、通常通りCSVインポートを完了します。ステップ7から始めてください。
完了すると、作成したこの部分CSVタグがCSVファイルのすべてのユーザーに適用されます。
