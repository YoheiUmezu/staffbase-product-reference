# サポートされているモバイルアプリのバージョン

**URL**: https://support.staffbase.com/hc/ja/articles/4414688840594
**取得日**: 2026-05-02 18:44:10 JST

## 内容

Staffbaseは、AndroidおよびiOSの従業員アプリを毎年4回のメジャーバージョンと、毎月のマイナーリリースで更新しています。組織が最新の状態かどうかは、こちらのアプリバージョンスケジュールでご確認ください。
Staffbaseは、モバイルアプリの直近4つのメジャーバージョンをサポートしています。
スマートデバイスは、少なくとも年に1回は新しいアプリバージョンに更新する必要があります。更新しない場合、従業員アプリへのアクセスができなくなります。
アプリのリリースにより、アプリが常に最新の機能を備え、従業員のセキュリティや組織のコンプライアンスを最高水準で保護します。
アプリの管理者として、あなたは組織内で近日中のリリースを積極的に告知することで、従業員が最新のアプリバージョンに更新するサポートができます。モバイルアプリの更新の2週間前に、組織にはリリースのお知らせと新機能のリストが届きますので、組織内で共有してください。
