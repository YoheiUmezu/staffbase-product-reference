# 従業員メールのオープンおよびクリック追跡のテスト

**URL**: https://support.staffbase.com/hc/ja/articles/4415102162834
**取得日**: 2026-05-02 18:55:49 JST

## 内容

オープンおよびクリック追跡のテストは、従業員メールが正しく設定されていることを確認するための素晴らしい方法です。テスト追跡に関する情報を以下に示しますので、何を期待すべきかを理解してください。
メールを追跡するには、送信ワークフローの一部であるオプションを使用してください。
クリック追跡をテストするには、送信するメッセージに少なくとも1つのリンクを含めるようにしてください。
受信した受信トレイからテストメールを開き、メッセージに含まれているリンクをクリックします。
時々、メールプログラムはデフォルトで画像をブロックします。画像を表示することは、オープンを判断する方法の重要な部分です。
テストオープンがカウントされるように、メールクライアントで画像を表示または同等のオプションをクリックして、メッセージ内にトラッキングピクセルが表示されるようにしてください。
ブラウザで、送信したテストメールの詳細を表示して、追跡されたオープンおよびクリックがどのように表示されるかを確認します。
Staffbaseは、従業員メールが追跡されたアクティビティを登録するまで、少なくとも30分待つことを推奨します。
