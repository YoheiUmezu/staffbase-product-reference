# Staffbase従業員アプリ向けMicrosoft Intune SDK統合の概要

**URL**: https://support.staffbase.com/hc/ja/articles/27439844952850
**取得日**: 2026-05-02 18:41:32 JST

## 内容

Microsoft Intune SDK の統合により、モバイルアプリケーションでエンタープライズモビリティとデータ保護が強化されます。スタッフベース従業員アプリへの統合により、Microsoft Intune をモバイルデバイス管理（MDM）やモバイルアプリケーション管理（MAM）に利用している組織の既存のITインフラとシームレスに連携できます。
この統合により、IT管理者はユーザーエクスペリエンスを損なうことなく、企業アプリやデータの設定や保護を行うことができます。スタッフベースは、統合の利用に必要なサービスをサポートしています。Microsoft Intune を従業員アプリと併用することで、組織は幅広いセキュリティおよびコンプライアンスポリシーを適用できるようになります。これには、以下のようなアプリ保護設定が含まれます：
コピー＆ペーストアクションの制限
PINアクセスの必須化
スクリーンショットの防止
準拠したデバイスのみがアプリにアクセスできるようにする
Microsoft Authentication Library（MSAL）は、MS Intune を利用したスタッフベース従業員アプリに対して、安全で最新の認証を提供します。すでに MS Intune でデバイスやアプリを管理している場合、同じポリシーを従業員アプリにも直接適用できます。MSAL を利用することで、ユーザーはより安全でスムーズなサインイン体験を得られ、最新の認証基準や条件付きアクセス ポリシーにも対応します。
このような管理性の高さにより、企業はセキュリティ基準を維持しながら、柔軟でモバイルファーストなワークフローをサポートしやすくなります。
