# アナリティクスダッシュボードでメールの指標を分析する

**URL**: https://support.staffbase.com/hc/ja/articles/25722108756626
**取得日**: 2026-05-02 18:58:41 JST

## 内容

アクティベーション
編集者
アナリティクスダッシュボードは、ユーザーがStaffbaseプラットフォームとどのように、いつ、どこで関与しているかについての深い洞察を提供します。アナリティクスダッシュボードは、メトリクスを深く掘り下げることができるように、さまざまなカテゴリに分かれています。
この記事では、メールタブの詳細、グラフや分析データを説明します。各メールには詳細なメトリクスが含まれており、Staffbaseメールアナリティクスの概要で詳しく学べます。 アナリティクスダッシュボード全体について、ユーザー役割のアクセスを含めて、アナリティクスダッシュボードの概要で詳しく学べます。
フィルター
メールタブの上部では、さまざまなフィルターを使用してデータを絞り込むことができます。フィルターを選択すると、メールタブ内のすべてのグラフに適用されます。日付範囲フィルターの場合、デフォルトの日付範囲は先週です。フィルタリングについて詳しく学ぶ。
トップメトリクス
ダッシュボードの上部には、概要を素早く把握するための4つの指標があります。アナリティクス用語について詳しくは、 アナリティクス用語集をご覧ください。
送信済みメール
受信者
開封率
クリック率
この指標は、選択された期間内に送信されたメールの数を表示します。
この指標は、少なくとも1通のメールを受け取ったユニークユーザーの数を表示します。
この指標は、送信されたメールのうち開封された割合を表示します。
この指標は、開封されたメールの中でリンクがクリックされた割合を表示します。
開封率とクリック率の横にあるパーセンテージは、過去7日間の傾向を示しています。例えば、先週の開封率またはクリック率が50%で、今週が55%の場合、5%の増加を示します。
グラフ
メールタブには、詳細な指標を示す4つのグラフがあります。
時間経過によるメール別総アクティビティグラフ
時間経過によるメールの総アクティビティグラフには、メールの総アクティビティとアクティビティ率を切り替えるためのドロップダウンがあります。グラフのいずれかのバージョンで任意の時点にカーソルを合わせると、追加の詳細が表示されます。
総アクティビティ 折れ線グラフには、次の指標が表示されます：
総受信者数
総開封数
トータルクリック数
アクティビティ率の棒グラフには、次の指標がパーセンテージで表示されます：
開封率
クリック率
グラフ上の特定の日付にカーソルを合わせると、選択した指標に応じてその日付の特定のデータを確認できます。フィルターを適用したり、指標を削除したりして、結果をさらに絞り込むことができます。
時間経過によるメールの総アクティビティと時間経過によるメールのアクティビティ率のグラフは、さまざまなメール指標に関する詳細な統計を表示します。各指標には、グラフ内で対応する色があります。デフォルトでは、グラフの両方のバージョンがすべての指標を表示します。このようにして、オプションがグラフ全体でどのように重なるかを確認できます。1つ以上の指標を選択解除して、グラフをカスタマイズできます。これにより、たとえば総受信者数とトータルクリック数など、特定の領域間の比較が可能になります。
グラフの右側の値も、選択した指標やフィルターに応じて適切な高値にシフトします。たとえば、総受信者数の指標が高い値を持っていて、それを選択解除すると、グラフの最高値は自動的に次の最高値に調整されます。グラフの下部の値も、選択した時間枠に応じて自動的にシフトします。1年などの大きな時間枠の場合、時間は月単位で表示されます。1週間などの短い時間枠の場合、時間は日単位で表示されます。
メール指標の選択解除
Studioで、アナリティクス > メールに移動します。
時間経過によるメール別総活動グラフに移動します。
グラフで非表示にしたいメトリックの横にあるチェックボックスをクリックします。
メトリックが削除され、グラフに表示されなくなります。
トータルオープン数
トータルオープン数の図は、ユニークオープンとリピートオープンに分けられたトータルオープン数を表示します。
各タイプにカーソルを合わせると、正確なオープン数とトータルに対する割合が表示されます。
トータルクリック数
トータルクリック数の図は、ユニーククリックとリピートクリックに分けられたトータルクリック数を表示します。
各タイプにカーソルを合わせると、正確なクリック数とトータルに対する割合が表示されます。
メールリスト
メールリストは、プラットフォーム内のメールの詳細な概要です。このリストには、メールの受信者、開封数、トータル開封数、クリック数、トータルクリック数、メール統計ページへのナビゲーションアイコンが表示されます。個別のメールアナリティクスについて詳しく学びましょう。
リストは毎日更新されます。メトリクスの処理と表示には最大24時間かかることがあります。
異なるテーブルヘッダーは次のとおりです：

デフォルトでは、テーブルは開封数でソートされています。特定のヘッダーをクリックすることで、さらにテーブルをソートできます。
メールリストのソート
Studioで、アナリティクス > メールに移動します。
メールリストテーブルに移動します。
リストをソートしたいヘッダーをクリックします。
リストは選択したヘッダーで自動的に降順にソートされます。
オプションで、同じヘッダーを再度クリックしてリストを昇順に再ソートします。
特定のヘッダーでメールリストをソートしました。メールタブから離れると、結果は自動的に開封数で表示されるようにリセットされます。
