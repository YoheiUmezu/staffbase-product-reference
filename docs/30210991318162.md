# Staffbaseメールにおける読了時間データの概要

**URL**: https://support.staffbase.com/hc/ja/articles/30210991318162
**取得日**: 2026-05-02 18:50:56 JST

## 内容

有効化
Staffbaseメールの読了時間は、受信者がメールにどれだけ積極的にエンゲージメントしているかを把握するのに役立ちます。このインサイトにより、どのメッセージが注目を集めているか、またエンゲージメントがどこで低下しているかを特定できます。特に、重要なお知らせや長めのニュースレター、定期的な社内更新の効果を評価する際に有用です。
Staffbaseは個々のメールの読了時間を提供します。メール 概要 の「最近送信済み」や、各フォルダの「送信済み」タブで、送信済みメールの統計にアクセスできます。「統計」タブでは、読了時間のダイアグラムで、合計開封数に基づく受信者の割合が表示されます。このダイアグラムでは、受信者がメールをどのくらいの時間閲覧したかによって、「一読」「流し読み」「熟読」の割合が示されます：
熟読：9秒以上積極的に閲覧されたメール
流し読み：3～8秒積極的に閲覧されたメール
一読：1～2秒積極的に閲覧されたメール
読了時間データが利用可能な場合
読了時間データは、受信者のメールクライアントがこの機能をサポートしている場合のみ生成されます。この機能をサポートするには、画像プロキシを使用せずにトラッキングピクセルを表示する必要があります。メールクライアントが画像をブロックまたはリダイレクトする場合、読了時間は記録できません。
