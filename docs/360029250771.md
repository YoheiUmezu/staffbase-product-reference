# ニュースチャンネルの更新情報の概要

**URL**: https://support.staffbase.com/hc/ja/articles/360029250771
**取得日**: 2026-05-02 18:57:32 JST

## 内容

新しいチャンネルを作成する際、著者がメッセージの構成を気にせずに個人的な内容を迅速に投稿できるようにするために、チャンネルタイプ 更新 を選択してください。更新投稿は、ユーザーが短いコンテンツを通じて活発で意味のある交流を行うためのソーシャルウォールとして機能します。
更新に最適なコンテンツは何ですか？
更新チャンネルでは、簡潔な通知を作成できます。たとえば、シフトスケジュールの変更や会社のイベントに関する情報など、従業員に組織の更新を提供するためです。また、CEOブログのような掲示板やマイクロブログとしても理想的に機能します。
短いテキスト
テキストは更新投稿の上部に表示され、最も多くの場合、短く要点を押さえたものです。更新の内容はしばしば時間に敏感で、短期間のみ関連性があります。そのため、洗練されたデザインを作成することよりも、情報をできるだけ早くユーザーに提供することに重点が置かれています。
オプションのメディア
短いテキストメッセージの下に、短いテキストを補完するために、いくつかの画像や動画を追加するオプションがあります。 画像や動画が4つを超える場合、ギャラリーはユーザーにとって見やすい展開可能および折りたたみ可能な形式で表示されます。
