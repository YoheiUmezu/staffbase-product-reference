# ユーザープロファイルにおけるMicrosoft Teamsコラボレーションの概要

**URL**: https://support.staffbase.com/hc/ja/articles/16228295645202
**取得日**: 2026-05-02 18:43:26 JST

## 内容

アクティベーション
Staffbaseプラットフォームは、Microsoft Teamsとシームレスにコラボレーションする柔軟性を提供します。Microsoft Teamsを頻繁に使用する組織内のチームの生産性とコラボレーションを向上させます。
ユーザープロファイルページからボタンをクリックするだけで、Microsoft Teamsのチャット、音声通話、またはビデオ通話オプションを使用してシームレスにコラボレーションできるようにします。
この統合は、日常的にMicrosoft Teamsを使用してコラボレーションする特定のチームが組織内にある場合に役立ちます。また、Teams Collaborateオプションを特定のチームのみに利用可能にすることもできます。
ユーザーにとってのMicrosoft Teamsでのコラボレーションの見え方
Staffbaseプラットフォームの再設計されたユーザープロファイルページでは、ユーザーがMicrosoft Teamsのドロップダウンを使用できます。サインインしたユーザーは、公開メールアドレスとして有効なMicrosoftメールアドレスが登録されている他のユーザーに連絡できます。
Microsoft Teamsのドロップダウンは、他のユーザーのプロファイルにのみ表示されます。自分のユーザープロファイルには表示されません。
ユーザーがMicrosoft Teamsをクリックすると、次のオプションを含むドロップダウンが開きます：
チャット
通話
ビデオ通話
オプションをクリックすると、ユーザーはMicrosoft Teamsの該当オプションに移動します。
