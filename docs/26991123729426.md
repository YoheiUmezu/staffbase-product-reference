# ミールプランプラグインの概要

**URL**: https://support.staffbase.com/hc/ja/articles/26991123729426
**取得日**: 2026-05-02 18:41:33 JST

## 内容

ミールプランプラグインを使うと、従業員にカフェテリアや近隣レストランで提供されるメニューを簡単に知らせることができます。編集者は、週ごとや今後のミールプランを公開でき、各メニューにはタイトル、説明、価格、日付を含めることができます。このプラグインはカテゴリ分けや無制限のメニュー登録に対応しており、多様なメニューにも最適です。
メニューの追加は迅速かつ効率的で、編集者はタイトルを入力することで以前作成したエントリーを再利用できます。プラグインは自動的に日付や曜日ごとにメニューを整理し、編集者はドラッグ＆ドロップで各日の順序を調整できるため、柔軟に管理できます。これにより、編集者はリアルタイムの更新や変更を反映した、分かりやすく魅力的な構成を維持できます。
ミールプランは、ユーザーがStaffbase従業員アプリやイントラネットを利用し始めるきっかけとして最適です。エンゲージメントを促進し、プラットフォームに慣れてもらうのに役立ちます。日常的に役立つ情報を提供することで、アプリの他の機能を知る入口にもなります。
