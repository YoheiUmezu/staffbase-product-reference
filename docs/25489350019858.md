# チャットの概要とFAQ

**URL**: https://support.staffbase.com/hc/ja/articles/25489350019858
**取得日**: 2026-05-02 18:41:55 JST

## 内容

チャットは、アプリ内での直接的な1対1またはグループメッセージングに最適なツールです。チャットは、同僚間のカジュアルなコミュニケーションを促進し、詳細な情報を迅速かつ直接に取得するためのシンプルなソリューションを提供します。チャットで利用可能な機能について詳しく学びましょう。
チャットではどのような会話が可能ですか？
1対1のチャット会話
これは、あなたと他の1人のユーザーとのプライベートな会話です。
グループチャット会話
グループチャット会話には、2人以上の個別ユーザー、ユーザーグループ、または個別ユーザーとユーザーグループの組み合わせが含まれることがあります。
新しいチャット会話を始めるとき、ユーザーをどのように検索できますか？
ユーザーは、名または姓で検索できます。組織の設定によっては、部門、職位、場所、その他の詳細でユーザーを検索することもできる場合があります。
グループチャット会話の最大参加者数は何人ですか？
チャット会話には最大128人の個別参加者が含まれます。ユーザーグループは1つの参加者としてカウントされます。
ユーザーグループを使用して128人以上のユーザーをチャット会話に追加することは技術的には可能ですが、Staffbaseはセキュリティと物流の理由からこの制限を設けています。
チャットでどのようなファイルを送信できますか？
モバイルおよびウェブアプリでさまざまな種類とサイズのファイルを送信できます。
チャットにアップロードされたすべての動画はmp4形式に変換され、720pにダウンサイジングされます。

チャットメッセージからメディアをダウンロードできますか？
チャットメッセージ内で送信された画像はダウンロードできます。モバイルアプリとウェブアプリの両方で、画像ファイルをダウンロードするには完全に開く必要があります。動画のダウンロードはできません。
チャットメッセージを編集できますか？
はい、組織で有効になっている場合、自分のチャットメッセージを編集できます。メッセージは必要なだけ何度でも編集できます。メッセージを編集した後、他のユーザーがメッセージに変更があったことを知るために、メッセージの横に表示されるインジケーターがあります。
特定のチャットメッセージを引用して返信できますか？
はい、会話内の特定のメッセージを引用して返信できます。詳細な手順は、チャットメッセージの引用で確認できます。
チャットメッセージや会話を削除できますか？
はい、組織で有効になっている場合、チャットメッセージや会話を削除できます。詳細な手順は、チャットメッセージまたは会話の削除で確認できます。
新しいメンバーはグループチャット会話の以前のチャット履歴を見ることができますか？
はい、新しいメンバーは、グループに参加する前に交換されたすべてのメッセージを含む完全なチャット履歴を見ることができます。
自分のチャットメッセージが読まれたかどうかをどうやって知ることができますか？
チャットには、1対1の会話用の既読通知機能があります。グループ会話にはまだ既読通知は利用できません。組織で有効になっている場合、会話が開かれると、最新のメッセージの下に目のアイコンが表示されます。また、個人設定で会話の既読通知を無効にすることもできます。こちらで詳しく学ぶ。
チャットメッセージのプッシュ通知を有効または無効にできますか？
はい、チャットのプッシュ通知をいつでも有効または無効にできます。詳細な手順は、チャットのプッシュ通知の設定で確認できます。
