# StaffbaseにCornerstone連携を追加する

**URL**: https://support.staffbase.com/hc/ja/articles/34769733663762
**取得日**: 2026-05-02 18:50:43 JST

## 内容

有効化
管理者
StaffbaseをCornerstone OnDemandと接続することで、学習コンテンツを従業員アプリやイントラネットに直接提供できます。Cornerstoneで設定を完了した後、StaffbaseスタジオでOAuth認証情報とディープリンク設定を使用して接続を構成できます。これにより、従業員は割り当てられたコースや推奨コースを閲覧し、進捗を追跡し、Cornerstoneが記録システムとして機能し続ける中で、シームレスにトレーニングを開始できます。
前提条件
Staffbaseにプラグインを追加しています。
CornerstoneのAPI管理でOAuthアプリケーションの設定を完了し、以下の情報を取得しています：
クライアント ID
クライアントシークレット
ユーザーマッピング用に、Cornerstoneのユーザーと一致するプロフィールフィールドを選択しています。
スタジオで設定 > 学習統合に移動します。
統合を追加をクリックします。
Cornerstone学習統合を追加ページが開きます。
ベースURLに、Cornerstone環境のベースURLを入力します。例：https://yourcompany.csod.com
クライアント IDに、CornerstoneでOAuth 2.0アプリケーションを登録した際に生成されたクライアントIDを入力します。OAuth 2.0アプリケーション。
クライアントシークレットに、クライアントIDと一緒にCornerstoneで生成されたOAuthクライアントシークレットを入力します。
ユーザーマッピングで、Cornerstoneの一意のユーザー識別子（User ID）と一致する値を含むStaffbaseのプロフィールフィールドを選択します。
一致するStaffbaseプロフィールフィールドがない場合は、マッピング用に新しく作成する必要があります。
作成するをクリックして接続を保存します。
統合が完了すると、HRサービスの関連機能がデフォルトで有効化されています。いつでも無効にすることができます。
