# ページの公開

**URL**: https://support.staffbase.com/hc/ja/articles/360027886812
**取得日**: 2026-05-02 19:01:32 JST

## 内容

ページを作成したら、ターゲットオーディエンスに見えるように公開できます。
メニューでのページの位置付け
作成した新しいページは、自動的にメニューのリストの一番下に表示されます。
スタジオで、コンテンツ > メニューに移動します。
ページをメニュー内の希望の位置にドラッグ＆ドロップします。フォルダーにページをドロップすることもできます。
保存をクリックします。
ページの公開
スタジオで、作成したばかりのページに移動します。
ページを公開をクリックします。
