# Gmailアカウントを従業員メールに接続する

**URL**: https://support.staffbase.com/hc/ja/articles/4415095674258
**取得日**: 2026-05-02 18:53:56 JST

## 内容

Gmailアカウントを従業員メールに接続すると、ドラフト、テンプレート、フォルダー、その他のリソースに、当社の拡張機能を使用してメールクライアントから直接アクセスできます。従業員メールの連絡先はGmailアドインに同期されないことに注意してください。連絡先を同期させるには、Google Contactsに連絡先を追加する必要があります。
手順
従業員メールウェブアプリの設定ページで、Email Connectボタンをクリックします。
このページからGmailと同期を選択し、Email Connectの設定を完了するための指示に従います。
Email Connectがアカウントと正常に同期された場合、メインのEmail Connectページに緑のバナーが表示され、アカウントが同期されたことを示します。
