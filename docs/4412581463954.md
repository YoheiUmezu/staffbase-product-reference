# メールデザインにおける画像の編集

**URL**: https://support.staffbase.com/hc/ja/articles/4412581463954
**取得日**: 2026-05-02 18:47:17 JST

## 内容

Employee Emailでデザインを編集する際は、内蔵の画像エディターを使用して、画像が美しく見えるようにし、読者を意図したページにリンクし、必要に応じて代替テキストを含めてください。

画像エディターメニューのナビゲート方法
画像をアップロードした後、Editボタンをクリックして画像エディターを開くことができます。画像エディター内には、いくつかのオプションが用意されています。
エディターウィンドウの上部には、UndoおよびRedoの矢印、さらにSaveおよびCancelボタンがあります。
画像の編集方法
トリミング
メールに画像を配置する際、全体の画像を含める必要がない場合や、希望する位置に配置されていない場合があります。画像をトリミングする際には、いくつかの選択肢があります。
トリミングメニューのCropセクションには、いくつかのプリセットアスペクト比が用意されています。特定のオプションを選択すると、アスペクト比が自動的に調整されます。画像のアスペクト比を維持しつつ、少しタイトにトリミングしたい場合は、Keep Resolutionのチェックボックスをオンにしてください。
画像の下には、さらにいくつかのボタンがあります。ここでは、画像を水平方向に反転したり、垂直方向に反転したり、0を左または右にクリック＆ドラッグして手動で画像を回転させたり、画像を90度左または右に回転させたりできます。
フィルター
画像の色を調整して、見た目を変更します。画像エディターのフィルターを使用します。
Filtersをクリックすると、いくつかのThemesが表示されます。これらのThemesの下には、選択肢が用意されています。それらの一つをクリックすると、適用したいフィルターの量を決定するためのスライダーが表示されます。
調整
画像を挿入する際に、修正したいわずかな欠陥に気付くことがあります。画像が暗すぎる、または少し色あせているように見える場合があります。良いニュースは、Adjustの下にこれらの問題を修正するための多くのオプションがあることです。
BrightnessやSaturationなどのBasicオプションを調整したり、HighlightsやExposureなどのRefinementsをさらに調整したりできます。
フォーカス
画像の特定の部分をさらに目立たせます。画像の一部に注意を引く素晴らしい方法は、興味のあるポイントの周りのフォーカスを調整することです。
画像エディターのFocusセクション内には、画像に歪みを追加するための素晴らしいオプションがあり、そのほとんどは移動やサイズ変更が可能です。これらのオプションをクリックすると、歪みの強度を選択できるスライダーが表示されます。
テキスト
Textツールを使用して、画像にコンテキストを簡単に追加できます。
左側のペインからこのオプションをクリックすると、画像エディターウィンドウの上部にNew Textボタンがすぐに表示されます。これをクリックして画像上にテキストボックスを作成し、ボックスをダブルクリックしてテキストを追加します。
テキストを配置したら、Font、Size、Alignment、Font Color、およびLine Spacingを調整できます。
追加のテキストボックスを追加する必要がある場合は、必要なだけNew Textをクリックするだけです！
ステッカー
画像にもう少し華やかさを加えましょう！
素晴らしいプリセットステッカーをチェックするか、Uploadボタンを使用して、コミュニケーションに独自のスタイルやブランディングを持ち込むことができます。
独自のステッカーをアップロードすることを選択すると、プリセットステッカーと独自のステッカーの間を迅速にナビゲートするためのCustomボタンが追加されます。
ブラシ
詳細を追加したり、画像の特定の部分に注意を引くためにブラシを使用します！このツールは、画像に手動で描画するために使用できます。
Brush Color、Brush Size、およびBrush Hardness（ブラシのエッジのテクスチャ）を変更するオプションがあります。ブラシツールで画像に線を追加したが、それを削除したい場合は、Undo矢印ボタンを使用してください。
フレーム
画像に境界線を追加し、視覚的な興味を高めます。このために、スタイルを追加するためのいくつかのプリセット画像フレームを提供しています。
左側のペインには、フレームスタイルをReplaceしたり、そのOpacityやWidthを変更するオプションがあります。
オーバーレイ
オーバーレイは、画像の見た目を楽しく興味深い方法で完全に刷新し、さまざまな効果のオプションを提供します。
左側のペインでオーバーレイを選択すると、そのオーバーレイのさまざまな効果をクリックして確認し、スライダーを使用してオーバーレイに適用される効果のレベルを決定できます。適用したオーバーレイを削除したい場合は、オーバーレイのリストの一番上までスクロールし、Noneを選択してください。
リンク
画像エディターを開くために使用するEditボタンのすぐ下に、Image Linkを挿入できるスペースがあります。ここに目的のURLを貼り付けることで、受信者が画像をクリックして、あなたが指示したいページに到達できるようにします。
代替テキスト
メッセージを表示している人が視覚障害を持っているため、画像が何であるかを簡単に判断できないことは珍しくありません。
代替テキストフィールドを使用して、画像に表示されている内容の書面による説明を提供してください。このアクセシビリティオプションは、銀行、教育、政府などの業界でのコミュニケーションに特に役立ちます。
