# ブリーフィングの使用

## 内容

ブリーフィングは、最初から必要なすべての重要な情報を確保することで、コンテンツ作成プロセスを効率化します。ブリーフィングを作成し、コンテンツ作成を担当するユーザーに割り当てます。ユーザーをブリーフィングに割り当てると、彼らはメール通知とスタジオ内のブリーフィングへのリンクを受け取ります。
概要ページには、作成されたブリーフィングがリストされます。あなたが作成したブリーフィングや、作成者によってあなたに割り当てられたブリーフィングを表示および編集できます。あなたが作成していない、または他のユーザーに割り当てられたブリーフィングは表示または編集できません。探しているブリーフィングを簡単に見つけるために、リストを最終更新日または締切でソートできます。
ブリーフィングは、シームレスなコラボレーションを確保するために、複数のユーザーに割り当てることができます。また、あなたが割り当てられたブリーフィングに他の受取人を追加することもできます。
ユーザーは、ブリーフィングへのアクセスリンクを含むメール通知を受け取るために、アカウントに有効なメールアドレスがリンクされている必要があります。
ブリーフィングの作成
スタジオで、計画 > ブリーフィング > ブリーフィングの作成に移動します。
ブリーフィングエディターが開きます。
件名の下に、ブリーフィングのタイトルを入力します。
対象読者の下に、1つ以上の対象者を入力します。
目的の下に、コミュニケーションの理由を入力します。
オプションで、コンテンツタイプの下に、必要なコンテンツの形式を提供します。
オプションで、言語の下に、コンテンツのための1つ以上の言語を選択します。
言語のドロップダウンには、あなたの組織で利用可能なすべてのコンテンツ言語が含まれています。
オプションで、トーンの下に、コンテンツのトーンを提供します。
オプションで、添付ファイルの下に、ファイルを参照をクリックしてファイルを追加します。
オプションで、追加情報の下に、さらに情報を追加します。
保存をクリックします。
ブリーフィングを作成しました。これで、1人以上のユーザーに割り当てることができます。
ブリーフィングの割り当て
ブリーフィングは、シームレスなコラボレーションを確保するために、1人以上のユーザーに割り当てることができます。
ユーザーは、ブリーフィングへのアクセスリンクを含むメール通知を受け取るために、アカウントに有効なメールアドレスがリンクされている必要があります。
締切の下に、コミュニケーションの終了日を提供します。
割り当て先の下に、ブリーフィングを割り当てるユーザーを検索または選択します。
保存をクリックします。
ブリーフィングを割り当てました。割り当てられたユーザーは、ブリーフィングへのリンクを含むメールを受け取りました。彼らは、ブリーフィングに提供された情報を使用してコンテンツを作成できます。
ブリーフィングの編集
あなたが作成したブリーフィングや、作成者によってあなたに割り当てられたブリーフィングを表示および編集できます。
スタジオで、計画 > ブリーフィングに移動します。
編集したいブリーフィングに移動します。
三点リーダーをクリックしてメニューを開きます。
ブリーフィングの編集をクリックします。
ブリーフィングエディターが開きます。
ブリーフィングの任意のセクションを編集します。
保存をクリックします。
ブリーフィングを編集しました。
ブリーフィングの削除
スタジオで、計画 > ブリーフィングに移動します。
削除したいブリーフィングに移動します。
三点リーダーをクリックしてメニューを開きます。
ブリーフィングの削除をクリックします。
ブリーフィングを削除しますか？ダイアログが開きます。
削除をクリックします。
ブリーフィングを削除しました。
