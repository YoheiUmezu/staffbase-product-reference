# Microsoft 365 ウィジェットの有効化

**URL**: https://support.staffbase.com/hc/ja/articles/4410113765010
**取得日**: 2026-05-02 18:52:04 JST

## 内容

管理者
Microsoft 365 ウィジェットは、Microsoft プラットフォームからのコンテンツやサービスを Pages 上のユーザーに提供します。ユーザーが Microsoft プラットフォームのウィジェット内のコンテンツを見ることができるように、Microsoft ウィジェットを有効化する必要があります。Microsoft 365 ウィジェットがすでにプラットフォームに追加されていると仮定します。この場合、編集者はそれらを有効化せずにページに追加することができます。しかし、ウィジェット内のコンテンツはユーザーに表示されません。その代わりに、管理者に連絡するようにというメッセージが表示されます。
前提条件
Microsoft 365 接続を追加しています。
スタジオで、設定 > サードパーティ接続に移動します。
Microsoft 365の下で、選択した Microsoft 365 ウィジェットのトグルを有効にします。
Microsoft 365 ウィジェットを有効化するためのダイアログが開きます。
有効化をクリックして、ユーザーのためにそれを有効にします。
次のステップ
Microsoft 接続に使用される Microsoft Entra アプリの権限を付与します。権限を付与すると、ユーザーはウィジェット内のコンテンツを見ることができます。
別の Microsoft 365 ウィジェットを有効化するたびに、Microsoft Entra アプリの権限を再度付与する必要があります。Staffbase は、すべての Microsoft 365 ウィジェットを一緒に有効化し、権限を一度だけ付与することを推奨します。
