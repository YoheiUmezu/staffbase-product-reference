# アプリとイントラネットの多言語オプション

**URL**: https://support.staffbase.com/hc/ja/articles/9718484610066
**取得日**: 2026-05-02 18:49:33 JST

## 内容

エディター
多言語オプションを使用すると、組織に設定された言語のいずれかでニュース、ページ、およびその他のプラグインのコンテンツを追加できます。
多言語オプションが利用できるプラグインは以下の通りです：
ニュース
ページ
フォーム
イベント登録
カレンダー
クイズカレンダー
調査
プラグインの多言語オプションは、組織に複数の言語が設定されている場合のみ利用可能です。
例えば、組織のために複数の言語を設定した後、ページは多言語設定を提供します。
複数の言語を使用する場合：ユーザーは何を見ることができますか？
ユーザーは、プロフィールメニューの設定 > 言語設定で、アプリまたはイントラネットコンテンツの言語を定義します。ユーザーが選択した言語がプラットフォームにどのように影響するかについては、言語管理の概要を参照してください。
多言語環境で全てのコンテンツを提供できない場合、デフォルト言語（選択した言語）が表示されます。つまり、英語や他の一般的に話される言語を代替として提供することができます。
インターフェース言語は、ユーザーのデバイスまたはブラウザの言語設定に基づいています。デバイスがインターフェース言語として利用できない言語に設定されている場合、デフォルト言語が代替として使用されます。
コンテンツが単一の言語で設定されており、ユーザーのデバイスと言語設定と異なる場合、これらの2つの設定はコンテンツに影響を与えません。この場合、コンテンツは作成された言語で表示されます。
