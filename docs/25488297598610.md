# リンクタイルウィジェットの設定

**URL**: https://support.staffbase.com/hc/ja/articles/25488297598610
**取得日**: 2026-05-02 18:48:43 JST

## 内容

エディター
リンクタイルウィジェットを使用すると、モバイルおよびウェブアプリにウェブサイトのリンクを追加できます。追加すると、これらのウェブサイトのリンクはタイルとして表示されます。ユーザーがタイルをクリックすると、それに関連付けられたリンクが開きます。
リンクタイルウィジェットを設定するには、以下の項目を設定する必要があります：
一般設定
リンクタイル
一般設定の構成
リンクタイルウィジェットを追加します。
リンクタイルウィジェットダイアログが開きます。
オプションで、タイトルを提供します。
タイトルはウィジェットの上に表示され、ユーザーに見えます。
ウィジェットが表示されるデバイスの種類を、対応するアイコンをクリックして選択します。
アプリ内の表示の下で、ウィジェットを表示する必要があるユーザーを選択します。
オプションで、モバイルブラウザーで開くのトグルを有効にします。
有効にすると、モバイルデバイスでは、定義されたターゲットURLが外部ブラウザーで開きます。
タイルの背景色とテキスト色を選択します。
一般設定のリンクタイルウィジェットを設定しました。次に、タイルをリンクできます。
リンクタイルの設定
リンクタイルのタイトルを提供します。
タイトルはユーザーに見えます。リンクタイルの内容を補完するタイトルを使用してください。
リンクしたいウェブサイトのURLを追加します。
オプションで、リンクタイルのアイコンを追加します。
標準アイコンライブラリからアイコンを選択するか、カスタムアイコンをアップロードするか、アイコンを含めないこともできます。
オプションで、ステップ1-3を繰り返してさらにリンクタイルを追加します。
すべてのリンクタイルを追加したら、OKをクリックします。リンクはいつでも調整できます。
リンクタイルウィジェットがページに追加されました。
