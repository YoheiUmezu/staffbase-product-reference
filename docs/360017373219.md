# ページテンプレートの削除

**URL**: https://support.staffbase.com/hc/ja/articles/360017373219
**取得日**: 2026-05-02 19:00:28 JST

## 内容

価格
アクティベーション
管理者
スタジオで、設定に移動します。
ページテンプレートをクリックします。
テンプレートの概要が表示されます。
アクション列で、削除したいページテンプレートの削除をクリックします。
ページテンプレートの削除を確認するダイアログが表示されます。
削除をクリックします。
ページテンプレートは完全に削除されました。エディターは新しいページを作成する際に、このオプションをテンプレートのドロップダウンで見ることができなくなります。ページテンプレートを削除しても、すでにそのテンプレートを使用して作成されたページには影響しません。
