# アプリまたはイントラネットを介して従業員のメールにアクセスする

**URL**: https://support.staffbase.com/hc/ja/articles/4404961967890
**取得日**: 2026-05-02 18:44:27 JST

## 内容

エディター
従業員メールデザイナーを使用して、アプリやイントラネットのコンテンツをカスタマイズされたニュースレター、メール、またはSlackメッセージに迅速に変換できます。使いやすく、エディターは魅力的なテンプレートの中から選択し、Staffbaseプラットフォームからニュース投稿を希望の場所にドラッグ＆ドロップするだけです。貴重な時間を節約し、コンテンツのリーチを拡大しましょう。
メール統合のインターフェースは現在、英語のみで利用可能です。
メールデザイナーを開く
従業員メールデザイナーから直接メールをデザインして送信します。
スタジオで、メールに移動します。
スタジオを通じてアクセスすると、自動的に従業員メールにサインインされます。
これで、既存のStaffbaseプラットフォームコンテンツを使用して新しいメールを作成できます。メールプラットフォームでのコンテンツ作成方法について詳しくは、Staffbaseサポートポータルをご覧ください。
メール設定と配信リストへのアクセス
スタジオの上部ナビゲーションバーで、設定または人のドロップダウンメニューに移動します。
追加の役立つ情報
メールデザインに記事投稿を追加する
Staffbaseの連絡先をBananatagにインポートする
