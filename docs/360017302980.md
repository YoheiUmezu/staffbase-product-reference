# アプリまたはイントラネットから Microsoft 365 にサインインする

**URL**: https://support.staffbase.com/hc/ja/articles/360017302980
**取得日**: 2026-05-02 18:44:42 JST

## 内容

価格
アクティベーション
従業員アプリまたはイントラネットから Microsoft 365 アカウントにサインインして、Microsoft 365 アプリケーションのコンテンツにアクセスできます。Microsoft 365 から特定のコンテンツを検索するか、Microsoft 365 ウィジェットを使用して表示されるコンテンツを確認してください。
従業員アプリまたはイントラネットで、ドロップダウンプロファイルメニューをクリックします。
プロファイルメニューが表示されます。
設定をクリックします。
設定の下で、アカウントをクリックします。
アカウントを接続の下で、Microsoft 365 のために サインイン をクリックします。
Microsoft 365 のログイン画面にリダイレクトされます。
Microsoft 365 のサインイン画面で、Microsoft アカウントを選択するか、別のアカウントを使用をクリックしてサインインの詳細を入力します。
サインインをクリックします。
従業員アプリまたはイントラネットから Microsoft 365 アカウントにサインインしました。同じセクションで サインアウト をクリックすることで、従業員アプリまたはイントラネットから Microsoft 365 アカウントをサインアウトできます。
Microsoft 365 ウィジェットから直接、または従業員アプリの検索機能の Microsoft 365 タブを通じて Microsoft 365 アカウントにサインインすることもできます。
