# 従業員メールデザイナーのドラフトにテキストを追加およびフォーマットする

**URL**: https://support.staffbase.com/hc/ja/articles/4412581467666
**取得日**: 2026-05-02 18:44:14 JST

## 内容

従業員メールデザイナーのドラフトにテキストを追加し、デザイン要件に合わせて編集します。
テキストを追加
ゼロから:
空のブロックレイアウトで+ボタンをクリックするか、作成パネルからテキスト要素をドラッグしてブロックにドロップします。
クイックブロックを使用する:
テキストや他の要素が含まれたクイックブロックをドラッグします。すべてのクイックブロックには、スペーサーを除いて、少なくとも1つのテキスト要素があります。
テキストをフォーマット
各テキスト要素には、次のプロパティごとに複数の設定を持つことができます:
フォント
ヘッダースタイル
フォントサイズ
配置
行間
1つのテキスト要素に複数の太字、下線、イタリックスタイル、および色を持つこともできます。
フォントやサイズを混ぜる場合は、別々のテキスト要素を使用することをお勧めします。
リストを追加
段落の一部にリストを追加したい場合は、テキストセクションをハイライトし、箇条書きリストまたは番号付きリストのいずれかを選択します。
リストのテキストの色を変更すると、テキストの色が変更されます。箇条書きや番号の色は常に黒です。
フォーマットをクリア
テキストのスタイルが気に入らない場合は、それをハイライトし、フォーマットをクリアボタンをクリックします。
リンクを挿入
リンクにしたいテキストをハイライトし、リンクを挿入ボタンをクリックして、リンク先のURLを追加します。
組織に特別なリンクスタイルがある場合、たとえば、すべてのリンクが赤で下線付きである必要がある場合は、ブランド設定ページでこれを設定できます。
目次を挿入
目次（TOC）機能を使用する予定がある場合、メールデザイナーのドラフト内の各ヘッダーに対してアンカーリンクを作成し、ヘッダーが表示される順序で配置されます。
目次をカスタマイズする方法について詳しく学びます。
