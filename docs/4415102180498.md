# なぜ「従業員メール」で追跡されたオープン詳細に「プロキシ」が表示されるのですか？

**URL**: https://support.staffbase.com/hc/ja/articles/4415102180498
**取得日**: 2026-05-02 18:49:46 JST

## 内容

なぜ、従業員メールで追跡された送信メールの詳細を表示すると、「Image Proxy」、「Gmail Proxy」または「Mountain View, CA」からのオープンが表示されるのですか？
これは、Gmailが画像のダウンロードを処理する方法によるもので、メールがいつ、どこで、どのように開かれたかを追跡するために使用されます。
Gmailが画像の自動ダウンロードをブロックするのをやめたとき、エンドユーザーを隠すためのプロキシも導入し、匿名性の層を提供しました。
つまり、Gmailでメールを開くと、画像をダウンロードするために使用されるデバイスがカリフォルニアのLinuxベースのGoogleプロキシであるため、あなたの詳細が記録されます。
従業員メールウェブアプリで送信されたメールのオープン詳細はこのようになります：
あなたの組織のGoogle Apps（GSuite）の管理者アカウントを持つ人々は、特定のドメインをホワイトリストに追加して、Gmail画像プロキシをバイパスできるようにすることができ、より正確な位置情報メトリクスを記録できます。
