# リカバリーコードの手動設定

**URL**: https://support.staffbase.com/hc/ja/articles/360015757232
**取得日**: 2026-05-02 18:50:28 JST

## 内容

管理者
スタジオで個々のユーザーに対して、永続的または一時的なリカバリーコードを手動で設定できます。これは、ユーザーがリカバリーコードを必要とする単一のケースに最適です。
スタジオで、ユーザー > 登録ユーザーに移動します。
更新したいユーザーアカウントの編集をクリックします。
リカバリーコードのフィールドにコードを入力します。
入力したリカバリーコードをクリップボードにコピーします。
リカバリーコードの有効期限を設定します。
保存をクリックします。
ユーザーのリカバリーコードを設定しました。コピーしたリカバリーコードをユーザーと共有できます。
