# リンクの開き方の変更

**URL**: https://support.staffbase.com/hc/ja/articles/360031040051
**取得日**: 2026-05-02 18:58:30 JST

## 内容

Linksプラグインを使用すると、別のウェブサイトにリンクし、そのリンクをアプリのメニューに配置できます。モバイルユーザーの場合、個々のリンクの開き方を設定できます：
アプリ内のブラウザを介して
アプリとは別の外部モバイルブラウザ（Google ChromeブラウザやSafariブラウザなど）で
モバイルユーザーのリンクの開き方の設定
スタジオで、Linksプラグインに移動します。
変更したいリンクを選択します。
設定を開くためにギアアイコンをクリックします。
プラグイン設定：リンクに移動します。
モバイルブラウザで開くのスライダーを動かして、リンクの開き方を定義します。
保存をクリックします。
