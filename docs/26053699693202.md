# Thank Youプラグインのインストール

**URL**: https://support.staffbase.com/hc/ja/articles/26053699693202
**取得日**: 2026-05-02 18:45:57 JST

## 内容

この機能はアドオンです。詳細はStaffbaseの価格をご参照ください。有効化についてはStaffbaseサポートまたはカスタマーサクセスマネージャーにお問い合わせください。
Thank Youプラグインは、Staffbase従業員エクスペリエンス（EX）ブースタープラグインです。
Thank Youプラグインは、従業員アプリやイントラネットから直接感謝の気持ちを簡単に共有できるようにすることで、感謝の文化を促進します。
前提条件
APIトークンを作成し、読み取り専用アクセス権を持っていること。
APIトークンを暗号化してsupport@staffbase.comに送信したこと。
APIトークンは、ユーザーAPIを通じてユーザープロフィール情報にアクセスするために必要です。APIトークンにより、プラグインはユーザープロフィール情報（プロフィール画像や権限など）にアクセスできます。これにより、許可されたユーザーのみがThank Youメッセージを投稿できるようになります。
Studioで、コンテンツ > コンテンツメニューに移動します。
メニューの一番下までスクロールし、プラグインを追加するをクリックします。
プラグインページが開きます。
Thank Youに移動します。
インストールをクリックします。
Thank Youを追加するダイアログが開きます。
Thank Youページのタイトルを入力します。
タイトルはプラグインの設定でいつでも変更できます。
表示ドロップダウンメニューから、Thank Youページを表示するユーザーを選択します。
Thank Youを追加するをクリックします。
Thank Youプラグインをインストールし、Thank Youページを追加しました。プラグインは左側のコンテンツセクションに表示されます。ページを設定して公開することで、ユーザーに表示できるようになります。
