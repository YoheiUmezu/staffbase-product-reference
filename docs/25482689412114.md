# スタッフベースメールデザイナーの概要

**URL**: https://support.staffbase.com/hc/ja/articles/25482689412114
**取得日**: 2026-05-02 19:01:52 JST

## 内容

スタッフベースのメールデザイナーを使用すると、モバイル対応のメールを簡単に作成することができます。タイトル、段落、列、動画、引用などを含む既製のブロックをデザインに直接ドラッグ＆ドロップするだけです。マイファイル統合により、メディアファイルの共有とアップロードがさらに簡単になり、必要なビジュアルにすばやくアクセスできます。このメールデザイナーについてのすべてをこの記事で学びましょう。
スタッフベースのメールデザイナーは、3つの主要なエリアで構成されています：
デザイナーパネル
編集者とプレビューエリア
設定パネル
デザイナーパネルを使用して、レイアウト、要素、およびクイックブロックを要件に基づいて選択します。↓
編集者とプレビューエリアを使用して、左側のパネルからレイアウト、要素、およびクイックブロックをドラッグ＆ドロップしてメールをデザインします。
単に要素を選択し、右側のパネルでその設定をカスタマイズしてニーズに合わせて構成します。↓
設定パネルを使用して、メールのレイアウト、個々のブロックまたは要素をさらに構成します。↓
デザイナーパネル
以下のセクションを探索して、レイアウト、コンテンツブロック、およびインタラクティブ要素を使用してメールをデザインおよび最適化する方法を学びましょう。
表示
このトグルを使用して、デザインモードとメールのプレビューを簡単に切り替えることができます。
プレビューモードでは、デスクトップまたはモバイルデバイスのプレビューを選択することができます。
アクション
矢印を使って、直前の変更を元に戻したりやり直したりできます。また、メールをHTMLファイルとしてダウンロードすることもできます。
AIサマリーを生成
生成する をクリックすると、メールの冒頭にサマリーブロックが追加されます。AIサマリーブロックについて詳しくはこちらをご覧ください。
レイアウト
左側のパネルで、メールに追加したい各ブロックのレイアウトを選択します。ブロックごとに1から4列の異なる幅と組み合わせを選択できます。使用したいレイアウトをクリックするか、編集者で必要な場所にドラッグアンドドロップします。
メール全体のレイアウトはブロックで構成されています。メールは単一のブロックまたは複数のブロックの組み合わせにすることができます。
ブロックをクリックすると、右側のパネルにブロック設定が表示され、次のことができます：
選択したブロックのみに変更を適用するか、このメールで使用されているすべてのブロックに変更を適用するかを決定する
ブロックをロック/アンロックする（ロックされたブロックは移動できず、そのコンテンツは編集できません）
ブロックを上下に移動、複製、または削除する
ブロックの外側と内側の背景の色を調整する
ブロックの上下のスペーシングを調整する
列の幅を調整する
編集者で空のブロックをクリックすると、いくつかのコンテキストメニューが表示されます：
ここで次のことができます：
ブロックをロック/アンロックする（ロックされたブロックは移動できず、そのコンテンツは編集できません）
ブロックを選択/選択解除する
青いプラスアイコンを介してブロックに直接要素を追加する
ブロックを上下に移動する
ブロックをコピーまたは削除する
ブロックを隣接するブロックとマージまたは交換する（ブロックの下の灰色の矢印を使用）
要素
Staffbaseメールでは、以下のタブに詳述されているさまざまな要素を追加および設定することで、メールデザインを強化できます。各要素の設定方法を学び、メールデザインを最大限に活用しましょう！
テキスト
テキスト要素をメールデザインにドロップすると、ダブルクリックして編集できます。
青いコンテンツメニューアイコンを使用して、次のこともできます:
要素を移動する
要素をコピーする
要素を削除する
テキスト要素をクリックすると、右側のパネルにテキスト設定が表示され、次のことができます:
間隔を調整する
目次を追加する
画像
画像要素をメールデザインにドロップすると、ダブルクリックして「マイファイル」から画像をアップロードできます。
青いコンテンツメニューアイコンを使用して、次のこともできます:
要素を移動する
要素をコピーする
要素を削除する
アップロードされた画像をクリックすると、右側のパネルに画像設定が表示され、次のことができます:
間隔を調整する
「マイファイル」を開いて画像を置き換える
画像を削除する
画像のリダイレクトリンクを入力する
代替テキストを入力する
選択したブロックレイアウトによって、以下の画像サイズを推奨します：
4ブロック：130 px
3ブロック：180 px
2ブロック：280 px
1ブロック：580 px
ボタン
ボタン要素をメールデザインにドロップすると、それをクリックして右側のパネルでボタン設定を開くことができます。
ここでできること:
スペーシングを調整する
ボタンリンクを編集する
ボタンテキストを調整する
ボタンテキストのフォント、サイズ、行の高さ、色、スタイルを調整する
ボタンの幅、高さ、角、配置、色の塗りを調整する
青いコンテンツメニューアイコンを使用して、次のこともできます:
要素を移動する
要素をコピーする
要素を削除する
動画
動画要素をメールデザインにドロップすると、それをクリックして右側のパネルで動画設定を開くことができます。
ここでできること:
スペーシングを調整する
動画リンクを追加する
マイファイルから画像を動画のサムネイルとして追加する
再生ボタンの色を選択する（黒または白）
青いコンテンツメニューアイコンを使用して、次のこともできます:
要素を移動する
要素をコピーする
要素を削除する
インライン動画再生は、Gmail、Outlook（Windows および Outlook.com）、Yahooメールなど、ほとんどの主要なメールクライアントではサポートされていません。これはセキュリティやパフォーマンス上の制限によるものであり、メールプロバイダーによって強制されているため、Staffbase で設定や上書きはできません。
スペーサー
スペーサー要素をメールデザインにドロップすると、それをクリックして右側のパネルでスペーサー設定を開くことができます。
ここでできること:
高さを調整する
色を選択する
青いコンテンツメニューアイコンを使用して、次のこともできます:
要素を移動する
要素をコピーする
要素を削除する
ソーシャルリンク
ソーシャルリンク要素をメールデザインにドロップすると、それをクリックして右側のパネルでソーシャルリンク設定を開くことができます。
ここでできること:
間隔を調整する
アイコンのスタイル、サイズ、配置を調整する
LinkedInやSlackのようなさらなるソーシャルリンクを追加する
選択したすべてのプラットフォームのリンクを追加する
青いコンテンツメニューアイコンを使用して、次のこともできます:
要素を移動する
要素をコピーする
要素を削除する
クイックブロック
スタッフベースメールのクイックブロックを使用すると、構造化されたコンテンツを簡単に追加でき、一貫したレイアウトを提供し、コンテンツを手動でフォーマットする必要がなくなるため、時間を節約できます。各クイックブロックは完全にカスタマイズ可能です。各クイックブロックを設定し、メールデザインを最適化する方法を学びましょう！
見出し
見出しクイックブロックをメールデザインにドロップすると、それをクリックしてプレースホルダーテキストを編集できます。見出しクイックブロックは複数のテキスト要素で構成されています。テキスト要素について詳しく学びましょう。
アラインメントサーベイ
メールがキャンペーンに割り当てられると、アラインメントサーベイクイックブロックをメールデザインにドラッグ＆ドロップして追加できます。メール受信者はメール内でサーベイを確認し、1通のメールにつき1回フィードバックを送信できます。
ニュース
ニュースクイックブロックは、従業員アプリまたはStaffbaseイントラネットをお持ちで、Staffbaseメールがアプリまたはイントラネットと一緒に使用されている場合にのみ利用可能です。メールスタンドアロンのお客様には利用できません。
ニュースクイックブロックをメールデザインにドロップすると、それをクリックして右側に最後に公開されたニュース投稿のリストを開くことができます。キーワードを使用してリストを検索できます。メールに含めたいニュース投稿をクリックまたはドラッグしてニュースクイックブロックに選択します。
サムネイル画像、記事のタイトル、および記事のティーザーテキストがブロックにコピーされ、記事自体へのリンクも含まれます。記事に画像がない場合や記事のティーザーテキストが空白の場合は、独自の画像とティーザーテキストをニュースブロックに追加できます。リンクテキストを含むすべてのテキストを編集することもできます。
画像段落
画像段落クイックブロックをメールデザインにドロップすると、それをクリックして画像とテキストのプレースホルダーの両方を編集できます。画像要素とテキスト要素について詳しく学びましょう。
画像テキスト列
画像テキスト列クイックブロックをメールデザインにドロップすると、それをクリックして画像、テキストのプレースホルダー、およびボタンを編集できます。画像要素、テキスト要素、およびボタン要素について詳しく学びましょう。
動画とテキスト
動画テキストのクイックブロックをメールデザインにドロップすると、それをクリックして動画とテキストのプレースホルダーを編集できます。動画要素とテキスト要素について詳しく学びましょう。
テーブル
テーブルのクイックブロックをメールデザインにドロップすると、それをクリックしてテーブルを編集できます。コンテキストメニューを使用して:
列を挿入または削除する
行を挿入または削除する
ヘッダー列またはヘッダー行を選択する
セルを結合する
セル内のテキストを編集するには、セルを選択して入力を開始します。
引用
引用のクイックブロックをメールデザインにドロップすると、それをクリックして引用と作成者のテキストプレースホルダーを編集できます。テキスト要素について詳しく学びましょう。
スペーサー
スペーサーのクイックブロックをメールデザインにドロップすると、それをクリックして編集できます。スペーサー要素について詳しく学びましょう。
Webバージョン
Webバージョンブロックをメールデザインにドロップすると、リンクテキストをクリックして編集できます。ブラウザーで開くリンクについて詳しくはこちらをご覧ください。
ソーシャルリンク
ソーシャルリンクのクイックブロックをメールデザインにドロップすると、それをクリックしてプレースホルダーテキストとソーシャルリンクを編集できます。テキスト要素とソーシャルリンク要素について詳しく学びましょう。
編集者
デザイナーパネルからクイックブロックやレイアウトをクリックまたはドラッグして、メールのデザインを開始します。編集者で要素を選択すると、関連する設定が右側のパネルに表示され、要件に基づいて調整できます。
