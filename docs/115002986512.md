# アプリ全体でのカスタムアイコン

**URL**: https://support.staffbase.com/hc/ja/articles/115002986512
**取得日**: 2026-05-02 18:52:43 JST

## 内容

カスタムアイコンを使用すると、Staffbaseプラットフォームのメニュー項目に個別のタッチを加えることができます。これには、ニュースページ、静的ページ、フォルダー、プラグインが含まれます。
カスタムアイコンの要件
サイズ: 96 x 96 px; メニューアイコンとしては18 x 18 pxに縮小されます
ファイル形式: png推奨、SVGはサポートされていません
解像度: 72 dpi
カラープロファイル: sRGB
ミニマリスティックな形状
推奨: キャンバス全体を境界まで使用する
アウトライン幅: アイコンの外枠幅として4〜8 pxのサイズを使用する
ネイティブアプリとWebアプリの異なる背景用に2つのバージョンが必要です:
透明な背景の明るい色
透明な背景の暗い色、
例えば、黒、標準のグレー（#666666）、またはアプリのブランディングに既に使用している他の暗い色
追加の役立つ情報
メニューにカスタムアイコンを追加する
