# アプリ内のコンテンツへのディープリンクによるリンク

**URL**: https://support.staffbase.com/hc/ja/articles/115002997132
**取得日**: 2026-05-02 18:52:41 JST

## 内容

通常のリンクまたはディープリンクを使用して、アプリ内の特定のコンテンツにどこからでもリンクできます。たとえば、ニュースレターやイントラネット内でアプリへのリンクを作成できます。通常のリンクは常にインターネットブラウザーでウェブアプリを開きます。
ディープリンクの大きな利点は、ネイティブアプリを開くことです。ディープリンクをクリックすると、アプリがインストールされているモバイルデバイスでは、自動的にネイティブアプリが開きます。ディープリンクは、アプリがインストールされていないコンピューターやモバイルデバイスでも機能します。モバイルデバイスでアプリがインストールされていない場合、アプリのダウンロードまたはウェブアプリの利用を選択するウェブサイトに誘導されます。コンピューターでは、ディープリンクはインターネットブラウザーでウェブアプリを開きます。
アプリ内のコンテンツを直接開くには、ユーザーがデバイス上でアプリにサインインしている必要があります。ユーザーがサインインしていない場合、ディープリンクはサインインを促すエラーページのみを開きます。サインイン後、ユーザーは再度ディープリンクをクリックして、アプリ内のコンテンツを開く必要があります。
ネイティブアプリまたはウェブアプリを開くための一般的なリンク
https://yourcompany.staffbase.com/openlink
「yourcompany.staffbase」をあなたのアプリのドメインまたはStaffbaseのサブドメインに置き換えてください。
特定のビューを開くディープリンク
ディープリンクは、記事、ページ、またはユーザーのプロフィール設定などのコンテンツを開きます。一部のコンテンツにはIDが必要です。詳細は、アプリコンテンツのIDを特定するを参照してください。
投稿、ニュースチャンネル、ニュースページを開くディープリンク：
https://yourcompany.staffbase.com/openlink/content/news/article/xxx
記事を開きます。xxxを記事のIDに置き換えてください。
https://yourcompany.staffbase.com/openlink/content/news/update/xxx
更新投稿を開きます。xxxを更新投稿のIDに置き換えてください。
https://yourcompany.staffbase.com/openlink/content/news/picture/xxx
画像投稿を開きます。xxxを画像投稿のIDに置き換えてください。
https://yourcompany.staffbase.com/openlink/content/news/archive/xxx
ニュースチャンネルを開きます。xxxをチャンネルのIDに置き換えてください。
https://yourcompany.staffbase.com/openlink/content/news/xxx
ニュースページを開きます。xxxをニュースページのIDに置き換えてください。
ページ、プラグイン、フォルダーを開くディープリンク：
https://yourcompany.staffbase.com/openlink/content/page/xxx
ページを開きます。xxxをページのIDに置き換えてください。
https://yourcompany.staffbase.com/openlink/content/chat/xxx
チャットプラグインを開きます。xxxをチャットページのIDに置き換えてください。
https://yourcompany.staffbase.com/openlink/content/chat/xxx/yyy
特定のチャット会話を開きます。xxxをチャットページのIDに、yyyをチャット会話のIDに置き換えてください。
https://yourcompany.staffbase.com/openlink/content/menu/xxx
特定のフォルダーを開きます。xxxをフォルダーのIDに置き換えてください。
https://yourcompany.staffbase.com/openlink/content/directory/xxx
アプリのすべてのユーザーを表示するディレクトリプラグインを開きます。xxxをディレクトリページのIDに置き換えてください。
https://yourcompany.staffbase.com/openlink/profile/xxx
特定のユーザーのプロフィールを開きます。xxxをユーザーページのIDに置き換えてください。
ユーザーのプロフィールメニューを開くディープリンク：
https://yourcompany.staffbase.com/openlink/settings
ユーザーのプロフィールメニューを開きます。
https://yourcompany.staffbase.com/openlink/settings/password
ユーザーのパスワード設定を開きます。
https://yourcompany.staffbase.com/openlink/settings/notifications
ユーザーの通知設定を開きます。
https://yourcompany.staffbase.com/openlink/bookmarks
ユーザーのブックマークを開きます。
https://yourcompany.staffbase.com/openlink/profile
ユーザーのプロフィールを開きます。
アプリに関する情報を開くディープリンク：
https://yourcompany.staffbase.com/openlink/about
Aboutページを開きます。
SSOサインインを開始するためのディープリンク：
https://yourcompany.staffbase.com/openlink/?action=restart-SSO
アプリを開き、シングルサインオン（SSO）でユーザーサインインを開始します。
