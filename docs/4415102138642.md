# メールアドインまたは拡張機能のアンインストール

**URL**: https://support.staffbase.com/hc/ja/articles/4415102138642
**取得日**: 2026-05-02 18:53:59 JST

## 内容

メールクライアントから Employee Email アドインまたは拡張機能をアンインストールする必要がある場合があります。
問題をトラブルシューティングしている場合は、現在のバージョンをアンインストールし、その後 最新バージョンをダウンロードしてインストールすることが役立つ場合があります。
Outlook Classic
Windows PCからOutlook Classic 3アドインをアンインストールします。
Outlookを終了します。
スタートを選択し、設定メニューを開くためにギアアイコンをクリックします。
アプリ > アプリと機能に移動します。
検索バーに「staffbase」と入力し始めます。
検索結果からStaffbase Email Classicを選択し、アンインストールをクリックします。
ダイアログが表示され、再度アンインストールをクリックするように促されます。
プロセスが完了したら、ダイアログで閉じるをクリックします。
Outlook 365
Windows、Mac、またはWeb上のOutlook用のOutlook 365アドインをアンインストールします。
通常、特別な権限は必要ありません。必要な権限が不足している場合は、ITチームにお問い合わせください。
https://outlook.office.com/owaに移動します。
新しいメッセージをクリックします。
メッセージの下にある三点リーダーをクリックします。
アドインの取得をクリックします。
マイアドインをクリックします。
アドインの下にある三点リーダーをクリックし、削除を選択します。
Outlookを閉じて再度開きます。
アドインが削除されます。
デスクトップでOutlookを閉じて再度開くと、アドインが削除されます。同じ変更がOutlook Webmailにも、セッションを更新した後に反映されます。
Gmail
ChromeウェブブラウザからGmail拡張機能をアンインストールします。
三つの縦の点をクリックしてChromeメニューを開きます。
その他のツール > 拡張機能に移動します。
拡張機能を見つけて削除をクリックし、Chromeからアンインストールします。
ダイアログが表示され、再度削除をクリックするように促されます。
Gmailを再読み込みします。
アドインが削除されます。
Outlook for Mac
通常、特別な権限なしでOutlook 365アドインをMacからアンインストールすることができます。必要な権限が不足している場合は、ITチームにお問い合わせください。
https://outlook.office.com/owaに移動します。
新しいメッセージをクリックします。
メッセージの下にある三点リーダーをクリックします。
アドインの取得をクリックします。
マイアドインをクリックします。
アドインの下にある三点リーダーをクリックし、削除を選択します。
Outlookを閉じて再度開きます。
アドインが削除されます。
