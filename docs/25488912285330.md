# キャンペーンのためのアライメント調査の概要

**URL**: https://support.staffbase.com/hc/ja/articles/25488912285330
**取得日**: 2026-05-02 18:56:36 JST

## 内容

アクティベーション
アライメント調査は、キャンペーンのナラティブに関する従業員からの直接的なフィードバックを提供します。キャンペーン関連のニュースコンテンツにアライメント調査を使用することで、コミュニケーションが従業員に与える影響を追跡できます。戦略的なナラティブが読者にどのように響いているかをステークホルダーに明確に示すことができます。アライメント調査を通じて、従業員がキャンペーンメッセージについてどのように感じているか、またキャンペーンの進行中にその感情が変化するかを学ぶことができます。
アライメント調査は、キャンペーンに割り当てられたニュース投稿の最後にオプションで追加できるスケール評価調査です。各キャンペーンにアライメント調査を追加できます。特定のキャンペーンに割り当てられたニュース投稿に調査を追加する必要があり、そうしないとユーザーには表示されません。同じ調査質問が各ニュース投稿に使用され、時間の経過とともにスコアを追跡できます。
システム全体のエディターロール以上のユーザーは、キャンペーンの詳細内で独自の質問を作成できます。回答スケールの範囲は1から5に設定されています。3つのオプションのプリセットスケール範囲ラベルがあり、カスタムラベルを作成することもできます。その後、キャンペーンに割り当てられたニュース投稿のためにアライメント調査オプションを切り替えることができます。詳細を学ぶ。
ユーザーはニュース投稿の下部で調査を確認し、各ニュース投稿に対して1つの評価を提出できます。ユーザーは新しいニュース投稿に調査が含まれるたびに回答できます。その場合、ユーザーが提出した最新のスコアがメトリック計算に使用されます。
アライメント調査の結果は、Smart Insightsダッシュボードのアライメントセクションに表示されます。機密性を維持するために、結果が表示される前に少なくとも5件の提出が必要です。 詳細を学ぶ。
アライメント調査の例
カーボンゼロへの道：持続可能性キャンペーンの先導
多様性と包摂を受け入れるキャンペーン
デジタル未来：変革への投資キャンペーン
