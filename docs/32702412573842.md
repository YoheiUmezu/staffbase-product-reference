# コンテンツデザイナーでページの以前のバージョンを復元する

**URL**: https://support.staffbase.com/hc/ja/articles/32702412573842
**取得日**: 2026-05-02 19:01:35 JST

## 内容

アクティベーション
編集者
Content Designer のページでバージョン履歴を確認し、以前のバージョンに復元できます。これにより、不要な変更を元に戻したり、以前のコンテンツを復元したり、ページを手動で再作成することなく、既知の良好な状態に戻すことが簡単にできます。
バージョン履歴には、ページを公開または更新したときに作成されたバージョンのみが保存されます。バージョンを復元しても他のバージョンは削除されないため、履歴を失うことなくバージョンを切り替えることができます。
バージョン履歴を閲覧する際、言語セレクターを変更して異なる言語のバージョンを確認・復元できます。
ページの直近100件までのバージョンを復元できます。
Studio で コンテンツ > ページ に移動します。
更新したいページを開きます。
上部メニューから三点リーダーを選択し、アクションメニューを開きます。
バージョン履歴 を選択します。
復元したいバージョンを選択します。
復元 をクリックします。
選択したバージョンが編集者で有効なバージョンになります。復元したコンテンツを公開または更新する前に確認できます。
