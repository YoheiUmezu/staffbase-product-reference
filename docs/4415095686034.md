# Gmail画像プロキシをバイパスしてメール位置情報メトリクスを取得する

**URL**: https://support.staffbase.com/hc/ja/articles/4415095686034
**取得日**: 2026-05-02 18:51:44 JST

## 内容

誰かがGmailを使用してあなたの追跡されたメールを開くと、Employee Emailによって記録されるデフォルトの位置情報は、メッセージ内の画像をダウンロードするために使用されるデバイスであるカリフォルニアのLinuxベースのGoogleプロキシです。
あなたの組織のGoogle Apps (GSuite)の管理者アカウントを持つ人々は、特定のドメインをホワイトリストに追加してGmail画像プロキシをバイパスすることができ、より正確な位置情報メトリクスを記録できるようになります。
Employee Emailの追跡ドメインはbl-1.comです。
このドメインをあなたの組織のGSuite/Google Appsアカウントの画像URLプロキシのホワイトリストに追加するには、Googleのヘルプサイトにあるこの記事の指示に従ってください：
https://support.google.com/a/answer/3299041?hl=en
正しく追加されると、ドメインbl-1.comがあなたの組織の画像URLプロキシのホワイトリストに表示されます。
