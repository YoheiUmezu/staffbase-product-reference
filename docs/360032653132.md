# ニュース投稿における画像のガイドライン

**URL**: https://support.staffbase.com/hc/ja/articles/360032653132
**取得日**: 2026-05-02 19:01:14 JST

## 内容

エディター
ニュース投稿にアップロードする画像は、ユーザーを投稿に引き付け、魅力的にするための主要な方法の一つです。以下のガイドラインは、記事チャンネル、メディアチャンネル、更新チャンネルの3つのチャンネルタイプにおいて、ニュース投稿にアップロードする画像の最適な視覚効果を確保するのに役立ちます。
最適な画像サイズ: 1900 x 1069 ピクセル
最適なアスペクト比: 16:9
最大画像ファイルサイズ: 25 MB
アップロードした画像が推奨サイズより小さい場合はどうなりますか？
画像が推奨サイズに合わない場合、画像は自動的に最適サイズにズームされ、品質が低下する可能性があります。
アップロードした画像が推奨アスペクト比を持たない場合はどうなりますか？
アスペクト比が一致しない場合、画像は自動的に16:9のアスペクト比にズームされ、画像の一部が切り取られるか、右側と左側にぼやけた背景が表示されることがあります。全画面モードで開くと、画像は元のアスペクト比で表示されます。
↻
画像を切り取ることは、結果として得られる画像にどのように影響しますか？
エディターで切り取られた画像は、アスペクト比に応じてぼやけた背景で表示されることがあります。
