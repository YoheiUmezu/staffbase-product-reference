# App Store Connectでアプリプロファイルを作成する

**URL**: https://support.staffbase.com/hc/ja/articles/115003481992
**取得日**: 2026-05-02 18:54:39 JST

## 内容

App Store Connectにサインインし、マイアプリに移動します。
新しいアプリをクリックします。
これにより、アプリのプロファイルが作成されます。プロファイルには、アプリおよびApple App Storeに関連するすべての設定が含まれます。
アプリ名を入力します。
アプリ名はAppleのレビュアーによって審査されます。アプリ名がAppleのガイドラインに合わない場合、アプリが拒否されることがあります。
アプリ名に「従業員アプリ」を使用しないでください。Appleは、アプリが特定のターゲットグループに焦点を当てるのではなく、公共の関連性を持つことを要求しています。
主要言語を設定します。
バンドルIDを入力します。このIDはエンドユーザーには表示されません。
SKUフィールドにユニークIDを入力します。このIDもエンドユーザーには表示されません。
作成をクリックします。
