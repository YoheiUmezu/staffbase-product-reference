# タスクの作成中および管理

**URL**: https://support.staffbase.com/hc/ja/articles/30886074874002
**取得日**: 2026-05-02 18:54:56 JST

## 内容

アクティベーション
効率的なタスク管理は、あなたが整理され、チームと連携を保つのに役立ちます。タスクを割り当て、詳細を更新し、古いアイテムを削除してプロジェクトを順調に進めたり、タスクを完了としてマークしたりできます。
各リストには最大1,000件のタスクを含めることができます。
タスクを作成する
タスクを作成して、ユーザーもしくはグループに割り当てる特定の作業内容を定義できます。日付や優先度を設定して、作業量や締め切りを管理することもできます。
アプリまたはイントラネットで、タスクを追加したいプロジェクトに移動します。
+ アイコンをクリックしてタスクを作成する。
タスクダイアログが開く。
タイトルと任意で説明を入力します。
必要に応じて、以下の詳細を設定します：
タスクのスケジュールを設定します。
タスクをユーザーもしくはユーザーグループに割り当てます。
優先度を選択します。
添付ファイルを追加する：画像、動画、またはPDF
Web：添付ファイルをドラッグそしてドロップします
モバイル：カメラを直接使用するか、デバイスからアップロードします
リストを選択するドロップダウンから、タスクを特定のリストに割り当てます。
作成するをクリックします。
タスクが作成され、選択したリストに表示されます。タスクはいつでも編集できます。
タスクを編集する
タスクを開いて、タイトル、担当者、期限などの情報を更新できます。
アプリまたはイントラネットで、プロジェクト > リスト > タスクに移動します。
タスクの詳細ダイアログが開く。
三点リーダーをクリックして、更にオプションメニューを開きます。
タスクを編集するをクリックします。
タスクの詳細を更新します。
保存するをクリックします。
タスクが編集されました。
タスクを削除します
不要になったタスクを削除することができます。
アプリまたはイントラネットで、プロジェクト > リスト > タスクに移動します。
タスクの詳細ダイアログが開く。
上部の三点リーダーをクリックして、更にオプションメニューを開きます。
削除するをクリックします。
必要に応じて、タスクを編集するをクリックし、ダイアログ下部のゴミ箱アイコンをクリックします。
タスクが削除されました。
