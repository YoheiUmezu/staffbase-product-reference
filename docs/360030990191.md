# オンデマンド翻訳の設定のカスタマイズ

**URL**: https://support.staffbase.com/hc/ja/articles/360030990191
**取得日**: 2026-05-02 18:56:11 JST

## 内容

アクティベーション
管理者
オンデマンド翻訳は、ユーザーがリクエストに応じて、ページ、投稿、コメントをユーザーのアプリ設定で事前に定義された言語に翻訳するのに役立ちます。さらに、ページや投稿にウィジェットが追加されている場合、それらもオンデマンドで翻訳されます。
スタジオで、設定 > 言語に移動します。
オンデマンド翻訳を好みに応じて有効または無効にします。
保存をクリックします。
オンデマンド翻訳は従業員にどのように見えるか？
最適なユーザーエクスペリエンスのために、ユーザーにはアプリプロファイルと同じ言語をブラウザーに設定することをお勧めします。
この例では、ユーザープロファイルでスペイン語が優先言語として設定されていますが、投稿の内容は英語のみで利用可能です。自動翻訳がこのギャップを埋め、コンテンツをスペイン語でも利用できるようにします。
追加の役立つ情報：
利用可能な言語の概要を含むMicrosoft Translatorのドキュメント
