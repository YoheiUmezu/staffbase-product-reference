# コミュニティプラグインをインストールする

**URL**: https://support.staffbase.com/hc/ja/articles/34481899285650
**取得日**: 2026-05-02 18:45:28 JST

## 内容

編集責任者
コミュニティは、ディスカッションや共有のための専用エリアであり、従業員がより強い接続を築いたり、重要な取り組みに関して連携を保つのに役立ちます。従業員はいつでも発見ページを使用して関連するコミュニティを見つけて参加でき、より接続されエンゲージメントの高い職場を実現します。
スタジオ内でコンテンツに移動します。
コンテンツメニューでプラグインを追加するをクリックします。
コミュニティに移動し、インストールするをクリックします。
コミュニティを追加するダイアログが開く。
コミュニティのタイトルを入力します。
ドロップダウンメニューからコミュニティの表示を設定します。
コミュニティを追加するをクリックします。
コミュニティの設定が開く。
コミュニティプラグインがインストールされ、最初のコミュニティが作成されました。今すぐコミュニティを設定することができます。
