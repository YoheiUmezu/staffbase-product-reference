# Microsoft 365 カレンダープラグインのインストールと設定

**URL**: https://support.staffbase.com/hc/ja/articles/360030784951
**取得日**: 2026-05-02 18:57:31 JST

## 内容

Microsoft 365 カレンダープラグインを使用すると、従業員はモバイルおよびデスクトップデバイスで Microsoft 365 カレンダーを利用できます。
Staffbase の Microsoft 365 カレンダープラグインは、現在アクティブな開発が行われておらず、セキュリティ関連のメンテナンスのみが行われています。Staffbase は、代わりにウィジェットの使用を推奨しています。ページに Microsoft 365 カレンダーウィジェットを追加することで、ユーザーにさらに良い体験を提供し、将来の更新や改善を受けることができます。
❯❯ Microsoft 365 カレンダーウィジェットの詳細を学ぶ。
Staffbase プラットフォームのユーザーは、カレンダーを表示およびアクセスするために Microsoft 365 ライセンスが必要です。
Staffbase における Microsoft 365 カレンダープラグインのインストールと設定
プラグインを初めてインストールする際は、グローバル管理者 Microsoft 365 の役割が必要です。
インストールが成功すると、通常の Microsoft 365 ユーザーが追加の Microsoft 365 カレンダーを追加および設定できます。
Microsoft 365 カレンダープラグインを有効にするには、Staffbase サポートまたはカスタマーサクセスマネージャーに連絡してください。
アプリに Microsoft 365 カレンダープラグインを追加する
スタジオで、プラグインを追加 に移動します。
Microsoft 365 カレンダープラグインの インストール をクリックします。
カレンダーを追加するためのダイアログが開きます。
カレンダーのタイトルを入力し、アクセスおよび表示設定を設定します。
カレンダーを追加 をクリックします。
Microsoft 365 カレンダープラグインと最初のカレンダーが追加されました。後でさらにカレンダーを追加できます。
アプリを介してアクセス権を付与する
プラグインに初めてアクセスする際は、アプリのアクセス権を付与する必要があります。
アクセス権を付与するには、グローバル管理者 Microsoft 365 の役割が必要です。
スタジオで、作成した Microsoft 365 カレンダーを開きます。
Microsoft 365 サインイン ページが開きます。
サインイン をクリックし、Microsoft アカウントにサインインする手順に従います。
組織を代表して同意する のチェックボックスを選択し、受け入れる をクリックしてください。
スタジオが開いているタブに戻り、作成した Office 365 カレンダーをさらに設定します。
個人 Microsoft 365 カレンダーの設定
個人カレンダーを追加すると、ユーザーはアプリ内で自分の個人 Microsoft 365 カレンダーを見ることができます。個人カレンダーは、イベントや予定のリストとして表示されます。
設定を構成するには、Microsoft 365 にサインインする必要があります。
スタジオで、作成した Microsoft 365 カレンダーを開きます。
カレンダーの種類として 個人カレンダー を選択します。
自分のカレンダーがプレビュー表示されます。設定後、アプリ内で各ユーザーは自分のカレンダーを見ることができます。
保存 をクリックします。
共有 Microsoft 365 カレンダーの設定
共有カレンダーは、共有 Microsoft 365 メールボックスに属するカレンダーです。このメールボックスにアクセスできるすべてのユーザーは、アプリメニューでカレンダーを見ることができます。共有カレンダーは、イベントや予定のリストとして表示されます。
設定を構成するには、Microsoft 365 にサインインする必要があります。
スタジオで、作成した Microsoft 365 カレンダーを開きます。
カレンダーの種類として 共有カレンダー を選択します。
ドロップダウンメニューから、共有カレンダーを表示したいメールボックスを選択します。
共有カレンダーがプレビュー表示されます。
保存 をクリックします。
Microsoft 365 グループカレンダーの設定
Microsoft 365 グループの場合、グループカレンダーをアプリに表示できます。グループのすべてのメンバーは、アプリメニューでカレンダーを見ることができます。グループカレンダーは、イベントや予定のリストとして表示されます。
設定を構成するには、Microsoft 365 にサインインする必要があります。
スタジオで、作成した Microsoft 365 カレンダーリストを開きます。
カレンダーの種類として グループカレンダー を選択します。
ドロップダウンメニューから、グループカレンダーを表示したいグループを選択します。
グループカレンダーがプレビュー表示されます。
保存 をクリックします。
アプリでの Microsoft 365 カレンダーの公開
スタジオで、公開したい Microsoft 365 カレンダーに移動します。
MS365 カレンダーを公開 をクリックします。
従業員にとっての Microsoft 365 カレンダーはどのように見えますか？
モバイルデバイスでは、Microsoft 365 カレンダーはアプリ内ブラウザーに表示されます。
従業員アプリをインターネットブラウザーで使用する場合、Microsoft 365 カレンダーはウェブアプリに統合されています。
