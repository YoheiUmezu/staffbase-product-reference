# ミールプランプラグインのインストール

**URL**: https://support.staffbase.com/hc/ja/articles/360032261692
**取得日**: 2026-05-02 18:47:55 JST

## 内容

ミールプランプラグインを使用すると、アプリ内でカフェテリアで利用可能な食事をユーザーに知らせることができます。追加する各食事について、詳細な説明、食事が利用可能な日付、さらにはユーザー向けの価格情報を提供できます。ミールプランプラグインをインストールし、最初の食事オプションを追加しましょう。
ミールプランプラグインのインストール
スタジオで、コンテンツメニューに移動します。
メニューの一番下までスクロールし、プラグインを追加をクリックします。
ミールプランに移動し、インストールをクリックします。
ミールプランを追加するためのダイアログが開きます。
ミールプランのタイトルを提供します。
ドロップダウンメニューからミールプランの可視性を設定します。
ミールプランを追加をクリックします。
最初のミールプランが追加されました。アプリ内でいつでも食事オプションや追加のミールプランを追加できます。ユーザーにミールプランを表示させるには、まずそのミールプランを公開する必要があります。
