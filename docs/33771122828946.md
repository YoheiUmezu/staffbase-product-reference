# キャンペーンのための多言語アラインメント調査の作成

**URL**: https://support.staffbase.com/hc/ja/articles/33771122828946
**取得日**: 2026-05-02 18:48:27 JST

## 内容

アクティベーション
システム全体のエディター
アラインメント調査は、キャンペーンのナarrativesに関する従業員からの直接的な定性的フィードバックを提供します。 詳細はこちら。 従業員アプリやイントラネットで多言語のコンテンツを作成できます。これにより、読者は選択したコンテンツ言語でコンテンツを読むことができます。アラインメント調査に多言語コンテンツを追加する際には、組織で有効になっている言語を選択できます。追加の言語が必要な場合は、管理者がそれを有効にできます。
多言語でコンテンツを作成することは、組織の設定が行われている場合のみ可能です。詳細はこちら。このオプションが必要な場合は、管理者に連絡してください。
アラインメント調査で多言語コンテンツを作成する方法は2つあります。
ゼロから始める
このオプションでは、手動で翻訳を追加し、アラインメント調査の選択した言語で新しいタブを作成します。テキストフィールドは空のままで、新しい言語で直接コンテンツを書くか、コピー＆ペーストできます。
コンテンツをコピーする
このオプションでは、アラインメント調査の選択した言語で新しいタブを作成し、元の言語のコンテンツをテキストフィールドにコピーします。
言語を追加ダイアログは、ユーザーのインターフェース言語で表示され、3つのプリセットスケール評価ラベルが含まれます。これらのラベルは、プレビューセクションおよび最終調査の言語タブに更新されます。
前提条件
あなたはミッションコントロールベータに参加しています。参加していない場合は、こちらから参加してください。
キャンペーンでアラインメント調査を作成しました。
多言語コンテンツを手動で追加する
このオプションでは、新しい言語で手動でコンテンツを追加できる新しいタブが作成されます。
スケール評価ラベルは、アラインメント調査の各言語バージョンで同じままです。新しい言語でラベルを更新すると、元の言語も自動的に更新されます。
スタジオで、計画 > キャンペーン > 特定のキャンペーンに移動します。
アラインメント調査を作成。
+ ボタンをクリックします。
言語を追加ダイアログが開きます。
ドロップダウンメニューから1つ以上の言語を選択します。
クリックして適用します。
選択してゼロから始める。
クリックして追加します。
新しい言語でアラインメント調査を設定します。
カスタムスケール評価ラベルの場合、新しい言語でテキストを手動で入力する必要があります。
クリックして保存します。
1つ以上の新しい言語をアラインメント調査に追加しました。各言語には調査の上部に異なるタブがあります。これで、その言語で調査の質問とスケールラベルを追加できます。
コピーによる多言語コンテンツの追加
このオプションでは、コピーしている言語の既存のコンテンツバージョンを含む新しいタブが作成されます。
プリセットのスケール評価ラベルは、アラインメント調査の各言語バージョンで同じままです。新しい言語でラベルを更新すると、元の言語も自動的に更新されます。
スタジオで、計画 > キャンペーン > 特定のキャンペーンに移動します。
アラインメント調査を作成。
+ ボタンをクリックします。
ダイアログが開きます。
ドロップダウンメニューから1つ以上の言語を選択します。
クリックして適用します。
選択してコンテンツをコピーします。
コピー元のコンテンツドロップダウンメニューが表示されます。
コンテンツをコピーする言語を選択します。
クリックして追加します。
新しい言語でアライメント調査を設定する。
カスタムスケール評価ラベルについては、新しい言語でテキストを手動で入力する必要があります。
保存をクリックします。
アライメント調査に1つ以上の新しい言語を追加しました。各言語はページの上部に異なるタブがあり、選択したソース言語から内容がコピーされています。
多言語コンテンツの削除
もはや必要ない場合や誤って追加した場合に備えて、言語コンテンツのバージョンを個別に削除できます。
スタジオで、計画 > キャンペーンに移動します。
特定のキャンペーンの隣にある3つのドットをクリックしてアクションメニューを開きます。
アライメント調査をクリックします。
ゴミ箱アイコンをクリックします。
言語を削除ダイアログが開きます。
ドロップダウンメニューから1つ以上の言語を選択します。
適用をクリックします。
クリックして削除します。
保存をクリックします。
選択した言語のアライメント調査バージョンが永久に削除されました。いつでも言語バージョンを再追加できます。
