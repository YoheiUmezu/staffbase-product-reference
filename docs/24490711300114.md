# スタッフベースメールによるターゲティングの概要

**URL**: https://support.staffbase.com/hc/ja/articles/24490711300114
**取得日**: 2026-05-02 18:42:39 JST

## 内容

この記事はAIを使用して翻訳されており、不正確な点が含まれている可能性があります。情報に不明な点がある場合は、正確性を確認するために英語の原文を参照してください。
スタッフベースメールでは、ユーザーグループを使用して正確なメールターゲティングを実現し、メッセージが毎回正しい視聴者に届くようにします。
従業員をプロフィールフィールド情報に基づいて、勤務地、部署、その他のプロフィールフィールドなどでセグメント化できます。グループが設定されると、新しい従業員はそのプロフィールフィールドがグループの条件に一致する場合に自動的に割り当てられます。同様に、既存の従業員もプロフィールフィールドがグループの基準に一致または異なるように変更されると、自動的に追加または削除されます。これにより、ターゲティングが正確で最新の状態に保たれます。
フォルダまたはメールの視聴者には、複数のユーザーグループや個別の受信者を含めることができます。メールを送信する際に、カスタムターゲットグループを含むCSVファイルをインポートすることもできます。
ユーザーグループについて詳しく知るには、以下のリンクを参照してください：
スタッフベースメールのユーザーグループ（スタッフベースメールを単独製品として使用する場合）
ユーザーグループの種類（スタッフベースメールをスタッフベースイントラネットまたは従業員アプリと一緒に使用する場合）
ターゲット視聴者の概要ページでは、グループがアイコンで表示され、マウスを重ねるとグループメンバーの数が表示されます。個別のユーザーは、アイコンと名前で表示されます。対応するプロフィールフィールドが設定されている場合、個別のユーザーは部署やユーザー役割によっても識別できます。
