# メールアイコン

**URL**: https://support.staffbase.com/hc/ja/articles/4407067687314
**取得日**: 2026-05-02 19:00:51 JST

## 内容

メールアイコンは、Staffbaseプラットフォームのコンテンツがメールにリンクされている場合に使用されます。メールアイコンは、リンクされたコンテンツのプレビューの上部に表示されます。
要件
サイズ: 幅360 px、高さ最大360 px
ファイル形式: png
解像度: 72 dpi
カラープロファイル: sRGB
ファイル名: Emailicon.png
透明度: 許可されている
カラーロゴ - グレーではない
