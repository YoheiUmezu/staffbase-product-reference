# グローバルメールハブの概要

**URL**: https://support.staffbase.com/hc/ja/articles/31207082240018
**取得日**: 2026-05-02 18:50:54 JST

## 内容

メールは、Studioの左側のナビゲーションにあり、アクセスできるすべてのメールを個別のフォルダを探し回ることなく、まとめて見つけて確認・管理できる中央の場所を提供します。このグローバルメールハブでは、タイトルで検索したり、フィルターオプションで絞り込んだりすることで、特定のメールを簡単に見つけることができます。
グローバルメールハブは、複数のスペースやフォルダを横断して作業するユーザーや、メールがどこで作成されたか分からない場合に特に役立ちます。また、多くのフォルダや大規模なメールライブラリを管理する役割の方にも、組織全体のメールコミュニケーションを効率的かつ一元的に見つけて操作できる便利な方法を提供します。
このハブは、あなたの権限やアクセスできるスペースに応じて表示が変わります。スペースにアクセスできるユーザーは、スペースやフォルダでフィルターできます。フォルダの編集者は、アクセス権のあるすべてのフォルダをまとめて表示できます。スペースピッカーにアクセスできないユーザーは、閲覧権限のあるすべてのメールが集約されたリストを表示します。
フォルダレベルでは、メールハブで次のメールをまとめて表示できます：
下書き
スケジュール
送信済み
検索する
グローバルメールハブのすべてのタブ（下書き、スケジュール、送信済み）で、メールのタイトルを検索できます。
フィルターオプション
検索を絞り込むための複数のフィルターオプションがあります。
下書き、スケジュール、送信済みタブのフィルターオプション
下書き、スケジュール、送信済みタブでは、次のフィルターを使って表示データを絞り込めます：
フィルタータイプ
フィルターオプションと詳細
作成者
メールを作成した作成者を選択します。
デフォルト値はすべての作成者です。
複数の作成者を同時に選択できます。
スペース
スペースにアクセスできるユーザーが利用できます。
メールが作成されたスペースを選択します。
デフォルト値はすべてのスペースです。
複数のスペースを同時に選択できます。
フォルダ
メールが保存されたフォルダを選択します。
デフォルト値はすべてのフォルダです。
複数のフォルダを同時に選択できます。
送信済みタブの追加フィルターオプション
フィルタータイプ
フィルターオプションと詳細
日付範囲
メールが送信された期間を選択します。
デフォルト値はすべての日付です。
利用可能なオプションは次の通りです：
すべての日付
過去7日間
過去30日間
過去3か月
過去6か月
過去1年
視聴者
メールが送信された視聴者を選択します。
デフォルト値はすべての視聴者です。
複数の視聴者を同時に選択できます。
送信者
メールを送信したユーザーを選択します。
デフォルト値はすべてのユーザーです。
複数のユーザーを同時に選択できます。
並べ替えオプション
下書き、スケジュール、送信済みタブでは、次の並べ替えオプションで表示データを並べ替えることができます：
新しい順
古い順
