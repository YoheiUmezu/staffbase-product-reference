# Googleデータセーフティセクションのデータ型

**URL**: https://support.staffbase.com/hc/ja/articles/4677057837074
**取得日**: 2026-05-02 18:43:57 JST

## 内容

Googleデータセーフティアンケートに回答して、従業員にとって安全なアプリにしましょう。このアンケートは、アプリを通じて収集または共有されるユーザーデータに関する詳細情報を提供する3つのセクションで構成されています。この記事のデータタイプの概要は、Staffbaseアプリのアンケートに回答するのに役立ちます。
Googleデータセーフティアンケートに記入する前に、法的アドバイザーに相談してください。この概要は、Staffbaseアプリで使用される一般的なデータタイプのコレクションです。独自のアプリのデータタイプは、カスタムプラグインやその他のカスタマイズの使用によって異なる場合があります。
アンケートセクション1：データ収集とセキュリティ

Googleは2023年4月にデータ収集とセキュリティアンケートに追加の質問を導入しました。既存の顧客の場合、2023年12月7日までに追加の質問に対する回答を提出する必要があります。

アンケートセクション2：データタイプ
