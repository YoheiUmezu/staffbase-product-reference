# スクリーンでのデバイス管理

**URL**: https://support.staffbase.com/hc/ja/articles/21703452837010
**取得日**: 2026-05-02 18:51:18 JST

## 内容

アクティベーション
管理エディター
オフィスのラウンジ、受付、またはカフェテリアにあるデバイスを、Screensのデバイス管理機能で登録します。Screensのデバイス管理ページでは、ユニークなコードを使用して、これらのデバイスにニュースやコンテンツを安全に表示することができます。
このページでは、すべてのデバイスの概要、現在の接続状況、およびどのスライドショーがどのデバイスで表示されているかを確認できます。これにより、明確な概要が得られ、関連するコンテンツが各デバイスに適切に表示されることが保証されます。
デバイスの状態
スライドショーが開かれ、リンクされ、再生されているとき、デバイスは接続されています。これは、ブラウザーとスタジオ間で通信が行われていることを意味します。接続状況にマウスを合わせると、デバイスの状態が最後に変更された時刻に関する情報が表示されます。
ブラウザーのタブを閉じるか、リンクがブラウザーで再生されなくなると、デバイスは切断されます。切断は次のような場合に発生する可能性があります：
部分的に切断された場合：スライドショーのリンクがデバイスで2時間以上開かれていない場合、デバイスは「切断済み」と表示されます。7日以内にデバイスでスライドショーを再度開くと、状態は「接続済み」に変更されます。
完全に切断された場合：スライドショーのリンクがデバイスで7日以上開かれていない場合、デバイスは「切断済み」と表示されます。これは、デバイスに問題があるか、インターネット接続に問題がある場合に発生する可能性があります。7日後、スライドショープレーヤーを再度開くと、デバイスのピンコードが表示されます。
デバイスの設定
スクリーンのデバイス管理ページから、デバイスを簡単に接続、名前変更、または削除できます。
デバイスの接続
スライドショーを表示するために、デバイスをスタジオに接続します。
スタジオで、Screens > Device Managementに移動します。
デバイス管理ページが開きます。
デバイスを接続をクリックします。
デバイスを接続ページが開き、URLが表示されます。
接続したいデバイスでURLを開きます。
確認コードが表示されます。
スタジオのデバイスを接続ページで、確認コードを貼り付けるか追加します。
青い矢印をクリックします。
デバイスをScreensに正常に接続しました。デバイス名がランダムに生成されましたので、すぐにデバイスの名前を変更するか、以下の手順に従ってください。これで、スライドショーをこのデバイスにリンクできます。
デバイスの名前変更
デバイスの名前を変更して、Screensのデバイス管理ページで見つけやすくすることができます。
スタジオで、Screens > Device Managementに移動します。
デバイス管理ページが開きます。
名前を変更したいデバイスを見つけるために、検索、フィルタリング、またはソートします。
アクションの下で、3つのドットをクリックしてアクションメニューを開きます。
名前変更をクリックします。
デバイスの名前変更ダイアログが開きます。
デバイス名フィールドに関連する名前を追加します。
名前変更をクリックします。
デバイスの名前を正常に変更しました。いつでもデバイスの名前を再度変更できます。これで、スライドショーをこのデバイスにリンクできます。
デバイスの削除
使用されていないデバイスを削除できます。
スタジオで、Screens > Device Managementに移動します。
デバイス管理ページが開きます。
削除したいデバイスを見つけるために、検索、フィルタリング、またはソートします。
アクションの下で、3つのドットをクリックしてアクションメニューを開きます。
削除をクリックします。
確認ダイアログが開きます。
削除をクリックします。
デバイスを正常に削除しました。いつでもデバイスを再接続できます。
