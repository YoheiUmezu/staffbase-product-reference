# スタッフベースメールで名前を使ってメールをパーソナライズする

視聴者と交流し、メールをより直接的かつ個人的にすることで影響力を高めましょう。Staffbaseメールの名前パーソナライズ機能を使用すると、メールの本文、件名、プレビュー テキストで受信者の名前を自動的に挨拶できます。
この機能は、Studioを介してStaffbaseメールから直接メールを送信する場合にのみ利用可能です。アドインや拡張機能をインストールしている場合や、メールクライアントから追跡メールを送信する場合は、この記事は該当しません。

パーソナライズタグとフォールバックテキスト
Staffbaseメールのパーソナライズは、Microsoft Officeスイートのメールマージ機能と同様に、マージタグを使用して機能します。例えば、「hey {{firstName}}」のように、次のプロフィールフィールドを参照します。
名
姓
また、名や姓のフィールドが空の場合に置き換えるカスタムフォールバックテキスト「everyone」などを設定することもできます。
タグとメールごとに定義できるフォールバックテキストは1つだけです。このフォールバックテキストは、メール内のすべてのタグのインスタンスに適用されます: メールテキスト、件名、プレビュー テキスト。
メールを送信すると、使用しているプロフィールフィールドと、どのプロフィールフィールドが埋まっているか、または空であるかに応じて、受信者に異なる挨拶が表示されます。例えば:
「Hey {{firstName}} {{lastName}}」 (フォールバックテキストが設定されていない場合)
「Hey John Doe」、名と姓のプロフィールフィールドが両方とも埋まっている場合。
「Hey John」、姓のプロフィールフィールドが空の場合。
「Hey Doe」、名のプロフィールフィールドが空の場合。
「Hey」、名と姓のプロフィールフィールドが両方とも空の場合。
「Hey {{firstName}}」
「Hey everyone」、名のフィールドが空で、フォールバックテキスト「everyone」を設定した場合。
「Hey」、名のフィールドが空で、フォールバックテキストが設定されていない場合。
メールのパーソナライズ
前提条件
意図した受信者のプロフィールの名前フィールドが正しく埋められており、メールに表示されること。
そうでない場合は、パーソナライズされたメールを送信する前に、管理者に連絡して名前フィールドを更新してください。
Studioで、メール > 下書きまたはテンプレート > メールデザイナーに移動します。
テキスト要素をクリックします。
ツールバーからパーソナライズをクリックし、{{firstName}}, {{lastName}}、または両方を選択します。
または、タグ{{firstName}}と{{lastName}}をテキスト要素に直接入力します。タグは大文字と小文字を区別するため、正しいスペルと大文字を使用してください。
名前タグがメールに追加されます。
オプションで、テキスト設定パネルのフォールバックテキストの下に、使用しているタグのフォールバック値を入力します。
メールを完成させ、次をクリックします。
サマリーページが開きます。
コンテンツの下で、パーソナライズをクリックして、件名またはプレビュー テキストにパーソナライズタグを追加します。
オプションで、フォールバックテキストの下に、使用しているタグのフォールバック値を入力します。
また、Staffbase AI Companionを使用してパーソナライズタグを含む件名とプレビュー テキストを生成するために、魔法の杖アイコンを使用するか、件名またはプレビュー テキストにタグ{{firstName}}と{{lastName}}を直接入力することもできます。この場合、タグは大文字と小文字を区別するため、正しいスペルと大文字を使用してください。
名前でメールをパーソナライズし、フォールバックテキストを提供しました。タグのフォールバックテキストは、現在編集しているメールにのみ影響します。他のメールの下書きやスケジュールされたメールのタグのフォールバックを更新することはありません。
パーソナライズされたメールのプレビュー
パーソナライズのタグを含む下書きまたはテンプレートで、プレビューモードに切り替えます。
パーソナライズドロップダウンを開きます。
フォールバックテキストを選択して、受信者のプロフィールフィールドが空の場合にメールがどのように見えるかをプレビューします。
オプションで、テストメールを送信して、送信されたメールでパーソナライズされた挨拶がどのように見えるかを確認します。
多言語メールでのパーソナライズの仕組み
メールの新しい言語バージョンを作成する場合、その新しい言語バージョンのフォールバックテキストを再入力する必要があります。フォールバックテキストは自動的に翻訳されません。
