# ジャーニーのアクティベーション

**URL**: https://support.staffbase.com/hc/ja/articles/8236641286034
**取得日**: 2026-05-02 18:56:56 JST

## 内容

アクティベーション
システム全体のエディター
ターゲットオーディエンスを設定し、希望するジャーニーコンテンツを作成したら、ユーザーのためにジャーニーをアクティブにする準備が整いました。
スタジオで、コンテンツ > ジャーニーに移動します。
アクティブにしたいジャーニーに移動します。
ジャーニーをアクティブ化をクリックします。。
ジャーニーは現在アクティブで、ユーザーはジャーニーコンテンツの受信を開始できます。
