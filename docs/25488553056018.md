# アプリグラフィックスの概要

**URL**: https://support.staffbase.com/hc/ja/articles/25488553056018
**取得日**: 2026-05-02 18:59:29 JST

## 内容

Staffbaseは、機能が豊富であるだけでなく、ロゴ、アイコン、画像で視覚的に強化された従業員アプリを提供します。これにより、従業員アプリに組織の独自のアイデンティティとその価値を与えます。
グラフィックファイルのチェックリスト
Staffbaseに送信する必要があるグラフィックファイルの完全なリストです。グラフィックファイルは、ZIPフォルダーにまとめてStaffbaseサポートまたはカスタマーサクセスマネージャーに送信するか、ダウンロードリンクを提供してください。
