# 高度な編集カレンダーでの週次ビューの使用

**URL**: https://support.staffbase.com/hc/ja/articles/19107705166354
**取得日**: 2026-05-02 18:49:11 JST

## 内容

アクティベーション
エディター
編集カレンダーの改善された週次ビューは、プラットフォーム内のニュース投稿やキャンペーンの概要を提供し、デザインが更新されました。キャンペーン、スペース、チャネルのフィルターを使用して、特定のコンテンツに絞り込むことができます。
改善された週次ビューから、便利に以下の操作ができます：
キャンペーン、ニュース投稿、ブロッカーを表示
ニュース投稿とブロッカーを作成
ニュース投稿とブロッカーをドラッグ＆ドロップ
ドラフト投稿をカレンダーに追加
ニュース投稿、キャンペーン、またはブロッカーをクリックして、主要情報を含むフライアウトを表示
フライアウト内のニュース投稿タイトルをクリックして、ニュース投稿を開く
フライアウトからキャンペーンの詳細に移動
フライアウト内の記事およびメディア投稿の主要情報を編集
フライアウト内のブロッカーを編集
キャンペーン、スペース、またはチャネルでビューをフィルタリング
月次ビューに切り替え
オプションのフル透明モードを使用して、通常アクセスできないコンテンツを表示（表示専用形式）
ポストフライアウトでは、以下のような有用な情報を表示できます：
ターゲットオーディエンス
アクティブな通知タイプ
作成者
投稿のステータス
投稿ステータスが公開の場合の投稿統計
キャンペーンフライアウトでは、以下のような有用な情報を表示できます：
キャンペーンのステータス
開始日と終了日
キャンペーンの目標
ブロッカーフライアウトでは、以下のような有用な情報を表示できます：
開始時刻と終了時刻
説明
作成者
