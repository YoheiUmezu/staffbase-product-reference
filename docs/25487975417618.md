# プラグインの削除

**URL**: https://support.staffbase.com/hc/ja/articles/25487975417618
**取得日**: 2026-05-02 18:53:22 JST

## 内容

削除したプラグインは30日以内に復元できます。
Staffbaseプラットフォームは、地図、食事プラン、イベント登録、カレンダーなど、さまざまなプラグインを提供しています。カレンダープラグインを例として使用していますが、従業員アプリから任意のプラグインを削除する手順に従うことができます。
スタジオで、削除したいプラグインに移動します。
ギアホイールをクリックして設定を開きます。
設定ページが開きます。
プラグインを削除します。
プラグインを削除するための確認メッセージが表示されます。
確認してプラグインを削除します。
