# 管理タスクの役割

**URL**: https://support.staffbase.com/hc/ja/articles/360025264511
**取得日**: 2026-05-02 18:56:12 JST

## 内容

管理者は、Staffbaseプラットフォームでユーザーが持つことができる最も強力な役割です。管理者は、機密のユーザー情報やアプリの機能、動作、外観を変更する基本的なアプリ設定にアクセスできます。
管理者の他に、プラットフォームの特定の領域に対してのみ管理権限を持つ役割があります。たとえば、スペース管理者やシステム全体の編集者などです。
通常、管理者は非常に少数で十分であり、Studioの使用について教育を受けていることを確認します。グループ、コンテンツ、スペースの管理には、必要な領域にのみアクセスできるそれぞれの役割を割り当てます。

グループ管理者
グループ管理者は、割り当てられたグループのグループ設定とグループメンバーを変更できます。グループ管理者は、ユーザー管理の他の部分にアクセスすることはできません。
これらの権限は、手動で作成されたグループにのみ適用されます。タグに基づく条件付きグループを使用する場合、グループ管理者はメンバーを追加または削除することはできません。
スペース管理者
スペース管理者は、コンテンツ管理とスペース設定に関するスペース全体の管理権限を持っています。しかし、スペース管理者はユーザー管理にアクセスすることはできません。
システム全体のエディター
システム全体のエディターは、通常のエディターよりも多くの権限を持ち、プラットフォーム内のスペース、ニュースチャンネル、投稿、ページ、およびその他のプラグインを管理できます。
管理エディター
管理エディターは、システム全体の権限を持ち、プラットフォーム内のスペースを表示し、ニュースチャンネル、投稿、ページ、およびその他のプラグインを追加できます。
管理者
管理者は、プラットフォームのすべての領域に対するシステム全体の管理権限を持っています。
