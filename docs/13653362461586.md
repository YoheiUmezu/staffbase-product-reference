# モバイルポッドキャストプレーヤーの概要

**URL**: https://support.staffbase.com/hc/ja/articles/13653362461586
**取得日**: 2026-05-02 18:59:36 JST

## 内容

従業員アプリでポッドキャスト、クリップ、またはその他の音声ファイルを簡単に聴くことができます。モバイルポッドキャストプレーヤーを使用すると、特定のニュース投稿やページを読んでいる間だけでなく、次のような状況でも音声を聴くことができます：
従業員アプリ内の他のコンテンツをスクロールしているとき
アプリがデバイスのバックグラウンドで開いているとき
モバイルデバイスがロックされているとき
音声ファイルを聴いている間、音声のタイトル、長さ、画像が表示されたミニプレーヤーが表示されます。また、いつでもポッドキャストを一時停止または閉じることができます。
音声を再生し続けるには、従業員アプリがバックグラウンドで開いている必要があります。アプリを積極的に使用していない場合やモバイルデバイスがロックされている場合でも、音声は再生されます。
ユーザーにとってのモバイルポッドキャストプレーヤーの外観
アプリを使用している間
従業員アプリで音声を再生すると、画面の下部にミニプレーヤーが表示されます。ミニプレーヤーは、投稿自体やアプリの他のエリアをスクロールしているときもこの位置に留まります。
モバイル画面がロックされている間
従業員アプリで音声を再生し、その後電話をロックすると、ロック画面に音声が表示されます。
