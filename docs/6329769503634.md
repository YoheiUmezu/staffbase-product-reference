# 従業員メールを使用して未開封者にフォローアップメールを送信する

**URL**: https://support.staffbase.com/hc/ja/articles/6329769503634
**取得日**: 2026-05-02 18:55:40 JST

## 内容

マネージャー
Power/Business または Enterprise プランが必要です。
有効にされている場合、機能アクセス設定により、すべてのアカウントタイプに対してメールの送信/スケジュール機能への個人のアクセスを追加または削除できます。
この機能は、従業員メールウェブアプリから直接送信されたメールにのみ利用可能です。
従業員メールのアドインや拡張機能を使用して送信されたメールには利用できません。
従業員メールウェブアプリからカスタム配信リストにメールを送信した後、元のメッセージを開かなかった受信者をターゲットにしたフォローアップメールを送信するオプションがあります。
フォローアップメールに対してフォローアップを送信することも可能です。
追跡された各メール（初回送信またはフォローアップのいずれか）は、それぞれ独自の分析セットを持つことに注意してください。
これにより、リマインダーが必要な人々の注意を再び引き、すぐにコミュニケーションを開いた人々には受信トレイの重複を減らすことで報います。
送信したメールへのフォローアップを送信する
従業員メールウェブアプリで、メール > 送信済みメールに移動します。
フォローアップしたいメールを開きます。
注: メールはウェブアプリから送信されている必要があります。各メールのアイコンと配信リスト名は、どのように送信されたかを示します。
ウェブアプリ:
アドイン:
新しいフォローアップをクリックします。
あなたのドラフトのいずれかを選択するか、テンプレートから、最初から、または送信済みメールから新しいドラフトを作成します。
Staffbaseは、ほぼ送信準備が整った保存済みドラフトを持つことを推奨します。新しいドラフトを生成し、複数のセッションで編集するよりも、ほぼ完成したドラフトを使用する方が良いです。
オプションで、Email Designerでドラフトフォローアップメッセージを編集して保存します。
フォローアップを送信する前に、最後の詳細を調整するために続行をクリックします。
オプションで、件名や追加フィールドを編集します。
注: 配信リストフィールドには、新しいリストが生成されます。このフォローアップリストの個々の受信者はプライバシーのために匿名化されており、メールに追加の配信リストを追加することはできません。
ボタンをクリックします。ラベルは今すぐ送信またはスケジュールを保存です。メールを後で送信するようにスケジュールしたかどうかに応じて。
元のメールを開かなかった人々に送信されるフォローアップメールを作成しました。
送信したフォローアップメールの表示
従業員メールウェブアプリで、メール > 送信済みメールに移動します。
送信済みメールのリストでは、フォローアップが明確にマークされ、元の送信メールに接続されています。
フォローアップメールが送信された受信者の数を確認するには、送信済みメールを開き、サマリータブを確認します。
追加のフォローアップを送信する
従業員メールウェブアプリで、メール > 送信済みメールに移動します。
フォローアップしたい送信済みフォローアップメールを開きます。
新しいフォローアップをクリックして、残りの未開封者をターゲットにします。
