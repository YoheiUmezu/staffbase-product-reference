# Microsoft 365 検索の概要

**URL**: https://support.staffbase.com/hc/ja/articles/360015269899
**取得日**: 2026-05-02 18:47:44 JST

## 内容

ベータ
アクティベーション
管理者
Microsoft 365 検索は、内部コミュニケーションプラットフォームのユーザーが、さまざまなプラットフォームでコンテンツを見つけるのをこれまで以上に簡単にします。Microsoft 365 検索を使用すると、ユーザーは Staffbase プラットフォーム内のコンテンツだけでなく、Microsoft 365 に存在するコンテンツもリモートで検索できます。SharePoint や OneDrive を含む Microsoft 365 からの検索結果は、Staffbase プラットフォームの別の検索タブに便利に表示されます。
シームレスな移行
Microsoft 365 検索タブの結果をクリックすると、ユーザーは直接 Microsoft 365 内の目的のコンテンツに移動します。これにより、ユーザーは別のプラットフォームで再度コンテンツを検索する手間が省け、同時にコンテンツ作成者はコンテンツの重複を避けることができます。ユーザーが Microsoft アカウントにサインインすると、Staffbase または Microsoft プラットフォームに存在するかどうかを気にすることなく、探しているコンテンツにシームレスに移行できます。
Microsoft 365 検索をユーザーに提供するには、Microsoft Connections を追加する必要があります。詳細を学ぶ。
