# グループサブスクリプションウィジェットの設定

**URL**: https://support.staffbase.com/hc/ja/articles/360019152219
**取得日**: 2026-05-02 19:00:27 JST

## 内容

ベータ
価格設定
アクティベーション
著者
グループサブスクリプションウィジェットは、ユーザーが興味のあるオープングループコンテンツに簡単にサブスクライブしたり、アプリやイントラネットで表示するコンテンツをカスタマイズしたりするためのコール・トゥ・アクションボタンです。グループサブスクリプションウィジェットを設定して、オープングループコンテンツの認知度を高め、リーチを拡大します。
スタジオで、グループサブスクリプションウィジェットを追加したいページに移動します。
グループサブスクリプションウィジェットを追加します。
グループサブスクリプションウィジェットを設定するためのダイアログが開きます。
オプションで、タイトルを提供します。
このタイトルはウィジェットの上に表示され、サブスクリプションボタンのテキストとしては表示されません。
ウィジェットが表示されるデバイスタイプを、対応するアイコンをクリックして選択します。
アプリ内の表示の下で、ウィジェットを表示したいユーザーを選択します。
ドロップダウンメニューから、ユーザーがボタンをクリックしたときにサブスクライブするオープングループを選択します。
オプションで、ユーザーがグループに参加または離脱する際にボタンに表示されるカスタムテキストを入力します。
オプションで、サブスクリプションボタンとテキストの色をカスタマイズします。
サブスクリプションボタンとテキストの色をカスタマイズしない場合、組織のブランドカラーがデフォルトで使用されます。
OKをクリックします。
アプリ内でのユーザーによるグループサブスクリプションウィジェットの表示方法
