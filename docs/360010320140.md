# ユーザーグループ向けのコンテンツターゲティングの適用

**URL**: https://support.staffbase.com/hc/ja/articles/360010320140
**取得日**: 2026-05-02 18:52:21 JST

## 内容

エディター
ユーザーグループを作成すると、すぐにコンテンツターゲティングに利用可能になります。コンテンツの可視性を調整することで、特定のコンテンツを1つまたは複数のユーザーグループに提供できます。ここでは、カレンダープラグインを例として使用します。
可視性を調整したいコンテンツに移動します。
ギアアイコンをクリックして設定を開きます。
設定ページが開きます。
アプリ内の可視性の下で、選択したユーザー、グループ、またはAPIトークン向けをドロップダウンメニューから選択します。
新しいフィールドが開きます。
ドロップダウンメニューからコンテンツを表示したいユーザーグループを選択します。
保存をクリックします。
このコンテンツは、選択したユーザーグループにすぐに利用可能になります。
