# スタッフベースメールで多言語メールを作成する

**URL**: https://support.staffbase.com/hc/ja/articles/24751859954578
**取得日**: 2026-05-02 18:46:10 JST

## 内容

編集者
AI
前提条件
コンテンツを作成するために選択した言語は、プラットフォームで利用可能でなければなりません。
管理者がシステム設定で言語を追加する方法を学ぶ。
Staffbase メールで多言語コンテンツを作成するには、3つの異なる方法があります：
ゼロから始める
このオプションでは、翻訳を手動で追加し、選択した言語の新しいタブをメールデザイナーに作成します。メールエディターは空のままで、新しい言語でメールを直接書いたり、コピー＆ペーストしたりできます。
コンテンツをコピーする
このオプションでは、選択した言語の新しいタブをメールデザイナーに作成し、元の言語のコンテンツをメールエディターにコピーします。
自動でコピーして翻訳する
このオプションでは、選択した言語の新しいタブをメールデザイナーに作成し、メールコンテンツを新しい言語に自動で翻訳します。そこから、必要に応じてメールを追加または編集できます。
ソースメールを更新すると、その翻訳版は自動的に更新されません。すべての言語バージョンと下書きに手動で変更を適用する必要があります。
多言語メールを手動で追加する
このオプションは、新しい言語でコンテンツを手動で追加できる新しいタブを作成します。
Studioで、メール > 言語を追加したいメールに移動します。
メールデザイナーが開きます。デフォルトのプラットフォーム言語として設定された言語が最初の言語タブとして選択されます。デフォルトのプラットフォーム言語について詳しく学ぶ。
+ ボタンをクリックします。
言語を追加する ダイアログが開きます。
ドロップダウンメニューから1つ以上の言語を選択し、適用する をクリックします。
最初から始める を選択します。
追加する をクリックします。
追加された各言語に対して、メールエディターの上部に新しいタブが作成されます。
追加された各言語のタブでメールを作成し、次 をクリックします。
サマリーページが開きます。
設定 セクションで、ターゲット視聴者 を選択します。
コンテンツ セクションで、フォールバック言語 ドロップダウンメニューからオプションを選択します。
これはメールの各言語バージョンに適用されます。
フォールバック言語が使用される場合について詳しく学ぶ。
各言語バージョンのメールに対して、件名、プレビューテキスト、およびオプションで添付ファイルを追加します。
送信する をクリックします。
多言語でメールを作成し、送信することに成功しました。受信者がプラットフォームで優先言語を設定していない場合、またはメールに対して優先言語が利用できない場合、選択したフォールバック言語でメールを受け取ります。
翻訳なしで多言語コンテンツを追加する
このオプションは、コピーする言語の既存のバージョンを含む新しいタブを作成します。
Studioで、メール > 言語を追加したいメールに移動します。
メールデザイナーが開きます。デフォルトのプラットフォーム言語として設定された言語が最初の言語タブとして選択されます。デフォルトのプラットフォーム言語について詳しく学ぶ。
+ ボタンをクリックします。
言語を追加する ダイアログが開きます。
ドロップダウンメニューから1つ以上の言語を選択し、適用する をクリックします。
コンテンツをコピーする を選択します。
コピー元のコンテンツ ドロップダウンメニューが表示されます。
コンテンツをコピーする言語を選択し、追加する をクリックします。
追加された各言語ごとに、メールエディターの上部に新しいタブが作成され、選択したソースからコンテンツがコピーされます。
新しい言語でメールの翻訳を追加し、次 をクリックします。
サマリーページが開きます。
設定 セクションで、ターゲット視聴者 を選択します。
コンテンツ セクションで、フォールバック言語 ドロップダウンメニューからオプションを選択します。
これはメールの各言語バージョンに適用されます。
フォールバック言語が使用される場合について詳しく学ぶ。
各言語バージョンのメールに件名、プレビューテキスト、およびオプションで添付ファイルを追加します。
送信する をクリックします。
複数の言語でメールを作成し、コンテンツをコピーし、翻訳し、送信することに成功しました。受信者がプラットフォームで優先言語を設定していない場合、またはメールに対して優先言語が利用できない場合、選択したフォールバック言語でメールを受け取ります。
自動翻訳を使用した多言語メールの追加
このオプションは、新しい言語に自動的に翻訳された既存のメールコンテンツを含む新しいタブを作成します。
Studioで、メール > 言語を追加したいメールに移動します。
メールデザイナーが開きます。デフォルトのプラットフォーム言語として設定された言語が最初の言語タブとして選択されます。デフォルトのプラットフォーム言語について詳しく学ぶ。
+ ボタンをクリックします。
言語を追加する ダイアログが開きます。
ドロップダウンメニューから1つ以上の言語を選択し、適用する をクリックします。
コピーして自動的に翻訳する を選択します。
コピー元のコンテンツ ドロップダウンメニューが表示されます。
コンテンツをコピーする言語を選択し、追加する をクリックします。
追加された各言語について、メールエディターの上部に新しいタブが作成され、選択したソースから新しい言語に翻訳されたコンテンツが表示されます。
新しいメールの翻訳を確認し、必要に応じて編集します。次 をクリックします。
サマリーページが開きます。
設定 セクションで、ターゲット視聴者 を選択します。
コンテンツ セクションで、フォールバック言語 ドロップダウンメニューからオプションを選択します。
これはメールの各言語バージョンに適用されます。フォールバック言語が使用される場合について詳しく学ぶ。
各言語バージョンのメールに件名、プレビューテキスト、およびオプションで添付ファイルを追加します。
送信する をクリックします。
複数の言語でメールを作成し、コンテンツをコピーし、翻訳し、送信することに成功しました。受信者がプラットフォームで優先言語を設定していない場合、またはメールに利用可能な優先言語がない場合、選択したフォールバック言語でメールを受け取ります。
多言語コンテンツの削除
必要がなくなった場合や誤って追加した場合、言語コンテンツのバージョンを個別に削除できます。
Studioで、メール に移動します。
言語を削除したいメールを開きます。
ゴミ箱アイコンをクリックします。
言語を削除する ダイアログが開きます。
ドロップダウンメニューから1つ以上の言語を選択します。
削除する をクリックします。
下書きを保存する または 次 をクリックして変更を保存します。
選択した言語バージョンのメールを永久に削除しました。言語バージョンをいつでも再追加できます。
ユーザーの言語設定はどのように設定されますか？
プラットフォームで優先言語を設定していないユーザー、またはメールで利用できない優先言語を持つユーザーは、送信時に選択したフォールバック言語でメールを受け取ります。
アプリ/イントラネット統合ユーザー
ユーザーは、Studioでデフォルト言語を設定して、優先するコンテンツ言語を定義できます。管理者は、ユーザーデータCSVエクスポートを実行して、優先するコンテンツ言語を確認できます。
メール単独ユーザー
ユーザーのコンテンツ言語の設定は、管理者によってアップロードされ、言語フィールドにマッピングされる必要があります。これは、CSVファイルまたはWorkday統合を通じて行うことができます。
言語コードは正しいロケールコードに従う必要があります。いくつかの言語には地域のバリエーションがあることに注意してください。例えば、カナダのフランス語にはfr_CAを、フランスで話されるフランス語にはfr_FRを使用します。
