# SharePoint OnlineにおけるニュースセントラルWebパーツの設定の構成

**URL**: https://support.staffbase.com/hc/ja/articles/11078089077522
**取得日**: 2026-05-02 18:55:38 JST

## 内容

アクティベーション
Azure Active Directoryでシングルサインオン（SSO）を設定した後、SharePointサイトでニュースセントラルWebパーツの設定を構成できます。設定では、Staffbase Webアプリを構成し、プラットフォームのメール通知の設定を定義できます。
前提条件:
テナントのアプリカタログでWeb管理権限以上を持っている必要があります。
Microsoft Entra IDとのシングルサインオン（SSO）を構成している必要があります。
SharePoint Onlineサイトで、ページの右下隅にある設定アイコンをクリックします。
ニュースセントラル設定 ダイアログが表示されます。
ページの右下隅にある設定アイコンは、ニュースセントラルを初めて構成する際にのみ利用可能です。構成後のニュースセントラルの設定の変更について詳しく学んでください。
Staffbase WebアプリURLフィールドに、Staffbase WebアプリのURLを入力します。
ニュースセントラルをWebパーツとして構成しました。これで、SharePointページにニュースセントラルを追加する準備が整いました。
保存をクリックします。

Staffbase Webアプリは、推奨オプションとしてデフォルトで選択されています。
