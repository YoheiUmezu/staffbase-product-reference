# キャンペーンマネージャーの権限

**URL**: https://support.staffbase.com/hc/ja/articles/23028595869714
**取得日**: 2026-05-02 18:58:02 JST

## 内容

ベータ
アクティベーション
管理者
キャンペーンマネージャーの権限は、どのユーザーまたはユーザーグループが以下にアクセスできるかを決定します：
キャンペーン
戦略的計画のタイムライン（組織で利用可能な場合）
スマートインパクト（組織で利用可能な場合）
キャンペーンマネージャーの権限は、ユーザーの役割に追加して割り当てられます。これにより、ユーザーが投稿を作成する権利が自動的に与えられるわけではありません。
キャンペーンマネージャーの追加
スタジオで、計画 > キャンペーンマネージャーに移動します。
キャンペーンマネージャーを追加をクリックします。
キャンペーンマネージャーを追加 ダイアログが開きます。
ユーザーとグループフィールドで、ドロップダウンからユーザーまたはユーザーグループを検索または選択します。
追加をクリックします。
ユーザーまたはユーザーグループをキャンペーンマネージャーとして追加しました。
キャンペーンマネージャーの削除
スタジオで、計画 > キャンペーンマネージャーに移動します。
削除したいユーザーまたはユーザーグループに対応する削除アイコンをクリックします。
ユーザーを削除しますか？ ダイアログが開きます。
削除をクリックします。
ユーザーまたはユーザーグループをキャンペーンマネージャーとして削除しました。
