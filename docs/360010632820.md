# 条件付きグループ割り当てのためのプロファイルフィールドのタグ付けを有効にする

**URL**: https://support.staffbase.com/hc/ja/articles/360010632820
**取得日**: 2026-05-02 18:44:57 JST

## 内容

タグはユーザーのプロファイルに追加される属性です。これらのタグを使用すると、特定の条件に基づいて自動化されたグループ、いわゆる条件付きグループを作成できます。タグに基づいてユーザーグループを作成することで、特定の部門、興味グループ、または場所に対してコンテンツをターゲットにすることができます。これらのタグベースのユーザーグループは手動で管理することはできず、CSV同期またはユーザーAPIを介してメンテナンスが必要です。
既存のプロファイルフィールドや新しいフィールドに対してタグ付けを有効にすることができます。これらのプロファイルフィールドは非表示にすることができ、ユーザーのプロファイルに公開されないようにすることができます。
スタジオで、設定に移動します。
カスタマイズセクションで、プロファイルフィールドを選択します。
カスタムフィールドの下で、編集したいプロファイルフィールドにカーソルを合わせます。
設定を編集するためのギアアイコンが表示されます。
ギアアイコンをクリックして設定を開きます。
プロファイルフィールドを編集するためのダイアログが開きます。
オプションの下で、グループタグ付けを有効にするトグルを有効にします。
OKをクリックします。
ダイアログが閉じます。
ページの上部で、保存をクリックします。
このプロファイルフィールドは、ユーザーグループのタグ付けに利用可能になりました。
