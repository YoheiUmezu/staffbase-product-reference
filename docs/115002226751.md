# 法的文書とは何か、なぜ必要なのか？

**URL**: https://support.staffbase.com/hc/ja/articles/115002226751
**取得日**: 2026-05-02 18:45:14 JST

## 内容

Staffbaseでは、従業員アプリを提供する会社（あなた）とユーザー（あなたの従業員）との間の相互作用のルールを、従業員アプリ内で直接説明することができます。必要な文書の範囲と種類は、サービスを提供する国や従業員が働く国によって大きく異なります。たとえば、ドイツでは個人データのプライバシーに関する法律が非常に厳格ですが、アメリカでは利用規約が重要です。
Apple App StoreとGoogle Play Storeの両方は、プラットフォームを通じてアプリを提出する際に、少なくともプライバシーポリシーを用意することを要求しています。
一般的に、以下の法的文書を提供することをお勧めします：
プライバシーポリシー
利用規約
詳細については、法的アドバイザーに相談してください。
