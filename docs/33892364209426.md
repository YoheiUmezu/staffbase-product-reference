# StaffbaseとScreenCloudのよくある質問（FAQ）

**URL**: https://support.staffbase.com/hc/ja/articles/33892364209426
**取得日**: 2026-05-02 18:40:48 JST

## 内容

スタッフベースの統合機能を使って、ScreenCloud上の画面にスタッフベースのニュースを表示できます。ScreenCloudをデジタルサイネージソリューションとして利用している場合、このシンプルな統合により、両方のプラットフォームをシームレスに接続できます。
デジタルサイネージソリューションを検討中の場合は、スタッフベースサポートまたはカスタマーサクセスマネージャーまでお問い合わせください。
詳細な設定手順については、ScreenCloud スタッフベースアプリガイドをご参照ください。
ScreenCloudでスタッフベースはどのように統合されますか？
ScreenCloudのスタッフベース統合により、ScreenCloudで管理されている画面にスタッフベースのニュースを表示できます。ScreenCloudでスタッフベースプラットフォームを接続した後、表示したいスタッフベースのニュースを選択します。ニュースのコンテンツはスタッフベースで設定し、画面のレイアウトや表示設定はScreenCloudで管理します。
スタッフベーススクリーンとScreenCloud、どちらを選ぶべきですか？
次のような場合は、スタッフベーススクリーンの利用をおすすめします。
完全にネイティブなスタッフベースソリューションを利用したい場合。
すべてを1つのプラットフォームで管理したい場合。
スタッフベースのニュース、動画、画像のみを表示したい場合。
次のような場合は、ScreenCloudが適しているかもしれません。
すでにScreenCloudをサイネージプロバイダーとして利用している場合。
高度なカスタマイズ、テンプレート、またはブランディングオプションが必要な場合。
ScreenCloudのエコシステムを通じて複数のコンテンツソースを組み合わせたい場合。
