# ミールプランプラグインへの食事の追加

**URL**: https://support.staffbase.com/hc/ja/articles/360019336000
**取得日**: 2026-05-02 18:50:09 JST

## 内容

エディター
ミールプランプラグインをインストールしたら、組織のためにミールプランを追加し始めることができます。ミールプランを使用して、従業員にその週および今後の週に地元のカフェテリアで提供されるメニューを知らせます。
前提条件
ミールプランプラグインをインストールしていること。
食事の追加
スタジオで、コンテンツ > ミールプランに移動します。
編集したいミールプランを選択するか、ミールプランを追加をクリックして新しいものを追加します。
ドロップダウンメニューから、食事を追加したい週を選択し、次に曜日のタブに移動します。
食事のタイトルを提供します。
オプションで、説明、価格、食事カテゴリなどの追加の食事詳細を提供します。
食事カテゴリを最大限に活用するためのいくつかのヒントは次のとおりです：
食事カテゴリを使用して、食事制限のあるユーザーにこの食事が適しているかどうかを迅速に知らせます。たとえば、食事がベジタリアンまたはグルテンフリーであるかどうかを示します。
カテゴリをより視覚的にするために絵文字を追加します。カテゴリフィールドを右クリックし、絵文字とシンボルを選択して絵文字を追加します。
保存をクリックします。
いつでも食事を編集または削除したり、ミールプランに追加の食事を追加したりできます。
ミールプランの公開
スタジオで、コンテンツ > ミールプランに移動します。
公開したいミールプランに移動します。
クリック ミールプランを公開。
