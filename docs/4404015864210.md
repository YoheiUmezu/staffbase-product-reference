# コメントの承認と復元

**URL**: https://support.staffbase.com/hc/ja/articles/4404015864210
**取得日**: 2026-05-02 18:52:07 JST

## 内容

エディター
コメントのモデレーションは、ユーザーが意見を自由に共有できる安全なプラットフォームを作るのに役立ちます。チャンネルのエディターは、モデレーションチームがより詳細にレビューできるように、潜在的に不適切なコメントを非表示にすることができます。コメントがプラットフォームのガイドラインに合致している場合、モデレーションチームはそれを承認し、復元することができます。コメントは再びチャンネル内で完全に表示されるようになります。
スタジオで、コンテンツ > システム > コメントに移動します。
コメントの概要で、承認して復元したいコメントに移動します。
承認をクリックします。
コメントを承認して復元するための確認ダイアログが開きます。
復元をクリックします。
コメントを承認し、復元しました。コメントはニュースチャンネル内で再びすべてのユーザーに表示され、著者は自分のコメントを再度編集および削除できます。
