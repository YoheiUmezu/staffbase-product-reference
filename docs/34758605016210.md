# Learningプラグインをインストールする

**URL**: https://support.staffbase.com/hc/ja/articles/34758605016210
**取得日**: 2026-05-02 18:45:25 JST

## 内容

アクティベーション
システム全体のエディター
Learningプラグインは、あなたの学習管理システム（LMS）をStaffbase従業員アプリまたはイントラネットに統合します。複数のプラットフォームからコースを集約し、1か所で表示するため、従業員が学習を簡単に発見、アクセス、追跡できるようになります。
Learningプラグインをインストールする
Studio内でコンテンツ > プラグインを追加するに移動します。マーケットプレイスページが表示されます。
Learningに移動し、インストールするをクリックします。
Learningを追加するダイアログが開きます。
Learningインスタンスのタイトルとスペースを入力します。
アプリ内での表示ドロップダウンから、Learningインスタンスを表示できるユーザーを選択します。
Add Learningをクリックします。最初のLearningインスタンスが追加されました。いつでも追加できます。Learningインスタンスを表示するには、編集して公開してください。
