# Apple Developer Programへの登録

**URL**: https://support.staffbase.com/hc/ja/articles/360002973351
**取得日**: 2026-05-02 18:54:32 JST

## 内容

始め方
あなたの組織はすでにApple Developer Programに登録されていますか？
はいの場合は、ここでの指示をスキップして、App Store ConnectアカウントへのStaffbaseの招待に進んでください。
あなたの組織にはApple IDがありますか？従業員アプリにそのApple IDを使用できますか？
はいの場合は、第二ステップに進んでください。
あなたの組織にはD-U-N-S番号がありますか？
はいの場合は、第二ステップをスキップして、Apple Developer Programへの登録を進めてください。
登録手順
Appleは、Apple Developer登録プロセス中に名前や法人名に以下をサポートしていません：
特殊文字: ラテンアルファベットのみを使用してください。特殊文字はサポートされていません。例えば、ハイフンなどです。
会社名にウムラウトやエスツェットが含まれている場合は、以下の基本文字に置き換える必要があります：
ä ⇒ a
ö ⇒ o
ü ⇒ u
ß ⇒ ss
100文字以上: 会社名が100文字を超える場合、法的な法人名を変更するために、ダン＆ブラッドストリートサポートに連絡してください。
AppleでApple IDを取得するために登録します。または、あなたの組織がすでにApple IDを持っているかどうかをAppleサポートページで確認してください。
Apple D-U-N-S登録ページでD-U-N-S®番号を登録します。D-U-N-S番号はダン＆ブラッドストリートによって管理されており、無料です。
Apple Developer Programに登録します。メンバーシップは$99 per yearです。
登録後数日以内に、Appleから電話がかかり、アプリの意図がAppleのガイドラインに沿っているか確認するためにいくつかの質問をされる可能性があります。以下は考えられる質問とその回答のアイデアです：
Q: 登録する組織の名前は何ですか？
A: 名前はD-U-N-S番号の登録に使用されたものと同じになります。
Q: 連絡先はどの部門で働いていますか？
A: Appleの連絡先であるあなたが働いている部門を述べてください。
Q: アプリの目的は何ですか？
A: Appleのガイドラインに従うために、アプリには公共の関連性が必要です。アプリが広範な公共の聴衆にとって価値のある情報を含むことを説明することで、公共の関連性を簡単に作成できます。例えば、潜在的な新しい従業員のための求人情報、パートナーや顧客のための組織ニュース、従業員スペースにサインインした後に利用可能な追加情報を提供することができます。
Apple IDでApp Store Connectにサインインして、すべてが正しく機能していることを確認します。
Apple Developer Programのメンバーシップを維持するために、以下を確認してください：
年次更新: アカウントが期限切れにならないように、毎年メンバーシップを更新してください。メンバーシップが期限切れになると、アプリはオフラインになります。
契約の受諾: Appleが定期的に更新する以下の契約を受け入れてください：
トレーダー契約。詳細を学ぶ。
Apple Developer Programライセンス契約。詳細を学ぶ。
有料アプリ契約。詳細を学ぶ。
追加の役立つ情報：
AppleがD-U-N-S番号について説明しています
