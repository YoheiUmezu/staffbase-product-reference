# サインアウト

**URL**: https://support.staffbase.com/hc/ja/articles/206478649
**取得日**: 2026-05-02 18:56:19 JST

## 内容

iOS
モバイルアプリで、右上のメニューを開きます。
歯車アイコンをクリックして設定を開きます。
アプリ設定の一番下で、サインアウトをクリックします。
モバイルアプリからサインアウトしました。
↻
Android
モバイルアプリで、左上のメニューを開きます。
設定をクリックします。
アプリ設定の一番下で、サインアウトをクリックします。
モバイルアプリからサインアウトしました。
↻
ウェブアプリ
ウェブアプリで、プロフィールメニューを開きます。
サインアウトをクリックします。
ウェブアプリからサインアウトしました。
↻
