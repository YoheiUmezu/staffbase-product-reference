# Microsoft 365 ウィジェットの概要

**URL**: https://support.staffbase.com/hc/ja/articles/25481210837394
**取得日**: 2026-05-02 18:42:29 JST

## 内容

アクティベーション
システム全体のエディター
Microsoft 365 ウィジェットは、他の Staffbase ウィジェットに似ています。これらは Microsoft プラットフォームからのコンテンツやサービスを、Pages や News 投稿でユーザーに提供します。コンパクトなレイアウトにより、Microsoft 365 ウィジェットは両プラットフォーム間の統合をシームレスな体験にします。
ユーザーが Microsoft プラットフォームのタスク、カレンダー、または概要を非 Microsoft コンテンツと同じページで表示できるようにします。エディターはテンプレートやカスタム CSS を使用する必要はありません。ウィジェットは数回のクリックで追加でき、Microsoft コンテンツを Staffbase プラットフォーム内にコンパクトで整然と表示します。
カレンダーウィジェット
Microsoft 365 カレンダーを News 投稿や Pages に追加するには、Microsoft 365 カレンダーウィジェットを使用します。ユーザーの個人用、共有、またはグループの Microsoft 365 カレンダーを提供し、ウィジェットからイベントの詳細を表示したり、オンライン会議に直接参加したりできます。
Microsoft 365 カレンダーウィジェットを News 投稿や Pages に統合する方法を学びましょう。
ファイルウィジェット
Pages や News 投稿に Microsoft ファイルを追加するには、Microsoft 365 ファイルウィジェットを使用します。さまざまな種類のファイルのドキュメントリストをページに追加できます。さらに、ウィジェットの検索機能により、ユーザーは探しているファイルを迅速に見つけることができます。ユーザーは、レイアウト機能を使用してファイルをリスト形式またはタイル形式で表示することを選択できます。
Microsoft 365 ファイルウィジェットを News 投稿や Pages に統合する方法を学びましょう。
ファイルビューアウィジェット
ページに単一のファイルを埋め込むには、Microsoft 365 ファイルビューアウィジェットを使用します。
たとえば、ユーザーがプレゼンテーションの各スライドを確認できる完全な PowerPoint プレゼンテーションや、SharePoint または OneDrive からの任意のファイルをウィジェット内に表示できます。
Microsoft 365 ファイルウィジェットを News 投稿やページに統合する方法を学びましょう。
タスクウィジェット
Microsoft Planner からユーザーのタスクをページやニュース投稿に追加するには、Microsoft 365 タスクウィジェットを使用します。このウィジェットは、良好な概要を提供するために 3 つのタブを表示します。
さらに、ウィジェットの検索およびフィルター機能により、ユーザーは探しているタスクを迅速に見つけることができます。
Microsoft 365 タスクウィジェットをニュース投稿やページに統合する方法を学びましょう。
Teams フィードウィジェット
Microsoft Teams フィードを Pages や News 投稿に追加するには、Microsoft 365 Teams フィードウィジェットを使用します。
このウィジェットは、ユーザーに次のタブを表示します：
フィード：ユーザーがアクセスできる Microsoft 365 Teams のすべてのチャンネル会話を更新リストの形式で表示します。
マイアクティビティ：@メンションやそのインタラクションなど、最近のアクティビティを表示します。
チャットメッセージ：Microsoft 365 Teams からの1対1およびグループチャットのすべてのチャットメッセージを表示します。
さらに、ウィジェットの検索およびフィルター機能により、ユーザーは探しているチームを迅速に見つけることができます。
Microsoft 365 Teams フィードウィジェットをニュース投稿やページに統合する方法を学びましょう。
Teams 概要ウィジェット
ユーザーが参加しているチームや、ユーザーが表示できるチームの概要を追加するには、Microsoft 365 Teams 概要ウィジェットを使用します。さらに、ウィジェットの検索機能により、ユーザーは探しているチームを迅速に見つけることができます。
Microsoft 365 Teams 概要ウィジェットを News 投稿や Pages に統合する方法を学びましょう。
サイトウィジェット
ユーザーが SharePoint で表示できるサイトの概要を追加するには、Microsoft 365 サイトウィジェットを使用します。さらに、ウィジェットの検索機能により、ユーザーは探しているサイトを迅速に見つけることができ、レイアウト機能により、ユーザーはサイトをリスト形式またはタイル形式で表示できます。
Microsoft 365 サイトウィジェットを News 投稿や Pages に統合する方法を学びましょう。
ドキュメントライブラリウィジェット
SharePoint ドキュメントライブラリまたはライブラリ内の特定のフォルダーを追加するには、Microsoft 365 ドキュメントライブラリウィジェットを使用します。このウィジェットは、ユーザーにパンくずリストを使用した簡単なナビゲーション、強力な検索機能を提供し、ファイルはアルファベット順または最終更新日でソートできます。
Microsoft 365 ドキュメントライブラリウィジェットをニュース投稿やページに統合する方法を学びましょう。
Viva コミュニティウィジェット
ベータ
Viva Engage コミュニティを Pages や News 投稿に追加するには、Microsoft 365 Viva コミュニティウィジェットを使用します。1 つ以上の Viva Engage コミュニティを宣伝し、コミュニティをカードとして表示できます。ユーザーがコミュニティカードをクリックすると、Viva Engage アプリ内のコミュニティスペースに移動します。
Microsoft 365 Viva コミュニティウィジェットを News 投稿や Pages に追加する方法を学びましょう。
Microsoft 365 ウィジェットをユーザーに利用可能にするには、Microsoft Connections を追加する必要があります。詳細はこちらをご覧ください。
