# Staffbase Emailのユーザーベース設定の概要

**URL**: https://support.staffbase.com/hc/ja/articles/24177352297746
**取得日**: 2026-05-02 18:42:49 JST

## 内容

Staffbase Emailをイントラネットまたは従業員アプリと一緒に使用する場合は、ユーザーベース設定の概要を参照してください。以下の情報は、Staffbase Emailがスタンドアロン製品として使用される場合にのみ適用されます。
Staffbase Emailを効果的に使用するには、まずユーザーベースを適切に設定する必要があります。あなたのStaffbase Emailユーザーベースは、2種類のユーザーで構成されています：
連絡先: メールを受信することしかできず、スタジオにアクセスできないユーザー
スタジオユーザー: スタジオにアクセスでき、役割に基づいてアクションを実行できるユーザー。たとえば、管理者として、あなたはスタジオユーザーです。
追加のユーザーを追加する前に、ユーザーベースを効果的に作成するために、以下のコンポーネントを同じ順序で設定してください：
プロファイルフィールド
これらは、ユーザー属性を含むフィールドで、個人を特定するために使用できます。ターゲット使用ケースに必要な追加のプロファイルフィールドを設定してください。
ユーザーの追加
プロファイルフィールドが設定された後、次のいずれかの方法を使用してユーザーを追加できます：
スタジオから
CSVインポート
SSOオンボーディング
Entra ID
Entra ID統合を使用すると、スタジオユーザーではない連絡先ユーザーを同期できます。
ユーザーグループ
これらは、ターゲットコミュニケーションと管理を可能にする共有属性を持つユーザーのコレクションです。プロファイルフィールドを定義し、ユーザーデータをインポートした後、次のいずれかの方法を使用して、これらのフィールドの情報に基づいて条件を設定することでユーザーグループを作成できます：
スタジオから
Entra ID
ユーザーベースを設定した後、必要に応じてプロファイルフィールドを更新できます。プロファイルフィールドの変更は、以降のユーザー同期に影響を与えることに注意してください。
