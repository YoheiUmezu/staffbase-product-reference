# 分析ダッシュボードを使用した検索メトリクスの分析

**URL**: https://support.staffbase.com/hc/ja/articles/20908732827666
**取得日**: 2026-05-02 19:01:25 JST

## 内容

スペース管理者
分析ダッシュボードは、ユーザーがStaffbaseプラットフォームとどのように、いつ、どこで関わっているかについての深い洞察を提供します。
この記事では、検索タブの詳細、グラフおよびその中の分析データについて説明します。 分析ダッシュボードの検索タブは、アプリとイントラネットのグローバル検索のメトリクスを表示します。
このタブは現在ベータ版であり、アクティベーションによってのみ利用可能です。詳細については、Staffbaseサポートまたはカスタマーサクセスマネージャーにお問い合わせください。
ユーザー役割のアクセスを含む、全体の分析ダッシュボードについて詳しくは、分析ダッシュボードの概要をご覧ください。

メトリクス
検索タブには、検索に関する2種類のメトリクスがあります。トップメトリクスと検索リストです。
トップメトリクス
ダッシュボードの上部には、迅速な概要のための3つのメトリクスがあります。
検索用語
全検索
検索ユーザー
このメトリクスは、ユーザーが指定された期間内に検索した異なる主題の数を表示します。
このメトリクスは、指定された期間内にユーザーが行った検索の数を表示します。
このメトリクスは、指定された期間内に検索を行ったユーザーの数を表示します。
メトリクスの下にあるラベルは、過去1か月間のメトリクスの増加または減少を示します。たとえば、先月の検索用語の数が100で、今月は50の場合、50%の減少が表示され、トレンドが下降していることを示します。
ラベルにカーソルを合わせると、先月と比較したメトリクスに関する追加情報が表示されます。
検索リスト
検索リストは、プラットフォームのグローバル検索における検索の広範な概要です。このリストには、検索用語、検索された回数、その用語の検索の増加または減少、用語を検索したユーザーの数、検索結果へのリンクが表示されます。検索の詳細については、検索の概要に関する記事をご覧ください。

検索リストのソート
スタジオで、分析 > 検索に移動します。
検索リストが表示されます。
リストをソートしたいヘッダーをクリックします。
そのヘッダーの下でリストが降順にソートされます。
オプションで、同じヘッダーを再度クリックしてリストを昇順に再ソートします。
特定のヘッダーで検索リストをソートしました。検索タブから移動すると、結果は自動的に検索語のアルファベット順にリセットされます。
