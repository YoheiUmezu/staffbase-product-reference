# 法的文書の再送信による確認

**URL**: https://support.staffbase.com/hc/ja/articles/360011304260
**取得日**: 2026-05-02 18:44:54 JST

## 内容

管理者
法的文書を更新した後、ユーザーに確認のために再送信することができます。
スタジオで、設定 > 法的文書に移動します。
確認設定をクリックします。
確認ダイアログの設定が開きます。
確認のために再送信のトグルを有効にします。
保存をクリックします。
すべてのユーザーに通知する確認メッセージが表示されます。
確認メッセージで、すべてに通知をクリックします。
ユーザーが次回アプリを開くと、法的文書を受け入れるためのダイアログが表示されます。受け入れた後でのみ、アプリを使用することができます。
