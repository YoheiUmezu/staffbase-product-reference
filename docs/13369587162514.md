# ジャーニーの複製

**URL**: https://support.staffbase.com/hc/ja/articles/13369587162514
**取得日**: 2026-05-02 18:58:04 JST

## 内容

アクティベーション
既存のジャーニーに似た新しいジャーニーを作成する際、複製することで多くの時間と労力を節約できます。複製することで、既存のジャーニーの内容を再利用できます。この方法で、自分自身のジャーニーテンプレートを作成することも可能です。
ジャーニーを複製する方法は以下の2つです：
ジャーニーを追加して複製する
スタジオで、コンテンツ > ジャーニーに移動します。
ジャーニーを追加をクリックします。
ジャーニーを追加ダイアログが開きます。
ジャーニーの名前を入力します。
ジャーニーを追加したいスペースを選択します。ジャーニーが複製されると、選択したスペースで開きます。
ジャーニーのステップを追加、更新、または削除するための協力者を選択します。
別のジャーニーからステップをコピーのチェックボックスを選択します。
ジャーニーを追加ダイアログが追加のフィールドで展開されます。
ジャーニーを選択をクリックします。
すべての既存のジャーニーが表示されるドロップダウンメニューが開きます。
複製したいジャーニーを選択します。
ジャーニーを追加をクリックします。
複製されたジャーニーが開きます。
すべてのステップを含むジャーニーを正常に複製しました。ジャーニーをアクティブにするには、開始点を設定する必要があります。
ジャーニーのステップをコピーして複製する
スタジオで、コンテンツ > ジャーニーに移動します。
複製したいジャーニーを選択します。
ステップをコピーをクリックします。
ジャーニーステップをコピーダイアログが開きます。
ジャーニーの名前を入力します。
ジャーニーを追加したいスペースを選択します。ジャーニーが複製されると、選択したスペースで開きます。
ジャーニーのステップを追加、更新、または削除するための協力者を選択します。
ステップをコピーをクリックします。
複製されたジャーニーが開きます。
すべてのステップを含むジャーニーを正常に複製しました。ジャーニーをアクティブにするには、開始点を設定する必要があります。
