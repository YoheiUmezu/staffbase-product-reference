# CSSによるStaffbaseプラットフォームのカスタムブランディング

**URL**: https://support.staffbase.com/hc/ja/articles/4410307430034
**取得日**: 2026-05-02 18:55:58 JST

## 内容

カスタムブランディングを使用すると、組織のニーズに基づいてStaffbaseプラットフォームの外観と感触をカスタマイズできます。スタジオでのブランディングの可能性に加えて、CSSカスタマイズを使用することで、プラットフォームの外観と感触をさらにカスタマイズし、企業のブランドアイデンティティに合わせることができます。
カスタムCSSは、正しく実装されていない場合、テーマを壊したり、特定の要素が正しく表示されない原因となる可能性があります。カスタムCSSは、経験がある場合のみ使用し、スタイルを適用した後にアプリの外観を必ず再確認してください。CSSスタイルの変更は慎重に扱う必要があり、ウェブ開発者やウェブデザイナーなどの上級ユーザーのみが行うべきです。
一般的に使用されるカスタマイズ
バッジ
CSSを使用して、ニュース投稿の「承認」および「重要」バッジをカスタマイズできます。
バッジのカスタマイズについて詳しくはこちら。
リンクタイルおよびリンクリストウィジェットのカスタムアイコン
HTMLを使用して、プラットフォームにカスタムアイコンを追加できます。
リンクタイルおよびリンクリストウィジェットのカスタムアイコンの追加について詳しくはこちら。
モバイルアプリ用のクイックリンクナビゲーションバー
CSSを使用して、モバイルアプリ用のクイックリンクナビゲーションバーをカスタマイズします。
モバイルデバイス用のクイックリンクナビゲーションバーのカスタマイズについて詳しくはこちら。
モバイルメニュー
CSSを使用して、ビジネスニーズに応じてモバイルサイドバーを変更します。
モバイルメニューのカスタマイズについて詳しくはこちら。
サインインダイアログ
サインインダイアログのスタイルを変更して、ユーザーにユニークな体験を提供します。サインインダイアログをカスタマイズするための以下のオプションのいずれかを使用します。
視覚的な要素の変更
テキストとレイアウトの変更
サインインボタンにヘルプテキストを追加する。
サインインダイアログのカスタマイズは、Staffbaseプラットフォームで使用されるサインイン方法に依存します。テキストベースの変更は、Staffbaseプラットフォームのオンボーディングプロセス中にのみStaffbaseテクニカルサポートチームが実施できます。プラットフォームの展開後の変更は、ネイティブアプリをアプリストアに再提出する必要があり、ユーザーがアプリを更新した後にのみ表示されます。

サインインボタンのヘルプテキスト
サインインダイアログのカスタマイズについて詳しくはこちら。
ウィジェット
カスタムCSSを使用して、プラットフォーム上のウィジェットのスタイルを設定し、ユニークな体験を作成します。
ウィジェットのカスタマイズについて詳しくはこちら。
