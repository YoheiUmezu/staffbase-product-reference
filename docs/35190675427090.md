# メディア投稿を作成する

**URL**: https://support.staffbase.com/hc/ja/articles/35190675427090
**取得日**: 2026-05-02 19:02:26 JST

## 内容

ニュースは、Staffbaseアプリとイントラネットで視聴者ごとに動的なコンテンツを公開することができます。コンテンツを効率的に整理し、ユーザーが簡単に利用できる形式で配信します。チャンネル内でニュース投稿記事を作成し、公開することができます。コンテンツ形式の外観は、チャンネルタイプによって異なります。この記事では、メディアチャンネルタイプ内で投稿を作成します。
Studioで、コンテンツ > ニュース > 投稿を追加したいメディアチャンネルに移動します。
投稿を作成するをクリックします。
メディアを作成するページが開きます。
ドロップダウンからオプションを選択して、投稿に画像または動画をアップロードします。
説明を入力します。
下書きを保存するをクリックします。
必要に応じて、プレビューをクリックして、投稿がユーザーにどのように表示されるかを確認します。
必要に応じて、別の言語でコンテンツを追加することもできます。
下書き投稿記事が作成されました。ユーザーに表示するには、まず公開する必要があります。
