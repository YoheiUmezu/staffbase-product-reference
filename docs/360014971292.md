# パスワードをリセットしようとしていますが、回復コードのメールが届きません

**URL**: https://support.staffbase.com/hc/ja/articles/360014971292
**取得日**: 2026-05-02 18:50:30 JST

## 内容

パスワードを忘れた場合は、リセットできます。以前にシステムにメールアドレスを入力している場合は、回復コードのメールを簡単にリクエストできます。
パスワードをリセットするためのメールが届かない場合は、以下を確認してください：
メールアドレスを正しく入力しましたか？
間違えた場合は、ログインフォームに戻って回復コードのメールを再度リクエストできます。
迷惑メールフォルダーを確認しましたか？
リンクを含むメールが迷惑メールとして分類されることがあります。
正しいメールアドレスを入力しましたか？
一部の従業員アプリでは、ユーザーにはログイン用のメールアドレスと異なる公開用のメールアドレスがあります。アカウントをリセットするには、ログイン用のメールアドレスを使用する必要があります。
アカウントに問題はありませんか？
アプリの管理者に連絡して、アカウントがまだアクティブであることを確認してください。アプリの管理者は、システム内からパスワードリセットのメールを送信することで支援できる場合もあります。
技術的な問題がありますか？
IT管理者に連絡して、会社の内部ファイアウォールがこの種のメールをブロックしているかどうかを確認してください。
