# アプリ内のスペースの表示順序の設定

**URL**: https://support.staffbase.com/hc/ja/articles/360010496219
**取得日**: 2026-05-02 18:59:08 JST

## 内容

スペースが表示される順序を選択できます。アプリでは、ユーザーはスタジオでスペースが配置された順序でスペースの内容を確認します。たとえば、全従業員スペースが最初にリストされ、その後にロンドン、ニューヨーク、営業などのスペースが続く場合、ロンドンに拠点を置く営業チームの従業員は、次の順序で内容を確認します：全従業員、ロンドン、営業。
スタジオで、システムメニューに移動します。
スペースをクリックします。
スペースの管理ページが開きます。
ハンドルを使用して、スペースを希望の位置にドラッグ＆ドロップします。
スペースの内容は、ソートした順序でアプリメニューに表示されます。
