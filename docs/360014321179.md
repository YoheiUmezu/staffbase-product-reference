# ウィジェットを使った魅力的なコンテンツの作成

**URL**: https://support.staffbase.com/hc/ja/articles/360014321179
**取得日**: 2026-05-02 18:52:18 JST

## 内容

ウィジェットを使用して、コンテンツを表示し、散在する情報をインタラクティブで魅力的な方法でユーザーに集中させることができます。
ウィジェットは以下の用途で利用可能です：
ページ
ニュース
ジャーニー
魅力的なコンテンツを簡単に作成
ウィジェットを使用すると、ソースコードに入ったり、HTMLの知識がなくても、エディターで簡単に魅力的なコンテンツを作成できます。創造的になりましょう！ウィジェットを自由に組み合わせて、静的および動的なコンテンツを表示できます。各ウィジェットのデザインと内容を、ユーザーの習慣やニーズに完璧に合わせて調整してください。ほとんどのウィジェットは、特定のデバイスの種類でのみ表示されるように設定することができ、すべてのデバイスで表示することも可能です。これにより、コンテンツは常に完璧なレイアウトになります。
さまざまなニーズに応えるウィジェットの配列があります：
コンテンツの構造を整える
情報が多いページでは、構造を作成することが重要です。ウィジェットを使用して、複雑なコンテンツ構造を簡単に整理し表示し、ユーザーが数秒で関連コンテンツを見つけられるようにします！たとえば、アコーディオンウィジェット、ヒーローイメージウィジェット、またはインフォボックスウィジェットを使用します。
明確でコンパクトな方法でリンクを提供
リンクは異なるコンテンツ間の架け橋を作ります。ウィジェットを使用して、内部および外部コンテンツへの便利なリンクを明確でコンパクトな方法で提供します。たとえば、ユーザープロファイルウィジェット、ファイルリストウィジェット、リンクリストウィジェット、リンクタイルウィジェット、またはボタンウィジェットを使用します。
コンテンツの舞台を整える
重要なニュースをニュースフィードとしてページに直接追加したり、フォームや食事プランなどのプラグインコンテンツをシームレスに埋め込んだり、Microsoftプラットフォームからのコンテンツやサービスを任意のページ、ジャーニー、またはニュース投稿に直接提供します。これにより、従業員は貴重なコンテンツを見逃すことがなくなります！たとえば、ニュースステージウィジェット、プラグインウィジェット、またはMicrosoft 365ウィジェットを使用します。
