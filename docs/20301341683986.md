# スマートインサイトダッシュボードによる可視性メトリクスの分析

**URL**: https://support.staffbase.com/hc/ja/articles/20301341683986
**取得日**: 2026-05-02 18:51:26 JST

## 内容

アクティベーション
システム全体のエディター
スマートインサイトダッシュボードは、各キャンペーンの詳細な統計を表示します。ダッシュボード内では、キャンペーンに割り当てられたニュース投稿の可視性、エンゲージメント、感情に関する詳細なメトリクスを確認できます。 詳細はこちら。
ダッシュボードには、分析ダッシュボードのキャンペーンタブからアクセスできます。スマートインサイトダッシュボードは、すべてのシステム全体のユーザーロールに対して表示されます。スマートインサイトダッシュボードは、スマートインパクト機能の一部です。詳細はこちら。
トップレベルメトリクスに関するより詳細な統計を提供するセクションがあります：
可視性
エンゲージメント
コメントの感情
アライメント
可視性は、キャンペーンコンテンツのリーチを示します。このメトリクスは、少なくとも1つのニュース投稿を閲覧したユーザーの割合を、キャンペーンコンテンツを閲覧できる潜在的なユーザー数と比較して計算されます。
時間経過に伴う可視性グラフ
時間経過に伴う可視性グラフは、キャンペーンコンテンツの可視性を一定期間にわたって示します。このチャートには、3つのデータ系列があります：
1つ以上の投稿を閲覧したユーザー：キャンペーン内で少なくとも1つの投稿を見たユーザーの割合、キャンペーンを見られる可能性のあるすべての視聴者の中で。
2つ以上の投稿を閲覧したユーザー：キャンペーン内で少なくとも2つの投稿を見たユーザーの割合、キャンペーンを見られる可能性のあるすべての視聴者の中で。
3つ以上の投稿を閲覧したユーザー：キャンペーン内で少なくとも3つの投稿を見たユーザーの割合、キャンペーンを見られる可能性のあるすべての視聴者の中で。
タイムライン内のデータポイントにカーソルを合わせると、フライアウトが開きます。フライアウトには以下が含まれます：
その時点での1つ以上、2つ以上、3つ以上の投稿を閲覧した正確な数値
その時点で公開された投稿のメトリクス
グラフ内のタイムラインは、キャンペーンの開始日と現在の日付に基づいています。開始日が設定されていない場合、キャンペーン作成日が代わりに使用されます。キャンペーンが終了すると、終了日がタイムラインの終了を示します。
グループごとの可視性テーブル
このテーブルは、個々のユーザーグループに基づく可視性を詳細に示します。これにより、どのユーザーグループがキャンペーンコンテンツを訪れているかを特定できます。グループにカーソルを合わせると、少なくとも1つのニュース投稿を訪れたユーザーの正確な数が表示されます。テーブルは、ドロップダウンメニューをクリックすることで、可視性率の高い順または低い順に並べ替えることができます。
