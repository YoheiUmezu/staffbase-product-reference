# Outlook Classic 3.0のナビゲーション

**URL**: https://support.staffbase.com/hc/ja/articles/4412592683666
**取得日**: 2026-05-02 18:44:20 JST

## 内容

この記事では、Employee Email 3.0アドインをOutlook Classicでナビゲートする方法について説明します。
メールデザイナードラフトの編集と送信
OutlookリボンでStaffbaseで作成をクリックして、メールデザイナードラフトのリストを開きます。
編集したいドラフトをクリックします。
Outlook Classicアドインの以前のバージョンと同様に、同じウィンドウ内でメールを編集して送信できます。ウェブアプリで利用可能なすべてのデザイン機能にアクセスできます。
ドラフトの編集が完了したら、送信をクリックします。
ドラフトを後で送信するようにスケジュールすることも可能ですが、Outlook Classicアドインを使用している場合は推奨されません。なぜなら、スケジュールされた時間に送信するには、デスクトップでOutlookを開いておく必要があるからです。
ドラフトが送信される予定の時間にOutlookが閉じている場合、次回Outlookが開かれたときに送信されます。
設定にアクセスする
Outlook Classicアドインの現在の設定メニューの概要や、ウェブアプリを通じてアクセスされる他の設定を確認してください。
アカウントの切り替え
いつでも、二人のアカウントを切り替えるアイコンをクリックするオプションがあります。これにより、新しいサインインウィンドウが開き、希望するEmployee Emailアカウントにサインインできます。
