# ユーザーデータをCSVファイルとしてエクスポートする

**URL**: https://support.staffbase.com/hc/ja/articles/24593477228178
**取得日**: 2026-05-02 18:57:58 JST

## 内容

ベータ
管理者
ユーザーベースデータをCSVファイルとしてダウンロードできます。エクスポートを開始すると、ファイルが準備でき次第、ダウンロードリンクを含むメールが届きます。
アカウントにプライマリメールアドレスが追加されている必要があります。Staffbaseプラットフォームの異なる種類のユーザーメールフィールドについて詳しく学んでください。
ユーザーデータエクスポートの開始
スタジオで、ユーザー ユーザーエクスポートに移動します。
ユーザーエクスポートページが開きます。
タイプの下で、次のいずれかを選択します：
すべてのユーザー：すべてのユーザーをエクスポートします。
選択したグループ：ドロップダウンから選択したグループをエクスポートします。
ユーザーステータスの下で、エクスポートしたいユーザーのステータスを選択します。
Staffbase Emailでは、連絡先という追加のユーザーステータスが利用可能です。
複数のステータスを選択できます。
ユーザーエクスポートを開始をクリックします。
ユーザーエクスポートプロセスを正常に開始しました。ダウンロードリンクを含むメールが届きます。
ユーザーデータのダウンロード
メールクライアントで、ユーザーエクスポートプロセスに関するメールを開きます。
ダウンロードリンクをクリックします。
スタジオが開きます。
ファイルをダウンロードをクリックします。
ファイルがコンピューターにダウンロードされます。
