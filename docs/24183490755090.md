# スタッフベース メールにおけるプロフィール フィールドの概要

## 内容

この記事はAIを使用して翻訳されており、不正確な情報が含まれている可能性があります。情報に不安がある場合は、正確性を確認するために英語の原文を参照してください。
StaffbaseメールをStaffbaseイントラネットまたは従業員アプリと一緒に使用する場合は、プロフィール フィールドの概要を参照してください。以下の情報は、Staffbaseメールをスタンドアロン製品として使用する場合にのみ適用されます。
プロフィールフィールドは、ユーザーに関する情報を保存します。これらのフィールドには通常、ユーザーの名前などの個人情報や、ユーザーを識別するのに役立つその他の詳細が含まれます。プロフィールフィールドは、メールをターゲットにしたユーザーグループを作成するのに役立ちます。
ユーザーをStaffbaseメールプラットフォームに追加する前に、組織に必要なプロフィールフィールドを設定する必要があります。
Staffbaseメールのデフォルトのプロフィールフィールド
Staffbaseメールには、次のデフォルトのプロフィールフィールドが含まれています:
部署
名
姓
所在地
電話番号
役職
プロフィールフィールドを削除したり、組織の要件に基づいて必要なだけ新しいフィールドを作成したりできます。
プロフィールフィールドを削除すると、それを条件として使用する既存のユーザーグループに影響を与える可能性があります。
