# HRウィジェットの利用可能設定

**URL**: https://support.staffbase.com/hc/ja/articles/27955352446994
**取得日**: 2026-05-02 18:55:01 JST

## 内容

有効化
管理者
組織の設定や統合された人事サービスに基づいて、ページに追加できるウィジェットを設定できます。データ同期が完了すると、デフォルトですべての関連機能が有効になり、編集者はすぐに対応するウィジェットへアクセスできます。管理者は、組織の設定に応じて機能の利用可否を簡単に変更できます。
前提条件
人事連携が設定され、同期されていること。
Studioで、設定 > HR連携に移動します。
連携を選択し、矢印をクリックして利用可能なウィジェットのリストを展開します。
ウィジェットをページで利用可能にするかどうかに応じて、トグルを調整します。
コンテンツエディターでHRウィジェットの利用可否を調整しました。
