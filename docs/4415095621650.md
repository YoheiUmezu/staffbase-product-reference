# 従業員メールアカウントプロファイルの変更

**URL**: https://support.staffbase.com/hc/ja/articles/4415095621650
**取得日**: 2026-05-02 18:49:50 JST

## 内容

あなたの 従業員メール アカウントでは、自分の表示名とメールアドレスを変更するオプションがあります。
アカウントに関連付けられたメールアドレスは、ウェブアプリにサインインするために使用されます。
このアドレスから送信したメールのみが、あなたのアカウントで追跡されます。
別のメールアドレスから送信されたメールを送信および追跡するには、そのメールアドレスに関連付けられた 従業員メール アカウントにアクセスできる必要があります。
Staffbase アプリ および/または イントラネット が 従業員メール と統合されている場合は、メインのStaffbaseプロファイルで名前とメールを編集してください。
表示名を変更する
設定ページで、アカウントをクリックします。
名および/または姓を編集し、保存して更新をクリックします。
「保存されました！」という通知が表示されます。
メールアドレスを変更する
設定ページで、アカウントをクリックします。
メールを変更をクリックします。
アカウントで使用したい新しいメールアドレスを入力します。
このプロセスを完了するには、新しいメールアカウントの受信トレイにすでにアクセスできる必要があります。
私はロボットではありませんというラベルの付いたボックスにチェックを入れ、メールを変更をクリックします。
確認メールが新しいメールアドレスに送信されます。
新しいメールアカウントの受信トレイで確認メールを確認します。
数分以内に受信トレイや迷惑メールフォルダーに見つからない場合は、Staffbaseサポートに連絡してください。
メールを開き、確認をクリックします。
確認ページがウェブブラウザーに表示されます。
ログインに戻るをクリックし、アカウントの新しいメールアドレスを使用してウェブアプリにサインインします。
パスワードは以前と同じままです。
この変更が完了すると、新しいメールアドレスから送信したメールのみがあなたの 従業員メール アカウントで追跡されます。
