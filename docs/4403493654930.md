# ゴーストライティングの概要

**URL**: https://support.staffbase.com/hc/ja/articles/4403493654930
**取得日**: 2026-05-02 19:03:13 JST

## 内容

アクティベーション
部門の責任者やCEOは従業員と多くのコミュニケーションを取る必要がありますが、コンテンツを作成してフォーマットする時間がほとんどありません。ゴーストライティングを利用することで、コンテンツチームはスタジオで重要な会社のニュースを簡単に準備し、特定のイニシアティブのリーダーの名義で投稿できます。たとえば、HR部門の責任者からのポリシー更新などです。
システム全体の編集者と管理者のみが投稿の著者を変更できますが、他の編集者は常に自分が誰の名義で書いているかを確認できます。投稿が公開されると、投稿の公的著者は「いいね」やコメントの通知を受け取り、個別の回答を行ったり、投稿に関する進行中の議論を把握したりできます。
ゴーストライティングを使用して、重要な会社のニュースを従業員により効率的に伝え、コンテンツ作成プロセスを合理化しましょう。
追加の有用な情報
投稿の著者を変更する
