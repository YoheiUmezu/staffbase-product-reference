# ジャーニーのパーソナライズ方法

**URL**: https://support.staffbase.com/hc/ja/articles/29518823014546
**取得日**: 2026-05-02 18:54:59 JST

## 内容

パーソナライズウィジェットを使って、ジャーニーに個人的な要素を追加できます。名や役職、オフィス所在地などのスマートフィールドを挿入すると、ユーザーのプロフィールから自動的にデータが取得されます。これにより、各ジャーニーステップが個人に合わせてカスタマイズされたように感じられます。
組織で設定されたすべての標準およびカスタムプロフィール フィールドが利用でき、パーソナライズのオプションはほぼ無限です。
Studioでコンテンツ > ジャーニー > パーソナライズしたいジャーニーステップに移動します。
編集するをクリックします。
ジャーニーステップのコンテンツ編集者が開きます。
編集者で、パーソナライズを追加したい位置にカーソルを置きます。
編集ツールバーで、ウィジェットを追加するのプラスアイコンをクリックします。
ウィジェットを追加するダイアログが開きます。
パーソナライズウィジェットを選択します。
パーソナライズ情報を追加するダイアログが開きます。
追加したいパーソナライズ情報を選択します。
OKをクリックします。
パーソナライズ情報が追加されます。
一度に選択できる情報は1つだけです。さらにパーソナライズ情報を追加するには、手順3から7を繰り返してください。
保存するをクリックします。
ジャーニーステップにパーソナライズを追加しました。ユーザーがジャーニーステップを表示すると、パーソナライズのプレースホルダーに該当するプロフィールのデータが表示されます。
