# 動画字幕の概要

**URL**: https://support.staffbase.com/hc/ja/articles/32707476248466
**取得日**: 2026-05-02 18:40:53 JST

## 内容

BETA
アクティベーション
AI
動画の字幕は、動画コンテンツをよりアクセスしやすく、理解しやすく、利用しやすくします。字幕は話されているコンテンツを画面上のテキストとして表示し、従業員がさまざまな環境、言語、アクセシビリティの状況で動画コンテンツを追いやすくなります。音声を再生できない場合や、従業員が読むことを好む場合、または複数の言語でコンテンツを共有する場合に、字幕は役立ちます。
字幕とキャプションという用語はしばしば同じ意味で使われます。一貫性のため、スタッフベースでは、動画に表示されるすべてのタイミング付きテキストを指す際に「字幕」という用語を製品およびドキュメント全体で使用しています。
動画字幕が役立つ場面
動画の字幕は、日常の職場のさまざまなシナリオをサポートします。例：
工場や店舗など、音声が聞き取りにくい騒がしい作業環境
音声を再生できない静かな環境や、ヘッドフォンが利用できない場合
従業員が母国語（ドイツ語、スペイン語、フランス語など）で読みたい英語のCEOメッセージ
聴覚障害のある従業員のサポート
認知や聴覚処理に違いがある場合など、読むことでコンテンツをより理解しやすい従業員のサポート
AIを活用した動画字幕により、スタッフベースは新しくアップロードされた動画に対して自動的に字幕と翻訳を生成します。これは手動の作業なしでバックグラウンドで行われます。完全にAIで生成された字幕は透明性のために明確にラベル付けされ、動画の長さや音声品質によって数分以内に利用可能になります。
AIで生成された字幕には不正確な部分が含まれる場合があります。これは、音声品質の低下、アクセント、方言、発音の誤り、バックグラウンドノイズや音楽などが原因となることがあります。正確性と一貫性を確保するため、特にブランド固有の用語、名前、技術用語については、公開前に必ず字幕を確認し、編集してください。
最新の対応アプリバージョン（最低v2025.3.463）で、すぐに動画字幕を利用開始できます。
字幕は公開ページでは利用できません。
動画字幕のメリット
時間とコストの削減
新しくアップロードされた動画には自動的に字幕が生成されるため、手動での書き起こし作業が不要になり、動画コミュニケーションの拡張が可能です。
多言語での理解促進
字幕は動画の元の言語で作成され、自動的に英語、ドイツ語、フランス語、スペイン語に翻訳されます。これにより、従業員は言語の壁を越えてコンテンツを理解できます。
アクセシビリティとインクルージョンの向上
字幕は聴覚障害のある従業員にも動画コンテンツを利用可能にし、環境や能力に関係なく包括的なコミュニケーションをサポートします。
柔軟で透明性のある字幕管理
コミュニケーターは字幕を確認、編集、差し替え、または特定の言語用にVTTファイルをアップロードできます。AIで生成された字幕は、字幕が確認されるまで明確にラベル付けされます。
ユーザーにとっての動画字幕の見え方
字幕は再生時にデフォルトで有効になっています。従業員は字幕のオン・オフや希望する言語の選択が可能です。言語の設定は、ブラウザやモバイルアプリなどデバイスごとに保存されます。字幕が完全にAIで生成され、まだ確認されていない場合は、「AI生成」ラベルが一時的に表示され、視聴者に通知されます。
