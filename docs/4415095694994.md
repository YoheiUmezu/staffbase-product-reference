# 最新の従業員メール Outlook 365 アドインへのアクセス

**URL**: https://support.staffbase.com/hc/ja/articles/4415095694994
**取得日**: 2026-05-02 18:55:45 JST

## 内容

製品名に関する注意: 2022年以降、BananatagはStaffbaseの従業員メール製品です。

最新の Outlook 365 アドイン用のエイリアスを追加する
組織の権限設定によっては、最新のアドインを承認するために Microsoft Entra  ID (以前の Azure AD) 管理者に連絡する必要がある場合があります。
このバージョンには、以下の追加スコープが必要です:
mail.readwrite.shared
mail.send.shared
2022年8月上旬、現在の従業員メール Outlook 365 アドインのサポートが終了します。
すべての従業員メール顧客は、できるだけ早く最新バージョンにアップグレードすることをお勧めします。
最新の Outlook 365 アドインへのアクセス方法
アドインのサイドバーを開くと、トグルが表示されます。
トグルをクリックします。サイドバーは自動的に閉じて、最新バージョンに更新されます。
サイドバーを再起動し、アカウントを認証します。
Bananatag Service APIによって要求された権限を受け入れます。
アップグレード中に承認が必要のダイアログが表示された場合、最も早く解決する方法は、Microsoft Entra ID 管理者にダイアログで資格情報を使用してサインインしてもらうことです。
通常通り送信します！
以前のアドインバージョンに戻す方法
アドインの送信またはドラフトサイドバーから、再度トグルをクリックします。
古いアドインに戻るために再認証を求められます。
トップに戻る
最新のアドインでのエイリアスによる送信の直感的な方法
トラッキングしたいメールを Outlook 365 から直接送信する場合:
以前の方法
アドインサイドバーのFromフィールドでメールアドレスを選択します。
新しい方法
メール作成画面のFromフィールドで直接メールアドレスを選択します。
トラッキングしたいEmail Designerのドラフトから送信する場合、プロセスは以前と同じです。
トップに戻る
最新のアドイン用のエイリアスを追加する
Email DesignerのドラフトのFromフィールドで、エイリアス選択のドロップダウンを開きます。
新しいエイリアスを追加をクリックします。
アカウントを追加をクリックします。
エイリアスとして使用したいメールアドレスを入力します。
このアドレスから送信するためのITまたは管理者の承認がある場合、保存に成功します。.
トップに戻る
