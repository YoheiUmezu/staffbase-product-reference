# メールデザインにおける高度な要素コントロールの使用

**URL**: https://support.staffbase.com/hc/ja/articles/4412584472850
**取得日**: 2026-05-02 18:51:56 JST

## 内容

Employee Emailでデザインを編集しているときは、強力な高度な要素コントロールを使用して、ブロック内の要素を分割、結合、入れ替えることができます。
この記事の内容
要素の分割、結合、入れ替えの方法
1つのブロック内で複数の要素を選択する方法
要素の分割、結合、入れ替え
要素の上にマウスを移動させると、要素の下に高度な要素コントロールメニューが表示されます。このメニューを使用すると、要素を分割、結合、入れ替えることができます。
要素の分割
分割機能を使用すると、1つの大きな要素を複数の小さな要素に分割できます。要素を分割すると、常に1つの要素が2つの要素に分割されます。
要素を分割するには、要素の上にマウスを移動させてからアイコンをクリックすると、要素が2つの新しい要素に分割されます。
要素の結合
結合機能を使用して要素を結合できます。これにより、任意の隣接要素と要素を結合し、より大きな要素を作成できます。
要素を結合するには、結合したい要素の上にマウスを移動させてから、左に結合または右に結合アイコンをクリックします。これにより、要素が結合されます。
結合される要素が異なる場合、結合ボタンをクリックしたときにマウスを移動させた要素が隣接要素を上書きします。
要素の入れ替え
場合によっては、ブロック内の異なる要素の順序を単に入れ替えたいことがあります。これを行うには、高度な要素コントロールメニューの入れ替え機能を使用します。
この機能を使用するには、入れ替えたい要素の上にマウスを移動させて、左に入れ替えまたは右に入れ替えアイコンをクリックします。いずれかのアイコンをクリックすると、現在の要素が隣接要素と入れ替わります。
1つのブロック内で複数の要素を選択する
1つのブロック内に複数の要素がある場合、複数の要素を選択して一括編集することが役立ちます。
これは、controlボタン（Macを使用している場合はcommand）を押しながら、選択したい各要素をクリックすることで行えます。
これが完了すると、画面の右側に複数要素メニューが表示されます。選択した要素の種類に応じて、編集のための異なるオプションが利用可能になります。
複数のテキスト要素または複数のボタン要素が選択された場合、次の変更が可能です：
要素間のパディング
フォント
テキストサイズ
行間
配置
それ以外の場合、利用可能なオプションは要素間のパディングを編集することだけになります。
