# 法的文書の作成

**URL**: https://support.staffbase.com/hc/ja/articles/115002233012
**取得日**: 2026-05-02 18:48:10 JST

## 内容

各法的文書は単一の文書として作成されます。たとえば、利用規約は1つの文書であり、プライバシーポリシーは別の文書です。法的文書を作成する方法は2つあります：
アプリ内の別ページとして作成する
外部ソースから文書へのリンクを作成する
法的文書を含むページの作成：
スタジオで、設定>法的文書に移動します。
ドロップダウンメニューから文書を作成を選択します。
タイトル、リンクアドレス、および内容を提供します。
リンクアドレスの最後の部分はショートネームとしても使用されます。ショートネームは、この法的文書へのリンクを確認ダイアログで使用できます。
保存をクリックします。
外部法的文書へのリンク作成
ブラウザでスタジオを開きます。
メニューで設定>法的文書をクリックします。
ドロップダウンメニューからリンクを作成を選択します。
タイトル、ショートネーム、およびリンクアドレスを提供します。
リンクアドレスはhttps://で始まる必要があります。
ショートネームは、タイトルに入力した内容から自動的に入力されます。このショートネームが適切でない場合は変更できます。ショートネームは、この法的文書へのリンクを確認ダイアログで使用できます。
保存をクリックします。
