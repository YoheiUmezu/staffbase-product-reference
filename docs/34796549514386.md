# メールのWebバージョンに関するよくある質問（FAQ）

**URL**: https://support.staffbase.com/hc/ja/articles/34796549514386
**取得日**: 2026-05-02 18:40:31 JST

## 内容

受信者はメールをPDFとして保存できますか？
はい。受信者がブラウザでメールを開くと、ブラウザの印刷機能を使って印刷したり、PDFとして保存したりできます。
Webバージョンでパーソナライズや動的コンテンツターゲットはサポートされていますか？
はい。パーソナライズと動的コンテンツターゲットの両方が、メールのブラウザバージョンで完全にサポートされています。
「ブラウザで開く」リンクは共有できますか？
はい。転送されたメールと同様に、リンクを共有できます。共有されたリンクは受信者のパーソナライズされたバージョンを反映するため、他の人がアクセスしても同じパーソナライズされたコンテンツが表示されます。Staffbaseでは、メールが内部利用のみを目的としている場合は、外部に共有しないよう従業員に注意喚起することを推奨しています。
Webバージョンでインライン動画の再生はサポートされていますか？
はい。インライン動画は完全にサポートされており、メールのブラウザバージョンで直接再生できます。
テストメールで「ブラウザで開く」リンクはサポートされていますか？
いいえ。「ブラウザで開く」リンクはテストメールではサポートされていません。
受信者はメールのWebバージョンでアンケートに参加できますか？
はい。受信者はブラウザバージョンで直接アンケートに参加でき、その回答は調査結果に含まれます。
Webバージョンでの受信者の動作はメールアナリティクスにどのように反映されますか？
ブラウザで開くリンクのクリックはクリックとして記録され、クリックマップに表示されます。
Webバージョンを開くとリピートオープンとしてカウントされ、トータルオープン率に反映されます。
Webバージョン内のボタンやリンクのクリックはリピートクリックとして記録され、トータルクリック数に追加されます。
閲覧時間はメールクライアントと同じ方法で計算されます。
デバイスタイプもメールクライアントと同じ方法で記録されます。
Webバージョンでインライン動画を再生すると、クリックとしてカウントされます。
