# ニュースに関するソーシャルインタラクションと通知チャネルの設定の概要

**URL**: https://support.staffbase.com/hc/ja/articles/33894088618386
**取得日**: 2026-05-02 19:00:56 JST

## 内容

ニュースは、スタッフベースアプリおよびイントラネットで視聴者ごとに動的なコンテンツを公開することができます。これは、ニュースページ、チャンネル、投稿の3層構造で構成されています。各チャンネル内では、編集責任者がすべての個別ニュース投稿記事のデフォルト設定をカスタマイズできます。
ソーシャルインタラクションや公開通知のデフォルトは、プラグインチャンネル設定で定義できます。これにより、チャンネル内で作成されるすべての投稿の基準が設定されます。変更は、保存後に作成された投稿にのみ適用されます。すでに公開された投稿には影響しません。デフォルトはチャンネル設定でのみ設定でき、個別の投稿内で上書きすることはできません。
設定のチャンネル機能では、ソーシャルインタラクションや通知設定のための複数のオプションがあります。これには以下が含まれます：
コメント
いいね／リアクション
承認
ハイライト
共有
プッシュ通知
メール通知
各インタラクションおよび公開通知設定の横には、許可されたとデフォルト設定の2つのオプションがあります。許可されたとして定義された設定は、投稿者以上のロールで個別のニュース投稿記事内で調整できます。下記の表は、2つの設定を組み合わせた場合の異なる結果を説明しています：
選択されたチャンネル設定
ニュース投稿記事の設定
例
設定が許可されていません。デフォルト設定はグレー表示されます。
設定はニュース投稿記事のオプション設定に表示されません。
コメントボックスはニュース投稿記事の下に表示されません。

設定はニュース投稿記事のオプション設定に表示されますが、自動的にはオンになりません。投稿者以上のロールでオンに切り替えることができます。
コメントは許可され、デフォルトで無効になっています。投稿者がオプションをオンにしない限り、コメントボックスはデフォルトで表示されません。
