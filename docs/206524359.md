# 言語の変更

**URL**: https://support.staffbase.com/hc/ja/articles/206524359
**取得日**: 2026-05-02 18:52:47 JST

## 内容

アプリ／イントラネットやメールの表示言語を変更できます。すべての投稿、ページ、メール、その他のコンテンツの表示言語を調整するには、個人設定でコンテンツ言語を直接変更してください。メニューやボタンの言語を変更するには、モバイルデバイスやブラウザからインターフェース言語を変更してください。
モバイルおよびウェブアプリでコンテンツ言語を変更する
プラットフォームは、ニュースやページを含むすべてのコンテンツを、アプリメニュー設定で表示されている言語で表示します。この設定を変更すると、表示されるコンテンツの言語が更新されます。
iOS
アプリメニューを開く。
歯車アイコンをタップして設定を開く。
設定内で言語をタップする。
言語設定が開く。
コンテンツ言語をタップする。
ドロップダウンメニューからアプリに表示したい言語を選択する。
モバイルアプリ内の 記事、ページ、その他のコンテンツの言語が変更されました。
↻
Android
ホーム画面からプロフィールメニューをタップする。
設定をタップする。
設定内で言語をタップする。
言語設定が開く。
コンテンツ言語をタップする。
ドロップダウンメニューからアプリに表示したい言語を選択する。
モバイルアプリ内の 記事、ページ、その他のコンテンツの言語が変更されました。
↻
Web App
プロフィールメニューを開き、設定をクリックする。
設定内で言語をクリックする。
言語設定が開く。
ドロップダウンメニューからアプリに表示したい言語を選択する。
ウェブアプリ内の 記事、ページ、その他のコンテンツの言語が変更されました。
↻
モバイルデバイスやブラウザからインターフェース言語を変更する
プラットフォームは、ボタンやメニューをモバイルデバイスやウェブブラウザで設定されている言語で表示します。モバイルデバイスやブラウザの言語を変更すると、これらの項目の表示言語が更新されます。
iOS および Android
デバイスの言語を確認する。
必要な言語にデバイスの言語を設定する。
必要に応じて、デバイスを再起動する。
モバイルデバイスのデバイス言語が変更されました。アプリは自動的に新しい言語に更新されます。
Firefox および Google Chrome
ブラウザの言語設定から直接言語を変更する。
予期しない結果を防ぐため、インストールされているすべてのブラウザをコンピュータの言語に設定することをおすすめします。
必要に応じて、ブラウザを再起動するか、ウェブページを更新する。
ブラウザの言語が変更されました。ウェブアプリは自動的に新しい言語に更新されます。
Microsoft Edge および Safari
コンピュータの言語を直接変更する。
ブラウザの言語設定は自動的に更新されます。
必要に応じて、ブラウザを再起動するか、ウェブページを更新する。
ブラウザの言語が変更されました。ウェブアプリは自動的に新しい言語に更新されます。
