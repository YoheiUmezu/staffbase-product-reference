# 投稿の削除

**URL**: https://support.staffbase.com/hc/ja/articles/360027933152
**取得日**: 2026-05-02 19:03:10 JST

## 内容

投稿は、公開した後でも削除できます。削除後、投稿はアプリのユーザーやスタジオの管理者には表示されなくなります。削除された投稿は、ゴミ箱機能を使用して削除から30日以内に復元できます。
スタジオで、削除したい投稿に移動します。
複数の投稿を一度に削除したい場合は、チェックボックスを選択してください。
削除をクリックします。
投稿を削除するための確認メッセージが表示されます。
削除をクリックします。
投稿が削除されます。コンテンツは削除から30日以内に復元できます。
