# ファイルマネージャーを使用した中央管理

**URL**: https://support.staffbase.com/hc/ja/articles/8987429409042
**取得日**: 2026-05-02 18:58:11 JST

## 内容

スタジオ全体でファイルやコレクションを一元管理できます。編集者がアップロードしたニュースやページのコンテンツに含まれるファイルや、スタジオユーザーが作成したコレクションを含む、任意のファイルを表示、置き換え、削除できます。ファイルをより良く管理するために、以下のオプションを使用してください：
アップロード: コンテンツ作成のためにデバイスからファイルを追加します
コレクション: ファイルの選択を作成し、以下の方法で管理します：
スタジオ内で表示または管理するための共有権限を設定する
コレクションにファイルを追加する
コレクションからファイルを削除する
コレクションを削除する
フィルター: ファイルタイプでファイルを絞り込みます
メタデータ: ファイルにカーソルを合わせると、簡単に理解できる形式で完全なメタデータが表示されます
サイズ: ファイルサイズを確認します
使用場所: ファイルが使用されている場所の数を確認します
タイムスタンプ: ファイルがアップロードされた日時を確認します
アップロード者: ファイルをアップロードしたユーザーの詳細を確認します
異なるビュー: 個人の好みに応じてグリッドまたはリストビューを使用します
ソート: 最新のファイルまたはアルファベット順にファイルをソートします
検索: 探している特定のファイルを見つけます
プレビュー: コンテンツを公開する前に画像や動画がどのように見えるかを確認します
ファイルを置き換える: リンクを変更せずに、古いまたは不正確なファイルを最新のファイルに置き換えます
ファイルを削除: 不適切またはもはや関連性のないファイルを削除します
ファイルの置き換え
スタジオ全体でユーザーがアップロードしたファイルにアクセスできます。新しいバージョンでファイルを更新する必要がある場合、すべてのユーザーがアクセスできる場所でファイルを置き換えることができます。ファイルが置き換えられても元のリンクは変更されず、混乱を防ぎます。 詳細を学ぶ。
ファイルの削除
ファイルを削除すると、ファイルマネージャー、プラットフォーム、および使用されているすべてのコンテンツから永久に削除されます。他の公開コンテンツで既に使用されているファイルを削除しないようにしてください。
スタジオで、ファイルに移動します。
ファイルマネージャーページが開きます。
オプションで、フィルターを使用して削除したいファイルを絞り込みます。
オプションで、ソートや異なるビューオプションを使用してファイルの全体像を把握します。
オプションで、プレビューを使用して画像や動画の詳細を確認します。
削除したいファイルを1つ以上選択します。
クリックして削除します。
ファイルを削除ダイアログが開きます。
ダイアログには、選択したファイルの数と、それが使用されている場所の数が表示されます。
クリックして削除します。
ファイルを正常に削除しました。
コレクションの削除
スタジオ全体でユーザーが作成したコレクションにアクセスできます。コレクションが冗長であるか、企業のガイドラインに違反している場合、中央のファイルマネージャーから永久に削除し、コレクションにアクセスしていたすべてのユーザーのファイルマネージャーからも削除できます。
コレクションを削除しても、コレクション内のファイルは削除されません。ファイルはすべてのファイルに引き続き利用可能です。
スタジオで、ファイルに移動します。
ファイルマネージャーページが開きます。
クリックしてコレクション。
削除したいコレクションを1つ以上選択します。
クリックしてコレクションを削除します。
コレクションを削除ダイアログが開きます。
ダイアログには、選択したコレクションの数と、それが使用されている場所の数が表示されます。
選択を確認した後、クリックして削除します。
コレクションを正常に削除しました。削除されたコレクションは、作成者やコレクションが共有されていたコラボレーターには表示されなくなります。
