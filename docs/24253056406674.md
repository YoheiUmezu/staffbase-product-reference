# Microsoft 365 Copilot 用の Staffbase エージェントの使用

**URL**: https://support.staffbase.com/hc/ja/articles/24253056406674
**取得日**: 2026-05-02 18:59:34 JST

## 内容

ベータ
アクティベーション
Microsoft 365 Copilot 用の Staffbase エージェントは、Microsoft Teams 内の Staffbase 拡張機能であり、Microsoft 365 Copilot が従業員アプリからの情報をソースリンクと共に応答できるようにします。リンクにカーソルを合わせると、情報ソースの詳細を表示するアダプティブカードが表示されます。また、要約、翻訳、または Microsoft 365 Copilot が提供するその他の機能を使用することもできます。
Microsoft 365 Copilot 用の Staffbase エージェントは、共有テナントデータを利用します。
前提条件
Microsoft 365 Copilot ライセンスを持っているか、メーター消費を使用しています。
管理者がMicrosoft 365 Copilot 用の Staffbase エージェントをインストールしています。
Microsoft 365 Copilot 用の Staffbase エージェントの選択
Microsoft Teams で、チャット > Copilot に移動します。
または、Copilot ピン留めアプリまたは Microsoft Teams ウェブインターフェイスに移動することもできます。
右側のパネルで、エージェントの下にある Microsoft 365 Copilot 用の Staffbase エージェント をクリックします。
エージェントの名前とアイコンは、組織が拡張機能をカスタマイズする方法によって異なる場合があります。
Microsoft 365 Copilot 用の Staffbase エージェントを選択しました。これで、従業員アプリからのターゲット応答を見つけることができます。
Microsoft 365 Copilot 用の Staffbase エージェントを使用して Copilot と対話する
Microsoft Teams で、チャット > Copilot に移動します。
または、Copilot ピン留めアプリまたは Microsoft Teams ウェブインターフェイスに移動することもできます。
右側のパネルで Microsoft 365 Copilot 用の Staffbase エージェントが選択されていることを確認してください。
質問やプロンプトを入力します。
従業員アプリにサインインするように求められる場合があります。
Microsoft 365 Copilot が応答します。
ソースリンクにカーソルを合わせて、Staffbase アダプティブカードを表示します。
Staffbase アダプティブカードで 開く をクリックして、ソース投稿にアクセスします。
Microsoft 365 Copilot が希望の情報を引き出さない場合は、プロンプトを調整または再構成して、Microsoft 365 Copilot を正しいソースに導くことを試みてください。
Microsoft 365 Copilot 用の Staffbase エージェントを使用して Microsoft 365 Copilot と対話しました。
