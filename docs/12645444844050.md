# コミュニケーションコントロールの共有ライブラリでの作業方法

**URL**: https://support.staffbase.com/hc/ja/articles/12645444844050
**取得日**: 2026-05-02 18:51:32 JST

## 内容

共有ライブラリには、あなたが共有したコンテンツや、あなたと共有されたコンテンツがすべて含まれています。
あなたと共有されたオブジェクトを追跡するために、共有ライブラリの各エリア（トピック、コンテンツ、アイデア、アセット）は、内部ユーザーと共有、外部ユーザーと共有、および私と共有のカテゴリに分かれています。
あなたと共有されたオブジェクトの編集オプション
あなたと共有されたトピック、コンテンツ、アイデア、アセットは、共有ライブラリで表示でき、あなた自身のチームにコピーできます。これを行う方法はこちらで確認できます。
共有オブジェクトを使用したくない場合は、ライブラリから削除できます。
あなたが共有したオブジェクトの編集オプション
他のチームと共有したトピック、コンテンツ、アイデア、アセットは、ここで表示でき、その共有オプションを編集できます。たとえば、後で日付や受取チームを変更したり、共有を停止したりできます。
