# 編集カレンダーの概要

**URL**: https://support.staffbase.com/hc/ja/articles/4407786275218
**取得日**: 2026-05-02 18:44:24 JST

## 内容

編集者
編集カレンダーは、編集者がアクセスできるスペースやチャネルにおける計画されたコンテンツを一目で確認できるようにします。これにより、編集者はコンテンツを視覚化し、特定の日に計画されているコンテンツが多すぎるかどうかを確認することが容易になります。その後、リーチを最大化するためにコンテンツを再配分できます。編集カレンダーは、編集者に圧倒されることなく包括的な概要を提供します。
編集カレンダーから直接、便利に以下の操作ができます：
ニュース投稿の追加、編集、スケジュール設定
ブロッカーの追加
ニュース投稿とブロッカーの詳細をフライアウトで表示
ニュース投稿のタイトルをクリックして開く
フライアウトでブロッカーの詳細を編集
スペース、チャネル、キャンペーンでニュース投稿をフィルタリング（組織で利用可能な場合）
クイック分析を表示
デフォルトでは、編集者はアクセスできるスペースやチャネルのニュースを表示および編集できます。ただし、オプションのフル透明モードを使用すると、すべてのスペースとチャネルのコンテンツがカレンダーに表示され、表示専用形式になります。フル透明モードの詳細を学ぶ。
