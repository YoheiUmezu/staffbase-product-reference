# アプリのアイコン

**URL**: https://support.staffbase.com/hc/ja/articles/360021949820
**取得日**: 2026-05-02 18:50:06 JST

## 内容

メニューアイコンは、ニュースページ、静的ページ、フォルダー、プラグインなど、すべてのメニュー項目の前に使用されます。さらに、モバイルナビゲーションバーにクイックリンク用のアイコンを追加することもできます。Staffbaseが提供するさまざまなアイコンの中から選択できます。
利用可能なStaffbaseアイコン
メニューを編集する際などに、アイコンピッカーからアイコンを選択します。
