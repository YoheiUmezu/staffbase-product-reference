# 従業員メールユーザーのパスワードをリセットする

**URL**: https://support.staffbase.com/hc/ja/articles/4415095625234
**取得日**: 2026-05-02 18:49:49 JST

## 内容

マネージャー
組織内の他の 従業員メール ユーザーのアカウントパスワードをリセットします。
管理者アカウントは、親管理者を除くすべてのユーザーのパスワードをリセットできます。
マネージャーアカウントは、自分のグループ内のマネージャーおよびベースユーザーのパスワードのみをリセットできます。
現在、Staffbase アプリ および/または イントラネット が 従業員メール と統合されている場合は、 スタジオ を使用してユーザーのパスワードリセットを支援します。
組織が 従業員メール との シングルサインオン (SSO) を使用している場合は、パスワードリセットの支援について中央サインインサービスの管理者に連絡してください。
手順
チーム ページで、ユーザーの横にある 三点リーダー をクリックし、パスワードをリセット をクリックします。
そのユーザーのために独自のパスワードを作成したい場合は カスタムパスワードを設定 をオンにし、自動生成されたパスワードを希望する場合はオフにします。
パスワードをリセット をクリックします。
ユーザーは新しいパスワードとさらなる指示を含むメールを受け取ります。
