# Android アダプティブアイコン

**URL**: https://support.staffbase.com/hc/ja/articles/360032772971
**取得日**: 2026-05-02 18:50:23 JST

## 内容

Android アプリのアイコンは、Android デバイスのホーム画面に表示されます。Android はこのアイコンをランチャーアイコンとも呼びます。アダプティブアイコンは、Android 8 以降のデバイスで使用されます。Android デバイスでは、各ランチャーがアプリアイコンを異なる方法で表示します。一部のランチャーはアイコンを四角い背景に配置し、他のランチャーはアイコンを丸い形に切り取ります。アダプティブアイコンは、Android 8 以降の特定の Android デバイス上で、すべてのアイコンがサイズと形状の観点でより一貫性を持つようにすることを目的として導入されました。
この新しいタイプのアイコンには、さまざまなアイコンタイプに対応するために常に表示される指定されたセーフゾーンが付いています。Staffbase はアダプティブアイコンをサポートしており、あなたのアプリアイコンはすべてのデバイスと Android バージョンで常に素晴らしく見えることを保証します。
要件
サイズ: 1024 x 1024 px
ファイル形式: png
解像度: 72 dpi
ファイル名: AppIcon_Android-adaptive.png
透明度: 許可されていません（アルファチャンネルを持ってはいけません）
Staffbase は、1 層のみのアダプティブアイコンをサポートしています。
角のある四角形
アプリアイコンに文字を使用する場合: 最大 6 - 10 文字（スペースを含む）
セーフゾーンに注意: ロゴをセーフゾーンの中央に配置し、エッジを空けておいてください。重要な要素はセーフゾーン内にのみ配置する必要があります。
セーフゾーンの外側の背景には、影、グラデーション、またはパターンを含めることができます。
提供するアダプティブアイコンのソースファイルは、後にデバイスに表示されるアプリアイコンとは異なります。提供するアダプティブアプリアイコンでは、角を埋めてください。
アダプティブアイコンの最終的な形状は制御できません。なぜなら、各ランチャーがそれぞれのデバイスで最終的な形状を作成するからです。
↻
Google Play™ 開発者ウェブサイトでは、新しいアプリアイコンの詳細な仕様とテンプレートを見つけることができます。
