# ヒーロー画像ウィジェットの設定

**URL**: https://support.staffbase.com/hc/ja/articles/25488219656850
**取得日**: 2026-05-02 18:56:37 JST

## 内容

価格
エディター
ヒーロー画像ウィジェットを使用して、大きな画像でページやジャーニーを視覚的にサポートします。ヒーロー画像ウィジェットは、ページを視覚的に魅力的にし、読者の重要なコンテンツへの注意を引きます。
ヒーロー画像ウィジェットの設定
ヒーロー画像ウィジェットを追加します。
ヒーロー画像ウィジェットを設定するためのダイアログが開きます。
オプションで、タイトルを提供します。
タイトルはウィジェットの上に表示され、ユーザーに見えるようになります。画像の内容を補完するタイトルを使用してください。
ウィジェットが表示されるデバイスタイプを、対応するアイコンをクリックして選択します。
ウィジェットが表示されるユーザーを、アプリ内の表示の下で選択します。
画像の下で、希望する画像をドラッグアンドドロップするか、フィールドをクリックして画像をアップロードします。
ヒーロー画像の理想的なサイズは1300 x 433ピクセルです。ただし、ヒーロー画像が追加されるセクションのサイズが100%未満の場合、画像は自動的に各側から一定の割合でトリミングされます。トリミングされる画像の領域は、セクションの幅と、ページカードモードが有効かどうかに基づいています。
モバイルデバイスで表示したい画像の領域にスライダーを配置します。
オプションで、カラーオーバーレイを有効にし、色と彩度のレベルを指定します。
OKをクリックします。
オプションで、エディター内の画像に移動し、画像にタイトルなどのテキストを追加します。
