# 個別のメールデザイナードラフトとテンプレートの共有

**URL**: https://support.staffbase.com/hc/ja/articles/4412584502546
**取得日**: 2026-05-02 18:49:53 JST

## 内容

製品名に関する注意: 2022年以降、BananatagはStaffbaseのEmployee Email製品です。
選択的に、個別のEmployee Emailデザイナードラフトまたはテンプレートをグループや特定のユーザーと共有して、時間を節約し、コンテンツの見た目と一貫性を向上させ、メールでのコラボレーションを行います。
また、共有ビューが有効になっている場合、グループのすべてのメンバーは、他のメンバーが送信した追跡メールにアクセスできるほか、他のメンバーのメールデザイナードラフトやテンプレートにもデフォルトでアクセスできます。
ドラフトの共有
Employee Emailウェブアプリで、メールデザイナードラフトのリストを開きます。
選択したドラフトの横にある三点アイコンをクリックし、次にドラフトを共有をクリックします。
ドラフトを共有したいユーザーまたはグループを見つけて、招待をクリックします。
表示されるドロップダウンメニューから、彼らの権限レベルを選択します。
完了をクリックします。
ドラフトの共有を解除するには、赤い点をクリックします。
テンプレートの共有
テンプレートはドラフトと同様に共有できますが、2つの重要な違いがあります:
個別のテンプレートは共有できず、フォルダーのみが共有可能です。
テンプレートはユーザーグループとのみ共有できます。
Employee Emailでのグループ設定について詳しくは、こちらをクリックしてください。
Employee Emailウェブアプリで、メールデザイナーテンプレートのリストを開きます。
選択したフォルダーの横にある三点アイコンをクリックし、次にフォルダーを共有をクリックします。
共有したいグループを見つけて、招待をクリックします。
表示されるドロップダウンメニューから、彼らの権限レベルを選択します。
完了をクリックします。
フォルダーの共有を解除するには、赤い点をクリックします。
アカウントの種類やグループの設定によっては、サブスクリプション内のすべてのグループ/ユーザーと共有できない場合があります。
