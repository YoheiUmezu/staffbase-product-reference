# 投稿の公開解除

**URL**: https://support.staffbase.com/hc/ja/articles/360027939932
**取得日**: 2026-05-02 19:02:44 JST

## 内容

すでに公開されている投稿の公開を解除することができます。投稿の公開を解除すると、ユーザーには表示されなくなります。公開解除を行うことで、投稿を編集し、後で再公開することが可能になります。
スタジオで、公開解除したい投稿に移動します。
編集をクリックします。
編集ページが開きます。
公開解除をクリックします。
投稿は公開解除されました。対象のオーディエンスには表示されなくなります。
