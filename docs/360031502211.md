# 通知センターとメニュー通知の違いは何ですか？

**URL**: https://support.staffbase.com/hc/ja/articles/360031502211
**取得日**: 2026-05-02 18:47:56 JST

## 内容

新しいコンテンツが利用可能になると、通知センターとアプリメニューに赤い通知バッジが表示されますが、これらの異なる場所で通知を受け取る方法と理由にはわずかな違いがあります。
通知センター
チャット通知を除く、プッシュまたはメール通知で受け取るすべての更新と通知は、通知センターに表示されます。ここでは、いつでも既読と未読の通知を手動で管理できます。通知センターで更新を受け取るのは、以下のケースです：
プッシュまたはメール通知で公開された投稿（アンケートや旅のステップを含む）
あなたが公開した投稿へのコメント
あなたのコメントへの直接の返信
メニュー通知
通知センターとは異なり、メニュー通知は新しい未読コンテンツが投稿されたニュースページ、フォルダー、またはチャットの上に直接表示されます。これらの通知は、新しいコンテンツが公開されたときに、プッシュまたはメール通知が送信されていなくても通知します。メニュー通知は、以下のケースでメニュー項目の上に表示されます：
メールまたはプッシュアラートで公開された投稿
アラートなしで公開された投稿
チャットメッセージ
