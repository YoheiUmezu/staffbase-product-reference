# スマートインサイトダッシュボードによるアライメントメトリクスの分析

## 内容

アクティベーション
ベータ
システム全体のエディター
スマートインサイトダッシュボードは、各キャンペーンの詳細な統計を表示します。ダッシュボード内では、キャンペーンに割り当てられたニュース投稿の可視性、エンゲージメント、感情に関する詳細なメトリクスを確認できます。 詳細はこちら。
ダッシュボードには、分析ダッシュボードのキャンペーンタブからアクセスできます。スマートインサイトダッシュボードは、すべてのシステム全体のユーザーロールに対して表示されます。スマートインサイトダッシュボードは、スマートインパクト機能の一部です。詳細はこちら。
より詳細な統計を提供するセクションがあります：
可視性
エンゲージメント
コメントの感情
アライメント
アライメントは、キャンペーンコンテンツが従業員にどのように響いているかを示します。このメトリクスは、キャンペーンに割り当てられたニュース投稿に対して提出されたアライメント調査からの直接的な従業員のフィードバックに基づいて計算されます。ニュース投稿にアクティブ化されると、調査はキャンペーンに割り当てられたニュース投稿の下部に表示されます。各従業員は、ニュース投稿ごとに1つの調査を提出できます。調査結果を機密に保つために、調査結果は最低5件のユーザー提出があった後にのみ表示されます。アライメント調査の詳細はこちら。
アライメントの時間経過に伴う平均グラフ
タイムラインは、特定の時点で収集されたアライメントデータの分布を示します。グラフには、調査フィードバックに基づく全体的なアライメントを示す単一の系列があります。調査結果を機密に保つために、調査結果は最低5件のユーザー提出があった後にのみ表示されます。
十分な回答がある場合、タイムラインのデータポイントにカーソルを合わせるとフライアウトが開きます。フライアウトには以下が含まれます：
平均アライメントメトリクス
評価の内訳（1から5）
特定のスコアを提出したユーザーの割合
ニュース投稿のタイトルと各投稿の参加者数
全体的なアライメント評価と投稿別評価テーブル
このテーブルは、関連するアライメント調査から収集されたスコアに基づくアライメントの詳細を示します。これにより、ユーザーがキャンペーンメッセージにどれだけ一致しているかを全体的に、または個々のニュース投稿ごとに特定できます。テーブルの上部にある表示ドロップダウンメニューを開くことで：
全体の平均または投稿に基づくアライメントを表示できます。
エンゲージメント率が最も高いまたは低いもの、古いまたは新しい投稿の日付でアライメントをソートできます。
テーブルの行にカーソルを合わせると、各投稿のアライメント平均とスコアの内訳が表示されます。
追加の有用な情報
