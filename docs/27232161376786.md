# メールのバージョンの表示と復元

**URL**: https://support.staffbase.com/hc/ja/articles/27232161376786
**取得日**: 2026-05-02 18:51:02 JST

## 内容

編集者
Staffbaseメールのバージョン管理機能は、下書きやテンプレートへのすべての変更を自動的に記録・保存します。ユーザーは、誰がいつ変更を行ったかを確認でき、コンテンツ作成プロセス全体で透明性と責任を確保できます。
Studioでメール > バージョン履歴を確認したいメールに移動します。
三点リーダーをクリックして、その他のオプションメニューを開きます。
バージョン履歴を表示を選択します。
バージョン履歴が現在のバージョンのプレビューの横に開きます。
バージョン履歴で、表示したいバージョンを選択します。
必要に応じて、バージョンを復元をクリックします。
バージョンを復元ダイアログが開きます。
復元するをクリックします。
復元されたメールのバージョンが編集モードで開きます。
メールのバージョンを確認し、復元しました。新しく復元されたバージョンは、バージョン履歴の現在のバージョンとして保存されます。
