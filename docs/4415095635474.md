# 他のメールユーザーとしてサインインおよび送信する

**URL**: https://support.staffbase.com/hc/ja/articles/4415095635474
**取得日**: 2026-05-02 18:55:51 JST

## 内容

マネージャー
新しい 従業員メールエイリアス送信 機能により、チームメンバーは追加のユーザーアカウントを必要とせずに共有エイリアスを使用してメールを送信できるようになりました。管理者はエイリアスの共有および送信オプションを作成および管理できます。別のユーザーとしてログインする代わりに 従業員メールエイリアス送信 を使用することをお勧めします。詳細については、従業員メールにおけるメールエイリアスの概要および従業員エイリアス送信への切り替えを参照してください。
組織内の他の 従業員メール ユーザーとしてサインインして、以下のことを行います：
現在のグループメンバーシップ、アカウントタイプ、および詳細な権限設定に基づいて、彼らが見るのと同じウェブアプリのビュー（下書き、テンプレート、メトリクスを含む）を確認します。
別の 従業員メール アカウント（たとえば、チームの共有メールアドレス用に作成されたアカウント）から追跡されたメールを送信し、そのアカウントのダッシュボードでメトリクスを受け取ります。
要件
送信元のメールアカウントには、関連付けられた 従業員メール アカウントが必要です。
送信元のメールアカウントとして送信する権限を持っている必要があります。
この権限は、組織のメール管理チームによって管理されています。
マネージャーとして、あなたは 従業員メール 内で同じグループに属する他のマネージャーまたはベースユーザーアカウントとしてサインインするオプションのみがあります。
手順
チーム ページで、サインインしたいユーザーの隣にある人物アイコンをクリックします。
そのアカウント内から、以下のことができます：
保存された下書きにアクセスする
彼らのテンプレートから新しい下書きを生成する
あなたの組織のメール管理チームによって送信する権限が付与されている場合、彼らのメールアドレスから下書きを送信する。
彼らのアカウントから以前に送信されたメールのメトリクスを表示する
メトリクスは常に送信元のアカウントに送信されることに注意してください。
他の人のアカウントを使用している間、リマインダーメッセージが画面に表示されます。
作業が完了したら、あなたのアカウントに戻るをクリックします。
この機能は慎重に使用してください。サインイン中に行ったすべての操作は、使用している人のアカウントに関連付けられます。その時点で誰がアカウントを使用しているかは明らかではない場合があります。したがって、この機能は注意して使用し、なりすましに関連する潜在的なセキュリティの影響を考慮してください。
従業員エイリアス送信への切り替え
既存のユーザー向け
従業員エイリアス送信を使用することに決定した場合、Staffbaseサポートまたはカスタマーサクセスマネージャーに連絡して、他のユーザーとしてのサインインを無効にしてください。
他のユーザーとしてのサインインが以前に使用されていて現在無効になっている場合、以下の方法でメールにアクセスできます：
ユーザーが機能の無効化前にメールの下書きを共有していた場合、アクセス権を持つ人は通常通り使用できます。
メールを表示するには、共有ビューのメールが有効になっていることを確認し、メールを送信したユーザーと同じグループに属しているか、管理者権限を持っている必要があります。
上記の条件が満たされない場合、その特定のユーザーとしてログインする以外にアカウントにアクセスする方法はありません。
新しいユーザー向け
新しいユーザーにとって、他のユーザーとしてのサインインはもはや利用できません。
代わりに従業員メールエイリアス送信を使用してください。詳細については、従業員メールにおけるメールエイリアスの概要を参照してください。
