# 事前フィルタリングされたオープングループ選択ツールの設定

**URL**: https://support.staffbase.com/hc/ja/articles/360011495720
**取得日**: 2026-05-02 19:01:41 JST

## 内容

ベータ
管理者
ユーザーが自由に参加および退会できるオープングループの大規模な選択肢は素晴らしいことです。しかし、興味グループから部門の更新、オフィスの食堂オプションまで、オープングループを事前にフィルタリングする方法がないと、ユーザーはすべてのオプションを一度に整理しようとして圧倒されることがあります。
そのため、アプリ内で事前フィルタリングされたグループ選択ツールへのリンクを提供する方法を今ご用意しています。これにより、特定のタイプのすべてのグループオプション（たとえば、オフィスの場所で利用可能なすべての興味グループやすべての会社の食堂オプション）を自動的に表示するリンクを作成できます。ユーザーがこれらのグループのいずれかに参加すると、アプリメニュー内でそのグループに関連するコンテンツを見つけることができます。
前提条件
オープングループを作成しています。
オープングループに適切な可視性設定でコンテンツを作成しています。
リンクプラグインをインストールしています。
オープングループコンテンツ用のフォルダーまたはニュースページを設定する
グループ選択ツールに表示したいコンテンツの種類に応じて、グループコンテンツが存在するフォルダーまたはニュースページを作成します：
ニュース関連のオープングループに事前フィルタリングされたグループ選択ツールを作成したい場合は、ニュースページを作成します。
プラグイン、ページ、ニュースなど、異なる種類のコンテンツに事前フィルタリングされたグループ選択ツールを作成したい場合は、フォルダーを作成します。
この例では、フォルダーを作成します。
スタジオで、メニューに移動します。
フォルダーを作成をクリックします。
新しいフォルダーを作成するためのダイアログが開きます。
フォルダーに名前を付け、フォルダーの説明を提供します。
作成をクリックします。
新しいフォルダーがメニューに表示されます。オプションで、メニュー内の位置を調整できます。
作成したフォルダーに必要なグループコンテンツを移動します。
この例では、すべての食事プランプラグインが食堂フォルダーに移動されます。
保存をクリックします。
コンテンツのIDを見つける
グループ選択ツールを事前フィルタリングするには、フィルタリングしたいオープングループコンテンツを含むフォルダーまたはニュースチャンネルのIDを見つける必要があります。この例では、ニュースページではなくフォルダーを使用します。
アプリメニューで、マイグループをクリックします。
事前フィルタリングされたグループ選択ツールに表示したいグループのいずれかに参加します。
これにより、アプリメニューでフォルダーを見ることができます。
フォルダーを開きます。
URLの最後にあるフォルダーIDをコピーします。
フォルダーIDは、最後のスラッシュ記号の後の全コードです。
フォルダーIDを保存し、再度アクセスできる場所に保管します。事前フィルタリングされたグループ選択ツールへのリンクを設定するためにIDが必要です。
事前フィルタリングされたグループ選択ツールへのリンクを作成する
スタジオで、リンクプラグインに移動します。
リンクを追加をクリックします。
新しいリンクを作成するためのダイアログが開きます。
リンクのタイトルを追加します。
ターゲットURLの下で、事前フィルタリングされたグループ選択ツールへのURLを3つのステップで構築します：
コンポーネント:
あなたのドメイン
メニューパス
(このコンポーネントをコピーして貼り付けます)
フォルダーまたはニュースページID
例:
https://showcase.staffbase.com
/settings/groups?contexttype=menufolder&contextid=
7r1847a4619979aa55a2df43
これら3つのコンポーネントがURLを構成します。次のようになります：
http://showcase.staffbase/settings/groups?contexttype=menufolder&contextid=7r1847a4619979aa5
リンクを追加をクリックします。
リンクを公開をクリックします。
リンクはアプリに表示されるようになります。スタジオのメニュー設定でリンクの配置をいつでも調整できます。
ユーザーがアプリで事前フィルタリングされたグループ選択ツールを表示する方法
