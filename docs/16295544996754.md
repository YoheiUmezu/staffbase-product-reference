# スタジオでのSMSの設定

**URL**: https://support.staffbase.com/hc/ja/articles/16295544996754
**取得日**: 2026-05-02 18:55:33 JST

## 内容

ベータ
アクティベーション
管理者
Staffbase SMSは、SMSを通じて従業員にテキストメッセージを送信することでコミュニケーションを強化します。Twilioのメッセージングプラットフォームと統合されており、信頼性が高く、高い開封率と簡単なアクセスを保証します。
設定の詳細
Twilioコンソールからアカウント設定を取得します。前提条件を参照してください。
受信者の電話番号は、スタジオのプロファイルフィールドから取得されます。
プロファイルフィールドにアクティブな電話番号がある場合、メッセージは以下に送信されます：
アクティブユーザー
保留中のユーザー
無効化されたユーザー
メッセージは以下には送信されません：
削除されたユーザー
前提条件
送信者の電話番号を持つ有料のTwilioアカウントを設定しています。この電話番号からスタジオでメッセージが送信されます。 詳細を学ぶ
アメリカ合衆国の受信者にSMSを送信するために、A2P登録を完了しています。
Staffbase StudioでSMSを設定するために、Twilioコンソールダッシュボードから以下の情報をコピーしました：
アカウントSID
認証トークン
私のTwilio電話番号
手順
スタジオで、SMS > SMS設定に移動します。
SMS設定ページが開きます。
アカウントSIDの下に、TwilioアカウントからアカウントSIDを貼り付けます。
認証トークンの下に、Twilioアカウントから認証トークンを貼り付けます。
送信者番号の下に、Twilioアカウントから私のTwilio番号を貼り付けます。
受信者電話番号の種類の下に、受信者のプロファイルに基づいて好みの番号の種類を選択します。例えば、勤務番号です。
保存をクリックします。
これで、Staffbase StudioからSMSを送信できるようになりました。 詳細を学ぶ
