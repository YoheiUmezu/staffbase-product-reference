# アコーディオンウィジェットの設定

**URL**: https://support.staffbase.com/hc/ja/articles/360020097859
**取得日**: 2026-05-02 18:59:03 JST

## 内容

価格
エディター
アコーディオンウィジェットを使用すると、任意のページ、ジャーニーステップ、またはニュース投稿にアコーディオンメニューを追加できます。ユーザーはすべてのパネルを一目で確認でき、アコーディオンパネルをクリックすることで対応するコンテンツを表示できます。アコーディオンは、コンテンツが多いページを整理するのに特に便利です。
アコーディオンウィジェットの設定
アコーディオンウィジェットを追加します。
アコーディオンウィジェットを設定するためのダイアログが開きます。
オプションで、タイトルを提供します。
タイトルはウィジェットの上に表示され、ユーザーに見えるようになります。アコーディオンの内容を補完するタイトルを使用してください。この例で使用されているタイトルは「FAQ」です:
ウィジェットが表示されるデバイスタイプを、対応するアイコンをクリックして選択します。
アプリ内の表示の下で、ウィジェットが表示されるユーザーを選択します。
アコーディオンの各パネルの見出しを追加します。
すべてのパネルを追加したら、OKをクリックします。いつでも追加のパネルを追加したり、パネルを調整したりできます。
アコーディオンウィジェットの設定が完了しました。これでアコーディオンにコンテンツを追加し、エディターでアコーディオンに移動してセクションの下に希望のコンテンツを入力できます。
