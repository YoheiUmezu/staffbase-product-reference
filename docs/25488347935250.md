# ニュース投稿およびページにおけるオーディオファイルの設定

**URL**: https://support.staffbase.com/hc/ja/articles/25488347935250
**取得日**: 2026-05-02 19:03:02 JST

## 内容

ユーザーがページやニュース記事内でポッドキャスト、音楽、または任意のオーディオファイルを直接聴取できるようにします。マイファイル、直接アップロード、またはドラッグアンドドロップを使用して、希望するコンテンツにオーディオファイルを含め、ユーザーが従業員アプリやフロントドアイントラネットでコンテンツを聴取できるようにします。その後、オーディオファイルを記事やページ内でいつでもドラッグアンドドロップして再配置できます。
Staffbaseは複数のオーディオファイル形式をサポートしています。詳細を学ぶ。
オーディオファイルはフロントドアイントラネットと従業員アプリの両方で使用できます。イントラネットでは、従業員がオーディオを聴取およびダウンロードできます。アプリでは、ユーザーがモバイルポッドキャストプレーヤーを使用して、柔軟に移動中に聴取できます。詳細を学ぶ。

オーディオファイルの追加
ニュース記事やページにオーディオファイルを簡単に追加できます。ファイルは常にカーソルが置かれているテキストボックスにアップロードされます。オーディオファイルのタイトルが正確であることを確認してください。これはコンテンツおよびモバイルポッドキャストプレーヤーに表示されます。以下のユーザーにとってのオーディオプレーヤーの見た目を参照してください。
ドラッグアンドドロップ
ニュース記事またはページで、コンテンツエディタに移動します。
オーディオファイルをテキストボックスにドラッグアンドドロップします。
保存をクリックします。
直接アップロード
ニュース記事またはページで、コンテンツエディタに移動します。
メディアアイコンをクリックしてメディアドロップダウンを開きます。
コンピュータからアップロードを選択します。
デバイスのアップロードダイアログが開きます。
オーディオファイルを選択してアップロードします。
保存をクリックします。
マイファイル
ニュース記事またはページで、コンテンツエディタに移動します。
メディアアイコンをクリックしてメディアドロップダウンを開きます。
マイファイルを選択します。
マイファイルダイアログが開きます。
アップロードしたいオーディオファイルを選択します。
ファイルを追加をクリックします。
保存をクリックします。
ニュース記事またはページにオーディオファイルをアップロードしました。ファイルはいつでも削除したり、ドラッグアンドドロップで再配置したりできます。記事やページがすでに公開されている場合、更新は自動的に表示されます。
オーディオファイルの再配置
オーディオファイルをニュース記事やページに追加した後、ファイルをドラッグアンドドロップしてコンテンツ内の位置を変更できます。
オーディオファイルを移動したいニュース記事またはページに移動します。
オーディオファイルを選択して保持し、コンテンツ内の配置したい場所に移動します。
保存をクリックします。
コンテンツ内のオーディオファイルの位置を更新しました。記事やページがすでに公開されている場合、更新は自動的に表示されます。
オーディオファイルのメタデータの編集
オーディオファイルのタイトルとURLを編集できます。
オーディオファイルを編集したいニュース記事またはページに移動します。
オーディオファイルをクリックします。
編集および削除アイコンが表示されます。
鉛筆アイコンをクリックして編集します。
添付ファイルの追加/編集ダイアログが開きます。
URLまたはタイトルを調整します。
追加をクリックします。
保存をクリックします。
コンテンツ内のオーディオファイルを編集しました。更新は即座に反映されます。いつでもファイルを再編集できます。
オーディオファイルの削除
もはや必要ない場合、コンテンツからオーディオファイルを簡単に削除できます。
削除したいオーディオファイルを追加したニュース記事またはページに移動します。
オーディオファイルをクリックします。
編集および削除アイコンが表示されます。
ゴミ箱アイコンを選択します。
オーディオファイルが削除されます。
保存をクリックします。
コンテンツからオーディオファイルを削除しました。いつでもオーディオファイルを再追加できます。記事やページがすでに公開されている場合、更新は即座に表示されます。
