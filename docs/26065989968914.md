# アイデア提案ページの設定

**URL**: https://support.staffbase.com/hc/ja/articles/26065989968914
**取得日**: 2026-05-02 19:01:51 JST

## 内容

システム全体の編集者
この機能はアドオンです。詳細はStaffbaseの価格をご参照ください。有効化についてはStaffbaseサポートまたはカスタマーサクセスマネージャーにお問い合わせください。
アイディエーションプラグインは、Staffbase従業員エクスペリエンス（EX）ブースタープラグインです。従業員がアイデアを共有し、コメントし、最も重要なことに投票できるダイナミックなツールです。アイディエーションプラグインをインストールした後、さらに設定を行い、最初のアイディエーションページを公開してユーザーに表示できるようにします。
アイディエーションページを設定する
Studioで、コンテンツ > アイディエーションに移動します。
設定したいアイディエーションページに移動します。
ページ下部のマニュアルをクリックして設定ガイドを開きます。
マニュアルを使って、ページをニーズに合わせて設定します。
保存するをクリックします。
アイディエーションページの設定が完了しました。ページを公開してユーザーに表示できるようになります。
アイディエーションページを公開する
Studioで、コンテンツ > アイディエーションに移動します。
公開したいアイディエーションページに移動します。
アイディエーションを公開するをクリックします。
アイディエーションページが公開されました。ユーザーに表示されます。
