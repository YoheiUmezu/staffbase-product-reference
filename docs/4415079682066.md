# Outlook または Gmail の配信リストへの送信

**URL**: https://support.staffbase.com/hc/ja/articles/4415079682066
**取得日**: 2026-05-02 18:49:42 JST

## 内容

Employee Email アドインを使用して Outlook または Gmail からメールを送信する際、配信リスト、連絡先グループ、または個別のアドレスのグループへの送信プロセスは同じです。ただし、いくつかの追跡制限に注意する必要があります。
要件
宛先 または CC フィールドに少なくとも 1 つのアドレスが入力されていること。
宛先 と CC フィールドの間に 25 件を超えるアドレスがないこと。
BCC フィールドのアドレス数に制限はありません。
配信リストまたは連絡先グループがすでに設定されていること。
Outlook Classic
Outlook で下書きメッセージを開き、宛先 または CC フィールドに配信リストまたは連絡先グループを入力します。
すべての受信者を BCC フィールドに入れることを希望する場合でも、少なくとも 1 つのメールアドレス（例えば、送信元のアドレスのコピー）を 宛先 または CC フィールドに入力する必要があります。
受信者カウント が Outlook Classic アドインの設定で有効になっている場合、すべてのフィールドのメールアドレスをカウントします。メールネットワーク、リスト構成、コンピュータの速度によっては、送信が遅れることがあります。
Gmail
Outlook で下書きメッセージを開き、宛先 または CC フィールドに配信リストまたは連絡先グループを入力します。
すべての受信者を BCC フィールドに入れることを希望する場合でも、少なくとも 1 つのメールアドレス（例えば、送信元のアドレスのコピー）を 宛先 または CC フィールドに入力する必要があります。
