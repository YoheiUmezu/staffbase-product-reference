# プラグインウィジェットの設定

**URL**: https://support.staffbase.com/hc/ja/articles/25488300689170
**取得日**: 2026-05-02 18:53:21 JST

## 内容

エディター
プラグインウィジェットを使用すると、フォームや地図などのプラグインコンテンツをページやニュース記事に直接埋め込むことができます。プラグインウィジェットは、以下を含むほとんどの標準Staffbaseプラグインをサポートしています：
フォームプラグイン
調査プラグイン
イベント登録プラグイン
食事プランプラグイン
カレンダープラグイン
地図プラグイン
その他にも多数あります！プラグインウィジェットでは、ユーザーのインタラクションを有効にすることもできます。つまり、ユーザーはページやニュース記事から離れることなく、フォームを送信したり、今週のカンティーンのオプションをスクロールしたりできます。
プラグインウィジェットの設定
プラグインウィジェットを追加します。
プラグインウィジェットを設定するためのダイアログが開きます。
オプションで、タイトルを提供します。
タイトルはウィジェットの上に表示され、ユーザーに見えるようになります。
ウィジェットが表示されるデバイスタイプを、対応するアイコンをクリックして選択します。
ウィジェットを表示したいユーザーを選択し、アプリ内の表示の下で設定します。
ドロップダウンメニューから埋め込みたいプラグインを選択します。
オプションで、デスクトップおよびモバイルデバイス用にウィジェットの高さをカスタマイズします。
インタラクションを有効にするトグルをオンにして、ウィジェットでのズームやクリックなどのユーザーインタラクションを有効にします。
スクロールを有効にするトグルをオンにして、ウィジェット内でのスクロールを有効にします。
OKをクリックします。
