# ショートの概要

**URL**: https://support.staffbase.com/hc/ja/articles/30379912327954
**取得日**: 2026-05-02 19:02:50 JST

## 内容

編集者
Shortsは、メディアニュースチャンネルの投稿をもとにした動画コンテンツを表示する、エンゲージメントを高める方法です。プラットフォームに親しみやすいソーシャルメディア風の体験をもたらします。ユーザーと短尺の魅力的なコンテンツをつなげ、リーダーシップの最新情報、学習コンテンツ、チームの表彰など、さまざまな用途に活用できます。
メディアチャンネルはShortsの基盤です。表示、編集者、権限の設定は、各チャンネルの設定で簡単にカスタマイズできます。ページ用Shortsウィジェットは次の機能を提供します：
さらなる表示とコンテンツターゲティングを実現
動画を横並びかつ時系列で一覧表示
ユーザーが概要をすばやく確認し、Shortsをクリックして開くことが可能
動画を全画面で表示
Shortsのスクロールやクイックな操作が可能
たとえばホームページでエンゲージメントを簡単に高めることが可能
Shortsを作成するのは簡単で、従業員アプリやイントラネットから直接作成できます。アプリでは、投稿者が縦型動画を録画またはアップロードし、プレビュー、キャプション追加、公開まで従業員アプリ内で完結します。イントラネットでは、メディアニュースチャンネルから直接Shortsを作成・公開でき、動画は全画面で表示されます。Shortsはコミュニケーションを迅速にし、表彰を簡単にし、企業ストーリーをより魅力的にします。
