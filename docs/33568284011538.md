# ニュース記事のハイライトと謝辞の概要

**URL**: https://support.staffbase.com/hc/ja/articles/33568284011538
**取得日**: 2026-05-02 19:02:28 JST

## 内容

ニュースは、スタッフベースアプリおよびイントラネットで視聴者ごとに動的なコンテンツを公開することができます。これは、ニュースページ、チャンネル、投稿の3層構造で構成されています。ハイライトと承認は、ニュース投稿記事にニュースフィード内でより注目を集めるための2つのオプションです。
ハイライト
ハイライトは、ニュースフィード内や投稿自体でニュース投稿記事に注目を集めます。記事の上部に重要ラベルが表示され、読者にとってその記事が最重要で見逃せないことを示します。
承認
承認は、従業員に特定のニュース投稿記事を読んだことを承認するよう求めるもので、必須またはセキュリティ関連の更新に最適です。これにより、お客様はコンプライアンスの向上、フォローアップ作業の削減、重要なコミュニケーションの信頼できる監査証跡の作成が可能になります。
承認はすべてのニュースチャンネルタイプで利用できます。特定の投稿のオプションセクションで有効にすると、投稿には承認が必要バッジが明確に表示され、読者はすぐにアクションが必要であることがわかります。投稿内で従業員は既読として承認ボタンをクリックし、その操作が記録されます。ユーザーは1回だけ投稿を承認できます。
Studioでは、編集者以上の権限を持つユーザーが、投稿の統計情報から承認数を確認したり、承認したユーザーのCSVリストをエクスポートしたりできます。承認率が希望より低い場合、編集者は未承認ユーザーのみに通知を再送信できます。
