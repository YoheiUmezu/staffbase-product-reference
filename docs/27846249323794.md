# システムメールのカスタマイズ

**URL**: https://support.staffbase.com/hc/ja/articles/27846249323794
**取得日**: 2026-05-02 18:41:24 JST

## 内容

管理者
Staffbaseを使用すると、Studioでシステムメールアドレスとアイコンを直接管理できます。このオプションにより、更新が簡素化され、手動作業が削減され、組織のブランディングが常に最新の状態に保たれます。
システムメールアドレスはメールのフッターに表示され、Studioのログイン画面で連絡先アドレスとして表示され、パスワードリセットの返信用メールアドレスとして使用されます。メールアイコンはメールのヘッダーに表示されます。
メールアイコンの要件
サイズ: 幅360ピクセル、高さ最大360ピクセル
ファイル形式: png
Studioで、設定 > システムメールに移動します。
メールアイコンの下で、コンピュータからファイルをアップロードまたはドラッグアンドドロップします。
メールフィールドに、システムメールおよびサインインページに表示する必要のあるメールアドレスを入力します。
保存をクリックします。
アイコンとシステムメールアドレスを追加しました。いつでも更新できます。
システムメールとメールアイコンがユーザーにどのように見えるか
Studioのログインページで、ユーザーはシステムメールアドレスを見ることができます。
招待状やパスワードリセットなどのシステム生成メールでは、ユーザーはヘッダーにメールアイコンを、フッターにメールを確認できます。
