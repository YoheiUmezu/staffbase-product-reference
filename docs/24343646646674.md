# Staffbaseメール用のカスタムフォントのアップロード

**URL**: https://support.staffbase.com/hc/ja/articles/24343646646674
**取得日**: 2026-05-02 18:51:11 JST

## 内容

管理者
カスタムフォントは、プラットフォーム上で簡単に利用できないフォントです。メールをさらにパーソナライズするために、以下の形式のカスタムフォントをアップロードできます：
ファイルサイズ：最大5MB
ファイルタイプ：.TTF, .OTF, .WOFF, .WOFF2
Staffbaseは可変フォントをサポートしていません。
カスタムフォントをアップロードする前に覚えておくべきこと：
カスタムフォントは、Staffbaseの契約に基づく顧客データと見なされます。プラットフォーム上での使用に関して適切なライセンスを確保する責任があります。アップロードすることで、必要なライセンスを保持していることを確認します。
プラットフォームにアップロードしたカスタムフォントを削除することはできません。
スタジオで、設定 メールフォントに移動します。
フォントページが開きます。
+ フォントを追加をクリックします。
アップロードおよびドラッグ＆ドロップのオプションが表示されます。
ファイルをアップロードまたはドラッグ＆ドロップします。
ファイルがフォントのリストに追加されます。
カスタムフォントのアップロードに成功しました。これで、デフォルトまたはセカンダリフォントとして設定し、その後フォールバックフォントを設定できます。
