# コミュニティ発見ページを公開する

**URL**: https://support.staffbase.com/hc/ja/articles/35131557865618
**取得日**: 2026-05-02 18:52:50 JST

## 内容

コミュニティは、ディスカッションや共有のための専用エリアであり、ユーザーがより強い接続を築いたり、重要な取り組みに関して連携を保つのに役立ちます。ユーザーは、コミュニティ発見ページを使って、いつでも関連するコミュニティを見つけて参加でき、より接続されたエンゲージメントの高い職場を実現します。
前提条件：
あなたはコミュニティ プラグインをインストールしています。
あなたはコミュニティを設定し、公開しています。
Studio XX内で、コンテンツ > メニュー > コミュニティに移動します。
矢印アイコンをクリックします。
プラグイン リンク内のコミュニティ リンクが開く。
公開するをクリックします。
確認ダイアログが表示されます。
公開するをクリックします。
コミュニティ発見ページが公開されました。メニュー順序でいつでもその場所を変更できます。すべての公開済みコミュニティは、ユーザーが見つけて参加できるようになりました。発見ページはいつでも非公開にできます。
