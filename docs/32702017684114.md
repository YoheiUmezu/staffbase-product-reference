# コンテンツデザイナーでクイックアクションメニューを使用する

**URL**: https://support.staffbase.com/hc/ja/articles/32702017684114
**取得日**: 2026-05-02 19:01:19 JST

## 内容

編集者
Content Designer では、ブロックやメディアの追加、セクションごとのアクションの管理をクイックアクションメニューで行うことができます。編集者を離れることなく、ページを効率的に作成・調整できます。
Studio で コンテンツ > ページ > 編集したいページに移動します。
Content Designer で、ブロックを追加したいカラムまたはコンテナを選択します。
Cmd/Ctrl + Shift + K を使ってクイックアクションメニューを開きます。
次のオプションが表示されます：
ブロックを追加する
ブロックを追加 を選択します。
ブロックを追加 のリストが表示されます。
リストからブロックを選択するか、検索フィールドで探して選択します。
クイックアクションメニューを使ってページにブロックを追加できました。
メディアを追加する
メディアを追加 を選択します。
メディアを追加 のリストが表示されます。
ファイルを選択 するか、検索フィールドで探して選択します。
クイックアクションメニューを使ってページにメディアを追加できました。
セクションのコンテキストアクションを利用する
セクションのコンテキストアクション を選択します。
コンテキストアクション のリストが表示されます。
リストからコンテキストアクションを選択します。
クイックアクションメニューを使ってコンテキストアクションにアクセスできました。
