# SharePointリダイレクトサイトを検索結果から除外する

**URL**: https://support.staffbase.com/hc/ja/articles/4410600114194
**取得日**: 2026-05-02 19:00:49 JST

## 内容

ニュースセンター用に設定したSharePointリダイレクトサイトは、StaffbaseとSharePointプラットフォーム間の安全でシームレスな認証プロセスを確保するページとして機能します。ユーザーは、アクセス権のあるすべてのサイトから検索結果を取得します。リダイレクトサイトは認証のみに使用されるため、検索結果から除外できます。
ニュースセンター用に設定したSharePointリダイレクトサイトのサイト設定に移動します。
サイト設定ページが開きます。
検索とオフラインの可用性をクリックします。
質問このサイトを検索結果に表示しますか？
に対していいえを選択します。
OKをクリックします。
リダイレクトサイトが検索結果に表示されないように除外されました。
