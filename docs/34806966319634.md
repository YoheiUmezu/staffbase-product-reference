# サービスナウ統合の概要

**URL**: https://support.staffbase.com/hc/ja/articles/34806966319634
**取得日**: 2026-05-02 18:40:28 JST

## 内容

有効化
Staffbaseはサービスナウと統合し、ユーザーが迅速にインフォメーションを検索したり、サポートリソースにアクセスしたり、プラットフォーム内から直接チケットを作成したりできるようにします。これらの統合により、内部コミュニケーションとITやサービスワークフローが接続され、業務の効率化が図れます。
Staffbaseは、特定のユースケースに合わせて設計された複数のサービスナウ統合を提供しています：
サービスナウ検索統合
サービスナウ検索統合を使用すると、ユーザーはStaffbaseから直接サービスナウコンテンツを検索できます。ユーザーは、プラットフォームを離れることなく、関連するナレッジベース記事、ドキュメント、またはサポートインフォメーションを見つけることができます。
サービスナウチケットウィジェット
サービスナウチケットウィジェットは、ユーザーがアプリやイントラネットを離れることなく有効なサービスナウチケットの概要を表示できます。ユーザーはリンクを開いて、該当するサービスナウページでチケットを表示、作成、更新、または返信することができます。
ナビゲーターとのサービスナウ統合
ナビゲーター内のサービスナウ統合は、複数の機能を統合した体験を提供します。ユーザーは、音声またはテキストで任意の言語で質問し、サービスナウ検索やサービスナウチケットウィジェットによるインスタントな回答を受け取ることができます。さらに、ナビゲーター設定で該当するチケットフォームが設定されていれば、ナビゲーターがユーザーにチケット作成を案内することも可能です。
各統合を使用するタイミング
ニーズに応じて、以下の統合を使用できます：
サービスナウ検索統合：ナレッジ発見やセルフサービス向け
サービスナウチケットウィジェット：既存チケットの追跡や管理向け
ナビゲーターとのサービスナウ統合：チケット作成を含む包括的なサポート体験向け
