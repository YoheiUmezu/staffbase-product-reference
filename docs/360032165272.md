# ファビコン

**URL**: https://support.staffbase.com/hc/ja/articles/360032165272
**取得日**: 2026-05-02 19:00:31 JST

## 内容

ファビコンはウェブアプリでのみ使用されます。これは、ウェブブラウザのタイトルバーやブックマークリストに表示されるアイコンです。すべてのプラットフォームで表示される一般的なアイコンを提供できます。また、iOSシステムのブラウザ用に異なるアイコンを提供することも可能です。
要件
サイズ: 最小 260px x 260 px
ファイル形式: png
解像度: 72 dpi
カラープロファイル: sRGB
ファイル名: Favicon.png
ミニマリスティックデザイン
ファビコンを提供しない場合、iOSアプリアイコンが再利用されます。
