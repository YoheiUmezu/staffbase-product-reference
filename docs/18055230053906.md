# Staffbaseコンパニオンエディターを使用した記事テキストの編集

**URL**: https://support.staffbase.com/hc/ja/articles/18055230053906
**取得日**: 2026-05-02 18:46:42 JST

## 内容

アクティベーション
エディター
AI
Staffbase Companion Editorは、コンテンツクリエイターが生成AIを使用して影響力のあるニュース記事を作成するのをサポートします。Companion Editorは、少量の寄稿者の入力に基づいてニュースコンテンツを生成、編集、洗練することができます。これは、ニュースのArticleチャネルタイプのStaffbase Studioで利用可能です。Staffbase Companion Editorの詳細を学ぶ。
StudioのArticleチャネルでCompanion Editorを使用すると、寄稿者の役割以上のユーザーは以下のことができます：
記事テキストの生成、編集、ドラフト作成、要約、トーンの調整
ティーザー文の生成
タイトルの生成
Companion Editorの生成AI機能は、Microsoft Azure OpenAI Serviceによって提供されています。Staffbase Companion Editorに特有のセキュリティに関する質問がある場合は、Staffbase Companion EditorセキュリティFAQで詳細を学んでください。
Staffbaseは、AI生成コンテンツを常にレビューし、公開前に特定のニーズに合わせて編集することを推奨します。StaffbaseでのAIコンテンツについての詳細を学ぶ。
記事テキストの生成
Companion Editorを使用して、特定のトピックに関するよく作られた記事を新規またはドラフト記事として作成できます。テキストの長さやトピックを含む、必要な仕様を短いプロンプトで記述してください。
Studioで、コンテンツ > ニュース > のArticleチャネルに移動します。
投稿を作成をクリックして新しい記事を作成するか、編集をクリックしてドラフト記事を開きます。
記事テキストボックスにスラッシュ ( / )を入力します。
Companion Editorのオプションが表示されます。
または、コンテンツエディターツールバーの杖アイコンをクリックします。
自分のプロンプトを使用をクリックします。
プロンプトテキストフィールドが開きます。
必要な詳細を挿入し、プロンプトをニーズに合わせて調整します。
AI生成テキストが表示され、レビューできます。
オプションで、プロンプトを変更をクリックして入力を調整するか、再生成をクリックして異なるテキストを作成します。
このテキストを挿入をクリックします。
テキストが記事テキストボックスに挿入されます。
自分のプロンプトから記事テキストを生成し、記事に挿入しました。これで記事の設定を完了できます。
