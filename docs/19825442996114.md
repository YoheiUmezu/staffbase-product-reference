# CSVファイルのインポート

**URL**: https://support.staffbase.com/hc/ja/articles/19825442996114
**取得日**: 2026-05-02 19:00:44 JST

## 内容

ベータ
アクティベーション
管理者
以下のステージでCSVファイルをインポートできます
下書き
ファイルをアップロードした直後、下書き状態のときにファイルをインポートできます。
スタジオで、ユーザー > ユーザーインポートと同期に移動します。
インポート開始
をクリックします。設定ページが開きます。
オプションで、設定を変更します。
次へをクリックします。
オプションで、マッピングを変更します。
ファイルをインポートをクリックします。
確認ダイアログが開きます。
ファイルをインポートをクリックします。
インポートが成功したという確認メッセージが表示されます。ファイルのステータスはインポート中に変更され、プロセスが終了すると、ステータスは完了に変更されます。
プレビュー
インポートしたいCSVをプレビューした後にファイルをインポートできます。
CSVファイルをプレビューします。
ファイルをインポートをクリックします。
確認ダイアログが開きます。
ファイルをインポートをクリックします。
インポートが成功したという確認メッセージが表示されます。ファイルのステータスはインポート中に変更され、プロセスが終了すると、ステータスは完了に変更されます。
