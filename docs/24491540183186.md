# スタッフベースメールでメールを複製する

**URL**: https://support.staffbase.com/hc/ja/articles/24491540183186

## 内容

この記事はAIを使用して翻訳されており、不正確な点が含まれている可能性があります。情報に不安がある場合は、正確性のために英語の原本を参照してください。
編集者
スタッフベースメールでは、下書き、テンプレート、送信済みメールを複製できます。
メールで複製したいメールに移動します。
その他のオプションメニューをクリックします。
ドロップダウンメニューが開きます。
複製をクリックします。
複製されたメールがメールデザイナーで開きます。
メールのコンテンツや視聴者を編集し、メールを送信するか下書きとして保存し続けます。
メールの複製に成功しました。
