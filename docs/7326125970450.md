# 従業員メール製品のアクセシビリティ

**URL**: https://support.staffbase.com/hc/ja/articles/7326125970450
**取得日**: 2026-05-02 18:43:51 JST

## 内容

Staffbaseでは、デジタルアクセシビリティにコミットしています。私たちは、アクセシビリティが不可欠であり、すべての人が私たちの製品を使用できるようにしたいと考えています。
私たちの取り組み
従業員メールは、添付されたレポートに基づき、WCAG 2.1のアクセシビリティ基準を満たしています。Staffbaseは、すべてのお客様にとって製品がアクセス可能であることを監査し、保証するために、外部のアクセシビリティパートナーと自主的に連携しました。私たちの製品をアクセシブルにすることは継続的なプロセスであり、アクセシビリティに関連する更新を定期的に行うために常に取り組んでいます。
私たちの自主的製品アクセシビリティテンプレート（VPAT）レポートのコピーをダウンロード – 2022年7月。
推奨スクリーンリーダーとブラウザー
一般的に、特定のニーズに最適な支援技術を使用できます。より良い体験を確保するために、以下の使用をお勧めします：
macOS：オペレーティングシステムに組み込まれたVoice Over技術を搭載した最新のSafariバージョン
Windows：最新のEdgeまたはGoogle Chromeブラウザーと最新のNVDAまたはJAWSスクリーンリーダーソフトウェアの組み合わせ
お問い合わせ
アクセシビリティへの取り組みに関するコメント、提案、または追加の質問がある場合は、support@staffbase.comまでご連絡ください。
