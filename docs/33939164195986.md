# ニュース投稿を別のチャンネルに移動する

**URL**: https://support.staffbase.com/hc/ja/articles/33939164195986
**取得日**: 2026-05-02 19:02:36 JST

## 内容

編集者
ニュースは、スタッフベースアプリとイントラネットで視聴者ごとに特化したコンテンツを公開することができます。ニュースチャンネル内でニュース投稿記事を作成することができます。また、同じ種類の別のチャンネルに移動することも可能です。
例えば、更新投稿を別の更新チャンネルタイプに移動できます。投稿を移動すると、そのコメントやいいねも一緒に移動されます。いいねやコメントを移動したくない場合は、投稿をコピーすることができます。
投稿が別のチャンネルに移動されると、以前の公開ステータスが保持されます。例えば、公開済みの投稿を別のチャンネルに移動すると、その投稿は公開モードのまま移動されます。
投稿を移動すると、新しいチャンネルの表示設定によってはユーザーがその投稿にアクセスできなくなる場合があります。
Studioでコンテンツ > ニュースに移動します。
チャンネルとコピーしたい投稿を探します。
オプションで、投稿を移動をクリックします。
投稿を移動ダイアログが開きます。
ドロップダウンから、投稿を移動したいチャンネルを選択します。
移動をクリックします。
投稿を移動したチャンネルが開きます。
投稿を新しいチャンネルに正常に移動しました。
