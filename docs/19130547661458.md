# Staffbaseをカスタムアプリとして追加する

**URL**: https://support.staffbase.com/hc/ja/articles/19130547661458
**取得日**: 2026-05-02 18:46:39 JST

## 内容

アクティベーション
管理者
StaffbaseプラットフォームをMicrosoft Teamsに簡単にインストールするには、カスタムアプリとしてアップロードします。これは、Staffbaseから提供されたアプリパッケージの.zipファイルをアップロードすることで行えます。ファイルを取得するには、Staffbaseサポートまたはカスタマーサクセスマネージャーにお問い合わせください。
前提条件
Microsoft 365でグローバル管理者またはTeamsサービス管理者の役割を持っている必要があります。
Staffbaseから受け取ったアプリパッケージを持っている必要があります。
Microsoft TeamsにStaffbaseプラットフォームをインストールする
Microsoft Teams管理センターで、Teamsアプリ > アプリの管理に移動します。
または、Microsoft 365管理センターからMicrosoft Teamsにアクセスすることもできます。
+ アップロードをクリックします。
新しいカスタムアプリをアップロードするためのダイアログが開きます。
Staffbaseから受け取ったアプリパッケージを選択してアップロードします。
ファイルのアップロードが成功すると、従業員アプリがMicrosoft Teams内で利用可能になります。
