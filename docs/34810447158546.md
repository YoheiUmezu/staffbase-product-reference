# AIページ生成をユーザーまたはユーザーグループに展開する

**URL**: https://support.staffbase.com/hc/ja/articles/34810447158546
**取得日**: 2026-05-02 18:50:39 JST

## 内容

管理者
AI
AIページ生成は、Content Designer使用時に利用できます。プロンプトに基づいて構造化されたページを素早く作成するのに役立ちます。手作業の負担を減らしながら、最終的なコンテンツを完全にコントロールできます。
AIページ生成機能へのアクセス権を持つユーザーまたはユーザーグループを管理できます。これにより、この機能を段階的に導入できます。たとえば、最初に少数の編集者グループに提供し、ベストプラクティスやワークフローを定義してから、あらゆるユーザーに展開できます。
この機能はあらゆるユーザーにも利用可能にできます。
AIページ生成を有効にする。
AIページガバナンスの表示を設定する。
保存するをクリックする。
AIページ生成を特定のユーザーまたはユーザー グループに展開しました。
