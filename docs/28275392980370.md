# 計画の概要

**URL**: https://support.staffbase.com/hc/ja/articles/28275392980370
**取得日**: 2026-05-02 18:41:23 JST

## 内容

キャンペーンマネージャー
概要ページは、キャンペーンマネージャーに計画中のコンテンツや有効および今後のキャンペーンを一元的に表示します。キャンペーンの全体的な紹介やその利点、インスピレーションとなる事前定義されたキャンペーンテンプレートが用意されており、新しい取り組みをすぐに始めることができます。これにより、キャンペーンマネージャーは効果的なキャンペーンを簡単に発見、計画、実行できます。
概要ページから直接、次の操作が便利に行えます：
キャンペーン、投稿を作成する、メールを作成する
今週の投稿を作成するやメールを日ごとに表示
有効および今後のキャンペーンを一目で確認
タイトルをクリックして投稿を作成する、メール、またはキャンペーンを開く
キャンペーンの目的や使い方について学ぶ
開始に役立つキャンペーンテンプレートを確認
キャンペーンマネージャーが概要ページから投稿を作成するやメールを作成するには追加の権限が必要です。
概要ページの利点
新規ユーザーのオンボーディング
概要ページには、キャンペーンの仕組みを理解し、コンテンツを作成するための導入ガイダンスと事前定義されたテンプレートが含まれています。
アラインメント
コンテンツ全体の状況をすぐに把握できます。概要ページでは、有効および今後のキャンペーン、さらに今週のすべてのメールや投稿を作成するが表示されます。
クイックアクセス
作成するボタンやクイックアクセスリンクにより、新しいキャンペーンの開始、メールや投稿を作成するの作成、表示されているキャンペーンやコンテンツへの移動が簡単に行えます。
