# ジャーニー概要の作成

**URL**: https://support.staffbase.com/hc/ja/articles/27609075544082
**取得日**: 2026-05-02 18:59:25 JST

## 内容

アクティベーション
システム全体の編集者
ジャーニー概要は、ユーザーがすべてのジャーニーを一元的に確認できるハブです。ジャーニーのステップのタイムラインと進捗状況が明確に表示されます。ジャーニー概要は1つだけ作成できます。
Studioでコンテンツ > ジャーニーに移動します。
ジャーニー概要の下で、追加するをクリックします。
ジャーニー概要を追加するダイアログが開きます。
タイトルの下に、ジャーニー概要の名前を入力します。
ジャーニー概要を追加するをクリックします。
ジャーニー概要が追加されました。従業員アプリに統合したり、必要に応じてジャーニー概要専用のランディングページを作成したりできます。
デフォルトでは、すべてのジャーニーがジャーニー概要に含まれます。特定のジャーニーを除外するには、ジャーニーの設定を開き、ジャーニー概要に表示トグルを無効にします。
