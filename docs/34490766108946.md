# コミュニティの作成

**URL**: https://support.staffbase.com/hc/ja/articles/34490766108946
**取得日**: 2026-05-02 18:48:24 JST

## 内容

システムワイド編集者
コミュニティは、ディスカッションや共有のための専用エリアであり、従業員がより強い接続を築いたり、重要な取り組みに関して連携を保つのに役立ちます。従業員は、発見ページを使用して、いつでも関連するコミュニティを見つけて参加でき、より接続されたエンゲージメントの高い職場を実現します。
前提条件:
コミュニティプラグインをインストールしています。
コミュニティを作成する
Studio内でコンテンツ > コミュニティに移動します。
コミュニティを追加するをクリックします。
コミュニティのタイトルを入力します。
作成するをクリックします。
コミュニティの設定が開きます。
コミュニティが作成され、設定を構成できるようになりました。
