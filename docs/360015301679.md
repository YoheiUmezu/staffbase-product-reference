# すぐに使える統合

**URL**: https://support.staffbase.com/hc/ja/articles/360015301679
**取得日**: 2026-05-02 18:52:13 JST

## 内容

最も一般的に使用される統合は、Staffbaseによって簡単に利用できるようになっています：
リンク
埋め込み
接続
リンク
リンクは、外部アプリケーションや外部ウェブサイトからのファイルをプラットフォーム上で利用可能にする、最もシンプルなタイプのすぐに使える統合です。リンクは、Staffbaseプラットフォームと外部コンテンツの間の橋渡しをします。ユーザーが外部コンテンツに迅速にアクセスできるようにしたい場合に最適です。
Staffbaseは、ハイパーリンクからウィジェットやプラグインまで、外部コンテンツをプラットフォームにリンクするさまざまなオプションを提供しています。ユーザーが従業員アプリやイントラネットからリンクをクリックすると、ユーザーは外部ウェブサイトやアプリケーションに移動してリンクされたコンテンツにアクセスします。
プラットフォーム内の以下の機能でコンテンツをリンクできます：
ニュースおよびページの外部リンク
ハイパーリンク：ページおよびニュースエディタでハイパーリンクを使用して、従業員プラットフォームのユーザーにクリック可能なリンクを提供します。ハイパーリンクをクリックすると、リンク先のウェブサイト、メール、または電話番号に直接移動します。
リンクウィジェット：リンクリストやリンクタイルウィジェットなどのリンクウィジェットを使用して、ユーザーにリストまたはタイルブロック形式で外部リンクを提供します。
アプリメニューの外部リンク
ウェブおよびモバイルアプリのメインメニューでユーザーに外部コンテンツを提供します。たとえば、ウェブサイトのホームページへのリンクをメニュー項目として提供したい場合に最適です。これにより、ユーザーは直接ホームページに移動します。このためには、リンクプラグインを使用します。
ネイティブアプリのクイックリンク
アプリメニューだけでなく、ナビゲーションバーから簡単にアクセスできるクイックリンクとして外部コンテンツを追加できます。このためには、リンクプラグインを使用します。
ランチャーパッドの集中リンク
ユーザーが定期的にアクセスするアプリケーションやウェブサイトを、従業員プラットフォーム内の専用の場所にまとめて提供し、ユーザーがすべてのアプリケーションに一か所からアクセスできるようにします。ランチャーパッドを使用して、組織が使用するさまざまなアプリケーションにリンクできます。詳細はこちら。
埋め込み
プラットフォームにコンテンツを埋め込むことは、リンクするコンテンツを一歩進める方法です。外部アプリケーションのユーザーインターフェースをStaffbaseプラットフォームで利用可能にします。ユーザーは、従業員アプリやイントラネットから離れることなく、同じユーザーエクスペリエンスと快適さを体験します。
埋め込みは、外部コンテンツを表示するための新しいユーザーインターフェースを提供することもできます。これにより、コンテンツが表示されるユーザーインターフェースが、従業員アプリやイントラネットのデザインやテーマに一致します。このようにして、ユーザーにとっては、従業員アプリやイントラネットの一部として実際に表示されます。
外部コンテンツの埋め込み
他のアプリケーションやウェブサイトから外部コンテンツを簡単に従業員プラットフォームに埋め込むことができます。このようにして、ユーザーは外部コンテンツに、まるで自分の従業員プラットフォームに存在するかのようにアクセスできます。外部コンテンツがパスワード保護されていて、SAMLプロトコルをサポートしていない場合、ユーザーは従業員プラットフォームに表示される外部コンテンツにアクセスするために、適切な資格情報でサインインする必要があります。このためには、埋め込みページプラグインを使用します。
サインイン保護された外部コンテンツの埋め込み
SAML SSOプロトコルを使用して、従業員プラットフォームにサインイン保護されたコンテンツを埋め込みます。このようにして、ユーザーはアカウント資格情報で積極的にサインインすることなく、外部コンテンツにアクセスできます。シングルサインオンの設定はバックグラウンドで機能し、シームレスでありながら安全なユーザーエクスペリエンスを実現します。ユーザーは、メニューやプラットフォーム内のページでコンテンツを確認できます。このためには、SAMLプラグインを使用します。
プラグインを介した埋め込み
さまざまな利用可能なプラグインを使用して、コンテンツやサービスをシームレスに追加します。プラグインは、技術的な実装パートナーやStaffbase自体から提供されることがあります。
スタジオのプラグインページを参照して、組織で利用可能なプラグインを確認できます。さらに、Staffbaseサポートやカスタマーサクセスマネージャーに連絡して、組織のためにより多くのプラグインを有効にしたい場合はお知らせください。いくつかの例は次のとおりです：
フォームプラグイン
マッププラグイン
クイズカレンダープラグイン
Staffbaseが提供するすべてのプラグインのリストを確認するには、プラグインクイックスタートを参照してください。
接続
外部システムと従業員アプリやイントラネットの間に接続を確立し、このシステムから特定の情報や、ユーザーのインタラクションとデータ交換のいくつかのステップを含むプロセスを表示します。外部システムへのデータ接続は、プラットフォームの使用の快適さと体験を一歩進めます。
このような接続は、Staffbaseプラットフォームとの間で双方向に機能することができます。この双方向アプローチにより、組織に最適なアプローチを選択する柔軟性が得られます。
Microsoft統合
Staffbaseは、SharePoint、Microsoft 365、Power AutomateなどのMicrosoftアプリケーションおよびプラットフォームとの統合を提供します。Staffbaseは、MicrosoftからStaffbaseへの統合とその逆も提供します。
以下は、Staffbaseが提供するMicrosoft統合の概要です：
Microsoft ←→ Staffbase
以下の統合機能を使用して、統合を開始しましょう：
Microsoft 365 Power Automateコネクタ
Microsoft 365拡張ページ
Microsoft Connections
Microsoft → Staffbase
以下の統合機能を使用して、Microsoftアプリケーションの機能やコンテンツをStaffbaseプラットフォームに取り込みます：
Microsoft 365検索
Microsoft 365ウィジェット
Staffbase → Microsoft
以下の統合機能を使用して、Staffbaseの機能やコンテンツをMicrosoftプラットフォームに取り込みます：
Staffbase 365ニュースセンター
Staffbase PnPモダン検索拡張ライブラリ
Microsoft Teamsアプリ
Teamsでのコラボレーション
Staffbaseには、レガシー機能としての追加のMicrosoft統合があります。Microsoft 365統合を参照してください。
Box検索統合
BoxをStaffbaseプラットフォームに接続して、専用タブ内でBoxのコンテンツを検索およびアクセスできます。詳細を学ぶ。
Confluence検索統合
AtlassianをStaffbaseプラットフォームに接続して、専用タブ内でConfluenceのコンテンツを検索およびアクセスできます。詳細を学ぶ。
ServiceNow検索統合
ServiceNowをStaffbaseプラットフォームに接続し、専用タブ内でServiceNowのナレッジベースリソースを検索およびアクセスできます。詳細を学ぶ。
RSSフィード
外部ウェブサイトのRSSフィードに接続して、そのコンテンツをプラットフォーム上に表示できます。詳細を学ぶ。
ユーザー管理
組織のさまざまなアプリケーションやシステムにおけるユーザーとその役割の管理は、大規模な組織にとって継続的な努力です。ユーザーデータを統合し、ユーザーに関連する役割やグループ属性を割り当てることで、ユーザーがStaffbaseプラットフォームを効率的に使用できるようにします。これにより、管理者のタスクが簡素化され、自動化されることで、管理者の権限が強化されます。組織のユーザー管理におけるさまざまなプロセスを異なるアプリケーションで統合することで、より信頼性が高く効率的になります。Staffbaseプラットフォームで利用可能な一般的な標準およびプロトコルは以下の通りです：

CSVインポート：CSVインポートを通じてユーザーライフサイクルを管理できます。新しいユーザーを作成したり、ユーザーを更新したり、組織を退職したユーザーをStaffbaseプラットフォームから削除したりできます。詳細を学ぶ。

SSO via SAML/OpenID：組織がすでにSSOアプリケーションを使用している場合、SSOアカウントを使用してユーザーをオンボードできます。詳細を学ぶ。
