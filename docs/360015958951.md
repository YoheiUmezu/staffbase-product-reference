# ユーザーのアカウント回復の開始

**URL**: https://support.staffbase.com/hc/ja/articles/360015958951
**取得日**: 2026-05-02 18:52:29 JST

## 内容

管理者はユーザーのアカウント回復プロセスを開始できます。これは、メールアドレスでサインインするユーザーにのみ可能です。
スタジオで、ユーザー設定 > 登録ユーザーに移動します。
必要なユーザーに移動し、編集をクリックします。
ユーザー編集ダイアログで、パスワードをリセットをクリックします。
ユーザーは回復コードを含むメールを受け取ります。
