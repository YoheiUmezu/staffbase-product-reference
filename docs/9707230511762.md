# アプリとイントラネットのためのRTL（右から左）言語サポート

**URL**: https://support.staffbase.com/hc/ja/articles/9707230511762
**取得日**: 2026-05-02 18:51:36 JST

## 内容

言語は、文字が書かれ読み取られる方向によって分類されます。左から右に書かれ読み取られる言語はLTR（左から右）言語と呼ばれます。右から左に書かれ読み取られる言語はRTL（右から左）言語と呼ばれます。Staffbaseは、ユーザーに最良の体験を提供するために、LTRおよびRTL言語の両方をサポートしています。
アプリとイントラネットでサポートされているRTL言語：
アラビア語
ヘブライ語
ウルドゥー語
パシュトゥー語
ペルシャ語
組織のために別のRTL言語をリクエストするには、Staffbaseサポートまたはカスタマーサポートマネージャーにお問い合わせください。
ユーザーがLTR言語からRTL言語に切り替えると、ユーザーインターフェースがミラーリングされます。これは、テキストだけでなく、全体のレイアウトやアイコンもRTLの慣習に合わせて配置されることを意味します。
RTL言語サポートの制限
組織のためにRTL言語を設定する際には、いくつかの制限に注意する必要があります。
スタジオ
スタジオのほとんどは、フォームおよびイベント登録プラグインの一部を除いて、まだRTLに準拠していません。
特定のプラグイン
RTLサポートは、以下のようなプラグインにはまだ完全には拡張されていません：
地図
ランチャーパッド
カスタムプラグイン
ユーザーに見える違い
インターフェース、レイアウト、テキストは、アプリまたはイントラネットで選択された言語に応じて適応します。
いくつかの例は以下の通りです：
検索
投稿の上にある統計と編集オプション
通知設定ページ
