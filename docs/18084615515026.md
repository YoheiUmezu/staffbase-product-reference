# 個人メールアドレスの変更

**URL**: https://support.staffbase.com/hc/ja/articles/18084615515026
**取得日**: 2026-05-02 18:56:49 JST

## 内容

ベータ
アクティベーション
アプリまたはイントラネットの個人メールアドレスはいつでも編集できます。
前提条件
プラットフォームの個人メールフィールドが編集可能になっています。
ウェブアプリ
イントラネットでプロフィールメニューに移動します。
プロフィールメニューが開きます。
設定 アカウントに移動します。
アカウント設定ページが開きます。
メールアドレスを変更をクリックします。
メールアドレスの変更ダイアログが開きます。
現在のメールアドレスフィールドは、登録時にメールアドレスを提供しなかった場合は空のままです。
新しいメールフィールドにメールアドレスを入力します。
メールアドレスを変更をクリックします。
成功メッセージが表示されます。
アプリまたはイントラネットの個人メールアドレスが変更されました。
iOS
モバイルアプリでメニューを開きます。
歯車アイコンをタップして設定を開きます。
アカウント設定の下で、メールアドレスを変更をタップします。
現在のメールアドレスフィールドは、登録時にメールアドレスを提供しなかった場合は空のままです。
新しいメールフィールドにメールアドレスを入力します。
メールアドレスを変更をタップします。
成功メッセージが表示されます。
アプリまたはイントラネットの個人メールアドレスが変更されました。
Android
モバイルアプリで画面右上の三点リーダーをタップしてプロフィールメニューを開きます。
設定の下で、アカウントをタップします。
アカウント設定の下で、メールアドレスを変更をタップします。
現在のメールアドレスフィールドは、登録時にメールアドレスを提供しなかった場合は空のままです。
新しいメールフィールドにメールアドレスを入力します。
メールアドレスを変更をタップします。
成功メッセージが表示されます。
アプリまたはイントラネットの個人メールアドレスが変更されました。
