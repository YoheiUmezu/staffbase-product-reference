# 投稿の通知を再送信する

**URL**: https://support.staffbase.com/hc/ja/articles/360017402559
**取得日**: 2026-05-02 19:02:41 JST

## 内容

エディター
まだ投稿を読んでいないユーザーにのみ通知を再送信することで、全体の通知数を減らし、ユーザーが重要なニュースを把握しやすくなります。投稿をまだ読んでいないか、認識していないユーザーに通知を再送信します。
スタジオで、再送信したい投稿のニュースチャンネルに移動します。
ユーザーに通知したい投稿を選択します。
編集をクリックします。
再送信をクリックします。
再送信通知の対象となるオーディエンスを選択します。
通知の種類を選択します。
今すぐ送信をクリックします。
