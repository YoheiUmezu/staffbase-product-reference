# Staffbaseメールのデフォルトフォントの設定

**URL**: https://support.staffbase.com/hc/ja/articles/24343489444626
**取得日**: 2026-05-02 18:46:18 JST

## 内容

管理者
デフォルトフォントとは、メール作成時に特定のフォントが選択されていない場合に自動的に適用されるフォントを指します。これは、メールブランディングページで「デフォルト」ラベルが付けられています。選択したフォントのオプションに応じて、太字、斜体、または太字斜体など、フォントのスタイルを選択することもできます。
スタジオで、設定 > メールに移動します。
メールブランディングページが開きます。
表示フォントの下で、デフォルトとして設定したいフォントを見つけます。
詳細オプションアイコンをクリックしてドロップダウンを開き、デフォルトとして設定を選択します。
選択したフォントがハイライトされ、デフォルトラベルが付けられます。
オプションで、デフォルトフォントタイルのレギュラーの隣にあるドロップダウンを選択します。
このオプションはすべてのフォントで利用できるわけではありません。利用可能な場合、レギュラーがデフォルトで選択されます。
ドロップダウンから、フォントのデフォルトスタイルとして選択するフォントスタイルの1つを選びます：
レギュラー
太字
斜体
太字斜体
選択したスタイルがデフォルトフォントタイルに表示されます。
保存をクリックします。
デフォルトフォントの設定が成功しました。いつでも変更できます。
