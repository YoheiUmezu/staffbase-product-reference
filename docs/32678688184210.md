# Staffbase Live の概要

**URL**: https://support.staffbase.com/hc/ja/articles/32678688184210
**取得日**: 2026-05-02 18:40:58 JST

## 内容

有効化
Staffbase Liveを使用すると、モデレーターはイベントをアプリやイントラネットに配信できます。これにより、組織全体の従業員とリアルタイムでやり取りし、エンゲージメントを高めることができます。
Staffbaseスタジオ内でEvents Centerプラグインを使って、ライブイベントの作成、スケジュール、管理ができます。イベントをスケジュールすると、参加者にはユニークな視聴リンクが送信されます。モデレーターはこのリンクをニュース投稿記事やダイレクトメッセージで共有し、対象の視聴者にリーチして参加率を高めることができます。
ライブイベントを録画して、後から視聴できるようにすることも可能です。これにより、ライブイベントを見逃した従業員も後で視聴できます。
Staffbase Liveのメリット
一元管理
Studioでライブイベントの計画、スケジュール、管理ができます。過去や今後のイベントの確認、参加者のモデレーション、イベントのコピー、リマインダーやタイムゾーン、共有アクセスリンクの利用が可能です。
魅力的なライブ体験
Staffbase Liveは、チャットや絵文字リアクションによるリアルタイムのエンゲージメントを実現します。モデレーターは視聴している従業員数を確認し、配信中にやり取りできます。
柔軟な配信オプション
外部配信ツールを使い、リアルタイムメッセージングプロトコル（RTMP）経由でイベントを配信できます。
信頼できるイベント後のアクセス
ライブイベントを録画し、オンデマンドで視聴できるようにします。これにより、重要な更新やメッセージを従業員がいつでも確認できます。
