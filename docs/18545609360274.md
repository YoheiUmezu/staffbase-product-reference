# セキュアメディアの概要

**URL**: https://support.staffbase.com/hc/ja/articles/18545609360274
**取得日**: 2026-05-02 18:46:40 JST

## 内容

セキュアメディアは、Staffbaseプラットフォーム内のメディアファイルを不正アクセスから保護するために設計された機能です。
セキュアメディアの利点
セキュア
新しくアップロードされたファイルは、認証されたユーザーのみがアクセスできるように制限された強化されたセキュリティ層の恩恵を受けます。
新しいメディアURLを異なる/シークレットブラウザにコピーして、ログインしていない状態でセキュアアクセスをテストできます。ファイルへのアクセスは制限されます。
堅牢
簡素化されたメディアURLは、APIリクエストのためのメディアIDの容易な識別を保証します。 詳細はこちら。
メディアURLの長さが短く、よりシンプルになっていることに気付くでしょう。形式は/media/secure/…/v2/です。
多用途
セキュアメディアの柔軟で機敏なアーキテクチャは、将来の技術的変化にスケーラブルで適応可能であることを保証します。
詳細については、Staffbaseサポートまたはお客様成功マネージャーにお問い合わせください。
