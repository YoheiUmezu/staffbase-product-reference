# CSVファイルを使用したユーザーデータのインポートと同期

**URL**: https://support.staffbase.com/hc/ja/articles/24593404676626
**取得日**: 2026-05-02 18:48:48 JST

## 内容

ベータ
アクティベーション
管理者
ユーザーベースを最新の状態に保つことは、管理者の最も重要なタスクの一つです。ユーザーインポートと同期は、CSVファイルを使用してスタジオから効率的に行うことができます。
CSVファイルを使用したユーザーインポートと同期が初めての場合、Staffbaseは最初の完全なインポートを試みる前に、少数のユーザーのみを含むテストから始めることを推奨します。プロセスに慣れている場合でも、ユーザーベースに誤って更新を行わないように、CSVファイルのインポートを適切に計画することをStaffbaseは推奨します。CSVファイルとユーザー管理にどのように使用するかについての詳細情報が必要な場合は、例を参照してください。
