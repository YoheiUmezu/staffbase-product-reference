# コンテンツデザイナーでページを削除する方法

**URL**: https://support.staffbase.com/hc/ja/articles/32702310703122
**取得日**: 2026-05-02 19:02:09 JST

## 内容

編集者
コンテンツデザイナーからページを削除できます。ページが不要になった場合や誤って作成された場合に便利です。
削除されたページは従業員に表示されなくなります。削除したページは30日以内であれば復元できます。それ以降はプラットフォームから完全に削除されます。
Studioでコンテンツ > ページに移動します。
削除したいページを開きます。
上部メニューから三点リーダーを選択し、アクションメニューを開きます。
削除するを選択します。
ページを削除しますか？の確認ダイアログが開きます。
削除するをクリックします。
ページが削除されました。
