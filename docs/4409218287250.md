# サポートされているビデオプラットフォーム

**URL**: https://support.staffbase.com/hc/ja/articles/4409218287250
**取得日**: 2026-05-02 18:54:12 JST

## 内容

Staffbaseは、投稿や記事にビデオコンテンツを追加するための複数のオプションを提供することで、メディア共有体験を向上させます。次のことができます：
アプリまたはイントラネットにビデオコンテンツを直接アップロードする
Microsoft Streamのビデオを追加する
次のような人気のビデオプラットフォームからビデオを埋め込む：
YouTube
Dacast
Wistia
Kaltura
3Q
Vimeo
iOSアプリでは、ドメインに制限されたVimeoビデオにアクセスすることは現在できません。iOSアプリユーザーがVimeoビデオにアクセスできるようにするには、このプライバシー設定を無効にしてください。
Staffbaseは、ビデオの比率が16:9であることを推奨します。これにより、埋め込まれたビデオが最適な解像度で表示され、ビデオ全体が表示されます。これは、Staffbaseプラットフォームの画像に対する提案でもあります。
特定のビデオプラットフォームからのビデオの埋め込みが、それぞれのプラットフォームによってブロックされる場合があります。その場合は、該当するビデオプラットフォームで埋め込みを許可する必要があります。
ビデオを埋め込む方法やアスペクト比を調整する方法については、プラットフォームのヘルプセンターを参照してください。
追加の役立つ情報
ページまたは投稿にビデオを埋め込む
YouTubeからビデオとプレイリストを埋め込む
Vimeoからビデオを埋め込む
Dacastからビデオを埋め込む
