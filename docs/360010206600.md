# アップロードが失敗した理由は何ですか？

**URL**: https://support.staffbase.com/hc/ja/articles/360010206600
**取得日**: 2026-05-02 19:00:54 JST

## 内容

Staffbaseプラットフォームは、アプリのすべてのコア機能へのアップロードに対してマルウェアスキャナーを実装しています。100メガバイト未満のすべてのファイルは、アップロードプロセス中に自動的に悪意のあるコードがスキャンされます。アップロードがキャンセルされた場合、それは悪意のあるコードが検出されたためです。
ファイルは、スキャンが正常に完了した場合にのみ、Staffbaseプラットフォームに保存されます。悪意のあるコードを含むファイルが検出された場合、ファイルが保存される前にアップロードプロセスは終了します。
マルウェアスキャナーについて詳しく知りたい場合は、アプリ管理者にお問い合わせください。
