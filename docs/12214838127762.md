# 分析チャットメトリクスとアナリティクスダッシュボード

**URL**: https://support.staffbase.com/hc/ja/articles/12214838127762
**取得日**: 2026-05-02 18:59:39 JST

## 内容

スペース管理者
アナリティクスダッシュボードは、ユーザーがStaffbaseプラットフォームとどのように、いつ、どこで関わっているかについての深い洞察を提供します。
この記事では、チャットタブについて詳細に説明し、その中にあるグラフと分析データを含みます。ユーザーの役割アクセスを含むアナリティクスダッシュボード全体については、アナリティクスダッシュボードの概要で詳しく学んでください。
チャットタブは、デフォルトで過去7日間のデータを表示します。日付範囲を変更し、結果を絞り込むためにさらにフィルターを使用できます。フィルタリングについて詳しく学ぶ。
チャットアクティビティグラフ
チャットアクティビティの折れ線グラフは、以下のメトリクスを表示します：
アクティブチャットユーザー
アクティブチャットグループ
アクティブ1:1チャット
アナリティクス用語については、アナリティクス用語集で詳しく学んでください。
チャットアクティビティ
チャットアクティビティの折れ線グラフは、異なるチャットユーザータイプに関する詳細な統計を表示します。各メトリクスには、グラフ内に対応する色があります。デフォルトでは、すべてのユーザーメトリクスが表示されます。このようにして、全体のグラフでオプションがどのように重なり合うかを見ることができます。特定の領域、たとえばアクティブチャットユーザーとアクティブ1:1チャットの間で比較できるように、1つまたは複数のメトリクスを選択解除してグラフをカスタマイズできます。
グラフ上の特定の日付にカーソルを合わせると、選択されたメトリクスに応じてその日の特定のデータを見ることができます。フィルターを適用したりメトリクスを削除したりすることで、結果をさらに絞り込むことができます。
グラフの右側の値も、選択された異なるメトリクスやフィルターに応じて適切な高値にシフトします。たとえば、アクティブチャットユーザーのメトリクスが高値で選択解除された場合、最高のグラフ値は自動的に次の高値に調整されます。グラフの下部の値も、選択された時間枠に応じて自動的にシフトします。1年などの大きな時間枠の場合、時間は月ごとに表示されます。1週間などの短い時間枠の場合、時間は日ごとに表示されます。
メトリクスは、ユーザー固有の情報や会話内で共有されたコンテンツ（プライベートな会話の詳細、誰が誰とチャットしているか、どのグループがアクティブかなど）を明らかにすることはありません。
異なるメトリクスの下にある色分けされたパーセンテージは、そのメトリクスが前回の期間からどのように変化したかを示します。緑の正の数は、メトリクスが増加したことを示します。赤の負の数は、メトリクスが減少したことを示します。
パーセンテージは、自動的に現在選択された日付範囲のメトリクスを前回のものと比較します。たとえば、現在の7日間と前の7日間です。
チャットメトリクスの選択解除
コンテンツリストは、すべての異なるテーブルヘッダーでソートできます。
スタジオで、アナリティクス > チャットに移動します。
チャットアクティビティグラフに移動します。
グラフに表示したくないメトリクスをクリックします。
メトリクスはグレーアウトされ、グラフに表示されなくなります。
メトリクスを選択解除し、グラフから削除しました。再度メトリクスをクリックすると、グラフに再度含めることができます。
CSVエクスポート
チャットタブからデータをCSVダウンロードで取得できます。
チャット時系列データのダウンロード
スタジオで、アナリティクス > チャットに移動します。
タブの上部に移動します。
オプションで、特定のデータに絞り込むためにフィルターを適用します。
CSVエクスポートをクリックします。
CSVエクスポートダイアログが開きます。
チャット時系列CSVをクリックします。
CSVダウンロードが自動的に開始されます。
チャットタブからデータをダウンロードしました。メトリクスは時系列でダウンロードされます。
