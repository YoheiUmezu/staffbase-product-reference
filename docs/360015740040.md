# ジャーニーの概要

**URL**: https://support.staffbase.com/hc/ja/articles/360015740040
**取得日**: 2026-05-02 18:44:44 JST

## 内容

アクティベーション
ジャーニーズプラグインは、コンテンツクリエイターに従業員プラットフォームでユーザーにカスタマイズされた体験を提供するための新しい可能性を提供します。ジャーニーズを使用すると、ビジネスケースごとにコンテンツを束ね、ユーザーの従業員ジャーニーの適切なタイミングで各コンテンツを配信することができます。
ジャーニーは、非常に特定のユーザーセットを対象としたコンテンツのタイムラインです。各コンテンツはジャーニーのステップであり、ユーザーがジャーニーを開始したときに即座に到着するように設定することも、特定の日付と時刻に到着するように設定することもできます。このようにして、管理者や編集者は、従業員を指導し、メンターとして支援し、組織内で成長を促すことができます。
これにより、適切なユーザーに関連するコンテンツを、ちょうど適切なタイミングで提供することができます。ジャーニーズは、以下のような場面で個人的なタッチを加えるのに最適です：
オンボーディング
役割変更の際の従業員のガイダンス
従業員のマイルストーンの祝賀
従業員トレーニング
会社のイニシアチブの紹介
新メンバーを興味グループに慣れさせる
新しいコンテンツクリエイターにベストプラクティスを紹介する
オフボーディング
ジャーニーズの利点は何ですか？
ターゲティングの集中
コンテンツのターゲティングは、誰がコンテンツを受け取るかだけでなく、いつ受け取るかも重要です。ジャーニーズを使用すると、アプリへの登録やユーザーグループへの参加などのユーザーグループやトリガーに基づいて、広範または狭いターゲットオーディエンスを選択する柔軟性があります。しかし、ユーザーが正確に適切なタイミングでコンテンツを受け取ることを確認することもできます。コンテンツを重要な瞬間に到着させることで、ユーザーのジャーニーを導く道標のようにスケジュールできます。
パーソナライズウィジェット
パーソナライズウィジェットは、ユーザーのジャーニーの各ステップに個人的なタッチを加えます。このウィジェットを使用すると、ユーザーのプロフィールからのデータを利用して、名前、役職、オフィスの場所などのスマートフィールドを投稿に挿入できます。ユーザーにとって、各ジャーニーステップはカスタマイズされたものとして表示されます。組織によって設定されたすべての標準およびカスタムプロフィールフィールドがパーソナライズウィジェットを使用して利用可能であり、可能性はほぼ無限です。ジャーニーポストの通知は、プッシュまたはメール通知を介して送信できます。
コンテンツのリサイクル
ジャーニーズの大きな利点は、編集者が新しいオーディエンスのために試行済みのコンテンツをリサイクルできることで、時間を大幅に節約できることです。コンテンツのリサイクルとターゲットユーザーの集中を組み合わせることで、必要なユーザーセグメントに基本的なコンテンツを、必要な瞬間に提供することができ、他のユーザーに馴染みのあるコンテンツで負担をかけることを恐れる必要がありません。編集者は、より大きなコンテンツを小さなブロックに分解し、ジャーニーのターゲットオーディエンスのニーズに応じてスケジュールすることもできます。
さらに、ウィジェット、リンク、メディアなどを追加するなど、ページエディタのすべての機能が、ジャーニーステップの作成時に編集者の手元にあります。これらを組み合わせることで、ジャーニーズの革新的な機能により、すべてのユーザーに対して魅力的でパーソナライズされた従業員体験を構築できます。
