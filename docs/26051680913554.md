# アイデア創出プラグインのインストール

**URL**: https://support.staffbase.com/hc/ja/articles/26051680913554
**取得日**: 2026-05-02 18:59:27 JST

## 内容

システム全体の編集者
この機能はアドオンです。詳細はStaffbase 価格をご参照ください。有効化についてはStaffbaseサポートまたはカスタマーサクセスマネージャーにお問い合わせください。
アイディエーションプラグインは、Staffbase従業員エクスペリエンス（EX）ブースタープラグインの一つです。
従業員がアイデアを共有し、コメントし、重要な内容に投票できるダイナミックなツールです。
前提条件
APIトークンを作成し、読み取り専用アクセス権を持っていること。
APIトークンを暗号化してsupport@staffbase.comに送信したこと。
APIトークンは、ユーザーAPIを通じてユーザープロフィール情報にアクセスするために必要です。APIトークンにより、プラグインはユーザープロフィール情報（プロフィール画像や権限など）にアクセスできます。これにより、承認されたユーザーのみがアイデアを投稿でき、新しいアイデアが共有された際に他のユーザーに通知されます。
Studioでコンテンツ > コンテンツメニューに移動します。
メニューの一番下までスクロールし、プラグインを追加するをクリックします。
プラグインページが開きます。
アイディエーションに移動します。
インストールをクリックします。
アイディエーションを追加するダイアログが開きます。
アイディエーションページのタイトルを入力します。
タイトルはプラグイン設定でいつでも変更できます。
表示ドロップダウンメニューから、どのユーザーがアイディエーションページを表示できるかを選択します。
アイディエーションを追加するをクリックします。
アイディエーションプラグインをインストールし、アイディエーションページを追加しました。プラグインは左側のコンテンツセクションに表示されます。ページを設定して公開することで、ユーザーに表示できるようになります。
