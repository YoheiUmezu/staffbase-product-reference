# ミールプランでユーザーをプラットフォームに誘導する

**URL**: https://support.staffbase.com/hc/ja/articles/30002559069586
**取得日**: 2026-05-02 18:41:10 JST

## 内容

ミールプラン プラグインを使うと、従業員にカフェテリアや近隣レストランで提供されるメニューを簡単に知らせることができます。ミールプランは、ユーザーにスタッフベース従業員アプリやイントラネットを紹介し、エンゲージメントを促進し、プラットフォームに慣れてもらうのに最適な方法です。日常的に役立つタッチポイントを提供し、アプリの他の機能を探すきっかけにもなります。
ユースケース：社内カフェテリアのメニュー
従業員の日常業務の重要な一部は、食事の選択です。従業員アプリやイントラネットに完全なミールプランがあることで、従業員は仕事の計画に必要な情報を得ることができます。また、プラットフォームへの訪問を促し、オンボーディング時にプラットフォームを紹介するのに最適です。ミールプラン プラグインを使えば、従業員にメニューを素早く簡単に共有でき、アプリやイントラネットに慣れてもらうことができます。
週間ミール情報を集める
さまざまな食事ニーズや好みに対応できるよう、メニューのバリエーションを承認する。
プラグインのクイックエントリー機能を使って新しい食事を追加するか、既存のものを再利用する。
必要に応じて食事をカテゴリ分けし、ドラッグ＆ドロップ機能で希望の順番に並べる。
ミールプランを従業員と共有
ミールプラン機能についての記事を作成し、全従業員に送信する。
プラグインウィジェットを使って、専用のカフェテリアやレストランのページにミールプランを表示する。
新しいミールプランを、週次や月次の会議のトピックとして取り上げる。
常に最新の状態に保つ
変更があった場合はリアルタイムでエントリーを更新し、従業員が常に正確で信頼できる情報を得られるようにする。
社外の飲食オプションをまとめたレストランページを作成し、そこにミールプランをリンクする。
ベストプラクティス
スタッフベースは、ユーザーに最適な体験を提供するためのいくつかの推奨事項を用意しています。
ミールプランをロケーションごとに調整
社内カフェテリアや地元レストランのパートナーから正確なミール情報を集める。
オフィス近くの各レストランごとに、同じ週でも別々のミールプランを作成する。
各オフィスロケーションやスペースごとに異なるミールプランを設定する。
整理と再利用
ベジタリアンやグルテンフリーなどのカテゴリを使い、簡単に閲覧できるようにする。
タイトルを入力して過去のエントリーを再利用し、時間を節約する。
多様なメニューに対応できるよう、無制限に食事を追加する。
可視性とエンゲージメントを高める
専用のカフェテリアやレストランのページにミールプランを表示する。
常に最新の状態に保ち、定期的な訪問を促す。
ミールプランの利用に加え、カフェテリアやレストランのメニューをPDFで専用ページにアップロードし、簡単にダウンロードできるようにする。
