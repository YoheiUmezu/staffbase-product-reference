# 従業員メールからの配信リストのCSVエクスポート

**URL**: https://support.staffbase.com/hc/ja/articles/17069180545810
**取得日**: 2026-05-02 18:46:45 JST

## 内容

親管理者
管理者
管理者または親管理者として、レポート、バックアップ、または監査のために配信リストのメンバーをCSVファイルにエクスポートする必要がある場合があります。CSVに含める属性を選択できます。たとえば、名前、場所、職位などです。
個人情報をエクスポートする際は、GDPRやその他の関連するプライバシー規制に準拠していることを確認してください。これらの法律に従って個人データの安全な取り扱いと保護に責任があります。
この機能は、従業員メールのWebアプリから直接メールを送信する場合にのみ利用可能です。アドインや拡張機能をインストールし、メールクライアントから追跡されたメールを送信している場合、この文書は関連しません。
エクスポートは、最大100,000件の連絡先を含む配信リストに対して利用可能です。
従業員メールのWebアプリで、連絡先 > 配信リストに移動します。
エクスポートしたい配信リストを選択し、三つ ドットをクリックしてアクションメニューを開きます。
エクスポートをクリックします。CSVとしてエクスポート。
CSVとしてエクスポートウィンドウが開きます。
エクスポートしたい列を選択するか、すべて選択をクリックします。
データ使用契約に同意するためのチェックボックスを選択します。
エクスポートCSVをクリックします。CSVファイルがコンピュータにダウンロードされます。
