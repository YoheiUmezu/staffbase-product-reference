# Microsoft 365 検索の有効化

**URL**: https://support.staffbase.com/hc/ja/articles/4416246788626
**取得日**: 2026-05-02 18:49:41 JST

## 内容

ベータ
管理者
Microsoft 365 検索を使用すると、ユーザーは従業員アプリやイントラネットから Microsoft プラットフォームのコンテンツを検索できます。
前提条件:
Microsoft Connections を Studio で構成している。
Studio で、設定 > サードパーティ接続に移動します。
Microsoft 365 の下で、Microsoft 365 検索のトグルを有効にします。
Microsoft 365 検索を有効化のダイアログが開きます。
有効化をクリックして、ユーザーのために有効にします。
次のステップ
Microsoft 接続に使用する Azure アプリの権限を付与する。権限を付与すると、ユーザーはウィジェット内のコンテンツを見ることができます。
Microsoft 365 ウィジェットを別のものを有効化するたびに、Azure アプリの権限を再度付与する必要があります。Staffbase は、検索とともにすべての Microsoft 365 ウィジェットを一緒に有効化し、権限を一度だけ付与することを推奨します。
