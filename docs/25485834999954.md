# 複数言語でのコンテンツタイトルの追加

**URL**: https://support.staffbase.com/hc/ja/articles/25485834999954
**取得日**: 2026-05-02 18:42:17 JST

## 内容

管理エディター
Staffbaseプラットフォームでは、ユーザーがアクセスできる複数の言語でコンテンツを作成できます。詳細については、アプリとイントラネットの多言語オプションをご覧ください。シームレスなユーザーエクスペリエンスを提供するために、複数の言語でコンテンツのタイトルを追加することもできます。このタイトルはメニューに表示されます。
多言語オプションは、組織に複数の言語が設定されている場合にのみ利用可能です。詳細はこちら。
スタジオで、既存のニュースチャンネル、ページ、またはプラグインに移動します。
設定を開くためにギアアイコンをクリックします。
言語フラグをクリックしてドロップダウンメニューを開きます。
すでに追加された言語タイトルがドロップダウンに表示されます。
別の言語を追加をクリックします。
言語を追加ダイアログが開きます。
ドロップダウンメニューから言語を選択します。
オプションで、トグルを有効にしコンテンツをコピーを選択し、どの言語からコンテンツをコピーするかを選択します。
追加をクリックします。
新しい言語タイトルフィールドが表示されます。
タイトルフィールドに、選択した言語でタイトルを入力します。
保存をクリックします。
コンテンツタイトルが更新されました。新しく追加されたタイトルは、ユーザーがアプリまたはイントラネットを追加した言語で使用している場合にメニューに表示されます。
