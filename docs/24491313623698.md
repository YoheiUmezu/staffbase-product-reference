# StaffbaseメールにおけるAIの概要

**URL**: https://support.staffbase.com/hc/ja/articles/24491313623698
**取得日**: 2026-05-02 18:42:35 JST

## 内容

アクティベーション
エディター
AI
Staffbase AIコンパニオンは、メールの件名やプレビュー文を作成・洗練することで時間を節約するのに役立ちます。ドラフトメールの内容を使用して、これらのテキストを生成します。インスピレーションを得るために、ワンドアイコンをクリックするだけです。テキストを1クリックで挿入または再生成できます。必要に応じて手動で編集することも可能です。
件名
メールの件名は、受信者がメールを受け取ったときに見る1行のテキストです。
プレビュー文
メールのプレビュー文は、件名の近くに表示され、メールの内容に関する追加の洞察を提供します。
