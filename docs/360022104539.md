# ユーザー変更履歴へのアクセス

**URL**: https://support.staffbase.com/hc/ja/articles/360022104539
**取得日**: 2026-05-02 18:44:31 JST

## 内容

ユーザー変更履歴は、管理者にユーザーアカウントの活動の概要を提供します。ユーザー変更履歴は、ユーザーアカウントに対して行われたすべての変更と、それを行った人物を追跡します。
ユーザー変更履歴は、ユーザープロフィールに対して行われた以下の変更を記録します：
ユーザーがあなたのStaffbaseプラットフォームに招待されたとき
ユーザーが初めてプラットフォームにログインし、利用規約に同意したとき
ユーザーがあなたのアプリに登録したとき
ユーザーのプロフィールフィールドに更新が行われたときと、それを行った人物
ユーザー変更履歴へのアクセス
スタジオで、ユーザー設定に移動します。
登録されたユーザーの概要が表示されます。
あなたのユーザーとグループのセクションで、次のタブのいずれかを選択します：
登録ユーザー
無効化されたユーザー
保留中のユーザー
変更履歴を表示したいユーザーに移動し、編集をクリックします。
ユーザーアカウントを編集するためのダイアログが開きます。
ユーザー変更履歴を表示をクリックします。
変更履歴が開きます。
ユーザー変更履歴から必要な情報を確認します。
OKをクリックして、変更履歴を終了します。
