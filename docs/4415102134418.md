# 従業員メールは私のメールメッセージを保存しますか？

**URL**: https://support.staffbase.com/hc/ja/articles/4415102134418
**取得日**: 2026-05-02 19:01:09 JST

## 内容

あなたのメールはプライベートなものであり、そのためそのように保持されるべきです。私たちの利用規約とプライバシーに従い、私たちはあなたのメールメッセージを記録、保存、または確認することはありません。
保存されるのは、あなたのメールメッセージの件名とメールリンクのみです（これらはあなたのBananatagアカウントで表示されます）。
ただし、Bananatagデザイナーを使用してメールを作成するユーザーは、テンプレートとドラフトが完全に暗号化されたサーバーに保存され、許可を与えた人のみがアクセスできるようになります。
