# 従業員メールのクッキーとキャッシュをブラウザーからクリアする

**URL**: https://support.staffbase.com/hc/ja/articles/4412584430354
**取得日**: 2026-05-02 19:01:28 JST

## 内容

製品名に関する注意: 2022年以降、BananatagはStaffbaseの従業員メール製品です。
トラブルシューティングの一環として、ブラウジング履歴、キャッシュされた画像とファイル、クッキーをクリアする必要がある場合があります。
このガイドでは、これを行う方法と、必要に応じてクッキーを保持する方法を示します。
Chrome
次のリンクをアドレスバーにコピーして貼り付けます:chrome://settings/clearBrowserData
表示されるウィンドウで、過去のアイテムをクリアのドロップダウンメニューがすべての時間に設定されていることを確認します。他のサイトのクッキーを保持したい場合は、クッキーと他のサイトデータを選択しないでください。
完了したら、データをクリアをクリックします。これで完了です！
Chromeでのクッキーの保持
ブラウザーのクッキーを保持したい場合、ブラウジング履歴とキャッシュファイルをクリアする際にこれを行うこともできます。
まず、上記の手順を完了します。ただし、ブラウザーデータをクリアする前にクッキーと他のサイトデータのチェックを外していることを確認してください。
クッキーページを開きます。これは、chrome://settings/siteData?search=cache をブラウザーにコピーして貼り付けることで行えます。
クッキーページに移動したら、Bananatagに関連するクッキーを探します。
ゴミ箱アイコンをクリックすることでBananatagのクッキーを削除できます。これにより、他のクッキーに影響を与えることなく、ブラウザーからBananatagのクッキーが削除されるはずです。
Firefox
次のリンクをブラウザーにコピーして貼り付けます about:preferences#privacy
クッキーとサイトデータというタイトルのセクションまでスクロールし、データをクリア...ボタンを選択します。
表示される新しいウィンドウでクリアを選択します。（他のサイトのクッキーを保持したい場合は、クッキーとサイトデータオプションのチェックを外してください）
Bananatagに関連するクッキーのみを削除するには、about:preferences#privacyに戻り、データを管理...ボタンを選択します。
新しいウィンドウで検索バーにBananatagと入力し、次に表示されたすべてを削除ボタンをクリックします:
Edge
Edgeでブラウジング履歴をクリアするには:
ブラウザーに次のリンクをコピーして貼り付けます: edge://settings/clearBrowserData?search=cache
表示されるウィンドウで、時間範囲のドロップダウンメニューがすべての時間に設定されていることを確認します（他のサイトのクッキーを保持したい場合は、クッキーと他のサイトデータを選択しないでください）:
Bananatagに関連するクッキーのみを削除するには、edge://settings/siteData?search=cookieに移動し、検索バーにBananatagと入力して表示されたすべてを削除を選択します。
