# メールのバージョン管理の概要とよくある質問

**URL**: https://support.staffbase.com/hc/ja/articles/27231694484882
**取得日**: 2026-05-02 18:53:13 JST

## 内容

Staffbaseメールは、変更の追跡と元に戻す機能を提供しており、品質管理やチームでの共同作業に不可欠です。メールのバージョン管理により、ユーザーは変更履歴を追跡し、メールの下書きやテンプレートの以前のバージョンに戻すことができます。これにより、コンテンツを編集プロセス全体で確認、復元、安全に管理できます。すべてのメール編集者は、利用可能なすべてのバージョンを表示および復元できるため、コンテンツ作成時の透明性と責任が確保されます。誤りの修正や以前のバージョンの参照など、バージョン管理によって信頼性の高い効率的なコンテンツ管理が可能です。
バージョン履歴はすべての言語バージョンで同じですか？
いいえ。バージョン履歴は、メール下書きの各言語バージョンごとに異なります。
各バージョンには何が含まれていますか？
各バージョンには以下が含まれます：
保存時点の下書きのコンテンツの状態
保存した日時
そのバージョンを保存したユーザーの名前
下書きでプロフィールフィールドのパーソナライズタグ（例：名前や姓）が使われている場合に追加されたフォールバックテキスト
メールバージョンには何が含まれますか？
メールバージョンには、その時点で保存されたメール下書きのコンテンツのみが含まれます。本文テキスト、画像、レイアウトが対象です。
メールで使用したファイルがタイトルから削除された場合はどうなりますか？
画像などの削除されたファイルは、それまで使用されていた下書きやテンプレートのバージョン履歴には表示されなくなります。つまり、古いバージョンのメールを復元しても、削除されたファイルは復元されたコンテンツに表示されません。
メールバージョンに含まれないものは何ですか？
メールバージョンには、メールタイトル、件名、プレビュー、視聴者、添付ファイルなどのメタデータは含まれません。
下書きやテンプレートの保存、作成、複製時にバージョン履歴はどうなりますか？
下書きをテンプレートとして保存、テンプレートから下書きを作成、下書きやテンプレートを複製する場合、いずれもコンテンツの最新バージョンのみが使用されます。バージョン履歴は引き継がれません。
下書きをテンプレートとして保存すると、最新バージョンのみが保存され、下書きのバージョン履歴はテンプレートに含まれません。
テンプレートから下書きを作成すると、テンプレートの最新バージョンのみが使用され、テンプレートのバージョン履歴は引き継がれません。
下書きやテンプレートを複製すると、現在のコンテンツのみがコピーされ、バージョン履歴は保存されません。
復元前にメールバージョンをプレビューできますか？
はい、ユーザーは利用可能なすべてのバージョンを復元前にプレビューできます。
現在のバージョンとは何ですか？
最新の保存済みバージョンが現在のバージョンです。コンテンツを編集中にバージョン履歴を開くと、手動で保存しなくても編集中のコンテンツが自動的に保存され、履歴の現在のバージョンとして表示されます。
復元ボタンが利用できないのはなぜですか？
復元オプションは、現在のバージョン以外をプレビューしている場合のみ有効になります。
メール下書きの複数言語バージョンを一度に復元できますか？
いいえ、下書きの1つの言語バージョンのみを一度に復元できます。
削除した下書きの言語バージョンを復元できますか？
はい、削除した言語バージョンも復元できます。
復元前に削除した言語を再追加した場合はどうなりますか？
復元したバージョンが、その言語で新たに追加したコンテンツを上書きします。
例：英語バージョンを削除後に再追加し、元の英語バージョンを復元すると、新たに追加したコンテンツは置き換えられます。
削除した言語を再追加した場合、バージョン履歴はどうなりますか？
同じ言語を再度追加した場合、バージョン履歴は最初に削除されたバージョンから継続されます。新規作成にはなりません。
復元後に新しいバージョンは自動的に保存されますか？
はい、バージョンを復元すると自動的に現在のバージョンとして保存されます。
復元に失敗した場合はどうなりますか？
復元に失敗した場合、ユーザーには復元失敗のエラーメッセージが表示されます。
コンテンツ言語から削除した言語のバージョンも復元できますか？
はい、コンテンツ言語から削除した言語でもバージョンを復元できます。
言語バージョンを完全に削除するにはどうすればよいですか？
下書きを完全に削除してください。下書きやテンプレートのUIから言語のみを削除しても、そのバージョン履歴は削除されません。
コンテンツ言語から言語を削除した場合はどうなりますか？
コンテンツ言語リストから言語を削除しても、下書きには影響しません。その言語のすべてのバージョンはバージョン履歴に残ります。
