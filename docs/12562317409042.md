# 従業員メールにおけるメールエイリアスの概要

**URL**: https://support.staffbase.com/hc/ja/articles/12562317409042
**取得日**: 2026-05-02 18:43:34 JST

## 内容

管理者
メールエイリアスを使用すると、別のメールアドレスからメールを送信することができ、メールが異なる人物や部門から送信されたように見えます。たとえば、組織の営業部門やCEOとしてメールを送信することができます。新しいメールボックスやアカウントを作成することなく、エイリアスを設定して使用することができます。
利点
メールエイリアスを使用して
主要なメールアドレスをプライベートかつ安全に保つ
特定のコミュニケーション目的や役割に使用されるアドレスで整理およびフィルタリングする。たとえば、営業やサポート情報を送信するためにエイリアスを使用する
複数のアドレスを1つの受信トレイに統合することで、メールを簡単に管理する
範囲
従業員メールでは、Webアプリまたはスタジオからメールを送信する際にメールエイリアスが利用可能です。アドインや拡張機能を使用してメールクライアントからメールを送信する場合はサポートされていません。
要件
メールアドレス
エイリアスに使用されるメールアドレスは、アカウントごとに一意でなければなりません
役割
親管理者と管理者は、自分の役割でエイリアスを作成および共有できます。
ユーザーとマネージャーは、自分に共有されたエイリアスのみを見ることができます。
エイリアスが共有され、メールを送信する権限があるユーザーは、そのエイリアスを使用してメールを送信できます。
ドメイン
エイリアスを使用してメールを正常に送信するには、エイリアスの送信ドメインがエイリアスを作成する前に構成され、確認されている必要があります。詳細については、従業員メールWebアプリでの組織のドメインの確認を参照してください。
メールエイリアスの使用
エイリアスを作成して共有し、チームで利用できるようにします
エイリアスを共有解除および無効化し、もはや必要ない場合は削除します
エイリアスの詳細とメトリクスを表示し、追跡したメールに関する柔軟で包括的な洞察を得ます
従業員メールで作成、送信、追跡
従業員メールで送信されたメールの詳細を表示
従業員メールのメトリクスを読み取り、解釈し、エイリアスと主要なメールアドレスで分けられた追跡メールに関する柔軟で包括的な洞察を得ます
既存の従業員メールエイリアスソリューション
すでに指定されたエイリアスとしてユーザーアカウントを設定している場合があります。この場合、別のユーザーアカウントを作成するのではなく、エイリアスを作成することをお勧めします。送信する相手を表す新しいエイリアスを作成したら、もはや必要ない場合は対応するユーザーアカウントを無効化してください。詳細については、従業員メールユーザーの無効化を参照してください。
送信されたメールデータは、レポートで生成され続けます。エイリアスの表示メールがユーザー（アクティブまたは無効）の表示メールと一致する場合、両方の送信メールデータはレポートで一緒に生成されます。
