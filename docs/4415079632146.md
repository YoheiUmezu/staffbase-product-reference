# 従業員メール追跡はどのように機能しますか？

**URL**: https://support.staffbase.com/hc/ja/articles/4415079632146
**取得日**: 2026-05-02 18:51:47 JST

## 内容

メールを追跡することで、送信後に受信者がメッセージとどのように対話しているかを知ることができます。従業員メールは、メールが開かれたときや、その中のリンクがクリックされたときに通知します。

メールを追跡する方法
追跡を始めるには：
組織が従業員メールをどのように使用しているかに応じて、ウェブアプリ送信（スタジオ経由でも利用可能）または通常のメールアプリケーション用のアドイン/拡張機能を設定していることを確認してください。
ウェブアプリまたはスタジオ経由で送信されたメールは自動的に追跡されます。
メールアドインまたは拡張機能を使用する場合は、送信前にメール作成ウィンドウで追跡がオンになっていることを確認してください。
Gmailでは、追跡ボックスが選択されていることを確認してから送信をクリックしてください。
Outlookでは、メールを追跡ボタンがハイライトされていることを確認してから送信をクリックしてください。
ブラウザで従業員メールアカウントにサインインして、統計を表示します！
オープンのカウント方法
従業員メールは、メールオープンを追跡するために、マスメールマーケティング会社と同様の技術を使用しています。これは、追跡されるメッセージに小さく透明な画像（またはトラッキングピクセル）を挿入することで実現されます。受信者がメールを表示すると（画像がダウンロードされると）、オープンが記録されます。
オープンに関する重要な考慮事項：
メッセージを開く人が画像を無効にしている場合、オープンは記録されません。
これは、スパマーによる悪用のために、一部のメールプログラムでデフォルトに設定されている場合があります。
受信者がプレーンテキストメールを使用している場合、オープンは記録されません。
ほとんどの人はプレーンテキストメールを使用していません（トラッキングピクセルが自動的にダウンロードされます）が、これは常にそうであるとは限らないことを覚えておくことが重要です。
追跡されたメールを開くと、受信者にクッキーを提供しようとします。このクッキーは非常に重要です。
このクッキーは、そのデバイス上でそのユーザーによる将来の再オープンが再オープンとしてカウントされ、追加のユニークオープンとしてカウントされないことを保証します。また、クッキーは、1人の受信者が短時間に複数のオープンをトリガーするのを防ぎます。
切り捨てられたメールを送信することは避けてください。メールが切り捨てられると、オープンが正確に登録されない可能性があります。メールエディタの警告に従って、メールを切り捨て制限以下に保ってください。
受信者が見るもの
追跡されたメールは、受信者にとって見た目が変わることはありません。受信者にメッセージが追跡されていることを通知する署名やその他の視覚的変更は追加されません。
唯一の違いは、メールに含まれる各リンクの先頭に追加のラッパーURLが追加されることですが、これは受信者には明らかではありません。

リンク追跡（クリック）
従業員メールでメールを追跡する際、メール内のすべてのリンクはデフォルトで自動的に追跡されます。
オプションとして、アドインまたは拡張機能からメールを送信する場合、アドインまたは拡張機能の設定でリンク追跡を無効にすることもできます。
エンゲージメントデータの正確性を確保するために、各従業員メールにはユニークなトラッキングピクセルと受信者ごとに特化したパーソナライズされたリンクが埋め込まれています。これらのメールを転送すると、トラッキングピクセルのユニーク性が損なわれ、メトリクスに潜在的な不正確さをもたらす可能性があることに注意してください。最良の結果を得るためには、データを正確かつ信頼できるものに保つために、メールを転送するのではなく、直接エンゲージメントを行うことをお勧めします。
