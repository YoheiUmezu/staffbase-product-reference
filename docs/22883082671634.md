# エディトリアルカレンダーでのクイック編集の使用

**URL**: https://support.staffbase.com/hc/ja/articles/22883082671634
**取得日**: 2026-05-02 18:51:16 JST

## 内容

エディター
クイック編集を使用すると、エディトリアルカレンダーを離れることなく、ニュース投稿の重要な情報を簡単に調整できます。
クイック編集オプションは、更新投稿には利用できません。このタイプのニュース投稿では、鉛筆アイコンをクリックするとコンテンツエディターが開きます。
クイック編集オプションを使用すると、次のことができます：
タイトルを編集する
公開日と時間を変更する
ティーザーテキストを編集する
スケジューリングを有効または無効にする
週次ビューでのクイック編集投稿
過去の公開日を持つドラフト投稿をクイック編集できます。ニュース投稿の公開日を変更してから、保存をクリックする必要があります。
スタジオで、プランニング > エディトリアルカレンダーに移動します。
編集したいニュース投稿のタイルをクリックします。
フライアウトメニューが開きます。
鉛筆アイコンをクリックします。
フライアウトが編集モードに変更されます。
ニュース投稿の詳細を編集します。
保存をクリックします。
ニュース投稿を編集しました。
