# Googleカレンダーウィジェットの設定

**URL**: https://support.staffbase.com/hc/ja/articles/4407741960850
**取得日**: 2026-05-02 18:56:00 JST

## 内容

ベータ
アクティベーション
エディター
デスクトップ用のGoogleカレンダーウィジェットを使用すると、従業員がページ上でGoogleカレンダーの会議リストを利用できるようになります。ウィジェットを設定して、従業員の個別カレンダーまたはチームカレンダーを表示できます。これにより、イントラネットが従業員のハブとなり、スケジュールを確認したり、ウィジェットから直接会議に参加したりすることができます。
前提条件
スタジオにてGoogle Workspace Connectionsを追加しています。
Googleカレンダーウィジェットをアクティブにしています。
チームカレンダーを設定する場合、そのカレンダーのカレンダーIDが必要です。
スタジオでウィジェットを追加したいページに移動します。
Googleカレンダーウィジェットを追加します。
Googleカレンダーウィジェットを設定するためのダイアログが開きます。
オプションで、タイトルを提供します。
このタイトルはウィジェットのコンテンツの上に表示されます。
モバイルアイコンの選択を解除し、デスクトップユーザーのみの表示に設定します。
アプリ内の表示の下で、ウィジェットを表示したいユーザーを選択します。
カレンダータイプの下で、個別またはチームカレンダーを表示するかを選択します。
チームカレンダーを設定する場合、カレンダーIDを提供します。
個別カレンダーにはカレンダーIDは必要ありません。
オプションで、デスクトップデバイス用にウィジェットの高さをカスタマイズします。
OKをクリックします。
ユーザーがGoogleカレンダーウィジェットを見る方法
ユーザーはウィジェットを初めて使用する際にGoogleログイン画面が表示されます。カレンダーを見る前に、ログインしてGoogleカレンダーをイントラネットに接続することに同意する必要があります。
