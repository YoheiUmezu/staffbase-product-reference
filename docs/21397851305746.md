# 代替テキストの概要とベストプラクティス

**URL**: https://support.staffbase.com/hc/ja/articles/21397851305746
**取得日**: 2026-05-02 18:49:01 JST

## 内容

代替テキスト（alt text）は、ウェブページ上の画像の外観や機能を説明します。スクリーンリーダーはこれを音声で読み上げ、視覚障害や低視力のある人々を支援します。
画像に代替テキストを使用することは、すべての従業員に対するアクセシビリティを確保するために重要です。画像に対して説明的なテキストを提供することで、代替テキストはスクリーンリーダーが重要な視覚情報を伝えることを可能にし、包括性を促進し、すべての人がコンテンツに関与できるようにします。
Staffbaseの代替テキスト機能により、編集者はニュース投稿やページの画像に代替テキストを追加できます。
代替テキスト使用のベストプラクティス
以下は、すべてのユーザーにとって役立ち、明確で包括的な効果的な代替テキストを書くためのベストプラクティスです：
簡潔でありながら説明的であること: 画像の重要な要素と目的を捉えた簡潔で意味のある説明を提供します。1～2文を目指しましょう。
文脈を考慮すること: 画像が表示される文脈に合わせて代替テキストを調整します。画像のすべての詳細ではなく、コンテンツや画像の目的にとって重要なことを説明します。
冗長なフレーズを避けること: スクリーンリーダーは画像に遭遇した際に自動的に通知します。「画像の」や「写真の」といったフレーズは不要です。
関連情報に焦点を当てること: インフォグラフィック内のテキストや写真内の特定のアクションなど、コンテンツ内で画像のメッセージや機能を伝える詳細を強調します。
適切な句読点を使用すること: スクリーンリーダーはテキストの構造を理解するために句読点に依存するため、代替テキストが文法的に正しいことを確認します。
画像内のテキストを説明すること: 画像に重要なテキスト（引用や見出しなど）が含まれている場合は、これを代替テキストに含めます。
キーワードを過剰に使用しないこと: 不要なキーワードで代替テキストを過剰に詰め込むことを避けます。明確さと可読性に焦点を当てます。
関連する場合のみ色を言及すること: 色が文脈にとって重要な場合のみ色を説明します。以下は例です：
関連性あり: 車を止める赤信号。
関連性なし: 公園を歩いている青いシャツを着た人。 - 青いシャツが文脈にとって重要でない限り。
