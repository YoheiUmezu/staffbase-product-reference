# Journeysプラグインのインストール

**URL**: https://support.staffbase.com/hc/ja/articles/360015330599
**取得日**: 2026-05-02 18:47:41 JST

## 内容

アクティベーション
システム全体のエディター
Journeysは、従業員が組織内での個々の旅をガイドすることを可能にします。カスタマイズされたコンテンツを作成し、従業員が新しいタスクをサポートするための個別のタイムラインを設定します。Journeysは、新しい従業員を組織に迎え入れるため、既存の従業員が新しいユーザーグループに適応するのを助けるため、その他多くの目的に最適です。
Journeysプラグインのインストール
スタジオで、コンテンツメニューに移動します。
メニューの一番下までスクロールし、プラグインを追加をクリックします。
Journeyに移動し、インストールをクリックします。
Journeysを追加するためのダイアログが開きます。
Journeyのタイトルを入力します。
Journeysを追加をクリックします。
これで最初のJourneyが追加されました。いつでもアプリにさらに多くのJourneysを追加できます。Journeyをユーザーに表示させるには、最初にJourneyをアクティブにする必要があります。
