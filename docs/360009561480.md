# 内部グループの作成

**URL**: https://support.staffbase.com/hc/ja/articles/360009561480
**取得日**: 2026-05-02 18:52:27 JST

## 内容

管理者
ユーザーは内部ユーザーグループに登録したり、登録を解除したりすることはできません。管理者は内部グループにユーザーを追加または削除できますが、ユーザー自身が追加または削除することはできません。これにより、内部グループは、部門特有のコンテンツやオフィスの所在地に基づく更新など、ユーザーにとって非常に関連性の高いユーザー特有のコンテンツを提供する信頼できる方法となります。内部グループを作成すると、コンテンツ作成者はこの特定のグループ向けにコンテンツの公開をすぐに開始できます。これにより、内部グループはアプリの最も強力なコンテンツターゲティングツールの一つとなります。内部ユーザーグループを作成し、関連するコンテンツをユーザーに提供しましょう。
スタジオで、ユーザー設定 > ユーザーグループに移動します。
グループを作成をクリックします。
ユーザーグループを作成するためのダイアログが開きます。
グループの名前を入力します。
ドロップダウンメニューからタイプ内部グループを選択します。
メンバーシップの下で、ドロップダウンメニューからグループのユーザー管理方法を選択します。
内部ユーザーグループの管理方法は、一度保存されると変更することはできません。
作成をクリックします。
内部ユーザーグループが作成されました。これで、グループ設定を編集し、1人以上のグループ管理者を追加できます。
