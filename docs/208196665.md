# 公開エリアとは何ですか、そしてなぜ必要ですか？

**URL**: https://support.staffbase.com/hc/ja/articles/208196665
**取得日**: 2026-05-02 18:45:17 JST

## 内容

公開エリアは、ユーザーがログインする前にアプリで表示される部分です。これは、顧客、パートナー、潜在的な従業員など、より広いターゲットグループ向けの情報を紹介するのに最適な場所です。このスペースを積極的に利用して、対外的なコミュニケーション活動を行うことができます。多くのStaffbaseの顧客は、公開エリアを使用して求人情報、プレスリリース、その他の一般向けのマーケティング資料を共有しています。
アプリをアプリストアを通じて配布する予定がある場合は、Staffbaseサポートまたはカスタマーサクセスマネージャーに連絡して、公開コンテンツの設定方法について詳しく学んでください。
