# StaffbaseへのBox接続の追加

**URL**: https://support.staffbase.com/hc/ja/articles/20497272943250
**取得日**: 2026-05-02 18:49:04 JST

## 内容

ベータ
アクティベーション
管理者
Box接続を追加して、ユーザーがStaffbaseプラットフォームからBoxコンテンツを検索およびアクセスできるようにします。
前提条件
あなたまたはBox管理者がBoxで以下の情報を生成しています：
StaffbaseがBoxインスタンスにアクセスするためのエンドポイント
クライアントID
クライアントシークレット
Studioで設定 > サードパーティ接続に移動します。
サードパーティ接続ページが開きます。
Boxの下で追加をクリックします。
Box接続を追加するためのダイアログが開きます。
Boxで生成した情報を使用して以下のフィールドに入力します。
- インスタンス名
- クライアントID
- クライアントシークレット
インスタンス名はbox.comの前にある名前です。box.comの前に名前がない場合は、「app」をインスタンス名として使用します。
ドロップダウンメニューからBox統合の可視性を選択します。
追加をクリックします。
BoxをStaffbaseに正常に接続しました。この接続ではまだBoxコンテンツにアクセスできません。ユーザーが利用できるようにするには、StudioでBox検索を有効にする必要があります。
StaffbaseでのBox検索の有効化
Studioで設定 > サードパーティ接続に移動します。
Boxの下で展開アイコンをクリックします。
Box検索のトグルを有効にします。
Box検索を有効にするためのダイアログが開きます。
アクティブ化をクリックします。
StaffbaseプラットフォームでBox検索を正常に有効にしました。これ以降、ユーザーがグローバル検索を使用すると、Box専用のタブが表示されます。Boxタブの下で、ユーザーがBoxアカウントにサインインしている場合、特定の検索用語に対するすべての検索結果がリストされます。
Box検索統合を有効にした後、初めてグローバル検索を使用する際には、ユーザーはBoxの資格情報でサインインする必要があります。
