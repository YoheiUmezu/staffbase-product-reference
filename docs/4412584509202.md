# メールデザイナーへのテキストの貼り付け

**URL**: https://support.staffbase.com/hc/ja/articles/4412584509202
**取得日**: 2026-05-02 18:47:13 JST

## 内容

Employee Email のメールデザイナーは、メールドラフトの作成、デザイン、フォーマット、レイアウトの配置に必要なすべての機能を提供します。
Staffbaseは、すべての製品内メールデザイナー機能を完全にサポートしています。
注意: Staffbaseのカスタマーエンゲージメントおよびサポートチームは、サードパーティツールやStaffbaseの合理的な管理を超えるその他の要因から生じる問題をカバーしていません。
サードパーティのテキスト編集ツールからフォーマットされたテキストをコピーして貼り付けると、メールデザイナーでのソースフォーマットの適用に制限がある場合があります。この制限は、テキストのソースや関与するフォーマットの種類によって異なります。
Staffbaseは、フォーマットなしでテキストを貼り付け、メールデザイナーで利用可能なフォーマットオプションを使用することを推奨します。
ただし、以下のサードパーティのテキスト編集製品については、Staffbaseは最善の努力をもってサポートを提供します:
Microsoft Word
Windows Notepad
Google Docs
Mac用TextEdit
最善の努力によるサポートとは、私たちのチームが特定の問題をレビューし、調査し、解決に向けて最善を尽くすことを意味します。ただし、Staffbaseは、私たちが制御できないサードパーティ製品に関連する問題の修正について完全に責任を負うことはできません。
コピーおよび貼り付け機能は、他のテキスト編集ツールからのフォーマットでも機能する場合があります。ただし、Staffbaseはそれらを積極的にサポートすることはありません。
メールデザイナーにテキストを貼り付ける際のフォーマットの削除
キーボードショートカットまたはマウスを使用して、元のフォーマットなしでメールデザイナーにテキストを貼り付けることができます。
注意: コピーしたテキストからすべてのフォーマットを削除する最も信頼性の高い方法は、最初にWindows NotepadやMac用TextEditなどのプレーンテキストエディタに貼り付け、その後プレーンテキストエディタからコピーしてメールデザイナーに貼り付けることです。
Windows
Windowsでキーボードショートカットを使用してフォーマットなしでコピー＆ペーストする方法:
サードパーティのテキストエディタで、コピーしたいテキストを選択します。
CTRL + Cを押します。
メールデザイナーのドラフトで、テキスト要素を右クリックします。
CTRL + SHIFT + Vを押します。
Windowsでマウスを使用してフォーマットなしでコピー＆ペーストする方法:
サードパーティのテキストエディタで、コピーしたいテキストを選択します。
選択したテキストを右クリックします。
コピーを選択します。
メールデザイナーのドラフトで、テキスト要素を右クリックします。
プレーンテキストとして貼り付けを選択します。
Mac
Mac OSでキーボードショートカットを使用してフォーマットなしでコピー＆ペーストする方法:
サードパーティのテキストエディタで、コピーしたいテキストを選択します。
CMD + Cを押します。
メールデザイナーのドラフトで、テキスト要素を右クリックします。
CMD + SHIFT + Vを押します。
Mac OSでマウスを使用してフォーマットなしでコピー＆ペーストする方法:
サードパーティのテキストエディタで、コピーしたいテキストを選択します。
選択したテキストを右クリックします。
コピーを選択します。
メールデザイナーのドラフトで、テキスト要素を右クリックします。
スタイルに合わせて貼り付けを選択します。
フォーマット付きのテキストをメールデザイナーに貼り付ける
キーボードショートカットまたはマウスを使用して、メールデザイナーにテキストを貼り付けることができます。

サードパーティのソースからのフォーマットの転送方法
フォーマットの種類やコピー元によって、フォーマット付きのテキストをメールデザイナーに貼り付けると異なる結果が得られます。
以下のスタイルタイプのコピー＆ペースト機能を優先します:
見出し
イタリック / 太字 / 下線
リスト
配置
注意: メールデザイナーはフォントサイズをピクセル(px)で測定します。これは、メールのフォーマットに最適です。Microsoft Wordのようなテキストエディタはフォントサイズをポイント(pt)で測定します。
これらの測定単位の違いは、貼り付けたテキストの見た目を変える可能性があります。ptでのソーステキストのサイズがpxにどのように変換されるかを予測するには、simplecssのような単位変換ツールを使用できます。
