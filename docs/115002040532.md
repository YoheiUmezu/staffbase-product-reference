# ログインロゴ

**URL**: https://support.staffbase.com/hc/ja/articles/115002040532
**取得日**: 2026-05-02 18:58:33 JST

## 内容

ログインロゴは、サインアップおよびログインフロー中に表示されます。ロゴは、サインアップまたはログインプロセスが完了するまで、画面の上部に表示されます。
ログインロゴは、灰色の背景に表示されます。
要件
幅360 px、高さ最大360 px
ファイル形式: png
解像度: 72 dpi
カラープロファイル: sRGB
LoginLogo.png
透明度: 許可されている
カラーロゴ - 灰色ではない
カスタムログインページロゴ
ログインページのロゴとその背景をカスタマイズして、組織のテーマや価値観に視覚的に合わせることもできます。
モバイルユーザー向け
モバイルデバイスでは、たとえば、異なる背景色を使用したり、組織のブランディングに合った背景画像を使用したりできます。興味がある場合は、Staffbaseサポートまたはカスタマーサクセスマネージャーにお問い合わせください。
標準の灰色背景
カスタム背景色または画像
ロゴのみ: 最大360 px x 360 px
カスタム背景画像: 550 px x 221 px
デスクトップユーザー向け
Webアプリでは、デスクトップユーザー向けのログインロゴが設定されている灰色の背景をカスタマイズできます。背景画像をスタジオを通じて設定する必要があります。この背景画像は、内部コミュニケーションプラットフォームに公開スペースがない場合にのみ表示されます。
標準の灰色背景
カスタム背景色または画像
