# スペースの復元

**URL**: https://support.staffbase.com/hc/ja/articles/360009559319
**取得日**: 2026-05-02 19:01:31 JST

## 内容

削除されたスペースは、削除後30日以内に復元できます。
スタジオで、ゴミ箱に移動します。
復元したいスペースのチェックボックスを選択します。複数のアイテムを選択することもできます。
復元をクリックします。
スペースが正常に復元されました。
スペース内のすべてのコンテンツは再びアプリユーザーに表示され、スペース管理者はスタジオで管理を続けることができます。
