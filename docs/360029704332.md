# チャット会話の作成と管理のための役割

**URL**: https://support.staffbase.com/hc/ja/articles/360029704332
**取得日**: 2026-05-02 18:52:28 JST

## 内容

チャットプラグインには、異なる権利と役割のセットによってサポートされる3つの明確なユースケースがあります。
プライベートチャット会話
ユーザーからアプリ管理者まで、すべての参加者がまったく同じ権利を持つチャット会話。
ユーザーに基づくグループチャット会話
特定のチャット会話の参加者の中で、会話管理者が他の参加者よりも多くの権利を持つチャット会話。
ユーザーグループに基づくグループチャット会話
管理者と管理編集者が強力な権利を持つチャット会話。ユーザーに基づくグループチャット会話と同様に、会話管理者が特定のチャット会話の他の参加者よりも多くの権利を持っています。

役割の詳細
ユーザー
アプリのユーザーは、他のアプリユーザーとプライベートまたはグループチャットの会話を開始することができます。
会話管理者
会話管理者は、グループ会話を管理するためのチャット特有の役割です。各グループチャット会話には、複数のチャット管理者が存在することができます。グループチャット会話の作成者は、自動的に最初の会話管理者となります。グループチャット会話の他のメンバーも会話管理者の役割を割り当てられることができます。
ユーザーグループ管理者
ユーザーグループ管理者は、ユーザーグループ特有の管理権限を持ち、それぞれのユーザーグループに基づいてチャットグループを作成することができます。これは、ユーザーグループに基づいてグループチャットを作成できる数少ない役割の一つであり、ユーザーグループのエンゲージメントを促進するための理想的な役割です。
スペース管理者
スペース管理者は、全体のスペースのコンテンツを管理する権限を持っています。しかし、チャットにおいては、特定のチャット会話の一部でない限り、権限を持ちません。
システム全体のエディター
システム全体のエディターは、チャット会話に関して権限を持ちませんが、チャットの一部である場合は例外です。
管理エディター
管理エディターは、内部またはオープンユーザーグループにアクセスできませんが、ユーザーグループに基づいてチャットグループを作成することができます。
管理者
管理者は、システム全体の管理権限を持ち、内部およびオープンユーザーグループにアクセスできます。したがって、管理者はユーザーグループに基づいてチャットグループを作成することができます。
アプリ管理者は、チャットプラグインの設定で新しい会話を開始できるユーザーを定義できます。デフォルトでは、すべてのユーザーがチャットを作成または参加する権利を持っています。特定のユーザーまたはユーザーグループのみがチャットを作成し、参加できるようにすることも可能です。その場合、チャット会話を開始することが制限されているユーザーは、チャット機能を表示できません。
