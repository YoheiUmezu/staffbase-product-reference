# クイズプラグインのインストール

**URL**: https://support.staffbase.com/hc/ja/articles/26049680779538
**取得日**: 2026-05-02 18:45:59 JST

## 内容

この機能はアドオンです。詳細はStaffbaseの価格をご参照ください。有効化についてはStaffbaseサポートまたはカスタマーサクセスマネージャーにお問い合わせください。
クイズプラグインは、Staffbase Employee Experience (EX) Boosterプラグインの1つです。
フレンドリーなコンペティションを実施する場合やeラーニングコンテンツを提供する場合でも、クイズプラグインは参加を促す効果的な方法です。
前提条件
APIトークンを作成し、読み取り専用アクセス権を付与していること。
APIトークンを暗号化してsupport@staffbase.comに送信していること。
APIトークンは、ユーザーAPIを通じてユーザープロフィール情報にアクセスするために必要です。APIトークンにより、プラグインはプロフィール画像などのユーザープロフィール情報にアクセスできます。
Studioで、コンテンツ > コンテンツメニューに移動します。
メニューの一番下までスクロールし、プラグインを追加をクリックします。
プラグインページが開きます。
クイズに移動します。
インストールをクリックします。
クイズを追加ダイアログが開きます。
クイズページのタイトルを入力します。
タイトルはプラグインの設定でいつでも変更できます。
表示ドロップダウンメニューから、クイズページを表示するユーザーを選択します。
クイズを追加をクリックします。
クイズプラグインをインストールし、クイズページを追加しました。プラグインは左側のコンテンツセクションに表示されます。ページを設定して公開することで、ユーザーに表示できるようになります。
