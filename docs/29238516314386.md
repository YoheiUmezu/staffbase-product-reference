# 多言語アンケート質問の作成

**URL**: https://support.staffbase.com/hc/ja/articles/29238516314386
**取得日**: 2026-05-02 18:50:57 JST

## 内容

この記事はAIを使用して翻訳されており、不正確な情報が含まれている可能性があります。情報に不安がある場合は、正確性のために英語のオリジナルバージョンを参照してください。
編集者
従業員アプリやイントラネットでコンテンツを複数の言語で作成できます。これにより、読者は選択したコンテンツ言語でコンテンツを読むことができます。アンケートに多言語コンテンツを追加する際には、組織で有効になっている言語を選択できます。追加の言語が必要な場合は、管理者がそれらを有効にすることができます。
複数の言語でコンテンツを作成するには、組織でこの設定がされている必要があります。詳細はこちら。このオプションが必要な場合は、管理者に連絡してください。
アンケートで多言語コンテンツを作成する方法は2つあります。
ゼロから始める
このオプションでは、翻訳を手動で追加し、アンケートの質問に選択した言語で新しいタブを作成します。テキストフィールドは空のままで、新しい言語でコンテンツを直接書いたり、コピー＆ペーストしたりできます。
コンテンツをコピーする
このオプションでは、アンケートの質問に選択した言語で新しいタブを作成し、元の言語のコンテンツをテキストフィールドにコピーします。
前提条件
アンケートを作成し、質問を追加しました。
アンケートは下書きであるか、公開予定です。
多言語の質問を手動で追加する
このオプションでは、アンケートの質問の上部に新しいタブを作成し、新しい言語でコンテンツを手動で追加できます。
Studioで、言語を追加したいアンケートの質問に移動します。
+ ボタンをクリックします。
言語を追加する ダイアログが開きます。
ドロップダウンメニューから1つ以上の言語を選択します。
適用する をクリックします。
ゼロから始める を選択します。
追加する をクリックします。
アンケートの質問に1つ以上の新しい言語を追加しました。各言語はアンケートの質問の上部に異なるタブがあります。
コピーによる多言語の質問を追加する
このオプションでは、アンケートの質問の上部に新しいタブを作成し、コピーする言語の既存のコンテンツを含めます。
Studioで、言語を追加したいアンケートの質問に移動します。
言語を追加したいニュース投稿を開きます。
+ ボタンをクリックします。
言語を追加する ダイアログが開きます。
ドロップダウンメニューから1つ以上の言語を選択します。
適用する をクリックします。
コンテンツをコピーする を選択します。
コピー元のコンテンツ ドロップダウンメニューが表示されます。
コンテンツをコピーする言語を選択します。
追加する をクリックします。
アンケートの質問に1つ以上の新しい言語を追加しました。各言語はアンケートの質問の上部に異なるタブがあり、選択したソース言語からコンテンツがコピーされています。
多言語コンテンツの削除
不要になった場合や誤って追加した場合は、個別に言語バージョンを削除できます。
Studioで、言語を削除したいアンケートに移動します。
アンケートの質問の横にある3つのドットをクリックして、アクションメニューを開きます。
編集する をクリックします。
ゴミ箱アイコンをクリックします。
言語を削除する ダイアログが開きます。
ドロップダウンメニューから1つ以上の言語を選択します。
適用する をクリックします。
削除する をクリックします。
保存する をクリックします。
選択した言語バージョンのアンケートの質問が完全に削除されました。言語バージョンはいつでも再追加できます。
