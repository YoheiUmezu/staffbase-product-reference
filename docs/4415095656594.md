# トラッキングされたメールの不明なIPロケーションのトラブルシューティング

**URL**: https://support.staffbase.com/hc/ja/articles/4415095656594
**取得日**: 2026-05-02 18:47:07 JST

## 内容

トラッキングされたメールを送信した後に受信する不明なロケーションデータが表示される理由はいくつかあります。
公開されているIPロケーション情報は、オープンを特定し、該当するオープンのロケーションデータを提供するために必要です。
モバイルデバイスの場合、このデータは最寄りのセルタワーから取得されます。一般的に、モバイルオープンのIPロケーションに関して問題は発生しません。
デスクトップデバイスの場合、表示される/ジオロケートされたIPアドレスは、あなたのローカルネットワークに基づいています。不明なロケーションの場合、このデータはおそらく、隠されたIPアドレスを持つメールサーバーから来ています。
これはあなたの会社内の内部機能であり、メール管理チームと調整の可能性について話し合うことができます。
