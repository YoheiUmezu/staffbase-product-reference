# iOSアプリIDの作成

**URL**: https://support.staffbase.com/hc/ja/articles/115003535352
**取得日**: 2026-05-02 18:48:04 JST

## 内容

アプリIDの作成
Apple Developerアカウントにサインインし、証明書、ID、およびプロファイル > 識別子 > アプリIDに移動します。
新しいアプリIDを追加します。
名前を入力します。この名前はエンドユーザーには表示されません。
明示的アプリIDを有効にします。
バンドルIDを入力します。ベストプラクティスは、外部ドメイン名を逆にしてアプリ名を付けることです。例えばcom.yourcompany.appのようにします。
アプリサービスのセクションで、デフォルトの設定をそのままにします。プッシュ通知を有効にします。
続行をクリックします。
入力したデータの概要が表示されます。
データを確認し、送信をクリックします。
