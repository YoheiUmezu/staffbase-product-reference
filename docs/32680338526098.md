# Staffbase Live用の追加イベントを作成する

**URL**: https://support.staffbase.com/hc/ja/articles/32680338526098
**取得日**: 2026-05-02 18:48:28 JST

## 内容

アクティベーション
編集責任者
Staffbase Liveを使用して、セッションをリアルタイムで配信するイベントを作成できます。イベントの設定時に、事前にスケジュールを設定したり、モデレーターをイベントのスピーカーやホストとして割り当てたり、どのユーザーやユーザーグループが視聴者として参加できるかを指定できます。
Staffbase Liveは最大100,000人の視聴者に対応しています。20,000人を超える視聴者が参加するイベントを計画している場合は、事前にStaffbaseサポートまたはカスタマーサクセスマネージャーまでご連絡ください。
前提条件
Events Centerプラグインをインストールしていること。
イベント情報を追加する
Studioでコンテンツ > Events Centerに移動します。
イベントを作成するをクリックします。
スペースのドロップダウンで、プラグインを追加したいスペースを選択します。
タイトル欄に、ライブ配信したいイベントのタイトルを入力します。
短い説明欄に、イベントの簡単な説明を入力します。
イベントをスケジュールするために、次の情報を選択します：
開始日
開始時刻
終了日
終了時刻
タイムゾーン
次をクリックします。
イベント情報が追加されました。次に設定を構成できます。
設定を構成する
設定の下で、イベントタイプはデフォルトでライブ配信に設定されています。
視聴者を選択するの下で、ユーザーグループまたはユーザーのドロップダウンを使用して、イベントに参加できる対象を選択します。
リマインダーを設定の下で、メールでを選択して、ライブ配信の1時間前にメールリマインダーを送信します。
モデレーターを選択するの下で、イベントで話すまたはホストする必要がある管理者を選択します。
モデレーターは、Staffbase Liveに関連するすべてのタスクを実行するために管理者ロールが必要です。
作成するをクリックします。
イベントが作成されました。モデレーターと視聴者には、イベントを閲覧またはモデレートするためのロール別リンクとカレンダー招待が記載されたメールが届きます。たとえば、Googleカレンダーのユーザーには、イベントをカレンダーに追加するよう促すメールが届きます。
イベント作成者にはカレンダー招待は送信されません。
