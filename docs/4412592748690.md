# メールデザイナーでのバージョン履歴の使用

**URL**: https://support.staffbase.com/hc/ja/articles/4412592748690
**取得日**: 2026-05-02 18:57:08 JST

## 内容

エンタープライズプランが必要です。詳細については、Staffbaseサポートまたはカスタマーサクセスマネージャーにお問い合わせください。
バージョン履歴を使用すると、チームと共同で従業員メールのドラフトやテンプレートを作成し、デザインに誰が参加しているかを確認し、変更を追跡できます。

バージョン履歴の仕組み
ドラフトやテンプレートに取り組んでいるチームがありますか？メールデザインの承認プロセスはありますか？メールデザイナーのコラボレーション機能を使用すると、進捗を簡単に共有し、一緒に作業できます。
コラボレーションにより、誰があなたのメールドラフト/テンプレートに取り組んでいるかを正確に確認できます。また、行われたすべての変更を追跡し、必要に応じて以前のバージョンに戻すこともできます。
バージョン履歴の使用
バージョン履歴を使用すると、変更が行われた正確な日時と、誰が変更を行ったかを確認し、以前のバージョンに戻すことができます。
以前のバージョンを表示
バージョン履歴を表示するには：
メールデザイナーウィンドウで、保存日時のタイムスタンプをクリックします。
表示したいバージョンをクリックします。
以前のバージョンに戻す
意図せずに変更を加えてしまいましたか？以前のバージョンに簡単に戻すことができます。
メールデザイナーウィンドウで、保存日時のタイムスタンプをクリックします。
表示したいバージョンをクリックします。
このバージョンを復元をクリックします。
以前のバージョンが自動的に復元されます。
