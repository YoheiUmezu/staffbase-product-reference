# 分析ダッシュボードにおけるエクスポート可能なデータの概要

**URL**: https://support.staffbase.com/hc/ja/articles/20578858349586
**取得日**: 2026-05-02 19:01:54 JST

## 内容

分析ダッシュボードは貴重な指標を提供します。エクスポートされたデータを使用して、編集者や著者と共有するレポートを作成し、コンテンツの成功を評価できます。管理者はフィルターを使用して、組織のさまざまなメンバー向けに異なるデータセットを作成できます。これらのフィルターにより、ユーザーインターフェイスで現在表示されているデータのみを抽出できます。
エクスポート生成の制限により、CSVエクスポートは1,000行のデータに制限されています。
