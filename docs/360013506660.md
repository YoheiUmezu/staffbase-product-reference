# モバイルデバイス向けクイックリンクナビゲーションの概要

**URL**: https://support.staffbase.com/hc/ja/articles/360013506660
**取得日**: 2026-05-02 18:59:54 JST

## 内容

アクティベーション
システム全体のエディター
モバイルデバイスでは、クイックリンクを使用することで、プラットフォーム内のコンテンツに簡単かつ直接にアクセスできます。重要なページや最も閲覧されているニュースチャンネルの隣に、ナビゲーションバーはプラグインや短期間に関連するコンテンツを表示するのに最適な場所でもあります。
コンテンツメニューの任意の第一レベルメニュー項目をクイックリンクナビゲーションバーに追加します。
さらに、次の機能をクイックリンクとして追加します：
検索
ランチパッド
任意のメニュー項目をクイックリンクナビゲーションバーに追加できます。これには特定のページも含まれます。この機能をアクティブにする前に、非第一レベルメニュー項目がすでにクイックリンクとして設定されていないことを確認してください。アクティブにすると、これらの項目はモバイルナビゲーションバーに自動的に表示されます。
2025.1より古いアプリバージョンのユーザーは、クイックリンクに加えた変更を確認できません。完全な機能を確保するために、最新のアプリバージョンに更新してください。詳細については、Staffbaseサポートまたはカスタマーサクセスマネージャーにお問い合わせください。
モバイルデバイスでのクイックリンクナビゲーションバーの外観
クイックリンクの最大数
ナビゲーションバーには最大5つのクイックリンクが表示されます。5つ未満のリンクも許可されており、1つのクイックリンクだけでも可能です。
5つ以上のメニュー項目がクイックリンクとして設定されている場合、最初の5つのみがナビゲーションバーに表示されます。
ナビゲーションバーにおけるクイックリンクの可視性はユーザーの権限に基づいています。ユーザーは、自分がアクセスできるクイックリンクのみを確認できます。たとえば、特定の場所に基づいてユーザーグループに表示される場所特有のページがある場合、各ユーザーはそれぞれの場所に対するクイックリンクのみを確認します。
ナビゲーションバーに表示されるクイックリンクの数は、コンテンツの可視性設定に応じてユーザーごとに異なる場合があります。ただし、クイックリンクの最大数は5つです。ユーザーがナビゲーションバーにリンクされているコンテンツにアクセスできない場合、そのユーザーにはナビゲーションバーは表示されません。
モバイルデバイス向けに再設計されたナビゲーションバーを使用する場合、クイックリンクの最大数は4つに制限されます。 詳細を学ぶ。
クイックリンクの並べ替え
クイックリンクに位置を追加しない場合、それらはコンテンツメニューの順序で並べ替えられます。
位置を指定すると、クイックリンクの順序に影響を与えます。1が最初の位置です。位置番号は1から5だけでなく、1から100まで指定できます。これにより、多くのコンテンツと異なるターゲットグループを持つ組織での細かい並べ替えが可能になります。
位置を持たないクイックリンクは、位置を持つものの前に表示されます。位置を持たないクイックリンクの順序は、コンテンツメニューの順序によって決まります。位置を持つクイックリンクは、その指定された位置によって並べ替えられて表示されます。
スペース用のクイックリンク
スペースを使用する場合、クイックリンクの設定にはプラットフォーム全体のグループとスペースの設定に関する非常に良い知識が必要です。Staffbaseは、全従業員スペースのメニュー項目に対してのみクイックリンクを使用することを推奨します。
複数のスペースに対してクイックリンクを追加する場合、それらはスペースの順序で並べ替えられます。各スペースは位置番号を1から再スタートします。異なるスペースからの最初の5つのクイックリンクのみが表示されます。管理スペースビューで下にあるスペースのクイックリンクは表示されない場合があります。
