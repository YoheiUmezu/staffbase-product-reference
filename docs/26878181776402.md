# スクリーンの役割

**URL**: https://support.staffbase.com/hc/ja/articles/26878181776402
**取得日**: 2026-05-02 18:45:51 JST

## 内容

この機能はアドオンです。詳細はStaffbaseの価格をご参照ください。有効化についてはStaffbaseサポートまたはカスタマーサクセスマネージャーにお問い合わせください。
スライドショーの作成、編集、公開の権限や、デバイスの追加や削除ができるユーザーの割り当てなど、Staffbase Screensでスライドショーの設定やデバイス管理に関するユーザーの役割と対応する権限について、この概要でご案内します。
スライドショーの設定とデバイス管理に関する権限と役割の概要表
アプリやイントラネットで権限や役割を設定するのは難しくありません。この概要表は、デジタルサイネージの役割設定に役立ちます。

役割の詳細
ユーザー
ユーザーはスライドショーやデバイスの追加、編集、削除はできません。
スペース管理者
スペース管理者はスライドショーやデバイスの追加、編集、削除はできません。
システム全体の編集者
システム全体の編集者はスライドショーやデバイスへのアクセスと編集ができます。システム全体の編集者はスライドショーやデバイスの追加や削除はできません。
編集責任者
編集責任者はScreensの管理に関するすべての権限を持っています。編集責任者はスライドショーやデバイスの追加、編集、削除ができます。
管理者
管理者はシステム全体の管理権限を持ち、Screensの管理も含まれます。
Staffbaseでは、管理者はユーザー管理やプラットフォームの設定にもアクセスできるため、コンテンツ管理にはこの役割の使用を推奨していません。
