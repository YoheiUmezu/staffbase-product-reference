# Staffbaseメール内のバウンスマネジメントの概要

**URL**: https://support.staffbase.com/hc/ja/articles/29333991344786
**取得日**: 2026-05-02 18:41:16 JST

## 内容

管理者
バウンスマネジメントとモニタリングは、配信に失敗したメールの状況を把握し、繰り返し発生する問題を特定するための重要な機能です。配信されなかったメール（バウンス）は、受信者側のさまざまな理由で発生することがあります。
ワークフローを効率的に維持するために、バウンスマネジメントページにはハードバウンスのみがリストされます。ハードバウンスは、無効なメールアドレスの修正など、手動での対応が必要な恒久的な配信失敗です。ソフトバウンス（受信トレイがいっぱいなどの一時的な問題）はこのリストには含まれません。システムが自動で再送信を試みるため、あなたによる即時のアクションは不要です。
これらのハードバウンスを管理することで、送信者としての評価を守り、メールが正しい受信者に確実に届くようになり、スパムフォルダに入るリスクを最小限に抑えることができます。
バウンスマネジメントリスト
受信者からのバウンスメールが登録されると、その連絡先がバウンスマネジメントリストに追加されます。バウンスマネジメントリストにある連絡先には、Staffbaseプラットフォームから今後メールが配信されません。管理者はリストを確認し、バウンスされた連絡先を更新、削除、または維持するかを決定できます。連絡先の削除について詳しくは、Staffbaseメールでのバウンス管理をご覧ください。
Staffbaseメールをイントラネット や従業員アプリと併用している場合、バウンスマネジメントリストにある連絡先には、プラットフォームに関連するすべてのメール通知が停止されます。リストから削除されると、連絡先は再びすべてのメールを受信します。
バウンスモニタリング
Studioのユーザー > バウンスマネジメントで、すべてのバウンスされた連絡先の概要を確認できます。バウンスマネジメントリストには以下が含まれます：
名
姓
メール
バウンス日（日付と時刻）
エラー（バウンスの理由）
ハードバウンスのエラータイプ
バウンスの理由は、次のいずれかのエラーとして表示されます：
エラー
説明
無効なメール
メールアドレスが無効、または存在しません。
配信に失敗しました
不明なエラーによりメールを配信できませんでした。
スパムとしてマークされています
受信者がこのメールをスパムとしてマークしました。
送信の停止
このメールアドレスで以前に問題が発生したため、送信がブロックされました。
