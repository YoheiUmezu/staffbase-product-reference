# スタジオへのアクセス

**URL**: https://support.staffbase.com/hc/ja/articles/206435179
**取得日**: 2026-05-02 18:48:14 JST

## 内容

この記事の情報は、従業員アプリまたはStaffbaseイントラネットを使用している場合、かつ従業員アプリまたはStaffbaseイントラネットと一緒にStaffbaseメールを使用している場合に適用されます。Staffbaseメールをスタンドアロン製品として使用している場合は、Staffbaseメールのためのスタジオへのアクセスを参照してください。
アカウントを登録した後、スタジオユーザーであれば、ウェブアプリからスタジオにアクセスできます。
ウェブブラウザーにアプリまたはイントラネットのドメインURLを入力します。例えば、https://mycompany.com
サインインページが開きます。
組織の認証方法（SSOや資格情報など）を使用してサインインします。
アプリまたはイントラネットが開きます。
プロフィールメニューに移動し、スタジオをクリックします。
スタジオが開きます。
スタジオに正常にアクセスしました。表示および使用できる機能や機能性は、あなたの権利と役割に依存します。
