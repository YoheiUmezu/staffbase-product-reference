# CSVインポートによるユーザー管理の自動化

**URL**: https://support.staffbase.com/hc/ja/articles/4402308161426
**取得日**: 2026-05-02 18:47:30 JST

## 内容

CSVファイルのインポートは、Staffbaseプラットフォーム上でユーザーデータを簡単に最新の状態に保つのに役立ちます。Staffbaseは手動および自動のCSVインポートの両方を提供しています。手動および自動CSVインポートは同じ機能と能力を持っていますが、自動CSVインポートでは自動データ同期のスケジュールを設定でき、頻繁な手動作業の負担を軽減できます。ユーザーベースが頻繁に変わる場合、自動CSVインポートは理想的です。そのような場合、週単位または日単位での手動ユーザーインポートはもはや実行可能ではありません。
スタジオで実行するすべてのCSVインポートアクションをスケジュールに従って自動化できます。これには、完全なインポートと部分的CSVインポートの両方が含まれます。プロセスを自動化するための最初のステップとして、ITチームはStaffbaseエンドポイントを使用してCSVファイルをStaffbase環境にアップロードする必要があります。その後、CSVファイルの属性をスタジオフィールドにマッピングし、Staffbaseエンドポイントを使用してスクリプトを更新できます。ITチームがスクリプトをテストした後、ビジネスニーズに合ったスケジュールでスクリプトを実行するように設定できます。
Staffbaseは、実際のインポートの前にテスト実行を推奨します。テスト実行により、行おうとしている変更をプレビューでき、ユーザーデータが意図しない上書きや重複から保護されます。適切なメンテナンスのために、自動CSV統合が望ましい通りに機能していることを確認するために、定期的にテストを行ってください。
