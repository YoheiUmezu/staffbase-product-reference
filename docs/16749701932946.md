# 2024年2月時点のGoogleおよびYahooによるメール送信者要件

**URL**: https://support.staffbase.com/hc/ja/articles/16749701932946
**取得日**: 2026-05-02 18:43:23 JST

## 内容

GoogleとYahooは、2024年2月から、1日あたり5,000通以上の商業メールを送信する送信者に対して新しい要件を発表しました。これらの要件を満たさない場合、メールが遅延したり、ブロックされたり、スパムに振り分けられたりする可能性があります。 この変更は、プライベートなGoogleおよびYahooアカウントに送信されるすべてのメールに影響します。Google Workspaceアカウントには影響しません。詳細については、GoogleおよびYahooの記事をご覧ください。
この変更は、主に以下のメール送信者グループに影響します：
1日に5,000以上のGmailまたはYahooのメールアドレスにメールを送信するバルク送信者。
送信するメールの数や送信先のドメインに関係なく、Gmailのメールアドレスから任意のメールサービスプロバイダーを通じてメールを送信する人。
送信者として知っておくべきことは何ですか？
すべての送信者が知っておくべきことは以下の通りです：
認証
ドメインのためにSPF（Sender Policy Framework）またはDKIM（Domain Key Identified Mail）、およびDMARC（Domain-based Message Authentication, Reporting and Conformance）のメール認証を設定する必要があります。
スパム率
スパム率を低く保つために、必要なメールのみを送信してください。
購読
1日あたり5,000通以上のメッセージを送信する場合、受信者のためにワンクリックでの簡単な退会を有効にする必要があります。
受信者にとっての変更点は何ですか？
StaffbaseからプライベートなGoogleおよびYahooアドレスに送信されるすべてのバルクメール（通知やニュースレターなど）には、ワンクリックの退会オプションが含まれています。これは、アプリ、イントラネット、従業員メールから送信されるすべてのメールに適用されます。
受信者が退会オプションをクリックすると：
従業員メールの場合：そのメールアドレスはメーリングリストから削除されます。
アプリまたはイントラネットの場合：そのメールアドレスの通知設定が無効になります。
