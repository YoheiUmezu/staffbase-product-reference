# Staffbaseメールのリンクスタイルのカスタマイズ

**URL**: https://support.staffbase.com/hc/ja/articles/24344013106450
**取得日**: 2026-05-02 18:58:48 JST

## 内容

管理者
スタジオで、設定 > メールに移動します。
メールブランディングページが開きます。
リンクの下で、テキストカラーのドロップダウンをクリックします。
ドロップダウンから、ブランドカラーを選択するか、カスタムカラーを設定します。
フォントスタイルのドロップダウンから、次のスタイルのいずれかを選択します：
太字
斜体
下線
なし
保存をクリックします。
リンクスタイルをカスタマイズしました。いつでも変更できます。
