# 検索可能なプロファイルフィールドの設定

**URL**: https://support.staffbase.com/hc/ja/articles/360010033180
**取得日**: 2026-05-02 18:52:24 JST

## 内容

価格
管理者
プロファイルフィールドの検索可能性を設定することで、アプリ内のグローバル検索結果に寄与するプロファイルフィールドを決定できます。たとえば、名と部署の両方が検索可能なプロファイルフィールドであり、ユーザーが「mar」で検索を開始した場合、マーケティング部門に所属するユーザーや、Markのようにこれらの文字で始まる名前を持つユーザーが検索結果に表示されます。
スタジオで、設定 > プロファイルフィールドに移動します。
アプリで検索可能にしたいプロファイルフィールドの隣にあるギアアイコンをクリックします。
プロファイルフィールドを編集するためのダイアログが開きます。
トグル検索用にアクセス可能にするを調整して、プロファイルフィールドをアプリで検索可能にするかどうかを定義します。
OKをクリックします。
検索可能な設定が有効になると、ユーザーはこのプロファイルフィールドから特定の名前や場所などの入力を検索できるようになります。
