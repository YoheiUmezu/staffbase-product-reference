# 従業員メールからエクスポートされた .CSV ファイルにおける特殊文字の表示

**URL**: https://support.staffbase.com/hc/ja/articles/4415095659922
**取得日**: 2026-05-02 18:59:42 JST

## 内容

従業員メールがデータをエクスポートする際に出力する .CSV ファイルは UTF-8 形式です。しかし、デフォルトでは Microsoft Excel は ANSI エンコーディングを使用するため、中国語の文字やログogram などの多くの特殊文字が正しく表示されません。
この問題を解決するには、Excel 内の設定を変更してください。
Excel で データ > 外部データのインポート > データのインポート/テキストからインポート に移動し、65001 Unicode (UTF-8) を選択します。
設定を保存し、ファイルを再度開いて特殊文字が正しく表示されることを確認します。
