# コンテンツデザイナーにおけるアクセシビリティの概要

**URL**: https://support.staffbase.com/hc/ja/articles/32702338837778
**取得日**: 2026-05-02 19:02:03 JST

## 内容

アクティベーション
編集者
コンテンツデザイナーのアクセシビリティ機能は、作成者と従業員の両方が、さまざまなアクセシビリティニーズやデバイスで公開されたコンテンツをナビゲートし、理解し、操作できるようにします。この概要では、従業員アプリとイントラネット全体で、包括的でユーザーフレンドリーなコンテンツ作成をサポートする主なツールと実践方法を紹介します。
コンテンツデザイナーは、ナビゲーション、読みやすさ、全体的なアクセシビリティを向上させる支援ツールを提供します。
操作性：キーボードによる編集とナビゲーション
コンテンツデザイナーはキーボードのみでの操作をサポートしており、ユーザーはマウスを使わずにコンテンツを作成・編集できます。これにより、運動障害のあるユーザーのアクセシビリティが向上し、キーボードユーザーの効率的な作業もサポートされます。
キーボードショートカット：コンテンツデザイナーで利用可能なすべてのキーボードショートカットの概要を表示します。
クイックアクションメニュー：キーボード（Cmd/Ctrl + Shift + K）でブロックを挿入できます。
コピー＆ペースト：ドラッグ＆ドロップを使わずに、ブロックやファイルを効率的に再利用できます。
キーボードナビゲーション：キーボードでセクション、コンテナ、ブロック間をナビゲートできます。
知覚性：画像の代替テキスト
編集者
代替テキストを設定することで、スクリーンリーダーを利用するユーザーを含むすべてのユーザーが画像にアクセスできるようになります。
操作性：コンテキストツールバーとフォーカス編集
編集者
コンテキストツールバーは、現在選択されているセクション、コンテナ、またはブロックに関連する編集オプションのみを表示します。
知覚性：色のコントラストと読みやすさ
編集者
テキストと背景の十分なコントラストを確保するために、編集者は以下のことができます：
テキストの色を調整する
セクションやコンテナの背景画像にカラーオーバーレイを適用する
理解しやすさと堅牢性：セマンティックな見出し構造
編集者
タイトルブロックはページの主見出しを定義し、異なるサイズの見出しと組み合わせて使用することで、明確なセマンティック階層を確立します。
