# Staffbaseメールのセカンダリフォントの設定

**URL**: https://support.staffbase.com/hc/ja/articles/24343544319250
**取得日**: 2026-05-02 18:48:55 JST

## 内容

管理者
セカンダリフォントは、あなたのお気に入りのフォントの2番目の選択肢です。これは、メールブランディングページのセカンダリラベルでマークされ、メールデザイナーでメールをデザインする際のフォントドロップダウンメニューの2番目のオプションとして表示されます。
スタジオで、設定 > メールに移動します。
メールブランディングページが開きます。
表示フォントの下で、セカンダリフォントとして設定したいフォントを見つけます。
さらにオプションアイコンをクリックしてドロップダウンを開き、セカンダリとして設定を選択します。
選択したフォントがハイライトされ、セカンダリラベルが設定されます。
保存をクリックします。
セカンダリフォントの設定が成功しました。いつでも変更できます。
