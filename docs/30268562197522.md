# メニュー内のフォルダをページに置き換える

**URL**: https://support.staffbase.com/hc/ja/articles/30268562197522
**取得日**: 2026-05-02 18:41:07 JST

## 内容

この記事はAIを使用して翻訳されており、不正確な情報が含まれている可能性があります。情報に不安がある場合は、正確性のために英語の原本を参照してください。
ベータ
アクティベーション
システム全体の編集者
サブメニュー項目を含むフォルダをページに置き換えることができます。これにより、メニュー構造がスリムでナビゲートしやすくなります。
Studioで、コンテンツ > メニューに移動します。
組織で利用可能な場合、スペース選択ツールでメニュー変更を適用したいスペースを選択します。
置き換えたいフォルダの横にある置換アイコンをクリックします。
フォルダをページに置き換えるダイアログが開きます。
ページドロップダウンメニューから、フォルダと置き換えたいページを選択します。
ドロップダウンには、サブメニュー項目や上位メニュー項目がなく、同じメニューに属するページのみが表示されます。
置換をクリックします。
フォルダをページに置き換えました。フォルダのサブメニュー項目はページに割り当てられ、フォルダは削除されました。該当ページのナビゲーションウィジェットは、フォルダIDをページIDに置き換えることで自動的に更新されます。
