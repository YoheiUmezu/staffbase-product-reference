# Android機能グラフィック

**URL**: https://support.staffbase.com/hc/ja/articles/115002040712
**取得日**: 2026-05-02 18:56:18 JST

## 内容

Android機能グラフィックは、アプリの本質を視覚的に要約したGoogleのプロモーション機能です。この機能グラフィックはAndroidデバイス上で表示され、さまざまなメッセンジャーやソーシャルメディアプラットフォームでアプリのプレビュー画像としても機能します。ただし、コンピュータのブラウザ版では表示されません。Android機能グラフィックは、Google Playストアを通じてアプリを配布するためにアップロードする必須のグラフィックの1つです。
要件
サイズ: 1024 x 500 px
カラープロファイル: sRGB
ファイル形式: png
解像度: 72 dpi
ファイル名: AndroidFeatureGraphic.png
透明度: 許可されていません（アルファチャンネルを含めてはいけません）
カラー画像
角がある長方形
