# メールデザインに記事投稿を追加する

**URL**: https://support.staffbase.com/hc/ja/articles/4412584520594
**取得日**: 2026-05-02 18:58:15 JST

## 内容

製品名に関する注意: 2022年以降、BananatagはStaffbaseの従業員メール製品です。
組織がStaffbase アプリ/イントラネットと 従業員メールを併用している場合、Staffbaseプラットフォームの組織のアーティクルチャンネルから既存の投稿をシームレスに引き出して、メールデザイナーでコンテンツを作成できます。
ニュースクイックブロックは、受信者がメールのアーティクル投稿と対話するために使用する画像、テキスト、ハイパーリンク要素を生成します。

メールデザインに記事投稿を追加する方法
アプリ/イントラネットと従業員メールアカウントの接続が完了すると、すぐにニュースクイックブロックを使用できるようになります！
メールデザイナーで新しいドラフトまたはテンプレートを読み込み、ドラッグアンドドロップエディタを使用してクイックブロックをドラフトに引き込んでください。
追加すると、メールデザイナーの右側に記事投稿が表示されます。
その後、記事をデザインにドラッグするか、記事をクリックして最新の空のニュースクイックブロックスペースに追加できます。
右側の画面で記事にマウスをホバーさせると、各記事がどのチャンネルから来ているかを詳しく知ることができます。
追加されると、記事のタイトル、主要画像、ティーザーテキスト、および元の記事投稿へのリンクがドラフトにコピーされます。他のメールデザイナー要素と同様に、このコンテンツは必要に応じて編集できます。
メール内のリンクは、デフォルトで受信者をイントラネット内の元の記事投稿に転送しますが、必要に応じて調整できます。
メールデザイナーで設定したブランド設定は、メールにコピーされた記事コンテンツに適用されます。
組織が記事内でオンデマンド翻訳を使用している場合、コンテンツを引き出す際に言語を選択することもできます。これは、記事をメールデザインに追加する前に言語の略称をクリックすることで行えます。
記事チャンネルは、組織内の特定のグループにのみターゲットされる場合がありますのでご注意ください。
受信者の体験を最適化するために、メールの受信者がリンクされた記事が投稿されたチャンネルにアクセスできることを必ず確認してください。そうしないと、受信者が受け取ったメール内のリンクをクリックしても記事投稿にアクセスできない場合があります。
チャンネルから引き出される投稿
現時点では、組織のアーティクルチャンネルに公開された投稿のみがメールデザインに追加できます。
作成できるチャンネルと投稿の種類について詳しく学んでください。
最近、アーティクルチャンネルで投稿を編集または公開したが、メールデザイナーに表示されない場合は、最新の投稿を引き出すためにリフレッシュボタンをクリックしてください。
記事投稿は、編集者によって削除、非公開、または変更される可能性があることに注意してください。リフレッシュボタンは、最新の投稿をメールに引き出すための優れた方法です！
組織がアーティクルチャンネルに投稿を作成していない場合、スタジオを開いて投稿を作成するためのリンクが表示されます。
誰がメールデザインに記事投稿を追加できるか
以下のアプリ/イントラネットの役割は、メールデザイナーのニュースクイックブロックにアクセスできます:
チャンネル編集者
スペース管理者
編集者
管理編集者
管理者
彼らはまた、組織のアーティクルチャンネルからコンテンツをメールデザインに引き出すこともできます。
この機能は、従業員メールのWebアプリ送信機能を使用しているユーザーのみに利用可能です。ニュースクイックブロックは、メールクライアントアドインを使用して送信しているユーザーには利用できません。
