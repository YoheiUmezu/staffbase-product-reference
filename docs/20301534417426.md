# スマートインサイトダッシュボードによるエンゲージメントメトリクスの分析

**URL**: https://support.staffbase.com/hc/ja/articles/20301534417426
**取得日**: 2026-05-02 18:53:31 JST

## 内容

アクティベーション
システム全体のエディター
スマートインサイトダッシュボードは、各キャンペーンの詳細な統計を表示します。ダッシュボード内では、キャンペーンに割り当てられたニュース投稿の可視性、エンゲージメント、感情に関する詳細なメトリクスを確認できます。 詳細はこちら。
ダッシュボードには、分析ダッシュボードのキャンペーンタブからアクセスできます。スマートインサイトダッシュボードは、すべてのシステム全体のユーザーロールに対して表示されます。スマートインサイトダッシュボードは、スマートインパクト機能の一部です。詳細はこちら。
トップレベルメトリクスに関するより詳細な統計を提供するセクションがあります：
可視性
エンゲージメント
コメントの感情
整合性
エンゲージメントは、キャンペーンコンテンツに関与したユーザーの数を示します。このメトリクスは、少なくとも1つのニュース投稿に関与したユーザーの割合に基づいて計算されます。
時間経過に伴うエンゲージメントグラフ
時間経過に伴うエンゲージメントグラフは、キャンペーンコンテンツのエンゲージメントを一定期間にわたって示します。このグラフには、全体のエンゲージメントを示す単一の系列があります。
タイムラインのデータポイントにカーソルを合わせると、フライアウトが開きます。フライアウトには以下が含まれます：
その時点でのエンゲージメントメトリクス
その時点で公開された投稿の反応とコメントの数
グラフ内のタイムラインは、キャンペーンの開始日と現在の日付に基づいています。開始日が設定されていない場合は、キャンペーン作成日が代わりに使用されます。キャンペーンが終了すると、終了日がタイムラインの終了を示します。
グループおよび投稿ごとのエンゲージメントテーブル
このテーブルは、個々のユーザーグループおよび投稿に基づくエンゲージメントの詳細を示します。これにより、どのユーザーグループがキャンペーンコンテンツと対話しているか、どの投稿が最も多くの対話を受けているかを特定できます。テーブルの上部にある表示ドロップダウンメニューを開くことで：
ユーザーグループまたは投稿に基づくエンゲージメントを表示
エンゲージメント率の高い順または低い順にエンゲージメントをソート
テーブル内の行にカーソルを合わせると、少なくとも1つのニュース投稿に関与したユーザーの正確な数が表示されます。
追加の役立つ情報
スマートインパクトの概要
分析ダッシュボードによるキャンペーンメトリクスの分析
