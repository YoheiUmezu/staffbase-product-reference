# CSVインポートを介して条件付きグループにメンバーを割り当てる

**URL**: https://support.staffbase.com/hc/ja/articles/360010753899
**取得日**: 2026-05-02 18:47:50 JST

## 内容

CSVファイルにタグを追加する
プロファイルフィールドのタグ付けを有効にすると、特定のオフィスの場所や部門などのタグを作成し、各ユーザーに割り当てることができます。
ユーザープロファイルに手動でタグを追加することはできません。CSVインポートまたはユーザーAPIを使用してください。
ユーザーデータを含むCSVファイルを開きます。
タグ付けが有効になっている各プロファイルフィールドのために列を作成します。
各ユーザーに適切な値を追加します。
この例では、ユーザーのオフィスの場所がロケーショングループタグの列に追加されています。
各ユーザーに対して1つの値のみを追加してください。
CSVファイルを保存します。
CSVインポートを介してユーザーを条件付きグループに同期する
スタジオで、ユーザー設定 > ユーザーインポートと同期に移動します。
新しいインポートをクリックします。
編集したCSVファイルをアップロードし、続行をクリックします。
ドロップダウンメニューを使用して、CSVの各列に対応するフィールドを選択し、続行をクリックします。.
オフィスの場所や部門などのシステムプロファイルフィールドに対応するタグを作成した場合は、システムプロファイルフィールドと作成したタグの両方にチェックを入れてください。
変更の概要を確認します。
インポートを実行をクリックします。
インポートを確認するように求めるダイアログが開きます。
インポートを続行するには、インポートを実行をクリックします。
成功したインポート後に変更の概要が表示されるダイアログが開きます。
