# Staffbaseメールにニュースブロックを統合する

**URL**: https://support.staffbase.com/hc/ja/articles/24490844367378
**取得日**: 2026-05-02 18:48:54 JST

## 内容

エディター
この記事の情報は、従業員アプリまたはStaffbaseイントラネットを使用している場合、かつStaffbaseメールがアプリまたはイントラネットと一緒に使用される場合に適用されます。
Staffbaseニュースクイックブロックを使用すると、アプリまたはイントラネットのコンテンツを直接メールに埋め込むことができます。
スタジオで、メールに移動します。
メールを作成をクリックするか、編集したいメールを開きます。
メールデザイナーで、左側のパネルからニュースクイックブロックをクリックします。
ニュースクイックブロックがメールに追加されます。
右側で、検索ボックスにキーワードを入力してニュース投稿を見つけるか、最近公開されたリストからニュース投稿を選択します。
サムネイル画像、記事のタイトル、およびティーザーテキストがニュースクイックブロックにコピーされ、記事へのリンクも含まれます。記事に画像がない場合やティーザーテキストが空白の場合は、ニュースブロックに独自の画像とティーザーテキストを追加できます。さらに、リンクテキストを含むすべてのテキストを編集できます。 メールデザイナーとクイックブロックについて詳しく学ぶ。
