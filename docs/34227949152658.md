# Studio でユーザーアカウントを削除する

**URL**: https://support.staffbase.com/hc/ja/articles/34227949152658
**取得日**: 2026-05-02 18:40:45 JST

## 内容

管理者
スタッフベースでは、ユーザー基盤を設定して、スタッフベースプラットフォームを効果的に利用できます。不要になったユーザーアカウントは、ユーザー基盤を管理するために削除できます。Studio で、1つまたは複数のユーザーアカウントを同時に削除できます。
ユーザーの削除は元に戻せません。特に複数のアカウントを一度に削除する場合は、必ず正しいユーザーアカウントを選択してから承認してください。
Studio で、ユーザーに移動します。
削除したいユーザーアカウントを検索して選択します。
フィルターを使って検索を絞り込めます。
ユーザーを削除するをクリックします。
ユーザーを削除しますか？ダイアログが開きます。
確認フィールドに、{number}人のユーザーを削除すると入力します。たとえば、4人のユーザーを削除する場合は、4人のユーザーを削除すると入力します。
削除するをクリックします。
ユーザーアカウントを完全に削除しました。
