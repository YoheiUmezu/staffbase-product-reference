# デスクトップ用カスタムヘッダーリンクの概要

**URL**: https://support.staffbase.com/hc/ja/articles/17825480190098
**取得日**: 2026-05-02 19:01:26 JST

## 内容

デスクトップのヘッダーでは、カスタムヘッダーリンクを追加できます。これらのリンクは、ユーザーを以下のいずれかに誘導できます：
内部コンテンツ
外部サイト
Staffbaseプラットフォーム内の任意のメニュー項目
ヘッダーリンクはお好みの順序で配置でき、可視性設定を使用して個別にターゲットできます。この機能について詳しくは、Staffbaseサポートまたはカスタマーサクセスマネージャーにお問い合わせください。
あなたのイントラネットヘッダーには、最大で4つのヘッダーリンクを表示できます。プロフィールメニュー、通知アイコン、ランチパッドを除くすべての項目はヘッダーリンクです。
ヘッダーリンクには2種類あります：
動的ヘッダーリンク
静的ヘッダーリンク
両方のタイプのヘッダーリンクを追加できます。これらはAPIコールを介して設定する必要があります。 詳しくはこちら。
動的ヘッダーリンク
動的ヘッダーリンクは、ユーザーを既存のメニュー項目に誘導するリンクです。このタイプのヘッダーリンクへのユーザーのアクセスは、対応するメニュー項目へのアクセスに依存します。ユーザーがメニュー項目を表示する権限を持っている場合、そのメニューのヘッダーリンクも表示できます。
静的ヘッダーリンク
静的ヘッダーリンクは、プラットフォーム内の非メニューコンテンツや外部ウェブサイトにユーザーを誘導するリンクです。これらはクイックリンクAPIを介して作成および管理されます。このタイプのヘッダーリンクへのユーザーのアクセスは、ユーザーIDまたはグループIDレベルで制限できます。 詳しくはこちら。
表示されるヘッダーリンクの数は、ユーザーの可視性設定に応じて異なる場合があります。
