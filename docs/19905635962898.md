Staffbase は、Microsoft 製品を使用する組織やチームを支援するために、Microsoft Stream プラットフォームからコンテンツにビデオを簡単に追加できるようにします。会議の録画やその他の重要な情報に関わらず、Microsoft Stream ビデオを追加して、読者を関連するコンテンツで最新の状態に保ち、関与させることができます。
Staffbase プラットフォームで Microsoft Stream ビデオを使用する方法は次のとおりです：
記事やメディア投稿のカバー ビデオとして
特定のセクションにさらなるコンテキストを提供するために投稿やページ内で
前提条件
Microsoft 365 Connections を Staffbase に追加している。
アプリまたはイントラネットから Microsoft 365 にサインインしている。まだサインインしていない場合は、サインインを促されます。
コンテンツの作成者と視聴者の両方が、Microsoft Stream コンテンツにアクセスするために Microsoft にサインインしている必要があります。Microsoft での権限に基づいてアクセスできるビデオのみが表示されます。
