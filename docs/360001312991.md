# 公共エリアの例

**URL**: https://support.staffbase.com/hc/ja/articles/360001312991
**取得日**: 2026-05-02 18:48:02 JST

## 内容

公共エリアに掲載する情報やメディアは、従業員以外の一般の聴衆にとって興味深いものであるべきです。たとえば、ここに会社のニュースや求人情報を掲載できます。Facebookフィードを追加するのも素晴らしいアイデアです。
以下の種類のコンテンツを使用できます：
ニュースフィード
ページ
地図、カレンダー、食事プラン、Facebookページなどのプラグイン
例
外部のインスピレーションは常に役立ちます。他の顧客が何をしているか見てみましょう：
