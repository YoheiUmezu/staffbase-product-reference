# スペースへのコンテンツの追加

**URL**: https://support.staffbase.com/hc/ja/articles/360010422340
**取得日**: 2026-05-02 18:58:29 JST

## 内容

スペースが作成されると、そのスペースが対象とするユーザーグループに関連するコンテンツを追加し始めることができます。スペースは特定のユーザーへのコンテンツのターゲティングを容易にします。
スペースにコンテンツを追加する方法はいくつかあります：
スペースに移動してコンテンツを追加します。
すべてのコンテンツの概要に移動し、Staffbaseプラットフォーム内のすべてのコンテンツを一覧表示し、コンテンツを追加します。コンテンツを追加する際に、スペースを選択するオプションが表示されます。
コンテンツを新しいスペースに移動したい場合は、その新しいスペースを作成し、コンテンツを移動できます。
スペースにさまざまな種類のコンテンツを追加します：
プラグイン
従業員ディレクトリとチャットプラグインを除く、利用可能なプラグインをスペースに追加できます。たとえば、食堂の位置情報を提供するために、ミールプランプラグインを使用します。
ページ
特定のスペースに関連するコンテンツを持つ静的ページを追加します。たとえば、営業部門の贈り物ポリシーや、トラック運転手が遵守すべき安全規則を記載したページなどです。
ニュースページ、チャンネル、および投稿
特定のスペース専用のニュース投稿を公開できますが、Staffbaseはニュースページとチャンネルはすべての従業員スペースに保持することを推奨しています。一般的にニュースはすべての登録ユーザーに利用可能であるべきですが、特定のユーザーにリーチするための専用チャンネルが必要な場合、スペースはユーザーグループと並んでニュースをターゲットにする方法の一つです。たとえば、ロンドンに拠点を置くユーザー専用の掲示板をそのスペースに作成できます。
スペースにコンテンツを最適に追加する方法についてさらに情報が必要な場合は、Staffbaseサポートまたはカスタマーサクセスマネージャーにお問い合わせください。
