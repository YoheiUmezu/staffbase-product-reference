# Microsoft Stream ビデオを投稿またはページに追加する

**URL**: https://support.staffbase.com/hc/ja/articles/19958185624978
**取得日**: 2026-05-02 18:46:34 JST

## 内容

ベータ
アクティベーション
エディター
Staffbase は、Microsoft Stream ビデオをコンテンツに追加するプロセスを簡素化し、複数のオプションを提供します。Microsoft Stream ビデオを投稿のカバー ビデオとして使用するか、投稿またはページの本文に追加します。
記事またはメディア投稿内で、埋め込まれた Microsoft Stream ビデオを再生すると、すべての Microsoft Stream ビデオ プレーヤー機能を使用できます。
Microsoft Stream ビデオは、ビデオの右下隅にある Stream で視聴 アイコンで他のビデオと区別できます。Stream で視聴 アイコンをクリックすると、Microsoft Stream でビデオが開きます。
前提条件
Microsoft 365 アカウントにサインインしています。まだサインインしていない場合は、サインインを求められます。
コンテンツ作成者とユーザーは、それぞれファイルにアクセスするための Microsoft 権限を持っています。
Microsoft Stream ビデオをカバー ビデオとして追加する
記事またはメディア投稿にカバー ビデオとして追加された Microsoft Stream ビデオは、投稿が表示される場所の投稿タイトルの横にサムネイルとして表示されます:
ニュースフィード
ユーザープロフィール
通知
ブックマーク
ウィジェット
検索
SharePoint の Staffbase News Central
スタジオで コンテンツ をクリックします。
ビデオを追加したいメディアまたは記事チャンネルに移動します。
記事画像/ビデオ の下で 選択する をクリックします。
Microsoft Stream をクリックします。
Microsoft Stream ビデオを追加 ダイアログが開きます。
ドロップダウンのオプションからビデオを検索して選択します。
ファイル ドロップダウンには、次の基準に基づいて利用可能なファイルが表示されます:
Microsoft でのあなたのアクセスと権限
Microsoft によって決定された関連性
追加 をクリックします。
ビデオのサムネイルが 記事画像/ビデオ の下に表示されます。
Staffbase からビデオのサムネイルを変更することはできません。埋め込まれたビデオを編集したい場合は、Microsoft Stream で行う必要があります。Staffbase に追加されたビデオが後で Microsoft Stream で編集されると、リンクが同じままであれば、Staffbase にも反映されます。
保存 をクリックします。
Microsoft Stream ビデオを投稿のカバー ビデオとして追加しました。ビデオを再生するには、投稿を開く必要があります。
Microsoft Stream ビデオを投稿またはページに追加する
Microsoft 365 オプションを使用して、メディアの追加ドロップダウンからファイル ビューア ウィジェットにアクセスし、投稿またはページの重要なポイントで関連するビデオを選択します。
ファイル ビューア ウィジェットを追加します。
ファイル ビューア ウィジェットを構成します。
Microsoft Stream ビデオを投稿またはページに追加しました。
