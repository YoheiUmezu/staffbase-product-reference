# 2026年第1四半期リリース

**URL**: https://support.staffbase.com/hc/ja/articles/32560682104850
**取得日**: 2026-05-02 18:45:34 JST

## 内容

今年最初のリリースは、待望の新機能を多数搭載してスタートします。AIによる動画字幕やAIポッドキャストにより、コンテンツの作成がより簡単になり、インクルーシブで手作業の負担も軽減されます。再設計されたページ用の新しいコンテンツデザイナーは、ページ作成をより迅速かつ直感的にし、アクセシビリティ機能も内蔵しています。メールのダイナミックブロックターゲティングやAIポッドキャストは、適切なコンテンツを適切な人に届けるのに役立ち、コンテンツガバナンスは公開済みページの監視と管理を簡素化します。Staffbase Liveは、リアルタイムの全社イベントを通じて従業員をつなげます。
これらのアップデートにより、適切な視聴者にリーチし、大規模でも高品質なコンテンツを維持できます。2026年2月12日に登場するその他の新機能もぜひご覧ください。
スタッフベースCampusで提供されているQ1 2026リリースコースに登録し、動画の字幕、再設計されたページ、コンテンツガバナンス、メールのダイナミックブロックターゲティングについて詳しく学びましょう。Campusが初めての方は、今すぐ登録してください。
ダイナミックブロックターゲティング
本番環境 自動展開
パーソナライズされたメールを複数バージョン作成せずに大規模に送信できます。編集者はコンテンツブロックにターゲティングルールを割り当て、各ブロックがどのユーザーグループやユーザー向けかを定義します。メール送信時、割り当てに基づいてブロックが自動的に表示または非表示になり、受信者ごとに関連する内容のみが表示されます。その結果、手作業が減り、ミスも少なくなり、1つの効率的なワークフローでより魅力的なメールを実現します。
Staffbase Live
本番環境 有効化 アドオン
組織全体でリアルタイム配信を実現します。Staffbase Liveは、モデレーターがタウンホールなどの主要イベントを従業員のアプリやイントラネットに直接配信できるようにします。Staffbaseプラットフォームをライブ・オンデマンドコミュニケーションのハブに変え、リアルタイムの絵文字リアクション、チャット、後から視聴できる録画機能も利用可能です。Staffbaseを、あらゆるコミュニケーションの入り口としてご活用ください。
コンテンツガバナンス
本番環境 有効化 アドオン
コンテンツガバナンスは、ページの公開後も継続的に監視し、古いコンテンツや未使用のコンテンツなどの問題を検出します。これらのインサイトにより、企業はコンテンツの質を向上させ、ページに関する見落としを最小限に抑えることができます。明確で一元化された概要により、コンテンツガバナンスがよりスマートかつ拡張性の高いものとなり、手動レビューへの依存を減らし、編集者が意思決定を完全にコントロールできます。
タスク
本番環境 有効化 アドオン
タスク（旧タスク管理）はベータ版を終了します。現場従業員がアプリ内で直接タスクを作成、割り当て、整理、追跡できるようになります。モバイルとデスクトップの両方で利用でき、タスクを一元管理することで責任感とコラボレーションを向上させます。コンテンツエディターの新しいタスクハブウィジェットは、すべての割り当て済みプロジェクトへの中央エントリーポイントを提供し、簡単にアクセスや切り替えが可能です。
ジャーニー概要
本番環境 自動展開 エンタープライズ
ジャーニー概要（旧ジャーニーナビゲーター）は、従業員に有効中および完了済みのジャーニーを一元的に表示します。タイムラインにはステップ、過去のメッセージ、今後のステップのプレビューが明確に表示されます。ジャーニー概要は、メニュー、モバイルクイックリンク、ヘッダー、ページからリンク可能です。さらに、システム全体の編集者は、追加情報やリソースを備えたジャーニーハブとして機能する専用のランディングページを作成できます。
再設計されたページ
ベータ 有効化
再設計されたページは、より優れたアクセシビリティと作成・編集体験を提供します。編集者は、クラシックエディターと新しいページ用コンテンツデザイナーから選択し、直感的な方法でコンテンツを作成できます。ドラッグ＆ドロップブロック、改良されたWYSIWYG編集、キーボード操作対応、ページプレビュー、公開済みページの下書き、最新のバージョン履歴により、ページ作成がより効率的で柔軟かつコントロールしやすくなります。
動画字幕
ベータ 有効化 AI搭載
AIによる字幕で動画コンテンツをよりアクセシブルかつインクルーシブにします。新しくアップロードされた動画には、元の言語（現在は英語、ドイツ語、スペイン語、フランス語）で自動的に字幕が付与されます。コミュニケーターは字幕テキストの確認・編集、カスタムVTTファイルのアップロード、字幕のレビュー済みマークが可能で、従業員はニュースやページで再生中に字幕を有効にしたり言語を切り替えたりできます。
AIポッドキャスト
ベータ 有効化 AI搭載
AIポッドキャストは、社内の最新情報をパーソナライズされたオーディオエピソードに変換し、従業員が通勤中や移動中、マルチタスク中でも情報を得られるようにします。管理者は既存のコンテンツを一貫性のある高品質なOn Airポッドキャストエピソードに簡単に変換でき、メッセージ、正確性、トーン、組織目標やコミュニケーション戦略との整合性を完全にコントロールできます。
改善点
自動展開
Microsoft 365 検索フィルター
Microsoft 365での検索性向上と効率化のため、ファイルタイプや最終更新日で検索結果を絞り込めるようになりました。これらのフィルターにより、より簡単に検索結果を絞り込み、関連ドキュメントを素早く見つけることができます。
ページオーナー
ページのコンテンツガバナンス向上のため、専用のページオーナーフィールドが追加されました。コンテンツを最新かつガイドラインに準拠させる責任者を明確にします。このフィールドは、今後のAIコンテンツガバナンスでのレビュー ワークフローの基盤となり、管理者や編集者が質問や更新の際に適切な連絡先をすぐに特定できるようにします。
ニュース用 Staffbase Companion Editor
2024年第2四半期リリースで初登場したAI搭載のStaffbase Companion Editorが、すべてのアプリおよびイントラネットのお客様向けにセルフサービスで有効化できるようになりました。Companion Editorの全機能を活用し、記事テキスト、ティーザーテキスト、タイトルを記事チャンネルタイプ全体で作成・洗練できます。管理者は、StudioのセルフサービスAI設定ページからこの機能をオプションで有効にできます。
モバイル検索の改善
ベータ版を終了し、モバイル検索の使いやすさとアクセシビリティが大幅に向上しました。スタートページヘッダーの検索アイコンがより目立つようになり、視認性の高いデザインで結果が見やすくなりました。ユーザーはGoogle Drive、Microsoft 365、Confluence、Box、ServiceNowなどの統合からサードパーティコンテンツにもシームレスにアクセスできます。フォーム & アンケートやカレンダー & イベントの新しい検索カテゴリで、より精度の高い検索が可能です。さらに、コンテンツ説明による文脈把握、フィルター、並べ替えオプション、直近の検索履歴へのクイックアクセス機能も追加されました。
対応iOSモバイルアプリバージョン
対応するiOSモバイルアプリの最小バージョンは17.0になりました。

2026年2月11日
Staffbaseメール
Staffbaseメールの改善点
カレンダーブロッカーが、Staffbaseメールを単体製品として利用しているお客様にもご利用いただけるようになりました。これにより、編集者がコミュニケーションを送信すべきでないタイミングを視覚的に確認できます。ただし、スケジュールされたブロッカー中にメールの送信自体をブロックする機能はありません。
フォルダ設定ページで、送信者名フィールドがドロップダウンから提案された名前を選択することも、カスタムの名前を直接入力することもサポートしていることを強調しました。
メールテンプレートは、保存時に現在のウェブプレビューに基づいて自動的にサムネイルプレビュー画像を生成するようになりました。テンプレートを保存するたびにサムネイルが更新されます。カスタムのカバー画像がサムネイルとして選択されている場合は、上書きされません。
2026年1月29日
Webアプリ/イントラネット
アプリとイントラネットの改善
ユーザーアカウントを編集する際、管理者はユーザーのプロフィールフィールド値を表示または編集できるようになりました。表示や編集を有効にするには、Staffbaseサポートまたはカスタマーサクセスマネージャーにお問い合わせください。
オープンまたは内部の手動グループに、識別子値に加えてメールアドレスを含むCSVファイルを使用して、ユーザーを追加または置き換えることができるようになりました。
CSVインポートのプレビュー手順で、再有効化されたユーザーの数と詳細も表示されるようになりました。
Studioの設定における日本語のローカライズ文字列が改善されました。
ServiceNowチケットウィジェットが、他のサポートされているモジュールに加えて、Strategic Portfolio Management（SPM）およびWorkplace Service Delivery（WSD）をサポートするようになりました。SPMはタスクと要求、WSDはケースとタスクをサポートします。
アプリとイントラネットのバグ修正
コレクション内のファイル数が、ファイルを削除した際に更新されませんでした。この問題は修正されました。
Staffbaseメール
Staffbaseメールの改善
すべてのアナリティクスページのデフォルトの期間が「今日」から「7日間」に変更されました。これにより、デフォルトでより多くのデータが表示され、全体的な閲覧体験が向上します。
管理者は、送信後のメール内のハイパーリンクを更新できるようになりました。これにより、メールを再送信することなく、壊れたリンクや古いリンクを簡単に修正できます。
メールを複製した際、送信者アドレスと送信者名が元のメールと一致するようになりました。
2026年1月7日
Webアプリ/イントラネット
アプリとイントラネットの改善
アナリティクスダッシュボードのニュースタブでのCSVダウンロード時、「いいね」指標が「投稿のいいね」と「コメントのいいね」に分かれて表示されるようになりました。
アナリティクスダッシュボードのニュースタブにあるニュースリストテーブルで、「投稿のいいね」と「コメントのいいね」が別々の列として表示されるようになりました。
タイトルおよびMy Filesで、Shiftキーを押しながら最初と最後のファイルを選択することで、範囲選択ができるようになりました。
タイトルで、名前やコンテンツでファイルを検索する際、結果が自動的に関連性順で並び替えられるようになりました。検索中は他の並び替えオプションは無効になり、検索が消去されると再び利用可能になります。
アイルランド語が新しいコンテンツ言語およびオンデマンド翻訳で利用可能になりました。
ジャーニーの概要ページで、ユーザー体験が向上しました。
Staffbaseメール
Staffbaseメールの改善
メールフォルダで、フォルダ管理者はユーザーグループまたはユーザーをフォルダの対象視聴者として割り当てることができるようになりました。
すべてのメールビューから、特定のフォルダでフィルタリングし、複数のメールを選択して、アクセス権のあるスペースまたはフォルダに移動できるようになりました。

2025年12月17日
Webアプリ/イントラネット
アプリとイントラネットの改善
ニュースおよびページのコンテンツエディターから、タスク管理ハブウィジェットを追加できるようになりました。これにより、従業員は割り当てられたすべてのタスクプロジェクトへの集中したエントリーポイントを得られます。複数のチーム、プロジェクト、または拠点で作業するユーザーにとって、メニューを移動せずにプロジェクト間を迅速かつ確実に切り替える方法を提供します。
Studioで、チャットプラグインがプラグイン設定のみを表示するように改善されました。リンクをクリックすると、イントラネット上のユーザーの個人チャットが開きます。すべての会話機能はアプリまたはイントラネットで引き続き利用可能です。
アナリティクスダッシュボードのニュースタブで、UIとグラフが改善され、ユーザー体験が向上しました。

Staffbaseメールのバグ修正
まれに、ブランディング設定で同じ名前のフォントファイルを2回アップロードすると、メールデザイナーに2回表示されることがありました。この問題は修正されました。
