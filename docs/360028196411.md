# ニュースページの作成

**URL**: https://support.staffbase.com/hc/ja/articles/360028196411
**取得日**: 2026-05-02 18:50:25 JST

## 内容

ニュースページは、新聞やソーシャルネットワークのように、新しいコンテンツが定期的に公開される場所です。必要に応じて、好きなだけニュースページを作成できます。
スタジオで、コンテンツに移動します。
メニューのシステムセクションの下で、メニューをクリックします。
編集ドロップダウンメニューから、ニュースページを作成を選択します。
ニュースページを作成ダイアログが開きます。
ページ名を入力します。
ソートドロップダウンメニューから、ソートオプションを選択します。
アプリのニュースフィードに表示するチャンネルごとの投稿数を定義します。
このニュースページに関連付けられたチャンネルをアプリのメニューにどのように表示するかを選択します。
作成をクリックします。新しいニュースページがメニューに追加されます。
保存をクリックします。
ニュースページが作成されました。次に、ニュースチャンネルを追加して、ニュース投稿を公開するために必要なすべての要素を設定する必要があります。
