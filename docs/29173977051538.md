# プラットフォームにおけるメディアの概要

**URL**: https://support.staffbase.com/hc/ja/articles/29173977051538
**取得日**: 2026-05-02 18:41:19 JST

## 内容

スタッフベースのさまざまな機能にメディアをアップロードできます。各メディアタイプや機能には特定の要件があります。たとえば、ページ、ニュース、フォーム、その他のプラグインに添付された動画やファイルは、同じ仕様に従います。
コンテンツがすべてのデバイスでプロフェッショナルに表示され、確実に動作するように、画像、動画、外部埋め込みメディア、Screensのメディア仕様に従ってください。
以下の仕様は推奨事項であり、すべてを網羅したものではありません。
