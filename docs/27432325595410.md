# プラグインの共通設定の概要

**URL**: https://support.staffbase.com/hc/ja/articles/27432325595410
**取得日**: 2026-05-02 18:45:50 JST

## 内容

共通設定セクションは、スタッフベースプラットフォーム内のプラグインで利用でき、各プラグインを正確な要件に合わせてカスタマイズできます。プラグインをインストールした後、歯車アイコンの背後にある設定を調整できます。これらの設定はいつでもさらに更新できます。共通設定に加えて、各プラグインには独自のプラグイン設定も含まれています。
設定オプション
タイトル
タイトルは、プラグインの特定のインスタンスの名前です。たとえば、アンケートの場合は、アンケートのタイトルになります。
アイコン
アイコン設定では、ドロップダウンからアイコンを選択するか、カスタムアイコンをアップロードできます。これはプラグインインスタンスのアイコンであり、プラットフォームのメニューに表示されます。
作成者
作成者は、プラグインインスタンスを作成した元のユーザーです。
スペース
有効化
プラグインインスタンスが配置されているスペースを表示します。
アプリでの表示
アプリでの表示設定では、どのユーザーがプラグインインスタンスを表示できるかを決定できます。ドロップダウンのオプションは次のとおりです：
すべてのスペースユーザー（スペースが有効な場合）
選択したスペースユーザーまたはグループ
表示しない
「表示しない」を選択した場合でも、インスタンスには直接リンクでアクセスできます。
編集者
編集者設定では、プラグインインスタンスを編集できる特定のユーザーやユーザーグループを選択できます。プラグイン編集者について詳しくはこちら。
メニューでの表示場所
メニューでの表示場所設定では、プラグインインスタンスをどのメニューに表示するかを指定できます。メニュー項目の表示設定について詳しくはこちら。
パンくずリスト設定
ベータ
有効化
パンくずリスト設定では、従業員アプリやイントラネットのパンくずナビゲーションでメニュー項目の表示を管理できます。パンくずナビゲーションについて詳しくはこちら。
