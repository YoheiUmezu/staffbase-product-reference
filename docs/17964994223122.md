# スマートインパクトの概要

**URL**: https://support.staffbase.com/hc/ja/articles/17964994223122
**取得日**: 2026-05-02 18:43:20 JST

## 内容

アクティベーション
スマートインパクトは、キャンペーンの効果を測定することを可能にします。キャンペーンの新しい指標を提供し、使いやすいダッシュボードを備えています。スマートインパクトは、戦略的にキャンペーンの結果を測定することによって、出力から成果へと移行する力を与えます。スマートインパクトは、Staffbaseプラットフォームの異なる領域にまたがる2つの機能セットで構成されています。
スマートインパクトには以下が含まれます：
分析ダッシュボードのキャンペーンタブ
詳細なキャンペーン指標を持つスマートインサイトダッシュボード
詳細については、カスタマーサクセスマネージャーまたはStaffbaseサポートにお問い合わせください。
セキュリティ
Staffbaseは、キャンペーンタブおよびスマートインサイトダッシュボードのすべての処理を第三者の関与なしに実施します。
スマートインサイトダッシュボードには、コメントの感情分析に基づく指標が含まれています。感情分析は、特定のキャンペーンに関連する投稿のすべてのコメントを処理し、ラベル付けすることによって行われます。この処理は、社内のワークフローを通じて機械学習モデルを使用して行われます。このパイプラインは、各コメントがポジティブ、中立、またはネガティブとしてラベル付けされる確率を提供します。完全に社内で実行されるため、Staffbaseは分析を実行するために第三者のプロセッサーを必要としません。
