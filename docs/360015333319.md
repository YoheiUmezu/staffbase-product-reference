# 失敗したiOSアプリの更新に関する次のステップ

**URL**: https://support.staffbase.com/hc/ja/articles/360015333319
**取得日**: 2026-05-02 18:44:46 JST

## 内容

Staffbaseは、ネイティブアプリの新しいバージョンを約3ヶ月ごとにリリースします。
iOSアプリの更新は、Apple App Storeアカウントで全てのライセンス契約が署名されている場合にのみ機能します。ライセンス契約が署名されていない場合、アプリの更新は公開できず、ユーザーは古いアプリバージョンのままとなり、プッシュ通知に問題が発生する可能性があります。
Appleが契約を継続的に更新しているため、定期的に確認することが最善です。最新の確認は、Staffbaseが新しいリリースを発表したときに行ってください。
Appleライセンス契約の確認と署名
要件
Apple Developer Programアカウントのアカウント保持者である必要があります。
App Store ConnectウェブサイトにApple App Storeアカウントでログインします。
https://appstoreconnect.apple.com/agreementsを開きます。
未署名のライセンス契約を探し、署名します。
質問がある場合や追加のサポートが必要な場合は、support@staffbase.comまでお問い合わせください。
