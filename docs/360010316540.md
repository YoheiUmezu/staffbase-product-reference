# ユーザーグループの削除

**URL**: https://support.staffbase.com/hc/ja/articles/360010316540
**取得日**: 2026-05-02 18:59:10 JST

## 内容

グループ管理者
ユーザーグループが不要になった場合は、いつでも削除できます。オープングループと内部グループの両方を削除できます。グループのメンバーは削除時にグループから除外され、削除されたグループのコンテンツを表示できなくなります。
ユーザーグループの削除は永久的な操作です。一度削除されると、ユーザーグループは復元できません。
スタジオで、ユーザー設定に移動します。
セクションあなたのユーザーとグループで、ユーザーグループを選択します。
削除したいグループを開きます。
グループを編集をクリックします。
グループを編集するためのダイアログが開きます。
ユーザーグループを永久に削除をクリックします。
削除を確認するためのダイアログが開きます。
削除をクリックします。
ユーザーグループが削除されました。
