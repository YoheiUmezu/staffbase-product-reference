# ビデオインサイトの概要

**URL**: https://support.staffbase.com/hc/ja/articles/11171865678610
**取得日**: 2026-05-02 18:43:41 JST

## 内容

アクティベーション
エディター
ビデオインサイトは、Staffbaseプラットフォームにアップロードされた投稿およびページの本文にあるビデオにのみ利用可能です。ビデオのメトリクスは、10秒未満のビデオや、YouTubeやVimeoなどの外部ビデオ共有プラットフォームからStaffbaseプラットフォームに埋め込まれたビデオには利用できません。
ビデオを含むコンテンツは、高いエンゲージメント効果と可視性を持っています。しかし、それを管理するにはより多くの労力が必要になることもあります。ビデオインサイトを使用することで、重要な分析とデータを用いてビデオコンテンツを効率的に管理できます。
ビデオインサイトは、詳細なビデオメトリクスを取得し、従業員の視聴習慣の概要を作成します。さらに、特定の再生時間と視聴時間のメトリクスを使用して、従業員がビデオコンテンツとどのようにインタラクトしているかを理解し、ユーザーの活動とエンゲージメント率を測定できます。最終的には、関連性のある従業員コンテンツを作成し、リーチを向上させるのに役立ちます。
追跡されるビデオメトリクスは以下の通りです：
総視聴回数
総視聴回数は、ビデオが5秒以上視聴された回数の合計を表示します。
ユニーク視聴回数
ユニーク視聴回数は、個々のユーザーがビデオを5秒以上視聴した回数の合計を表示します。
視聴時間
視聴時間は、ユーザーがビデオを一時停止したり、他のページに移動する前に視聴した平均時間を表示します。
