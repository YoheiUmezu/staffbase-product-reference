# Androidプッシュアイコン

**URL**: https://support.staffbase.com/hc/ja/articles/115002026611
**取得日**: 2026-05-02 18:54:44 JST

## 内容

プッシュ通知は、プッシュ通知機能が有効になっている場合にアプリがモバイルユーザーに送信する自動通知です。プッシュ通知が送信されると、Androidデバイスはステータスバーと通知ドロワーに小さなプッシュアイコンを表示します。
要件
サイズ: 96 x 96 px
ファイル形式: png
解像度: 72 dpi
カラープロファイル: sRGB
ファイル名: PushIcon.png
色: #FFFFFF
透明な背景
形状: アウトラインのみ
デフォルトプッシュアイコン
カスタムプッシュ通知アイコンを提供しない場合、Staffbaseはアプリ用のデフォルトプッシュアイコンを提供します:
追加の有用な情報:
プッシュアイコンは、AndroidデバイスやAndroidバージョンによって異なる場合があります。たとえば、一部のAndroidデバイスではプッシュアイコンの代わりにアプリアイコンが使用されます。
