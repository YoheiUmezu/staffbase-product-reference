# Staffbaseスタジオとダッシュボード

すべての内部コミュニケーションチャンネルを一か所で管理します。Studioでは、ネイティブアプリ、イントラネット、メールなどの異なるチャンネルを簡単に見つけて管理し、切り替えることができます。ダッシュボードは内部コミュニケーションタスクへの入り口であり、プラットフォーム上のエンゲージメントとアクティビティの全体像を提供します。
最適な視覚およびユーザーエクスペリエンスのために、Staffbaseは1440px以上の解像度の画面でStudioを使用することを推奨しています。ただし、Staffbaseメールの場合、1100px以上の解像度がサポートされています。
水平Studioナビゲーションには、ダッシュボードがあり、プラン、役割、権限に応じて追加のメニューポイントがあります。プランニング、コンテンツ、メール、SMS、スクリーン、ファイル、アナリティクス、リソースセンター、ユーザー管理、設定の間を素早く切り替えて概要を把握します。
ダッシュボード
ダッシュボード は、プラットフォームのエンゲージメントとアクティビティの全体像を提供し、最新かつ最もパフォーマンスの高いコンテンツに関する非常に関連性の高いアナリティクスを提供します。ダッシュボードから、クイックアクションを通じて最も重要なタスクにワンクリックでアクセスできます。
プランニング
プランニングには、編集カレンダー、キャンペーン、ブリーフィング、戦略的プランニング、キャンペーンマネージャーが利用可能です。これはすべてのプランニングニーズに対応するワンストップの目的地です。コミュニケーションのプランニングを次のレベルに引き上げましょう。
コンテンツ
ニュース、ページ、スペース、メニュー、コメント、およびすべての他のプラグインがコンテンツの下にまとめられ、コンテンツの作成と管理の最適な概要を提供します。
メール
Staffbase メールはStudioに統合されています。ワンクリックでStaffbaseメールのすべてのお気に入りの機能に切り替えます。統合されたナビゲーションを使用して、メールをデザイン、送信、追跡、共有します。
SMS
Staffbase SMSは、分散した労働力との信頼性の高いコミュニケーションへの迅速かつ簡単なアクセスを可能にし、重要な更新やアラートをテキストで送信し、Twilioのプラットフォームを使用してStudioから直接フロントラインおよびデスクレスワーカーに到達します。
スクリーン
スクリーンを使用して、デバイス管理でデバイスを管理し、スライドショーを作成するを使用してスライドショーを構成し、レイアウトタブでブランディングをカスタマイズして、一貫したブランドの視聴体験を提供します。
ファイル
ファイルは、コンテンツファイルを管理するための中央ライブラリであり、コンテンツ作成者と管理者の両方をサポートします。マイファイルを使用してファイルをアップロード、再利用、整理し、管理者はすべてのファイルを中央で管理します。ファイルを置き換えるやコレクションなどの機能でコンテンツを整理し、最新の状態に保ちます。
アナリティクス
トップナビゲーションからアプリとイントラネットのアナリティクスにアクセスします。アナリティクスデータからユーザーエンゲージメントを学び、それに応じてコミュニケーションを計画します。Staffbaseメールのアナリティクスはメールセクションで見つけることができます。
リソースセンター、ユーザー、設定
トップナビゲーションバーからリソースセンター、ユーザー管理、および設定に直接アクセスします。リソースセンターで利用可能な最新のニュースとリソースについて学びます。アプリとイントラネットのユーザー、メールの連絡先、および異なるメールの権限と役割をユーザー管理で見つけます。歯車アイコンをクリックすると、プラットフォームの設定が開きます。
サポート
質問やフィードバックがある場合は、右下のサポートボタンを使用してお問い合わせください。
FAQ
新しいStudioでロゴが表示されないのはなぜですか？
旧Studioでは、組織のロゴでStudioをブランディングすることが可能でした。新しいStudioでは、このオプションが削除され、常にStaffbaseのロゴが表示されます。ロゴの設定はStudioにのみ影響します。アプリやイントラネットでは、ユーザーはブランドのロゴと色を確認できます。
スマートフォンのプレビューがもう見られないのはなぜですか？
旧Studioでは、ダッシュボードにスマートフォンのフレーム内でコンテンツのプレビューが表示されていました。このスマートフォンのプレビューは新しいStudioでは利用できなくなり、削除されました。
ダッシュボードタイルにアクセスすると「何かがうまくいかなかった」というエラーメッセージが表示され続けるのはなぜですか？
これは、ブラウザに広告ブロッカーがインストールされている場合に発生する可能性があります。その場合、広告ブロッカーがアナリティクスの表示もブロックします。問題を解決する方法を学びましょう。
