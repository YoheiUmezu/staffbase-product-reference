# StaffbaseをあなたのApp Store Connectアカウントに招待する

**URL**: https://support.staffbase.com/hc/ja/articles/360002975631
**取得日**: 2026-05-02 18:59:14 JST

## 内容

StaffbaseをあなたのApp Store Connectアカウントに招待するには、次のいずれかの役割が必要です：
アカウントホルダー（チームエージェント）
管理者
Staffbaseが招待に必要な名前とメールアドレスを送信します。
App Store Connectにサインインします。
ユーザーとアクセスをクリックします。
+ボタンをクリックして新しいユーザーを追加します。
名前とメールアドレスを入力します。
役割としてアプリマネージャーを選択します。
証明書、識別子およびプロファイルへのアクセスを付与するためのチェックボックスを選択します。
オプションで、アカウント内のアプリへのアクセスを設定します。
App Store Connectアカウントに複数のアプリがある場合、Staffbaseユーザーの権限をStaffbaseアプリに明示的に制限できます。AppIDが作成されると、Staffbaseアプリの設定が利用可能になります。
この制限を今設定する必要はありません。後でいつでも設定できます。Staffbaseはこの制限を設定または変更することはできません。これは管理者の役割を持つユーザーのみが許可されています。
招待をクリックします。
追加の役立つ情報
Appleのリソース：App Store Connectにおける役割とアクセス
開発者向け：証明書、ID、およびプロファイルへのアクセス
