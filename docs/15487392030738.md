# ユーザーグループのCSVインポート概要

**URL**: https://support.staffbase.com/hc/ja/articles/15487392030738
**取得日**: 2026-05-02 19:00:48 JST

## 内容

Staffbaseは、コンテンツを効果的にターゲットするためのさまざまなタイプのユーザーグループを提供しています。内部の手動グループやオープングループのユーザーを手動で管理できます。より迅速な一括管理のために、CSVファイルをインポートしてグループ内のユーザーを追加または置き換えることができます。必要なのは、プラットフォームに既に存在するユーザーの識別子値だけです。
CSVインポート機能は、条件付きグループには利用できません。これらは、設定した条件に基づいて自動的に管理されます。
ユーザーグループにCSVインポートを使用するタイミング
CSVインポートオプションは、グループ内の複数のユーザーを迅速に更新または管理する必要がある状況に最適です。この方法は時間を節約し、エラーを減らし、グループメンバーシップを最新の状態に保つための簡単な方法を提供します。
一般的なシナリオには以下が含まれます:
採用率の向上: すべての保留中のユーザーのCSVをアップロードし、登録を促すカスタマイズされたメールコミュニケーションを送信するためにリストを使用します。
新入社員のオンボーディング: 新しい従業員を迅速に重要なシステム、リソース、およびコミュニケーションチャンネルに追加するために、HRが提供するCSVリストをインポートします。
イベント参加者の追加: 会議、ウェビナー、またはワークショップの登録参加者をインポートして、スケジュールやリソースを共有します。
ターゲットを絞ったコミュニケーションの送信: セッション参加者、製品試用者、レガシーシステムユーザーなどの特定のユーザーグループのCSVをアップロードして、正確なフォローアップを行います。
