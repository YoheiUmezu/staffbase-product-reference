# 全期間の従業員メール統計のエクスポート

**URL**: https://support.staffbase.com/hc/ja/articles/9053063681426
**取得日**: 2026-05-02 18:58:56 JST

## 内容

管理者
従業員メールによって収集されたデータを、初めて使用された日から現在までの間に、あなたの組織のために表示します。
オプションとして、このデータをCSVファイルとしてエクスポートし、オフラインアーカイブを作成したり、自分のツールセットで分析したりできます。
前提条件
従業員メールの管理者または親管理者アカウントを持っていること
従業員メールを使用して、少なくともいくつかのメールが送信されていること
従業員メールの使用開始日を確認する
Webアプリ
メール Webアプリに移動し、メール > 送信済みメールに進みます。
表示がすべてのユーザーによって送信されたメールを全期間で表示し、すべて表示に設定されていることを確認します。
送信日でソートが選択されていることを確認します。
ソートの順序を昇順に選択します。
リストの最上部に表示されるメールの送信日は、従業員メールがあなたの組織のデータを収集し始めた最も早い日付です。
スタジオ
スタジオに移動し、メール > 送信済みメールに進みます。
表示がすべてのユーザーによって送信されたメールを全期間で表示し、すべて表示に設定されていることを確認します。
送信日でソートが選択されていることを確認します。
ソートの順序を昇順に選択します。
リストの最上部に表示されるメールの送信日は、従業員メールがあなたの組織のデータを収集し始めた最も早い日付です。
すべての統計レポートの生成とエクスポート
Webアプリ
メール Webアプリに移動し、レポートに進みます。
プリセットセクションで、すべての統計をクリックします。
レポートのための日付範囲を選択します。
全期間のすべての統計を表示するには、送信済みメールを確認して、あなたの組織が従業員メールを使用し始めた日を確認します。
終了日はデフォルトで現在の日付になります。
任意のレポート名を入力します。
レポートを生成をクリックします。
レポートを保存するには、保存アイコンをクリックします。
レポートをCSVファイルとしてエクスポートするには、ダウンロードアイコンをクリックします。
スタジオ
スタジオに移動し、メール > レポートに進みます。
プリセットセクションで、すべての統計をクリックします。
レポートのための日付範囲を選択します。
全期間のすべての統計を表示するには、送信済みメールを確認して、あなたの組織が従業員メールを使用し始めた日を確認します。
終了日はデフォルトで現在の日付になります。
任意のレポート名を入力します。
レポートを生成をクリックします。
レポートを保存するには、保存アイコンをクリックします。
レポートをCSVファイルとしてエクスポートするには、ダウンロードアイコンをクリックします。
