# プロフィールの編集

**URL**: https://support.staffbase.com/hc/ja/articles/207179675
**取得日**: 2026-05-02 18:48:12 JST

## 内容

ユーザープロフィールには、あなたの個人情報が含まれています。ユーザープロフィールでは、電話番号、メールアドレス、LinkedInのURL、その他の個人情報を提供できます。これにより、同僚があなたについてもっと知ったり、簡単に連絡を取ったりすることができます。あなたのプロフィールは、すべてのアプリユーザーに表示されます。
組織の設定によっては、あなたのプロフィールまたはその一部が読み取り専用である場合があります。
プロフィールを編集するには、以下の手順に従ってください：
Webアプリ
プロフィールメニューを開きます。
あなたの名前をクリックします。
プロフィールが開きます。
プロフィールを編集をクリックします。
編集モードが開きます。
情報を入力し、変更を保存をクリックします。
モバイルアプリ
アプリメニューを開きます。
あなたの名前をタップします。
プロフィールが開きます。
編集をタップします。
編集モードが開きます。
情報を入力し、保存をタップします。
追加の役立つ情報
ユーザープロフィールに画像を追加する
