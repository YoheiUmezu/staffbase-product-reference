# イントラネットからAtlassianアカウントにサインインする

**URL**: https://support.staffbase.com/hc/ja/articles/20502945298706
**取得日**: 2026-05-02 18:51:23 JST

## 内容

ベータ
アクティベーション
イントラネットからAtlassianアカウントにサインインして、Confluenceのコンテンツを検索およびアクセスできます。
イントラネットで、プロフィールメニューに移動します。
設定 > アカウントに移動します。
接続されたアカウントの下で、Atlassianのサインインをクリックします。
Atlassianのサインイン画面にリダイレクトされます。
Atlassianのサインイン画面で、ユーザー名とパスワードを入力します。
アプリを使用するサイトのドロップダウンメニューから、アプリを使用したいサイトを選択します。
承認をクリックして、アプリがAtlassianアカウントに接続するための権限を付与します。
イントラネットからAtlassianアカウントにサインインしました。同じセクションでサインアウトをクリックすることで、Atlassianアカウントからサインアウトできます。
また、Confluenceの検索タブやイントラネットの検索機能から直接Atlassianアカウントにサインインすることもできます。
