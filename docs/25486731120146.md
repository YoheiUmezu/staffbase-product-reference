# デスクトップユーザー向けの水平メニュー

**URL**: https://support.staffbase.com/hc/ja/articles/25486731120146
**取得日**: 2026-05-02 18:42:14 JST

## 内容

アクティベーション
デスクトップアプリのユーザーにとって、水平メニューバーはメニューのフライアウトや深いナビゲーションレベルを通じてコンテンツへのアクセスを容易にします。水平メニューをアクティブにすると、デスクトップユーザーにのみ影響します。モバイルデバイスでは、ナビゲーションは縦の配置のままです。フライアウトメニュー内の第一レベルの項目をクリックすると、メニューのパスの画像と説明が表示され、第二レベルの項目に分岐します。第二レベルの項目にマウスをホバーすると、次のレベルが表示されます。最大三レベルのフライアウトは、デスクトップユーザーにシームレスなナビゲーション体験を提供し、メニュー内の項目を明確に区分することでナビゲーションをより直感的にします。
Front Door Packageについて詳しく知りたい場合は、Staffbaseサポートまたはカスタマーサクセスマネージャーにお問い合わせください。
初めて水平メニューに切り替えると、ナビゲーションの外観と感触が変わります。フォルダーを活用してアプリメニューを再構築および再編成し、各スペースに対してコンテンツの整理がうまく機能することを確認してください。
