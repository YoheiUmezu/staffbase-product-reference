# 従業員メールパルス調査の使用

**URL**: https://support.staffbase.com/hc/ja/articles/4412581473426
**取得日**: 2026-05-02 18:54:07 JST

## 内容

従業員メールパルス調査を通じて、コミュニケーションの受取人から追加の洞察を得ることができます。メールを送信する際、メッセージの内容に関連して受取人からフィードバックを収集する機会があるかもしれません。
前提条件
管理者権限が必要です。

パルス調査の挿入
パルス調査を数秒で挿入できます！
マルチセクションまたはシングルセクションブロック内にパルス調査を設定することができます：
ブロックを選択します
調査要素を選択したセクションにドラッグします
パルス調査のスタイル
メール内にパルス調査を含める際、異なるスタイルの調査のいくつかの選択肢があります：
反応
反応：受取人がさまざまな絵文字やシンプルなサムズアップで反応する機会を提供します。
範囲
範囲：受取人がメールに対する感情をより詳細に提供するために、さまざまな反応を使用できるようにします。
バイナリ
バイナリ：直接的なフィードバックのためのシンプルな「はい」または「いいえ」スタイルの調査。
調査のカスタマイズ
パルス調査をカスタマイズすることで、伝えたい正確な内容とメッセージに調査を合わせることができます。
さまざまなカスタマイズオプションについて学ぶ
パディング
パディング：メール内の調査の周囲のスペースを調整します。これらの値は個別に変更することも、およびボタンを使用して5px単位で一括変更することもできます。
調査タイトル
調査タイトル：これは必須です。タイトルはメールのメトリクス内で表示され、公開されることはありません。
調査スタイル
調査スタイル：受取人の反応方法に関して異なるオプションを持つ、上記の調査タイプの中から選択します。
フィードバックページ
フィードバックページ：受取人が調査に回答した後にコメントを書くことを許可し、より多くのフィードバックを収集できます。ユーザーはフィードバックを匿名化するオプションがあります。
調査サイズ
調査サイズ：メール内の調査の投票ボタンのサイズを調整します。これは1pxから50pxの範囲です。
調査の追加カスタマイズオプション
調査をメールに追加する際に提示されるオプションの他に、調査をさらにカスタマイズするための追加のブランディングを使用できます。
前提条件
ブランディングをカスタマイズするには管理者権限が必要です。
ナビゲーション
調査カスタマイズページにアクセスします：
設定ページでブランディングをクリックします。
調査フィードバックをクリックします。
ブランドロゴ
ブランドロゴをカスタマイズします：
アップロードをクリックしてロゴを選択します。
ロゴの向きを選択します。
ロゴを更新すると、自動的に調査ページに表示されます。
透明な背景を持つ.pngファイルを使用すると、最良の結果が得られます。.opngファイルは、ブランディングおよび資産ガイドまたはマーケティング部門から直接入手できます。
多くの場合、組織のロゴは検索エンジンで簡単に見つけることができます。いくつかの高度な検索オプションを使用すると、主に透明な背景を持つ .png 画像を見つけるために検索を制限することができます。
メッセージング
フィードバックメッセージと感謝メッセージの両方をカスタマイズできます：
フィードバックメッセージまたは感謝メッセージをクリックします。
メッセージを更新します。
プレビューをクリックしてメッセージを確認します。
ユーザーがフィードバックを提供する方法
ユーザーは自分の名前とメールアドレスを提供するか、フィードバックを匿名化することができます。
