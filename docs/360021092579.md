# キャンペーンの概要

**URL**: https://support.staffbase.com/hc/ja/articles/360021092579
**取得日**: 2026-05-02 18:44:34 JST

## 内容

アクティベーション
キャンペーンは、戦略的なコンテンツ目標を設定、追跡、分析する方法です。ゼロからコンテンツ戦略を作成するのは大変ですが、キャンペーンを使えば簡単に始められます。すでに戦略がある場合は、データを統合する方法を探しているかもしれませんが、キャンペーンはコンテンツとメトリクスへのアクセスを1つの簡単なダッシュボードにまとめます。詳細については、Staffbaseサポートまたはカスタマーサクセスマネージャーにお問い合わせください。
キャンペーンの利点
戦略を立てる
コンテンツチームのために戦略的目標を明確かつアクセスしやすくします。作成するキャンペーンごとに、キャンペーン目標を追加できます。この目標は、キャンペーンの概要および詳細ビューに表示されるため、システム全体の編集者や管理者は、コンテンツを管理および評価する際に常に念頭に置くことができます。
コンテンツの作成と管理
投稿の編集タイムラインを一目で確認できます。この便利な概要を超えて、キャンペーンの詳細ビューから新しい投稿を直接作成したり、ドラフトを編集したりできます。各割り当てられた投稿の公開状況を示すラベルにより、公開済み、スケジュール済み、パイプラインにある投稿の数をすぐに確認できます。
分析する
キャンペーンでは、マクロおよびマイクロレベルでの分析を行うことができます。キャンペーン内でどの投稿が最も良い結果を出したか、また、より広いスケールでどのキャンペーンが読者の間で最もエンゲージメントを生み出しているかを並べて見ることができます。各投稿およびキャンペーン全体のいいね、コメント、訪問数などの統計がすぐに利用可能です。
コンテンツを戦略に合わせ、キャンペーンを通じてコミュニケーションがビジネス目標に与える影響を理解します。
キャンペーンの編集
キャンペーンを作成したら、ニーズに合わせていつでも編集できます。
スタジオで、プランニング > キャンペーン > 編集したいキャンペーンに移動します。
三点リーダーメニューをクリックします。
クリックして編集します。
キャンペーンを編集ダイアログが開きます。
キャンペーンの任意のセクションを編集します。
クリックして保存します。
キャンペーンを編集しました。変更はすぐに反映されます。
キャンペーンの削除
キャンペーンを削除すると、すべてのニュース投稿はキャンペーンから割り当て解除されます。投稿とその統計は、それぞれのチャンネルで利用可能なままです。削除されたキャンペーンは復元できません。
スタジオで、プランニング > キャンペーン > 削除したいキャンペーンに移動します。
三点リーダーメニューをクリックします。
クリックして削除します。
キャンペーンを削除ダイアログが開きます。
クリックして削除を確認するか、キャンセルして戻ります。
キャンペーンを正常に削除しました。このキャンペーンに以前割り当てられていた投稿はすべて割り当て解除されました。
