# SMS用のTwilioサブアカウントの作成

**URL**: https://support.staffbase.com/hc/ja/articles/16295596965010
**取得日**: 2026-05-02 18:51:29 JST

## 内容

テストアプリと本番アプリの両方がある場合、Twilioではそれぞれのサブアカウントを作成できます。各サブアカウントには、独自のTwilio電話番号が必要です。
Twilioコンソールにログインし、ダッシュボードに移動します。
アカウントをクリックし、サブアカウントを選択します。
サブアカウントの作成をクリックします。
Twilioサブアカウントが作成されました。
詳細はTwilioのドキュメントをご覧ください。
