# Staffbase Companion Editorを使用した記事のティーザーテキスト

**URL**: https://support.staffbase.com/hc/ja/articles/18055102356882
**取得日**: 2026-05-02 18:51:28 JST

## 内容

アクティベーション
エディター
ai
Staffbase Companion Editorは、コンテンツクリエイターが生成AIを使用してインパクトのあるニュース記事を作成するのをサポートします。Companion Editorは、少量の寄稿者の入力に基づいてニュースコンテンツを生成、編集、洗練することができます。これは、ニュースのArticleチャネルタイプのStaffbase Studioで利用可能です。Companion Editorの詳細を学ぶ。
StudioのArticleチャネルでCompanion Editorを使用すると、寄稿者の役割以上のユーザーは以下のことができます：
記事のテキストを生成、編集、ドラフト、要約、トーンを調整する
ティーザーテキストを生成する
タイトルを生成する
Companion Editorの生成AI機能は、Microsoft Azure OpenAI Serviceによって提供されています。Staffbase Companion Editorに特有のセキュリティに関する質問がある場合は、Staffbase Companion EditorセキュリティFAQで詳細を学んでください。
Staffbaseは、AI生成コンテンツを常にレビューし、公開前に特定のニーズに合わせて編集することを推奨します。
ティーザーテキストの生成
Companion Editorは、ニュースコンテンツの魅力的なティーザーテキストを作成するのを助けます。Article Textボックスにテキストを追加すると、Companion Editorはワンクリックで完璧なティーザーテキストを生成します。
Studioで、コンテンツ > ニュース > Articleチャネルに移動します。
投稿を作成をクリックして新しい記事を作成するか、編集をクリックしてドラフト記事を開きます。
ティーザーテキストボックスの上にあるAIで生成をクリックします。
AI生成のテキストがテキストボックスの下に表示され、レビューできます。
オプションで、再生成をクリックして異なるティーザーテキストを作成します。
このティーザーを挿入をクリックします。
テキストがティーザーテキストボックスに挿入されます。
ティーザーテキストを生成し、挿入しました。これで、記事の設定を完了できます。
