# ページの以前のバージョンを復元する

**URL**: https://support.staffbase.com/hc/ja/articles/360017402519
**取得日**: 2026-05-02 18:59:49 JST

## 内容

アクティベーション
管理者
ページの履歴には、最後に保存された50のページバージョンが表示され、選択したバージョンに復元できます。
ページ履歴は、2020年8月以前に作成されたページには利用できません。ページ履歴は2020年8月にプラットフォームに初めて導入されました。古いページバージョンは復元できません。
スタジオで、コンテンツ > ページに移動します。
以前のバージョンに復元したいページに移動します。
ページ履歴をクリックします。
復元したいバージョンを選択します。
次のオプションがあります：
現在のバージョンに戻り、未保存の変更を削除する
公開されているページの以前のバージョンを復元する
ページの以前のドラフトバージョンを復元する
復元または戻るをクリックします。復元するバージョンに応じて選択してください。
変更はすべての言語に影響します。ページは、選択したタイムスタンプで各言語でどのように見えていたかに復元されます。
ページが復元されました。
