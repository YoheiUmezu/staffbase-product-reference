# タイムゾーン設定の変更

**URL**: https://support.staffbase.com/hc/ja/articles/24213725987858
**取得日**: 2026-05-02 18:51:12 JST

## 内容

正確なタイムゾーン設定をプロフィールに維持することは、スケジュールされた通信が正しい時間に送信されることを保証するために重要です。ユーザーのプロフィールのタイムゾーン設定が実際の位置や現在の夏時間（DST）/標準時間と一致しない場合、スケジュールされたメールが正しい時間に送信されない可能性があります。
この記事では、メールを送信またはスケジュールする前に、プロフィールのタイムゾーン設定を確認および更新する方法を示します。
プロフィールのタイムゾーン設定の変更
従業員メールのウェブアプリで、あなたのイニシャルが表示されたアカウントアイコンに移動し、設定をクリックします。
従業員メール設定ページで、アカウントタイルをクリックします。
アカウント設定が開きます。
タイムゾーンがあなたの位置と一致しているか確認します。
このリンクを使用して、あなたの位置に適したタイムゾーンを見つけることができます。
必要に応じて、ドロップダウンメニューから正しいタイムゾーンを選択します。
保存して更新をクリックします。
あなたの位置に合わせてタイムゾーンが更新されました。
