# 統合の用語集

**URL**: https://support.staffbase.com/hc/ja/articles/360015287500
**取得日**: 2026-05-02 18:58:28 JST

## 内容

プラグインとは何ですか？
プラグインは、Staffbaseプラットフォームに特定の機能を追加できるミニアプリケーションのようなものです。これらはStaffbaseプラットフォームにシームレスに統合されます。ユーザーはプラグインを使用していることに気づかないかもしれません。なぜなら、プラグインはプラットフォームの一部として機能するからです。これらのプラグインはStaffbaseプラットフォームでの使用に最適化されており、シームレスなシングルサインオンを提供します。管理者は、プラグインをインストールすることができます。これは、すでに組織に利用可能な場合です。
ほとんどの場合、これらはプラグアンドプレイのように開発されていますが、特定の外部システムとStaffbaseプラットフォーム間の通信を確立するために、コネクタを選択する必要がある場合もあります。
また、プラットフォームにカスタム機能を追加するためのプラグインを構築することもできます。カスタムプラグインについて詳しく学んでください。
コネクタとは何ですか？
コネクタは、Staffbaseにインストールされたプラグインと統合されている広く使用されている外部システム間の通信を確立するために、APIやWebサービスプロトコルを自動的に消費する仲介者として機能します。コネクタは通常、組織のITチームによって設定されるべきであり、管理者がプラグインをそれぞれのシステムに簡単に接続できるようにします。
例：SAP HCMコネクタ
ウィジェットとは何ですか？
ウィジェットは、ページやニュースに特定のコンテンツブロックを追加するWebコンポーネントです。これらのWebコンポーネントは、ニュースやページエディタのデザイン要素として機能し、追加のレイアウトやテンプレートを使用せずに構造化された方法でコンテンツブロックを追加することを可能にします。これにより、同じページ上で異なる種類のコンテンツを整理された、消費しやすい方法で構造化することができます。
Staffbaseは、標準ウィジェットのセットと、サードパーティ統合を提供する他のウィジェットを提供しています。詳細については、こちらをご覧ください。
さらに、ウィジェットを構築して、ビジネスのユースケースに基づいてページやニュースにカスタムコンテンツブロックを追加することもできます。これらのカスタムウィジェットは、プラットフォームにインストールされると、Staffbaseが提供する他のウィジェットのように機能します。
