# 分析ハッシュタグメトリクスと分析ダッシュボード

**URL**: https://support.staffbase.com/hc/ja/articles/13519081402514
**取得日**: 2026-05-02 19:01:07 JST

## 内容

スペース管理者
分析ダッシュボードは、ユーザーがStaffbaseプラットフォームとどのように、いつ、どこで関わっているかについての深い洞察を提供します。
この記事では、ハッシュタグタブの詳細、グラフ、およびそれらに含まれる分析データについて説明します。このタブは現在ベータ版であり、アクティベーションによってのみ利用可能です。詳細については、Staffbaseサポートまたはカスタマーサクセスマネージャーにお問い合わせください。
ユーザー役割のアクセスを含む、全体の分析ダッシュボードについては、分析ダッシュボードの概要をご覧ください。
ハッシュタグリスト
ハッシュタグタブには、コンテンツに関する詳細なメトリクスを含むリストがあります。
ハッシュタグリストは、プラットフォームで使用されているハッシュタグの広範な概要です。このリストは、ニュース投稿、ページ、およびコメントで使用されているハッシュタグを表示します。ハッシュタグの詳細、ハッシュタグ結果ページについては、概要記事をご覧ください。
異なるテーブルヘッダーは次のとおりです：
ヘッダー
説明
ハッシュタグ
ハッシュタグの名前
投稿
ハッシュタグを含む投稿の数
投稿訪問数
ハッシュタグを含むすべての投稿のユーザー訪問の合計数
ハッシュタグクリック数
ユーザーがハッシュタグをクリックした回数またはハッシュタグ結果ページを訪れた回数
合計使用数
ハッシュタグを含むコンテンツアイテムの合計数
コンテンツアイテムは投稿、ページ、またはコメントである可能性があります
アクション
関連するハッシュタグ結果ページ
デフォルトでは、テーブルはハッシュタグ名でソートされています。投稿、投稿訪問数、ハッシュタグクリック数のヘッダーをクリックすることで、さらにテーブルをソートできます。
ハッシュタグリストのソート
ハッシュタグリストは、投稿、投稿訪問数、ハッシュタグクリック数の3つの異なるテーブルヘッダーでソートできます。
スタジオで、分析 > ハッシュタグに移動します。
ハッシュタグリストが表示されます。
リストをソートしたいヘッダーをクリックします。
リストは自動的に選択したヘッダーでソートされます。
異なるヘッダーでハッシュタグリストをソートしました。ハッシュタグタブから移動すると、結果は自動的にハッシュタグ名で表示されるようにリセットされます。
