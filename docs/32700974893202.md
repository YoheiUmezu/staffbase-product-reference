# 動画の字幕を管理する

**URL**: https://support.staffbase.com/hc/ja/articles/32700974893202
**取得日**: 2026-05-02 18:50:46 JST

## 内容

ベータ版
有効化
AI
動画の字幕は、AIを使用して新しくアップロードされた動画に自動的に生成されます。透明性を確保するため、AI生成のラベルが動画プレーヤーに一時的に表示されます。このラベルは、字幕が確認された後は表示されなくなります。
字幕の確認が必要な場合、編集者にも通知されます。
この通知は次の場所で確認できます：
ファイルマネージャーの動画タイル上
字幕エディター内の動画上
編集者は、正確性と品質を確保するために、これらの字幕を確認し、修正することを推奨します。
動画字幕へアクセスする
字幕を確認ラベルで、確認が必要な動画を特定できます。
Studioでファイルに移動し、編集したい動画を選択します。
三点リーダーをクリックして、その他のオプションメニューを開きます。
字幕を選択します。
動画字幕エディターが開きます。
字幕がまだ生成中の場合、生成には数分かかる旨のメッセージが表示されます。
動画字幕エディターにアクセスできました。
字幕を確認する
字幕が生成された後、誤字や名前、企業固有の用語を確認・修正できます。
動画の再生中、エディターはオーディオに合わせてスクロールし、文脈の中でテキストを確認しやすくなります。
字幕は動画の元の言語で作成され、自動的に英語、ドイツ語、フランス語、スペイン語に翻訳されます。ご注意ください：
元の言語で字幕を編集すると、すべての利用可能な翻訳が更新されます。
翻訳言語で字幕を編集すると、その言語のみが変更されます。
編集できるのはテキストのみで、タイムスタンプは変更できません。
字幕エディターで元の言語タブを選択します（推奨）。
字幕テキストを確認し、必要に応じて編集します。
保存するをクリックします。
字幕を編集しました。他の言語についても同じ手順を繰り返してください。
字幕を確認済みにする
視聴者向けのAI生成ラベルと編集者向けの字幕を確認ラベルは、字幕が確認済みとしてマークされた後にのみ消えます。
続行する前に、次のことを確認してください：
元の言語の字幕が確認され、保存されていること。
すべての翻訳が該当する言語の編集者によって確認されていること。
字幕エディターで、すべての言語バージョンが確認済みであることを承認します。
確認済みにするをクリックします。
字幕を確認インジケーターが削除され、エディター内で字幕を確認済みに置き換わります。誰が確認したかも確認できます。
