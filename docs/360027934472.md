# 投稿の公開

**URL**: https://support.staffbase.com/hc/ja/articles/360027934472
**取得日**: 2026-05-02 19:02:39 JST

## 内容

投稿を作成した後、ターゲットオーディエンスに表示させるために公開する必要があります。投稿は即時に公開することも、後で公開するようにスケジュールすることもできます。デフォルト設定で投稿を公開するか、設定をカスタマイズすることができます。
投稿はタイムゾーンの影響を受けます。たとえば、ベルリンにいる投稿者が10:00 CETに投稿を公開すると、ロンドンの従業員は9:00 GMTにその投稿を見ることができます。
投稿を即時に公開する
ドラフトビュー内で直接投稿を公開できます。
スタジオで、コンテンツ > ニュースに移動します。
作成した投稿を開きます。
オプションで、投稿の設定をカスタマイズすることができます。
今すぐ公開をクリックします。
投稿が公開され、ユーザーに表示されるようになりました。プッシュ通知またはメール通知のオプションを有効にした場合、ユーザーはすぐに通知を受け取ります。
投稿の公開をスケジュールする
事前に計画を立てて、投稿をスケジュールします。
スタジオで、コンテンツ > ニュースに移動します。
作成した投稿を開きます。
公開時間の下で、即時の隣にあるトグルを有効にします。
投稿を公開したい日付と時間を選択します。
オプションで、投稿の設定をカスタマイズすることができます。
スケジュールをクリックします。
投稿の公開をスケジュールしました。ドラフト内でいつでもスケジュールを更新またはキャンセルできます。プッシュ通知またはメール通知のオプションを有効にした場合、投稿が公開されるとユーザーに通知されます。
