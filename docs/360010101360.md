# ユーザープロファイルにおけるプロファイルフィールドの可視性の設定

**URL**: https://support.staffbase.com/hc/ja/articles/360010101360
**取得日**: 2026-05-02 18:50:16 JST

## 内容

管理者
部門、電話番号、所在地、またはその他のプロファイルフィールドなどの情報が、ユーザープロファイルに表示されるかどうかを決定します。
ユーザープロファイルに表示されない情報は、従業員ディレクトリに表示される可能性があります。機密情報については、ユーザープロファイルと従業員ディレクトリの両方で可視性を無効にすることを確認してください。
スタジオで、設定 > プロファイルフィールドに移動します。
ユーザープロファイルに表示したいプロファイルフィールドの隣にあるギアアイコンをクリックします。
プロファイルフィールドを編集するためのダイアログが開きます。
トグルユーザープロファイルに表示を調整して、プロファイルフィールドをユーザープロファイルに表示するかどうかを定義します。
OKをクリックします。
可視性設定が有効になると、プロファイルフィールドはユーザープロファイルから直接表示されます。
