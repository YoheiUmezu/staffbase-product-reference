# パスワードをリセットするにはどうすればよいですか？

**URL**: https://support.staffbase.com/hc/ja/articles/207179525
**取得日**: 2026-05-02 18:48:13 JST

## 内容

パスワードをリセットする方法は、アプリにサインインする方法と、アプリにメールアドレスを保存しているかどうかによって異なります。
中央サインインサービス（例：SSO）を介してサインインする場合：
この場合、中央サインインサービスを介してパスワードをリセットします。中央サインインサービスの管理者に連絡してください。
ユーザー名でサインインする場合：
リカバリーコードを手元に用意してください。
リカバリーコードがわからない場合は、アプリの管理者に連絡してください。
サインインダイアログで、パスワードを忘れた。をクリックします。
ユーザー名を入力します。
リカバリーコードを入力します。
パスワードをリセットをクリックします。
新しいパスワードを入力します。
メールアドレスでサインインする場合：
サインインダイアログで、パスワードを忘れた。をクリックします。
メールアドレスを入力します。アプリにサインインするために使用したメールアドレスを使用してください。
リカバリーコードが記載されたメールが届きます。
リカバリーコードを入力します。
パスワードをリセットをクリックします。
新しいパスワードを入力します。
