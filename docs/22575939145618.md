# 投稿の公開解除をスケジュールする

**URL**: https://support.staffbase.com/hc/ja/articles/22575939145618
**取得日**: 2026-05-02 19:02:43 JST

## 内容

寄稿者
投稿の公開解除をスケジュールすると、選択した日時にその投稿が自動的に下書きに戻ります。これは、イベントのリアルタイム更新など、期限があるコンテンツを整理し、プラットフォームを清潔に保つのに役立ちます。
記事が公開解除され、従業員に表示されなくなる日時をスケジュールすることが可能です。これは、公開するためにスケジュールするのと同時に行います。公開プロセス中に投稿の公開解除をスケジュールしない場合は、いつでも手動で投稿を公開解除することができます。
即時公開
スタジオで、 コンテンツ > ニュースに移動します。
公開解除をスケジュールしたい投稿の横にある 編集をクリックします。
公開解除時間の下で、 スケジュールされていない。の横にあるトグルを有効にします。
投稿を公開解除したい日時を選択します。
オプションで投稿の設定をカスタマイズします。
今すぐ公開をクリックします。
投稿が公開され、ユーザーに表示されるようになりました。選択した時間に公開解除されます。プッシュ通知またはメール通知のオプションを有効にした場合、ユーザーはすぐに通知を受け取ります。投稿が公開解除されたとき、ユーザーには通知されません。
スケジュールされた公開
スタジオで、 コンテンツ > ニュースに移動します。
公開解除をスケジュールしたい投稿の横にある 編集をクリックします。
公開時間の下で、 すぐにの横にあるトグルを有効にします。
投稿を公開したい日時を選択します。
公開解除時間の下で、 スケジュールされていないの横にあるトグルを有効にします。
投稿を公開解除したい日時を選択します。
オプションで投稿の設定をカスタマイズします。
スケジュールをクリックします。
特定の時間に公開および公開解除されるように投稿をスケジュールしました。下書きの中でいつでもスケジュールを更新またはキャンセルできます。プッシュ通知またはメール通知のオプションを有効にした場合、投稿が公開されたときにユーザーに通知されます。投稿が公開解除されたとき、ユーザーには通知されません。
