# Staffbase Emailにおけるファイル管理の概要

**URL**: https://support.staffbase.com/hc/ja/articles/24491248942610
**取得日**: 2026-05-02 18:42:36 JST

## 内容

StaffbaseメールのMy Filesを使用すると、チームで画像、動画、コレクションの共有ライブラリにアクセスできます。この連携により、繰り返し作業が減り、すべてのコミュニケーションで一貫したビジュアルを保つことができます。My Filesには、メールデザイナーから直接アクセスできます。メールに画像 や動画 要素を追加するには、プレースホルダーをクリックしてアップロードアイコンを表示します。
サポートされているメディアファイルサイズ
メール内でサポートされている画像の最大サイズは25MBです。
Staffbaseメールでの添付の最大サイズは7.5MBです。Staffbaseメールの添付についてさらに見る。
マイファイルからアップロード
アップロードアイコンをクリックすると、「マイファイル」が直接開き、メールにアップロードしたいファイルを選択できます。
インライン動画再生は、Gmail、Outlook（Windows および Outlook.com）、Yahooメールなど、ほとんどの主要なメールクライアントではサポートされていません。これはセキュリティやパフォーマンス上の制限によるものであり、メールプロバイダーによって強制されているため、Staffbase で設定や上書きはできません。
