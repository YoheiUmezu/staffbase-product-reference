# Staffbaseメールのフォールバックフォントの設定

**URL**: https://support.staffbase.com/hc/ja/articles/24343895873938
**取得日**: 2026-05-02 18:53:27 JST

## 内容

管理者
カスタムフォントをデフォルトまたはセカンダリフォントとして使用する場合に備えて、フォールバックフォントを定義できます。フォールバックフォントを定義しない場合、カスタムフォントがユーザーのメールクライアントで利用できないときは、デフォルトでメールクライアントはArialフォントにフォールバックします。
このオプションは、カスタムフォントをデフォルトまたはセカンダリフォントとして設定した場合にのみ表示されます。
スタジオで、設定 > メールに移動します。
メールブランディングページが開きます。
フォールバックフォントの下で、フォールバックとして追加したいフォントを選択します。
保存をクリックします。
カスタムフォントのフォールバックフォントが設定されました。いつでも変更できます。
