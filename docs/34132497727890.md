# 投稿の通知設定を行う

**URL**: https://support.staffbase.com/hc/ja/articles/34132497727890
**取得日**: 2026-05-02 19:01:34 JST

## 内容

ニュースは、スタッフベースアプリおよびイントラネットで、視聴者ごとに動的なコンテンツを公開できます。これは、ニュースページ、チャンネル、投稿を作成するの3層構造で構成されています。
ニュースチャンネルでは、編集責任者がすべての新しいニュース投稿記事に対してデフォルト設定を設定します。各投稿記事ごとに、有効な設定を個別にカスタマイズできます。これにより、各投稿記事ごとに公開やその他のオプションを調整できます。
プッシュ通知
ユーザーは、モバイルデバイスの従業員アプリに直接プッシュ通知を受け取ります。
メール通知
ユーザーは、アプリまたはイントラネットのユーザーアカウントに紐づけられたメールアドレスにメール通知を受け取ります。
ユーザーは個人設定で公開通知の設定を調整できます。投稿記事でプッシュ通知が有効になっていても、ユーザーが自身のアカウントで無効にしている場合は通知を受け取りません。
プッシュ通知を設定する
Studioでコンテンツ > ニュースに移動します。
通知を設定したい投稿記事の横にある編集するをクリックします。
編集ページが開きます。
公開の下で、プッシュのトグルを調整します。
保存するをクリックします。
メール通知を設定する
Studioでコンテンツ > ニュースに移動します。
通知を設定したい投稿記事の横にある編集するをクリックします。
編集ページが開きます。
公開の下で、メールのトグルを調整します。
保存するをクリックします。
