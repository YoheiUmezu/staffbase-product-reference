# 調査の概要

**URL**: https://support.staffbase.com/hc/ja/articles/14224399832722
**取得日**: 2026-05-02 18:43:30 JST

## 内容

従業員アプリやイントラネットの調査を使用すると、貴重な従業員の洞察やフィードバックを収集できます。調査を通じて、従業員にアプローチし、フィードバックを集め、プロセスを改善し、その後の変更を確認することができます。
通知により、ユーザーが定期的に参加しやすくなります。定期的にスケジュールされた調査リズムを選択することで、洞察を一度または時間をかけて定期的に収集できます。Staffbaseには、組織が選択した料金プランに応じて、2種類の異なる調査があります。
クイックパルス
クイックパルス調査はすべてのプラットフォームで利用可能です。これは、ユーザーが5つ星評価形式で回答を提出できる1問のユーザー調査です。従業員の健康状態を確認したり、新しい方針やプロセスの成功と受け入れを監視するために、クイックパルス調査を使用してください。
複数の質問
これらの調査は複数の質問を含んでおり、特定のトピックを深く掘り下げることで包括的な洞察を得るのに役立ちます。1つの調査内で追加、順序変更、編集できる5種類の質問タイプから選択できます。結果は質問ごとに分解され、リアルタイムで利用可能です。また、1つのトピックに関する洞察を時間をかけて収集する定期的な調査リズムか、1回限りの調査を選択できます。
