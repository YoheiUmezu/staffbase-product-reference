# メインメニューのコンテンツの配置

**URL**: https://support.staffbase.com/hc/ja/articles/360015613519
**取得日**: 2026-05-02 18:58:26 JST

## 内容

システム全体のエディター
メインメニューに表示されるメニュー項目の順序を選択できます。ユーザーは、スタジオで配置された内容をそのままメニューで見ることができます。このようにして、プラットフォーム内に論理的な階層を作成し、ユーザーが大量のコンテンツを簡単にナビゲートできるようにします。
スタジオで、 コンテンツ > メニュー に移動します。
プラットフォームのコンテンツを希望の位置にドラッグアンドドロップします。
新しく追加されたコンテンツは、デフォルトでメニューの下部に表示されます。これらのコンテンツを希望の位置にドラッグアンドドロップしてください。
保存 をクリックします。
コンテンツの順序が保存されました。
スタジオで配置されたコンテンツがプラットフォームに表示される方法
