# Microsoft 365 Teams フィードウィジェットの設定

**URL**: https://support.staffbase.com/hc/ja/articles/360021062599
**取得日**: 2026-05-02 18:54:16 JST

## 内容

アクティベーション
システム全体のエディター
Microsoft 365 ウィジェットを使用して、従業員アプリやイントラネット内で Microsoft Teams のすべての関連アクティビティやチャットメッセージの概要をユーザーに提供します。さらに、ウィジェット内の検索およびフィルター機能により、ユーザーは探しているチームアクティビティを迅速に見つけることができます。
前提条件
あなたは Microsoft 365 接続を Staffbase に追加しました。
あなたは アプリまたはイントラネットから Microsoft 365 にサインインしました。
あなたは Microsoft 365 Teams フィードウィジェットをアクティブ化しました。
Microsoft 365 Teams フィードウィジェットを追加します。
ウィジェットを設定するためのダイアログが開きます。
オプションで、タイトルを提供します。
タイトルはウィジェットの上に表示され、ユーザーに見えるようになります。
ウィジェットを表示できるデバイスタイプを、対応するアイコンをクリックして選択します。
ウィジェットを表示したいユーザーをアプリ内の可視性の下で選択します。
ウィジェットに表示するフィードアイテムの数を選択します。この数のアイテムが自動的にウィジェットに表示されます。さらにフィードアイテムはスクロールすることで読み込まれます。
OKをクリックします。
ウィジェットがページに追加されます。
変更を保存します。
Microsoft 365 Teams フィードウィジェットを設定して追加しました。
ユーザーが Microsoft 365 Teams フィードウィジェットを表示する方法
ウィジェットにアクセスする際、ユーザーはまだサインインしていない場合、従業員アプリまたはイントラネットでMicrosoft 365 アカウントにサインインするように促されます。
