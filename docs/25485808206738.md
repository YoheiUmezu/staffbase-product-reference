# StaffbaseメールのCSVファイル例

**URL**: https://support.staffbase.com/hc/ja/articles/25485808206738
**取得日**: 2026-05-02 18:42:21 JST

## 内容

ユーザーベースを最新の状態に保つことは、管理者の最も重要なタスクの一つです。ユーザーインポートと同期を使用すると、スタジオからCSVファイルを使って効率的にこれを行うことができます。StaffbaseでのCSVインポートについて詳しく学ぶ。CSVインポートでユーザーデータを管理する予定がある場合は、まずインポート用のCSVファイルを設定する必要があります。添付のCSV例をダウンロードし、自分のCSVファイルの出発点として使用できます。
すべてのユーザーが有効なメールアドレスを持っていることを確認してください。メールアドレスを持たないユーザーは連絡先としてインポートされますが、ターゲットオーディエンスを選択する際にメールデザイナーに表示されません。
CSVファイルの例
Staffbaseは、Staffbaseメールをスタンドアロン製品として使用するか、従業員アプリまたはイントラネットと組み合わせて使用するかに応じて、2種類のCSVファイルの例を提供しています。
CSVファイルの種類

言語付きCSVファイルの例
Staffbaseメールをスタンドアロン製品として使用する場合、ユーザーが好みの言語でメールを受け取るために、ユーザーの言語を設定する必要があります。これを行うには、CSVファイル内で言語ロケールを使用してユーザーの言語を追加します。
正しい言語ロケールを設定しないと、プラットフォームのデフォルトコンテンツ言語に戻ります。たとえば、de_DEの代わりにドイツ語を使用すると、デフォルト言語が英語に設定されている場合、英語に戻ります。
