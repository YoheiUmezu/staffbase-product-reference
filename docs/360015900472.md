# ユーザーにコンテンツ特有の権限を付与する

**URL**: https://support.staffbase.com/hc/ja/articles/360015900472
**取得日**: 2026-05-02 18:59:12 JST

## 内容

特定のコンテンツに対してユーザーに拡張権限を付与することが可能です。これにより、適切なコンテンツに対して適切なユーザーに作業を委任できます。たとえば、財務チームの更新用のチャネルに対して、財務担当者に 作成者 の権限を付与し、このチャネル内でコンテンツを作成および公開できるようにする一方で、財務マネージャーには同じチャネル内でコンテンツを管理および編集できる 編集者 の権限を付与することができます。
スタジオで、作成者または編集者を割り当てたいコンテンツに移動します。たとえば、ニュースチャネル、ページ、またはプラグインです。
設定を開くために ギアアイコン をクリックします。
編集者 または 作成者 の設定に移動します。
ドロップダウンメニューからユーザーまたはユーザーグループを選択します。
保存 をクリックします。
作成者は編集者の隣にある特別な役割です。ニュースチャネルには両方の役割があります。他のすべてのコンテンツタイプには、編集者のみを割り当てることができます。
寄稿者（作成者） は編集者よりも権限が少ないです。作成者の役割はコンテンツの寄稿専用に設計されており、他の寄稿者の投稿を編集または削除することはできません。
編集者 は拡張権限を持ち、チャネル内のすべての寄稿者の投稿を管理できます。
