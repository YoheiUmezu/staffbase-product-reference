# ニュースフィードウィジェットの設定

**URL**: https://support.staffbase.com/hc/ja/articles/25488184434962
**取得日**: 2026-05-02 18:59:31 JST

## 内容

エディター
ニュースフィードウィジェットを使用すると、ニュースチャンネルやニュースサイトからの投稿をページに追加できます。表示する投稿の数、投稿の並べ替え方法、ニュースフィードの表示レイアウトを定義することができます。
レイアウトオプションは、ニュースフィードのソースとしてニュースチャンネルを選択した場合にのみ利用可能です。以下のレイアウトから選択できます：
拡張レイアウト：このレイアウトは、大きなサムネイル、タイトル、および各投稿のティーザーテキストを詳細に表示します。メディアを前面に出すのに最適です。
コンパクトレイアウト：このレイアウトは、小さなサムネイル、タイトル、および各投稿のティーザーテキストを表示します。一目で多くの投稿を表示するのに理想的です。
標準レイアウト：このレイアウトは、拡張レイアウトとコンパクトレイアウトの組み合わせです。ニュースフィードの最初の投稿は拡張レイアウトで表示され、以降の投稿はコンパクトレイアウトで表示されます。
リストレイアウト：このレイアウトは、タイトルとオプションのメディアサムネイルを持つミニマリストリストを表示します。メディアサムネイルが有効になっている場合、ユーザーは再生アイコンのおかげで動画をすぐに認識し、直接再生できます。できるだけ多くの投稿を表示するのに最適です。
ニュースフィードウィジェットの一般設定の構成
ニュースフィードウィジェットを追加します。
ニュースフィードウィジェットを設定するためのダイアログが開きます。
オプションで、タイトルを提供します。
タイトルはウィジェットの上に表示され、ユーザーに見えるようになります。
ウィジェットが表示されるデバイスタイプを、対応するアイコンをクリックして選択します。
ウィジェットが表示されるユーザーをアプリ内の表示の下で選択します。
ウィジェットに表示するニュースページまたはチャンネルをソースの下で選択します。
ウィジェットにすべての利用可能な投稿を表示するか、限られた数の投稿を表示するかを表示タイプの下で選択します。
OKをクリックしてウィジェットを追加するか、高度な設定の構成を続けます。
ニュースフィードウィジェットの高度な設定の構成
ニュースフィードウィジェットを設定するためのダイアログで、高度な設定をクリックします。
オプションで、ウィジェット内の投稿の並べ替え方法を並べ替えの下で選択します。
ニュースフィードのソースとしてニュースチャンネルを選択した場合、投稿は自動的に時系列で並べ替えられます。ニュースページをソースとして選択した場合、チャンネル全体の時系列並べ替えとチャンネル特有の並べ替えのいずれかを選択できます。
オプションで、スキップ投稿の下で除外する投稿の数を入力します。たとえば、3と入力すると、ニュースフィードは4番目の投稿から始まります。
オプションで、チャンネル名を表示のトグルを有効にします。
オプションで、ニュースソースへのリンクのトグルを有効にし、リンクテキストを提供します。
リンクテキストはニュースフィードの下にボタンとして表示され、ユーザーを元のニュースソースにリダイレクトし、すべての利用可能な投稿を表示します。
ニュースフィードのソースとしてニュースチャンネルを選択した場合、オプションでレイアウトの下でフィードの表示方法を選択します。
OKをクリックして設定を保存し、ウィジェットをページに追加します。
