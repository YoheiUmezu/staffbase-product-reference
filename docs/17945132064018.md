# 高度な編集カレンダーでのキャンペーンの使用

**URL**: https://support.staffbase.com/hc/ja/articles/17945132064018
**取得日**: 2026-05-02 18:53:36 JST

## 内容

アクティベーション
エディター
高度な編集カレンダーでは、開始日と終了日を持つキャンペーンが表示されます。これにより、キャンペーンとニュース投稿を視覚的にまとめ、戦略的コミュニケーションの取り組みをより効率的に監視できます。詳細については、Staffbaseサポートまたはカスタマーサクセスマネージャーにお問い合わせください。
キャンペーン作成時に選択された色は、アクティブな日付の下にある編集カレンダーに表示されます。キャンペーンに割り当てられた投稿は、それぞれのキャンペーンカラーで色分けされます。編集カレンダーの投稿と同様に、キャンペーンに割り当てられたニュース投稿にアクセスできます。 詳細を学ぶ。
色が割り当てられていないが、開始日と終了日が割り当てられているキャンペーンは、編集カレンダーでデフォルトの薄い青色で表示されます。
キャンペーン詳細へのナビゲーション
スタジオで、プランニング > 編集カレンダーに移動します。
色付きのバーをクリックしてキャンペーンを開きます。 フライアウトメニューが開きます。
クリック キャンペーン詳細を開く。 個別のキャンペーン概要が開きます。
キャンペーン詳細に移動しました。キャンペーンを編集するか、ニュース投稿を開きます。
