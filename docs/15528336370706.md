# Staffbaseにサービスナウ接続を追加する

**URL**: https://support.staffbase.com/hc/ja/articles/15528336370706
**取得日**: 2026-05-02 18:49:20 JST

## 内容

有効化
管理者
Staffbaseプラットフォームとサービスナウを連携するために、サービスナウ接続を追加してください。
前提条件
あなた、またはサービスナウの管理者が、サービスナウで以下を生成している必要があります：
Staffbaseがサービスナウインスタンスにアクセスするためのエンドポイント
インスタンス名
クライアント ID
クライアントシークレット
Studio内で設定 > サードパーティ接続に移動します。
サードパーティ接続ページが開きます。
サービスナウの下で、追加するをクリックします。
サービスナウ接続を追加するダイアログが開きます。
サービスナウで生成した情報を、以下のフィールドに入力します。
インスタンス名：サービスナウインスタンスの名前を入力します。
クライアント ID：サービスナウで作成したクライアント IDを入力します。
クライアントシークレット：サービスナウで生成したクライアントシークレットを入力します。
カスタムURL：組織でカスタムサービスナウURLを使用している場合は入力します。
チケット作成URL：サービスナウチケットウィジェットから新しいチケットを作成するために使用されるフルURLを入力します。
ドロップダウンメニューからサービスナウ連携の表示を選択します。
追加するをクリックします。
サービスナウとStaffbaseの接続が正常に完了しました。この接続だけではサービスナウのコンテンツへのアクセスはまだ付与されません。ユーザーが利用できるようにするには、サービスナウでKnowledge APIをインストールし、Studioで有効にする必要があります。
