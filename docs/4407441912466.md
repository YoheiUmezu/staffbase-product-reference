# メニューにカスタムアイコンを追加する

**URL**: https://support.staffbase.com/hc/ja/articles/4407441912466
**取得日**: 2026-05-02 18:54:15 JST

## 内容

システム全体のエディター
アプリやイントラネットのメニュー項目をユーザーが簡単に識別できるように、カスタムアイコンを追加します。任意のメニュー項目に対して、ユーザーがアプリをナビゲートする際にメニュー名の横に表示される独自のアイコンや画像をアップロードできます。
スタジオで、カスタムアイコンを追加したいメニュー項目に移動します。
メニュー項目名の横にあるギアアイコンをクリックします。
メニュー項目を編集するためのダイアログが開きます。
チェックボックス  カスタムアイコン を選択します。
以下のアイコンをアップロードします：
ダークアイコン：これは、デスクトップメニューなどの明るい背景に対して使用されます。
ライトアイコン：これは、モバイルアプリメニューなどの暗い背景に対して使用されます。
プレビューでは、アイコンが明るい背景または暗い背景に対してどのように見えるかを確認し、コントラストが十分に高いことを確認できます。
OK をクリックします。
ダイアログが閉じます。
保存 をクリックします。
追加の役立つ情報
アプリ全体のカスタムアイコン
