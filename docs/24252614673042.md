# プロフィールホバーカードの概要

**URL**: https://support.staffbase.com/hc/ja/articles/24252614673042
**取得日**: 2026-05-02 18:56:42 JST

## 内容

ベータ
アクティベーション
プロフィールホバーカードは、従業員のユーザープロフィールのクイックビューを表示します。これは、イントラネットのさまざまなエリアで従業員に関する重要な情報を一目で提供します。ホバーカードを使用すると、特定の詳細を確認したり、完全なプロフィールページに移動したり、チャット会話を開始したりできます。
デスクトップユーザーは、プロフィールホバーカードで以下の情報を見ることができます：
プロフィールフィールド：従業員のユーザープロフィールからのプロフィールフィールド。表示される内容は各企業ごとに異なりますが、名前、アバター画像、役職、部門、所在地などが含まれる場合があります。プロフィールホバーカードはシステムプロフィールフィールドのみを表示し、カスタムプロフィールフィールドは表示しません。
プロフィールを見る：プロフィールを見るボタンをクリックすると、従業員の完全なユーザープロフィールが開きます。
メッセージ：メッセージボタンをクリックすると、従業員とのチャット会話が開きます。このボタンは、会社がStaffbaseプラットフォームでチャットを有効にしている場合にのみ表示されます。
プロフィールホバーカードは、イントラネット内の従業員の名前にカーソルを合わせることで見つけることができます。以下の場所で確認できます：
