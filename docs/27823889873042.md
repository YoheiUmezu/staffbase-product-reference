# HR 統合の概要

**URL**: https://support.staffbase.com/hc/ja/articles/27823889873042
**取得日**: 2026-05-02 18:41:26 JST

## 内容

有効化
Staffbase の軽量な HR 統合は、HR システムとのすぐに使える接続を提供します。これらの統合は Merge によって提供され、ユーザーはアプリやイントラネット内で給与明細や欠勤管理などの主要な HR インフォメーションにアクセスできます。
以下の HR システムと統合を設定できます：
Workday
SAP SuccessFactors
最大 3 つの HR 統合を設定できます。各統合は Studio 内で個別のエントリーとして表示されます。
複数の構成を使用して、よくあるシナリオに対応できます。例：
地域ごとに異なる統合：組織が国や地域ごとに異なる HR サービスを利用している場合があります。たとえば、Workday の統合を 2 つ、SAP SuccessFactors の統合を 1 つ作成できます。
同じ統合への複数の接続：子会社を持つ大規模な組織では、各子会社が同じ HR システムの独自インスタンスに接続している場合があります。たとえば、Workday の統合を 3 つ作成できます。
ユーザーごとのサービス接続：複数の部門で役割を持つユーザーは、異なる HR システムにアクセスできる場合があります。たとえば、ある部門は SAP SuccessFactors を使用し、別の部門は Workday を使用している場合です。複数の HR システムからデータが提供されている場合、ユーザーは希望するデータソースを選択できます。
メリット
迅速な設定
カスタム開発なしで Studio 内で簡単に統合を設定できます。
複数の HR システム対応
さまざまな HR システムと接続し、アプリやイントラネットでユーザーにリアルタイムデータを提供します。
統合の簡素化
Staffbase は Merge.dev を統合レイヤーとして使用し、接続された HR システムから API を集約しデータを同期することで、技術的な複雑さを最小限に抑えます。
HR 統合の設定概要
管理者は Staffbase と HR システム間の接続を確立する必要があります：
HR システムを統合のために準備する：HR システム内で許可や要件を設定します：
SAP Success Factors
Workday
HR 統合を追加して設定する。
HR 機能を有効にして、ウィジェットを通じて編集者が利用できるようにします。
ウィジェットがウィジェット追加ダイアログで利用可能になると、編集者はウィジェットをページに追加し、ページを公開してサービスをユーザーに提供できます。
ユーザー向けの HR 統合の見え方
利用可能な HR サービスに基づいて、ユーザーはアプリやイントラネットで給与や欠勤情報にアクセスできます。お知らせはデフォルトで有効化されていますが、ユーザーはアプリの通知設定でいつでもオフにできます。
