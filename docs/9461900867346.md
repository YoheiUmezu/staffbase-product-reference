# スタジオダッシュボードと分析ダッシュボードの読み込み問題の修正

**URL**: https://support.staffbase.com/hc/ja/articles/9461900867346
**取得日**: 2026-05-02 18:43:47 JST

## 内容

スタジオダッシュボードのタイルや分析ダッシュボードで「何かがうまくいかなかった」という読み込みエラーメッセージが繰り返し表示される場合、これは広告ブロッカーが原因である可能性が高いです。あなたの広告ブロッカーが、Staffbase Analytics APIがダッシュボードにデータを読み込むのをブロックしている可能性があります。
Staffbaseは、プラットフォーム上で貴重な分析情報にアクセスできるようにするために、以下のいずれかを推奨します：
アプリまたはイントラネットのドメインを広告ブロッカーの信頼済みサイトに追加する
広告ブロッカーが分析データをブロックしないようにするためには、アプリまたはイントラネットのドメインを広告ブロッカーの信頼済みサイトに追加する必要があります。
ブラウザの拡張機能リストを参照して、ブラウザにインストールされている広告ブロッカーを確認してください。
Google Chrome
Chromeブラウザで、メニューを開くために三つの点をクリックします。
メニューで、設定をクリックします。
拡張機能をクリックします。
インストールされている広告ブロッカーを見つけます。
Mozilla Firefox
Firefoxブラウザで、メニューを開くために三本の線をクリックします。
メニューで、設定をクリックします。
拡張機能とテーマをクリックし、拡張機能に移動します。
インストールされている広告ブロッカーを見つけます。
Safari
Safariを開き、Safariメニューバーを開きます。
Safari拡張機能をクリックします。
インストールされている広告ブロッカーを見つけます。
Microsoft Edge
Edgeブラウザで、メニューを開くために三つの点をクリックします。
メニューで、拡張機能をクリックします。
拡張機能の管理をクリックします。
インストールされている広告ブロッカーを見つけます。
今、広告ブロッカーのドキュメントを確認して、ドメインを信頼済みサイトとして追加できます。たとえば、AdBlockをブラウザの広告ブロッカーとして使用している場合は、ドメインで広告を許可するを確認してください。
広告ブロッカーを無効にする
広告ブロッカーはブラウザの拡張機能として機能します。ブラウザの設定で広告ブロッカーを無効にする必要があります。
ブラウザを選択し、広告ブロッカーを無効にする手順に従ってください：
Google Chrome
Chromeブラウザで、メニューを開くために三つの点をクリックします。
メニューで、設定をクリックします。
拡張機能をクリックします。
インストールされている広告ブロッカーを見つけ、トグルをスライドして無効にします。
Mozilla Firefox
Firefoxブラウザで、メニューを開くために三本の線をクリックします。
メニューで、設定をクリックします。
拡張機能とテーマをクリックし、拡張機能に移動します。
インストールされている広告ブロッカーを見つけ、トグルをスライドして無効にします。
Safari
Safariを開き、Safariメニューバーを開きます。
Safari拡張機能をクリックします。
インストールされている広告ブロッカーを見つけます。
開くをクリックして広告ブロッカーを選択します。
拡張機能ダイアログが開きます。
広告ブロッカーのチェックボックスをオフにします。
Microsoft Edge
Edgeブラウザで、メニューを開くために三つの点をクリックします。
メニューで、拡張機能をクリックします。
拡張機能の管理をクリックします。
インストールされている広告ブロッカーを見つけ、トグルをスライドして無効にします。
