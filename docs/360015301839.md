# カスタムビルド統合

**URL**: https://support.staffbase.com/hc/ja/articles/360015301839
**取得日**: 2026-05-02 18:54:21 JST

## 内容

使用している社内HRシステムが組織向けにカスタムビルドされている場合、Staffbaseがそれとの即使用可能な統合を提供していない可能性があります。そのような場合でも、即使用可能なソリューションがない場合、Staffbaseは特定の統合ニーズを満たすためにカスタム統合を構築することを可能にします。これらは、特定のビジネスケースを解決するために、自分自身またはベンダーや技術実装パートナーの助けを借りて構築できる統合です。カスタムビルド統合が完成したら、組織の要件に応じてStaffbaseプラットフォームで使用できます。
カスタムビルド統合は、ユーザー向けのソリューションをどのように想像するかに依存します。これには、プラットフォーム内でユーザー向けにソリューションを埋め込むことや、APIの助けを借りて外部システム内のデータを直接変更することが含まれます。カスタムビルド統合には、埋め込みと接続があります。
埋め込み
コンテンツを埋め込むことで、外部コンテンツを表示するための新しいユーザーインターフェースが提供され、コンテンツが従業員プラットフォームの一部としてユーザーに表示されます。この方法は、プラットフォームに埋め込むことを想像したソリューションが即使用可能な統合によって実現できない場合にのみ使用する必要があります。
プラットフォーム内で埋め込まれたコンテンツをカスタムビルドするには、以下の方法を使用できます：
カスタムプラグインによる埋め込み
非常に特定のビジネスケースを解決するために、自分自身のWebアプリケーションを開発します。このようなソリューションを社内で構築するか、ベンダーや技術実装パートナーの助けを得ることができます。その後、このWebアプリケーションをStaffbaseプラットフォームにカスタムプラグインとして埋め込むことができます。これは、Staffbaseが提供する標準プラグインと同様にシームレスに機能します。カスタムプラグインの詳細についてはこちらをご覧ください。
カスタムウィジェットによる埋め込み
カスタムウィジェットを使用して特定のビジネスケースを解決するために、自分自身のWebコンポーネントを開発します。カスタムウィジェットを開発したら、Staffbaseプラットフォームにインストールできます。カスタムウィジェットをインストールした後は、Staffbaseが提供する標準ウィジェットのように、ニュースやページのエディターに利用可能になります。カスタムウィジェットの詳細についてはこちらをご覧ください。
接続
APIを介してStaffbaseプラットフォームに接続を確立できます。APIを使用して、Staffbaseまたは外部プラットフォーム内のデータやコンテンツを追加、変更、抽出、または削除できます。
Staffbase APIによる接続
Staffbaseは、Staffbaseまたは外部プラットフォーム内のコンテンツと簡単に対話できる一般的なシナリオ用のAPIを提供しています。
