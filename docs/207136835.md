# メールを通じてアプリにユーザーを手動で招待する

**URL**: https://support.staffbase.com/hc/ja/articles/207136835
**取得日**: 2026-05-02 18:45:19 JST

## 内容

メール招待を使用すると、個々のユーザーに一時的なパスワードを手動で送信できます。このパスワードは一度だけ有効で、ユーザーがアカウントを設定できることを保証します。
スタジオで、ユーザー設定に移動します。
ユーザーを追加をクリックします。
新しいユーザーを招待するためのダイアログが開きます。
メール招待をクリックします。
ユーザーの名前とメールアドレスを入力します。
ドロップダウンメニューから、アプリ内のユーザーのアクセス権を選択します。
ユーザーを追加をクリックします。
ユーザーにランダムに生成された一時的なパスワードとアプリへのリンクを含むメール招待が送信されました。
追加の有用な情報
メールを通じてユーザーを追加ダイアログの詳細設定の下で、ユーザーの識別子、一時的なパスワード、またはユーザーグループへの事前追加を設定するオプションもあります。
ユーザーにカスタマイズされた招待を送信したい場合は、招待メールを調整の下で、メールの言語を英語とドイツ語の間で変更し、招待メールに個人的なメッセージを追加することが可能です。
