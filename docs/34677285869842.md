# Space グループでローカル視聴者を管理する

**URL**: https://support.staffbase.com/hc/ja/articles/34677285869842
**取得日**: 2026-05-02 19:00:02 JST

## 内容

有効化
スペース管理者
Space グループは、スペース管理者が自分のスペース内でグループを作成および管理できるようにすることで、ユーザーグループとスペースの概念を拡張します。
あなたはスペース管理者であるスペース内のみでグループを表示および管理できます。
自分の Space グループを表示する
複数のスペースのスペース管理者である場合、グループを管理したいスペースを選択する必要があります。
Studio で、ユーザー > グループに移動します。
Space グループ選択ツールから、表示したいスペースを選択します。
希望するスペースが選択されました。
スペースが選択されると、ユーザーグループと同じ方法でグループを管理できます。
スペース内のグループは次の方法で管理できます：
内部グループを作成する
オープングループを作成する
ユーザーグループにグループ管理者を追加する
グループメンバーシップを管理する
CSV を使用してグループメンバーシップを管理する
