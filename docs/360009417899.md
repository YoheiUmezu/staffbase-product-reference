# オープングループを通じてコンテンツにアクセスする

**URL**: https://support.staffbase.com/hc/ja/articles/360009417899
**取得日**: 2026-05-02 18:45:03 JST

## 内容

オープングループを利用すると、コンテンツの選択肢から自由に発見、購読、購読解除ができます。オープングループに購読すると、そのグループのために公開されたすべてのコンテンツ（投稿、カレンダー、イベント登録、チャットメッセージなど）にアクセスできます。オープングループのコメントで言及された場合、そのオープングループにアクセスしてコンテンツを発見できます。
オープングループは、ファンや音楽愛好者など、同じ興味を持つユーザーをつなげます。オープングループのもう一つの一般的な使用法は、異なるオフィスの場所のためです。これにより、異なるオフィスの場所を訪れる際にユーザーが簡単に購読し、重要なローカル情報にアクセスできます。
従業員アプリでオープングループをブラウズし、最初のオープングループに購読することができます。
オープングループに購読する
アプリメニューで、 マイグループ をクリックします。
利用可能なすべてのオープングループをリストしたダイアログが開きます。
右側で、購読したいグループを選択します。
閉じる をクリックします。
アプリのコンテンツが更新され、新しく追加されたオープングループのコンテンツが反映されます。新しいメニュー項目、プラグイン、またはチャットの会話が表示されることがあります。
オープングループから購読解除する
いつでもオープングループから購読解除できます。
オープングループに購読する際と同じ手順に従い、購読解除したいグループのチェックを外します。
