# SharePointサイトにアプリを追加する

**URL**: https://support.staffbase.com/hc/ja/articles/11077835469842
**取得日**: 2026-05-02 18:53:43 JST

## 内容

SharePointサイトのアプリカタログにアプリを追加できます。これにより、編集者はサイトのページに追加できるようになります。
次のいずれかを追加できます：
Staffbase 365 News Central
PnP Modern Search Extensibility Library
前提条件：
SharePointサイトオーナーまたはサイトコレクション管理者の役割を持ち、SharePoint Onlineサイトにアプリを追加できます。
追加したいアプリ（例：Staffbase 365 News CentralまたはStaffbase PnP Modern Search Extensibility Library）をすでにインストールしています。
Staffbase 365 News Centralを追加したいSharePointサイトに移動します。
+ 新規 をクリックし、 アプリを選択します。
マイアプリ ページが表示されます。
追加したいアプリをクリックします。
追加をクリックします。
アプリが追加され、設定できます。
