# Microsoft 365 サイトウィジェットの設定

**URL**: https://support.staffbase.com/hc/ja/articles/360021063959
**取得日**: 2026-05-02 18:50:07 JST

## 内容

アクティベーション
システム全体のエディター
Staffbase プラットフォームに Microsoft Sites の概要を追加するには、Microsoft 365 サイトウィジェットを使用します。ユーザーに、フォローしているサイトや SharePoint で表示する権限があるサイトの概要を提供します。さらに、ウィジェットの検索機能により、ユーザーはサイトを迅速に見つけることができ、レイアウト機能により、サイトをリスト形式またはタイル形式で表示できます。
前提条件
あなたは Microsoft 365 接続を Staffbase に追加しました。
あなたは アプリまたはイントラネットから Microsoft 365 にサインインしました。
あなたは Microsoft 365 サイトウィジェットをアクティブ化しました。
Microsoft 365 サイトウィジェットを追加します。
ウィジェットを設定するためのダイアログが開きます。
オプションで、タイトルを提供します。
タイトルはウィジェットの上に表示され、ユーザーに見えるようになります。
ウィジェットを表示できるデバイスタイプを、対応するアイコンをクリックして選択します。
アプリ内の可視性の下で、ウィジェットを表示したいユーザーを選択します。
ウィジェットに表示するサイトの数を選択します。この数のサイトが自動的にウィジェットに表示されます。さらにサイトはスクロールすることで読み込まれます。
オプションで、デフォルトで検索フィールドに表示される検索テキストを選択します。
OKをクリックします。
ウィジェットがページに追加されます。
変更を保存します。
Microsoft 365 サイトウィジェットを設定して追加しました。
ユーザーが Microsoft 365 サイトウィジェットを表示する方法
ウィジェットにアクセスする際、ユーザーはまだサインインしていない場合、従業員アプリまたはイントラネットでMicrosoft 365 アカウントにサインインするように促されます。
