# Microsoft 365 拡張機能の概要

**URL**: https://support.staffbase.com/hc/ja/articles/12223499964434
**取得日**: 2026-05-02 18:43:38 JST

## 内容

アクティベーション
管理者
Microsoft 365 拡張機能ページには、通信プラットフォームとの有意義な統合を構築するのに役立つすべての Microsoft 365 インストールパッケージが一覧表示されています。該当する統合の最新のインストールパッケージをダウンロードし、ファイルをインストールして統合を開始できます。
Microsoft 365 拡張機能ページには、以下の統合のダウンロードパッケージが含まれています：
Staffbase 365 ニュースセントラル
Staffbase PnP モダン検索拡張ライブラリ
Viva Connections の Staffbase ニュース
Microsoft 365 拡張機能ページには、上記の統合を構築するために必要なインストールパッケージが含まれています。これは、既存の Microsoft 365 ウィジェット、プラグイン、または 接続 の動作には影響しません。
統合パッケージのダウンロード
スタジオで、設定 > 拡張機能 > Microsoft 365 に移動します。
Microsoft 365 拡張機能 ページが開きます。
お好みの統合の ダウンロード をクリックします。
インストールパッケージファイルのダウンロードに成功しました。次に、該当する統合のセットアップ手順に従ってインストールを完了してください。
追加の役立つ情報
Staffbase 365 ニュースセントラルのインストールと構成の概要
Power Automate 用ニュース同期テンプレートの開始方法
