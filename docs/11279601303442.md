# ニュースチャンネルの並べ替え

**URL**: https://support.staffbase.com/hc/ja/articles/11279601303442
**取得日**: 2026-05-02 19:02:15 JST

## 内容

エディター
ニュースプラグインでは、公開済みおよび未公開のチャンネルを並べ替えたり、順序を変更したりすることができます。これにより、組織やエディターにとって最適な方法でチャンネルを整理できます。ニュースプラグイン内でチャンネルの順序を変更すると、従業員アプリやイントラネットのメニュー順序も変更されます。
ニュースチャンネルの概要に関する並べ替えプロセスの詳細は、この記事をご覧ください。
スタジオで、 コンテンツ > ニュースに移動します。
チャンネルをドラッグ＆ドロップして順序を変更します。
チャンネルの順序は自動的に更新されます。
ニュースチャンネルを並べ替えました。この変更はすべてのユーザーに表示されます。
