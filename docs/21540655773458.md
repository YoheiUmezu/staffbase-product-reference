# 高度な編集カレンダーでの月間ビューの使用

**URL**: https://support.staffbase.com/hc/ja/articles/21540655773458
**取得日**: 2026-05-02 18:46:25 JST

## 内容

アクティベーション
エディター
月間カレンダーのビューでは、週次ビューだけではなく、さらに先を計画することができます。広範で全体的な概要を使用して、どのキャンペーンがアクティブであるか、どのニュース投稿が公開されたか、または公開される予定であるか、すべてのブロッカーを確認できます。これにより、長期的なパターンを理解し、問題を特定し、コンテンツ計画を改善するのに役立ちます。
月間ビューでは、次のことができます：
計画されたニュース投稿、公開されたニュース投稿、すべてのキャンペーン、およびブロッカーを表示する
新しい投稿とブロッカーを作成する
ニュース投稿またはキャンペーンをクリックして、主要情報を含むフライアウトを表示する
ニュース投稿のタイトルをクリックして開く
キャンペーンのフライアウトからその詳細に移動する
キャンペーン、スペース、またはチャネルでビューをフィルタリングする
キャンペーンを表示または非表示にする
月間ビューへの切り替え
スタジオで、計画 > 編集カレンダーに移動します。
週次ドロップダウンに移動します。
月間を選択します。
月間ビューが表示されます。
編集カレンダーのビューを週次から月間に切り替えました。いつでも週次ビューに戻すことができます。
