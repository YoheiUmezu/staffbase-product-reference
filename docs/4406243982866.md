# Staffbaseサービスのステータス

**URL**: https://support.staffbase.com/hc/ja/articles/4406243982866
**取得日**: 2026-05-02 18:44:26 JST

## 内容

私たちのステータスページでStaffbaseサービスのステータスを確認してください:
https://status.staffbase.com/
アプリやイントラネットに問題が発生していますか？
例えば:
プッシュ通知が送信されない
メールが送信されない
コンテンツが読み込まれない、または非常に遅く読み込まれる
私たちは、あなたのアプリケーションが常に稼働していることを目指しています。サービスの稼働時間を最優先し、問題を調査し解決するためにチームが待機しています。
ステータスページでは、障害、遅延、またはその他の問題に関する情報を見つけることができます。常に最新情報を受け取るために、メールでの更新を購読してください。
報告されたインシデントがない場合でも、問題が発生した場合は、サポートチームにお問い合わせください。
