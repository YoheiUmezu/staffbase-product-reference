# Staffbaseメール内のクリックマップの概要

**URL**: https://support.staffbase.com/hc/ja/articles/27233672829842
**取得日**: 2026-05-02 18:53:11 JST

## 内容

Staffbaseメールのクリックマップは、受信者がメールコンテンツ内のリンクやボタンにどれだけアクションしたかを視覚的に表示します。各要素ごとにクリック率が直接ハイライトされるため、どの部分が最もエンゲージメントを集めているか、リンクやCTA（コールトゥアクション）ボタンの配置がパフォーマンスにどのように影響しているかを簡単に把握できます。クリックマップは、送信済みメールごとに専用のクリックマップタブで利用できます。
フィルター
クリックマップは、送信済みメールごとに自動で生成され、最後のクリックの日付と時刻が表示されます。また、以下のフィルターでデータを表示できます：
メールの言語（マルチ言語メールの場合）
ユーザーグループ
部門
位置
場所
主な指標
クリックマップでは、メールのパフォーマンスを一目で評価できる高レベルの指標も提供します：
受信者：メールを受信した受信者数。
クリック率：開封済みメールのうち、リンクがクリックされた割合。
合計クリック数：メール内のリンクがクリックされた合計回数（リピートとユニーククリックを含む）。
ユニーククリック：特定の受信者が初めてリンクをクリックした回数。
ビジュアルクリックマップ
タイルの下には、メールの画像が表示され、クリック率がコンテンツ上に直接重ねて表示されます。メール内でパフォーマンスが高いエリアと低いエリアを特定しやすくするため、クリックマップはどの要素が最もエンゲージメントを得たかを視覚的に示します。このデータは、インタラクションの強度に応じて色分けされています。色が濃いほどクリックボリュームが高いことを示します。
リンク概要
ビジュアルクリックマップの横には、メール内に含まれるすべてのリンクとボタンのリストと、それぞれのクリック率を示すテーブルが表示されます。テーブル内のリンクにカーソルを合わせると、プレビュー画像が自動的にそのリンクの位置までスクロールし、メールコンテンツ内の位置を簡単に見つけることができます。テーブルは、矢印ボタンでクリック率の割合順にソートできます。
アクションメニューから、リンクを開いたり編集したりできます。
すでに送信済みのメール内のリンクを変更した場合、更新は今すぐ反映されます。新しいリンクは、すでにメールを開封したり、以前のリンクをクリックした受信者も含め、すべての受信者に適用されます。
