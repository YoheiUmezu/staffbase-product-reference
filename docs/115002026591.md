# 会社のロゴ

**URL**: https://support.staffbase.com/hc/ja/articles/115002026591
**取得日**: 2026-05-02 19:00:00 JST

## 内容

アプリの設定ページに会社のロゴが表示されます。このロゴは、ログインページのロゴやアプリアイコンとは異なる場合があります。たとえば、色、透明度、形状に制限がないため、実際の会社のロゴを使用することもできます。
要件
サイズ: 幅720 px
ファイル形式: png
解像度: 72 dpi
カラープロファイル: sRGB
ファイル名: CompanyLogo.png
透明度: 許可されている
白い背景に合うように会社のロゴをスタイル設定してください
