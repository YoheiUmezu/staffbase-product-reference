# 初めてのアプリをApple App Storeに提出する

**URL**: https://support.staffbase.com/hc/ja/articles/115003482052
**取得日**: 2026-05-02 18:59:59 JST

## 内容

次のステップには、Application Loaderソフトウェアを搭載したMacが必要です。他のオペレーティングシステムで作業している場合は、Staffbaseサポートまたはカスタマーサクセスマネージャーに連絡して、さらなるサポートを受けてください。
Staffbaseがipaファイルを提供します。
App Store Connectにサインインします。
Macで、アプリケーションApplication Loaderを開きます。
ipaファイルをアップロードします。
これには数分かかる場合があります。アップロードが成功すると、成功メッセージが表示されます。
Appleがアプリを処理しており、アプリがApp Store Connectで利用可能になるまでに数分かかることがあります。
App Store Connectで、作成したアプリのプロファイルに移動します。
Buildセクションで新しくアップロードしたアプリを選択します。
すべての入力内容を再確認します。
Saveをクリックします。
すべての質問にNoで回答します。
Submitをクリックします。
アプリはAppleのレビュアーに送信されます。レビューのステータスはApp Store Connectで確認できます。レビューが成功すると、アプリはすぐにストアで利用可能になるか、指定した日付に利用可能になります。
