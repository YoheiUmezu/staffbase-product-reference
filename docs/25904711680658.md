# StaffbaseでのServiceNow機能の有効化

**URL**: https://support.staffbase.com/hc/ja/articles/25904711680658
**取得日**: 2026-05-02 18:55:08 JST

## 内容

有効にする
管理者
StaffbaseスタジオでServiceNow連携を有効にするには、ServiceNow環境をStaffbaseプラットフォームに接続する手順に従ってください。次の連携から選択できます：ServiceNowナレッジ検索：ユーザーがStaffbaseの検索を通じてServiceNowのコンテンツを検索できるようにします。ServiceNowチケット連携：ウィジェットを通じてユーザーがServiceNowチケットを閲覧できるようにします。
Studioで、設定 > サードパーティ接続に移動します。
ServiceNowの下で、展開アイコンをクリックします。
メニューが展開されます。
必要な連携のトグルをオンにします。
機能を有効にするダイアログが開きます。
有効にするをクリックします。
ServiceNow連携が正常に有効になりました。
初めてServiceNow連携にアクセスする際は、ServiceNowの認証情報でサインインして利用を開始してください。
