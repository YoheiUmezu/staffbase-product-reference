# 複数の質問を含む調査の公開スケジュールの設定

**URL**: https://support.staffbase.com/hc/ja/articles/27986840753426
**取得日**: 2026-05-02 18:53:08 JST

## 内容

エディター
従業員アプリやイントラネットの調査を使用すると、貴重な従業員の洞察やフィードバックを収集できます。
調査を公開する際には、次のいずれかを選択できます：
スケジュール設定
即時公開
調査をスケジュール設定すると、ユーザーにいつ利用可能になるかを事前に計画できます。一度限りの調査は特定の瞬間のスナップショットを提供しますが、定期的な調査では特定の質問に関するデータを時間の経過とともに比較できます。調査を即時に公開すると、ユーザーができるだけ早くフィードバックを受け取るために、すぐに利用可能になります。
調査が公開されると、もはや編集できません。
前提条件
複数の質問を含む調査を作成しています。
一度限りの調査のスケジュール設定
調査を作成したら、ユーザーにどの時期にどのくらいの期間利用可能にするかを選択します。
スタジオで、コンテンツ > 調査 > スケジュール設定したい調査を選択します。
サイドメニューの公開 の下で、調査をスケジュール設定を選択します。
スケジュールタイプの下で、一度限りを選択します。
調査の開始日と時間、終了日と時間を指定します。
オプションでプッシュ通知を送信の選択を解除します。
スケジュールをクリックします。
プッシュ通知が有効になっている場合は、スケジュールをクリックします 調査をスケジュール設定ダイアログで。
調査がスケジュールされました。調査が公開されると、ユーザーは通知を受け取ります。調査が終了すると、自動的に非公開になります。
定期的な調査のスケジュール設定
調査を作成したら、ユーザーにどの時期にどのくらいの期間利用可能にするか、また調査がどのくらいの頻度で繰り返されるかを選択します。
スタジオで、コンテンツ > 調査 >  スケジュール設定したい調査を選択します。
サイドメニューの公開の下で、調査をスケジュール設定を選択します。
スケジュールタイプの下で、定期的を選択します。
最初の調査の公開日と時間を指定します。
ドロップダウンメニューから調査の繰り返し頻度を選択します。
調査の期間の下で、ユーザーが調査にアクセスして回答できる期間を選択します。
オプションでプッシュ通知を送信の選択を解除します。
スケジュールをクリックします。
プッシュ通知が有効になっている場合は、スケジュールをクリックします 調査をスケジュール設定ダイアログで。
調査がスケジュールされました。新しい調査が利用可能になるたびに、ユーザーは通知を受け取ります。
調査を即時に公開する
調査を作成したら、即時に公開することを選択し、ユーザーにどのくらいの期間利用可能にするかを決定できます。この調査は一度だけ公開されます。定期的な調査はスケジュール設定する必要があります。
スタジオで、コンテンツ > 調査 >  スケジュール設定したい調査を選択します。
サイドメニューの公開の下で、今すぐ公開を選択します。
調査の終了日と時間を指定します。
オプションでプッシュ通知を送信の選択を解除します。
今すぐ公開をクリックします。
プッシュ通知が有効になっている場合は、公開をクリックします 調査を公開ダイアログで。
調査を公開しました。ユーザーは新しい調査が利用可能であることを通知されます。調査が終了すると、自動的に非公開になります。
調査スケジュールの編集
調査が公開される前に、調査スケジュールを編集できます。調査が公開された後は、もはや編集できません。
スタジオで、コンテンツ > 調査に移動します。
スケジュール設定したい調査の横にある編集をクリックします。
サイドメニューの公開の下で、スケジュールを編集をクリックします。
スケジュール設定オプションが開きます。
スケジュールを編集します。
クリックしますスケジュール。
調査のスケジュールを編集して更新しました。
調査スケジュールの削除
調査が公開される前に、調査スケジュールを削除できます。調査が公開された後は、スケジュールを削除することはできず、調査を終了することしかできません。
スタジオで、コンテンツ > 調査。
スケジュールを削除したい調査の横にある編集をクリックします。
サイドメニューの 公開 の下で、スケジュールの削除 をクリックします。
調査のスケジュールが削除されました。
調査の終了
公開された調査は、予定された終了日より前であればいつでも簡単に終了できます。
スタジオで、コンテンツ >調査 に移動します。
終了したい公開された 編集 をクリックします。
サイドメニューの 公開 の下で、調査の終了
調査は即座に終了し、公開が解除されます。ユーザーはもはやアクセスできません。
