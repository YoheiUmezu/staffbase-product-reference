# 利用可能なCSSセレクター

**URL**: https://support.staffbase.com/hc/ja/articles/207492945
**取得日**: 2026-05-02 18:58:34 JST

## 内容

カスタムCSS機能を使用すると、アプリのスタイルを手動で変更し、既存のスタイルルールを上書きできます。
スタイルをここで適用します: 設定 > ブランド設定 > カスタムスタイリング
カスタムCSSを使用すると、予測不可能で不正確な表示が発生する可能性があります。合理的な経験がある場合のみカスタムCSSを使用し、スタイルを適用した後は常にアプリの表示を再確認してください。
ターゲティングセレクター
特定のプラットフォーム、画面サイズ、または機能をターゲットにするためのクラスが多数あります。
これらはルートDOMノードに適用され、必要に応じて各セレクターの前に追加する必要があります。

ID特有のセレクター
特定のアプリコンテンツのIDをCSSセレクターに追加できます。詳細はアプリコンテンツのIDを特定するを参照してください。
IDをCSSセレクターと一緒に使用する方法の例
/* フィードとアーカイブ内の特定のチャンネルを選択 */
.feed .feed-channel.channel-56376cf7e4b07ab9337fad37,
.archive .feed-channel.channel-56376cf7e4b07ab9337fad37 { ... }
/* 特定のチャンネルの見出しを選択 */
.feed .feed-channel.channel-56376cf7e4b07ab9337fad37 > h2,
.archive .feed-channel.channel-56376cf7e4b07ab9337fad37 > h2 { ... }
/* 特定の投稿を選択 */
.feed .feed-post.post-56376db0e4b07ab9337fad3d,
.archive .feed-post.post-56376db0e4b07ab9337fad3d { ... }
/* 特定の単一投稿ビューを選択 */
.page.post-56376db0e4b07ab9337fad3d .page-content { ... }
/* 特定のページ（または他のプラグイン）を選択 */
.page.plugin-558c017ee4b029c8c2e50dbc .page-content { ... }
/* 特定のメニューページを選択 */
.page.menupage-55954d7ae4b0e81bc5b1d9bb .page-content { ... }
