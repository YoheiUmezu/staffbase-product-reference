# ニュース用プラグインチャンネル設定の概要

**URL**: https://support.staffbase.com/hc/ja/articles/33893189243666
**取得日**: 2026-05-02 18:59:20 JST

## 内容

ニュースは、スタッフベースアプリおよびイントラネットで、視聴者ごとに動的なコンテンツを公開することができます。これは、ニュースページ、チャンネル、投稿を作成するという3層構造で構成されています。
チャンネル内では、編集責任者がすべての個別ニュース投稿記事のデフォルト設定をカスタマイズできます。これらの設定は、チャンネル内で作成されるすべての投稿を作成するための基本構成となります。チャンネル設定の変更は、変更を保存した後に作成された投稿にのみ適用されます。すでに公開された投稿は変更されません。デフォルトはチャンネル設定でのみ設定でき、個別の投稿内で上書きすることはできません。
特定の設定が許可された場合、ニュース投稿者は必要に応じて有効または無効にすることができます。
レイアウト
記事チャンネルでは、3種類の異なるレイアウトから選択して、ニュースフィード内の記事を表示できます。各レイアウトは異なるユースケースに対応しています。
チャンネル機能
プラグインチャンネルの設定で、ソーシャルインタラクションや公開通知のデフォルトを定義できます。これらの機能には以下が含まれます：
コメント
ユーザーが投稿を作成するにコメントできます。コメントは投稿にアクセスできるすべてのユーザーに表示されます。コメントはエンゲージメントを促進し、トップダウン型のコミュニケーションをインタラクティブなものに変えます。
リアクションまたは「いいね」
ユーザーがワンクリックで投稿を作成するにリアクションできます。リアクションは投稿にアクセスできるすべてのユーザーに表示されます。ユーザーの反応を素早く把握したり、作成者への認知を与えたり、組織内のトレンドを特定するのに役立ちます。
承認する
ユーザーが投稿を作成するを読んで理解したことを承認する必要があります。主に重要な必須情報に使用され、コンテンツが確実に確認されたことをサポートします。
ハイライト表示
投稿を作成するが重要であることを、投稿の上部にバッジで即座にユーザーに示します。これにより、ニュースフィード内でより目立つようになります。
共有
ユーザーが投稿を作成するを社内の同僚や特定のプラットフォームを通じて外部に素早く共有できます。
プッシュ通知
ユーザーのモバイルデバイスの従業員アプリに直接通知を送信します。
メール通知
ユーザーのアカウントにメールアドレスが登録されている場合、メールで通知を送信します。
作成者を表示
投稿の作成者名をニュース投稿記事の末尾に表示するように設定できます。
関連ニュースページ
ニュースページはニュースチャンネルのコレクションです。ニュースページには任意の数のチャンネルを含めることができ、チャンネルを任意の数のニュースページにリンクできます。
パンくずリスト
パンくずリストは、ユーザーがプラットフォーム内の自分の位置を理解するのに役立ちます。ユーザーがプラットフォーム内の場所を簡単に移動できるナビゲーションパスを提供します。
ソース（RSSフィード）
RSSフィードを使用して、外部ウェブサイトのコンテンツをStaffbaseプラットフォームに表示できます。記事または更新するニュースチャンネルタイプで利用できます。RSSフィードは独立したニュースチャンネルとしても、組織独自のニュースコンテンツと組み合わせて使用することもできます。
