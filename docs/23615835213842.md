# オフラインで表示するためのページのダウンロード

**URL**: https://support.staffbase.com/hc/ja/articles/23615835213842
**取得日**: 2026-05-02 18:46:22 JST

## 内容

ベータ
アクティベーション
オフラインモードを使用すると、従業員アプリでページをダウンロードでき、インターネット接続なしで重要な情報にアクセスできます。安全プロトコルや人事ガイドラインなど、特定のページを保存して、オフラインのときでもいつでも表示できます。 詳細はこちら。
タイトルの横にあるアイコンをタップすることで、簡単に任意のページをダウンロードできます。また、ダウンロード済みのアイコンを選択して確認することで、既存のダウンロードページを削除できます。オフラインのときは、ダウンロードしたすべてのページのリストを整理された概要で表示できるため、必要なものを簡単に選択してアクセスできます。
オフラインでページを表示しているときは、テキスト、画像、PDFにアクセスできます。ただし、リンクやページのインタラクティブな部分、従業員アプリの他の領域にはアクセスできません。
ページのダウンロード
従業員アプリで、ダウンロードしたいページに移動します。
ページタイトルの横にあるダウンロードアイコンをタップします。
ページがダウンロードされます。
ページがダウンロードされました。インターネット接続なしでアプリを開くと、ダウンロードした他のページのリストにそのページが表示されます。タイトルの横にあるダウンロードアイコンをタップすることで、ダウンロードしたページをいつでも削除できます。
ダウンロードしたページをオフラインで表示する
従業員アプリを開きます。
オフラインで利用可能なダウンロードページのリストが表示されます。
開きたいページをタップします。
ページが開きます。
オフラインでダウンロードしたページを表示しました。コンテンツは、オンラインのときにページを再度開くまで同じままであり、その際に更新があればリフレッシュされます。
