# ユーザーページの概要

**URL**: https://support.staffbase.com/hc/ja/articles/27820426774802
**取得日**: 2026-05-02 18:41:29 JST

## 内容

スタッフベースでは、複数の方法でユーザーを作成することができます。これらの方法で作成されたすべてのユーザーは、Studio のユーザーページに表示されます。主な方法は次のとおりです。
CSV インポート
SCIM
SSO
ユーザー API
手動で作成
Studio のユーザーページは、管理者がすべてのユーザーを一元的に確認できるページです。ユーザーは、ステータス、役割、作成方法、ユーザータイプごとに分類されます。これにより、管理者はユーザー管理に必要な情報を簡単に把握できます。
ユーザー概要ページでは、次のことができます。
特定のユーザーアカウントを検索する
詳細なユーザープロフィールや所属グループを表示する
ユーザーアカウントを編集または削除する
役割、ステータス、グループ、作成方法、ユーザータイプでユーザーを絞り込む
作成日、最終編集日、またはアルファベット順でユーザーを並べ替える
このシンプルな操作により、ユーザーデータの監視、管理、アクションが一元的に簡単に行えます。
ユーザー概要のメリット
グループ管理の一元化
すべてのユーザーを一か所で表示・管理でき、複数のページを移動する手間を減らします。
重要な情報への素早いアクセス
役割、ステータス、グループ、作成方法、ユーザータイプで簡単に検索、絞り込み、並べ替えができ、目的のユーザーをすぐに見つけられます。
アクションの効率化
概要ページから直接、編集、削除、ユーザー詳細やグループ割り当ての確認など、よく使う作業をすばやく行えます。
