# ニュースチャンネルの公開範囲を更新する

**URL**: https://support.staffbase.com/hc/ja/articles/33742475672850
**取得日**: 2026-05-02 19:01:48 JST

## 内容

編集責任者
ニュースは、スタッフベースアプリおよびイントラネットで視聴者ごとに動的なコンテンツを公開することができます。これは、ニュースページ、チャンネル、投稿を作成するの3層構造で構成されています。
チャンネルの表示設定では、ニュースチャンネルの対象グループを選択できます。チャンネル内のすべての投稿を作成するは同じ表示設定となります。表示設定はチャンネル作成時に設定しますが、いつでもチャンネルの設定で調整できます。
スタッフベースでは、表示設定を構成する際にチャンネルで公開するコンテンツの種類を考慮することを推奨しています。たとえば、チャンネルで北米向けの更新するを投稿する場合は、北米のユーザーやユーザーグループに表示設定を設定してください。
チャンネルの表示設定を更新する
Studioで、コンテンツ > ニュースに移動します。
表示設定を更新したいチャンネルを開きます。
歯車アイコンをクリックして設定を開きます。
設定ページが開きます。
アプリでの表示のドロップダウンを開きます。
チャンネルを表示するユーザーやユーザーグループを選択します。
保存するをクリックします。
チャンネルの表示設定を更新しました。設定はいつでも調整できます。
