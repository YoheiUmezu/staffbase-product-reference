# 従業員メールでのカスタム配信リストの作成

**URL**: https://support.staffbase.com/hc/ja/articles/4412584449426
**取得日**: 2026-05-02 18:44:17 JST

## 内容

あなたが 従業員メール ディレクトリに連絡先をアップロードまたは同期した後、異なる属性に基づいてカスタム配信リストを作成するためにそれらをフィルタリングできます。
配信リストは、同じタイミングで同じコミュニケーションを受け取る必要がある連絡先のグループです。たとえば、バンクーバーにいる連絡先はすべて同じ配信リストに含まれます。
属性は、受信者を他の受信者と区別する特性であり、これによりオーディエンスをセグメント化し、特定のグループに対してコミュニケーションをターゲットにすることができます。属性に基づいてフィルタリングすることで、データを完全に制御し、選択したセグメントにコミュニケーションを送信できます。
従業員メール ウェブアプリには、配信リストの共有オプションがあり、個々のユーザーやグループがカスタムおよび同期された配信リストにメールを送信したり、編集したりする権限を制御できます。
注意: この機能は、従業員メール ウェブアプリから直接メールを送信する場合にのみ利用可能です。アドインや拡張機能をインストールし、メールクライアントから追跡されたメールを送信する場合は、この文書は関連しません。
前提条件
あなたの組織は、以下のいずれかの方法を使用して 従業員メール ウェブアプリに連絡先を追加しました:
カンマ区切り値 (CSV) ファイルのアップロード
Microsoft Entra ID (以前の Azure AD) との自動同期の設定
Workdayとの自動同期の設定
あなたの 従業員アプリ/イントラネット からの自動同期の設定
カスタム配信リストの種類
人々は、自分に明確に関連するメールを開く可能性が高く、特に自分のタイムゾーンにとって便利な時間に受信トレイにアクセスできるときにそうなります。
プライベートなカスタム配信リストを作成する柔軟性を持つことで、さまざまな方法で連絡先をフィルタリングして、組織に最適な方法を見つけることができます。
単一のフィルターに基づいてシンプルなリストを作成できます。たとえば:
特定のチーム/部門
管理層
オフィスの場所
または、複雑なリストを作成することもできます。たとえば:
特定の場所を除く同じ管理層の連絡先
特定の場所の最近の採用者、特定の職種を除く
カスタム配信リストの作成
従業員メール ウェブアプリの連絡先ページで、フィルターを作成をクリックします。
ターゲットにするグループを定義するために含めるおよび/または除外する属性を選択します。
作成しているフィルターに追加するには、適用をクリックします。
注意: メール 属性を選択し、特定の連絡先をキーワードやメールアドレスの文字で検索するオプションがあります。
オプションで、フィルタリングしたい属性のサブグループを追加します。
クリック リストとして保存。
リストに名前を付けます。
動的 または 静的 としてリストを保存します。
注意: 動的リストは、連絡先の属性が変更されると常に更新されます。
静的リストは、属性が変更されても手動で編集された連絡先のリストを保持します。
カスタム配信リストが作成され、配信リスト タブで表示、編集、共有できます。
これで、カスタムリストにメールを送信できます。
配信リストをCSVファイルとしてエクスポートする
配信リストをCSVファイルとしてエクスポートする方法の詳細については、この記事をお読みください。
