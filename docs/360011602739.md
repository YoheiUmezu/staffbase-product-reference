# 確認ダイアログに法的文書のリストを追加する

**URL**: https://support.staffbase.com/hc/ja/articles/360011602739
**取得日**: 2026-05-02 18:44:51 JST

## 内容

ベータ
アクティベーション
管理者
法的文書の確認を有効にしたら、確認ダイアログに法的文書のリストを直接追加できます。これにより、ユーザーは新しいおよび更新された法的文書を探す必要がなくなり、確認ダイアログから直接簡単にアクセスできるようになります。
この法的文書処理のベータ版は改善中であり、アクティベートする必要があります。この機能を最適に使用するために、Staffbaseサポートまたはお客様のカスタマーサクセスマネージャーがすべての要件を満たしているか確認します。たとえば、すべてのモバイルアプリユーザーが最新のアプリバージョンを持っている必要があります。詳細については、Staffbaseサポートまたはカスタマーサクセスマネージャーにお問い合わせください。
前提条件
法的文書を作成しています。
法的文書の確認を有効にしています。
法的文書へのリンク
スタジオで、設定 > 法的文書に移動します。
確認設定をクリックします。
以前に作成した確認ダイアログのメッセージが開きます。
ユーザーのための短い確認メッセージを提供します。
このメッセージは法的文書のリストの上に表示されます。以前の法的文書処理のバージョンからアップグレードする場合、波括弧を使用してリンクされた以前の法的文書は手動で削除する必要があります。たとえば、{データ保護契約}。
確認ダイアログでリンクする必要がある文書を選択します。
以前の法的文書処理のバージョンからアップグレードする場合、以前にリンクされた法的文書は確認ダイアログから削除されます。
文書をリンクするために、関連する法的文書を再度選択する必要があります。
確認のプレビューリンクをクリックして確認ダイアログのプレビューを表示します。
プレビューは別のダイアログで開きます。
プレビューモードでは、すべての必要な法的文書がリンクされていることを確認してください。
各ユーザーは、法的文書の可視性設定に応じて異なる文書のリストを表示します。
受け入れるをクリックしてプレビューを閉じます。
保存をクリックします。
異なるユーザーが確認ダイアログをどのように表示するか
確認ダイアログ内に表示される法的文書のリストは、文書がユーザーにとって関連性があるかどうかに応じて、各ユーザーによって異なります。たとえば、ユーザーAは3つの異なる法的文書を読み、確認する必要がありますが、ユーザーBは1つだけです。したがって、各ユーザーは確認ダイアログで異なる法的文書を受け取ります。
