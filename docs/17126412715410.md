# ユーザーのプライマリメールアドレスの編集

**URL**: https://support.staffbase.com/hc/ja/articles/17126412715410
**取得日**: 2026-05-02 18:55:31 JST

## 内容

管理者
ユーザーのプライマリメールアドレスは、アカウント作成プロセス中にユーザーのメールアドレスを提供すると表示されます。 詳細はこちら。
プライマリメールアドレスは、オンボーディング中に管理者またはITチームによって提供された場合、スタジオのユーザー編集ダイアログで利用可能です。メールアドレスが利用できない場合は、新しいメールアドレスを提供して保存できます。ただし、ユーザーが登録時に個人のメールアドレスを追加した場合、メールフィールドは編集できません。
スタジオで、ユーザー管理 > 登録ユーザーに移動します。
また、非アクティブユーザーや保留中のユーザーに移動することもできます。
プライマリメールアドレスを更新したいユーザーの編集をクリックします。
編集ダイアログが開きます。
メールフィールドに追加したいメールアドレスを入力します。
保存をクリックします。
ユーザーのプライマリメールアドレスを保存しました。これからは、ユーザーがプラットフォームからのメール通知を受け取る際やパスワードリセットをリクエストする際に、新しいメールアドレスが使用されます。
