# スタッフベースメールでのスケジューリング

**URL**: https://support.staffbase.com/hc/ja/articles/24491514294802
**取得日**: 2026-05-02 19:00:12 JST

## 内容

この記事はAIを使用して翻訳されており、不正確な情報が含まれている可能性があります。情報に不安がある場合は、正確性のために英語の原文を参照してください。
編集者
スタッフベースメールでメールをスケジュールすることで、効率的にコミュニケーションを計画し自動化できます。特定の日付と時間にメールをスケジュールする前に、以下を行うことができます:
ターゲット視聴者、送信者の詳細、およびキャンペーンを設定する
件名とプレビューテキストを追加し、AIを活用した提案を使用するオプションを利用する
必要に応じて添付ファイルを追加する
メールスケジュールを数回のクリックでスケジュール、編集、または削除する方法を学びましょう。
メールをスケジュールする
Studioで、メール > スケジュールしたいメールに移動します。
メールを開き、次をクリックします。
サマリーページが開きます。
設定セクションで、ターゲット視聴者を選択します。
メールが作成されたフォルダに割り当てられたユーザーとグループのみを選択できます。すべてのフォルダユーザーを選択するか、ドロップダウンメニューから個々のユーザーとグループを手動で選択できます。
次のフィールドに正確な情報が入力されていることを確認します:
送信者アドレス
送信者名
返信オプション
キャンペーン
件名
プレビューテキスト
添付ファイル
送信オプションセクションで、後でスケジュールを選択します。
ダイアログが展開されます。
メールをスケジュールしたい日付と時間を選択します。
スケジュールをクリックします。
確認ダイアログが開きます。
スケジュールをクリックします。
メールのスケジュールが正常に設定されました。送信されていない限り、このメールのスケジュールを編集または削除できます。
スケジュールの削除または編集
Studioで、メール > スケジュールされたメールが作成されたフォルダに移動します。
スケジュール済みタブを開きます。
編集したいメールで、その他のオプションメニューをクリックします。
ドロップダウンメニューから再スケジュールを選択します。
サマリーページが開きます。
送信オプションの下で、スケジュールを削除をクリックします。
スケジュールを削除ダイアログが開きます。
削除をクリックします。
オプションで、新しい日付と時間を追加し、スケジュールをクリックします。
新しいスケジュールを追加しない場合は、保存をクリックします。
このメールのスケジュールが正常に削除および編集されました。
