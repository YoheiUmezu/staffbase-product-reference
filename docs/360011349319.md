# コンテンツのURLを特定する

**URL**: https://support.staffbase.com/hc/ja/articles/360011349319
**取得日**: 2026-05-02 18:47:46 JST

## 内容

コンテンツにリンクするには、コンテンツのURLを取得する必要があります。特定のコンテンツのURLをあなたのアプリで特定する方法は2つあります。
デスクトップまたはモバイルブラウザーでアプリを使用している場合にのみ、URLを特定することが可能です。モバイルアプリではコンテンツのURLは表示されません。
Webアプリを通じてコンテンツのURLを特定する
Webアプリで、URLを取得したいコンテンツに移動します。
ブラウザーからURLをコピーします。
スタジオを通じてコンテンツのURLを特定する
スタジオで、リンクしたいページ、ニュース記事、またはプラグインを開きます。
プレビュー モードに移動します。
プレビュー ダイアログが開きます。
プレビュー画像の下にあるリンクを右クリックし、コピーをクリックします。
