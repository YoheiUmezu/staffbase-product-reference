# クイズの概要

**URL**: https://support.staffbase.com/hc/ja/articles/26065979906450
**取得日**: 2026-05-02 18:41:45 JST

## 内容

システム全体の編集者
この機能はアドオンです。詳細はStaffbase の価格をご参照ください。有効化については Staffbase サポートまたはカスタマーサクセスマネージャーにお問い合わせください。
クイズプラグインは Staffbase エンプロイーエクスペリエンス（EX）ブースタープラグインです。
フレンドリーなコンペティションを実施する場合や、eラーニングコンテンツを提供する場合でも、クイズプラグインは参加を促す実績のある方法です。無制限の質問数と最大4つの回答オプションで、ユーザーに柔軟で魅力的な体験を提供します。各回答後に即時かつカスタマイズ可能なフィードバックを提供でき、最後に参加者のランキングを表示して競争心を高めることができます。
デスクトップとモバイル向けに設計されており、クイズプラグインはあらゆるワークフローにシームレスに組み込むことができます。一度限りまたは匿名での参加をサポートし、結果はCSVファイルとして簡単にエクスポートでき、さらなる評価に活用できます。ランダムな質問モードや再利用可能なテンプレートにより、クイズを常に新鮮でダイナミックに保ち、プラットフォームでのエンゲージメントを高めます。
