# Gmailでの複数のエイリアスの取り扱い

**URL**: https://support.staffbase.com/hc/ja/articles/4415102172178
**取得日**: 2026-05-02 19:00:21 JST

## 内容

すでにGmailで複数のメールアドレスから送信している場合、Employee Email拡張機能でそれらを使用するオプションもあります。
Gmailで、Employee Email拡張機能の設定ボタンをクリックし、次にエイリアス管理をクリックします。
複数のエイリアスを使用する際のオプション：
プライマリアカウントから追跡すること。つまり、john@gmail.comにログインしてエイリアス(johndoe@gmail.com)からのメッセージを追跡すると、そのメールはEmployee Emailのjohn@gmail.comアカウントで追跡されます。これはあなたのプライマリ Gmailアカウントです。
独自のアカウントに追跡すること。つまり、john@gmail.comにログインしてエイリアス(johndoe@gmail.com)からのメッセージを追跡すると、そのメールはEmployee Emailのjohndoe@gmail.comアカウントで追跡されます。このオプションでは、エイリアスとプライマリアカウントは完全に別々であることに注意してください。
注意: あなたの追跡オプションはローカルに保存されます。異なるコンピュータやブラウザを使用する場合、または履歴/キャッシュをクリアした場合は、再度使用したいオプションを選択する必要があります。
