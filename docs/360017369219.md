# テンプレートを使用してページを作成する

**URL**: https://support.staffbase.com/hc/ja/articles/360017369219
**取得日**: 2026-05-02 18:59:06 JST

## 内容

アクティベーション
エディター
ページテンプレートを使用すると、事前に構造化され、フォーマットされたページを作成できるため、エディターはコンテンツの提供に集中することができます。プラットフォーム全体で一貫した品質と視覚的に魅力的な構造を持つテンプレートを使用して、すべてのページへの簡単なナビゲーションを実現します。
以下の標準ページテンプレートから選択できます：
スタートページ
サービスページ
ロケーションページ
これらのすぐに使用できるページテンプレートに加えて、管理者は組織の特定のニーズに応じたカスタムページテンプレートを作成できます。
スタジオで、コンテンツ > ページに移動します。
ページを追加をクリックします。
ページを追加ダイアログが開きます。
ページのタイトルを入力します。
テンプレートドロップダウンメニューから使用するテンプレートを選択します。
ページの可視性を設定します。
ページを追加をクリックします。
ページテンプレートの事前定義された構造がページエディターに読み込まれます。これで、テンプレートにコンテンツを挿入し、ユーザー向けにページを公開できます。
