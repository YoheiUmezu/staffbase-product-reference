# ニュースページ、チャンネル、および投稿の関係

**URL**: https://support.staffbase.com/hc/ja/articles/360029246651
**取得日**: 2026-05-02 18:47:58 JST

## 内容

Staffbaseプラットフォームは、オーディエンス特化型の動的コンテンツの公開を可能にします。これを実現するために、ニュースページ、チャンネル、および投稿という3つの強力な要素を使用します。
ニュースページ
ニュースページは、新聞やソーシャルネットワークのようなもので、新しいコンテンツが定期的に公開されます。最新の投稿は常に上部に表示されるため、古い投稿は徐々にニュースフィードやユーザーの視界から消えていきます。これは、コンテンツの寿命が短いか中程度であることを意味します。
コンテンツを長期間アクセス可能にしたい場合は、ページを使用してください。
ニュースページは、必要なだけ作成できます。各ニュースページでは、その中のチャンネルや投稿へのアクセスを制御できます。ユーザーはアプリのメニュー項目としてニュースページを表示します。管理者や編集責任者は、メニュー内のニュースページの位置を整理できるため、ユーザーがアプリ内で情報を見つける場所を決定できます。
ニュースページには任意の数のチャンネルを含めることができ、チャンネルはさらに任意の数の投稿を持つことができます。チャンネルとニュースページを活用することで、特定のターゲットグループやトピックに向けたコンテンツを提供できます。たとえば、販売に関するニュースや、スペインのロケーションに関するニュース専用のチャンネルをニュースページの下に持つことができます。
ニュースページを設定するには、ニュースページの作成に関する記事を参照してください。
チャンネル
Staffbaseプラットフォームにおけるソーシャルインタラクションの中心は、チャンネルと投稿にあります。チャンネルは、新聞のニュースセグメントやこの場合のニュースページのようなものです。たとえば、新聞ではスポーツセグメントはスポーツに関連するニュースのみを扱います。同様に、チャンネルには多数の投稿が含まれますが、同じタイプの投稿のみを扱うように設計されています。チャンネル内の投稿は動的で、主に短期から中期の情報を含みます。プラットフォームにおけるコンテンツの寿命を読んで、Staffbaseプラットフォームがサポートするコンテンツの種類について、寿命に基づいて詳しく学んでください。
必要なだけチャンネルを作成できます。各チャンネルをニュースページに割り当てる必要があり、そうすることで、メディア、更新、記事などの投稿がアプリ内の特定のターゲットオーディエンスに表示されます。チャンネルを複数のニュースページに割り当てることもでき、大規模な企業でのスペースや異なるグループに役立ちます。可視性設定に応じて、ユーザーはアプリのメニューで親ニュースページのみを表示するか、ニュースページに含まれる各チャンネルをサブアイテムとして表示します。
アプリへのニュース配信に必要な3つの要素の中で、チャンネルはコンテンツの使用ケースに最適なプラットフォームを作成するのに役立つ要素です。チャンネルは、特定のチャンネル内の投稿を誰が見ることができるかについて、非常に細かい可視性設定を可能にします。また、チャンネル設定で権限のある編集者を構成することもできます。これらの設定により、非常に異なる使用ケースを構成することが可能になります。たとえば：
「CEOからのニュース」
このチャンネルは全員に表示されます。しかし、ここにニュースを公開できるのは少数の編集者のみです。
「チームニュース」
このチャンネルは特定のチームにのみ表示されます。しかし、チーム全体がここにニュースを公開できます。
チャンネルはまた、チャンネル内の投稿が表示される形式を定義します。異なる使用ケースをサポートするために、3つの異なるチャンネルタイプがあります：
記事：テキスト、画像、その他のメディア。
メディア：短いキャプション付きの画像または動画。
更新：メディアのオプションを含む短いテキスト。
チャンネル設定を作成およびカスタマイズするには、ニュースクイックスタートガイドを参照してください。
投稿
投稿は、新聞のニュース項目のようなものです。さらに詳しく説明すると、新聞ではニュース項目はそれぞれのニュースセグメントに基づいてグループ化されます。たとえば、新聞の国際ニュースセグメントは、世界中のニュース項目を扱います。同様に、この場合、投稿はそれぞれのチャンネルに依存します。チャンネルには、従業員アプリに放送される動的コンテンツが含まれています。
投稿はアプリ内のニュースフィードとして表示され、最新の投稿のリストです。アプリのスタートページには一般的なニュースフィードがあります。また、各ニュースページやチャンネルを選択して、そのニュースページまたはチャンネルのニュースフィードのみを表示することができます。
投稿はチャンネル内で作成されます。投稿は、それが属するチャンネルのタイプの形式を取ります。たとえば、記事チャンネルを作成すると、このチャンネルタイプ内で作成された投稿はすべて記事になります。
新しい投稿が公開されると、古い投稿はニュースフィードの下部に移動し、徐々にニュースフィードから消えていきます。古い投稿はニュースフィードから消えますが、検索を通じて見つけることができます。
投稿を作成するには、投稿の作成に関する記事を参照してください。
