# ページ用の多言語コンテンツの作成

**URL**: https://support.staffbase.com/hc/ja/articles/115002543151
**取得日**: 2026-05-02 19:00:34 JST

## 内容

寄稿者
従業員アプリまたはイントラネットで多言語のコンテンツを作成できます。これにより、読者は選択したコンテンツ言語でコンテンツを読むことができます。ページに多言語コンテンツを追加する際には、組織で有効になっている言語を選択できます。追加の言語が必要な場合は、管理者がそれらを有効にできます。
さらに、他の言語から既存のコンテンツをコピーし、機械翻訳を使用して追加した言語に自動的に翻訳することができます。機械翻訳されたコンテンツは、翻訳に誤りがないことを確認するために、必ず校正することをお勧めします。
多言語でのコンテンツ作成は、組織の設定が行われている場合にのみ可能です。このオプションが必要な場合は、管理者にお問い合わせください。
ページで多言語コンテンツを作成する方法は3つあります。
ゼロから始める: このオプションでは、手動で翻訳を追加し、選択した言語の新しいタブをページに作成します。テキストフィールドは空のままで、新しい言語で直接コンテンツを書くか、コピー＆ペーストできます。
コンテンツをコピー: このオプションでは、ページに選択した言語の新しいタブを作成し、元の言語のコンテンツをテキストフィールドにコピーします。
自動翻訳でコピー: このオプションでは、ページに選択した言語の新しいタブを作成し、コンテンツを新しい言語に自動的に翻訳します。そこから、必要に応じてコンテンツを追加または編集できます。
手動で多言語コンテンツを追加する
このオプションでは、新しい言語で手動でコンテンツを追加できる新しいタブが作成されます。
スタジオで、コンテンツ > ページに移動します。
言語を追加したいページを開きます。
+ ボタンをクリックします。
言語を追加するダイアログが開きます。
ドロップダウンメニューから1つ以上の言語を選択します。
ゼロから始めるを選択します。
追加をクリックします。
新しい言語のタブが表示されます。
ページに1つ以上の新しい言語を追加しました。各言語はページの上部に異なるタブを持っています。これで、それらの言語で新しいコンテンツを書くことができます。
翻訳なしで多言語コンテンツを追加する
アクティベーション
このオプションでは、コピーしている言語の既存のコンテンツバージョンを含む新しいタブが作成されます。
スタジオで、コンテンツ > ページに移動します。
言語を追加したいページを開きます。
+ ボタンをクリックします。
言語を追加するダイアログが開きます。
ドロップダウンメニューから1つ以上の言語を選択します。
コンテンツをコピーを選択します。
コピー元のコンテンツドロップダウンメニューが表示されます。
コンテンツをコピーする言語を選択します。
追加をクリックします。
保存をクリックします。
ページに1つ以上の新しい言語を追加しました。各言語はページの上部に異なるタブを持ち、選択したソースからコピーされたコンテンツがあります。
自動翻訳で多言語コンテンツを追加する
アクティベーション
このオプションでは、新しい言語に自動的に翻訳された既存のコンテンツを含む新しいタブが作成されます。
スタジオで、コンテンツ > ページに移動します。
言語を追加したいページを開きます。
+ ボタンをクリックします。
言語を追加するダイアログが開きます。
ドロップダウンメニューから1つ以上の言語を選択します。
自動翻訳でコピーを選択します。
自動翻訳元のコピードロップダウンメニューが表示されます。
コンテンツをコピーする言語を選択します。
追加をクリックします。
保存をクリックします。
ページに1つ以上の新しい言語を追加しました。各言語はページの上部に異なるタブを持ち、選択したソースから新しい言語に翻訳されたコンテンツがあります。
多言語コンテンツを削除する
不要になった場合や誤って追加した場合は、言語コンテンツのバージョンを個別に削除できます。
スタジオで、コンテンツ > ページに移動します。
言語を削除したいページを開きます。
ゴミ箱アイコンをクリックします。
言語を削除するダイアログが開きます。
ドロップダウンメニューから1つ以上の言語を選択します。
クリックして削除します。
言語を削除するダイアログが開きます。
削除をクリックします。
保存をクリックします。
選択した言語バージョンのページは永久に削除されました。いつでも言語バージョンを再追加できます。
