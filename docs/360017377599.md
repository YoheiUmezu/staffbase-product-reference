# FormsプラグインとPower Automateワークフローを統合するためのトリガーURLの取得

**URL**: https://support.staffbase.com/hc/ja/articles/360017377599
**取得日**: 2026-05-02 18:50:11 JST

## 内容

FormsプラグインとPower Automateワークフローを統合することで、フォームの送信によってトリガーされる自動フローを作成できます。このようなワークフローは、HRシステム内の従業員データの自動更新や、外部メンテナンス会社への故障修理の自動通知などが含まれます。Formsプラグインを使用してPower Automateと統合するには、Power Automateで作成したワークフローからトリガーURLを取得する必要があります。その後、スタジオ管理者がFormsプラグインにURLを追加できます。
前提条件
Power Automateのライセンスを持っていること。
Power Automateでワークフローを作成し、保存していること。
ワークフローのいずれかのステップでFormsプラグインを使用していること。
Power Automateで、FormsプラグインからのHTTPリクエストを使用しているワークフローに移動します。
編集をクリックしてワークフローを開きます。
HTTPリクエストを設定したステップをクリックします。
ステップの詳細が表示されます。
URLをコピーアイコンをクリックして、フォームのトリガーURLをコピーします。
ワークフローをFormsプラグインと統合するためのトリガーURLをコピーしました。これをスタジオ管理者に提供して、Formsプラグインとリンクを統合できます。詳細については、Formsプラグインを使用したPower Automateワークフローの設定を参照してください。
