# プラットフォーム内のコンテンツのリンク

**URL**: https://support.staffbase.com/hc/ja/articles/207493455
**取得日**: 2026-05-02 18:50:38 JST

## 内容

Staffbaseプラットフォーム内では、1つのコンテンツから別のコンテンツにリンクを作成できます。これには、他の内部ページ、ニュース、プラグイン、または同じページ内の見出しへのリンクが含まれます。
内部コンテンツをURLとしてリンクする
内部コンテンツをリンクする最も基本的な方法は、リンクしたいコンテンツのURLをコピーし、それをコンテンツのテキストに直接貼り付けることです。
アプリまたはスタジオで、内部リンクを挿入したいコンテンツに移動します。
編集をクリックします。
コンテンツを編集するためのダイアログが開きます。
クリップボードからURLを直接テキストの本文に貼り付けます。
変更を保存します。
内部コンテンツがリンクされました。
内部コンテンツをハイパーリンクとしてリンクする
ニュース投稿やページでは、ハイパーリンクを使用して内部リンクを作成できます。これにより、特定のテキスト内にリンクが埋め込まれます。この場合、ユーザーはリンクされたテキストをクリックしてリンクを開きます。
アプリまたはスタジオで、内部リンクを挿入したいコンテンツに移動します。
編集をクリックします。
コンテンツを編集するためのダイアログが開きます。
ハイパーリンクにしたいテキストを選択します。
エディタツールバーのリンクアイコンをクリックします。
ハイパーリンクを追加するためのダイアログが開きます。
クリップボードからURLをターゲットURLフィールドに貼り付けます。
追加をクリックします。
変更を保存します。
内部コンテンツがリンクされました。
ページの特定のポイントにリンクする
ページでは、ユーザーをページの特定のポイントに移動させるリンクを設定できます。このタイプのリンクはアンカーリンクと呼ばれます。ページ上でアンカーにしたいポイントを選択し、そのポイントを同じページまたは別のページの他の場所にハイパーリンクします。ユーザーがアンカーリンクをクリックすると、アンカーされたポイントにジャンプします。これにより、長いコンテンツのページで関連するトピック間を簡単に移動できます。
アンカーリンクの設定
アプリまたはスタジオで、アンカーを作成したいページに移動します。
アンカーとして設定し、リンクしたいテキストを選択します。
エディタツールバーのアンカーアイコンをクリックします。
アンカーを設定するためのダイアログが開きます。
アンカーIDを提供します。このIDはアンカーリンクの末尾に表示されます。
ページに複数のアンカーリンクを設定する予定がある場合、各リンクのアンカーIDは一意でなければなりません。アンカーIDは小文字の文字と数字で構成される必要があります。大文字は機能しません。ハイフン、ダッシュ、アンダースコアもアンカーIDに使用できますが、IDはこれらの文字で始まったり終わったりしてはいけません。
オプションで、アンカーテキストを調整します。
アンカーリンクをコピーするためにコピーアイコンをクリックします。
OKをクリックします。
アンカーへのリンク
スタジオの同じページまたは別のページで、アンカーリンクを配置したい場所を選択します。
エディタツールバーのリンクアイコンをクリックします。
ハイパーリンクを追加するためのダイアログが開きます。
クリップボードからアンカーリンクをターゲットURLフィールドに貼り付けます。
追加をクリックします。
コンテンツを保存します。
