# 分析ダッシュボードを使用したコンテンツメトリクスの分析

**URL**: https://support.staffbase.com/hc/ja/articles/11985191555218
**取得日**: 2026-05-02 18:58:06 JST

## 内容

編集者
アナリティクス ダッシュボードは、ユーザーがStaffbaseプラットフォームをどのように、いつ、どこで利用しているかについて、より深いインサイトを提供します。アナリティクス ダッシュボードは異なるカテゴリーに分かれており、メトリクスを詳しく分析できます。
この記事では、コンテンツタブの詳細、グラフや分析データについて説明します。アナリティクス ダッシュボード全体やユーザー役割のアクセスについては、アナリティクス ダッシュボードの概要をご覧ください。
フィルター
コンテンツタブは、過去7日間のデータを自動的に表示します。日付範囲を変更したり、さらにフィルターを使用して結果を絞り込むことができます。
コンテンツの種類フィルターは、コンテンツタブ特有のものです。このフィルターを使うと、リストに表示したい特定のコンテンツの種類を選択できます。これは動的で、あなたの組織がコンテンツとして追加したものに応じて変化します。たとえば、アンケートプラグインがインストールされていても、フォームプラグインがインストールされていない場合、フォームプラグインは表示されません。
フィルターの詳細はこちら。
メトリクス
コンテンツタブには、コンテンツに関する詳細なメトリクスを含むテーブルが1つあります。
コンテンツリスト
コンテンツリストは、プラットフォーム内でインタラクションがあったさまざまな種類のコンテンツの広範な概要です。デフォルトでは、タブを最初に開いたとき、日付範囲フィルターは7日間に設定されています。このリストには、その期間内に作成、編集、公開、または削除されたコンテンツが表示されます。別の日付範囲を選択すると、さらに多くのコンテンツが表示されます。
コンテンツには以下が含まれます：
フォルダ
サブコンテンツを持たない個別のプラグイン（例：ネットワークやチャット）
他のプラグイン内の個別コンテンツ（例：単一ページやアンケート）
異なるテーブルヘッダーは、コンテンツの詳細を示します。テーブルヘッダーは以下の通りです：
ナビゲーション ヘッダー
説明
コンテンツ名
コンテンツの名前またはタイトル
スペース
コンテンツが配置されているスペース
訪問
コンテンツが合計で訪問された回数
訪問者
一意の個別訪問者数
パブリック
コンテンツがプラットフォームの公開エリアで表示されているかどうか
アクション
ユーザー表示でコンテンツを開く
コンテンツ名をクリックすると、そのコンテンツを開くことができます。コンテンツリストは自動的に訪問でソートされています。さらにリストをカスタマイズし、各カテゴリーでソートできます。
コンテンツリストをソートする
コンテンツリストは、さまざまなテーブルヘッダーでソートできます。
Studioで、アナリティクス > コンテンツに移動します。
コンテンツリストテーブルに移動します。
リストをソートしたいヘッダーをクリックします。
別のヘッダーでコンテンツリストがソートされました。コンテンツタブから離れると、結果は自動的に訪問での表示にリセットされます。
CSVエクスポート
コンテンツタブからユーザーデータをCSVダウンロードでダウンロードできます。エクスポート可能なデータの詳細はこちら。
CSVデータのダウンロード
Studioで、アナリティクス > コンテンツに移動します。
必要に応じて、特定のデータに絞り込むためにフィルターを適用します。
CSVエクスポートをクリックします。
コンテンツ テーブルCSVをクリックします。
CSVのダウンロードが自動的に始まります。
コンテンツタブからデータをダウンロードしました。メトリクスはテーブルとしてダウンロードされます。
