# ニュースの受信確認を使用する

**URL**: https://support.staffbase.com/hc/ja/articles/33568358597010
**取得日**: 2026-05-02 19:02:30 JST

## 内容

エディター
ニュースは、スタッフベースアプリとイントラネットで視聴者ごとに動的なコンテンツを公開することができます。ニュース投稿記事の承認は、従業員に特定のニュース投稿記事を読んだことを承認するよう求めるもので、必須またはセキュリティ関連の更新に最適です。
承認はすべてのニュースチャンネルタイプで利用できます。特定の投稿のオプションセクションで有効にすると、承認が必要バッジが明確に表示され、読者はすぐにアクションが必要であることがわかります。投稿内で従業員は既読として承認ボタンをクリックし、その操作が記録されます。
投稿に承認を追加する
Studioでコンテンツ > ニュースに移動します。
承認を追加したいニュースチャンネルと投稿を選択します。
オプションで、承認トグルを有効にします。
下書き保存または保存をクリックします。
これで投稿に承認が追加されました。
承認のCSVをエクスポートする
Studioでコンテンツ > ニュースに移動します。
承認をダウンロードしたいニュースチャンネルと投稿を選択します。
投稿の横にある三点リーダーをクリックしてアクションメニューを開きます。
統計をクリックします。
投稿統計が開きます。
承認の下で、CSVファイルをダウンロードするをクリックします。
CSVファイルが自動的にダウンロードされます。
これで承認をCSVファイルとしてエクスポートできました。
