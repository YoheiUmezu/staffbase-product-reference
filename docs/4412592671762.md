# 従業員メールへのサインイン

**URL**: https://support.staffbase.com/hc/ja/articles/4412592671762
**取得日**: 2026-05-02 18:44:22 JST

## 内容

2022年現在、BananatagはStaffbaseの従業員メール製品です。
あなたの組織が従業員メールをどのように設定し統合しているかによって、Webアプリへのサインインオプションが異なる場合があります。わからない場合は、まずIT部門に連絡して支援を受けてください。
手順
あなたの所在地に応じて、以下のいずれかのURLにアクセスしてください：
北米のお客様 - https://app.bananatag.com
ヨーロッパのお客様 - https://app.de.bananatag.com
オプションとして、Staffbaseのアプリおよび/またはイントラネットが従業員メールと統合されている場合は、こちらをクリックして代替のナビゲーション手順を確認してください。
組織で使用されている認証情報でサインインします。
