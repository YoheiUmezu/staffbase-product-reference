# メールにアンケートを追加する

**URL**: https://support.staffbase.com/hc/ja/articles/27337745778450
**取得日**: 2026-05-02 18:48:38 JST

## 内容

編集者
アンケートを使うと、従業員から貴重なインサイトやフィードバックを収集できます。アンケートを利用して従業員にリーチし、フィードバックを集め、プロセスを改善し、その後の変化を確認できます。アンケート結果は質問ごとに分かれており、リアルタイムで確認できます。
アンケートを作成した後、アンケートブロックをメール本文に直接追加できます。ブロックはメール内のどこにでもドラッグ＆ドロップできます。詳しくはStaffbaseメールのクイックブロックについてをご覧ください。メール受信者はメール内でアンケートを確認し、ランディングページで1回だけフィードバックを送信します。
アンケートクイックブロックを追加したら、右側のアンケートサイドパネルからメールに含めたいアンケートを選択できます。このパネルにはアクセス権のあるアンケートのみが表示されます。アンケートにカーソルを合わせると、スケジュールや作成者などの詳細が表示されます。
すべての受信者には個別のアンケートリンクが送信されます。テストメールではアンケートリンクは有効にならないため、テストクリックは記録されません。
前提条件
アンケートを作成していること。
アンケートを公開または公開予定にしていること。
メールにアンケートを追加する
Studioでメールに移動します。
アンケートを追加したいメールに移動します。
クイックブロックの下で、アンケートクイックブロックに移動します。
アンケートクイックブロックをメール本文にドラッグ＆ドロップします。
右側のアンケートサイドパネルから特定のアンケートをドラッグ＆ドロップします。
アンケートは自動的にアンケートクイックブロックに反映されます。
必要に応じて、アンケートブロックの色やスタイルを調整します。
これでメールにアンケートが含まれました。ユーザーは送信されたメール本文でアンケートを確認できます。送信前であれば、メールからアンケートを削除できます。メール送信後は編集できません。
