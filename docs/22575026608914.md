# チャット会話のアーカイブ

**URL**: https://support.staffbase.com/hc/ja/articles/22575026608914
**取得日**: 2026-05-02 18:55:24 JST

## 内容

チャットは、アプリ内での直接的な1対1またはグループメッセージングに最適なツールです。 詳細はこちら。チャットでの会話をアーカイブすると、それらはアクティブな会話リストから移動され、削除されることはありません。これにより、受信トレイが整理され、古い会話も将来の参照のためにアクセス可能な状態が保たれます。これにより、進行中の会話に集中し、整理された状態を維持できます。
アーカイブされた会話はアーカイブフォルダーにあります。他のチャット参加者の会話の場所には影響しません。また、参加者が新しいメッセージを送信すると、会話は自動的に受信トレイに戻ります。会話をアーカイブしても通知には影響しません。
会話のアーカイブ
モバイルアプリ
チャットで、アーカイブしたいチャット会話に移動します。
会話を左にスライドさせてアクションアイテムを表示します。
タップしてアーカイブします。
会話はアーカイブに移動されます。参加者が新しいメッセージを送信すると、会話は自動的に受信トレイに移動します。いつでも会話を復元できます。
ウェブアプリ
チャットで、アーカイブしたいチャット会話に移動します。
会話の上にカーソルを合わせ、3つのドットをクリックしてアクションメニューを開きます。
クリックしてアーカイブします。
会話はアーカイブに移動されました。参加者が新しいメッセージを送信すると、会話は自動的に受信トレイに移動します。いつでも会話を復元できます。
アーカイブされた会話の復元
以前にアーカイブされた会話を復元し、メインのチャット受信トレイに戻すことが可能です。
モバイルアプリ
チャットで、受信トレイをタップします。
タップしてアーカイブします。
アーカイブが表示されます。
復元したい会話を左にスライドさせます。
タップして復元します。
会話は受信トレイに戻されました。いつでも会話を再度アーカイブできます。
ウェブアプリ
チャットで、受信トレイの下部にあるアーカイブをクリックします。
アーカイブが表示されます。
復元したい会話の上にカーソルを合わせます。
3つのドットをクリックしてアクションメニューを開きます。
復元を選択します。
会話は受信トレイに戻されました。いつでも会話を再度アーカイブできます。
