# カスタムメールフォントの有効化

**URL**: https://support.staffbase.com/hc/ja/articles/4412592744722
**取得日**: 2026-05-02 18:49:54 JST

## 内容

従業員メールは、カスタムフォントを使用して送信することをサポートしています。ただし、ニュースレターの受信者がそのフォントを受信できるようにするために、いくつかの手順を踏む必要があります。
要件
受信者はカスタムフォントをダウンロードしている必要があります。
ニュースレターの受信者がカスタムフォントを表示できるようにするためには、受信者のコンピュータにダウンロードされ、必要に応じてデスクトップのOutlookアプリケーション設定に追加されている必要があります。
この件については、システム管理者またはITチームにお問い合わせください。
GmailおよびOWAユーザー向け：
ウェブセーフフォントを使用している場合、受信者はデバイスにダウンロードする必要がないかもしれません。
受信者はカスタムフォントを使用するためのライセンスを持っている必要があります。
上記を満たすために必要な場合、受信者はそのフォントを使用するためのライセンスを所有している必要があります。
フォントファイルを提供する必要があります。
フォントファイルは、アカウントマネージャーに送信する必要があります。そうすることで、私たちのチームのメンバーがアカウントにフォントを追加できます。
ファイル要件：
フォントファイルは.ttf形式である必要があります。
ファイルに保存されているフォントはレギュラーウェイトである必要があります。
この件については、システム管理者またはITチームにお問い合わせください。
管理者または親管理者アカウントを持っている必要があります。
この権限レベルは、新しいメールデザインのデフォルトとしてカスタムフォントを設定するために必要です。
始めるには
アカウントにカスタムフォントを追加することに興味がある場合は、アカウントマネージャーにお問い合わせください。
