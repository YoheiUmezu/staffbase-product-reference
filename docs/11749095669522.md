# チャットでの音声メッセージの送信

**URL**: https://support.staffbase.com/hc/ja/articles/11749095669522
**取得日**: 2026-05-02 18:49:26 JST

## 内容

音声メッセージを送信することで、チャットの会話に迅速に応答し、参加できます。モバイルアプリから簡単に音声メッセージを録音して送信できます。音声メッセージは最大2分間の長さに対応しています。音声メッセージは、モバイルアプリとウェブアプリの両方で再生できます。
音声とテキストを1つのメッセージに組み合わせることはできません。音声メッセージを録音するためにマイクアイコンを有効にするには、メッセージフィールド内のテキストを削除する必要があります。
音声メッセージの送信
モバイルアプリ
チャットで、音声メッセージを送信したい会話を開きます。
マイクアイコンをタップして長押しします。
最大2分間の音声メッセージを録音します。
オプションで、ゴミ箱アイコンをクリックしてメッセージを破棄し、再録音します。
紙飛行機アイコンをクリックしてメッセージを送信します。
音声メッセージを録音して送信しました。他のユーザーは、モバイルアプリまたはウェブアプリであなたの音声メッセージを聞くことができます。
