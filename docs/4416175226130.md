# ニュースチャンネルのパンくずリストナビゲーションの管理

**URL**: https://support.staffbase.com/hc/ja/articles/4416175226130
**取得日**: 2026-05-02 19:03:18 JST

## 内容

アクティベーション
パンくずリストナビゲーションは、ユーザーが内部コミュニケーションプラットフォーム内での位置を把握するための直感的で実績のある方法です。デスクトップユーザーがナビゲーションを把握できるように、各ニュースチャンネルのパンくずリストナビゲーションを管理します。各ニュースチャンネルに対して、1つ以上のニュースページを関連付けることができます。
スタジオで、コンテンツ > ニュースに移動します。
パンくずリストナビゲーションを管理したいニュースチャンネルを選択し、設定を開くためにギアアイコンをクリックします。
関連ニュースページのセクションで、ニュースページを選択します。ニュースページがそのニュースチャンネルに既に接続されている場合、ここに表示されます。
パンくずリストのニュースページの下で、ニュースチャンネルのパンくずリストナビゲーションに表示されるニュースページを選択します。
オプションで、パンくずリストプレビューをクリックして、ユーザーにとってのパンくずリストナビゲーションの見え方を確認します。パンくずリストナビゲーションが期待通りに表示されるか確認します。
保存をクリックします。
ニュースチャンネルのパンくずリストナビゲーションが設定されました。ユーザーはこのニュースチャンネルでパンくずリストを使用できるようになります。
