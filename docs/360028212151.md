# 投稿の復元

**URL**: https://support.staffbase.com/hc/ja/articles/360028212151
**取得日**: 2026-05-02 19:03:11 JST

## 内容

削除されたコンテンツは30日以内に復元できます。
スタジオで、コンテンツ > ゴミ箱に移動します。
復元したい投稿のチェックボックスを選択します。複数のアイテムを選択することもできます。
復元をクリックします。
投稿が復元されました。復元された投稿は、それぞれのチャンネルに再表示され、以前の公開モードで表示されます。たとえば、公開された投稿が削除された場合、復元されると再び公開された状態で表示されます。
