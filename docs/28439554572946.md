# ソーシャルウォールで従業員同士をつなぐ

**URL**: https://support.staffbase.com/hc/ja/articles/28439554572946
**取得日**: 2026-05-02 18:41:22 JST

## 内容

コミュニケーション
ピア・ツー・ピア
労力：低
ソーシャルウォールは、従業員がニュース、意見、個人的な近況を共有しながら同僚とつながるための専用スペースを提供します。これはコミュニティ意識を高め、企業文化を強化するための強力なツールであり、特にリモートチームや現場従業員とのつながりに重要です。ソーシャルウォールは柔軟かつカスタマイズ可能で、組織に最適なテーマに基づいて設定できます。以下のユースケースは、従業員のためのハブとしてソーシャルウォールを構築するのに役立ちます。

主なポイント
課題：リモートや現場の従業員は、同僚とつながるための非公式なチャンネルが不足し、孤立感や企業文化からの疎外感を感じることがあります。
ビジネス目標：ピア・ツー・ピアの交流を促進するための中央デジタルスペースを導入し、従業員のエンゲージメントを高め、コミュニティ意識を醸成し、士気を向上させます。
Staffbase 機能：
ニュース：ソーシャルウォールは専用のニュースチャンネルで、すべてのユーザーまたは特定のグループに公開できます。権限を管理することで、非公式のユーザー生成コンテンツと公式ニュースを分けることができます。
投稿ウィジェット：ユーザーが特定のページから直接ソーシャルウォールのニュース投稿を作成できるようにします。
ページ：ページ上でソーシャルウォールを紹介することで利用を促進し、従業員が簡単に見つけられるようにします。
視聴者：すべての従業員、特に地理的に分散したチーム、リモートワーカー、現場従業員。これらの従業員は従来の社内コミュニケーションチャンネルへの継続的なアクセスが難しい場合があります。
成果：ピア・ツー・ピアの交流が増え、企業文化が強化されます。従業員は同僚や組織とのつながりをより感じ、士気や帰属意識が向上します。
動画チュートリアル
手順
このステップバイステップガイドを使って、Staffbase でこのユースケースを設定できます。
インタラクティブなニュースチャンネルを作成する
編集責任者
Studio でコンテンツ > ニュースに移動します。
チャンネルを追加するをクリックします。
ソーシャルウォールの名前を入力します。
チャンネルタイプでアップデートを選択します。
保存するをクリックします。
投稿者ドロップダウンで、インタラクティブニュースチャンネルに投稿できるユーザーやユーザーグループを選択します。
保存するをクリックします。
設定に移動し、プッシュ通知やメール通知のオプションを調整します。
チャンネルを公開するをクリックします。
チャンネルの作成やニュースチャンネルの種類について詳しく学べます。
投稿でチャンネルを紹介する
編集者
ソーシャルウォールで投稿を作成するをクリックします。
チャンネルをユーザーに紹介するニュース投稿を書きます。
下書きを保存するをクリックします。
必要に応じて、プレビューをクリックしてユーザー向けの表示を確認します。
公開するをクリックします。
投稿の作成や投稿の公開について詳しく学べます。
ニュースフィードと投稿ウィジェットをページに追加する
編集者
ニュースフィードウィジェットと投稿ウィジェットをページに追加し、従業員がソーシャルウォールを利用しやすくします。
両方のウィジェットのソースとして新しいソーシャルウォールチャンネルを選択します。
保存するをクリックします。
公開するをクリックします。
アイスブレイクとソーシャルウォールの促進
フレンドリーな最初のメッセージを投稿して雰囲気を作りましょう。
ストーリーや写真、質問を共有し、他の人が参加しやすくします。
リーダーに最初の投稿を依頼し、メディアを含めることを提案します。
リーダーの可視的な参加は信頼を築き、ウォールが安全なスペースであることを示します。さまざまな投稿タイプ（テキスト、写真、動画）があると、より興味深くエンゲージメントも高まります。
社内ニュースレター、会議、アプリのホームページでソーシャルウォールを紹介します。
さらに、ニュースフィードウィジェットをページに直接追加してチャンネルを目立たせましょう。
投稿やコメント欄を定期的に確認してアクティビティをモデレートします。
モデレーションはポジティブな環境を維持し、積極的で意味のある参加を促します。
成功の測定
このユースケースの成功を追跡するさまざまな方法があります。
リーチとサイレントエンゲージメント
閲覧数が多い場合、ウォールが情報ハブとして機能し、認知度が高まっていることを示します。
アナリティクスダッシュボードのニュースタブを使って、新しいコンテンツの投稿頻度や、従業員が投稿を閲覧した回数（反応がなくても）を確認できます。
投稿ごとのエンゲージメント
早い段階でエンゲージメント目標を設定し、ベンチマークとします。高く増加傾向のエンゲージメントは、つながりのあるアクティブなコミュニティを示します。
投稿統計を使って、投稿ごとの平均いいね数やコメント数を測定し、反応の度合いを把握します。
従業員のフィードバックと感情
ポジティブな感情やエピソードは、ウォールの文化的・感情的な影響を示します。
ウォールのコメント、アンケート、会話から定量的・定性的なフィードバックを集めましょう。
ベストプラクティス
Staffbase では、ユーザーに最適な体験を提供するためのいくつかの推奨事項があります。
ソーシャルウォールのテーマを決める
誰でも役割や場所に関係なく参加しやすい、シンプルで魅力的なテーマを選びましょう。
幅広く包括的なテーマは創造性を引き出し、参加を最大化します。例：「1枚の写真で今週を表現」や「私のワークスペース紹介」など。
ソーシャルウォールをモデレートする
コミュニティガイドラインに沿った、敬意と包括性、目的意識のある交流を促すエチケットを策定し、共有しましょう。
不適切なコンテンツへの迅速な対応や、積極的で意味のある参加を促すために、モデレーションに時間とリソースを割きましょう。
イベント専用ソーシャルウォールを作成する
イベント前にウォールを公開し、期待感を高めたり、事前の自己紹介やコンテンツ共有を促進します。
イベント中は従業員にウォールの利用を促し、ライブの瞬間やリアクション、舞台裏の更新を記録します。
カスタマーストーリー
他のStaffbaseのお客様がこのユースケースをどのように活用しているかをご紹介します。
Yageoは、全地域の従業員にソーシャルウォール機能を活用して部門のニュースやイベント情報を投稿するよう促しました。この戦略により従業員のエンゲージメントが高まり、ソーシャルウォールは週6,000回以上の閲覧と、全世界の拠点の80%以上からの積極的な参加を実現しました。その結果、世界中の従業員が会社の活動によりつながりを感じ、結束感や帰属意識が高まりました。
Brussels Airlinesは、従業員が仕事についての考えを共有し、組織がアクションを取れるようなスペースとしてアプリ内にソーシャルウォールを導入しました。それ以前は、従業員がFacebookなどのソーシャルネットワーク上にページを作成していました。この変更によりコミュニティ意識が醸成され、機密情報の外部流出も防止できました。ソーシャルウォールの利用は日常的なコミュニケーションにとどまらず、従業員がアイデアを交換したり、質問したり、専門的な知見を共有することも可能になりました。
