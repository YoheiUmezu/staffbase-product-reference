# Microsoft 365 アプリケーションの権限の概要

**URL**: https://support.staffbase.com/hc/ja/articles/4410113144466
**取得日**: 2026-05-02 18:57:12 JST

## 内容

Microsoft Connections、ウィジェット、およびプラグインは、Azure グローバル管理者が Azure ポータルで管理できるさまざまな Microsoft API 権限を使用します。これには、Microsoft 365 環境から内部コミュニケーションプラットフォームがアクセスするコンテンツの管理が含まれます。Microsoft Connections、ウィジェット、およびプラグインで使用されるデフォルトの権限はここにリストされています。

Microsoft Connection の設定に使用したアプリの下で、次の権限を使用してください。
デフォルトの Azure アプリを使用した場合は、Staffbase Integrations. の下で権限を見つけることができます。

権限の種類

サインインしてユーザープロファイルを読み取る
User.Read
委任
いいえ
ユーザーがアプリにサインインし、アプリがサインインしたユーザーのプロファイルを読み取ることを許可します。また、アプリがサインインしたユーザーの基本的な会社情報を読み取ることも許可します。
すべてのサイトコレクションのアイテムを読み取る
Sites.Read.All
委任
いいえ
サインインしたユーザーに代わって、アプリがすべてのサイトコレクション内のドキュメントおよびリストアイテムを読み取ることを許可します。
すべてのユーザーのファイルを読み取る
Files.Read
委任
いいえ
アプリがサインインしたユーザーのファイルを読み取ることを許可します。
ユーザーがアクセスできるすべてのファイルを読み取る
Files.Read.All
委任
いいえ
アプリがサインインしたユーザーがアクセスできるすべてのファイルを読み取ることを許可します。
すべてのユーザーの完全なプロファイルを読み取る
User.Read.All
委任
はい
サインインしたユーザーに代わって、アプリが組織内の他のユーザーのプロファイルプロパティ、レポート、およびマネージャーの完全なセットを読み取ることを許可します。
すべてのグループを読み取る
Group.Read.All
委任
はい
サインインしたユーザーに代わって、アプリがグループをリストし、そのプロパティおよびすべてのグループメンバーシップを読み取ることを許可します。また、アプリがサインインしたユーザーがアクセスできるすべてのグループのカレンダー、会話、ファイル、およびその他のグループコンテンツを読み取ることも許可します。
ユーザーのカレンダーを読み取る
Calendars.Read
委任
いいえ
アプリがユーザーのカレンダー内のイベントを読み取ることを許可します。
ユーザーおよび共有カレンダーを読み取る
Calendars.Read.Shared
委任
いいえ
アプリがユーザーがアクセスできるすべてのカレンダー内のイベントを読み取ることを許可します。これには、委任カレンダーおよび共有カレンダーが含まれます。
ユーザーのチャットメッセージを読み取る
Chat.Read
委任
いいえ
アプリがサインインしたユーザーに代わって、1対1またはグループチャットスレッドを読み取ることを許可します。
ユーザーデータにいつでもアクセス
offline_access
委任
いいえ
アプリがユーザーデータを読み取りおよび更新することを許可します。ユーザーがアプリを現在使用していない場合でも可能です。
