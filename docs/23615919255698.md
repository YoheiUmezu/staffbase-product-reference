# ページにおけるオフラインモードのベストプラクティス

**URL**: https://support.staffbase.com/hc/ja/articles/23615919255698
**取得日**: 2026-05-02 18:48:58 JST

## 内容

ベータ
アクティベーション
オフラインモードを使用すると、従業員は従業員アプリでページをダウンロードでき、インターネット接続なしで関連情報にアクセスできます。ダウンロードされたコンテンツ（テキスト、画像、PDFを含む）は、オフライン時に整理された概要ページを介してアクセス可能です。オフラインモードは、遠隔地、倉庫、またはオフショアの場所にいる従業員にとって実用的です。ユーザーは、自分の役割に関連するコンテンツのみをダウンロードできるため、迅速にアクセスできます。 詳細はこちら。
ベストプラクティス
Staffbaseは、ユーザーにとって最高のオフライン体験を確保するためのいくつかの推奨事項を提供しています。
ページコンテンツの最適化
組織がダウンロードの可能性が高いと考えるページを更新し、磨きをかける時間を取ります。
休暇ポリシーやHRの福利厚生などの高トラフィックページを編集し、視覚的に魅力的で事実に基づいたものにします。
ページにはテキスト、画像、PDFのみを含めます。これらの形式はダウンロード時に最適に機能し、オフラインでのユーザー体験を向上させます。
インターネット接続なしでは再生できないため、動画やGIFは避けます。
ユーザーの有効化
新しいオフラインモードについてユーザーに積極的に知らせ、彼らがそれを活用できるようにします。
ダウンロード方法を示すウォークスルーを作成します。
オフラインで機能するものとオンラインで必要なものについて明確な期待を設定します。
更新の通知
重要または高トラフィックのページが更新された場合、ユーザーに通知します。ダウンロードされたページは、オンラインで再訪問したときのみ更新されます。
更新後に重要なページを再訪問するようユーザーに促し、更新を受け取るようにします。
追加の役立つ情報
ページのオフラインモードの概要とFAQ
オフラインで表示するためのページのダウンロード
