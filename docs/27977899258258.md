# 複数の質問を含む調査の作成

**URL**: https://support.staffbase.com/hc/ja/articles/27977899258258
**取得日**: 2026-05-02 18:45:45 JST

## 内容

アクティベーション
エディター
複数の質問を含む調査は、特定のトピックを深く掘り下げることで包括的な洞察を提供します。1つの調査内で追加、順序変更、編集できる3種類の質問タイプから選択できます。結果は質問ごとに分解され、リアルタイムで利用可能です。また、時間をかけて1つのトピックに関する洞察を収集する定期的な調査リズムと、1回限りの調査のいずれかを選択できます。
複数の質問を含む調査の追加
スタジオで、コンテンツ>調査に移動します。
調査を作成をクリックします。
新しい調査を作成するためのダイアログが開きます。
調査のタイトルを入力します。
ドロップダウンメニューから調査の可視性を選択します。
調査を追加をクリックします。
最初の質問の追加
調査を追加すると、調査エディターが自動的に開きます。既存の調査にさらに質問を追加するには、複数の質問を含む調査の編集を参照してください。
作成した調査で、開始するをクリックして質問の追加を開始します。
質問を追加セクションで、追加したい調査質問のタイプを選択します。
質問フィールドが自動的に開きます。
調査質問を入力します。
さまざまな質問タイプの設定に関する詳細なガイダンスについては、以下の質問タイプを参照してください。
オプションで、ユーザーにより多くのコンテキストを提供するために追加情報を追加します。
保存をクリックします。
オプションで、プレビューをクリックして、ユーザーに対する調査の見え方を確認します。
さらに質問を追加するには、ステップ3から繰り返します。
質問が保存されました。調査が更新されましたが、従業員には表示されません。さらに調査質問を追加するか、調査の公開をスケジュールします。
質問タイプ
Surveysプラグインを使用すると、5種類の異なる質問タイプを設定できます：
5つ星評価
スケール評価
テキスト
複数選択
NPS質問
5つ星質問の設定
調査の質問を追加セクションで、5つ星評価を選択します。
調査質問を提供します。
オプションで、ユーザーにより多くのコンテキストや評価の指示を提供するために追加情報を追加します。
保存をクリックします。
質問が保存され、調査質問の下に自動的に表示されます。
スケール評価質問の設定
調査の質問を追加セクションで、スケール評価を選択します。
ドロップダウンメニューからスケール範囲を選択します。
調査質問を提供します。
オプションで、ユーザーにより多くのコンテキストを提供するために追加情報を追加します。
スケール範囲ラベルの下に、ユーザーを方向付けるためにスケールの両端の説明を提供します。
保存をクリックします。
質問が保存され、調査質問の下に自動的に表示されます。
テキスト質問の設定
調査の質問を追加セクションで、テキストを選択します。
調査質問を提供します。
オプションで、ユーザーにより多くのコンテキストを提供するために追加情報を追加します。
保存をクリックします。
質問が保存され、調査質問の下に自動的に表示されます。
複数選択質問の設定
調査の質問を追加セクションで、複数選択を選択します。
調査質問を提供します。
オプションで、ユーザーにより多くのコンテキストを提供するために追加情報を追加します。
回答選択肢を提供します。
オプションで、ユーザーが選択できる回答の最大数を定義します。
デフォルトでは、ユーザーはすべての回答を選択できます。最大選択ドロップダウンを使用して、選択できる回答の数を制限します。
最大選択ドロップダウンは、提供した回答選択肢の総数よりも1つ少ない数を表示します。この数は、提供する回答選択肢の数に基づいて自動的に調整されます。
保存をクリックします。
質問が保存され、アンケート質問の下に自動的に表示されます。
ネットプロモータースコア (NPS) 質問の設定
アンケートの質問を追加のセクションで、スケール評価を選択します。
アンケート質問を提供します。
オプションで、ユーザーにより多くのコンテキストを提供するために追加情報を追加します。
スケール範囲ラベルの下に、ユーザーを方向付けるためにスケールの両端の説明を提供します。
保存をクリックします。
質問が保存され、アンケート質問の下に自動的に表示されます。
