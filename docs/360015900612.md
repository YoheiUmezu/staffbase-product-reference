# ユーザーにアプリ全体の拡張権限を付与する

**URL**: https://support.staffbase.com/hc/ja/articles/360015900612
**取得日**: 2026-05-02 19:00:32 JST

## 内容

アプリ全体にわたってユーザーに システム全体の役割 を割り当てることができます。ユーザーにシステム全体の権限を付与することで、アプリメニュー、分析、システム全体の管理者の場合はアプリ設定やユーザー管理への拡張アクセスを許可します。システム全体の権限を付与する前に、ユーザーがスタジオの使用方法を理解していることを確認してください。
スタジオで、 ユーザー設定 > 登録ユーザー または 保留中のユーザー に移動します。
拡張権限を付与したいユーザーに移動します。
編集 をクリックします。
アクセス 設定で、ドロップダウンメニューから システム全体の 役割を選択します。
保存 をクリックします。
