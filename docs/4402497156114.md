# ユーザープロフィール写真の推奨サイズ

**URL**: https://support.staffbase.com/hc/ja/articles/4402497156114
**取得日**: 2026-05-02 18:50:04 JST

## 内容

プロフィール写真はユーザープロフィールをより個人的なものにし、新しい従業員がより早く周囲を把握するのに役立ちます。以下のガイドラインは、適切なプロフィール写真を見つける手助けをします：
最小画像サイズ：400ピクセル x 400ピクセル
最大ファイルサイズ：25 MB
画像が推奨サイズに合わない場合、自動的に最適なサイズにリサイズされます。アスペクト比は保持され、さらに画像をトリミングすることもできます。
