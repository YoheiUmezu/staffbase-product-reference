# Staffbaseメールでフォローアップメールを送信する

**URL**: https://support.staffbase.com/hc/ja/articles/27735334852242
**取得日**: 2026-05-02 18:56:32 JST

## 内容

編集者
フォローアップメールは、あなたのメッセージが見逃されないようにする最適な方法です。Staffbaseメールでは、メールアナリティクスタブから未開封者や元の視聴者に直接メッセージを再送信できます。フォローアップメールは元のメールをコピーし、再送信前に編集するオプションもあります。これにより、重要な更新を強調したり、ミスを簡単に修正したりできます。各フォローアップは自動的に元のメールとリンクされ、編集者に完全な透明性を保ちます。各元のメールにつき1通のフォローアップメールを送信できます。
Studioでメール > フォローアップを送信したいメールに移動します。
三点リーダーをクリックして、その他のオプションメニューを開きます。
アナリティクスを選択します。
フォローアップ作成をクリックします。
必要に応じて、デザイン編集をクリックしてメールコンテンツに追加や変更を行います。
未開封者または全ての元の受信者のいずれに送信するかを選択します。
送信するをクリックします。
フォローアップメール送信ダイアログが開きます。
送信するをクリックします。
フォローアップメールの送信が完了しました。メールは「フォローアップ」という接頭辞付きで「最近送信済み」に表示されます。フォローアップメールの詳細を開くと、設定や元のメールへのリンクを確認できます。
