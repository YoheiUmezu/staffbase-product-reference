# Microsoft Stream ビデオをコンテンツから削除する

**URL**: https://support.staffbase.com/hc/ja/articles/19961420342418
**取得日**: 2026-05-02 18:49:05 JST

## 内容

ベータ
アクティベーション
エディター
古くなった、関連性のない、または誤って追加されたビデオを削除することで、コンテンツを正確かつ関連性のあるものに保ちます。
カバービデオの削除
スタジオで、コンテンツ > ニュースに移動します。
Microsoft Stream ビデオを削除したい投稿に移動します。
記事画像/ビデオの下で、ゴミ箱アイコンをクリックします。
保存をクリックします。
投稿からカバーマイクロソフト Stream ビデオを削除しました。
投稿またはページからビデオを削除する
スタジオで、ビデオを削除したい投稿またはページに移動します。
コンテンツエディターで、ファイルビューアウィジェットの上にカーソルを合わせます。
ゴミ箱アイコンをクリックします。
ウィジェットを削除ダイアログが開きます。
削除をクリックします。
保存をクリックします。
投稿またはページから Microsoft Stream ビデオを削除しました。

投稿の作成
