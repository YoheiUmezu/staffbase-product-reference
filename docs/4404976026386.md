# プラットフォームのホスティングインフラストラクチャとベースURLの特定

**URL**: https://support.staffbase.com/hc/ja/articles/4404976026386
**取得日**: 2026-05-02 18:47:27 JST

## 内容

管理者
Staffbaseは、プラットフォームとデータをホストするために、異なる国にデータセンターを提供しています。Studioでは、プラットフォームがどこにホストされているか、またAPI認証に使用する必要があるプラットフォームのベースURLを確認できます。
Studioで、システム > APIアクセスに移動します。
APIトークンページが開きます。
APIトークン見出しの横にあるラベルを確認します:
US: プラットフォームは米国のホスティングインフラストラクチャにホストされています。
DE: プラットフォームはヨーロッパのホスティングインフラストラクチャにホストされています。
AU: プラットフォームはオーストラリアのホスティングインフラストラクチャにホストされています。
APIリクエストで使用するため、または統合を構成するためにプラットフォームのベースURLをコピーします。
プラットフォームのホスティングインフラストラクチャとベースURLを正常に特定しました。これで、API統合を開始できます。
