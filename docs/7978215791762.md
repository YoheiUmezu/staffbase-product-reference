# チャットメッセージまたは会話の削除

**URL**: https://support.staffbase.com/hc/ja/articles/7978215791762
**取得日**: 2026-05-02 18:56:59 JST

## 内容

チャット内で、Staffbaseは不要なコンテンツを削除するためのさまざまな方法を提供しています。会話内の個別のメッセージや、全体のチャット会話を削除することができます。
削除メッセージ機能を使用して、会話内の冗長または不正確なメッセージを削除します。
削除会話機能を使用して、不要なチャットをクリアし、チャットスペースを整理された状態に保ちます。
Staffbaseは、削除されたメッセージや会話の記録をいかなる期間も保持しません。メッセージまたは会話が削除されると、復元することはできません。
チャットメッセージの削除
アクティベーション
前提条件
プラットフォーム管理者がチャットメッセージの削除を有効にしています。
グループまたは1対1の会話で、自分のチャットメッセージを永久に削除できます。削除されると、メッセージは自分や他の参加者には表示されなくなります。
モバイルアプリ
チャットに移動します。
編集したい会話を開き、削除したいメッセージに移動します。
チャットメッセージをタップして保持します。
アクションメニューが開きます。
メッセージを削除をタップします。
確認ダイアログが開きます。
削除をタップします。
チャットメッセージを削除しました。チャット参加者にはもはや表示されません。
ウェブアプリ
チャットに移動します。
編集したい会話を開き、削除したいメッセージに移動します。
ゴミ箱アイコンをクリックします。
確認ダイアログが開きます。
削除をクリックします。
チャットメッセージを削除しました。チャット参加者にはもはや表示されません。
1対1の会話の削除
1対1の会話は、どちらの参加者でも削除できます。ユーザーが1対1の会話を削除すると、その会話はこのユーザーにとって永久に削除されますが、他の参加者には削除されません。
チャットに移動します。
削除したい会話を開きます。
情報アイコンをクリックします。
アクションメニューが開きます。
削除をクリックします。
確認ダイアログが開きます。
削除をクリックします。
自分の1対1のチャット会話を削除しました。他の参加者は、モバイルまたはウェブアプリでメッセージと会話を引き続き見ることができます。
グループ会話の削除
管理者
グループ会話は、会話の管理者のみが削除できます。削除されると、会話はすべての参加者にとって永久に削除されます。
チャットに移動します。
削除したいグループ会話を開きます。
情報アイコンをクリックします。
グループ情報が開きます。
削除をクリックします。
確認ダイアログが開きます。
削除をクリックします。
グループ会話を削除しました。チャット参加者にはもはやアクセスできません。
