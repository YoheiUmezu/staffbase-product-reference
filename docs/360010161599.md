# ページにセクションを追加する

**URL**: https://support.staffbase.com/hc/ja/articles/360010161599
**取得日**: 2026-05-02 19:00:55 JST

## 内容

アクティベーション
エディター
セクションは、ページ上に視覚的な区切りを作るデザイン要素です。ページエディターでは、さまざまなセクションタイプから選択できます。ページにセクションを追加した後、テキスト、メディア、またはウィジェットを追加します。
スタジオで、コンテンツ > ページに移動します。
セクションを追加したいページをクリックします。
ページを編集が開きます。
エディターのツールバーで、セクションアイコンをクリックします。
ページレイアウトオプションのツールバーが開きます。
追加したいセクションタイプをクリックします。
セクションにコンテンツを追加します。
セクションに別のセクションを追加して、高度なページレイアウトを作成できます。
保存をクリックして、ページへの変更を保存します。
