# 従業員メールで送信済みメールの詳細を表示する

**URL**: https://support.staffbase.com/hc/ja/articles/4415095641362
**取得日**: 2026-05-02 18:44:04 JST

## 内容

管理者
エディター
従業員メールで追跡している各送信メールの詳細情報にアクセスできます。これには、あなたの主要なアドレスまたはエイリアスから送信されたメールが含まれます。
表示される情報は役割によって異なります：
管理者はすべての情報にアクセスできます。
ユーザーは自分のアカウントと共有されたエイリアスの情報を見ることができます。
従業員メールのウェブアプリにサインインするか、スタジオを開く。
メール > 送信済みメールに移動します。
「送信済みメール」ページが開きます。
「送信済みメール」ページには、最近送信されたメールが表示されます。ユーザーは自分の送信済みメールを見ることができます。
ユーザーのフィルタ
管理者はすべてのタブを見ることができます。ユーザーは自分のアカウントタブと共有されたエイリアスのみを見ることができます。
アカウントタブをクリックして、現在使用しているメールアカウントから送信されたメールのリストを表示します。
すべてのユーザーのメールタブをクリックして、すべてのユーザーのメールのリストを表示します。エイリアスも含まれます。ユーザーのドロップダウンから1つ以上の名前を選択できます。
エイリアスからタブをクリックして、エイリアスから送信されたメールのリストを表示します。エイリアスのドロップダウンから1つ以上の名前を選択できます。
詳細のフィルタ
リストに表示されている任意のメールをクリックして、そのメール統計ページを表示します。このページには、送信された日時や場所、受信者の反応を示すメトリクスが含まれています。
マイフォルダーをクリックして、特定のフォルダーに保存されたメールを表示します。
送信済みメールをフォルダーに整理するオプションもあります。
すべて表示をクリックして、開封済み、未開封、リンクがクリックされたなど、さまざまなステータスで送信済みメールのリストをフィルタリングします。
すべての時間をクリックして、今日、過去7日間、今月など、さまざまな期間で送信済みメールのリストをフィルタリングします。
ソートオプション
送信日でソートのドロップダウンをクリックするか、ドロップダウンから更新日でソートを選択して、表示されるメールリストをそれに応じてソートします。
降順のドロップダウンをクリックして最新のメールを最初に表示します。あるいは、ドロップダウンから昇順を選択して最も古いメールを最初に表示します。
