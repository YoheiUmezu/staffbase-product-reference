# ターゲットユーザーへの確認のための法的文書の再送信

**URL**: https://support.staffbase.com/hc/ja/articles/360011535879
**取得日**: 2026-05-02 18:44:53 JST

## 内容

ベータ
アクティベーション
管理者
法的遵守により、アプリのポリシーに変更があった場合は、アプリユーザーにその変更を提示し、変更を承認してもらう必要があります。影響を受けるユーザーのみから再確認を要求するには、まず更新された法的文書を選択し、再確認が送信されるタイミングを決定します。
この法的文書処理のベータ版はアクティブ化する必要があります。この機能を最適に使用するために、Staffbaseサポートまたはお客様成功マネージャーがすべての要件を満たしているか確認します。たとえば、すべてのモバイルアプリユーザーが最新のアプリバージョンを持っている必要があります。詳細については、Staffbaseサポートまたはお客様成功マネージャーにお問い合わせください。
確認のための法的文書のマーク
法的文書が大幅に更新され、影響を受けるユーザーからの再確認が必要な場合、その文書にマークを付けることができます。これは、次回の法的文書再確認がユーザーに送信される際に、以前にマークされたすべての法的文書がその一部となることを意味します。
スタジオで、設定 > 法的文書に移動します。
編集をクリックして、更新したい法的文書を開きます。
文書の編集ページが開きます。
文書を更新内容に合わせて修正します。
確認のチェックボックスを選択します。
これは大幅な更新であり、ユーザーからの確認が必要な場合のみこのオプションを選択してください。
保存をクリックします。
法的文書を関連するユーザーに再送信するために成功裏にマークしました。次に、マークされた法的文書を確認のために再送信します。
確認のための法的文書の再送信
スタジオで、設定 > 法的文書に移動します。
確認設定をクリックします。
確認ダイアログの設定が開きます。
確認のために再送信のトグルを有効にします。
保存をクリックします。
ユーザーに通知する確認メッセージが開きます。
すべてに通知をクリックします。
法的文書を確認のために成功裏に再送信しました。これはターゲットユーザーのみに送信されます。ユーザーは法的文書を受け入れるためのダイアログも受け取ります。
