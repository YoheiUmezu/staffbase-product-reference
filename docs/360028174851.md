# コンテンツを別のスペースに移動する

**URL**: https://support.staffbase.com/hc/ja/articles/360028174851
**取得日**: 2026-05-02 18:59:56 JST

## 内容

ニュースチャンネル、ページ、またはプラグインを別のスペースに移動する
ニュースチャンネル、ページ、またはプラグインを1つのスペースから別のスペースに移動できます。コンテンツを別のスペースに移動すると、コンテンツは他のスペースにコピーされるのではなく、移動されます。チャンネル、ページ、リンクなどのコンテンツを移動できます。
前提条件
複数のスペースが存在すること。
ニュース投稿を除くすべてのコンテンツを1つのスペースから別のスペースに移動するには、以下の手順を使用してください。ニュース投稿を移動するには、ニュース投稿を別のスペースに移動するをお読みください。
スタジオで、スペース選択ツールに移動し、すべてのコンテンツを選択します。
または、移動したいコンテンツのあるスペースにさらに移動できます。
移動したいコンテンツに移動します。
設定を開くためにギアアイコンをクリックします。
設定ページが開きます。
移動をクリックします。
コンテンツを移動したいスペースを選択します。
移動をクリックします。
コンテンツを別のスペースに正常に移動しました。コンテンツは新しいスペースに表示され、以前のスペースからは削除されます。
ニュース投稿を別のスペースに移動する
ニュース投稿を別のスペースに移動できますが、同じタイプのチャンネルに移動する必要があります。たとえば、更新投稿は更新タイプのチャンネルにのみ移動できます。投稿を別のスペースに移動すると、その投稿に関連するコメントやいいねも移動されます。いいねやコメントを移動したくない場合は、投稿を別のスペースにコピーすることができます。投稿が別のチャンネルに移動されると、以前の公開状態を保持します。たとえば、公開された投稿を別のチャンネルに移動すると、その投稿は公開モードのままで他のチャンネルに移動されます。
前提条件
複数のスペースが存在すること。
投稿を別のスペースに移動できるのは、すべてのコンテンツビューからのみです。スペースビューから別のスペースに投稿を移動することはできません。
投稿を移動すると、ユーザーは新しいチャンネルの可視性設定の変更により、投稿へのアクセスを失う可能性があります。
スタジオで、スペース選択ツールに移動し、すべてのコンテンツを選択します。
ニュース > チャンネルに移動します。
投稿を移動したいチャンネルをクリックします。
選択したチャンネルが開きます。
移動したい投稿を選択します。
移動をクリックします。
投稿を移動するためのダイアログが開きます。
ドロップダウンメニューから、投稿を移動したいチャンネルを選択します。
更新投稿を移動する場合、メニューには更新タイプのチャンネルのみが表示されます。
移動をクリックします。
投稿を別のスペースに正常に移動しました。投稿は、移動元のチャンネルには表示されなくなります。
