# ユーザープロファイルでの組織図の表示

**URL**: https://support.staffbase.com/hc/ja/articles/13833982083346
**取得日**: 2026-05-02 18:55:36 JST

## 内容

アクティベーション
組織図はユーザープロファイルに表示され、組織の階層構造の包括的な概要を提供します。これにより、マネージャー、直接の同僚、およびレポートを簡単に特定できます。
組織図はユーザーによって編集することはできません。これは、マネージャープロファイルフィールドの情報に基づいて自動的に生成されます。このフィールドは、組織の人事またはITシステムによって入力されます。詳細についてはこちらをご覧ください。
前提条件
マネージャーシステムプロファイルフィールドがアクティブであり、入力されていること。
組織図は3つのレベルで構成されています：
現在のユーザーのマネージャー
現在のユーザーと同じマネージャーにレポートする同僚
そのユーザーにレポートするユーザー
組織図へのアクセス方法
従業員アプリまたはイントラネットで、プロファイルメニューを開きます。
自分の名前をクリックします。
ユーザープロファイルが開きます。
組織図がユーザープロファイルに表示されます。
他のユーザーのカードをクリックして、そのユーザープロファイルを開きます。
オプションで、さらに表示をクリックして、利用可能な追加の従業員を表示します。
ユーザーにとっての組織図の見え方
