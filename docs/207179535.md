# パスワードの変更

**URL**: https://support.staffbase.com/hc/ja/articles/207179535
**取得日**: 2026-05-02 18:52:49 JST

## 内容

パスワードを変更するには、次の手順に従ってください：
iOS
モバイルアプリで、右上のメニューを開きます。
歯車アイコンをタップして設定を開きます。
設定の下で、アカウントをタップします。
アカウント設定の下で、パスワードを変更をタップします。
パスワードを変更ページが開きます。
現在のパスワードフィールドに、現在のパスワードを入力します。
新しいパスワードフィールドに、新しいパスワードを入力します。
新しいパスワードの確認フィールドに再度新しいパスワードを入力して確認します。
パスワードを変更をタップします。
パスワードが正常に変更されました。
↻
Android
モバイルアプリで、画面右上の三つの点をタップしてプロフィールメニューを開きます。
設定をタップします。
設定の下で、アカウントをタップします。
アカウント設定の下で、パスワードを変更をタップします。
パスワードを変更ページが開きます。
現在のパスワードフィールドに、現在のパスワードを入力します。
新しいパスワードフィールドに、新しいパスワードを入力します。
新しいパスワードの確認フィールドに再度新しいパスワードを入力して確認します。
パスワードを変更をタップします。
パスワードが正常に変更されました。
↻
Webアプリ
Webアプリで、プロフィールメニューを開き、設定をクリックします。
設定の下で、アカウントをクリックします。
アカウント設定のセクションで、パスワードを変更をクリックします。
パスワードを変更ページが開きます。
現在のパスワードフィールドに、現在のパスワードを入力します。
新しいパスワードフィールドに、新しいパスワードを入力します。
新しいパスワードの確認フィールドに再度新しいパスワードを入力して確認します。
パスワードを変更をクリックします。
パスワードが正常に変更されました。
↻
