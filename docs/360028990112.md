# 静的コンテンツ用のページ

**URL**: https://support.staffbase.com/hc/ja/articles/360028990112
**取得日**: 2026-05-02 19:01:16 JST

## 内容

ページを使うと、情報を常にアクセス可能な状態にできます。ページは、長期間にわたって重要で、時々小さな更新だけが必要な静的コンテンツに最適です。
例としては、以下のようなものがあります：
企業に関する情報
各部門の連絡先
アクセス方法や敷地図
安全規則
ページを作成するには、ページプラグインを追加する必要があります。ページ上のコンテンツはニュースページやチャンネルに依存しません。作成した各ページはメニューに表示されます。
ページの作成と管理方法については、ページ クイックスタートガイドをご覧ください。
