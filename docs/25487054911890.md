# 組織図の概要

**URL**: https://support.staffbase.com/hc/ja/articles/25487054911890
**取得日**: 2026-05-02 19:00:41 JST

## 内容

アクティベーション
組織図は、ユーザーが従業員アプリまたはイントラネットで最新の組織図を表示できるようにします。これにより、ユーザーが組織構造の中でどのように位置付けられているかを視覚化します。この方法で、誰に報告しているのか、同じチームの同じレベルの同僚、そして部下を確認できます。ユーザープロファイルには、すべての従業員を表示するグローバルな組織図はありません。
組織図は、マネージャープロファイルフィールドに見つかった情報に基づいて自動的に生成されます。そのフィールドは、組織の人事またはITシステムによって入力されます。一度設定されると、ユーザーには読み取り専用モードで提供されます。ユーザーが編集することはできず、ユーザープロファイルの他の部分とは異なります。詳細についてはこちらをご覧ください。
組織図の利点は何ですか？
素晴らしい概要
一目で、現在のユーザーのマネージャー、直接の同僚、および部下を確認できます。ユーザープロファイルに組織図を表示することで、報告プロセスが容易になるだけでなく、同僚が不在の場合の代理人を特定することも可能です。
簡単な接続
リモートワークの時代において、特に同僚間の関係を築くことがますます重要になっています。組織図から表示されたユーザーのカードをクリックすることで、異なるユーザープロファイルに直接ジャンプできます。これにより、部門や場所を超えたつながりを築くことができます。また、新入社員が同僚と会う前に知り合う素晴らしい機会でもあります。
低い労力
組織図は人事またはITシステムから自動的に生成されるため、管理が容易でエラーが発生しにくいです。設定後、組織図はユーザーに読み取り専用モードで提供されます。組織図は、特に急成長している組織にとって、新しい同僚を紹介し統合する素晴らしい機会です。
