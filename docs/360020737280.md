# 単一ユーザーの識別子の設定

**URL**: https://support.staffbase.com/hc/ja/articles/360020737280
**取得日**: 2026-05-02 18:47:34 JST

## 内容

管理者
外部システムでユーザーを識別するために、各ユーザーに識別子を設定できます。ユーザーの識別子を設定するための方法は多数あります。単一ユーザーの識別子を設定する場合、手動での方法が理想的です。
以下の手順では、従業員IDを識別子として使用します。
スタジオで、ユーザー > 登録ユーザーに移動します。
識別子は、保留中および無効化されたユーザーを含む任意のユーザーに設定できます。
編集をクリックしてユーザープロファイルを編集します。
ユーザープロファイルを編集するためのダイアログが開きます。
識別子フィールドに、ユーザーの識別子を入力します。
保存をクリックします。
