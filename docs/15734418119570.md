# キャンペーンからニュース投稿を作成する

**URL**: https://support.staffbase.com/hc/ja/articles/15734418119570
**取得日**: 2026-05-02 18:49:19 JST

## 内容

アクティベーション
システム全体のエディター
ニュース、更新、メディア投稿を作成し、それらをキャンペーンに一度のステップで割り当てることができます。
キャンペーンからニュース記事投稿を作成する
スタジオで、プランニング > キャンペーンに移動します。
ニュース記事投稿を作成したいキャンペーンに移動します。
+ アイコンをクリックします。
または、キャンペーンの概要を開いて投稿を作成をクリックします。
投稿を作成するためのダイアログが開きます。
ニュース記事チャンネルを選択します。
ダイアログが展開されます。
投稿のタイトルを入力します。
投稿のティーザーテキストを入力します。
投稿の日付と時間を入力します。
作成をクリックします。
ニュース記事投稿を作成し、それをキャンペーンに割り当てました。個々のキャンペーンの概要で、この投稿とそのリーチメトリクスを、同じキャンペーンに割り当てられた他の投稿と一緒に表示できます。
キャンペーンから更新投稿を作成する
スタジオで、プランニング > キャンペーンに移動します。
更新投稿を作成したいキャンペーンに移動します。
+ アイコンをクリックします。
または、キャンペーンの概要を開いて投稿を作成をクリックします。
投稿を作成するためのダイアログが開きます。
更新チャンネルを選択します。
ダイアログが展開されます。
投稿の説明を入力します。
投稿の日付と時間を入力します。
作成をクリックします。
更新投稿を作成し、それをキャンペーンに割り当てました。個々のキャンペーンの概要で、この投稿とそのリーチメトリクスを、同じキャンペーンに割り当てられた他の投稿と一緒に表示できます。
キャンペーンからメディア投稿を作成する
スタジオで、プランニング > キャンペーンに移動します。
メディア投稿を作成したいキャンペーンに移動します。
+ アイコンをクリックします。
または、キャンペーンの概要を開いて投稿を作成をクリックします。
投稿を作成するためのダイアログが開きます。
メディアチャンネルを選択します。
エディターを開くをクリックします。
コンテンツエディターが開きます。
投稿の画像または動画を選択します。
投稿の説明を入力します。
ドラフトを保存をクリックします。
メディア投稿を作成し、それをキャンペーンに割り当てました。個々のキャンペーンの概要で、この投稿とそのリーチメトリクスを、同じキャンペーンに割り当てられた他の投稿と一緒に表示できます。
