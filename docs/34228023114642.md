# フィルターを使ってユーザーアカウントを絞り込む

**URL**: https://support.staffbase.com/hc/ja/articles/34228023114642
**取得日**: 2026-05-02 18:40:44 JST

## 内容

管理者
ユーザーページでは、Staffbaseプラットフォーム上のユーザー基盤の概要を確認できます。役割、ステータス、グループ、作成方法、またはユーザータイプでユーザーを検索、フィルター、並べ替えして、必要なアカウントをすばやく見つけることができます。特定したアカウントは、編集や削除を行い、ユーザー基盤を効果的に管理できます。
Studioでユーザーに移動します。
フィルターを表示するをクリックします。
必要に応じて、以下のいずれかのフィルターオプションを1つ以上選択します。
役割
ステータス
グループ
招待タイプ
ユーザータイプ
適用するをクリックします。
必要なすべてのフィルターを適用するまで、手順3～4を繰り返します。
必要なオプションでユーザーアカウントがフィルターされました。
