# ポートフォリオウィジェットの設定

**URL**: https://support.staffbase.com/hc/ja/articles/25488197350290
**取得日**: 2026-05-02 18:57:56 JST

## 内容

エディター
ポートフォリオウィジェットを使用すると、カバー画像、タイトル、説明文を含むアルバム形式でコンテンツを表示できます。タイルはアプリ内のコンテンツや外部サイトへのリンクを持つことができます。
ポートフォリオウィジェットの設定
ポートフォリオウィジェットを追加します。
ポートフォリオウィジェットを設定するためのダイアログが開きます。
オプションで、タイトルを提供します。
タイトルはウィジェットの上に表示され、ユーザーに見えるようになります。
ウィジェットが表示されるデバイスタイプを、対応するアイコンをクリックして選択します。
ウィジェットが表示されるユーザーをアプリ内の表示で選択します。
ポートフォリオウィジェットの設定が完了しました。保存する前に、ポートフォリオアイテムを作成する必要があります。
ポートフォリオアイテムの追加
ポートフォリオアイテムのタイトルを提供します。
タイトルはユーザーに見えます。アイテムの内容を補完するタイトルを使用してください。
オプションで、ポートフォリオアイテムの画像を追加します。
オプションで、ポートフォリオアイテムのリンクを追加します。
ポートフォリオアイテムにリンクを追加すると、そのアイテムをクリックすることで自動的にリンク先のリソースに移動します。
オプションで、ステップ1-3を繰り返してポートフォリオにさらにアイテムを追加します。
すべてのアイテムを追加したらOKをクリックします。いつでもさらにアイテムを追加したり、調整したりできます。
オプションで、エディターで作成したばかりのポートフォリオに移動し、タイトルの下に説明文を追加してユーザーにさらなる有益な情報を提供します。
