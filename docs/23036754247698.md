# ジャーニーのための多言語コンテンツの作成

**URL**: https://support.staffbase.com/hc/ja/articles/23036754247698
**取得日**: 2026-05-02 18:58:49 JST

## 内容

アクティベーション
システム全体のエディター
従業員アプリまたはイントラネットで、複数の言語でジャーニーを作成できます。これにより、ジャーニーの対象オーディエンスは、自分が選択したコンテンツ言語でコンテンツを読むことができます。ジャーニーに多言語コンテンツを追加する際には、組織で有効になっている言語を選択できます。追加の言語が必要な場合は、管理者がそれを有効にできます。
さらに、他の言語から既存のコンテンツをコピーし、機械翻訳を使用して追加した言語に自動的に翻訳することができます。 Staffbaseは、機械翻訳されたコンテンツに誤りがないことを確認するために、常に校正することを推奨しています。
複数の言語でコンテンツを作成することは、組織の設定が行われている場合にのみ可能です。 詳細はこちら。このオプションが必要な場合は、管理者に連絡してください。
ジャーニーで多言語コンテンツを作成する方法は3つあります。
ゼロから始める
このオプションでは、手動で翻訳を追加し、ジャーニーのステップで選択した言語の新しいタブを作成します。テキストフィールドは空のままで、新しい言語で直接コンテンツを書くか、コピー＆ペーストできます。
コンテンツをコピー
このオプションでは、ジャーニーのステップで選択した言語の新しいタブを作成し、元の言語のコンテンツをテキストフィールドにコピーします。
自動的にコピーして翻訳
このオプションでは、ジャーニーのステップで選択した言語の新しいタブを作成し、コンテンツを新しい言語に自動的に翻訳します。そこから、必要に応じてコンテンツを追加または編集できます。
手動で多言語コンテンツを追加する
このオプションでは、新しい言語で手動でコンテンツを追加できる新しいタブが作成されます。
スタジオで、コンテンツ > ジャーニーに移動します。
言語を追加したいジャーニーを開きます。
ジャーニーのステップに移動し、編集をクリックします。
+ ボタンをクリックします。
言語を追加ダイアログが開きます。
ドロップダウンメニューから1つ以上の言語を選択します。
ゼロから始めるを選択します。
追加をクリックします。
新しい言語のタブが表示されます。
ジャーニーのステップのタイトルを入力します。
保存をクリックします。
1つ以上の新しい言語をジャーニーのステップに追加しました。各言語には、ステップの上部に異なるタブがあります。これで、それらの言語で新しいコンテンツを書くことができます。すべてのジャーニーステップに対してこの手順を繰り返してください。
翻訳なしで多言語コンテンツを追加する
アクティベーション
このオプションでは、コピーしている言語の既存のコンテンツのバージョンを含む新しいタブが作成されます。
スタジオで、コンテンツ > ジャーニーに移動します。
言語を追加したいジャーニーを開きます。
ジャーニーのステップに移動し、編集をクリックします。
+ ボタンをクリックします。
言語を追加ダイアログが開きます。
ドロップダウンメニューから1つ以上の言語を選択します。
コンテンツをコピーを選択します。
コピー元のコンテンツドロップダウンメニューが表示されます。
コンテンツをコピーする言語を選択します。
追加をクリックします。
保存をクリックします。
1つ以上の新しい言語をジャーニーのステップに追加しました。各言語には、ステップの上部に異なるタブがあり、選択したソースからコピーされたコンテンツがあります。すべてのジャーニーステップに対してこの手順を繰り返してください。
自動翻訳で多言語コンテンツを追加する
アクティベーション
このオプションでは、新しい言語に自動的に翻訳された既存のコンテンツを含む新しいタブが作成されます。
スタジオで、コンテンツ > ジャーニーに移動します。
言語を追加したいジャーニーを開きます。
ジャーニーのステップに移動し、編集をクリックします。
+ ボタンをクリックします。
言語を追加ダイアログが開きます。
ドロップダウンメニューから1つ以上の言語を選択します。
自動的にコピーして翻訳を選択します。
自動的にコピーして翻訳元ドロップダウンメニューが表示されます。
翻訳するコンテンツの言語を選択します。
追加をクリックします。
保存をクリックします。
1つ以上の新しい言語を旅のステップに追加しました。各言語はステップの上部に異なるタブがあり、選択したソースから新しい言語に翻訳されたコンテンツがあります。すべての旅のステップに対してこの手順を繰り返します。
多言語コンテンツの削除
もはや必要ない場合や誤って追加した場合に備えて、言語コンテンツのバージョンを個別に削除できます。
スタジオで、コンテンツ > 旅に移動します。
言語を削除したい旅を開きます 。
旅のステップに移動し、編集をクリックします。
ゴミ箱アイコンをクリックします。
言語を削除ダイアログが開きます。
ドロップダウンメニューから1つ以上の言語を選択します。
削除をクリックします。
言語を削除ダイアログが開きます。
削除をクリックします。
保存をクリックします。
選択した言語バージョンの旅のステップは永久に削除されました。いつでも言語バージョンを再追加できます。
