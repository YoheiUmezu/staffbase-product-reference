# メールデザインにおけるテーブルの使用

**URL**: https://support.staffbase.com/hc/ja/articles/19542431325970
**取得日**: 2026-05-02 18:43:13 JST

## 内容

ベータ
アクティベーション
エディター
メール内でテーブルを簡単に作成および編集できます。メールデザイナーを使用して、新しいテーブルを設計し、データを明確に整理するか、既存のテーブルをコピーして簡単に編集します。これにより、スムーズで効率的なコミュニケーションが確保され、生産性が向上し、メールにプロフェッショナルな印象を与えます。
ゼロからテーブルを作成する
従業員メールのウェブアプリで、テーブルを追加したいメールのドラフトを開きます。
テーブルを追加したいブロックを選択します。
クイックブロックの下で、テーブルをクリックします。
テーブルブロックがメールのドラフトに挿入されます。
テーブルブロックをクリックします。
メニューが開きます。
メニューから、列または行のアイコンを選択して、テーブルに行または列を追加します。
トグルを調整して、ヘッダー行またはヘッダー列を追加します。
メールにテーブルを作成しました。いつでもテーブルを編集できます。
既存のテーブルをメールにコピーする
従業員メールのウェブアプリで、テーブルを追加したいメールのドラフトを開きます。
テーブルを追加したいブロックを選択します。
クイックブロックの下で、テーブルをクリックします。
テーブルブロックがメールのドラフトに挿入されます。
クリップボードからテーブルをテーブルブロックに貼り付けます。
テーブルをメールにコピーしました。いつでもテーブルを編集できます。
テーブルと一緒にコピーされたカスタムフォントは、編集中のメールデザイナーでは表示されない場合があります。ただし、プレビューや送信されたメールでは表示されます。
テーブルの編集
テーブルを作成またはコピーした後、行や列を追加または削除したり、セルを結合したり、その他の変更を加えたりして編集できます。
編集したいセル、行、または列をクリックします。
メニューが開きます。
メニューから、テーブルのデザインを変更する方法を選択します。
カスタマイズは、メールデザイナーに直接表示されます。
オプションで、セル内のテキストをクリックして編集または削除します。
Staffbaseは、テーブル内に画像などの非テキスト要素を追加することを推奨していません。
テーブルのデザインと内容を編集し、メールのプレビュー、送信、またはスケジュールの準備が整いました。
