# コメントを非表示にする

**URL**: https://support.staffbase.com/hc/ja/articles/4404007650450
**取得日**: 2026-05-02 18:50:01 JST

## 内容

エディター
コメントのモデレーションは、従業員が意見を共有するためのプラットフォームをより安全で安心な場所にするのに役立ちます。エディターが不適切なコメントの可能性があるものを見た場合、モデレーションチームによるレビューのためにそれらを非表示にすることができます。
ニュースチャンネルのエディターのみが非表示のコメントを見ることができます。他のユーザーは、コメントが現在レビュー中であることを示すプレースホルダーを見ることになります。コメントへの既存の返信は引き続き表示され、コメントに返信することも可能です。
コメントは以下の場所から非表示にできます：
スタジオのコメント概要
ニュース投稿から直接
コメント概要からコメントを非表示にする
スタジオで、コンテンツ > システム > コメント。
コメント概要で、非表示にしたいコメントを探します。
モデレーションのために非表示にするをクリックします。コメントを非表示にするための確認ダイアログが開きます。
非表示をクリックします。
コメントが非表示になりました。コメントはユーザーには表示されず、モデレーションチームによるレビューが可能になります。
投稿ビューから直接コメントを非表示にする
投稿で、非表示にしたいコメントを探します。
アクションメニューを開くために三つの点をクリックします。
モデレーションのために非表示をクリックします。
コメントを非表示にするための確認ダイアログが開きます。
非表示をクリックします。
コメントが非表示になりました。コメントはユーザーには表示されず、モデレーションチームによるレビューが可能になります。
