# サードパーティブラウザクッキー

**URL**: https://support.staffbase.com/hc/ja/articles/4412022029458
**取得日**: 2026-05-02 18:44:23 JST

## 内容

iframeを介して別のウェブサイトを埋め込むことは、最も簡単な統合オプションの1つです。しかし、外部コンテンツが埋め込み可能であることが重要です。たとえば、必要なクッキー設定が整っている必要があります。
2021年以降、多くのブラウザは埋め込まれたコンテンツのSet-Cookieヘッダーに対して以下の設定を要求しています。クッキー設定は次のようになります：
Set-Cookie: session=a-session; SameSite=None; Secure
外部埋め込みコンテンツがStaffbaseプラットフォームで正しく表示されない場合は、この情報をIT部門または外部コンテンツの所有者に転送してください。Staffbaseはプラットフォーム外のコンテンツを修正することはできません。
