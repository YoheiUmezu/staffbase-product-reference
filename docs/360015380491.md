# アカウント回復方法の選択

**URL**: https://support.staffbase.com/hc/ja/articles/360015380491
**取得日**: 2026-05-02 18:45:08 JST

## 内容

Staffbaseは、必要に応じてユーザーアカウントを簡単かつ安全に回復する方法を提供します。
永久回復コード
設定
次のいずれかの方法で回復コードを設定する必要があります：
ユーザーデータのインポート（例：CSVインポート）による回復コードのインポート
CSVインポート時にシステムが回復コードを生成する
回復コードを手動で設定する
詳細
ユーザー名でオンボーディングされたユーザーは、永久回復コードを使用してアカウントを回復します。
永久回復コードは特定のユーザーに属します。これらの事前定義された回復コードの有効期限を1時間や1年などの時間範囲に制限できます。時間範囲の設定は非常に柔軟です。
一時回復コード
設定
特に何もする必要はありません。ただし、ユーザーがメールアドレスでサインインすることが重要です。
詳細
メールを通じてオンボーディングされたユーザーは、サインイン用のメールアドレスを使用して一時回復コードを受け取ることでアカウントを回復します。
一時回復コードは、パスワードリセットのプロセスが開始されたときにのみ要求に応じて生成されます。一時回復コードは48時間のみ有効であり、送信されたメールアドレスでのみ使用できます。同じユーザーが再度アカウントを回復する必要がある場合、システムは新しい一時コードを生成します。
