# カードスタイルでのページ表示

**URL**: https://support.staffbase.com/hc/ja/articles/360010073620
**取得日**: 2026-05-02 19:02:19 JST

## 内容

アクティベーション
エディター
ページのスタイルとして以下のいずれかを選択できます:
カードスタイル
非カードスタイル
スタジオで、コンテンツ > ページに移動します。
レイアウトを変更したいページに移動します。
ページを編集 が開きます。
エディターのツールバーで、セクションアイコンをクリックします。
ページレイアウトオプションのツールバーが開きます。
トグルページをカードとして表示をスライドさせて、ページのレイアウトを定義します。
保存 をクリックして、ページの変更を保存します。
アプリでのカードスタイルページと非カードスタイルページのユーザー表示
