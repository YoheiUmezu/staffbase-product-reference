# RSSフィードの統合

**URL**: https://support.staffbase.com/hc/ja/articles/360027894572
**取得日**: 2026-05-02 18:57:34 JST

## 内容

RSSフィードを使用して、外部ウェブサイトのコンテンツをStaffbaseプラットフォームに表示できます。これらは、記事または更新ニュースチャンネルタイプで使用できます。RSSフィードは独立したニュースチャンネルとして使用することも、組織の独自のコンテンツと一緒に使用することもできます。
RSSフィードに新しいコンテンツが利用可能になると、それはStaffbase内のニュースチャンネルに転送されます。更新は毎時行われます。RSSフィードには、画像や動画などの外部ウェブサイトからのメディアは含まれません。表示できるのは、ソースに基づいた1つの言語のみです。
RSSフィードのサイズは300 KBを超えてはいけません。スクリプトヘッダーには、言語、タイトル、発行日を含める必要があります。ソースが300 KBを超えるか、これらの属性がない場合、URLは拒否されます。フィードが機能しない場合は、フィード検証ツールを使用してURL属性を確認するか、ソースの作成者に連絡してください。
スタジオで、コンテンツ > ニュースに移動し、RSSフィードを追加したいチャンネルに移動します。
設定を開くためにギアアイコンをクリックします。
チャンネルの設定ページが開きます。
プラグイン設定: チャンネル > ソースに移動します。
ソースを追加をクリックします。
ソースを追加ダイアログが開きます。
ソースURLとソース名を提供します。
追加をクリックします。
ニュースチャンネルにRSSフィードを追加しました。外部ウェブサイトからのテキストコンテンツは自動的にチャンネルに転送されます。
