# アナリティクス・ダッシュボードを使用したナビゲーター指標の分析

**URL**: https://support.staffbase.com/hc/ja/articles/34804442272658
**取得日**: 2026-05-02 19:00:35 JST

## 内容

有効化
編集者
アナリティクスダッシュボードは、ユーザーがStaffbaseプラットフォームをどのように、いつ、どこで利用しているかについて、より深いインサイトを提供します。アナリティクスダッシュボードは異なるカテゴリーに分かれており、指標を詳しく分析できます。
この記事では、ナビゲータタブの詳細、テーブルや分析データについて説明します。アナリティクスダッシュボード全体やユーザー役割のアクセスについては、アナリティクスダッシュボードの概要をご覧ください。
アナリティクス用語についてはアナリティクス用語集 をご覧ください。管理者がアクセス権を付与している場合、ナビゲータタブにアクセスできます。
フィルター
ナビゲータタブは、デフォルトで先週のデータを表示します。日付範囲を変更したり、さらにフィルターを使って結果を絞り込むことができます。フィルターの詳細はこちら
指標
ナビゲータタブには、詳細な指標を含むテーブルと、会話ログごとの個別指標があります。
会話リスト
会話リストは、プラットフォーム内でやり取りがあったさまざまな種類のコンテンツの広範な概要です。デフォルトでは、タブを開いたときに日付範囲フィルターが7日間に設定されています。このリストには、その期間中にナビゲーターで行われたすべての会話が表示されます。別の日付範囲を選択すると、さらに多くの会話が表示されます。
テーブルの各ヘッダーは、コンテンツの詳細を示します。テーブルヘッダーは以下の通りです：
ナビゲーション ヘッダー
説明
会話
会話の簡単なまとめ
モード
音声またはテキストの入力方法が使用されたかどうか
デバイス
やり取りに使用されたデバイス
言語
会話中に使用された言語
メッセージ
会話中に送信されたメッセージ数
報告された問題
やり取り中にユーザーから報告された問題
会話タイトルをクリックすると、会話ログを開くことができます。テーブルは自動的に最新順でソートされます。フィルターを使ってリストをさらにカスタマイズできます。
個別会話ログ
会話を開くと、ナビゲーターとの特定のやり取りについて詳細情報を確認できます。これらの詳細を確認することで、知識のギャップを特定し、応答品質の向上に役立ちます。ユーザーの身元は表示されません。また、プライバシー設定が有効な場合、会話は非表示となり、ナビゲーターに送信されたプロンプトのみが表示されます。
各ログには以下の情報が含まれます：
まとめ：やり取りの簡単な概要
インテント：クエリの背後にあるユーザーの意図
報告された問題：やり取り中にユーザーからフラグされた問題
解決状態：問題の現ステータス
トランスクリプト：ユーザーとナビゲーター間の全会話。プライバシー設定が有効な場合は、ナビゲーターによる検索プロンプトのみが表示されます。
