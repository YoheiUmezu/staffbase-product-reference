# Staffbase EmailでのCSVインポートの設定

**URL**: https://support.staffbase.com/hc/ja/articles/25485833074834
**取得日**: 2026-05-02 18:42:18 JST

## 内容

ベータ
アクティベーション
管理者
Staffbase EmailをStaffbase IntranetまたはEmployee Appと一緒に使用する場合は、CSVインポートの設定を参照してください。以下の情報は、Staffbase Emailがスタンドアロン製品として使用される場合にのみ適用されます。
CSVファイルをアップロードした後、インポートの準備を行うことができます。
設定の構成
スタジオで、ユーザー > ユーザーインポートと同期に移動します。
インポートしたいファイルに対応するアクションメニューから、インポートを開始をクリックします。
インポートを開始するオプションは、ドラフトステータスのファイルにのみ表示されます。
設定ページが開きます。
スコープの下で、次のいずれかを選択します：
ユーザーインポート
このオプションは、スタジオユーザーのデータを更新し、アカウントをスタジオに登録するためのメール招待を送信します。
連絡先インポート
このオプションは、メールを送信できる連絡先のユーザーデータを更新します。連絡先にアカウントをスタジオに登録するためのメール招待は送信されません。
インポートタイプの下で、次のいずれかを選択します：
選択したユーザー
このオプションは、CSVファイルに含まれるユーザーのデータのみを更新します。プラットフォーム内の残りのユーザーデータはそのまま保持されます。
このオプションはデルタインポートとも呼ばれます。デルタインポートを使用してユーザーをオフボードする場合は、各ユーザーのアカウントステータスを含む列をCSVファイルに追加する必要があります。詳細を学ぶ。
すべてのユーザー
このオプションは、CSVファイル内のユーザーとプラットフォーム内のユーザーを含むすべてのユーザーを更新します。また、アクティブなユーザーを非アクティブにし、CSVファイルに含まれていない保留中のユーザーを削除します。
テキストエンコーディングのドロップダウンから、CSVファイルで使用されているテキストエンコーディングの種類を選択します。
セル区切り文字フィールドに、CSVファイル内のセルを区切るために使用される値を入力します。
テキストエンコーディングのドロップダウンから、CSVファイルで使用されているテキストエンコーディングの種類を選択します。
次へをクリックします。
マッピングページが開きます。
マッピングの設定
マッピングページで、識別子がマップされるの下にあるチェックボックス識別子を選択します。
プラットフォーム内のプロファイルフィールドに従って、CSVの列名をマッピングします。
デルタインポートを使用してユーザーをオフボードするには、アカウントステータス値を含むCSV列をステータスにマッピングします。
オプションで、次のオプションを選択します：
回復コードを自動生成しますか？：ユーザーのための回復コードを生成するため
新しいユーザーをメールで招待または保留中のユーザーをメールで招待：メール招待を送信するため
ユーザーをメールで招待するオプションは、CSV列を識別メールアドレスにマッピングした場合にのみ表示されます。
プレビューを作成をクリックします。
直接ファイルをインポートをクリックすることもできますが、Staffbaseはプレビューを確認してユーザーベースに加えようとしている更新を確認することを推奨します。
ファイルをマッピングし、プレビューを作成しました。今、あなたはインポートを確認します。
