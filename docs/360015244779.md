# ジャーニーの作成

**URL**: https://support.staffbase.com/hc/ja/articles/360015244779
**取得日**: 2026-05-02 18:52:17 JST

## 内容

アクティベーション
システム全体のエディター
Journeysプラグインをインストールしたら、個々の従業員のジャーニーに合わせたコンテンツの作成を開始できます。Journeysを使用して新しい従業員を歓迎し、オンボードし、ユーザーが達成したベンチマークを祝ったり、新しいユーザーグループでユーザーがくつろげるようにします。
ジャーニーへの貢献者の追加
ジャーニーを作成する際に、ジャーニーのステップを追加、更新、または削除するために貢献者を割り当てることができます。これらの貢献者は、ジャーニーの設定を変更したり、ジャーニーをアクティブまたは非アクティブにすることはできません。このようにして、必要のない権限を与えることなく、貢献者にコンテンツ管理のタスクを委任できます。
スタジオで、コンテンツ > ジャーニーに移動します。
ジャーニーの追加をクリックします。
ジャーニーの追加ダイアログが開きます。
ジャーニーの名前を提供します。
ジャーニーのスペースを選択します。
ジャーニーのステップを追加、更新、または削除するための貢献者を選択します。
ジャーニーの追加をクリックします。
ターゲットオーディエンスを設定するためのジャーニーの開始点の設定
ジャーニーのターゲットオーディエンスを設定する際、誰が特定のジャーニーを開始し、どのように開始するかを決定します。ターゲットオーディエンスの設定を開始するために、2つの開始点のいずれかを選択します：
新しいアプリユーザーのオンボード
すべての新しいアプリユーザーが自動的にジャーニーを開始するようにしたい場合、新しいユーザーがアプリに登録すると同時にジャーニーが開始されるように設定できます。
スタジオで、ジャーニーに移動します。
設定したいジャーニーを選択するか、ジャーニーの追加をクリックして新しいジャーニーを追加します。
開始点の設定をクリックします。
ジャーニーのターゲットグループを設定するためのダイアログが開きます。
新しいユーザーのオンボードをクリックします。
次へをクリックします。
ドロップダウンメニューから、すべての新しいユーザーまたは選択したユーザーグループの新しいユーザーのみがジャーニーコンテンツを受け取るかを選択します。
完了をクリックします。
ジャーニーのターゲットオーディエンスが保存されました。
ユーザーグループへのユーザーのオンボード
特定のユーザーグループに新しいメンバーをオンボードするためにジャーニーを使用することは、新しいグループメンバーを歓迎するだけでなく、グループの特性や目的を紹介する素晴らしい方法です。これにより、新しいグループメンバーにグループの重要な基本を提供し、既存のグループメンバーに繰り返しコンテンツで圧倒されることなく情報を伝えることができます。
スタジオで、ジャーニーに移動します。
編集したいジャーニーを選択するか、ジャーニーの追加をクリックして新しいジャーニーを追加します。
開始点の設定をクリックします。
ジャーニーのターゲットグループを設定するためのダイアログが開きます。
特定のグループへのユーザーのオンボードをクリックします。
次へをクリックします。
アプリ内の可視性の下で、新しいユーザーがジャーニーコンテンツを受け取るユーザーグループを選択します。
ジャーニーへの再参加の下で、ユーザーがジャーニーを完了する前にユーザーグループを離れて再参加した場合に何が起こるかを選択します。
ジャーニーを開始するグループメンバーの下で、ジャーニーを新しいメンバーのみにアクティブにするか、すべてのグループメンバーにアクティブにするかを選択します。
完了をクリックします。ジャーニーのターゲットオーディエンスが保存されました。
ターゲットグループの設定が完了したら、最初のジャーニーステップを追加してジャーニーのコンテンツ作成を開始できます。
