# Microsoft Teamsにおけるアプリのブランディングのカスタマイズ

**URL**: https://support.staffbase.com/hc/ja/articles/19127717557010
**取得日**: 2026-05-02 18:55:29 JST

## 内容

アクティベーション
管理者
Microsoft Teams環境内でアプリの外観をカスタマイズして、認識しやすくすることができます。機能を変更することなく、アプリの外観を組織のブランディングガイドラインに合わせて修正できます。
Microsoft Teams管理センターで、Teamsアプリ > アプリの管理に移動します。
または、Microsoft 365管理センターからMicrosoft Teamsにアクセスします。
カスタマイズしたいアプリを選択します。
アプリの詳細が表示されます。
設定とカスタマイズで、初めてアプリをカスタマイズする際は、カスタマイズの追加をクリックします。
次回以降は、アプリを選択して編集をクリックします。
カスタマイズの編集ダイアログが開きます。
カスタマイズの編集で、アプリの詳細、アイコン、色を更新します。詳細を学ぶ。
適用をクリックします。
アプリのブランディングをカスタマイズしました。カスタマイズの更新がユーザーに表示されるまでに最大24時間かかる場合があります。
