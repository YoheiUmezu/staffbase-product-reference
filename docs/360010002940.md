# Appleビジネスマネージャーアカウントの設定

**URL**: https://support.staffbase.com/hc/ja/articles/360010002940
**取得日**: 2026-05-02 18:50:21 JST

## 内容

アクティベーション
管理者
Appleビジネスマネージャーアカウントの登録リクエストが承認されると、アカウントを設定できます。
追加の管理者を追加する
ABMアカウントに追加の管理者を追加することが重要です。初期の管理者がサインインできない場合、追加の管理者がいることで、組織がアカウントを回復できるようになります。
Appleビジネスマネージャーにサインインします。
アカウントに移動します。
新しい管理者のために追加のアカウントを追加します。
管理者に必要な詳細を提供し、保存します。
追加の管理者に招待を送信します。
カスタムアプリの配布を有効にする
Appleビジネスマネージャーで、設定 > 登録情報に移動します。
組織の税情報を追加します。
ダウンロードページを使用したアプリ配布オプションには、後のプロセスで引換コードが必要です。引換コードを購入するためには、組織の税情報を追加する必要があります。この購入には費用はかかりませんが、Appleビジネスマネージャーでの購入として処理されます。
カスタムアプリに移動します。
カスタムアプリの配布を有効にします。
後で、アプリがStaffbaseによってレビューされ承認されると、カスタムアプリの下に表示されます。
