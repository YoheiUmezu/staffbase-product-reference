# ユーザー権限と役割

**URL**: https://support.staffbase.com/hc/ja/articles/208282825
**取得日**: 2026-05-02 18:45:15 JST

## 内容

役割によるユーザー権限管理
アプリを管理し、運用を維持するためには、アプリを担当するチームが必要です。通常のユーザーはアプリを見ることしかできません。コンテンツを追加したり、コンテンツをレビューしたり、ユーザーを管理したりするには、特定のユーザーに拡張権限を付与する必要があります。
Staffbaseは、役割によって異なる目的のための権限を管理します。ユーザーに特定の役割を与えると、そのユーザーは単なるユーザー以上の権限を得ることができます。コンテンツの作成や編集に明確に焦点を当てた役割もあれば、ユーザー管理やアプリ設定の調整など、より広範なタスクをカバーする役割もあります。
コンテンツ特化型の役割
例えば、特定のニュースチャンネルでニュース投稿を投稿、編集、削除することのみ許可されているコンテンツエディターを持つことができます。この権限は、他のニュースチャンネル、ページ、またはプラグインには影響しません。コンテンツ特化型の役割は、コンテンツに直接設定されます。
システム全体の役割
一方、システム全体のエディターは、アプリ全体のコンテンツを管理することが許可されています。システム全体の役割は、ユーザー管理で設定されます。
