# ニュース投稿を別のスペースにコピーする

**URL**: https://support.staffbase.com/hc/ja/articles/360010698080
**取得日**: 2026-05-02 19:00:29 JST

## 内容

ニュース投稿を異なるスペースにコピーできますが、同じタイプのチャンネルに移動する必要があります。たとえば、更新投稿は更新タイプのチャンネルにのみコピーできます。投稿を別のチャンネルにコピーすると、他のチャンネルにドラフトモードで投稿が複製されます。その後、修正して公開できます。
前提条件
1つ以上のスペースが存在すること。
投稿を別の投稿にコピーできるのは、すべてのコンテンツビューからのみです。スペースビューから別のスペースに投稿を移動することはできません。
スタジオで、スペース選択ツールに移動し、すべてのコンテンツを選択します。
ニュース > チャンネルに移動します。
コピーしたい投稿のあるチャンネルをクリックします。
選択したチャンネルが開きます。
コピーしたい投稿を選択します。
コピーをクリックします。
投稿をコピーするためのダイアログが開きます。
ドロップダウンメニューから、投稿をコピーしたいチャンネルを選択します。
更新投稿をコピーする場合、メニューには更新タイプのチャンネルのみが表示されます。
コピーをクリックします。
投稿が正常にコピーされました。投稿は両方のチャンネルで利用可能です。
