# Microsoft Teamsでのコラボレーションの有効化

**URL**: https://support.staffbase.com/hc/ja/articles/16228455211154
**取得日**: 2026-05-02 18:46:46 JST

## 内容

ベータ
有効化
管理者
StaffbaseプラットフォームからTeams内で他の同僚とコラボレーションできるようにするために、Microsoft Teamsのコラボレーションオプションを有効化する必要があります。
前提条件:
StudioでMicrosoft Connectionsを設定していること。
このコラボレーションの利用可能性を特定のユーザーグループに制限したい場合は、Microsoft Connectionsの設定でアプリの可視性を変更する必要があります。他のMicrosoft接続統合がある場合、可視性設定はそれらにも影響します。
再設計されたユーザープロファイルページ（ベータ、ヘッダー画像で認識可能）が有効化されていること。
ユーザーのMicrosoft Teamsのメールアドレスを、Staffbaseプラットフォームのユーザーのデフォルトの公開メールアドレスとして追加していること。
Studioで、設定 > サードパーティ接続に移動します。
Microsoft 365の下で、Microsoft Teamsでのコラボレーションのトグルを有効にします。
Microsoft 365ウィジェットを有効化するためのダイアログが開きます。
有効化をクリックします。
Microsoft Teamsのドロップダウンはユーザープロファイルにのみ表示されます:
他のユーザーの（自分のユーザープロファイルには表示されません）。
Staffbaseプラットフォームで公開メールアドレスとして有効なMicrosoftメールアドレスを持つ。
ユーザーのプロファイルでMicrosoft Teamsでのコラボレーションが有効になりました。このオプションはいつでも再度無効にできます。
