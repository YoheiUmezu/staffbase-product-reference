# Navigator の概要

**URL**: https://support.staffbase.com/hc/ja/articles/34666654272786
**取得日**: 2026-05-02 18:40:35 JST

## 内容

アクティベーション
AI
Navigatorは、Staffbaseアプリとイントラネットをインテリジェントな職場ガイドに変えます。ユーザーは音声またはテキストで、どの言語でも質問でき、企業知識に基づいたインスタントな回答を受け取ることができます。
Navigatorは、組み込みのカスタマイズ、多言語サポート、アナリティクスを備えた、一貫性がありブランド化されたスケーラブルなAI搭載のユーザー体験を提供します。
さらに、プラットフォームにサービスナウ連携がある場合、Navigatorと一緒に使用できます。
Navigatorを使用するメリット
インフォメーションへの迅速かつ簡単なアクセス
ユーザーは会話のきっかけを使い、複数のページを移動することなく素早く回答を見つけることができ、検索時間を短縮し生産性を向上させます。
一貫性があり包括的なコミュニケーション
Navigatorは一貫したトーンで回答を提供し、複数の言語をサポートするため、デスクトップと現場のチームの両方がインフォメーションにアクセスできます。
インサイトと継続的な改善
アナリティクスとユーザーフィードバックにより、ユーザーのニーズの可視化が可能になり、組織は知識のギャップを特定しコンテンツを改善できます。
Navigatorのユーザー画面
ユーザーはさまざまな方法でインテリジェントな職場ガイドとやり取りできます。
