# Outlook Classic アドインでのリンクトラッキングの無効化

**URL**: https://support.staffbase.com/hc/ja/articles/4415079668626
**取得日**: 2026-05-02 18:51:45 JST

## 内容

トラッキングされたメールに含まれる各リンクへのクリック数を測定するか無視するかを選択できます。これは通常、Outlook Classic アドインでデフォルトで有効になっていますが、設定でオフにすることもできます。
リンクトラッキングを無効にする
Outlook リボンで、 Staffbase で作成をクリックします。
保存したメールデザイナのドラフトのリストが表示されます。
ギアアイコン をクリックして設定メニューを開きます。
リンクトラッキングを有効にするを オフに切り替えます。
この設定により、トラッキングされたメールに含まれるリンクはEmployee Emailによって無視されます。
