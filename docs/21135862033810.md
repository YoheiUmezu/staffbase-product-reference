# ポストウィジェットの設定

**URL**: https://support.staffbase.com/hc/ja/articles/21135862033810
**取得日**: 2026-05-02 18:46:28 JST

## 内容

エディター
ポストウィジェットを使用すると、ユーザーはページから直接ニュース投稿を作成できます。ポストウィジェットを任意のページに戦略的に配置することで、ユーザーが迅速かつ直感的に投稿できるようになります。ユーザーは投稿を書くためのプロンプトを見て、投稿が公開されると、ニュースチャンネルが表示される場所であればどこでも見ることができます。
モバイルアプリでは、入力フィールドを画面の端に浮かぶボタンに置き換えるオプションもあり、ユーザーは数秒で新しいコンテンツを追加できます。
ポストウィジェットは、従業員が会社全体のイベントに対する感想を書き込むことを促すのに最適です。イベントのランディングページにウィジェットを含めるだけで、ユーザーはリアルタイムで更新を投稿できます。
前提条件
ユーザーが投稿できるインタラクティブなニュースチャンネルを作成しています。詳細を学ぶ。
ポストウィジェットの設定
スタジオで、コンテンツ > ページ > ポストウィジェットを含めたいページに移動します。
ポストウィジェットを追加します。
ポストウィジェットを設定するためのダイアログが開きます。
オプションで、タイトルを提供します。
タイトルはウィジェットの上に表示され、ユーザーに見えます。
ウィジェットが表示されるデバイスタイプを、対応するアイコンをクリックして選択します。
ウィジェットが表示されるユーザーをアプリ内の可視性で選択します。
OK.をクリックします。
保存をクリックします。
ポストウィジェットの設定が完了しました。保存する前に、さらに設定を指定する必要があります。
ポストウィジェットの設定を指定する
ポストウィジェットの一般設定を構成した後、従業員が投稿できるニュースページまたはチャンネルを選択する必要があります。
ポスト先セクションで、ユーザーがニュースページまたはニュースチャンネルに投稿するかを選択します。
ドロップダウンメニューからニュースページまたはニュースチャンネルを選択します。
モバイルでのウィジェットの位置の下で、ウィジェットを次のように設定するかを選択します：
配置したページの特定のエリアに表示する
ユーザーがナビゲートする場所の上に浮かぶ
オプションで、プロンプトテキストを含めます。このテキストは、ユーザーが書き始める前にテキストボックス内に表示されます。
OK.をクリックします。
保存をクリックします。
ページにポストウィジェットを追加しました。ウィジェットをページ上の希望の位置にドラッグアンドドロップできます。位置やウィジェットの設定はいつでも変更できます。
