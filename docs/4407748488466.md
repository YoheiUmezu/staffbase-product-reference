# チームカレンダーのGoogleカレンダーIDを見つける

**URL**: https://support.staffbase.com/hc/ja/articles/4407748488466
**取得日**: 2026-05-02 18:57:15 JST

## 内容

StaffbaseプラットフォームのページにGoogle環境からチームカレンダーを埋め込むには、カレンダーIDが必要です。
https://calendar.google.com/にアクセスします。
埋め込みたいチームカレンダーの三点リーダーメニューをクリックします。
メニューで、設定をクリックします。
カレンダーを統合の下で、カレンダーIDをコピーします。
カレンダーIDをコピーしたので、Googleカレンダーウィジェットを設定できます。
