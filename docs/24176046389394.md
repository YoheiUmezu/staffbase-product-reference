# ユーザーベースの設定概要

**URL**: https://support.staffbase.com/hc/ja/articles/24176046389394
**取得日**: 2026-05-02 18:42:51 JST

## 内容

この記事の情報は、従業員アプリまたはStaffbaseイントラネットを使用している場合、及びStaffbase Emailが従業員アプリまたはイントラネットと一緒に使用される場合に適用されます。Staffbase Emailをスタンドアロン製品として使用している場合は、Staffbase Emailにおけるユーザーベースの設定概要を参照してください。
まず、Staffbaseプラットフォームを効果的に使用するために、ユーザーベースを適切に設定する必要があります。プラットフォームのユーザーベースは、2種類のユーザーで構成されています：
ユーザー: 組織が使用するStaffbase製品に応じて、以下のいずれかの操作を行うことができる個人です：
イントラネットにアクセスする
アプリにアクセスする
メールを受信する
スタジオユーザー: 自身の役割に基づいてアクションを実行できるスタジオへのアクセスを持つ個人です。たとえば、管理者として、あなたはスタジオユーザーです。
追加のユーザーを追加する前に、ユーザーベースを効果的に作成するために、以下のコンポーネントを同じ順序で設定してください：
プロフィールフィールド
これらは、個人を特定するためのユーザー属性を含むフィールドです。ターゲット使用ケースに必要な追加のプロフィールフィールドを設定してください。
ユーザーの追加
プロフィールフィールドが設定された後、以下のいずれかの方法を使用してユーザーを追加できます：
手動
CSVインポート
SSOオンボーディング
SCIM
ユーザーAPI
ユーザーベースを設定した後、必要に応じてプロフィールフィールドを更新できます。プロフィールフィールドの変更は、以降のユーザー同期に影響を与えることに注意してください。
ユーザーグループ
これらは、ターゲットコミュニケーションと管理を可能にする共有属性を持つユーザーのコレクションです。プロフィールフィールドを定義し、ユーザーデータをインポートした後、プロフィールフィールド属性に基づいてユーザーグループを作成できます。
