# ニュースチャンネル型メディアの概要

**URL**: https://support.staffbase.com/hc/ja/articles/360028940272
**取得日**: 2026-05-02 18:58:32 JST

## 内容

新しいチャンネルを作成する際、チャンネルタイプ メディア を選択すると、チャンネル内で作成する投稿がメディアを取り入れ、画像や動画を前面に出すことができます。
メディアに最適なコンテンツは？
チャンネルタイプ「メディア」では、著者が画像または動画を含む投稿を作成し、その後に短い説明を追加できます。これは、会社の生活を垣間見るための完璧な方法であり、動画チュートリアルシリーズを公開したり、従業員が「今日の一枚」を共有する場所を提供したり、さらには「今月の写真」コンテストを開始するのにも最適です。
画像または動画
投稿には、単一の画像または動画のみを含めることができます。メディア投稿は、動的で短く、最も重要なのは視覚的なコンテンツに関するものです。このチャンネルは、画像や動画のための完璧なスポットライトです。
説明
画像または動画の下に、短い説明を追加できます。説明を使用して、視覚的な内容の背後にあるストーリーを視聴者に伝え、興味を引き起こしましょう。
