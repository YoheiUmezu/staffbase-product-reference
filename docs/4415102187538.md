# Outlookでの連絡先グループの作成

**URL**: https://support.staffbase.com/hc/ja/articles/4415102187538
**取得日**: 2026-05-02 18:55:47 JST

## 内容

同じ受信者グループに頻繁にメールを送信していますか？Outlook 2016でカスタム連絡先グループを作成することで、生産性を最大化し、時間を節約できます！この機能は、ビジネスまたは学生アカウントを持っている場合にのみ機能し、個人のOffice 365アカウントでは利用できません。
Outlookの連絡先グループ機能を使用すると、1つの名前（例："親しい友人"や"サポートチーム"）の下で、メールアドレスのグループに迅速にメールを送信できます。連絡先グループにメールを送信すると、そのグループに指定したすべての受信者に送信されます。これは配布リストとほぼ同じように機能しますが、ローカルのOutlookクライアントを通じて直接設定されます。
設定方法は次のとおりです：
ステップ 1: Outlookを開き、左下のナビゲーションペインで連絡先アイコンをクリックします。
ステップ 2: ホームタブの下にある新しい連絡先グループボタンをクリックします。新しいウィンドウが開きます。
ステップ 3: 連絡先グループタブの下で、メンバーの追加をクリックし、Outlookの連絡先またはアドレス帳から選択します。
ステップ 4: 連絡先グループに追加したい各メール連絡先をダブルクリックし、OKをクリックします。
ステップ 5: グループの名前を入力します。
ステップ 6:連絡先グループリボンの下で保存して閉じるをクリックします。
この新しい連絡先グループに送信するには、通常通りOutlookで新しいメールを作成し、TO/CCまたはBCCフィールドに連絡先グループの名前を入力します。
連絡先グループ名の横にあるプラス（+）記号をクリックすることで、配布リストに誰が含まれているかを確認でき、必要に応じて連絡先グループ自体にメールアドレスを追加または削除できます。
これは、より大きな受信者グループに一斉メールを送信する際のOutlookでの最高のメールショートカットの1つです。ただし、これはローカルのOutlookクライアントにのみ適用され、同じメールサーバー上の他のユーザーには利用できないことに注意してください（つまり、配布リスト）。
