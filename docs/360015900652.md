# ユーザーにスペースの拡張権限を付与する

**URL**: https://support.staffbase.com/hc/ja/articles/360015900652
**取得日**: 2026-05-02 18:59:58 JST

## 内容

スペースに対して拡張権限を持つスペース管理者を1人以上割り当てることができます。このスペース内では、すべてのニュースチャンネル、ページ、プラグインを追加、編集、削除したり、スペースの設定を変更したりできます。
スタジオで、スペースの設定に移動します。
スペース管理者を追加したいスペースに移動します。
スペースを編集をクリックします。
管理者のドロップダウンからユーザーを選択します。
保存をクリックします。
