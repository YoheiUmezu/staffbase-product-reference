# Navigator XX内サービスナウ連携の概要

**URL**: https://support.staffbase.com/hc/ja/articles/34807123041298
**取得日**: 2026-05-02 18:50:41 JST

## 内容

有効化
AI
NavigatorのServiceNow統合は、ServiceNowサービスへの集中化されたエントリーポイントを提供します。検索、チケットの表示、チケットの作成を1つのシンプルな体験にまとめています。従業員は音声またはテキストで、どの言語でも質問でき、以下からインスタントな回答を受け取ることができます：
ServiceNowナレッジベース
ServiceNowカタログ
ServiceNowチケット
あなたのServiceNowの許可に応じて、ServiceNowからインフォメーションを表示できます。また、NavigatorでServiceNowフォームが設定されている場合は、フォームを使ってチケットを作成することもできます。
NavigatorでServiceNow統合を設定するには、次のステップに従ってください：
まだプラットフォームに設定されていない場合は、StaffbaseにServiceNow接続を追加する
スタジオで次のServiceNow接続を有効にする：
ServiceNow ナレッジ検索
ServiceNowチケットの統合
ServiceNowカタログ項目検索
Navigatorの設定にServiceNowチケットフォームを追加する。
イントラネットからServiceNowアカウントにサインインする。
