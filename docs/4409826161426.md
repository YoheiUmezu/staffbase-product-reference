# SharePoint サイトページに News Central を追加する

**URL**: https://support.staffbase.com/hc/ja/articles/4409826161426
**取得日**: 2026-05-02 18:57:13 JST

## 内容

ニュースセントラルのセットアップを構成したら、ニュースセントラル Web パーツを追加して、ニュースソースとレイアウトを構成できます。
ページにニュースセントラルを追加する
前提条件:
Web パーツを追加するために、SharePoint サイトのメンバーまたはそれ以上の権限を持っていること。
ニュースセントラルを SharePoint サイトに構成していること。
ニュースセントラルを構成した SharePoint サイトのページに移動します。
ページの右上にある 編集 をクリックします。
ページの上にマウスをホバーします。
新しい Web パーツを追加するための + アイコンが表示されます。
+ アイコンをクリックします。
利用可能な Web パーツのダイアログが表示されます。
ニュースセントラルを検索して選択します。
ニュースセントラル Web パーツを追加しましたので、これから構成できます。
ニュースセントラルのニュースソースとレイアウトを構成する
前提条件:
ニュースソースとしてチャンネルを選択するには、そのチャンネルのスタジオでの著者または編集者の役割が必要です。
SharePoint サイトで、Web パーツを追加したページに移動します。
編集 アイコンをクリックして、ニュース セントラル ダイアログを開きます。
ニュースセントラル ダイアログが開き、Staffbase にサインインするように促されます。
Staffbase プラットフォームにサインインします。
ニュースソースの下で、ニュースを取得したいニュースページまたはチャンネルを選択します。
メディアおよび更新チャンネルは、カルーセルおよびタイルレイアウトではサポートされていません。
レイアウトの下で、Web パーツのタイトルを表示するかどうかを定義し、タイトルと「すべて表示」のトグルを調整してリンクを提供します。
レイアウトとして、次のいずれかを選択します：
カルーセル
カルーセルレイアウトは、ユーザーにニュースストーリーのスライドショーを表示します。
投稿数のスライダーを使用して、含める投稿の数を定義します。
次の設定を有効または無効にすることで調整します：
自動再生: カルーセル内でニュースを自動的に次々と再生します。
カルーセルナビゲーションドット: カルーセル内のニュース投稿の数をナビゲーションドットのシリーズで表示します。
チャンネル名: 表示されるニュース投稿のチャンネル名を表示します。
ニュース投稿タイトル: ニュース投稿のタイトルを表示します。
ティーザーテキスト: ニュースからのティーザーテキストを表示します。
いいねとコメント: ユーザーが投稿にいいねやコメントを追加できるようにします。
タイル
タイルレイアウトは、ユーザーにニュースをタイル形式で表示します。
投稿数のスライダーを使用して、含める投稿の数を定義します。
次の設定を有効または無効にすることで調整します：
チャンネル名: 表示されるニュース投稿のチャンネル名を表示します。
ニュース投稿タイトル: ニュース投稿のタイトルを表示します。
横並び
横並びレイアウトは、ユーザーにニュースを二列リストで表示します。
投稿数のスライダーを使用して、含める投稿の数を定義します。
次の設定を有効または無効にすることで調整します：
チャンネル名: 表示されるニュース投稿のチャンネル名を表示します。
ニュース投稿タイトル: ニュース投稿のタイトルを表示します。
ティーザーテキスト: ニュースからのティーザーテキストを表示します。
いいねとコメント: ユーザーが投稿にいいねやコメントを追加できるようにします。
リスト
リストレイアウトは、ユーザーにニュースをリスト形式で表示します。
投稿数のスライダーを使用して、含める投稿の数を定義します。
次の設定を有効または無効にすることで調整します：
チャンネル名: 表示されるニュース投稿のチャンネル名を表示します。
ニュース投稿タイトル: ニュース投稿のタイトルを表示します。
ティーザーテキスト: ニュースからのティーザーテキストを表示します。
いいねとコメント: ユーザーが投稿にいいねやコメントを追加できるようにします。
ニュースソースとニュースセントラルのレイアウトを構成しました。
