# Box検索統合の概要

**URL**: https://support.staffbase.com/hc/ja/articles/20496753468946
**取得日**: 2026-05-02 18:43:06 JST

## 内容

ベータ
アクティベーション
StaffbaseとのBox統合により、ユーザーはプラットフォーム全体でコンテンツを見つけるのがこれまで以上に簡単になります。このFront Door Intranet用の統合を使用すると、Staffbaseプラットフォーム内のコンテンツだけでなく、Boxのコンテンツにもアクセスできます。これにより、別のプラットフォームで再度コンテンツを検索する手間が省け、コンテンツ作成者はコンテンツの重複を避けることができます。Boxからの検索結果は、Staffbaseプラットフォームの別の検索タブに表示されます。
ユーザーが検索を利用できるようにするには、設定し、Box接続をStaffbaseに追加する必要があります。
ユーザーにとってのBox統合の見え方
ユーザーがBoxアカウントにサインインすると、Box検索タブでコンテンツを検索できます。詳細を学ぶ。検索タブの結果をクリックすると、ユーザーはBox内の目的のコンテンツに直接移動します。
