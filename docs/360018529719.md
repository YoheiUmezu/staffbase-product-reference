# Appleアプリのプライバシーラベルに関するFAQ

**URL**: https://support.staffbase.com/hc/ja/articles/360018529719
**取得日**: 2026-05-02 18:59:04 JST

## 内容

Appleプライバシーラベルとは何ですか？
AppleはApp Storeにプライバシーラベルを設けており、ユーザーにアプリが使用するデータについての透明性を提供しています。
アプリのプライバシーに関する質問に回答することは義務ですか？回答しないとどうなりますか？
はい、義務です。アプリのプライバシーに関する質問に回答しない場合、新しいバージョンのアプリをユーザーにリリースすることはできません。
ラベルは新しいアプリと既存のアプリの両方に必要であり、組織のプライバシーポリシーへのリンクを求める既存の要件に加わります。
Staffbaseプラットフォームを使用するアプリは一般的にどのようなデータを収集しますか？
通常、Staffbaseプラットフォームを使用するアプリは、ユーザーがアプリを安全に使用できるようにするために必要なデータと、ユーザーに関連するコンテンツを提供するために必要なデータのみを収集します。収集される情報の種類は、アプリの具体的な使用ケースによって異なる場合があります。このため、アプリのデータ処理活動に関するAppleのプライバシーに関する質問に対して、明確な回答を提供することはできません。詳細情報はこちらをご覧ください。
Staffbaseは、保護された健康情報や財務情報を含むがこれに限定されない特定の情報のアプリ内での使用を制限しています。詳細については、当社の利用規約をご参照ください。
なぜStaffbaseはアプリのプライバシー情報を提供できないのですか？
Staffbaseがこの情報を提供できない理由はいくつかあります：
アカウント保持者または管理者のみがApp Store Connectに情報を入力できます。
アプリの所有者であるあなたが、App Storeの関連規約を遵守する責任があります。
あなたはアプリの使用ケースを完全に理解しており、ユーザーがアプリを使用する際にどの情報が収集されるかを知っています。
アプリのプライバシー情報を提供する頻度はどのくらいですか？
アプリのプライバシー情報を提供すると、それは今後のリリースやバグ修正に使用されます。データ使用の慣行が変更されない限り、ラベルを更新する必要はありません。
アプリ内のデータ使用の慣行が変更された場合、アプリのプライバシー質問票の情報を更新し維持する必要があります。データプライバシーラベルを公開した後でも、再度質問に回答し、新しいデータプライバシーラベルを公開することができます。アプリのプライバシーラベルの更新は即座に適用されます。アプリを再構築したり再提出したりする必要はありません。
なぜアプリのプライバシーラベルが見えないのですか？
App Storeでアプリのプライバシーラベルを見るためには、以下が必要です：
iOS 14.4以上
macOS 11.1以上
