# モバイルデバイスのクイックリンクナビゲーションバーに検索を追加する

**URL**: https://support.staffbase.com/hc/ja/articles/360013465420
**取得日**: 2026-05-02 19:00:52 JST

## 内容

ベータ
アクティベーション
システム全体のエディター
検索機能をナビゲーションバーにクイックリンクとして追加することで、モバイルデバイスでのアクセスをさらに向上させることができます。これにより、ユーザーはアプリを開いた瞬間にコンテンツやユーザーを検索するための即時アクセスを得ることができます。
検索機能へのリンクを作成する
スタジオで、コンテンツ > リンクに移動します。
リンクを追加をクリックします。
新しいリンクを追加するためのダイアログが開きます。
リンクにタイトルを付けます。このタイトルは、アプリ内のエンドユーザーには表示されません。
ターゲットURLの下に、アプリのURLを挿入し、その後に/search-mobileを追加します。
リンクは次のようになります:
ドロップダウンメニューからリンクの可視性を選択します。
リンクを追加をクリックします。
モバイルのクイックリンクナビゲーションバーの検索を設定する
メニューに移動し、作成したリンクのためにクイックリンクを設定する手順に従います。
クイックリンク設定で、デバイスの可視性をモバイルのみに設定します。
設定が成功したら、リンク > 以前に作成した検索リンクに移動します。
リンクを公開をクリックします。
検索がモバイルデバイスのクイックリンクナビゲーションバーに表示されるようになります。
