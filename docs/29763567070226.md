# 代替テキストでアクセシビリティを向上させる

**URL**: https://support.staffbase.com/hc/ja/articles/29763567070226
**取得日**: 2026-05-02 19:00:06 JST

## 内容

コミュニケーション
フロントライン
労力：中
画像に代替テキスト（altテキスト）を追加することは、すべての人にコンテンツを利用しやすくするシンプルで効果的な方法です。altテキストは、スクリーンリーダーや他の支援技術を利用する従業員、または接続が制限されて画像を読み込めない従業員にも、コミュニケーションの内容が正しく伝わるようにします。明確で説明的なaltテキストを書くことで、WCAGなどのアクセシビリティ基準に準拠するだけでなく、すべての従業員が平等にコンテンツに参加できるインクルーシブな文化を築くことができます。このユースケースでは、altテキストを通じて社内コミュニケーションの明確さ、一貫性、信頼性を向上させる方法を説明します。
この記事の内容：
Key Facts | Steps | Measure Success | Best Practices | How Alt Text Looks For Users
Key Facts
Problem Statement: スクリーンリーダーを利用する従業員や技術的な問題に直面している従業員は、画像内に埋め込まれた重要なコンテンツを見逃す可能性があり、情報へのアクセスにギャップが生じ、インクルーシブ性が低下します。
Business Objective: アクセシビリティガイドラインへの準拠を強化し、視覚コンテンツへの平等なアクセスを確保することで、従業員全体の信頼を高めること。
Staffbase Features:
メールデザイナー：すべての画像にaltテキストを追加して、メールコンテンツをアクセシブルにします。
ページ：社内ドキュメントや更新情報にアクセシブルな画像を組み込みます。
ニュース：説明的なaltテキストを埋め込むことで視覚的なストーリーテリングのインクルーシブ性を高め、画像が読み込めない場合には代替表示されるため、インターネット接続が弱いユーザーにも有効です。
Audience: スクリーンリーダーを利用する従業員や、デバイスやインターネットに制限のある従業員。
Outcome: 視覚能力やデバイス、インターネットの制限に関わらず、すべての従業員が完全に理解できるアクセシブルなコミュニケーション。
Steps
このステップバイステップガイドを使って、Staffbaseでこのユースケースを設定できます。
ニュースまたはページの画像にAltテキストを追加する
編集者
Studioでコンテンツ > altテキストを追加したいページまたは投稿を開きます。
altテキストを追加したい画像をクリックします。
ALTボタンをクリックします。
Altテキストを入力します。
適用するをクリックします。
保存するをクリックします。
ニュースやページでの画像へのaltテキスト追加について詳しくはこちら。
メールの画像にAltテキストを追加する
編集者
Studioでメール > 送信またはスケジュールしたいメールの下書きを開きます。
メールを開きます。
画像をクリックし、右側のペインで画像設定に移動します。
代替テキスト欄にaltテキストを入力します。
保存するをクリックします。
メール画像へのaltテキスト追加について詳しくはこちら。
セットアップと継続的なガバナンス
定期的にコンテンツを監査し、画像に説明的なaltテキストが含まれていることを確認します。
監査は、すべてのコミュニケーションチャンネルで一貫性と品質を維持するのに役立ちます。
コンテンツ作成者に効果的なaltテキストの書き方をトレーニングします。
トレーニングはaltテキストの重要性を強調し、すべての編集者が同じ基準とガイドラインに従うことを保証します。
altテキストをWCAGアクセシビリティ基準に合わせます。
編集者がアクセシビリティ要件を理解することで、コミュニケーションのコンプライアンスを維持できます。
Measure Success
このユースケースの成果を追跡する方法です。
従業員からのフィードバック収集
アンケートを作成し、altテキストを利用する従業員からの意見を集め、編集チーム内で共有します。
支援技術を利用する従業員とのライブセッションからフィードバックを収集し、繰り返し発生する課題を特定して、今後のコンテンツ作成者向けトレーニングに活かします。
Best Practices
Staffbaseは、ユーザーに最適な体験を提供するためのいくつかの推奨事項を用意しています。
説明的にする
画像の視覚的な詳細だけでなく、文脈での目的に焦点を当てましょう。
1～2文で意図が伝わるよう簡潔にまとめましょう。
画像内のテキスト（引用、見出しなど）も含めましょう。
簡潔かつ関連性を持たせる
「画像」や「写真」などの表現は省略しましょう。
色については、理解を助ける場合のみ言及しましょう。
スクリーンリーダーで読みやすいように明確な句読点を使いましょう。
プラットフォーム全体で一貫性を保つ
メール、ニュース、ページのすべての画像にaltテキストが含まれていることを確認しましょう。
組織全体のアクセシビリティガイドラインを適用し、一貫性を維持しましょう。
スクリーンリーダーでテストし、明確さと使いやすさを確認しましょう。
