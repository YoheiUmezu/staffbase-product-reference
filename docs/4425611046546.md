# プラットフォーム用のビデオ変換

**URL**: https://support.staffbase.com/hc/ja/articles/4425611046546
**取得日**: 2026-05-02 18:57:03 JST

## 内容

Staffbaseは、ビデオが迅速に再生され、良好な品質を保つために、いくつかのガイドラインに従うことを推奨しています。一般的なルールとして、アップロードや再生時の長い待機時間を避けるために、ファイルサイズをできるだけ小さく保つようにしてください。ビデオの最大ファイルサイズは4GBです。
ビデオストリーミングの体験は、インターネット接続の強度によって異なる場合があります。良好なビデオ品質と合理的なサイズのバランスを考慮して、720p（1280 x 720px）解像度のファイルを使用することをお勧めします。データ使用量が増加し、ビデオストリーミングの体験に影響を与える可能性があるため、より高解像度のファイルも使用できます。Staffbaseがサポートするメディアフォーマットについて詳しく学ぶ。
最良の結果を得るために、以下の設定を使用して直接ビデオを録画するか、HandBrakeなどのビデオコンバータを使用して変換してください。
ビデオ設定
フォーマット: mp4, ウェブ最適化
ビデオエンコーダ: H.264 (x264)
フレームレート: 30 FPS PFR
オーディオ: AAC (CoreAudio), 128 kbps, ステレオ
HandBrakeを使用したビデオの変換
HandBrakeはサードパーティのサービスであることに注意してください。Staffbaseは、このサービスの使用結果について責任を負いません。
前提条件
HandBrakeソフトウェアがインストールされていること。
手順
HandBrakeを開きます。
ソースを開くをクリックし、変換したいビデオファイルを選択します。
概要タブに移動し、ウェブ最適化チェックボックスを選択します。
プリセット > ウェブをクリックし、ドロップダウンメニューからソーシャル 50 MB 5 分 720p30を選択します。
名前を付けて保存の下に、ビデオファイルの名前を入力します。
保存先の下で、ビデオファイルを保存するフォルダーを選択します。
開始をクリックして、ビデオを変換します。
結果
ビデオは変換され、選択したフォルダーに保存されました。これで、プラットフォームにアップロードできます。
