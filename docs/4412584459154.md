# Outlook Classic 3.0 アドイン FAQ

**URL**: https://support.staffbase.com/hc/ja/articles/4412584459154
**取得日**: 2026-05-02 18:47:18 JST

## 内容

Outlook Classic 3.0 は、私たちの人気のある Outlook 用アドイン（非 365 バージョン）の完全に再構築されたバージョンです。このアドインは、Windows デスクトップ上の Outlook のみをサポートし、.exe ファイルを介してローカルにインストールされます。
このリリースは、私たちの最大の顧客の大多数が使用している Outlook のバージョンであるため、重要です。
バージョン 3.0 の新機能は何ですか？
パフォーマンスの向上
Web アプリとより一貫性のあるメールデザイナーインターフェース
将来の更新が少なくて済む単一のインストール/アップグレード
これは誰のためのものですか？
現在、私たちの Outlook アドイン（1.0 または 2.0）を使用しているすべての顧客は、Classic 3.0 への更新から恩恵を受けます。これは、Windows デスクトップ上の非 365 用 Outlook の新しく改善されたアドインのバージョンです。
何が変わりますか？
Outlook Classic 3.0 は、Employee Email サイドバーとリボンボタンを、リボン上の単一の Create with Staffbase ボタンに統合します。これは、Employee Email への単一のエントリーポイントとして設計されており、Web アプリを使用したことがある人には馴染みのある体験となります。
リボンからの単一のエントリーポイントに移行することで、Outlook での Web アプリの使用体験に合わせることができます。
ユーザーは現在の Outlook アドイン（つまり Classic 2.0）をアンインストールする必要がありますか？
いいえ！従来、私たちは顧客に新しいバージョンのアドインを更新およびインストールする際に「古い」バージョンのアドインをアンインストールすることを要求していましたが、このバージョンは単にインストールされ、2.0 を置き換えます。
同様に、更新後、ユーザーは 3.0 をアンインストールすることなく 2.0 を再インストールできます。
IT チームが考慮すべき 2.0 から 3.0 への変更はありますか？
リリース時、3.0 では Outlook からの配布リスト受信者カウントがデフォルトで無効になっています。これは、Outlook の配布リストにいる人数でメールレポートを事前に入力する機能です。
Email Designer 2.0 および 3.0 のドラフトは Outlook Classic 3 で利用可能ですか？
はい！
すべての設定は 2.0 と同じですか？
こちらをクリックして、このアドインの現在の設定メニューの概要をご覧ください。
