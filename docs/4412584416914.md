# 従業員メールの返信アドレスの変更

**URL**: https://support.staffbase.com/hc/ja/articles/4412584416914
**取得日**: 2026-05-02 18:54:09 JST

## 内容

従業員メール ウェブアプリまたはスタジオ統合を使用してメールを送信する際に、追跡されたメールの返信アドレスを変更することで、コミュニケーションへの返信がどこに送信されるかを正確に定義できます。
この変更は、以下の場合に役立ちます：
組織の幹部など、個人を代表してコミュニケーションを送信している場合で、彼らの受信トレイに多くの返信が届くことを望まない。
別の部門のためにコミュニケーションを送信しており、彼らが返信にアクセスし、管理したい場合。
メールドラフトの返信アドレスの変更
ドラフトを送信またはスケジュールする準備ができたら、送信詳細ページにアクセスします。
「その他の送信オプション」セクションで、返信メールの変更設定のトグルボタンを有効にします。
これで、このコミュニケーションの新しい返信メールを入力できるようになります。
メッセージを送信すると、更新された返信メールが有効になります。
組織のメールサーバーの設定によっては、一部の自動返信（不在通知など）がカスタムメールではなく、元のメールに送信される場合があります。
