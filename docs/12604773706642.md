# 従業員メールにおけるエイリアスの詳細とメトリクスの表示

**URL**: https://support.staffbase.com/hc/ja/articles/12604773706642
**取得日**: 2026-05-02 18:49:23 JST

## 内容

管理者
エイリアスから送信されたメールの表示
従業員メールでは、Webアプリ/スタジオがエイリアスを使用して送信されたメールを含む、詳細なステータス情報を持つメールの概要を提供します。表示される情報は、それぞれの役割に依存します：
管理者役割以上はすべての情報にアクセスできます。
ユーザーは、自分と共有されたエイリアスの情報を見ることができます。
詳細については、従業員メールにおける送信メールの詳細の表示.
エイリアスに関連するメトリクスの表示
エイリアスとプライマリメールアドレスの区別により、追跡されたメールに関する柔軟で包括的な洞察が得られます。
プライマリアカウントと共有されたエイリアスの間で、概要ページで簡単に切り替えることができます。
デフォルトレポートには、ログインユーザーのメールと共有エイリアスから送信されたメールのすべての追跡メールが含まれます。
レポートページでは、表示するメールをフィルタリングし、特定のユーザーからのデータを生成できます。
エイリアスの表示メールがユーザーの表示メール（アクティブまたは非アクティブ）と一致する場合、両方の送信メールデータがレポートで一緒に生成されます。
詳細については、従業員メールにおけるメトリクスの読み取りと解釈を参照してください。
