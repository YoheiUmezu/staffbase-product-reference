# ニュース投稿の高度なレビューページの使用

**URL**: https://support.staffbase.com/hc/ja/articles/25488775351442
**取得日**: 2026-05-02 19:02:56 JST

## 内容

アクティベーション
従業員を引き込む魅力的なニュースを作成するには、時間、労力、チームワークが必要です。ニュース投稿の高度なコンテンツレビュー機能を使用すると、アイデアを簡単に議論し、フィードバックを求めたり、承認を得たりできます。これはすべてのニュースチャンネルタイプで利用可能です。
投稿作成者は、ニュース投稿を共有することで、コラボレーターをコンテンツに招待できます。招待されたコラボレーターは、メールを通じてアクセスを受け取り、その後、ニュース投稿のレビュータブでレビューページにコメントを残すことができます。
ニュース投稿のレビューページでは、次のことができます：
コメントを残し、他のコメントを見る
コメントを編集する
コメントを削除する
ユーザーを@の後にユーザー名を入力してメンションする
ユーザーにタスクを割り当てる
ユーザーに承認をリクエストする
利用可能な場合は複数の言語間で切り替える
多言語ニュース投稿のレビューページでの言語切り替え
レビューページでは、作成された任意の言語でニュース投稿を表示できます。
メールの受信トレイで、ニュース投稿に関するメールを見つけて、開くをクリックします。
ニュース投稿が新しいタブで開きます。
ニュース投稿の上にある言語タブに移動します。
言語タブをクリックします。
レビューページに選択した言語でニュース投稿が表示されます。
ニュース投稿のレビューページで言語を切り替えました。いつでも別の言語に切り替えることができます。
レビューページでのニュース投稿へのコメント
レビューページでは、新しいコメントを追加したり、既存のコメントに返信したりして、ニュース投稿にフィードバックを残すことができます。これにより、明確で構造化されたコメント履歴が確保されます。

ニュース投稿レビューページでのコメントの編集
レビューページでは、スペルミスを修正したり、フィードバックを更新したり、考えを洗練させたりするために、提出したコメントを編集できます。これにより、クリーンで一貫したレビューのスレッドが確保されます。
返信を受けていない場合にのみ、自分のコメントを編集できます。コメントに返信があった場合、そのコメントは編集できなくなります。スレッド内では、最も最近追加されたコメントのみが編集可能です。

コメントでの@メンションの使用
@メンション機能を使用すると、ニュース投稿のレビュータブで同僚をタグ付けできます。メンションされたユーザーはメールを受け取り、レビュープロセスをできるだけ迅速かつ効率的にします。
前提条件
メンションされたユーザーが、個別またはユーザーグループの一部として、スペースの可視性フィールド選択に含まれていること。
メンションされたユーザーが、個別またはユーザーグループの一部として、チャンネルのアプリ内可視性フィールド選択に含まれていること。
メンションされたユーザーが、共有機能を使用してニュース投稿に招待されていること。
可視性要件が満たされていない場合、@の後に名前を入力してもドロップダウンメニューにユーザーは表示されません。これには、エディター、管理エディター、管理者などのシステム全体の権限を持つユーザーが含まれます。可視性要件が満たされているが、ユーザーがニュース投稿に招待されていない場合、@メンションに関するメールを受け取ります。しかし、メール内のリンクをクリックするとエラーページが表示されます。

コメントでのタスクの割り当て
レビューページからコラボレーターにタスクを割り当てることで、ニュース投稿作成中の最良のコラボレーションを確保します。
未完了のタスクは、投稿の公開を妨げません。
