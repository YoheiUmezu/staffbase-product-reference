# ページまたはニュース投稿の画像をトリミングする

**URL**: https://support.staffbase.com/hc/ja/articles/360013545199
**取得日**: 2026-05-02 19:01:29 JST

## 内容

エディター
トリミング機能を使用すると、画像の未選択の周囲を削除できます。トリミングツールが付属しており、位置をドラッグしてサイズを変更できます。
この機能は以下に利用可能です：
ニュース投稿のメイン画像
ニュースまたはページエディターに追加された画像
画像の編集ダイアログを開く
記事またはメディア投稿のメイン画像
スタジオで、メイン画像を編集する投稿に移動します。
編集をクリックします。
編集ページが開きます。
画像の詳細を編集をクリックします。
画像の詳細を編集ダイアログが開きます。
画像をトリミングする際のオプションについて詳しく学ぶ。
ニュース投稿またはページエディターの画像
スタジオで、編集したいファイルを追加したページまたは投稿に移動します。
エディターで、編集したいファイルをクリックします。
編集および削除アイコンが表示されます。
鉛筆アイコンをクリックして編集します。
画像をトリミングするためのダイアログが開きます。
画像をトリミングする際のオプションについて詳しく学ぶ。
画像をトリミングする
画像の編集ダイアログで、トリミングツールをドラッグして、最終画像に含めたい画像の部分に位置を合わせます。
トリミングツールはデフォルトで固定アスペクト比に設定されています。この固定アスペクト比は、画像の推奨アスペクト比になっています。
オプションで、トグル固定アスペクト比を無効にして、トリミングツールの固定アスペクト比を変更します。
別のアスペクト比を選択すると、右側と左側にぼやけた背景が表示されます。
保存をクリックします。
ユーザーがエディターでトリミングされた画像を表示する方法
固定アスペクト比トグルが有効な場合のトリミングされた画像。
例えば、200 x 50px
固定アスペクト比トグルが無効でカスタムアスペクト比が選択された場合のトリミングされた画像。
例えば、50 x 100px
保存時、横長アスペクト比の結果画像がそのまま表示されます。
保存時、縦長アスペクト比の結果画像がぼやけた背景と共に表示されますが、フルスクリーンモードで開くと表示されません。
