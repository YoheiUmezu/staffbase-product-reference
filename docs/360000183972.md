# スタッフベースへのアプリ詳細の送信

**URL**: https://support.staffbase.com/hc/ja/articles/360000183972
**取得日**: 2026-05-02 18:56:13 JST

## 内容

証明書とプロビジョニングプロファイルの設定が完了したら、アプリの準備を進める準備が整いました。ファイルをメールで送信し、メール本文に情報を追加してください。
Apple Developerアカウントから以下のファイルが必要です：
以前にダウンロードして.p12に変換したiOS配布証明書ファイル
以前にダウンロードして.p12に変換したiOSプロダクションSSL証明書ファイル
.mobileprovisionとしてダウンロードしたApp StoreタイプのiOSプロビジョニングプロファイルファイル
.mobileprovisionとしてダウンロードしたAd HocタイプのiOSプロビジョニングプロファイルファイル
App Store Connectのアプリプロファイルから以下の情報が必要です：
バンドルID（例：com.yourcompany.appname）
Apple ID（例：1234567890）
