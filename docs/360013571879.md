# 多言語グループ名の設定

**URL**: https://support.staffbase.com/hc/ja/articles/360013571879
**取得日**: 2026-05-02 18:57:23 JST

## 内容

グループ管理者
オープングループおよび必須オープングループの場合、グループ名はグループ選択ツールでユーザーに表示されます。組織のすべてのコンテンツ言語でグループ名を追加することで、ユーザーは自分のニーズや興味に合ったコンテンツを見つけることができます。
複数の言語でグループ名を作成する機能は、アプリがそのように設定されている場合のみ利用可能です。このオプションがない場合は、アプリ管理者に連絡してください。
スタジオで、ユーザー設定 > ユーザーグループに移動します。
設定を編集したいグループをクリックします。
グループを編集をクリックして設定を開きます。
グループ設定のダイアログが開きます。
言語のドロップダウンメニューを開きます。
追加をクリックします。
追加したい言語を選択します。
ドロップダウンメニューには、プラットフォームで有効になっているすべてのコンテンツ言語が含まれています。追加の言語が必要な場合は、管理者に有効にするよう依頼してください。
グループの場合、言語設定はグループ選択ツールでのグループ名にのみ影響します。
OKをクリックします。
新しい言語でグループの名前を入力します。
グループ名はグループ選択ツールに表示されます。
保存をクリックします。
