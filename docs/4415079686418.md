# Outlook 365の従業員メールアドインでのエイリアスの使用

**URL**: https://support.staffbase.com/hc/ja/articles/4415079686418
**取得日**: 2026-05-02 18:59:41 JST

## 内容

他の誰かの代理でメッセージを送信し、それが直接その人から来ているように見せたい場合は、エイリアスを使用できます。エイリアスを使用すると、特定のコミュニケーションのために異なるメールアドレスからメールを送信でき、メッセージが代表している人から発信されている印象を与えます。
前提条件
あなたのExchangeアカウントは、エイリアスメールアドレスに対して送信者として送信の権限を持っています。
Microsoft Exchangeの権限レベルについて詳しく学ぶ。
Outlookでエイリアスから未追跡のメールを送信できるかテストする。
従業員メールユーザーアカウントがエイリアスメールアドレスのために作成されています。
最新のアドインのためのエイリアスの追加
メールデザイナーのドラフトの送信者フィールドで、エイリアス選択のドロップダウンを開きます。
新しいエイリアスを追加をクリックします。
アカウントを追加をクリックします。
エイリアスとして使用したいメールアドレスを入力します。
このアドレスから送信するためのITまたは管理者の承認がある場合、保存に成功します。
エイリアスからの送信
お好みのプラットフォームでOutlook 365のStaffbaseメールサイドバーを開き、認証します。
ドラフトセクション（2番目のタブ）に移動します。
エイリアスで送信したいドラフトにカーソルを合わせます。
送信方法を選択します。
ドラフトを送信またはドラフトをスケジュールウィンドウ内で、送信者ドロップダウンをクリックしてメールを選択します。
新しいエイリアスを追加をクリックします。
アカウントを追加をクリックします。
エイリアスとして使用するメールアドレスを入力し、保存をクリックします。
認証されたユーザーとして、追加したメールアドレスからメールを受信します。このメールアドレスは、別のアカウントとして送信ウィンドウで利用可能になります。
このウィンドウを閉じると、追加したエイリアスが送信者ドロップダウンで利用可能になります。
認証されたユーザーは、今やこのエイリアスでドラフトを送信できます。
O365統合メカニズムは、Microsoft Graph APIを通じてエイリアスを確認するためにメールを送信します。ユーザーがGraph APIを通じてメールを送信できる場合、O365アドイン内でエイリアス機能を使用できます。
