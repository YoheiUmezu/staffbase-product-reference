# 組織全体でトピックの専門家を見つける

**URL**: https://support.staffbase.com/hc/ja/articles/29739502049042
**取得日**: 2026-05-02 18:59:24 JST

## 内容

従業員は、特定の知識やスキルを持つ同僚をすばやく特定する方法を必要としています。特に大規模または分散型の組織では重要です。このユースケースでは、スタッフベース プラットフォームを活用し、ユーザープロフィールにカスタムフィールドを追加してスキルベースの検索を有効にすることで、従業員が社内の専門家を簡単に見つけられるようにします。その結果、検索可能な人材プールが生まれ、部門間の壁をなくし、問題解決力を高め、全社的なコラボレーションを促進します。
この記事の内容:
Key Facts | Steps | Measure Success | Best Practices | How Finding Topic Experts Looks for Users
Key Facts
Problem Statement: 従業員は、同僚のスキルが見えにくいため、メールのやり取りや外部コンサルタントに頼って専門知識を探すのに時間を費やすことがよくあります。
Business Objective: 社内の専門知識を検索可能にすることで、時間を節約し、コストを削減し、チーム間のコラボレーションを促進します。
Staffbase Features:
プロフィール フィールド: スキル、資格、言語などの構造化された専門知識データを収集します。
検索する: 専門分野フィールドをインデックス化し、従業員が特定の用語で同僚を見つけられるようにします。
従業員ディレクトリ: スキル、部門、勤務地での絞り込みや閲覧が可能です。
Audience: 適切な専門知識を持つ同僚をすばやく見つけたい従業員。
Outcome: 従業員は専門知識を自分で探せるようになります。これにより、問題解決が早くなり、イノベーションが促進され、社内のつながりが強化されます。
Steps
このステップバイステップガイドを使って、スタッフベースでこのユースケースを設定できます。
新しいプロフィール フィールドを手動で追加する
管理者
Studio で、設定 > プロフィール フィールドに移動します。
フィールドを追加するをクリックします。
フィールドのタイトルを入力します。例：スキル。
入力タイプで、デフォルトのテキストを選択したままにします。
次のトグルを有効にします。
-ユーザープロフィールに表示
-従業員ディレクトリと検索結果に表示
-検索で利用可能にする
-ユーザー編集を有効
保存するをクリックします。
プロフィール フィールドや、SCIM、CSV、ユーザー APIによるプロフィール フィールドの自動作成について詳しく学べます。
フィールドを入力する
すべての役割
アプリまたはイントラネットで、プロフィールメニュー > 自分のプロフィールに移動します。
プロフィールを編集するをクリックします。
該当するスキル関連のフィールドに入力します。
変更を保存するをクリックします。
プロフィールの編集について詳しく学べます。
スキルを検索する
すべての役割
アプリまたはイントラネットで、グローバル検索に移動します。
探しているスキルを入力します。
検索結果ページで、人物に移動し、プロフィールにスキルが記載されている同僚を確認します。
名前をクリックしてプロフィールを開きます。
Measure Success
このユースケースの成功を追跡するさまざまな方法があります。
プロフィールの完成度
ユーザーデータをエクスポートして、スキルフィールドを入力した従業員の割合を計算します。
高い完成率は、機能の利用が進んでいることを示します。
検索の利用状況
アナリティクス ダッシュボードの検索タブを使って、従業員がスキルキーワードをどのくらい検索しているかを確認します。
検索でよく使われるキーワードを特定し、スキルフィールドの追加や改善に役立てます。
つながりとフィードバック
部門を超えた連絡の増加を確認します。
アンケートを作成することで、従業員の体験に関するフィードバックを収集します。
Best Practices
スタッフベースは、ユーザーに最適な体験を提供するためのいくつかの推奨事項を用意しています。
明確で一貫性のあるフィールドを定義する
事前定義されたスキルリストを使用し、フリーテキストフィールドの入力を標準化して一貫性を促します。
検索データを定期的に確認し、キーワードを改善してユーザーが一貫した用語を使うようにします。
プロフィールを完全かつ最新に保つことを促す
従業員がプロフィールを入力・更新する動機付けとしてインセンティブを提供します。
定期的にリマインダーを送信し、プロフィール情報が最新であることを確認します。
知識共有の文化を促進する
成功事例やリーダーシップからのメッセージを共有し、専門家発見を企業文化に組み込みます。
従業員ディレクトリやQ&Aフォーラム、イントラネットブログなどの知識ツールを活用し、継続的な知識共有と専門家とのつながりを支援します。
