# Staffbaseメールのボタンスタイルのカスタマイズ

**URL**: https://support.staffbase.com/hc/ja/articles/24344044853266
**取得日**: 2026-05-02 18:57:59 JST

## 内容

管理者
ボタンの色とスタイル、テキストの色を設定することで、メール内のボタンの外観をカスタマイズできます。
スタジオで、設定 > メールに移動します。
メールブランディングページが開きます。
ボタンの下で、テキストカラーのドロップダウンをクリックします。
ドロップダウンから、ブランドカラーを選択するか、カスタムカラーを設定します。
背景色のドロップダウンから、次のスタイルのいずれかを選択します：
太字
斜体
下線
なし
保存をクリックします。
ボタンスタイルをカスタマイズしました。いつでも変更できます。
