# モバイルデバイスのナビゲーションバーにクイックリンクを追加する

**URL**: https://support.staffbase.com/hc/ja/articles/15396749708818
**取得日**: 2026-05-02 19:00:17 JST

## 内容

スタジオで、メニューに移動します。
ナビゲーションバーに追加したいメニュー項目に移動します。
モバイルメニューで表示が有効になっている第一階層のメニュー項目のみがクイックリンクとして追加できます。
任意のメニュー項目をクイックリンクナビゲーションバーに追加できます。これには特定のページも含まれます。この機能を有効にする前に、第一階層以外のメニュー項目がすでにクイックリンクとして設定されていないことを確認してください。有効にすると、これらの項目は自動的にモバイルナビゲーションバーのクイックリンクとして表示されます。詳細については、Staffbaseサポートまたはカスタマーサクセスマネージャーにお問い合わせください。
ギアアイコンをクリックします。
メニュー項目を編集するためのダイアログが開きます。
アイコンの下で、このリンクのために表示したいアイコンを選択します。
クイックリンクのトグルを有効にします。
クイックリンクのトグルを有効にすると、クイックリンクバーの追加設定のための新しいフィールドが表示されます。
短い名前の下で、クイックリンクの名前を提供します。
短い名前はクイックリンクのみに使用されます。この名前は、モバイルクイックリンクバーのエンドユーザーに表示されます。短い名前を追加しない場合、アイコンのみが表示されます。
オプションで、位置の下に、クイックリンクを並べる順序を示す番号を提供します。
位置番号は、コンテンツの表示設定を考慮した相対的な位置です。したがって、クイックリンクナビゲーションバーは、コンテンツの表示設定と位置番号に応じて、ユーザーごとに異なる場合があります。クイックリンクナビゲーションバーには最大5つのクイックリンクが保持されます。
OKをクリックします。
保存をクリックします。
