# アプリまたはイントラネットからのGoogleアカウントへのサインイン

**URL**: https://support.staffbase.com/hc/ja/articles/8267095468690
**取得日**: 2026-05-02 18:43:48 JST

## 内容

ベータ
アクティベーション
アプリまたはイントラネットからGoogle Workspaceアカウントにサインインして、Google Driveのコンテンツを検索およびアクセスできます。
iOS
モバイルアプリで、右上のメニューを開きます。
歯車アイコンをタップして設定を開きます。
プロファイルメニューが表示されます。
設定の下で、アカウントをタップします。
接続されたアカウントの下で、Googleのサインインをタップします。
Googleのサインイン画面にリダイレクトされます。
Googleのサインイン画面で、Googleアカウントを選択するか、別のアカウントを選択してサインイン情報を入力します。
アカウントにサインインします。
許可をタップして、アプリがGoogleアカウントに接続するための権限を付与します。
アプリからGoogleアカウントにサインインしました。
Android
モバイルアプリで、画面の右上隅にある三つの点をタップしてプロファイルメニューを開きます。
設定の下で、アカウントをタップします。
接続されたアカウントの下で、Googleのサインインをタップします。
Googleのサインイン画面にリダイレクトされます。
Googleのサインイン画面で、Googleアカウントを選択するか、別のアカウントを選択してサインイン情報を入力します。
アカウントにサインインします。
タップして許可を選択し、アプリがGoogleアカウントに接続するための権限を付与します。
アプリからGoogleアカウントにサインインしました。
Webアプリ
イントラネットで、プロファイルメニューに移動します。
プロファイルメニューが開きます。
設定 > アカウントに移動します。
接続されたアカウントの下で、Googleのサインインをクリックします。
Googleのサインイン画面にリダイレクトされます。
Googleのサインイン画面で、Googleアカウントを選択するか、別のアカウントを選択してサインイン情報を入力しますをクリックします。
サインインをクリックします。
許可をクリックして、アプリがGoogleアカウントに接続するための権限を付与します。
イントラネットからGoogleアカウントにサインインしました。
イントラネットからGoogleアカウントをサインアウトするには、同じセクションのサインアウトをクリックします。
また、従業員アプリのGoogle Drive検索タブまたはイントラネットの検索機能から直接Googleアカウントにサインインすることもできます。
