# Staffbaseメールでテストメールを送信する

**URL**: https://support.staffbase.com/hc/ja/articles/24491578563474
**取得日**: 2026-05-02 18:59:32 JST

## 内容

このテストメールを送信機能を使用すると、同僚とドラフトを共有し、迅速かつ簡単にレビューを行うことができます。受信者には、メールの上部に黒いバナーとして表示されるプライベートノートを含めることができます。また、送信者の名前も表示されます。レビュアーが返信をクリックすると、その応答は送信したエディターに送信され、フィードバックを収集し、レビューのプロセスを効率化するシームレスな方法となります。
メールを設定し、作成する。
テストメールを送信をクリックします。
テストメールを送信ダイアログが開きます。
受信者のドロップダウンメニューからレビュアーを選択します。
オプションで、テストメールのプライベートノートフィールドに追加情報を記入します。
送信をクリックします。
確認ダイアログが開きます。
閉じるをクリックします。
受信者におけるテストメールの見え方
あなたのテストメールは、テストメールの受信者の受信トレイに、件名が[Test]で始まり、その後にプライベートノートが続く形で表示されます。
メールの上部には、あなたの名前とプライベートノートが表示された黒いバナーが見えます。レビュアーが返信をクリックすると、その応答はあなたのメールボックスに送信されます。
