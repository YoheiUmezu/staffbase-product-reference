# モバイルデバイスのクイックリンクナビゲーションバーにランチパッドを追加する

**URL**: https://support.staffbase.com/hc/ja/articles/22497976435218
**取得日**: 2026-05-02 19:01:03 JST

## 内容

アクティベーション
システム全体のエディター
ランチパッドをナビゲーションバーにクイックリンクとして追加することで、モバイルデバイスでのアクセスをさらに向上させることができます。これにより、ユーザーはアプリを開いた瞬間にランチパッドに即座にアクセスできるようになります。
ランチパッドへのリンクを作成する
スタジオで、コンテンツ > リンクに移動します。
リンクを追加をクリックします。
新しいリンクを追加するためのダイアログが開きます。
リンクにタイトルを付けます。このタイトルは、アプリ内のエンドユーザーには表示されません。
ターゲットURLにアプリのURLを挿入し、その後に/launchpadを追加します。
ドロップダウンメニューからリンクの可視性を選択します。
リンクを追加をクリックします。
モバイル用クイックリンクナビゲーションバーのランチパッドを設定する
メニューに移動し、作成したリンクのためにクイックリンクを設定する手順に従います。
クイックリンク設定で、デバイスの可視性をモバイルのみに設定します。
設定が成功したら、リンク > 以前に作成したランチパッドリンクに移動します。
リンクを公開をクリックします。
ランチパッドがモバイルデバイスのクイックリンクナビゲーションバーに表示されるようになります。
