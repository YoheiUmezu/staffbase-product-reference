# スタッフベースメールにおけるリアルタイムコラボレーションの概要

ベータ版
スタッフベースメールのリアルタイムコラボレーションにより、複数のチームメンバーが同時に同じメールの下書きを編集できます。これにより、投稿者はフィードバックを待ったり、交代で変更を加えたりすることなく、メールに共同で取り組むことができます。特に、スピードと明確さが重要な緊急のコミュニケーションに役立ちます。
同時編集をサポートすることで、スタッフベースメールはバージョンの競合や上書きの問題を回避するのに役立ちます。チームは会議やメッセージスレッドを必要とせずに、ブレインストーミング、レビュー、コンテンツの最終化を共同で行うことができます。この機能は効率を向上させ、やり取りを減らし、大規模なコミュニケーションチーム間でのシームレスなコラボレーションをサポートするように設計されています。リアルタイムコラボレーションはテキストエディターモードでも利用可能です。
下書きのコラボレーション
他の人が同時に編集している下書きやテンプレートを開くと、その人のプロフィール写真やアバターが下書きの上部に表示されます。プロフィール写真やアバターにカーソルを合わせると、その人の名前と現在作業している言語バージョンがツールチップに表示されます。
ブロックが編集されている間は、その作業をしている人だけが変更を確認できます。ブロックがロック解除されると、更新されたコンテンツが全員に表示されます。
各下書きには複数の言語バージョンがあり、言語タブで確認できます。リアルタイムコラボレーションとブロックロックは各言語に個別に適用されるため、ユーザーは異なる言語で同時に作業しても競合しません。メールの下書きを保存するユーザーは、現在作業中の全員の変更を保存します。
ブロックロック
レイアウトやコンテンツブロックは一度に一人しか編集できません。誰かがクリックするとブロックがロックされ、他の人は変更を加えることができません。現在ブロックを編集している人だけがロック中のブロックを確認でき、他の人はロックされたブロックに透明なオーバーレイが表示されます。ブロックにカーソルを合わせると、編集しているユーザーが表示されます。ブロックは以下の場合に自動的にロック解除されます：
その人が選択をやめたとき
10分間活動がないとき
その人が下書きを離れたとき
ブロックがロック解除されると、更新されたコンテンツが全員に表示されます。
