# クイズカレンダープラグインのインストール

**URL**: https://support.staffbase.com/hc/ja/articles/4403494439314
**取得日**: 2026-05-02 18:47:29 JST

## 内容

クイズカレンダーは、重要な会社のイベント、マイルストーン、または休日に向けてユーザーを楽しませるための楽しい方法です。カレンダーの各日ごとにユーザーがアンロックできるカスタムクイズ質問、画像、またはテキストを設定できます。これにより、ユーザーは毎日戻ってくる理由ができ、プラットフォームを使用しながら楽しむことができます。クイズカレンダープラグインをインストールして、最初のカレンダーを作成しましょう。
クイズカレンダープラグインのインストール
スタジオで、コンテンツ > コンテンツメニューに移動します。
メニューの一番下までスクロールし、プラグインを追加をクリックします。
クイズカレンダーに移動し、インストールをクリックします。
クイズカレンダーを追加するためのダイアログが開きます。
クイズカレンダーのタイトルを入力します。
ドロップダウンメニューからクイズカレンダーの可視性を選択します。
クイズカレンダーを追加をクリックします。
最初のクイズカレンダーを追加しました。すぐにプラグインに移動して設定を完了します。クイズカレンダーをユーザーに表示するには、最初にクイズカレンダーを公開する必要があります。
ユーザーにとってのクイズカレンダーの見た目
これは、カスタム画像背景を持つクイズカレンダーがユーザーにどのように見えるかの例です。
