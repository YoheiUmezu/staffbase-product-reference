# アプリコンテンツのIDを特定する

**URL**: https://support.staffbase.com/hc/ja/articles/115002996811
**取得日**: 2026-05-02 19:02:07 JST

## 内容

ニュースチャンネルのID
ニュースチャンネルのIDはスタジオで確認できます。スタジオでニュースチャンネルに移動します。IDはURLの最後の部分です。
例:
https://app.yourcompany.com/admin/plugin/news/56376cf7e4b07ab9337fad37
記事、ページ、またはプラグインのID
記事、ページ、またはプラグインのIDはアプリ自体で確認できます。ウェブアプリを開き、必要なコンテンツに移動します。IDはURLの最後の部分です。
例:
https://app.yourcompany.com/content/page/news/558c017ee4b029c8c2e50dbc
