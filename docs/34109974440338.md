# Studioでプロフィールフィールドデータを編集する

**URL**: https://support.staffbase.com/hc/ja/articles/34109974440338
**取得日**: 2026-05-02 18:40:47 JST

## 内容

管理者
プロフィール フィールドは、ユーザーに関する情報を保存する領域です。Studioでユーザーのプロフィール フィールドのデータを編集できます。
SCIM、CSV import、ユーザーAPIなどの自動ユーザー同期連携を使用している場合は、変更内容を必ず維持してください。Studioでプロフィール フィールドに加えた変更は、同期元でもそのフィールドを更新しない限り、次回の同期時に上書きされます。プロフィール フィールドを編集すると、ユーザーの条件付きグループメンバーシップにも影響します。
Studioで、ユーザー > 編集したいユーザーアカウントに移動します。
三点リーダーをクリックしてアクションメニューを開きます。
編集するをクリックします。編集するページが開きます。
プロフィール フィールドタブで、編集したいフィールドの横にある鉛筆アイコンをクリックします。
新しい値を入力します。
チェックマークアイコンをクリックします。これでユーザーアカウントのプロフィール フィールドデータを編集しました。
