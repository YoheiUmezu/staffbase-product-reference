# メールデザインにビデオリンクを追加する

**URL**: https://support.staffbase.com/hc/ja/articles/4412592734610
**取得日**: 2026-05-02 18:51:54 JST

## 内容

ビデオは、迅速で消化しやすい形式でコンテンツを提供することにより、ユーザーのメールへのエンゲージメントを向上させる素晴らしい方法です。
Employee EmailのEmail Designerツールを使用すると、受信者をYouTubeやVimeoなどの外部プラットフォームにホストされているビデオにリンクするプレビュー画像をメールに追加できます。
ビデオのプレビュー画像は、プラットフォームによって使用されるデフォルトのサムネイルまたはお好みのカスタム画像にすることができます。受信者がクリックすると、新しいブラウザウィンドウでビデオが表示されるプレイボタンが表示されます。
注意: 異なるメールクライアントやサーバーには、それぞれビデオファイルの取り扱いや配信可能な最大添付ファイルサイズに関する独自のルールがあります。
できるだけ多くの受信者に対してビデオが適切に読み込まれ再生されるようにするために、Employee Emailは、画像をアップロードするのと同じ方法でメールデザインにビデオをアップロードしません。同様の理由から、メールにビデオファイルを添付として追加することも推奨されません。
前提条件
YouTubeやVimeoなどの外部プラットフォームにホストされているビデオへのリンクを持っていること。
注意: Staffbaseは、組織にファイアウォールや受信者がYouTubeなどの外部ビデオホスティングプラットフォームにアクセスできない設定がある場合は、IT部門に確認することを推奨します。
また、地域の制限や設定に応じて、内部でホストされているビデオへのリンクを共有するオプションがあるかもしれません。たとえば、安全なイントラネット上にホストされている場合です。
組織がStaffbaseのIntranetおよび/またはAppソリューションを使用している場合は、プラットフォームへのビデオやその他のメディアファイルのアップロードについて詳しく学ぶことができます。
デザインにビデオリンクを追加する
注意: ビデオ要素は、メールデザインにビデオをアップロードしません。
コンピュータにローカルに保存されているビデオがある場合は、メールデザインに追加する前に、YouTubeやVimeoなどの外部プラットフォームにビデオをアップロードする必要があります。
メールのドラフトまたはテンプレートを編集している間に、テキスト付きビデオのクイックブロックを追加します。
または、ビデオ要素を含むカスタムブロックを作成します。
ビデオ要素をクリックします。
ビデオ設定パネルが開きます。
ビデオリンクフィールドに、メールに表示したいビデオへのリンクをコピーして貼り付けます。
ビデオのサムネイルを作成する
ほとんどの受信者は、サムネイル画像があるビデオを好みます。これは、ビデオをクリックして視聴するよう促すからです。
通常、ビデオがホストされているプラットフォームはデフォルトのサムネイルを自動的に生成しますが、お好みのカスタム画像を使用するオプションもあります。
メールのドラフトまたはテンプレートを編集している間に、ビデオリンクを持つビデオ要素をクリックします。
カスタム画像をアップロードをクリックします。
サムネイル用の画像ファイルを選択します。
プレイボタンのためにホワイトまたはブラックを選択します。
アップロードした画像は、ビデオ要素に表示され、上にプレイボタンが重ねられます。
YouTubeにホストされているビデオのサムネイル画像をアップロードする場合、アップロードした画像ファイルはデフォルトのYouTubeサムネイルを置き換えます。
受信者がビデオとどのように対話するか
受信者がメール内のビデオをクリックすると、ビデオがホストされている外部プラットフォームにリダイレクトされ、そこで視聴したり、反応やコメントを追加したりできます。
注意: IT部門にファイアウォールや外部ビデオホスティングプラットフォームへのアクセスを制限する設定がある場合、受信者がビデオを視聴しようとすると問題が発生する可能性があります。
このような場合は、IT部門に連絡して、ビデオをホストするための代替方法についての支援やアドバイスを求めてください。
