# チャンネルの公開

**URL**: https://support.staffbase.com/hc/ja/articles/360028231791
**取得日**: 2026-05-02 19:02:06 JST

## 内容

ニュースページへの関連付け
公開する前に、作成したチャンネルをニュースページに関連付ける必要があります。
デフォルトでは、新しいチャンネルはメニュー内で最初に作成されたニュースページに関連付けられます。ターゲットグループにチャンネルを公開するには、手動で意図したニュースページを選択する必要があります。
チャンネルを複数のニュースページに関連付けることができます。
スタジオで、公開したいチャンネルに移動します。
歯車アイコンをクリックして設定を開きます。
チャンネルの設定ページが開きます。
関連付けられたニュースページフィールドで、チャンネルを関連付けたいニュースページを1つ以上選択します。
保存をクリックします。
チャンネルの公開
スタジオで、公開したいチャンネルに移動します。
チャンネルを公開をクリックします。
チャンネルは公開されましたが、少なくとも1つの記事が公開されている場合にのみ、ユーザーに表示されます。
