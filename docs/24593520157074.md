# CSVファイルの削除

**URL**: https://support.staffbase.com/hc/ja/articles/24593520157074
**取得日**: 2026-05-02 18:56:39 JST

## 内容

ベータ
アクティベーション
管理者
アップロードまたはインポートされたファイルは、過去30日以内であれば削除できます。30日以上前にインポートされたファイルは、自動的に概要ページから削除されます。すでにインポートされたファイルを削除してもインポートプロセスは元に戻らないことに注意してください。これは、概要ページからCSVインポート記録を削除するだけです。
スタジオで、ユーザー ユーザーインポートと同期に移動します。
削除したいファイルに対応する3つのドットのアクションメニューから、ファイルを削除をクリックします。
確認ダイアログが開きます。
ファイルを削除をクリックします。
概要ページからCSVファイルを削除しました。
