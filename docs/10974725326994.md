# ニュースチャンネルの概要の使用

**URL**: https://support.staffbase.com/hc/ja/articles/10974725326994
**取得日**: 2026-05-02 19:02:47 JST

## 内容

アクティベーション
ベータ
エディター
ニュースチャンネルの概要は、すべてのニュースチャンネルに関する情報を一目で確認できる使いやすいページです。多くのニュースチャンネルを持つ大規模な企業組織に最適で、概要を使用して既存のチャンネルを並べ替えたり表示したり、新しいチャンネルを作成したりできます。
すべてのニュースチャンネルは「すべてのコンテンツ」概要ページで表示するか、各個別のスペースの概要を確認できます。デフォルトでは、概要は最初の5つのチャンネルを表示しますが、結果のドロップダウンを使用して、5、10、25、50、または100のチャンネルを一度に表示するようにカスタマイズできます。並べ替えビューに切り替えることで、チャンネルの順序を簡単に変更することも可能です。
各ニュースチャンネルには、以下のオプションを持つ独自の行があります：
メタデータを表示
投稿数：公開済みおよびドラフトの投稿数
ユーザー：コンテンツを表示できるユーザーの数
スペース：チャンネルが見つかるスペース
最終投稿の公開日：最後に公開された投稿の日付
公開/未公開：チャンネルが公開されているか未公開か
公開：チャンネルが公開スペースに含まれている場合に表示
チャンネルを開く
チャンネルのタイトルをクリックして、投稿の詳細ビューを開き、投稿の統計とアクションを含めます
投稿を作成
チャンネル内で直接新しいドラフト投稿を作成します
アクションメニューを開く
チャンネル設定を管理します
公開ステータスを変更します
新しいチャンネルを作成
管理エディター
スタジオで、コンテンツ > ニュースに移動します。
ニュースチャンネルページが開きます。
チャンネルを作成をクリックします。
チャンネルを作成ダイアログが開きます。
名前を入力し、チャンネルタイプを選択します。
組織がスペースを使用している場合、すべてのコンテンツエリアから新しいチャンネルを作成する際には、スペースも選択する必要があります。
保存をクリックします。
新しいチャンネルの設定ページが開きます。
新しいチャンネルが作成されました。概要ページの最後の結果として表示されます。
チャンネルに新しい投稿を作成
スタジオで、コンテンツ > ニュースに移動します。
ニュースチャンネルページが開きます。
投稿を追加したいチャンネルに移動します。
プラスアイコンをクリックします。
記事を作成ドラフトページが開きます。
チャンネルの概要から新しいドラフト投稿を作成しました。いつでもチャンネルまたはチャンネルの概要に戻ることができます。
チャンネル設定を開く
スタジオで、コンテンツ > ニュースに移動します。
ニュースチャンネルページが開きます。
設定を開きたいチャンネルに移動します。
三点リーダーをクリックします。
アクションメニューが開きます。
設定をクリックします。
チャンネル設定が開きます。
チャンネル設定に移動しました。設定を更新するか、チャンネルまたはチャンネルの概要に戻ります。
チャンネルを公開
スタジオで、コンテンツ > ニュースに移動します。
ニュースチャンネルページが開きます。
未公開のチャンネルに移動します。
三点リーダーをクリックします。
アクションメニューが開きます。
公開をクリックします。
チャンネルを公開ダイアログが開きます。
公開をクリックします。
チャンネルを公開しました。これで、従業員に表示されるようになりました。概要には公開済みラベルが表示されます。
チャンネルを未公開にする
スタジオで、コンテンツ > ニュースに移動します。
ニュースチャンネルページが開きます。
公開済みのチャンネルに移動します。
三点リーダーをクリックします。
アクションメニューが開きます。
未公開にするをクリックします。
チャンネルを未公開にするダイアログが開きます。
未公開にするをクリックします。
チャンネルを未公開にしました。これで、従業員には表示されなくなりました。概要には未公開ラベルが表示されます。
ニュースチャンネルを並べ替える
公開済みおよび未公開のチャンネルを並べ替えたり再配置したりすることができます。これにより、組織に最適な方法でチャンネルを整理できます。
ニュースプラグイン内のチャンネルの順序を変更すると、従業員アプリやイントラネットのメニュー内での順序も変更されます。この変更はすべてのユーザーに表示されます。
スタジオで、コンテンツ > ニュースに移動します。
ニュースチャンネルページが開きます。
概要の下部にある並べ替えビューに切り替えをクリックします。
並べ替えビューが開きます。
チャンネルをドラッグ＆ドロップして順序を変更します。
順序は自動的に更新されます。
概要に戻るをクリックします。
ニュースチャンネルの概要が新しいチャンネルの順序で更新されます。
ニュースチャンネルを並べ替え、ニュースチャンネルの順序を変更しました。いつでもチャンネルを再度並べ替えることができます。
