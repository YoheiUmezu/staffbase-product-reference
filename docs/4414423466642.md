# Microsoft Connections のための Azure アプリの権限付与

**URL**: https://support.staffbase.com/hc/ja/articles/4414423466642
**取得日**: 2026-05-02 18:54:01 JST

## 内容

Microsoft 365 の検索とウィジェットを Staffbase プラットフォームに接続し、Microsoft アプリからのコンテンツを表示するために、Microsoft Connections に追加した Microsoft Azure アプリに権限を付与する必要があります。これらの権限により、Azure アプリは Microsoft プラットフォームからコンテンツを取得し、従業員アプリまたはプラットフォームに表示できるようになります。
前提条件
Studio で Microsoft 365 Connections を追加しています。
あなたは Azure グローバル管理者です。
Staffbase は、Microsoft 365 の検索とウィジェットを一緒に有効化し、すべてのサービスに対して一度に権限を付与することを推奨します。そうしないと、有効化後に再度権限を付与する必要があります。
Staffbase プラットフォームのユーザービューで、設定 > アカウントに移動します。
接続されたアカウントの下で、Microsoft 365 の サインイン をクリックします。
Microsoft 365 にサインインするためのダイアログが開きます。
サインインダイアログで、Microsoft 365 アカウントの詳細を入力するか、アカウントを選択します。
サインインをクリックします。
権限リクエストのダイアログが開きます。
組織を代表して同意するのチェックボックスを選択し、受け入れるをクリックします。
権限はいつでも変更できます。詳細を学ぶ。
あなたは Staffbase プラットフォームに対して Microsoft Connections に権限を付与しました。
