# ページのオフラインモードに関する概要とFAQ

**URL**: https://support.staffbase.com/hc/ja/articles/23615493853074
**取得日**: 2026-05-02 18:42:57 JST

## 内容

ベータ
アクティベーション
オフラインモードを使用すると、従業員は従業員アプリでページをダウンロードでき、インターネット接続なしで関連情報にアクセスできます。ユーザーは、安全プロトコルや人事ガイドラインなど、オフラインで読むための特定のページをダウンロードできます。ダウンロードされたコンテンツ（テキスト、画像、PDFを含む）は、オフライン時に整理された概要ページを介してアクセス可能です。
オフラインモードは、遠隔地、倉庫、またはオフショアの場所にいる従業員にとって実用的です。ユーザーは、自分の役割に関連するコンテンツのみをダウンロードでき、迅速なアクセスを確保します。ページのオフラインモードについて詳しく学ぶ。 これにより、接続が限られている地域でも重要な情報へのアクセスを維持することで、コンプライアンスと運用効率がサポートされます。
アクセシビリティの向上
オフラインモードにより、従業員はネットワーク接続のない場所（遠隔地、倉庫、オフショアプラットフォームなど）でも重要な情報にいつでもアクセスできます。これにより、重要なリソースへの途切れのないアクセスが保証されます。
パーソナライズされたコンテンツ管理
従業員は、自分の役割に関連するページ（ツールの指示やトレーニング資料など）を選択してダウンロードでき、カスタマイズされた体験を提供します。この柔軟性により、ユーザーは最も必要なコンテンツを優先できます。
安全で信頼性の高い機能
ダウンロードされたコンテンツはユーザーのアプリに安全に保存され、データが保護されます。オフラインモードは、セキュリティ対策（セッションタイムアウトやユーザー認証など）を維持しながら、コンプライアンスと運用ニーズをサポートします。
FAQ
オフラインで使用するために何をダウンロードできますか？
オフラインで読むためのページをダウンロードできます。ダウンロードされたページは、アプリをオフラインで開いたときに「オフラインで利用可能」リストに表示されます。
ダウンロードされたコンテンツはどこに保存されますか？
ダウンロードされたデータは、ユーザーのデバイスに暗号化されて保存されます。
ユーザーがログアウトしたり、セッションがタイムアウトした場合はどうなりますか？
ユーザーは以前にダウンロードしたコンテンツにアクセスできません。これにより、コンテンツの不正アクセスが防止されます。
ユーザーが新しいモバイルデバイスに切り替えた場合はどうなりますか？
ダウンロードされたページは新しいデバイスに転送されません。ユーザーは再度ページをダウンロードする必要があります。
組織に短いセッションタイムアウトが実装されている場合はどうなりますか？
ユーザーが以前にダウンロードしたコンテンツにアクセスできなくなる短いセッションタイムアウトの場合、オフラインモードは組織に適していない可能性があります。セキュリティはStaffbaseにとって高い優先事項です。
ユーザーが組織を退職した場合はどうなりますか？
ユーザーが組織を退職し、ユーザーアカウントが無効化されると、ユーザーはオフラインのままでダウンロードしたコンテンツにアクセスできます。次回ユーザーがオンラインでアプリを開くと、アプリはブロックされます。その後、ユーザーはダウンロードしたコンテンツにアクセスできなくなります。
特定のページのみオフラインモードにすることは可能ですか？
現時点では不可能です。オフラインモードは、組織のアプリ内のすべてのページに自動的に適用され、ユーザーがダウンロードできるようになります。
ダウンロードされたページに表示されるメディアの種類は何ですか？
オフラインモードは、テキスト、画像、PDFをサポートしています。サポートされている画像ファイル形式は、pdf、jpeg、png、webP、gif、svgです。
ユーザーがオフラインの間にニュース記事などアプリの別のエリアを開こうとした場合はどうなりますか？
オフラインのユーザーはダウンロードされたページのみを表示でき、アプリの他の部分と対話することはできません。
