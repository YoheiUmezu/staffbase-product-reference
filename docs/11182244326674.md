# Staffbase PnP モダン検索拡張ライブラリの概要

**URL**: https://support.staffbase.com/hc/ja/articles/11182244326674
**取得日**: 2026-05-02 18:43:40 JST

## 内容

アクティベーション
Staffbase PnP モダン検索拡張ライブラリは、PnP モダン検索 ウェブパーツと Staffbase ニュース API を活用して、SharePoint 内の Staffbase ニュースを検索します。ユーザーはキーワードを入力して Staffbase ニュース投稿を見つけ、検索結果から投稿を開くことができます。この統合を使用して、ビジネスのユースケースに応じて、ユーザーの検索体験を向上させることができます。
この統合を使用して、ビジネスのユースケースをサポートします：
SharePoint 内に Staffbase ニュースの検索ハブを作成します。
ページに専用の検索ボックスを追加します。ユーザーがキーワードを検索すると、検索ページにリダイレクトされます。
最新の更新または公開された Staffbase ニュースを表示する結果ページを追加します。
Staffbase ニュースを独自の検索結果タブに表示します。
検索表示の外観をカスタマイズして、企業のブランドアイデンティティに合わせます。
