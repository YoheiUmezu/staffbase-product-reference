# Staffbaseメールにおけるブロックターゲティングに関するよくある質問（FAQ）

**URL**: https://support.staffbase.com/hc/ja/articles/32661366270866
**取得日**: 2026-05-02 18:53:03 JST

## 内容

ダイナミックブロックターゲティングはどこで利用できますか？
テキストエディターまたはメールデザイナーで作成されたメールの下書きで利用できます。
誰がブロックターゲティングの設定を編集できますか？
メールの下書きを編集する権限を持つユーザーは、コンテンツブロックに割り当てられたターゲット視聴者も編集できます。
メールテンプレートでブロックをターゲティングできますか？
いいえ、ブロックターゲティングは下書きメールでのみ利用できます。メールの下書きをテンプレートとして保存した場合、ブロックターゲティングの設定は保存されません。
スケジュール済みメールのブロックターゲティングを変更できますか？
はい、スケジュール済みメールのブロックターゲティングを変更することができます。
CSVでアップロードしたカスタムターゲットグループを持つメールでダイナミックブロックターゲティングを利用できますか？
いいえ、メールのターゲット視聴者はすべてのフォルダユーザーまたはユーザーグループとユーザーである必要があります。
複数のブロックを同時にターゲティングできますか？
いいえ、1回につき1つのブロックのみを特定のユーザーまたはユーザーグループにターゲティングできます。
1つのブロックに割り当てられるユーザーやユーザーグループの数に制限はありますか？
いいえ、制限はありません。
ターゲティングに選択できるユーザーやユーザーグループはどれですか？
ダイナミックブロックターゲティングは既存のフォルダ視聴者に基づいています。ブロックをターゲティングする際、メール自体がより限定された視聴者に送信される場合でも、フォルダの全視聴者から選択できます。ブロックは、そのターゲット視聴者が実際のメールのターゲット視聴者に含まれている場合のみ送信されます。
ユーザーが複数のターゲットユーザーグループに属している場合はどうなりますか？
ユーザーが複数のユーザーグループに属し、それぞれのグループに異なるブロックがターゲティングされている場合、そのユーザーは自分のグループにターゲティングされたすべてのブロックと、すべてのユーザーにターゲティングされたすべてのブロックを受け取ります。
ターゲティングされたブロックをプレビューするにはどうすればよいですか？
特定のユーザーやユーザーグループでプレビューする場合、プレビューには以下が表示されます：
すべてのユーザーにターゲティングされたブロック
そのユーザーやグループに特定にターゲティングされたブロック
ターゲティングされたコンテンツのテストメールを送信できますか？
はい。以下のテストメールを送信できます：
ブロックターゲティングで使用されている特定のユーザーやユーザーグループ
すべてのブロックを含むバージョン
「すべてのユーザー」にターゲティングされたすべてのブロックを含むバージョン
追加情報：
テストメールには、選択されたユーザーやユーザーグループのリストを示す情報バナーが含まれます。
プレビュービューからテストメールを送信する場合、選択したプレビュー視聴者が自動的に適用されます。
ユーザーやユーザーグループが空のメールを受け取ることはありますか？
はい、各受信者は自分にターゲティングされたブロックのみを受け取ります。自分にターゲティングされたブロックがない場合、メールは空になります。そのため、少なくとも1つのブロックをすべてのユーザーにターゲティングすることを強く推奨します。
メールの目次にはすべてのブロックが表示されますか？
はい、目次にはターゲティングによってメールに含まれないブロックも表示されます。回避策として、各ターゲットグループごとに目次を作成し、そのグループに合わせて手動で調整し、該当するブロックをそのターゲットグループにターゲティングする方法があります。
ダイナミックブロックターゲティングは多言語メールでどのように機能しますか？
ターゲティングされたブロックを含むメールをコピーまたは翻訳すると、ブロックターゲティングの設定は新しい言語バージョンにも自動的に引き継がれます。コピー後に言語バージョンで行った変更は、他の言語バージョンには自動的に適用されません。
メールを複製した場合はどうなりますか？
メールの下書きを複製すると、すべてのブロックターゲティング設定が引き継がれます。
HTMLダウンロードには何が含まれますか？
HTMLダウンロードには、ダウンロード時に表示されているコンテンツのみが含まれます。
たとえば、特定のユーザーグループ向けのコンテンツをプレビューしている状態でHTMLをダウンロードすると、そのバージョンのみがダウンロードに含まれます。
ターゲティングされたブロックを含むメールのパフォーマンスはどのように測定されますか？
メールアナリティクスやクリックマップは、選択したユーザーグループごとに表示できます。クリックマップには、選択したグループが受信したコンテンツのみが反映されます。グループ単位でのみフィルタリングでき、個別の受信者単位ではできません。
