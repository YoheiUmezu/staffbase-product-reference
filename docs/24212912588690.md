# スタッフベースメールでStudioユーザーを作成する

**URL**: https://support.staffbase.com/hc/ja/articles/24212912588690
**取得日**: 2026-05-02 18:53:28 JST

## 内容

この記事はAIを使用して翻訳されており、不正確な情報が含まれている可能性があります。情報に不安がある場合は、正確性のために英語の原本を参照してください。
管理者
Staffbase EmailをStaffbase Intranetまたは従業員アプリと一緒に使用する場合は、ユーザーの作成を参照してください。以下の情報は、Staffbase Emailをスタンドアロン製品として使用する場合にのみ適用されます。
ユーザーを作成する前に、Staffbase Emailの権限と役割を理解する必要があります。プラットフォームで1人または2人のユーザーを作成する場合は、Studioユーザーを手動で作成できます。大量作成の場合は、CSVインポートを使用できます。
前提条件:
組織に必要なプロフィールフィールドを設定しています。
Studioでユーザーに移動します。
ユーザーを作成をクリックします。
メールでユーザーを作成して招待が開きます。
名と姓を入力します。
ユーザーの名と姓の両方を必ず入力してください。姓が欠けていると、役割に基づいてStudioで適切な権限を受け取れない可能性があります。
メールフィールドに有効なメールアドレスを入力します。
役割のドロップダウンから役割を選択します。
作成して招待をクリックします。
新しいユーザーを作成しました。ユーザーは保留中のユーザー状態でリストされます。ユーザーはアカウントを登録するためのメールを受け取ります。ユーザーがアカウントを登録すると、ユーザーの状態は登録済みに変わります。
