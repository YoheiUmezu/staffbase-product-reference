# パンくずナビゲーション

**URL**: https://support.staffbase.com/hc/ja/articles/4414905352082
**取得日**: 2026-05-02 18:44:08 JST

## 内容

アクティベーション
パンくずリストは、ユーザーインターフェースにおけるナビゲーション補助です。これにより、ユーザーは内部コミュニケーションプラットフォーム内での自分の位置を把握できます。メニュー構造に基づいて、ニュース投稿、ページ、またはその他の場所にいる場合でも、ユーザーのコンテンツ内での閲覧位置を示します。ユーザーは、プラットフォーム内の自分の位置を一目で把握できます。パンくずリストは、ウェブアプリおよびほとんどのコンテンツタイプで利用可能です。パンくずリストは、ブックマーク、チャット、従業員ディレクトリ、エクスペリエンススタジオ、設定、およびユーザープロファイルでは利用できません。
個々のニュースチャンネルに対して、ユーザーに対するパンくずリストの表示方法を管理することが重要です。
